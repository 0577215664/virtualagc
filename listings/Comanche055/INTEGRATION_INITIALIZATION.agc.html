056613,000035:                                                                                                 #  ----------------
056614,000036: 
056615,000037:                                                                                                 #  FROM A USER'S POINT OF VIEW, ORBITAL INTEGRATION IS ESSENTIALLY THE SAME AS THE 278 INTEGRATION
056616,000038:                                                                                                 #  PROGRAM.  THE SAME ENTRANCES TO THE PROGRAM WILL BE MAINTAINED, THE SAME STALLING ROUTINE WILL BE USED AND
056617,000039:                                                                                                 #  OUTPUT WILL STILL BE VIA THE PUSHLIST.  THE PRIMARY DIFFERENCES TO A USER INVOLVE THE ADDED CAPABILITY OF
056618,000040:                                                                                                 #  TERMINATING INTEGRATION AT A SPECIFIC FINAL RADIUS AND THE DIFFERENCE IN STATE VECTOR SCALING INSIDE AND OUTSIDE
056619,000041:                                                                                                 #  THE LUNAR SPHERE OF INFLUENCE.
056620,000042: 
056621,000043:                                                                                                 #  IN ORDER TO MAKE THE CSM(LEM)PREC AND CSM(LEM)CONIC ENTRANCES SIMILAR TO FLIGHT 278, THE INTEGRATION PROGRAM
056622,000044:                                                                                                 #  WILL ITSELF SET THE FINAL RADIUS (RFINAL) TO 0 SO THAT REACHING THE DESIRED TIME ONLY WILL TERMINATE
056623,000045:                                                                                                 #  INTEGRATION.  THE DP REGISTER RFINAL MUST BE SET BY USERS OF INTEGRVS AND INTEGRV, AND MUST BE DONE AFTER THE
056624,000046:                                                                                                 #  CALL TC INTSTALL.
056625,000047: 
056626,000048:                                                                                                 #  WHEN THE LM IS ON THE LUNAR SURFACE (INDICATED BY LUNAR SURFACE FLAG SET) CALLS TO LEMCONIC, LEMPREC, AND
056627,000049:                                                                                                 #  INTEGRV WITH VINFLAG = 0 WILL RESULT IN THE USE OF THE PLANETARY INERTIAL ORIENTATION SUBROUTINES TO PROVIDE
056628,000050:                                                                                                 #  BOTH THE LM'S POSITION AND VELOCITY IN THE REFERENCE COORDINATE SYSTEM.
056629,000051:                                                                                                 #  THE PROGRAM WILL PROVIDE OUTPUT AS IF INTEGRATION WAS USED.  THAT IS, THE PUSHLIST WILL BE SET AS NOTED BELOW AND
056630,000052:                                                                                                 #  THE PERMANENT STATE VECTOR UPDATED WHEN SPECIFIED BY AN INTEGRV CALL.
056631,000053: 
056632,000054:                                                                                                 #  USERS OF INTEGRVS DESIRING INTEGRATION (INTYPFLG = 0) SHOULD NOTE THAT THE OBLATENESS PERTURBATION COMPUTATION
056633,000055:                                                                                                 #  IN LUNAR ORBIT IS TIME DEPENDENT.  THEREFORE, THE USER SHOULD SUPPLY AN INITIAL STATE VECTOR VALID AT SOME REAL
056634,000056:                                                                                                 #  TIME AND THE DESIRED TIME (TDEC1) ALSO AT SOME REAL TIME.  FOR CONIC "INTEGRATION" THE USER MAY STILL USE ZERO
056635,000057:                                                                                                 #  AS THE INITIAL TIME AND DELTA TIME AS THE DESIRED TIME.

056637,000059:                                                                                                 #  2.0 CENTRAL DESCRIPTION
056638,000060:                                                                                                 #  -----------------------
056639,000061: 
056640,000062:                                                                                                 #  THE INTEGRATION PROGRAM OPERATES AS A CLOSED INTERPRETIVE SUBROUTINE AND PERFORMS THESE FUNCTIONS --
056641,000063:                                                                                                 #         1) INTEGRATES (PRECISION OR CONIC) EITHER CSM OR LM STATE VECTOR
056642,000064:                                                                                                 #         2) INTEGRATES THE W-MATRIX
056643,000065:                                                                                                 #         3) PERMANENT OR TEMPORARY UPDATE OF THE STATE VECTOR
056644,000066: 
056645,000067:                                                                                                 #  THERE ARE SIX ENTRANCES TO THE INTEGRATION PROGRAM.  FOUR OF THESE (CSMPREC, LEMPREC, CSMCONIC, LEMCONIC) SET
056646,000068:                                                                                                 #  ALL THE FLAGS REQUIRED IN THE INTEGRATION PROGRAM ITSELF TO CAUSE THE PRECISION OR CONIC INTEGRATION (KEPLER) OF
056647,000069:                                                                                                 #  THE LM OR CSM STATE VECTOR, AS THE NAMES SUGGEST.  ONE ENTRANCE (INTEGRVS) PERMITS THE CALLING PROGRAM TO
056648,000070:                                                                                                 #  PROVIDE A STATE VECTOR TO BE INTEGRATED.  THE CALLING PROGRAM MUST SET THE FLAGS INDICATING (1) PRECISION OR
056649,000071:                                                                                                 #  CONIC INTEGRATION, (2) IN OR OUT OF LUNAR SPHERE, (3) MIDCOURSE OR NOT, AND THE INTEGRATION PROGRAM COMPLETES
056650,000072:                                                                                                 #  THE FLAG SETTING TO BYPASS W-MATRIX INTEGRATION.  THE LAST ENTRANCE (INTEGRV, USED IN GENERAL BY THE
056651,000073:                                                                                                 #  NAVIGATION PROGRAMS) PERMITS THE CALLER TO SET FIVE FLAGS (NOT MOONFLAG OR MIDFLAG) BUT NOT TO INPUT A STATE
056652,000074:                                                                                                 #  VECTOR.  ANY PROGRAM WHICH CALLS INTEGRVS OR INTEGRV MUST CALL INTSTALL BEFORE IT SETS THE INTEGRATION FLAGS
056653,000075:                                                                                                 #  AND/OR STATE VECTOR.
056654,000076: 
056655,000077:                                                                                                 #  THREE SETS OF 42 REGISTERS AND 2 FLAGS ARE USED FOR THE STATE VECTORS.  TWO SETS, WHICH MAY NOT BE OVERLAYED, ARE
056656,000078:                                                                                                 #  USED FOR THE PERMANENT STATE VECTORS FOR THE CSM AND LM.  THE THIRD SET, WHICH MAY BE OVERLAYED WHEN INTEGRATION
056657,000079:                                                                                                 #  IS NOT BEING DONE, IS USED IN THE COMPUTATIONS.
056658,000080: 
056659,000081:                                                                                                 #  THE PERMANENT STATE VECTORS WILL BE PERIODICALLY UPDATED SO THAT THE VECTORS WILL NOT BE OLDER THAN 4 TIMESTEPS.
056660,000082:                                                                                                 #  THE PERMANENT STATE VECTORS WILL ALSO BE UPDATED WHENEVER THE W-MATRIX IS INTEGRATED OR WHEN A CALLER OF INTEGRV
056661,000083:                                                                                                 #  SETS STATEFLG (THE NAVIGATION PROGRAMS P20, P22.)
056662,000084: 
056663,000085:                                                                                                 #  Page 1310
056664,000086:                                                                                                 #  APPENDIX B OF THE USERS' GUIDE LISTS THE STATE VECTOR QUANTITIES.
056665,000087: 
056666,000088:                                                                                                 #  2.1 RESTARTS
056667,000089: 
056668,000090:                                                                                                 #  PHASE CHANGES WILL BE MADE IN THE INTEGRATION PROGRAM ONLY FOR THE INTEGRV ENTRANCE (I.E., WHEN THE W-MATRIX IS
056669,000091:                                                                                                 #  INTEGRATED OR PERMANENT STATE VECTOR IS UPDATED.)  THE GROUP NUMBER USED WILL BE THAT FOR THE P20-25 PROGRAMS
056670,000092:                                                                                                 #  (I.E., GROUP2) WINCE THE INTEGRV ENTRANCE WILL ONLY BE USED BY THESE PROGRAMS.  IF A RESTART OCCURS DURING AN
056671,000093:                                                                                                 #  INTEGRATION OF THE STATE VECTOR ONLY, THE RECOVERY WILL BE TO THE LAST PHASE IN THE CALLING PROGRAM.  CALLING
056672,000094:                                                                                                 #  PROGRAMS WHICH USE THE INTEGRV OR INTEGRVS ENTRANCE OF INTEGRATION WHOULD ENSURE THAT IF PHASE CHANGING IS DONE
056673,000095:                                                                                                 #  THAT IT IS PRIOR TO SETTING THE INTEGRATION INPUTS IN THE PUSHLIST.
056674,000096:                                                                                                 #  THIS IS BECAUSE THE PUSHLIST IS LOST DURING A RESTART.
056675,000097: 
056676,000098:                                                                                                 #  2.2 SCALING
056677,000099: 
056678,000100:                                                                                                 #  THE INTEGRATION ROUTIEN WILL MAINTAIN THE PERMANENT MEMORY STATE VECTORS IN THE SCALING AND UNITS DEFINED IN
056679,000101:                                                                                                 #  APPENDIX B OF THE USERS' GUIDE.  THE SCALING OF THE OUTPUT POSITION VECTOR DEPENDS ON THE ORIGIN OF THE COORDINATE
056680,000102:                                                                                                 #  SYSTEM AT THE DESIRED INTEGRATION TIME.  THE COORDINATE SYSTEM TRANSFORMATION WILL BE DONE AUTOMATICALLY ON
056681,000103:                                                                                                 #  MULTIPLE TIMESTEP ENCKE INTEGRATION ONLY.  THUS IT IS POSSIBLE TO HAVE OUTPUT FROM SUCCESSIVE INTEGRATIONS IN
056682,000104:                                                                                                 #  DIFFERENT SCALING.
056683,000105:                                                                                                 #  HOWEVER, RATT, VATT WILL ALWAYS BE SCALED THE SAME.
