056964,000042:                                                                                                  #  1.0 INTRODUCTION
056965,000043:                                                                                                  #  ----------------
056966,000044: 
056967,000045:                                                                                                  #  FROM A USERS POINT OF VIEW, ORBITAL INTEGRATION IS ESSENTIALLY THE SAME AS THE 278 INTEGRATION
056968,000046:                                                                                                  #  PROGRAM.  THE SAME ENTRANCES TO THE PROGRAM WILL BE MAINTAINED, THE SAME STALLING ROUTINE WILL BE USED AND
056969,000047:                                                                                                  #  OUTPUT WILL STILL BE VIA THE PUSHLIST.  THE PRIMARY DIFFERENCES TO A USER INVOLVE THE ADDED CAPABILITY OF
056970,000048:                                                                                                  #  TERMINATING INTEGRATION AT A SPECIFIC FINAL RADIUS AND THE DIFFERENCE IN STATE VECTOR SCALING INSIDE AND OUT-
056971,000049:                                                                                                  #  SIDE THE LUNAR SPHERE OF INFLUENCE.
056972,000050: 
056973,000051:                                                                                                  #  IN ORDER TO MAKE THE CSM(LEM)PREC AND CSM(LEM)CONIC ENTRANCES SIMILAR TO FLIGHT 278, THE INTEGRATION PROGRAM
056974,000052:                                                                                                  #  WILL ITSELF SET THE FINAL RADIUS (RFINAL) TO 0 SO THAT REACHING THE DESIRED TIME ONLY WILL TERMINATE
056975,000053:                                                                                                  #  INTEGRATION.  THE DP REGISTER RFINAL MUST BE SET BY USERS OF INTEGRVS AND INTEGRV, AND MUST BE DONE AFTER THE
056976,000054:                                                                                                  #  CALL TC INTSTALL.
056977,000055: 
056978,000056:                                                                                                  #  WHEN THE LM IS ON THE LUNAR SURFACE (INDICATED BY LUNAR SURFACE FLAG SET) CALLS TO LEMCONIC, LEMPREC, AND
056979,000057:                                                                                                  #  INTEGRV WITH VINFLAG = 0 WILL RESULT IN THE USE OF THE PLANETARY INERTIAL ORIENTATION SUBROUTINES TO PROVIDE
056980,000058:                                                                                                  #  BOTH THE LMS POSITION AND VELOCITY IN THE REFERENCE COORDINATE SYSTEM.
056981,000059:                                                                                                  #  THE PROGRAM WILL PROVIDE OUTPUT AS IF INTEGRATION WAS USED.  THAT IS, THE PUSHLIST WILL BE SET AS NOTED BELOW AND
056982,000060:                                                                                                  #  THE PERMANENT STATE VECTOR UPDATED WHEN SPECIFIED BY AN INTEGRV CALL.
056983,000061: 
056984,000062:                                                                                                  #  USERS OF INTEGRVS DESIRING INTEGRATION (INTYPFLG = 0) SHOULD NOTE THAT THE OBLATENESS PERTURBATION COMPUTATION
056985,000063:                                                                                                  #  IN LUNAR ORBIT IS TIME DEPENDENT.  THEREFORE, THE USER SHOULD SUPPLY AN INITIAL STATE VECTOR VALID AT SOME REAL
056986,000064:                                                                                                  #  TIME AND THE DESIRED TIME (TDEC1) ALSO AT SOME REAL TIME.  FOR CONIC ,,INTEGRATION,, THE USER MAY STILL USE ZERO
056987,000065:                                                                                                  #  AS THE INITIAL TIME AND DELTA TIME AS THE DESIRED TIME.
056988,000066: 
056989,000067:                                                                                                  #  2.0 GENERAL DESCRIPTION
056990,000068:                                                                                                  #  -----------------------
056991,000069: 
056992,000070:                                                                                                  #  THE INTEGRATION PROGRAM OPERATES AS A CLOSED INTERPRETIVE SUBROUTINE AND PERFORMS THESE FUNCTIONS ---
056993,000071:                                                                                                  #        1) INTEGRATES (PRECISION OR CONIC) EITHER CSM OR LM STATE VECTOR
056994,000072:                                                                                                  #        2) INTEGRATES THE W-MATRIX
056995,000073:                                                                                                  #        3) PERMANENT OR TEMPORARY UPDATE OF THE STATE VECTOR
056996,000074: 
056997,000075:                                                                                                  #  THERE ARE SIX ENTRANCES TO THE INTEGRATION PROGRAM.  FOUR OF THESE (CSMPREC, LEMPREC, CSMCONIC, LEMCONIC) SET
056998,000076:                                                                                                  #  ALL THE FLAGS REQUIRED IN THE INTEGRATION PROGRAM ITSELF TO CAUSE THE PRECISION OR CONIC INTEGRATION (KEPLER) OF
056999,000077:                                                                                                  #  THE LM OR CSM STATE VECTOR, AS THE NAMES SUGGEST.  ONE ENTRANCE (INTEGRVS) PERMITS THE CALLING PROGRAM TO
057000,000078:                                                                                                  #  PROVIDE A STATE VECTOR TO BE INTEGRATED.  THE CALLING PROGRAM MUST SET THE FLAGS INDICATING (1) PRECISION OR
057001,000079:                                                                                                  #  CONIC INTEGRATION, (2) IN OR OUT OF LUNAR SPHERE, (3) MIDCOURSE OR NOT, AND THE INTEGRATION PROGRAM COMPLETES
057002,000080:                                                                                                  #  THE FLAG SETTING TO BYPASS W-MATRIX INTEGRATION.  THE LAST ENTRANCE (INTEGRV, USED IN GENERAL BY THE
057003,000081:                                                                                                  #  NAVIGATION PROGRAMS) PERMITS THE CALLER TO SET FIVE FLAGS (NOT MOONFLAG OR MIDFLAG) BUT NOT TO INPUT A STATE
057004,000082:                                                                                                  #  VECTOR.  ANY PROGRAM WHICH CALLS INTEGRVS OR INTEGRV MUST CALL INTSTALL BEFORE IT SETS THE INTEGRATION FLAGS
057005,000083:                                                                                                  #  AND/OR STATE VECTOR.
057006,000084: 
057007,000085:                                                                                                  #  THREE SETS OF 42 REGISTERS AND 2 FLAGS ARE USED FOR THE STATE VECTORS.  TWO SETS, WHICH MAY NOT BE OVERLAYED, ARE
057008,000086:                                                                                                  #  USED FOR THE PERMANENT STATE VECTORS FOR THE CSM AND LM.  THE THIRD SET, WHICH MAY BE OVERLAYED WHEN INTEGRATION
057009,000087:                                                                                                  #  IS NOT BEING DONE, IS USED IN THE COMPUTATIONS.
057010,000088: 
057011,000089:                                                                                                  #  THE PERMANENT STATE VECTORS WILL BE PERIODICALLY UPDATED SO THAT THE VECTORS WILL NOT BE OLDER THAN 4 TIMESTEPS.
057012,000090:                                                                                                  #  THE PERMANENT STATE VECTORS WILL ALSO BE UPDATED WHENEVER THE W-MATRIX IS INTEGRATED OR WHEN A CALLER OF INTEGRV
057013,000091:                                                                                                  #  SETS STATEFLG (THE NAVIGATION PROGRAMS P20, P22.)

057021,000099:                                                                                                  #  INTEGRATED OR PERMANENT STATE VECTOR IS UPDATED.)  THE GROUP NUMBER USED WILL BE THAT FOR THE P20-25 PROGRAMS

057025,000103:                                                                                                  #  THAT IT IS PRIOR TO SETTING THE INTEGRATION INPUTS IN THE PUSHLIST.

057028,000106:                                                                                                  #  2.2 SCALING
057029,000107: 
057030,000108:                                                                                                  #  THE INTEGRATION ROUTINE WILL MAINTAIN THE PERMANENT MEMORY STATE VECTORS IN THE SCALING AND UNITS DEFINED IN
057031,000109:                                                                                                  #  APPENDIX B OF THE USERS GUIDE.  THE SCALING OF THE OUTPUT POSITION VECTOR DEPENDS ON THE ORIGIN OF THE COORDINATE
057032,000110:                                                                                                  #  SYSTEM AT THE DESIRED INTEGRATION TIME.  THE COORDINATE SYSTEM TRANSFORMATION WILL BE DONE AUTOMATICALLY ON
057033,000111:                                                                                                  #  MULTIPLE TIMESTEP ENCKE INTEGRATION ONLY.  THUS IT IS POSSIBLE TO HAVE OUTPUT FROM SUCCESSIVE INTEGRATIONS IN
057034,000112:                                                                                                  #  DIFFERENT SCALING.
057035,000113:                                                                                                  #  HOWEVER, RATT, VATT WILL ALWAYS BE SCALED THE SAME.
