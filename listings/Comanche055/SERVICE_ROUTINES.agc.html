Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Colossus 249 (Apollo 9) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 055 OF AGC PROGRAM COMANCHE BY NASA 2021113-051
	10:28 APR.  1,1969
Note that the date is the date of the printout, not the date of the program revision.

064705,000084:                                                                                                  #  UPFLAG AND DOWNFLAG ARE ENTIRELY GENERAL FLAG SETTING AND CLEARING SUBROUTINES.  USING THEM, WHETHER OR
064706,000085:                                                                                                  #  NOT IN INTERRUPT, ONE MAY SET OR CLEAR ANY SINGLE, NAMED BIT IN ANY ERASABLE REGISTER, SUBJECT OF COURSE TO
064707,000086:                                                                                                  #  EBANK SETTING.  A "NAMED" BIT, AS THE WORD IS USED HERE, IS ANY BIT WITH A NAME FORMALLY ASSIGNED BY THE YUL
064708,000087:                                                                                                  #  ASSEMBLER.
064709,000088: 
064710,000089:                                                                                                  #  AT PRESENT THE ONLY NAMED BITS ARE THOSE IN THE FLAGWORDS.  ASSEMBLER CHANGES WILL MAKE IT POSSIBLE TO
064711,000090:                                                                                                  #  NAME ANY BIT IN ERASABLE MEMORY.
064712,000091: 
064713,000092:                                                                                                  #  CALLING SEQUENCES ARE AS FOLLOWS:-
064714,000093:                                                                                                  #                TC      UPFLAG                  TC      DOWNFLAG
064715,000094:                                                                                                  #                ADRES   NAME OF FLAG            ADRES   NAME OF FLAG
064716,000095: 
064717,000096:                                                                                                  #  RETURN IS TO THE LOCATION FOLLOWING THE "ADRES" ABOUT .58 MS AFTER THE "TC".

064821,000200:                                                                                                  #  GENTRAN, A BLOCK TRANSFER ROUTINE.
064822,000201:                                                                                                  #  WRITTEN BY D. EYLES
064823,000202:                                                                                                  #  MOD 1 BY KERNAN                               UTILITYM REV 17 11/18/67
064824,000203:                                                                                                  #  MOD 2 BY SCHULENBERG  (REMOVE RELINT)  SKIPPER REV 4 2/28/68
064825,000204: 
064826,000205:                                                                                                  #  THIS ROUTINE IS USEFULL FOR TRANSFERING N CONSECUTIVE ERASABLE OR FIXED QUANTITIES TO SOME OTHER N
064827,000206:                                                                                                  #  CONSECUTIVE ERASABLE LOCATIONS.  IF BOTH BLOCKS OF DATA ARE IN SWITCHABLE EBANKS, THEY MUST BE IN THE SAME ONE.
064828,000207: 
064829,000208:                                                                                                  #  GENTRAN IS CALLABLE IN A JOB AS WELL AS A RUPT.  THE CALLING SEQUENCE IS:
064830,000209:                                                                                                  #        I       CA      N-1             # OF QUANTITIES MINUS ONE.
064831,000210:                                                                                                  #        I +1    TC      GENTRAN         IN FIXED-FIXED.
064832,000211:                                                                                                  #        I +2    ADRES   L               STARTING ADRES OF DATA TO BE MOVED.
064833,000212:                                                                                                  #        I +3    ADRES   M               STARTING ADRES OF DUPLICATION BLOCK.
064834,000213:                                                                                                  #        I +4                            RETURNS HERE.
064835,000214: 
064836,000215:                                                                                                  #  GENTRAN TAKES 25 MCT'S (300 MICROSECONDS) PER ITEM + 5 MCT'S (60 MICS) FOR ENTERING AND EXITING.
064837,000216:                                                                                                  #  A, L AND ITEMP1 ARE NOT PRESERVED.
