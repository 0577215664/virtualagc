The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Colossus 249 (Apollo 9) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 055 OF AGC PROGRAM COMANCHE BY NASA 2021113-051

044776,000278:                                                                                                  #  WHEN AUTOMATIC MANEUVERS ARE BEING PERFORMED, THE FOLLOWING ANGLE ADDITION MUST BE MADE TO PROVIDE A SMOOTH
044777,000279:                                                                                                  #  SEQUENCE OF ANGULAR COMMANDS TO THE AUTOPILOT--
044778,000280: 
044779,000281:                                                                                                  #        CDUXD = CDUXD + DELCDUX         (DOUBLE PRECISION)
044780,000282:                                                                                                  #        CDUYD = CDUYD + DELCDUY         (DOUBLE PRECISION)
044781,000283:                                                                                                  #        CDUZD = CDUZD + DELCDUZ         (DOUBLE PRECISION)
044782,000284: 
044783,000285:                                                                                                  #  THE STEERING PROGRAMS:
044784,000286:                                                                                                  #        1) ATTITUDE MANEUVER ROUTINE
044785,000287:                                                                                                  #        2) LEM TRACKING
044786,000288: 
044787,000289:                                                                                                  #  SHOULD GENERATE THE DESIRED ANGLES (CDUXD, CDUYD, CDUZD) AS WELL AS THE INCREMENTAL ANGLES (DELCDUX, DELCDUY,
044788,000290:                                                                                                  #  DELCDUZ) SO THAT THE GIMBAL ANGLE COMMANDS CAN BE INTERPOLATED BETWEEN UPDATES.
044789,000291: 
044790,000292:                                                                                                  #  HOLDFLAG CODING:
044791,000293: 
044792,000294:                                                                                                  #        + = GRAB PRESENT CDU ANGLES AND STORE IN THETADX, THETADY, THETADZ
044793,000295:                                                                                                  #            AND PERFORM ATTITUDE HOLD ABOUT THESE ANGLES
044794,000296:                                                                                                  #            ALSO IGNORE AUTOMATIC STEERING
044795,000297:                                                                                                  #            SET = + BY
044796,000298:                                                                                                  #                1) INITIALIZATION PHASE OF AUTOPILOT
044797,000299:                                                                                                  #                2) OCCURANCE OF RHC COMMANDS
044798,000300:                                                                                                  #                3) FREE MODE
044799,000301:                                                                                                  #                4) SWITCH OVER TO ATTITUDE HOLD FROM AUTO
044800,000302:                                                                                                  #                   WHILE DOING AUTOMATIC STEERING (IN THIS CASE
044801,000303:                                                                                                  #                   HOLDFLAG IS NOT ACTUALLY SET TO +, BUT THE LOGIC
044802,000304:                                                                                                  #                   FUNCTIONS AS IF IT WERE.)
044803,000305:                                                                                                  #                5) S/C CONTROL SWITCH IS SCS
