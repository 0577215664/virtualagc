Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Colossus 249 (Apollo 9) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 055 OF AGC PROGRAM COMANCHE BY NASA 2021113-051
	10:28 APR.  1,1969
Note that the date is the date of the printout, not the date of the program revision.

060318,000034:                                                                                                  #  PROGRAM NAME -- SELF-CHECK                    LOG SECTION:  AGC BLOCK TWO SELF-CHECK
060319,000035:                                                                                                  #  MOD NO -- 1                                   ASSEMBLY SUBROUTINE UTILITYM REV 25
060320,000036:                                                                                                  #  MOD BY -- GAUNTT
060321,000037: 
060322,000038:                                                                                                  #  FUNCTIONAL DESCRIPTION
060323,000039:                                                                                                  #        PROGRAM HAS TWO MAIN PARTS.  THE FIRST IS SELF-CHECK WHICH RUNS AS A ZERO PRIORITY JOB WITH NO CORE SET, AS
060324,000040:                                                                                                  #        PART OF THE BACK-UP IDLE LOOP.  THE SECOND IS SHOW-BANKSUM WHICH RUNS AS A REGULAR EXECUTIVE JOB WITH ITS OWN
060325,000041:                                                                                                  #        STARTING VERB.
060326,000042: 
060327,000043:                                                                                                  #        THE PURPOSE OF SELF-CHECK IS TO CHECK OUT VARIOUS PARTS OF THE COMPUTER AS OUTLINED BELOW IN THE OPTIONS.
060328,000044: 
060329,000045:                                                                                                  #        THE PURPOSE OF SHOW-BANKSUM IS TO DISPLAY THE SUM OF EACH BANK, ONE AT A TIME.
060330,000046: 
060331,000047:                                                                                                  #        IN ALL THERE ARE 7 POSSIBLE OPTIONS IN THIS BLOCK II VERSION OF SELF-CHECK.  MORE DETAIL DESCRIPTION MAY BE
060332,000048:                                                                                                  #        FOUND IN E-2065 BLOCK II AGC SELF-CHECK AND SHOW BANKSUM BY EDWIN D. SMALLY DECEMBER 1966, AND ADDENDA 2 AND 3.
060333,000049: 
060334,000050:                                                                                                  #        THE DIFFERENT OPTIONS ARE CONTROLLED BY PUTTING DIFFERENT NUMBERS IN THE SMODE REGISTER (NOUN 27).  BELOW IS
060335,000051:                                                                                                  #        A DESCRIPTION OF WHAT PARTS OF THE COMPUTER THAT ARE CHECKED BY THE OPTIONS, AND THE CORRESPONDING NUMBER, IN
060336,000052:                                                                                                  #        OCTAL, TO LOAD INTO SMODE.
060337,000053:                                                                                                  #                +-4             ERASABLE MEMORY
060338,000054:                                                                                                  #                +-5             FIXED MEMORY
060339,000055:                                                                                                  #                +-1,2,3,6,7,10  EVERYTHING IN OPTIONS 4 AND 5.
060340,000056:                                                                                                  #                -0              SAME AS +-10 UNTIL AN ERROR IS DETECTED.
060341,000057:                                                                                                  #                +0              NO CHECK, PUTS COMPUTER INTO THE BACKUP IDLE LOOP.
060342,000058: 
060343,000059:                                                                                                  #  WARNINGS
060344,000060:                                                                                                  #        USE OF E MEMORY RESERVED FOR SELF-CHECK (EVEN IN IDLE LOOP) AS TEMP STORAGE BY OTHER PROGRAMS IS DANGEROUS.
060345,000061:                                                                                                  #        SMODE SET GREATER THAN OCT 10 PUTS COMPUTER INTO BACKUP IDLE LOOP.
060346,000062: 
060347,000063:                                                                                                  #  CALLING SEQUENCE
060348,000064:                                                                                                  #        TO CALL SELF-CHECK KEY IN
060349,000065:                                                                                                  #                V 21 N 27 E     OPTION NUMBER E
060350,000066:                                                                                                  #        TO CALL SHOW-BANKSUM KEY IN
060351,000067:                                                                                                  #                V 91 E          DISPLAYS FIRST BANK
060352,000068:                                                                                                  #                V 33 E          PROCEED, DISPLAYS NEXT BANK
060353,000069: 
060354,000070:                                                                                                  #  EXIT MODES, NORMAL AND ALARM
060355,000071:                                                                                                  #        SELF-CHECK NORMALLY CONTINUES INDEFINITELY UNLESS THERE IS AN ERROR DETECTED.  IF SO + OPTION NUMBERS PUT
060356,000072:                                                                                                  #        COMPUTER INTO BACKUP IDLE LOOP, - OPTION NUMBERS RESTART THE OPTION.
060357,000073: 
060358,000074:                                                                                                  #        THE -0 OPTION PROCEEDS FROM THE LINE FOLLOWING THE LINE WHERE THE ERROR WAS DETECTED.
060359,000075: 
060360,000076:                                                                                                  #        SHOW-BANKSUM PROCEEDS UNTIL A TERMINATE IS KEYED IN (V 34 E).  THE COMPUTER IS PUT INTO THE BACKUP IDLE LOOP.
060361,000077: 
060362,000078:                                                                                                  #  OUTPUT

Page 1395 

060364,000080:                                                                                                  #        SELF-CHECK UPON DETECTING AN ERROR LOADS THE SELF-CHECK ALARM CONSTANT (01102) INTO THE FAILREG SET AND
060365,000081:                                                                                                  #        TURNS ON THE ALARM LIGHT.  THE OPERATOR MAY THEN DISPLAY THE THREE FAILREGS BY KEYING IN V 05 N 09 E.  FOR FURTHER
060366,000082:                                                                                                  #        INFORMATION HE MAY KEY IN V 05 N 08 E, THE DSKY DISPLAY IN R1 WILL BE ADDRESS+1 OF WHERE THE ERROR WAS DETECTED,
060367,000083:                                                                                                  #        IN R2 THE BBCON OF SELF-CHECK, AND IN R3 THE TOTAL NUMBER OF ERRORS DETECTED BY SELF-CHECK SINCE THE LAST MAN
060368,000084:                                                                                                  #        INITIATED FRESH START (SLAP1).
060369,000085: 
060370,000086:                                                                                                  #        SHOW-BANKSUM STARTING WTIH BANK 0 DISPLAYS IN R1 THE BANK SUM (A +-NUMBER EQUAL TO THE BANK NUMBER), IN R2
060371,000087:                                                                                                  #        THE BANK NUMBER, AND IN R3 THE BUGGER WORD.
060372,000088: 
060373,000089:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED
060374,000090:                                                                                                  #        ACCOMPLISHED BY FRESH START
060375,000091:                                                                                                  #                SMODE SET TO +0
060376,000092: 
060377,000093:                                                                                                  #  DEBRIS
060378,000094:                                                                                                  #        ALL EXITS FROM THE CHECK OF ERASABLE (ERASCHK) RESTORE ORIGINAL CONTENTS TO REGISTERS UNDER CHECK.
060379,000095:                                                                                                  #        EXCEPTION IS A RESTART.  RESTART THAT OCCURS DURING ERASCHK RESTORES ERASABLE, UNLESS THERE IS EVIDENCE TO DOUBT
060380,000096:                                                                                                  #        E MEMORY, IN WHICH CASE PROGRAM THEN DOES A FRESH START (DOFSTART).
