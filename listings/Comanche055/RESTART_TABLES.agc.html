Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Colossus 249 (Apollo 9) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 055 OF AGC PROGRAM COMANCHE BY NASA 2021113-051
	10:28 APR.  1,1969
Note that the date is the date of the printout, not the date of the program revision.

008933,000037:                                                                                                  #  --------------
008934,000038: 
008935,000039:                                                                                                  #  THERE ARE TWO FORMS OF RESTART TABLES FOR EACH GROUP.  THEY ARE KNOWN AS THE EVEN RESTART TABLES AND THE ODD
008936,000040:                                                                                                  #  RESTART TABLES.  THE ODD TABLES HAVE ONLY ONE ENTRY OF THREE LOCATIONS WHILE THE EVEN TABLES HAVE TWO ENTRIES
008937,000041:                                                                                                  #  EACH USING THREE LOCATIONS.  THE INFORMATION AS TO WHETHER IT IS A JOB, WAITLIST, OR A LONGCALL IS GIVEN BY THE
008938,000042:                                                                                                  #  WAY THINGS ARE PUT IN TO THE TABLES.
008939,000043: 
008940,000044:                                                                                                  #  A JOB HAS ITS PRIORITY STORED IN A PRDTTAB OF THE CORRECT PHASE SPOT -- A POSITIVE PRIORITY INDICATES A
008941,000045:                                                                                                  #  FINDVAC JOB, A NEGATIVE PRIORITY A NOVAC.  THE 2CADR OF THE JOB IS STORED IN THE CADRTAB.
008942,000046:                                                                                                  #  FOR EXAMPLE,
008943,000047: 
008944,000048:                                                                                                  #                5.7SPOT         OCT     23000
008945,000049:                                                                                                  #                                2CADR   SOMEJOB
008946,000050: 
008947,000051:                                                                                                  #  A RESTART OF GROUP 5 WITH PHASE SEVEN WOULD THEN CAUSE SOMEJOB TO BE RESTARTED AS A FINDVAC WITH PRIORITY 23.
008948,000052: 
008949,000053:                                                                                                  #                5.5SPOT         OCT     -23000
008950,000054:                                                                                                  #                                2CADR   ANYJOB
008951,000055: 
008952,000056:                                                                                                  #  HERE A RESTART OF GROUP 5 WITH PHASE 7 WOULD CAUSE ANYJOB TO BE RESTARTED AS A NOVAC WITH PRIORITY 23.
008953,000057:                                                                                                  #  A LONGCALL HAS ITS GENADR OF ITS 2CADR STORED NEGATIVELY AND ITS BBCON STORED POSITIVELY.  IN ITS PRDTTAB IS
008954,000058:                                                                                                  #  PLACED THE LOCATION OF A DP REGISTER THAT CONTAINS THE DELTA TIME THAT LONGCALL HAD BEEN ORIGINALLY STARTED
008955,000059:                                                                                                  #  WITH.  EXAMPLE,
008956,000060: 
008957,000061:                                                                                                  #                3.6SPOT         GENADR  DELTAT
008958,000062:                                                                                                  #                                -GENADR LONGTASK
008959,000063:                                                                                                  #                                BBCON   LONGTASK
008960,000064: 
008961,000065:                                                                                                  #                                OCT     31000
008962,000066:                                                                                                  #                                2CADR   JOBAGAIN
008963,000067: 
008964,000068:                                                                                                  #  THIS WOULD START UP LONGTASK AT THE APPROPRIATE TIME, OR IMMEDIATELY IF THE TIME HAD ALREADY PASSED.  IT SHOULD
008965,000069:                                                                                                  #  BE NOTED THAT IF DELTAT IS IN A SWITCHED E BANK, THIS INFORMATION SHOULD BE IN THE BBCON OF THE 2CADR OF THE
008966,000070:                                                                                                  #  TASK.  FROM ABOVE, WE SEE THAT THE SECOND PART OF THIS PHASE WOULD BE STARTED AS A JOB WITH A PRIORITY OF 31.
008967,000071: 
008968,000072:                                                                                                  #  WAITLIST CALLS ARE IDENTIFIED BY THE FACT THAT THEIR 2CADR IS STORED NEGATIVELY.  IF PRDTTAB OF THE PHASE SPOT
008969,000073:                                                                                                  #  IS POSITIVE, THEN IT CONTAINS THE DELTA TIME, IF PRDTTAB IS NEGATIVE THEN IT IS THE -GENADR OF AN ERASABLE
008970,000074:                                                                                                  #  LOCATION CONTAINING THE DELTA TIME, THAT IS, THE TIME IS STORED INDIRECTLY. IT SHOULD BE NOTED AS ABOVE, THAT
008971,000075:                                                                                                  #  IF THE TIME IS STORED INDIRECTLY, THE BBCON MUST CONTAIN THE NECESSARY E BANK INFORMATION IF APPLICABLE.  WITH
008972,000076:                                                                                                  #  WAITLIST WE HAVE ONE FURTHER OPTION, IF -0 IS STORED IN PRDTTAB, IT WILL CAUSE AN IMMEDIATE RESTART OF THE
008973,000077:                                                                                                  #  TASK.  EXAMPLES,
008974,000078: 
008975,000079:                                                                                                  #                                OCT     77777           # THIS WILL CAUSE AN IMMEDIATE RESTART
008976,000080:                                                                                                  #                                -2CADR  ATASK           # OF THE TASK "ATASK"
008977,000081: 
008978,000082:                                                                                                  #                                DEC     200             # IF THE TIME OF THE 2 SECONDS SINCE DUMMY
008979,000083:                                                                                                  #                                -2CADR  DUMMY           # WAS PUT ON THE WAITLIST IS UP, IT WILL BEGIN
008980,000084:                                                                                                  #                                                        # IN 10 MS, OTHERWISE IT WILL BEGIN WHEN
008981,000085:                                                                                                  #                                                        # IT NORMALLY WOULD HAVE BEGUN.
008982,000086:                                                                                                  #  Page 212
008983,000087:                                                                                                  #                                -GENADR DTIME           # WHERE DTIME CONTAINS THE DELTA TIME
008984,000088:                                                                                                  #                                -2CADR  TASKTASK        # OTHERWISE THIS IS AS ABOVE
