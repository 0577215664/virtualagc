Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Colossus 249 (Apollo 9) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 055 OF AGC PROGRAM COMANCHE BY NASA 2021113-051
	10:28 APR.  1,1969
Note that the date is the date of the printout, not the date of the program revision.

008991,000040:                                                                                                  #  RESTART TABLES
008992,000041:                                                                                                  #  --------------
008993,000042: 
008994,000043:                                                                                                  #  THERE ARE TWO FORMS OF RESTART TABLES FOR EACH GROUP.  THEY ARE KNOWN AS THE EVEN RESTART TABLES AND THE ODD
008995,000044:                                                                                                  #  RESTART TABLES.  THE ODD TABLES HAVE ONLY ONE ENTRY OF THREE LOCATIONS WHILE THE EVEN TABLES HAVE TWO ENTRIES
008996,000045:                                                                                                  #  EACH USING THREE LOCATIONS.  THE INFORMATION AS TO WHETHER IT IS A JOB, WAITLIST, OR A LONGCALL IS GIVEN BY THE
008997,000046:                                                                                                  #  WAY THINGS ARE PUT IN TO THE TABLES.
008998,000047: 
008999,000048:                                                                                                  #  A JOB HAS ITS PRIORITY STORED IN PRDTTAB OF THE CORRECT PHASE SPOT - A POSITIVE PRIORITY INDICATES A
009000,000049:                                                                                                  #  FINDVAC JOB, A NEGATIVE PRIORITY A NOVAC.  THE 2CADR OF THE JOB IS STORED IN THE CADRTAB.
009001,000050:                                                                                                  #  FOR EXAMPLE,
009002,000051: 
009003,000052:                                                                                                  #                5.7SPOT         OCT     23000
009004,000053:                                                                                                  #                                2CADR   SOMEJOB
009005,000054: 
009006,000055:                                                                                                  #  A RESTART OF GROUP 5 WITH PHASE SEVEN WOULD THEN CAUSE SOMEJOB TO BE RESTARTED AS A FINDVAC WITH PRIORITY 23.
009007,000056: 
009008,000057:                                                                                                  #                5.5SPOT         OCT     -23000
009009,000058:                                                                                                  #                                2CADR   ANYJOB
009010,000059: 
009011,000060:                                                                                                  #  HERE A RESTART OF GROUP 5 WITH PHASE 7 WOULD CAUSE ANYJOB TO BE RESTARTED AS A NOVAC WITH PRIORITY 23.
009012,000061:                                                                                                  #  A LONGCALL HAS ITS GENADR OF ITS 2CADR STORED NEGATIVELY AND ITS BBCON STORED POSITIVELY.  IN ITS PRDTTAB IS
009013,000062:                                                                                                  #  PLACED THE LOCATION OF A DP REGISTER THAT CONTAINS THE DELTA TIME THAT LONGCALL HAD BEEN ORIGINALLY STARTED
009014,000063:                                                                                                  #  WITH.  EXAMPLE,
009015,000064: 
009016,000065:                                                                                                  #                3.6SPOT         GENADR  DELTAT
009017,000066:                                                                                                  #                                -GENADR LONGTASK
009018,000067:                                                                                                  #                                BBCON   LONGTASK
009019,000068: 
009020,000069:                                                                                                  #                                OCT     31000
009021,000070:                                                                                                  #                                2CADR   JOBAGAIN
009022,000071: 
009023,000072:                                                                                                  #  THIS WOULD START UP LONGTASK AT THE APPROPRIATE TIME, OR IMMEDIATELY IF THE TIME HAD ALREADY PASSED.  IT SHOULD
009024,000073:                                                                                                  #  BE NOTED THAT IF DELTAT IS IN A SWITCHED E BANK, THIS INFORMATOIN SHOULD BE IN THE BBCON OF THE 2CADR OF THE
009025,000074:                                                                                                  #  TASK.  FROM ABOVE, WE SEE THAT THE SECOND PART OF THIS PHASE WOULD BE STARTED AS A JOB WITH A PRIORITY OF 31.
009026,000075: 
009027,000076:                                                                                                  #  WAITLIST CALLS ARE IDENTIFIED BY THE FACT THAT THEIR 2CADR IS STORED NEGATIVELY.  IF PRDTTAB OF THE PHASE SPOT
009028,000077:                                                                                                  #  IS POSITIVE, THEN IT CONTAINS THE DELTA TIME, IF PRDTTAB IS NEGATIVE THEN IT IS THE -GENADR OF AN ERASABLE
009029,000078:                                                                                                  #  LOCATION CONTAINING THE DELTA TIME, THAT IS, THE TIME IS STORED INDIRECTLY. IT SHOULD BE NOTED AS ABOVE, THAT
009030,000079:                                                                                                  #  IF THE TIME IS STORED INDIRECTLY, THE BBCON MUST CONTAIN THE NECESSARY E BANK INFORMATION IF APPLICABLE.  WITH
009031,000080:                                                                                                  #  WAITLIST WE HAVE ONE FURTHER OPTION, IF -0 IS STORED IN PRDTTAB, IT WILL CAUSE AN IMMEDIATE RESTART OF THE
009032,000081:                                                                                                  #  TASK.  EXAMPLES,
009033,000082: 
009034,000083:                                                                                                  #                                OCT     77777           THIS WILL CAUSE AN IMMEDIATE RESTART
009035,000084:                                                                                                  #                                -2CADR  ATASK           OF THE TASK :ATASK:
009036,000085: 
009037,000086:                                                                                                  #                                DEC     200             IF THE TIME OF THE 2 SECONDS SINCE DUMMY
009038,000087:                                                                                                  #                                -2CADR  DUMMY           WAS PUT ON WAITLIST IS UP, IT WILL BEGIN
009039,000088:                                                                                                  #                                                        IN 10 MS, OTHERWISE IT WILL BEGIN WHEN
009040,000089:                                                                                                  #                                                        IT NORMALLY WOULD HAVE BEGUN.
