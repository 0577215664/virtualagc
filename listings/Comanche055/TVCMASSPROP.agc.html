Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Colossus 249 (Apollo 9) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 055 OF AGC PROGRAM COMANCHE BY NASA 2021113-051
	10:28 APR.  1,1969
Note that the date is the date of the printout, not the date of the program revision.

042468,000036:                                                                                                  #  PROGRAM NAME....MASSPROP
042469,000037:                                                                                                  #  LOG SECTION....TVCMASSPROP            PROGRAMMER...MELANSON (ENGEL, SCHLUNDT)
042470,000038: 
042471,000039:                                                                                                  #  FUNCTIONAL DESCRIPTION:
042472,000040: 
042473,000041:                                                                                                  #        MASSPROP OPERATES IN TWO MODES: (1) IF LEM MASS OR CONFIGURATION ARE UPDATED (MASSPROP DOES NOT TEST
042474,000042:                                                                                                  #        FOR THIS) THE ENTIRE PROGRAM MUST BE RUN THROUGH, BREAKPOINT VALUES AND DERIVATIVES OF THE OUTPUTS WITH
042475,000043:                                                                                                  #        RESPECT TO CSM MASS BEING CALCULATED PRIOR TO CALCULATION OF THE OUTPUTS.  (2) OTHERWISE, THE OUTPUTS CAN BE
042476,000044:                                                                                                  #        CALCULATED USING PREVIOUSLY COMPUTED BREAKPOINT VALUES AND DERIVATIVES.
042477,000045: 
042478,000046:                                                                                                  #  CALLING SEQUENCES
042479,000047: 
042480,000048:                                                                                                  #        IF LEM MASS OR CONFIGURATION HAS BEEN UPDATED, TRANSFER TO MASSPROP, OTHERWISE TRANSFER TO FIXCW.
042481,000049:                                                                                                  #                L       TC      BANKCALL OR IBNKCALL
042482,000050:                                                                                                  #                L+1     CADR    MASSPROP
042483,000051:                                                                                                  #                        OR
042484,000052:                                                                                                  #                L+1     CADR    FIXCW
042485,000053:                                                                                                  #                L+2     RETURNS VIA Q
042486,000054: 
042487,000055:                                                                                                  #  CALLED   IN PARTICULAR BY DONOUN47 (JOB) AND TVCEXECUTIVE (TASK)
042488,000056: 
042489,000057:                                                                                                  #  JOBS OR TASKS INITIATED - NONE
042490,000058: 
042491,000059:                                                                                                  #  SUBROUTINES CALLED - NONE
042492,000060: 
042493,000061:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED
042494,000062: 
042495,000063:                                                                                                  #        LEMMASS MUST CONTAIN LEM MASS SCALED AT B+16 IN KILOGRAMS
042496,000064:                                                                                                  #        CSMMASS MUST CONTAIN CSM MASS SCALED AT B+16 IN KILOGRAMS
042497,000065:                                                                                                  #        DAPDATR1 MUST BE SET TO INDICATE VEHICLE CONFIGURATION.
042498,000066:                                                                                                  #                BITS (15,14,13)  =  ( 0 , 0 , 1 )       LEM OFF
042499,000067:                                                                                                  #                                    ( 0 , 1 , 0 )       LEM ON (ASCNT,DSCNT)
042500,000068:                                                                                                  #                                    ( 1 , 1 , 0 )       LEM ON (ASCNT ONLY)
042501,000069: 
042502,000070:                                                                                                  #  ALARMS - NONE
042503,000071: 
042504,000072:                                                                                                  #  EXIT -   TC   Q
042505,000073: 
042506,000074:                                                                                                  #  OUTPUTS:
042507,000075: 
042508,000076:                                                                                                  #        (1)     IXX, SINGLE PRECISION SCALED AT B+20 IN KG-M SQ.
042509,000077:                                                                                                  #        (2)     IAVG, SINGLE PRECISION SCALED AT B+20 IN KG-M SQ.
042510,000078:                                                                                                  #        (3)     IAVG/TLX, SINGLE PRECISION, SCALED AT B+2 SEC-SQD
042511,000079:                                                                                                  #        
042512,000080:                                                                                                  #        THEY ARE STORED IN CONSECUTIVE REGISTERS IXX0, IXX1, IXX2
042513,000081:                                                                                                  #        CONVERSION FACTOR:  (SLUG-FTSQ) = 0.737562 (KG-MSQ)

042516,000084:                                                                                                  #  OUTPUTS ARE CALCULATED AS FOLLOWS:
042517,000085: 
042518,000086:                                                                                                  #        (1)     IF LEM DOCKED, LEMMASS IS FIRST ELIMINATED AS A PARAMETER
042519,000087: 
042520,000088:                                                                                                  #                VARST0 = INTVALUE0 + LEMMASS(SLOPEVAL0)         IXX             BREAKPOINT VALUE
042521,000089:                                                                                                  #                VARST1 = INTVALUE1 + LEMMASS(SLOPEVAL1)         IAVG            BREAKPOINT VALUE
042522,000090:                                                                                                  #                VARST2 = INTVALUE2 + LEMMASS(SLOPEVAL2)         IAVG/TLX        BREAKPOINT VALUE
042523,000091: 
042524,000092:                                                                                                  #                VARST3 = INTVALUE3 + LEMMASS(SLOPEVAL3)         IAVG/TLX        SLOPE FOR CSMMASS > 33956 LBS (SPS > 10000 LBS)
042525,000093:                                                                                                  #                VARST4 = INTVALUE4 + LEMMASS(SLOPEVAL4)         IAVG            SLOPE FOR CSMMASS > 33956 LBS (SPS > 10000 LBS)
042526,000094: 
042527,000095:                                                                                                  #                VARST5 = INTVALUE5 + LEMMASS(SLOPEVAL5)         IXX             SLOPE FOR ALL VALUES OF CSMMASS
042528,000096: 
042529,000097:                                                                                                  #                VARST6 = INTVALUE6 + LEMMASS(SLOPEVAL6)         IAVG            SLOPE FOR CSMMASS < 33956 LBS (SPS < 10000 LBS)
042530,000098:                                                                                                  #                VARST7 = INTVALUE7 + LEMMASS(SLOPEVAL7)         IAVG/TLX        SLOPE FOR CSMMASS < 33956 LBS (SPS < 10000 LBS)
042531,000099: 
042532,000100:                                                                                                  #                VARST8 = INTVALUE8 + LEMMASS(SLOPEVAL8)         IAVG            DECREMENT TO BRKPT VALUE WHEN LEM DSCNT STAGE OFF
042533,000101:                                                                                                  #                VARST9 = INTVALUE9 + LEMMASS(SLOPEVAL9)         IAVG/TLX        DECREMENT TO BRKPT VALUE WHEN LEM DSCNT STAGE OFF
042534,000102: 
042535,000103:                                                                                                  #        (2)     IF LEM NOT DOCKED
042536,000104: 
042537,000105:                                                                                                  #                VARST0 = NOLEMVAL0      WHERE THE MEANING AND SCALING OF VARST0
042538,000106:                                                                                                  #                   .          .         TO VARST9 ARE THE SAME AS GIVEN ABOVE
042539,000107:                                                                                                  #                   .          .
042540,000108:                                                                                                  #                   .          .         NOTE... FOR THIS CASE, VARST8,9 HAVE NO
042541,000109:                                                                                                  #                VARST9 = NOLEMVAL9      MEANING (THEY ARE COMPUTED BUT NOT USED)
042542,000110: 
042543,000111:                                                                                                  #        (3)     THE FINAL OUTPUT CALCULATIONS ARE THEN DONE
042544,000112: 
042545,000113:                                                                                                  #                IXX0 = VARST0 + (CSMMASS + NEGBPW)VARST5                IXX
042546,000114: 
042547,000115:                                                                                                  #                IXX1 = VARST1 + (CSMMASS + NEGBPW)VARST(4 OR 6)         IAVG
042548,000116: 
042549,000117:                                                                                                  #                IXX2 = VARST2 + (CSMMASS + NEGBPW)VARST(3 OR 7)         IAVG/TLX
042550,000118: 
042551,000119:                                                                                                  #        THE DATA USED CAME FROM CSM/LM SPACECRAFT OPERATIONAL DATA BOOK.
042552,000120:                                                                                                  #                VOL. 3, NASA DOCUMENT SNA-8-D-027 (MARCH 1968)
042553,000121: 
042554,000122:                                                                                                  #        PERTINENT MASS DATA:            CSM WEIGHT      (FULL)  64100 LBS
042555,000123:                                                                                                  #                                                        (EMPTY) 23956 LBS
042556,000124:                                                                                                  #                                        LEM WEIGHT      (FULL)  32000 LBS
042557,000125:                                                                                                  #                                                        (EMPTY) 14116 LBS
042558,000126: 
042559,000127:                                                                                                  #        (WEIGHTS ARE FROM AMENDMENT #1 (APRIL 24, 1968) TO ABOVE DATA BOOK)
