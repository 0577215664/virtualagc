Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Colossus 249 (Apollo 9) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 055 OF AGC PROGRAM COMANCHE BY NASA 2021113-051
	10:28 APR.  1,1969
Note that the date is the date of the printout, not the date of the program revision.

042473,000036:                                                                                                  #  PROGRAM NAME....MASSPROP
042474,000037:                                                                                                  #  LOG SECTION....TVCMASSPROP            PROGRAMMER...MELANSON (ENGEL, SCHLUNDT)
042475,000038: 
042476,000039:                                                                                                  #  FUNCTIONAL DESCRIPTION:
042477,000040: 
042478,000041:                                                                                                  #        MASSPROP OPERATES IN TWO MODES: (1) IF LEM MASS OR CONFIGURATION ARE UPDATED (MASSPROP DOES NOT TEST
042479,000042:                                                                                                  #        FOR THIS) THE ENTIRE PROGRAM MUST BE RUN THROUGH, BREAKPOINT VALUES AND DERIVATIVES OF THE OUTPUTS WITH
042480,000043:                                                                                                  #        RESPECT TO CSM MASS BEING CALCULATED PRIOR TO CALCULATION OF THE OUTPUTS.  (2) OTHERWISE, THE OUTPUTS CAN BE
042481,000044:                                                                                                  #        CALCULATED USING PREVIOUSLY COMPUTED BREAKPOINT VALUES AND DERIVATIVES.
042482,000045: 
042483,000046:                                                                                                  #  CALLING SEQUENCES
042484,000047: 
042485,000048:                                                                                                  #        IF LEM MASS OR CONFIGURATION HAS BEEN UPDATED, TRANSFER TO MASSPROP, OTHERWISE TRANSFER TO FIXCW.
042486,000049:                                                                                                  #                L       TC      BANKCALL OR IBNKCALL
042487,000050:                                                                                                  #                L+1     CADR    MASSPROP
042488,000051:                                                                                                  #                        OR
042489,000052:                                                                                                  #                L+1     CADR    FIXCW
042490,000053:                                                                                                  #                L+2     RETURNS VIA Q
042491,000054: 
042492,000055:                                                                                                  #  CALLED   IN PARTICULAR BY DONOUN47 (JOB) AND TVCEXECUTIVE (TASK)
042493,000056: 
042494,000057:                                                                                                  #  JOBS OR TASKS INITIATED - NONE
042495,000058: 
042496,000059:                                                                                                  #  SUBROUTINES CALLED - NONE
042497,000060: 
042498,000061:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED
042499,000062: 
042500,000063:                                                                                                  #        LEMMASS MUST CONTAIN LEM MASS SCALED AT B+16 IN KILOGRAMS
042501,000064:                                                                                                  #        CSMMASS MUST CONTAIN CSM MASS SCALED AT B+16 IN KILOGRAMS
042502,000065:                                                                                                  #        DAPDATR1 MUST BE SET TO INDICATE VEHICLE CONFIGURATION.
042503,000066:                                                                                                  #                BITS (15,14,13)  =  ( 0 , 0 , 1 )       LEM OFF
042504,000067:                                                                                                  #                                    ( 0 , 1 , 0 )       LEM ON (ASCNT,DSCNT)
042505,000068:                                                                                                  #                                    ( 1 , 1 , 0 )       LEM ON (ASCNT ONLY)
042506,000069: 
042507,000070:                                                                                                  #  ALARMS - NONE
042508,000071: 
042509,000072:                                                                                                  #  EXIT -   TC   Q
042510,000073: 
042511,000074:                                                                                                  #  OUTPUTS:
042512,000075: 
042513,000076:                                                                                                  #        (1)     IXX, SINGLE PRECISION SCALED AT B+20 IN KG-M SQ.
042514,000077:                                                                                                  #        (2)     IAVG, SINGLE PRECISION SCALED AT B+20 IN KG-M SQ.
042515,000078:                                                                                                  #        (3)     IAVG/TLX, SINGLE PRECISION, SCALED AT B+2 SEC-SQD
042516,000079:                                                                                                  #        
042517,000080:                                                                                                  #        THEY ARE STORED IN CONSECUTIVE REGISTERS IXX0, IXX1, IXX2
042518,000081:                                                                                                  #        CONVERSION FACTOR:  (SLUG-FTSQ) = 0.737562 (KG-MSQ)

042521,000084:                                                                                                  #  OUTPUTS ARE CALCULATED AS FOLLOWS:
042522,000085: 
042523,000086:                                                                                                  #        (1)     IF LEM DOCKED, LEMMASS IS FIRST ELIMINATED AS A PARAMETER
042524,000087: 
042525,000088:                                                                                                  #                VARST0 = INTVALUE0 + LEMMASS(SLOPEVAL0)         IXX             BREAKPOINT VALUE
042526,000089:                                                                                                  #                VARST1 = INTVALUE1 + LEMMASS(SLOPEVAL1)         IAVG            BREAKPOINT VALUE
042527,000090:                                                                                                  #                VARST2 = INTVALUE2 + LEMMASS(SLOPEVAL2)         IAVG/TLX        BREAKPOINT VALUE
042528,000091: 
042529,000092:                                                                                                  #                VARST3 = INTVALUE3 + LEMMASS(SLOPEVAL3)         IAVG/TLX        SLOPE FOR CSMMASS > 33956 LBS (SPS > 10000 LBS)
042530,000093:                                                                                                  #                VARST4 = INTVALUE4 + LEMMASS(SLOPEVAL4)         IAVG            SLOPE FOR CSMMASS > 33956 LBS (SPS > 10000 LBS)
042531,000094: 
042532,000095:                                                                                                  #                VARST5 = INTVALUE5 + LEMMASS(SLOPEVAL5)         IXX             SLOPE FOR ALL VALUES OF CSMMASS
042533,000096: 
042534,000097:                                                                                                  #                VARST6 = INTVALUE6 + LEMMASS(SLOPEVAL6)         IAVG            SLOPE FOR CSMMASS < 33956 LBS (SPS < 10000 LBS)
042535,000098:                                                                                                  #                VARST7 = INTVALUE7 + LEMMASS(SLOPEVAL7)         IAVG/TLX        SLOPE FOR CSMMASS < 33956 LBS (SPS < 10000 LBS)
042536,000099: 
042537,000100:                                                                                                  #                VARST8 = INTVALUE8 + LEMMASS(SLOPEVAL8)         IAVG            DECREMENT TO BRKPT VALUE WHEN LEM DSCNT STAGE OFF
042538,000101:                                                                                                  #                VARST9 = INTVALUE9 + LEMMASS(SLOPEVAL9)         IAVG/TLX        DECREMENT TO BRKPT VALUE WHEN LEM DSCNT STAGE OFF
042539,000102: 
042540,000103:                                                                                                  #        (2)     IF LEM NOT DOCKED
042541,000104: 
042542,000105:                                                                                                  #                VARST0 = NOLEMVAL0      WHERE THE MEANING AND SCALING OF VARST0
042543,000106:                                                                                                  #                   .          .         TO VARST9 ARE THE SAME AS GIVEN ABOVE
042544,000107:                                                                                                  #                   .          .
042545,000108:                                                                                                  #                   .          .         NOTE... FOR THIS CASE, VARST8,9 HAVE NO
042546,000109:                                                                                                  #                VARST9 = NOLEMVAL9      MEANING (THEY ARE COMPUTED BUT NOT USED)
042547,000110: 
042548,000111:                                                                                                  #        (3)     THE FINAL OUTPUT CALCULATIONS ARE THEN DONE
042549,000112: 
042550,000113:                                                                                                  #                IXX0 = VARST0 + (CSMMASS + NEGBPW)VARST5                IXX
042551,000114: 
042552,000115:                                                                                                  #                IXX1 = VARST1 + (CSMMASS + NEGBPW)VARST(4 OR 6)         IAVG
042553,000116: 
042554,000117:                                                                                                  #                IXX2 = VARST2 + (CSMMASS + NEGBPW)VARST(3 OR 7)         IAVG/TLX
042555,000118: 
042556,000119:                                                                                                  #        THE DATA USED CAME FROM CSM/LM SPACECRAFT OPERATIONAL DATA BOOK.
042557,000120:                                                                                                  #                VOL. 3, NASA DOCUMENT SNA-8-D-027 (MARCH 1968)
042558,000121: 
042559,000122:                                                                                                  #        PERTINENT MASS DATA:            CSM WEIGHT      (FULL)  64100 LBS
042560,000123:                                                                                                  #                                                        (EMPTY) 23956 LBS
042561,000124:                                                                                                  #                                        LEM WEIGHT      (FULL)  32000 LBS
042562,000125:                                                                                                  #                                                        (EMPTY) 14116 LBS
042563,000126: 
042564,000127:                                                                                                  #        (WEIGHTS ARE FROM AMENDMENT #1 (APRIL 24, 1968) TO ABOVE DATA BOOK)
