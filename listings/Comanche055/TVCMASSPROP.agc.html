Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Colossus 249 (Apollo 9) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 055 OF AGC PROGRAM COMANCHE BY NASA 2021113-051
	10:28 APR.  1,1969
Note that the date is the date of the printout, not the date of the program revision.

042256,000032:                                                                                                  #  PROGRAM NAME....MASSPROP
042257,000033:                                                                                                  #  LOG SECTION....TVCMASSPROP            PROGRAMMER...MELANSON (ENGEL, SCHLUNDT)
042258,000034: 
042259,000035:                                                                                                  #  FUNCTIONAL DESCRIPTION:
042260,000036: 
042261,000037:                                                                                                  #        MASSPROP OPERATES IN TWO MODES: (1) IF LEM MASS OR CONFIGURATION ARE UPDATED (MASSPROP DOES NOT TEST
042262,000038:                                                                                                  #        FOR THIS) THE ENTIRE PROGRAM MUST BE RUN THROUGH, BREAKPOINT VALUES AND DERIVATIVES OF THE OUTPUTS WITH
042263,000039:                                                                                                  #        RESPECT TO CSM MASS BEING CALCULATED PRIOR TO CALCULATION OF THE OUTPUTS.  (2) OTHERWISE, THE OUTPUTS CAN BE
042264,000040:                                                                                                  #        CALCULATED USING PREVIOUSLY COMPUTED BREAKPOINT VALUES AND DERIVATIVES.
042265,000041: 
042266,000042:                                                                                                  #  CALLING SEQUENCES
042267,000043: 
042268,000044:                                                                                                  #        IF LEM MASS OR CONFIGURATION HAS BEEN UPDATED, TRANSFER TO MASSPROP, OTHERWISE TRANSFER TO FIXCW.
042269,000045:                                                                                                  #                L       TC      BANKCALL or IBNKCALL
042270,000046:                                                                                                  #                L+1     CADR    MASSPROP
042271,000047:                                                                                                  #                        OR
042272,000048:                                                                                                  #                L+1     CADR    FIXCW
042273,000049:                                                                                                  #                L+2     RETURNS VIA Q
042274,000050: 
042275,000051:                                                                                                  #  CALLED:  IN PARTICULAR BY DONOUN47 (JOB) AND TVCEXECUTIVE (TASK)
042276,000052: 
042277,000053:                                                                                                  #  JOBS OR TASKS INITIATED:  NONE
042278,000054: 
042279,000055:                                                                                                  #  SUBROUTINES CALLED:  NONE
042280,000056: 
042281,000057:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED
042282,000058: 
042283,000059:                                                                                                  #        LEMMASS MUST CONTAIN LEM MASS SCALED AT B+16 KILOGRAMS
042284,000060:                                                                                                  #        CSMMASS MUST CONTAIN CSM MASS SCALED AT B+16 KILOGRAMS
042285,000061:                                                                                                  #        DAPDATR1 MUST BE SET TO INDICATE VEHICLE CONFIGURATION.
042286,000062:                                                                                                  #                BITS (15,14,13)  =  ( 0 , 0 , 1 )       LEM OFF
042287,000063:                                                                                                  #                                    ( 0 , 1 , 0 )       LEM ON (ASCNT,DSCNT)
042288,000064:                                                                                                  #                                    ( 1 , 1 , 0 )       LEM ON (ASCNT ONLY)
042289,000065: 
042290,000066:                                                                                                  #  ALARMS:  NONE
042291,000067: 
042292,000068:                                                                                                  #  EXIT:         TC      Q
042293,000069: 
042294,000070:                                                                                                  #  OUTPUTS:
042295,000071: 
042296,000072:                                                                                                  #        (1)     IXX, SINGLE PRECISION SCALED AT B+20 IN KG-M SQ.
042297,000073:                                                                                                  #        (2)     IAVG, SINGLE PRECISION SCALED AT B+20 IN KG-M SQ.
042298,000074:                                                                                                  #        (3)     IAVG/TLX, SINGLE PRECISION, SCALED AT B+2 SEC-SQD
042299,000075:                                                                                                  #        
042300,000076:                                                                                                  #        THEY ARE STORED IN CONSECUTIVE REGISTERS IXX0, IXX1, IXX2
042301,000077:                                                                                                  #        CONVERSION FACTOR:  (SLUG-FTSQ) = 0.737562 (KG-MSQ)

042303,000079: 
042304,000080:                                                                                                  #  OUTPUTS ARE CALCULATED AS FOLLOWS:
042305,000081: 
042306,000082:                                                                                                  #        (1)     IF LEM DOCKED, LEMMASS IS FIRST ELIMINATED AS A PARAMETER
042307,000083: 
042308,000084:                                                                                                  #                VARST0 = INTVALUE0 + LEMMASS(SLOPEVAL0)         IXX             BREAKPOINT VALUE
042309,000085:                                                                                                  #                VARST1 = INTVALUE1 + LEMMASS(SLOPEVAL1)         IAVG            BREAKPOINT VALUE
042310,000086:                                                                                                  #                VARST2 = INTVALUE2 + LEMMASS(SLOPEVAL2)         IAVG/TLX        BREAKPOINT VALUE
042311,000087: 
042312,000088:                                                                                                  #                VARST3 = INTVALUE3 + LEMMASS(SLOPEVAL3)         IAVG/TLX        SLOPE FOR CSMMASS > 33956 LBS (SPS > 10000 LBS)
042313,000089:                                                                                                  #                VARST4 = INTVALUE4 + LEMMASS(SLOPEVAL4)         IAVG            SLOPE FOR CSMMASS > 33956 LBS (SPS > 10000 LBS)
042314,000090: 
042315,000091:                                                                                                  #                VARST5 = INTVALUE5 + LEMMASS(SLOPEVAL5)         IXX             SLOPE FOR ALL VALUES OF CSMMASS
042316,000092: 
042317,000093:                                                                                                  #                VARST6 = INTVALUE6 + LEMMASS(SLOPEVAL6)         IAVG            SLOPE FOR CSMMASS < 33956 LBS (SPS < 10000 LBS)
042318,000094:                                                                                                  #                VARST7 = INTVALUE7 + LEMMASS(SLOPEVAL7)         IAVG/TLX        SLOPE FOR CSMMASS < 33956 LBS (SPS < 10000 LBS)
042319,000095: 
042320,000096:                                                                                                  #                VARST8 = INTVALUE8 + LEMMASS(SLOPEVAL8)         IAVG            DECREMENT TO BRKPT VALUE WHEN LEM DSCNT STAGE OFF
042321,000097:                                                                                                  #                VARST9 = INTVALUE9 + LEMMASS(SLOPEVAL9)         IAVG/TLX        DECREMENT TO BRKPT VALUE WHEN LEM DSCNT STAGE OFF
042322,000098: 
042323,000099:                                                                                                  #        (2)     IF LEM NOT DOCKED
042324,000100: 
042325,000101:                                                                                                  #                VARST0 = NOLEMVAL0      WHERE THE MEANING AND SCALING OF VARST0
042326,000102:                                                                                                  #                        .               TO VARST9 ARE THE SAME AS GIVEN ABOVE
042327,000103:                                                                                                  #                        .
042328,000104:                                                                                                  #                        .               NOTE... FOR THIS CASE, VARST8,9 HAVE NO
042329,000105:                                                                                                  #                VARST9 = NOLEMVAL9      MEANING (THEY ARE COMPUTED BUT NOT USED)
042330,000106: 
042331,000107:                                                                                                  #        (3)     THE FINAL OUTPUT CALCULATIONS ARE THEN DONE
042332,000108: 
042333,000109:                                                                                                  #                IXX0 = VARST0 + (CSMMASS + NEGBPW)VARST5                IXX
042334,000110: 
042335,000111:                                                                                                  #                IXX1 = VARST1 + (CSMMASS + NEGBPW)VARST(4 OR 6)         IAVG
042336,000112: 
042337,000113:                                                                                                  #                IXX2 = VARST2 + (CSMMASS + NEGBPW)VARST(3 OR 7)         IAVG/TLX
042338,000114: 
042339,000115:                                                                                                  #        THE DATA USED CAME FROM THE CSM/LM SPACECRAFT OPERATIONAL DATA BOOK
042340,000116:                                                                                                  #                VOL. 3, NASA DOCUMENT SNA-8-D-027 (MARCH 1968)
042341,000117: 
042342,000118:                                                                                                  #        PERTINENT MASS DATA:            CSM WEIGHT      (FULL)  64100 LBS.
042343,000119:                                                                                                  #                                                        (EMPTY) 23956 LBS.
042344,000120:                                                                                                  #                                        LEM WEIGHT      (FULL)  32000 LBS.
042345,000121:                                                                                                  #                                                        (EMPTY) 14116 LBS.
042346,000122: 
042347,000123:                                                                                                  #        (WEIGHTS ARE FROM AMMENDMENT #1 (APRIL 24, 1968) TO ABOVE DATA BOOK)
