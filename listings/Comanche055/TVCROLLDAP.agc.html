Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Colossus 249 (Apollo 9) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 055 OF AGC PROGRAM COMANCHE BY NASA 2021113-051
	10:28 APR.  1,1969
Note that the date is the date of the printout, not the date of the program revision.

044054,000041:                                                                                                  #  FUNCTIONAL DESCRIPTION....
044055,000042: 
044056,000043:                                                                                                  #       *AN ADAPTATION OF THE LEM P-AXIS CONTROLLER
044057,000044:                                                                                                  #       *MAINTAIN OGA WITHIN 5 DEG DEADBND OF OGAD, WHERE OGAD = OGA AS SEEN
044058,000045:                                                                                                  #        BY IGNITION (P40)
044059,000046:                                                                                                  #       *MAINTAIN OGA RATE LESS THAN 0.1 DEG/SEC LIMIT CYCLE RATE
044060,000047:                                                                                                  #       *SWITCHING LOGIC IN PHASE PLANE.... SEE GSOP CHAPTER 3
044061,000048:                                                                                                  #       *USES T6 CLOCK TO TIME JET FIRINGS
044062,000049:                                                                                                  #       *MAXIMUM JET FIRING TIME = 2.56 SECONDS, LIMITED TO 2.5 IF GREATER
044063,000050:                                                                                                  #       *MINIMUM JET FIRING TIME = 15 MS
044064,000051:                                                                                                  #       *JET PAIRS FIRE ALTERNATELY
044065,000052:                                                                                                  #       *AT LEAST 1/2 SECOND DELAY BEFORE A NEW JET PAIR IS FIRED
044066,000053:                                                                                                  #       *JET FIRINGS MAY NOT BE EXTENDED, ONLY SHORTENED, WHEN RE-EVALUATION
044067,000054:                                                                                                  #        OF A JET FIRING TIME IS MADE ON A LATER PASS
044068,000055: 
044069,000056:                                                                                                  #  CALLING SEQUENCE....
044070,000057: 
044071,000058:                                                                                                  #       *ROLLDAP CALL VIA WAITLIST, IN PARTICULAR BY TVCEXEC (EVERY 1/2 SEC)
044072,000059:                                                                                                  #        WITH A 3CS DELAY TO ALLOW FREE TIME FOR OTHER RUPTS (DWNRPT, ETC.)
044073,000060: 
044074,000061:                                                                                                  #  NORMAL EXIT MODES.... ENDOFJOB
044075,000062: 
044076,000063:                                                                                                  #  ALARM OR ABORT EXIT MODES.... NONE
044077,000064: 
044078,000065:                                                                                                  #  SUBROUTINES CALLED.....NONE
044079,000066: 
044080,000067:                                                                                                  #  OTHER INTERFACES....
044081,000068: 
044082,000069:                                                                                                  #       *TVCEXEC SETS UP ROLLDAP TASK EVERY 1/2 SECOND AND UPDATES 1/CONACC
044083,000070:                                                                                                  #        EVERY 10 SECONDS (VIA MASSPROP AND S40.15)
044084,000071:                                                                                                  #       *RESTARTS SUSPEND ROLL DAP COMPUTATIONS UNTIL THE NEXT 1/2 SEC
044085,000072:                                                                                                  #        SAMPLE PERIOD.  (THE PART OF TVCEXECUTIVE THAT CALLS ROLL DAP IS
044086,000073:                                                                                                  #        NOT RESTARTED.)  THE OGAD FROM IGNITION IS MAINTAINED.
044087,000074:

044111,000098:                                                                                                  #                                  *
044112,000099:                                                                                                  #   * * * * * * * * * * *          *
044113,000100:                                                                                                  #                                  *     (REGION 1, SEE TEXT BELOW)

044126,000113:                                                                                                  #                         *        *         *      (REGION 6-PRIME)
044127,000114:                                                                                                  #                                  *         *      (SEE TEXT BELOW)
044128,000115:                                                                                                  #                           *      *           *
044129,000116:                                                                                                  #                                  *             *     ...STRAIGHT LINE
044130,000117:                                                                                                  #     (FIRE POS ROLL JETS)     *   *               *  .
044131,000118:                                                                                                  #                                  *   (COAST)       *
044132,000119:                                                                                                  #                                  *                   * * * * * * * * * *
044133,000120:                                                                                                  #                                  *                         -MINLIM
044134,000121:                                                                                                  #                                  *    *
044135,000122:                                                                                                  #                                  *
044136,000123:                                                                                                  #                                  *          * * * * * * * * * * * * * * *
044137,000124:                                                                                                  #                                  *                        -MAXLIM
044138,000125:                                                                                                  #                                  *
044139,000126:                                                                                                  #                                  *
044140,000127: 
044141,000128:                                                                                                  #  SWITCHING PARABOLAS ARE CONTROL PARABOLAS, THUS REQUIRING KNOWLEDGE OF
044142,000129:                                                                                                  #        CONTROL ACCELERATION CONACC, OR ITS RECIPROCAL, 1/CONACC, THE TVC
044143,000130:                                                                                                  #        ROLL DAP GAIN  (SEE TVCEXECUTIVE VARIABLE GAIN PACKAGE).  JET
044144,000131:                                                                                                  #        FIRING TIME IS SIMPLY THAT REQUIRED TO ACHIEVE THE DESIRED OGARATE,
044145,000132:                                                                                                  #        SUBJECT TO THE LIMITATIONS DISCUSSED UNDER FUNCTIONAL DESCRIPTION,
044146,000133:                                                                                                  #        ABOVE.
044147,000134: 
044148,000135:                                                                                                  #  THE THREE CONTROL REGIONS  (+, -, AND ZERO TORQUE) ARE COMPRIZED OF
044149,000136:                                                                                                  #        TWELVE SUBSET REGIONS  ( 1...6, AND THE CORRESPONDING 1-PRIME...
044150,000137:                                                                                                  #        6-PRIME )  SEE SECTION 3 OF THE GSOP  (SUNDISK OR COLOSSUS)

044152,000139: 
044153,000140:                                                                                                  #  GIVEN THE OPERATING POINT NOT IN THE COAST REGION, THE DESIRED OGARATE
044154,000141:                                                                                                  #        IS AT THE POINT OF PENETRATION OF THE COAST REGION BY THE CONTROL
044155,000142:                                                                                                  #        PARABOLA WHICH PASSES THROUGH THE OPERATING POINT.  FOR REGION 3
044156,000143:                                                                                                  #        DESIRED OGARATE IS SIMPLY +-MAXLIM.  FOR REGIONS 1 OR 6 THE SOLUTION
044157,000144:                                                                                                  #        TO A QUADRATIC IS REQUIRED (THE PENETRATION IS ALONG THE STRAIGHT
044158,000145:                                                                                                  #        LINE OR MINLIM BOUNDRY SWITCH LINES).  AN APPROXIMATION IS MADE
044159,000146:                                                                                                  #        INSTEAD. TAKE AN OPERATING POINT IN REGION 6'.  PASS A TANGENT TO
044160,000147:                                                                                                  #        THE CONTROL PARABOLA THROUGH THE OPERATING POINT, AND FIND ITS
044161,000148:                                                                                                  #        INTERSECTION WITH THE STRAIGHT LINE SECTION OF THE SWITCH CURVE...
044162,000149:                                                                                                  #        THE INTERSECTION DEFINES DESIRED OGARATE.  IF THE OPERATING POINT IS
044163,000150:                                                                                                  #        CLOSE TO THE SWITCH LINE, THE APPROXIMATION IS QUITE GOOD (INDEED
044164,000151:                                                                                                  #        THE APPROXIMATE AND QUADRATIC SOLUTIONS CONVERGE IN THE LIMIT AS
044165,000152:                                                                                                  #        THE SWITCH LINE IS APPROACHED).  IF THE OPERATING POINT IS NOT CLOSE
044166,000153:                                                                                                  #        TO THE SWITCH LINE, THE APPROXIMATE SOLUTION GIVES VALID TREND
044167,000154:                                                                                                  #        INFORMATION (DIRECTION OF DESIRED OGARATE) AT LEAST.  THE 
044168,000155:                                                                                                  #        RE-EVALUATION OF DESIRED OGARATE IN SUBSEQUENT ROLL DAP PASSES (1/2
044169,000156:                                                                                                  #        SECOND INTERVALS) WILL BENEFIT FROM THE CONVERGENT NATURE OF THE
044170,000157:                                                                                                  #        APPROXIMATION.
044171,000158: 
044172,000159:                                                                                                  #  FOR LARGE OGAERROR THE TANGENT INTERSECTS +-MINLIM SWITCH BOUNDRY BEFORE
044173,000160:                                                                                                  #        INTERSECTING THE STRAIGHT LINE SWITCH.  HOWEVER THE MINLIM IS
044174,000161:                                                                                                  #        IGNORED IN COMPUTING THE FIRING TIME, SO THAT THE EXTENSION (INTO
044175,000162:                                                                                                  #        THE COAST REGION) OF THE STRAIGHT LINE SWITCH IS WHAT IS FIRED TO.
044176,000163:                                                                                                  #        IF THE ROLL DAP FINDS ITSELF IN THE COAST REGION BEFORE REACHING
044177,000164:                                                                                                  #        THE DESIRED INTERSECTION (IE, IN THE REGION BETWEEN THE MINLIM
044178,000165:                                                                                                  #        AND THE STRAIGHT LINE SWITCH) IT WILL EXIBIT NORMAL COAST-REGION
044179,000166:                                                                                                  #        BEHAVIOR AND TURN OFF THE JETS.  THE PURPOSE OF THIS FIRING POLICY
044180,000167:                                                                                                  #        IS TO MAINTAIN STATIC ROLL STABILITY IN THE EVENT OF A JET
044181,000168:                                                                                                  #        FAILED-ON.
044182,000169: 
044183,000170:                                                                                                  #  WHEN THE OPERATING POINT IS IN REGION 1 THE SAME APPROXIMATION IS
044184,000171:                                                                                                  #        MADE, BUT AT AN ARTIFICIALLY-CREATED OR DUMMY OPERATING POINT,
044185,000172:                                                                                                  #        DEFINED BY.. OGAERROR = INTERSECTION OF CONTROL PARABOLA AND
044186,000173:                                                                                                  #        OGAERROR AXIS, OGARATE = +-LMCRATE WHERE SIGN IS OPPOSITE THAT OF
044187,000174:                                                                                                  #        REAL OPERATING POINT RATE.  WHEN THE OPERATING POINT HAS PASSED
044188,000175:                                                                                                  #        FROM REGION 1 TO REGION 6', THE DUMMY POINT IS NO LONGER REQUIRED,
044189,000176:                                                                                                  #        AND THE SOLUTION REVERTS TO THAT OF A REGULAR REGION 6' POINT.
044190,000177: 
044191,000178:                                                                                                  #  EQUATION FOR SWITCHING PARABOLA (SEE FIGURE ABOVE)....
044192,000179:                                                                                                  #                                    2
044193,000180:                                                                                                  #        SOGAERROR = (DB - (SOGARATE) (1/CONACC)/2) SGN(SOGARATE)
044194,000181: 
044195,000182:                                                                                                  #  EQUATION FOR SWITCHING STRAIGHT LINE SEGMENT....
044196,000183: 
044197,000184:                                                                                                  #        SOGARATE = -(-SLOPE)(SOGAERROR) - SGN(SOGARATE) INTERCEP
044198,000185: 
044199,000186:                                                                                                  #                WHERE  INTERCEP = DB(-SLOPE) - LMCRATE

044203,000190: 
044204,000191:                                                                                                  #        DOGADOT = NUM/DEN, WHERE
044205,000192:                                                                                                  #                                       2
