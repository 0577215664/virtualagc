Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Colossus 249 (Apollo 9) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 055 OF AGC PROGRAM COMANCHE BY NASA 2021113-051
	10:28 APR.  1,1969
Note that the date is the date of the printout, not the date of the program revision.

043822,000037:                                                                                                  #  FUNCTIONAL DESCRIPTION....
043823,000038: 
043824,000039:                                                                                                  #       *AN ADAPTATION OF THE LEM P-AXIS CONTROLLER
043825,000040:                                                                                                  #       *MAINTAIN OGA WITHIN 5 DEG DEADBND OF OGAD, WHERE OGAD = OGA AS SEEN
043826,000041:                                                                                                  #        BY IGNOVER (P40)
043827,000042:                                                                                                  #       *MAINTAIN OGA RATE LESS THAN 0.1 DEG/SEC LIMIT CYCLE RATE
043828,000043:                                                                                                  #       *SWITCHING LOGIC IN PHASE PLANE.... SEE GSOP CHAPTER 3
043829,000044:                                                                                                  #       *USES T6 CLOCK TO TIME JET FIRINGS.
043830,000045:                                                                                                  #       *MAXIMUM JET FIRING TIME = 2.56 SECONDS, LIMITED TO 2.5 IF GREATER
043831,000046:                                                                                                  #       *MINIMUM JET FIRING TIME = 15 MS
043832,000047:                                                                                                  #       *JET PAIRS FIRE ALTERNATELY
043833,000048:                                                                                                  #       *AT LEAST 1/2 SECOND DELAY BEFORE A NEW JET PAIR IS FIRED
043834,000049:                                                                                                  #       *JET FIRINGS MAY NOT BE EXTENDED, ONLY SHORTENED, WHEN RE-EVALUATION
043835,000050:                                                                                                  #        OF A JET FIRING TIME IS MADE ON A LATER PASS
043836,000051: 
043837,000052:                                                                                                  #  CALLING SEQUENCE....
043838,000053: 
043839,000054:                                                                                                  #       *ROLLDAP CALL VIA WAITLIST, IN PARTICULAR BY TVCEXEC (EVERY 1/2 SEC)
043840,000055:                                                                                                  #        WITH A 3CS DELAY TO ALLOW FREE TIME FOR OTHER RUPTS (DWNRPT, ETC.)
043841,000056: 
043842,000057:                                                                                                  #  NORMAL EXIT MODES.... ENDOFJOB
043843,000058: 
043844,000059:                                                                                                  #  ALARM OR ABORT EXIT MODES.... NONE
043845,000060: 
043846,000061:                                                                                                  #  SUBROUTINES CALLED.....NONE
043847,000062: 
043848,000063:                                                                                                  #  OTHER INTERFACES....
043849,000064: 
043850,000065:                                                                                                  #       *TVCEXEC SETS UP ROLLDAP TASK EVERY 1/2 SECOND AND UPDATES 1/CONACC
043851,000066:                                                                                                  #        EVERY 10 SECONDS (VIA MASSPROP AND S40.15)
043852,000067:                                                                                                  #       *RESTARTS SUSPEND ROLL DAP COMPUTATIONS UNTIL THE NEXT 1/2 SEC
043853,000068:                                                                                                  #        SAMPLE PERIOD.  (THE PART OF TVCEXECUTIVE THAT CALLS ROLL DAP IS
043854,000069:                                                                                                  #        NOT RESTARTED.)  THE OGAD FROM IGNITION IS MAINTAINED.

043920,000135: 
043921,000136:                                                                                                  #  GIVEN THE OPERATING POINT NOT IN THE COAST REGION, THE DESIRED OGARATE
043922,000137:                                                                                                  #        IS AT THE POINT OF PENETRATION OF THE COAST REGION BY THE CONTROL
043923,000138:                                                                                                  #        PARABOLA WHICH PASSES THROUGH THE OPERATING POINT.  FOR REGION 3
043924,000139:                                                                                                  #        DESIRED OGARATE IS SIMPLY +-MAXLIM.  FOR REGIONS 1 OR 6 THE SOLUTION
043925,000140:                                                                                                  #        TO A QUADRATIC IS REQUIRED (THE PENETRATION IS ALONG THE STRAIGHT
043926,000141:                                                                                                  #        LINE OR MINLIM BOUNDRY SWITCH LINES).  AN APPROXIMATION IS MADE
043927,000142:                                                                                                  #        INSTEAD.  CONSIDER AN OPERATING POINT IN REGION 6'.  PASS A TANGENT TO
043928,000143:                                                                                                  #        THE CONTROL PARABOLA THROUGH THE OPERATING POINT, AND FIND ITS
043929,000144:                                                                                                  #        INTERSECTION WITH THE STRAIGHT LINE SECTION OF THE SWITCH CURVE...
043930,000145:                                                                                                  #        THE INTERSECTION DEFINES THE DESIRED OGARATE.  IF THE OPERATING POINT IS
043931,000146:                                                                                                  #        CLOSE TO THE SWITCH LINE, THE APPROXIMATION IS QUITE GOOD (INDEED
043932,000147:                                                                                                  #        THE APPROXIMATE AND QUADRATIC SOLUTIONS CONVERGE IN THE LIMIT AS
043933,000148:                                                                                                  #        THE SWITCH LINE IS APPROACHED).  IF THE OPERATING POINT IS NOT CLOSE
043934,000149:                                                                                                  #        TO THE SWITCH LINE, THE APPROXIMATE SOLUTION GIVES VALID TREND
043935,000150:                                                                                                  #        INFORMATION (DIRECTION OF DESIRED OGARATE) AT LEAST.  THE 
043936,000151:                                                                                                  #        RE-EVALUATION OF DESIRED OGARATE IN SUBSEQUENT ROLL DAP PASSES (1/2
043937,000152:                                                                                                  #        SECOND INTERVALS) WILL BENEFIT FROM THE CONVERGENT NATURE OF THE
043938,000153:                                                                                                  #        APPROXIMATION.
043939,000154: 
043940,000155:                                                                                                  #  FOR LARGE OGAERROR THE TANGENT INTERSECTS +-MINLIM SWITCH BOUNDRY BEFORE
043941,000156:                                                                                                  #        INTERSECTING THE STRAIGHT LINE SWITCH.  HOWEVER THE MINLIM IS
043942,000157:                                                                                                  #        IGNORED IN COMPUTING THE FIRING TIME, SO THAT THE EXTENSION (INTO
043943,000158:                                                                                                  #        THE COAST REGION) OF THE STRAIGHT LINE SWITCH IS WHAT IS FIRED TO.
043944,000159:                                                                                                  #        IF THE ROLL DAP FINDS ITSELF IN THE COAST REGION BEFORE REACHING
043945,000160:                                                                                                  #        THE DESIRED INTERSECTION (I.E., IN THE REGION BETWEEN THE MINLIM
043946,000161:                                                                                                  #        AND THE STRAIGHT LINE SWITCH) IT WILL EXHIBIT NORMAL COAST-REGION
043947,000162:                                                                                                  #        BEHAVIOR AND TURN OFF THE JETS.  THE PURPOSE OF THIS FIRING POLICY
043948,000163:                                                                                                  #        IS TO MAINTAIN STATIC ROLL STABILITY IN THE EVENT OF A JET
043949,000164:                                                                                                  #        FAILED-ON.
043950,000165: 
043951,000166:                                                                                                  #  WHEN THE OPERATING POINT IS IN REGION 1 THE SAME APPROXIMATION IS
043952,000167:                                                                                                  #        MADE, BUT AT AN ARTIFICIALLY-CREATED OR DUMMY OPERATING POINT,
043953,000168:                                                                                                  #        DEFINED BY:  OGAERROR = INTERSECTION OF CONTROL PARABOLA AND
043954,000169:                                                                                                  #        OGAERROR AXIS, OGARATE = +-LMCRATE WHERE SIGN IS OPPOSITE THAT OF
043955,000170:                                                                                                  #        REAL OPERATING POINT RATE.  WHEN THE OPERATING POINT HAS PASSED
043956,000171:                                                                                                  #        FROM REGION 1 TO REGION 6', THE DUMMY POINT IS NO LONGER REQUIRED,
043957,000172:                                                                                                  #        AND THE SOLUTION REVERTS TO THAT OF A REGULAR REGION 6' POINT.
043958,000173: 
043959,000174:                                                                                                  #  EQUATION FOR SWITCHING PARABOLA (SEE FIGURE ABOVE)....
043960,000175:                                                                                                  #                                    2
043961,000176:                                                                                                  #        SOGAERROR = (DB - (SOGARATE) (1/CONACC)/2) SGN(SOGARATE)
043962,000177: 
043963,000178:                                                                                                  #  EQUATION FOR SWITCHING STRAIGHT LINE SEGMENT....
043964,000179: 
043965,000180:                                                                                                  #        SOGARATE = -(-SLOPE)(SOGAERROR) - SGN(SOGARATE) INTERCEP
043966,000181: 
043967,000182:                                                                                                  #                WHERE  INTERCEP = DB(-SLOPE) - LMCRATE

043971,000186: 
043972,000187:                                                                                                  #        DOGADOT = NUM/DEN, WHERE
043973,000188:                                                                                                  #                                       2
043974,000189:                                                                                                  #                NUM = (-SLOPE)(OGARATE) (1/CONACC)
