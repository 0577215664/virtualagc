Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Colossus 249 (Apollo 9) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 055 OF AGC PROGRAM COMANCHE BY NASA 2021113-051
	10:28 APR.  1,1969
Note that the date is the date of the printout, not the date of the program revision.

044051,000041:                                                                                                  #  FUNCTIONAL DESCRIPTION....
044052,000042: 
044053,000043:                                                                                                  #       *AN ADAPTATION OF THE LEM P-AXIS CONTROLLER
044054,000044:                                                                                                  #       *MAINTAIN OGA WITHIN 5 DEG DEADBND OF OGAD, WHERE OGAD = OGA AS SEEN
044055,000045:                                                                                                  #        BY IGNITION (P40)
044056,000046:                                                                                                  #       *MAINTAIN OGA RATE LESS THAN 0.1 DEG/SEC LIMIT CYCLE RATE
044057,000047:                                                                                                  #       *SWITCHING LOGIC IN PHASE PLANE.... SEE GSOP CHAPTER 3
044058,000048:                                                                                                  #       *USES T6 CLOCK TO TIME JET FIRINGS
044059,000049:                                                                                                  #       *MAXIMUM JET FIRING TIME = 2.56 SECONDS, LIMITED TO 2.5 IF GREATER
044060,000050:                                                                                                  #       *MINIMUM JET FIRING TIME = 15 MS
044061,000051:                                                                                                  #       *JET PAIRS FIRE ALTERNATELY
044062,000052:                                                                                                  #       *AT LEAST 1/2 SECOND DELAY BEFORE A NEW JET PAIR IS FIRED
044063,000053:                                                                                                  #       *JET FIRINGS MAY NOT BE EXTENDED, ONLY SHORTENED, WHEN RE-EVALUATION
044064,000054:                                                                                                  #        OF A JET FIRING TIME IS MADE ON A LATER PASS
044065,000055: 
044066,000056:                                                                                                  #  CALLING SEQUENCE....
044067,000057: 
044068,000058:                                                                                                  #       *ROLLDAP CALL VIA WAITLIST, IN PARTICULAR BY TVCEXEC (EVERY 1/2 SEC)
044069,000059:                                                                                                  #        WITH A 3CS DELAY TO ALLOW FREE TIME FOR OTHER RUPTS (DWNRPT, ETC.)
044070,000060: 
044071,000061:                                                                                                  #  NORMAL EXIT MODES.... ENDOFJOB
044072,000062: 
044073,000063:                                                                                                  #  ALARM OR ABORT EXIT MODES.... NONE
044074,000064: 
044075,000065:                                                                                                  #  SUBROUTINES CALLED.....NONE
044076,000066: 
044077,000067:                                                                                                  #  OTHER INTERFACES....
044078,000068: 
044079,000069:                                                                                                  #       *TVCEXEC SETS UP ROLLDAP TASK EVERY 1/2 SECOND AND UPDATES 1/CONACC
044080,000070:                                                                                                  #        EVERY 10 SECONDS (VIA MASSPROP AND S40.15)
044081,000071:                                                                                                  #       *RESTARTS SUSPEND ROLL DAP COMPUTATIONS UNTIL THE NEXT 1/2 SEC
044082,000072:                                                                                                  #        SAMPLE PERIOD.  (THE PART OF TVCEXECUTIVE THAT CALLS ROLL DAP IS
044083,000073:                                                                                                  #        NOT RESTARTED.)  THE OGAD FROM IGNITION IS MAINTAINED.
044084,000074: 
044085,000075:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED....

044087,000077:                                                                                                  #       *1/CONACC                                (S40.15)

044106,000096:                                                                                                  #                               OGARATE
044107,000097:                                                                                                  #                                  *
044108,000098:                                                                                                  #                                  *
044109,000099:                                                                                                  #   * * * * * * * * * * *          *
044110,000100:                                                                                                  #                                  *     (REGION 1, SEE TEXT BELOW)
044111,000101:                                                                                                  #                             *    *
044112,000102:                                                                                                  #                                  *

044114,000104:                                                                                                  #                *                 *    .
044115,000105:                                                                                                  #                  *               *   *
044116,000106:                                                                                                  #                    *             *                (FIRE NEG ROLL JETS)
044117,000107:                                                                                                  #                      *           *      *

044121,000111:                                                                                                  #  ************************************************************************
044122,000112:                                                                                                  #                                  *         *                (-AK, OGAERR)
044123,000113:                                                                                                  #                         *        *         *      (REGION 6-PRIME)
044124,000114:                                                                                                  #                                  *         *      (SEE TEXT BELOW)
044125,000115:                                                                                                  #                           *      *           *
044126,000116:                                                                                                  #                                  *             *     ...STRAIGHT LINE
044127,000117:                                                                                                  #     (FIRE POS ROLL JETS)     *   *               *  .
044128,000118:                                                                                                  #                                  *   (COAST)       *
044129,000119:                                                                                                  #                                  *                   * * * * * * * * * *
044130,000120:                                                                                                  #                                  *                         -MINLIM
044131,000121:                                                                                                  #                                  *    *
044132,000122:                                                                                                  #                                  *
044133,000123:                                                                                                  #                                  *          * * * * * * * * * * * * * * *
044134,000124:                                                                                                  #                                  *                        -MAXLIM
044135,000125:                                                                                                  #                                  *
044136,000126:                                                                                                  #                                  *
044137,000127: 
044138,000128:                                                                                                  #  SWITCHING PARABOLAS ARE CONTROL PARABOLAS, THUS REQUIRING KNOWLEDGE OF
044139,000129:                                                                                                  #        CONTROL ACCELERATION CONACC, OR ITS RECIPROCAL, 1/CONACC, THE TVC
044140,000130:                                                                                                  #        ROLL DAP GAIN  (SEE TVCEXECUTIVE VARIABLE GAIN PACKAGE).  JET
044141,000131:                                                                                                  #        FIRING TIME IS SIMPLY THAT REQUIRED TO ACHIEVE THE DESIRED OGARATE,
044142,000132:                                                                                                  #        SUBJECT TO THE LIMITATIONS DISCUSSED UNDER FUNCTIONAL DESCRIPTION,
044143,000133:                                                                                                  #        ABOVE.
044144,000134: 
044145,000135:                                                                                                  #  THE THREE CONTROL REGIONS  (+, -, AND ZERO TORQUE) ARE COMPRIZED OF
044146,000136:                                                                                                  #        TWELVE SUBSET REGIONS  ( 1...6, AND THE CORRESPONDING 1-PRIME...
044147,000137:                                                                                                  #        6-PRIME )  SEE SECTION 3 OF THE GSOP  (SUNDISK OR COLOSSUS)

044149,000139: 
044150,000140:                                                                                                  #  GIVEN THE OPERATING POINT NOT IN THE COAST REGION, THE DESIRED OGARATE
044151,000141:                                                                                                  #        IS AT THE POINT OF PENETRATION OF THE COAST REGION BY THE CONTROL
044152,000142:                                                                                                  #        PARABOLA WHICH PASSES THROUGH THE OPERATING POINT.  FOR REGION 3
044153,000143:                                                                                                  #        DESIRED OGARATE IS SIMPLY +-MAXLIM.  FOR REGIONS 1 OR 6 THE SOLUTION
044154,000144:                                                                                                  #        TO A QUADRATIC IS REQUIRED (THE PENETRATION IS ALONG THE STRAIGHT
044155,000145:                                                                                                  #        LINE OR MINLIM BOUNDRY SWITCH LINES).  AN APPROXIMATION IS MADE
044156,000146:                                                                                                  #        INSTEAD. TAKE AN OPERATING POINT IN REGION 6'.  PASS A TANGENT TO
044157,000147:                                                                                                  #        THE CONTROL PARABOLA THROUGH THE OPERATING POINT, AND FIND ITS
044158,000148:                                                                                                  #        INTERSECTION WITH THE STRAIGHT LINE SECTION OF THE SWITCH CURVE...
044159,000149:                                                                                                  #        THE INTERSECTION DEFINES DESIRED OGARATE.  IF THE OPERATING POINT IS
044160,000150:                                                                                                  #        CLOSE TO THE SWITCH LINE, THE APPROXIMATION IS QUITE GOOD (INDEED
044161,000151:                                                                                                  #        THE APPROXIMATE AND QUADRATIC SOLUTIONS CONVERGE IN THE LIMIT AS
044162,000152:                                                                                                  #        THE SWITCH LINE IS APPROACHED).  IF THE OPERATING POINT IS NOT CLOSE
044163,000153:                                                                                                  #        TO THE SWITCH LINE, THE APPROXIMATE SOLUTION GIVES VALID TREND
044164,000154:                                                                                                  #        INFORMATION (DIRECTION OF DESIRED OGARATE) AT LEAST.  THE 
044165,000155:                                                                                                  #        RE-EVALUATION OF DESIRED OGARATE IN SUBSEQUENT ROLL DAP PASSES (1/2
044166,000156:                                                                                                  #        SECOND INTERVALS) WILL BENEFIT FROM THE CONVERGENT NATURE OF THE
044167,000157:                                                                                                  #        APPROXIMATION.
044168,000158: 
044169,000159:                                                                                                  #  FOR LARGE OGAERROR THE TANGENT INTERSECTS +-MINLIM SWITCH BOUNDRY BEFORE
044170,000160:                                                                                                  #        INTERSECTING THE STRAIGHT LINE SWITCH.  HOWEVER THE MINLIM IS
044171,000161:                                                                                                  #        IGNORED IN COMPUTING THE FIRING TIME, SO THAT THE EXTENSION (INTO
044172,000162:                                                                                                  #        THE COAST REGION) OF THE STRAIGHT LINE SWITCH IS WHAT IS FIRED TO.
044173,000163:                                                                                                  #        IF THE ROLL DAP FINDS ITSELF IN THE COAST REGION BEFORE REACHING
044174,000164:                                                                                                  #        THE DESIRED INTERSECTION (IE, IN THE REGION BETWEEN THE MINLIM
044175,000165:                                                                                                  #        AND THE STRAIGHT LINE SWITCH) IT WILL EXIBIT NORMAL COAST-REGION
044176,000166:                                                                                                  #        BEHAVIOR AND TURN OFF THE JETS.  THE PURPOSE OF THIS FIRING POLICY
044177,000167:                                                                                                  #        IS TO MAINTAIN STATIC ROLL STABILITY IN THE EVENT OF A JET
044178,000168:                                                                                                  #        FAILED-ON.
044179,000169: 
044180,000170:                                                                                                  #  WHEN THE OPERATING POINT IS IN REGION 1 THE SAME APPROXIMATION IS
044181,000171:                                                                                                  #        MADE, BUT AT AN ARTIFICIALLY-CREATED OR DUMMY OPERATING POINT,
044182,000172:                                                                                                  #        DEFINED BY.. OGAERROR = INTERSECTION OF CONTROL PARABOLA AND
044183,000173:                                                                                                  #        OGAERROR AXIS, OGARATE = +-LMCRATE WHERE SIGN IS OPPOSITE THAT OF
044184,000174:                                                                                                  #        REAL OPERATING POINT RATE.  WHEN THE OPERATING POINT HAS PASSED
044185,000175:                                                                                                  #        FROM REGION 1 TO REGION 6', THE DUMMY POINT IS NO LONGER REQUIRED,
044186,000176:                                                                                                  #        AND THE SOLUTION REVERTS TO THAT OF A REGULAR REGION 6' POINT.
044187,000177: 
044188,000178:                                                                                                  #  EQUATION FOR SWITCHING PARABOLA (SEE FIGURE ABOVE)....
044189,000179:                                                                                                  #                                    2
044190,000180:                                                                                                  #        SOGAERROR = (DB - (SOGARATE) (1/CONACC)/2) SGN(SOGARATE)
044191,000181: 
044192,000182:                                                                                                  #  EQUATION FOR SWITCHING STRAIGHT LINE SEGMENT....
044193,000183: 
044194,000184:                                                                                                  #        SOGARATE = -(-SLOPE)(SOGAERROR) - SGN(SOGARATE) INTERCEP
044195,000185: 
044196,000186:                                                                                                  #                WHERE  INTERCEP = DB(-SLOPE) - LMCRATE

044198,000188: 
044199,000189:                                                                                                  #  EQUATION FOR INTERSECTION, CONTROL PARABOLA AND STRAIGHT SWITCH LINE....
044200,000190: 
044201,000191:                                                                                                  #        DOGADOT = NUM/DEN, WHERE
044202,000192:                                                                                                  #                                       2
044203,000193:                                                                                                  #                NUM = (-SLOPE)(OGARATE) (1/CONACC)
044204,000194:                                                                                                  #                      +SGN(DELOGA)(-SLOPE)(OGAERROR - SGN(DELOGA)(DB))
044205,000195:                                                                                                  #                      +LMCRATE
044206,000196: 
044207,000197:                                                                                                  #                DEN = (-SLOPE)(LMCRATE)(1/CONACC) - SGN(DELOGA)
044208,000198:                                                                                                  #                                                  2
044209,000199:                                                                                                  #                DELOGA = OGAERROR - (DB -  OGARATE (1/CONACC)/2)SGN(OGARATE)

044214,000204:                                                                                                  #        AND OGARATE, WHERE THE DUMMY POINT IS GIVEN BY....
