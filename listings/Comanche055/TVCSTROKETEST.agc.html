Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Colossus 249 (Apollo 9) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 055 OF AGC PROGRAM COMANCHE BY NASA 2021113-051
	10:28 APR.  1,1969
Note that the date is the date of the printout, not the date of the program revision.

043556,000035:                                                                                                  #  MODIFIED BY SCHLUNDT                          21 OCTOBER 1968
043557,000036: 
043558,000037:                                                                                                  #  FUNCTIONAL DESCRIPTION....
043559,000038:                                                                                                  #        STROKE TEST PACKAGE GENERATES A WAVEFORM DESIGNED TO EXCITE BENDING
043560,000039:                                                                                                  #        STRKTSTI (STROKE TEST INITIALIZATION) IS CALLED AS A JOB BY VB68.
043561,000040:                                                                                                  #                IT INITIALIZES ALL ERASABLES REQD FOR A STROKE TEST, AND
043562,000041:                                                                                                  #                THEN TESTS FOR CSM/LM (BIT 13 OF DAPDATR1). IF CSM/LM,
043563,000042:                                                                                                  #                IN EITHER HIGH OR LOW-BANDWIDTH MODE, THE TEST IS STARTED
043564,000043:                                                                                                  #                IMMEDIATELY.  IF NOT CSM/LM, PROGRAM EXITS WITH NO ACTION.
043565,000044:                                                                                                  #        HACK (STROKE TEST) GENERATES THE WAVEFORM BY DUMPING PULSE BURSTS
043566,000045:                                                                                                  #                OF PROPER SIGN AND IN PROPER SEQUENCE DIRECTLY INTO
043567,000046:                                                                                                  #                TVCPITCH, WORKING IN CONJUNCITON WITH BOTH PITCH AND YAW
043568,000047:                                                                                                  #                TVC DAPS, WITH INTERMEDIAT WAITLIST CALLS.  NOTE, HOWEVER
043569,000048:                                                                                                  #                THAT THE STROKE TEST IS PERFORMED ONLY IN THE PITCH AXIS.
043570,000049:                                                                                                  #                AN EXAMPLE WAVEFORM IS GIVEN BELOW, TO DEMONSTRATE STROKE-
043571,000050:                                                                                                  #                TEST PARAMETER SELECTION.
043572,000051:                                                                                                  #        RESTARTS CAUSE TEST TO BE TERMINATED.  ANOTHER V68 REQD IF TEST
043573,000052:                                                                                                  #                IS TO BE RE-RUN.
043574,000053:                                                                                                  #        PULSE BURST SIZE IS PAD-LOADED (ESTROKER) SO THAT AMPLITUDE OF
043575,000054:                                                                                                  #                WAVEFORM CAN BE CHANGED.  THERE ARE TEN PULSE BURSTS IN
043576,000055:                                                                                                  #                THE HALF-AMPLITUDE OF THE FIRST FREQUENCY SET IN THE
043577,000056:                                                                                                  #                STANDARD WAVEFORM.  AMPLITUDE IS 10(ESTROKER)(1/42.15),
043578,000057:                                                                                                  #                NOMINALLY 50/42.15 = 1.185 DEG
043579,000058: 
043580,000059:                                                                                                  #  CALLING SEQUENCE....
043581,000060:                                                                                                  #        EXTENDED VERB 68 SETS UP STRKTSTI JOB
043582,000061:                                                                                                  #        PITCH AND YAW TVCDAPS, FINDING STROKER NON-ZERO, DO A "TC HACK"
043583,000062:                                                                                                  #        AN INTERNALLY-GENERATED WAITLIST CALL ENTERS AT "HACKWLST"
043584,000063: 
043585,000064:                                                                                                  #  NORMAL EXIT MODES....
043586,000065:                                                                                                  #        TC BUNKER ("Q" IF ENTRY FROM DAP, "TCTSKOVR" IF FROM WAITLIST) LIST
043587,000066: 
043588,000067:                                                                                                  #  SUBROUTINES CALLED....
043589,000068:                                                                                                  #        WAITLIST
043590,000069: 
043591,000070:                                                                                                  #  ALARM OR ABORT EXIT MODES....
043592,000071:                                                                                                  #        NONE
043593,000072: 
043594,000073:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED....
043595,000074:                                                                                                  #        ESTROKER (PAD-LOAD)
043596,000075:                                                                                                  #        STROKER, CADDY, REVS, CARD, N
