Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Colossus 249 (Apollo 9) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 055 OF AGC PROGRAM COMANCHE BY NASA 2021113-051
	10:28 APR.  1,1969
Note that the date is the date of the printout, not the date of the program revision.

053227,000040:                                                                                                  #  MOD NO - 2                                                    LOG SECTION - WAITLIST

053232,000045:                                                                                                  #  FUNCTIONAL DESCRIPTION -
053233,000046:                                                                                                  #        PART OF A SECTION OF PROGRAMS,- WAITLIST, TASKOVER, T3RUPT, USED TO CALL A PROGRAM, (CALLED A TASK),
053234,000047:                                                                                                  #        WHICH IS TO BEGIN IN C(A) CENTISECONDS.  WAITLIST UPDATES TIME3, LST1 AND LST2.  THE MEANING OF THESE LISTS
053235,000048:                                                                                                  #        FOLLOW.
053236,000049: 
053237,000050:                                                                                                  #                C(TIME3) = 16384 -(T1-T) CENTISECONDS, (T=PRESENT TIME, T1-TIME FOR TASK1)
053238,000051: 
053239,000052:                                                                                                  #                        C(LST1)         =       -(T2-T1)+1
053240,000053:                                                                                                  #                        C(LST1 +1)      =       -(T3-T2)+1
053241,000054:                                                                                                  #                        C(LST1 +2)      =       -(T4-T3)+1
053242,000055:                                                                                                  #                                        .
053243,000056:                                                                                                  #                                        .
053244,000057:                                                                                                  #                        C(LST1 +6)      =       -(T8-T7)+1
053245,000058:                                                                                                  #                        C(LST1 +7)      =       -(T9-T8)+1
053246,000059: 
053247,000060:                                                                                                  #                        C(LST2)         =       2CADR OF TASK1
053248,000061:                                                                                                  #                        C(LST2 +2)      =       2CADR OF TASK2
053249,000062:                                                                                                  #                                        .
053250,000063:                                                                                                  #                                        .
053251,000064:                                                                                                  #                        C(LST2 +14)     =       2CADR OF TASK8
053252,000065:                                                                                                  #                        C(LST2 +16)     =       2CADR OF TASK9
053253,000066: 
053254,000067:                                                                                                  #  WARNINGS-
053255,000068:                                                                                                  #  --------
053256,000069:                                                                                                  #        1)      1 <= C(A) <= 16250D (1 CENTISECOND TO 162.5 SEC)
053257,000070:                                                                                                  #        2)      9 TASKS MAXIMUM
053258,000071:                                                                                                  #        3)      TASKS CALLED UNDER INTERRUPT INHIBITED
053259,000072:                                                                                                  #        4)      TASKS END BY TC TASKOVER
053260,000073: 
053261,000074:                                                                                                  #  CALLING SEQUENCE -
053262,000075:                                                                                                  #        L-1     CA      DELTAT  (TIME IN CENTISECONDS TO TASK START)
053263,000076:                                                                                                  #        L       TC      WAITLIST
053264,000077:                                                                                                  #        L+1     2CADR   DESIRED TASK
053265,000078:                                                                                                  #        L+2     (MINOR OF 2CADR)
053266,000079:                                                                                                  #        L+3     RELINT          (RETURNS HERE)
053267,000080: 
053268,000081:                                                                                                  #  TWIDDLE-
053269,000082:                                                                                                  #  -------
053270,000083:                                                                                                  #        TWIDDLE IS FOR USE WHEN THE TASK BEING SET UP IS IN THE SAME EBANK AND FBANK AS THE USER.  IN
053271,000084:                                                                                                  #        SUCH CASES, IT IMPROVES UPON WAITLIST BY ELIMINATING THE NEED FOR THE BBCON HALF OF THE 2CADR,

Page 1222 

053273,000086:                                                                                                  #        SAVING A WORD.  TWIDDLE IS LIKE WAITLIST IN EVERY RESPECT EXCEPT CALLING SEQUENCE, TO WIT-
053274,000087:                                                                                                  #                L-1     CA      DELTAT
053275,000088:                                                                                                  #                L       TC      TWIDDLE
053276,000089:                                                                                                  #                L+1     ADRES   DESIRED TASK
053277,000090:                                                                                                  #                L+2     RELINT          (RETURNS HERE)
053278,000091: 
053279,000092:                                                                                                  #  NORMAL EXIT MODES -
053280,000093:                                                                                                  #        AT L+3 OF CALLING SEQUENCE
053281,000094: 
053282,000095:                                                                                                  #  ALARM OR ABORT EXIT MODES -
053283,000096:                                                                                                  #        TC      ABORT
053284,000097:                                                                                                  #        OCT     1203    (WAITLIST OVERFLOW - TOO MANY TASKS)
053285,000098: 
053286,000099:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED -
053287,000100:                                                                                                  #        ACCOMPLISHED BY FRESH START,--  LST2, ..., LST2 +16 = ENDTASK
053288,000101:                                                                                                  #                                        LST1, ..., LST1 +7  = NEG1/2
053289,000102: 
053290,000103:                                                                                                  #  OUTPUT --
053291,000104:                                                                                                  #        LST1 AND LST2 UPDATED WITH NEW TASK AND ASSOCIATED TIME.
053292,000105: 
053293,000106:                                                                                                  #  DEBRIS -
053294,000107:                                                                                                  #        CENTRALS - A,Q,L
053295,000108:                                                                                                  #        OTHER    - WAITEXIT, WAITADR, WAITTEMP, WAITBANK
053296,000109: 
053297,000110:                                                                                                  #  DETAILED ANALYSIS OF TIMING -
053298,000111:                                                                                                  #        CONTROL WILL NOT BE RETURNED TO THE SPECIFIED ADDRESS (2CADR) IN EXACTLY DELTA T CENTISECONDS.
053299,000112:                                                                                                  #        THE APPROXIMATE TIME MAY BE CALCULATED AS FOLLOWS
053300,000113:                                                                                                  #                LET T0 = THE TIME OF THE TC WAITLIST
053301,000114:                                                                                                  #                LET TS = T0 +147U + COUNTER INCREMENTS (SET UP TIME)
053302,000115:                                                                                                  #                LET X  = TS -(100TS)/100  (VARIANCE FROM COUNTERS)
053303,000116:                                                                                                  #                LET Y  = LENGTH OF TIME OF INHIBIT INTERRUPT AFTER T3RUPT
053304,000117:                                                                                                  #                LET Z  = LENGTH OF TIME TO PROCESS TASKS WHICH ARE DUE THIS T3RUPT BUT DISPATCHED EARLIER.
053305,000118:                                                                                                  #                (Z=0, USUALLY)
053306,000119:                                                                                                  #                LET DELTD = THE ACTUAL TIME TAKEN TO GIVE CONTROL TO 2CADR
053307,000120:                                                                                                  #        THEN DELTD = TS+DELTA T -X +Y +Z +1.05MS* +COUNTERS*
053308,000121:                                                                                                  #                *-THE TIME TAKEN BY WAITLIST ITSELF AND THE COUNTER TICKING DURING THIS WAITLIST TIME.
053309,000122:                                                                                                  #        IN SHORT, THE ACTUAL TIME TO RETURN CONTROL TO A 2CADR IS AUGMENTED BY THE TIME TO SET UP THE TASK:S
053310,000123:                                                                                                  #        INTERRUPT, ALL COUNTERS TICKING, THE T3RUPT PROCESSING TIME, THE WAITLIST PROCESSING TIME AND THE POSSIBILITY
053311,000124:                                                                                                  #        OF OTHER TASKS INHIBITING THE INTERRUPT.
