Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Colossus 249 (Apollo 9) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 055 OF AGC PROGRAM COMANCHE BY NASA 2021113-051
	10:28 APR.  1,1969
Note that the date is the date of the printout, not the date of the program revision.

052940,000034:                                                                                                  #  PROGRAM DESCRIPTION                                           DATE -- 10 OCTOBER 1966
052941,000035:                                                                                                  #  MOD NO -- 2                                                   LOG SECTION -- WAITLIST
052942,000036:                                                                                                  #  MOD BY -- MILLER      (DTMAX INCREASED TO 162.5 SEC)          ASSEMBLY -- SUNBURST REV 5
052943,000037:                                                                                                  #  MOD 3 BY KERNAN       (INHINT INSERTED AT WAITLIST) 2/28/68 SKIPPER REV 4
052944,000038:                                                                                                  #  MOD 4 BY KERNAN       (TWIDDLE IN 54) 3/28/68 SKIPPER REV 13.
052945,000039: 
052946,000040:                                                                                                  #  FUNCTIONAL DESCRIPTION --
052947,000041:                                                                                                  #        PART OF A SECTION OF PROGRAMS -- WAITLIST, TASKOVER, T3RUPT, USED TO CALL A PROGRAM (CALLED A TASK),
052948,000042:                                                                                                  #        WHICH IS TO BEGIN IN C(A) CENTISECONDS.  WAITLIST UPDATES TIME3, LST1, AND LST2.  THE MEANING OF THESE LISTS
052949,000043:                                                                                                  #        FOLLOW.
052950,000044: 
052951,000045:                                                                                                  #                C(TIME3) = 16384 -(T1-T) CENTISECONDS, (T=PRESENT TIME, T1-TIME FOR TASK1)
052952,000046: 
052953,000047:                                                                                                  #                        C(LST1)         =       -(T2-T1)+1
052954,000048:                                                                                                  #                        C(LST1 +1)      =       -(T3-T2)+1
052955,000049:                                                                                                  #                        C(LST1 +2)      =       -(T4-T3)+1
052956,000050:                                                                                                  #                                       ...
052957,000051:                                                                                                  #                        C(LST1 +6)      =       -(T8-T7)+1
052958,000052:                                                                                                  #                        C(LST1 +7)      =       -(T9-T8)+1
052959,000053: 
052960,000054:                                                                                                  #                        C(LST2)         =       2CADR OF TASK1
052961,000055:                                                                                                  #                        C(LST2 +2)      =       2CADR OF TASK2
052962,000056:                                                                                                  #                                       ...
052963,000057:                                                                                                  #                        C(LST2 +14)     =       2CADR OF TASK8
052964,000058:                                                                                                  #                        C(LST2 +16)     =       2CADR OF TASK9
052965,000059: 
052966,000060:                                                                                                  #  WARNINGS --
052967,000061:                                                                                                  #        1)      1 <= C(A) <= 16250D (1 CENTISCOND TO 162.5 SEC)
052968,000062:                                                                                                  #        2)      9 TASKS MAXIMUM
052969,000063:                                                                                                  #        3)      TASKS CALLED UNDER INTERRUPT INHIBITED
052970,000064:                                                                                                  #        4)      TASKS END BY TC TASKOVER
052971,000065: 
052972,000066:                                                                                                  #  CALLING SEQUENCE --
052973,000067:                                                                                                  #        L-1     CA      DELTAT  (TIME IN CENTISECONDS TO TASK START)
052974,000068:                                                                                                  #        L       TC      WAITLIST
052975,000069:                                                                                                  #        L+1     2CADR   DESIRED TASK
052976,000070:                                                                                                  #        L+2     (MINOR OF 2CADR)
052977,000071:                                                                                                  #        L+3     RELINT          (RETURNS HERE)
052978,000072: 
052979,000073:                                                                                                  #  TWIDDLE --
052980,000074:                                                                                                  #        TWIDDLE IS FOR USE WHEN THE TASK BEING SET UP IS IN THE SAME EBANK AND FBANK AS THE USER.  IN
052981,000075:                                                                                                  #        SUCH CASES, IT IMPROVES UPON WAITLIST BY ELIMINATING THE NEED FOR THE BBCON HALF OF THE 2CADR,

Page 1222 

052983,000077:                                                                                                  #        SAVING A WORD.  TWIDDLE IS LIKE WAITLIST IN EVERY RESPECT EXCEPT CALLING SEQUENCE, TO WIT,
052984,000078:                                                                                                  #                L-1     CA      DELTAT
052985,000079:                                                                                                  #                L       TC      TWIDDLE
052986,000080:                                                                                                  #                L+1     ADRES   DESIRED TASK
052987,000081:                                                                                                  #                L+2     RELINT          (RETURNS HERE)
052988,000082: 
052989,000083:                                                                                                  #  NORMAL EXIT MODES --
052990,000084:                                                                                                  #        AT L+3 OF CALLING SEQUENCE.
052991,000085: 
052992,000086:                                                                                                  #  ALARM OR ABORT EXIT MODES --
052993,000087:                                                                                                  #        TC      ABORT
052994,000088:                                                                                                  #        OCT     1203    (WAITLIST OVERFLOW -- TOO MANY TASKS)
052995,000089: 
052996,000090:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED --
052997,000091:                                                                                                  #        ACCOMPLISHED BY FRESH START --  LST2, ..., LST2 +16 = ENDTASK
052998,000092:                                                                                                  #                                        LST1, ..., LST1 +7  = NEG1/2
052999,000093: 
053000,000094:                                                                                                  #  OUTPUT --
053001,000095:                                                                                                  #        LST1 AND LST2 UPDATED WITH NEW TASK AND ASSOCIATED TIME.
053002,000096: 
053003,000097:                                                                                                  #  DEBRIS --
053004,000098:                                                                                                  #        CENTRALS -- A,Q,L
053005,000099:                                                                                                  #        OTHER    -- WAITEXIT, WAITADR, WAITTEMP, WAITBANK
053006,000100: 
053007,000101:                                                                                                  #  DETAILED ANALYSIS OF TIMING --
053008,000102:                                                                                                  #        CONTROL WILL NOT BE RETURNED TO THE SPECIFIED ADDRESS (2CADR) IN EXACTLY DELTA T CENTISECONDS.
053009,000103:                                                                                                  #        THE APPROXIMATE TIME MAY BE CALCULATED AS FOLLOWS:
053010,000104:                                                                                                  #                LET T0 = THE TIME OF THE TC WAITLIST
053011,000105:                                                                                                  #                LET TS = T0 +147U + COUNTER INCREMENTS (SET UP TIME)
053012,000106:                                                                                                  #                LET X  = TS -(100TS)/100  (VARIANCE FROM COUNTERS)
053013,000107:                                                                                                  #                LET Y  = LENGTH OF TIME OF INHIBIT INTERRUPT AFTER T3RUPT
053014,000108:                                                                                                  #                LET Z  = LENGTH OF TIME TO PROCESS TASKS WHICH ARE DUE THIS T3RUPT BUT DISPATCHED EARLIER.
053015,000109:                                                                                                  #                (Z=0, USUALLY).
053016,000110:                                                                                                  #                LET DELTD = THE ACTUAL TIME TAKEN TO GIVE CONTROL TO 2CADR
053017,000111:                                                                                                  #        THEN DELTD = TS+DELTA T -X +Y +Z +1.05MS* +COUNTERS*
053018,000112:                                                                                                  #                *THE TIME TAKEN BY WAITLIST ITSELF AND THE COUNTER TICKING DURING THIS WAITLIST TIME.
053019,000113:                                                                                                  #        IN SHORT, THE ACTUAL TIME TO RETURN CONTROL TO A 2CADR IS AUGMENTED BY THE TIME TO SET UP THE TASK'S
053020,000114:                                                                                                  #        INTERRUPT, ALL COUNTERS TICKING, THE T3RUPT PROCESSING TIME, THE WAITLIST PROCESSING TIME AND THE POSSIBILITY
053021,000115:                                                                                                  #        OF OTHER TASKS INHIBITING THE INTERRUPT.
