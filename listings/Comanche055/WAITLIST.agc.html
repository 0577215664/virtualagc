Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Colossus 249 (Apollo 9) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 055 OF AGC PROGRAM COMANCHE BY NASA 2021113-051
	10:28 APR.  1,1969
Note that the date is the date of the printout, not the date of the program revision.

053235,000045:                                                                                                  #  FUNCTIONAL DESCRIPTION -
053236,000046:                                                                                                  #        PART OF A SECTION OF PROGRAMS,- WAITLIST, TASKOVER, T3RUPT, USED TO CALL A PROGRAM, (CALLED A TASK),
053237,000047:                                                                                                  #        WHICH IS TO BEGIN IN C(A) CENTISECONDS.  WAITLIST UPDATES TIME3, LST1 AND LST2.  THE MEANING OF THESE LISTS
053238,000048:                                                                                                  #        FOLLOW.
053239,000049: 
053240,000050:                                                                                                  #                C(TIME3) = 16384 -(T1-T) CENTISECONDS, (T=PRESENT TIME, T1-TIME FOR TASK1)
053241,000051: 
053242,000052:                                                                                                  #                        C(LST1)         =       -(T2-T1)+1
053243,000053:                                                                                                  #                        C(LST1 +1)      =       -(T3-T2)+1
053244,000054:                                                                                                  #                        C(LST1 +2)      =       -(T4-T3)+1
053245,000055:                                                                                                  #                                        .
053246,000056:                                                                                                  #                                        .
053247,000057:                                                                                                  #                        C(LST1 +6)      =       -(T8-T7)+1
053248,000058:                                                                                                  #                        C(LST1 +7)      =       -(T9-T8)+1
053249,000059: 
053250,000060:                                                                                                  #                        C(LST2)         =       2CADR OF TASK1
053251,000061:                                                                                                  #                        C(LST2 +2)      =       2CADR OF TASK2
053252,000062:                                                                                                  #                                        .
053253,000063:                                                                                                  #                                        .
053254,000064:                                                                                                  #                        C(LST2 +14)     =       2CADR OF TASK8
053255,000065:                                                                                                  #                        C(LST2 +16)     =       2CADR OF TASK9
053256,000066: 
053257,000067:                                                                                                  #  WARNINGS-
053258,000068:                                                                                                  #  --------
053259,000069:                                                                                                  #        1)      1 <= C(A) <= 16250D (1 CENTISECOND TO 162.5 SEC)
053260,000070:                                                                                                  #        2)      9 TASKS MAXIMUM
053261,000071:                                                                                                  #        3)      TASKS CALLED UNDER INTERRUPT INHIBITED
053262,000072:                                                                                                  #        4)      TASKS END BY TC TASKOVER
053263,000073: 
053264,000074:                                                                                                  #  CALLING SEQUENCE -
053265,000075:                                                                                                  #        L-1     CA      DELTAT  (TIME IN CENTISECONDS TO TASK START)
053266,000076:                                                                                                  #        L       TC      WAITLIST
053267,000077:                                                                                                  #        L+1     2CADR   DESIRED TASK
053268,000078:                                                                                                  #        L+2     (MINOR OF 2CADR)
053269,000079:                                                                                                  #        L+3     RELINT          (RETURNS HERE)
053270,000080: 
053271,000081:                                                                                                  #  TWIDDLE-
053272,000082:                                                                                                  #  -------
053273,000083:                                                                                                  #        TWIDDLE IS FOR USE WHEN THE TASK BEING SET UP IS IN THE SAME EBANK AND FBANK AS THE USER.  IN
053274,000084:                                                                                                  #        SUCH CASES, IT IMPROVES UPON WAITLIST BY ELIMINATING THE NEED FOR THE BBCON HALF OF THE 2CADR,

Page 1222 

053276,000086:                                                                                                  #        SAVING A WORD.  TWIDDLE IS LIKE WAITLIST IN EVERY RESPECT EXCEPT CALLING SEQUENCE, TO WIT-
053277,000087:                                                                                                  #                L-1     CA      DELTAT
053278,000088:                                                                                                  #                L       TC      TWIDDLE
053279,000089:                                                                                                  #                L+1     ADRES   DESIRED TASK
053280,000090:                                                                                                  #                L+2     RELINT          (RETURNS HERE)
053281,000091: 
053282,000092:                                                                                                  #  NORMAL EXIT MODES -
053283,000093:                                                                                                  #        AT L+3 OF CALLING SEQUENCE
053284,000094: 
053285,000095:                                                                                                  #  ALARM OR ABORT EXIT MODES -
053286,000096:                                                                                                  #        TC      ABORT
053287,000097:                                                                                                  #        OCT     1203    (WAITLIST OVERFLOW - TOO MANY TASKS)
053288,000098: 
053289,000099:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED -
053290,000100:                                                                                                  #        ACCOMPLISHED BY FRESH START,--  LST2, ..., LST2 +16 = ENDTASK
053291,000101:                                                                                                  #                                        LST1, ..., LST1 +7  = NEG1/2
053292,000102: 
053293,000103:                                                                                                  #  OUTPUT --
053294,000104:                                                                                                  #        LST1 AND LST2 UPDATED WITH NEW TASK AND ASSOCIATED TIME.
053295,000105: 
053296,000106:                                                                                                  #  DEBRIS -
053297,000107:                                                                                                  #        CENTRALS - A,Q,L
053298,000108:                                                                                                  #        OTHER    - WAITEXIT, WAITADR, WAITTEMP, WAITBANK
053299,000109: 
053300,000110:                                                                                                  #  DETAILED ANALYSIS OF TIMING -
053301,000111:                                                                                                  #        CONTROL WILL NOT BE RETURNED TO THE SPECIFIED ADDRESS (2CADR) IN EXACTLY DELTA T CENTISECONDS.
053302,000112:                                                                                                  #        THE APPROXIMATE TIME MAY BE CALCULATED AS FOLLOWS
053303,000113:                                                                                                  #                LET T0 = THE TIME OF THE TC WAITLIST
053304,000114:                                                                                                  #                LET TS = T0 +147U + COUNTER INCREMENTS (SET UP TIME)
053305,000115:                                                                                                  #                LET X  = TS -(100TS)/100  (VARIANCE FROM COUNTERS)
053306,000116:                                                                                                  #                LET Y  = LENGTH OF TIME OF INHIBIT INTERRUPT AFTER T3RUPT
053307,000117:                                                                                                  #                LET Z  = LENGTH OF TIME TO PROCESS TASKS WHICH ARE DUE THIS T3RUPT BUT DISPATCHED EARLIER.
053308,000118:                                                                                                  #                (Z=0, USUALLY)
053309,000119:                                                                                                  #                LET DELTD = THE ACTUAL TIME TAKEN TO GIVE CONTROL TO 2CADR
053310,000120:                                                                                                  #        THEN DELTD = TS+DELTA T -X +Y +Z +1.05MS* +COUNTERS*
053311,000121:                                                                                                  #                *-THE TIME TAKEN BY WAITLIST ITSELF AND THE COUNTER TICKING DURING THIS WAITLIST TIME.
053312,000122:                                                                                                  #        IN SHORT, THE ACTUAL TIME TO RETURN CONTROL TO A 2CADR IS AUGMENTED BY THE TIME TO SET UP THE TASK:S
053313,000123:                                                                                                  #        INTERRUPT, ALL COUNTERS TICKING, THE T3RUPT PROCESSING TIME, THE WAITLIST PROCESSING TIME AND THE POSSIBILITY
053314,000124:                                                                                                  #        OF OTHER TASKS INHIBITING THE INTERRUPT.
