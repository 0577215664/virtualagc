Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Colossus 249 (Apollo 9) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 055 OF AGC PROGRAM COMANCHE BY NASA 2021113-051
	10:28 APR.  1,1969
Note that the date is the date of the printout, not the date of the program revision.

053230,000045:                                                                                                  #  FUNCTIONAL DESCRIPTION -
053231,000046:                                                                                                  #        PART OF A SECTION OF PROGRAMS,- WAITLIST, TASKOVER, T3RUPT, USED TO CALL A PROGRAM, (CALLED A TASK),
053232,000047:                                                                                                  #        WHICH IS TO BEGIN IN C(A) CENTISECONDS.  WAITLIST UPDATES TIME3, LST1 AND LST2.  THE MEANING OF THESE LISTS
053233,000048:                                                                                                  #        FOLLOW.
053234,000049: 
053235,000050:                                                                                                  #                C(TIME3) = 16384 -(T1-T) CENTISECONDS, (T=PRESENT TIME, T1-TIME FOR TASK1)
053236,000051: 
053237,000052:                                                                                                  #                        C(LST1)         =       -(T2-T1)+1
053238,000053:                                                                                                  #                        C(LST1 +1)      =       -(T3-T2)+1
053239,000054:                                                                                                  #                        C(LST1 +2)      =       -(T4-T3)+1
053240,000055:                                                                                                  #                                        .
053241,000056:                                                                                                  #                                        .
053242,000057:                                                                                                  #                        C(LST1 +6)      =       -(T8-T7)+1
053243,000058:                                                                                                  #                        C(LST1 +7)      =       -(T9-T8)+1
053244,000059: 
053245,000060:                                                                                                  #                        C(LST2)         =       2CADR OF TASK1
053246,000061:                                                                                                  #                        C(LST2 +2)      =       2CADR OF TASK2
053247,000062:                                                                                                  #                                        .
053248,000063:                                                                                                  #                                        .
053249,000064:                                                                                                  #                        C(LST2 +14)     =       2CADR OF TASK8
053250,000065:                                                                                                  #                        C(LST2 +16)     =       2CADR OF TASK9
053251,000066: 
053252,000067:                                                                                                  #  WARNINGS-
053253,000068:                                                                                                  #  --------
053254,000069:                                                                                                  #        1)      1 <= C(A) <= 16250D (1 CENTISECOND TO 162.5 SEC)
053255,000070:                                                                                                  #        2)      9 TASKS MAXIMUM
053256,000071:                                                                                                  #        3)      TASKS CALLED UNDER INTERRUPT INHIBITED
053257,000072:                                                                                                  #        4)      TASKS END BY TC TASKOVER
053258,000073: 
053259,000074:                                                                                                  #  CALLING SEQUENCE -
053260,000075:                                                                                                  #        L-1     CA      DELTAT  (TIME IN CENTISECONDS TO TASK START)
053261,000076:                                                                                                  #        L       TC      WAITLIST
053262,000077:                                                                                                  #        L+1     2CADR   DESIRED TASK
053263,000078:                                                                                                  #        L+2     (MINOR OF 2CADR)
053264,000079:                                                                                                  #        L+3     RELINT          (RETURNS HERE)
053265,000080: 
053266,000081:                                                                                                  #  TWIDDLE-
053267,000082:                                                                                                  #  -------
053268,000083:                                                                                                  #        TWIDDLE IS FOR USE WHEN THE TASK BEING SET UP IS IN THE SAME EBANK AND FBANK AS THE USER.  IN
053269,000084:                                                                                                  #        SUCH CASES, IT IMPROVES UPON WAITLIST BY ELIMINATING THE NEED FOR THE BBCON HALF OF THE 2CADR,

Page 1222 

053271,000086:                                                                                                  #        SAVING A WORD.  TWIDDLE IS LIKE WAITLIST IN EVERY RESPECT EXCEPT CALLING SEQUENCE, TO WIT-
053272,000087:                                                                                                  #                L-1     CA      DELTAT
053273,000088:                                                                                                  #                L       TC      TWIDDLE
053274,000089:                                                                                                  #                L+1     ADRES   DESIRED TASK
053275,000090:                                                                                                  #                L+2     RELINT          (RETURNS HERE)
053276,000091: 
053277,000092:                                                                                                  #  NORMAL EXIT MODES -
053278,000093:                                                                                                  #        AT L+3 OF CALLING SEQUENCE
053279,000094: 
053280,000095:                                                                                                  #  ALARM OR ABORT EXIT MODES -
053281,000096:                                                                                                  #        TC      ABORT
053282,000097:                                                                                                  #        OCT     1203    (WAITLIST OVERFLOW - TOO MANY TASKS)
053283,000098: 
053284,000099:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED -
053285,000100:                                                                                                  #        ACCOMPLISHED BY FRESH START,--  LST2, ..., LST2 +16 = ENDTASK
053286,000101:                                                                                                  #                                        LST1, ..., LST1 +7  = NEG1/2
053287,000102: 
053288,000103:                                                                                                  #  OUTPUT --
053289,000104:                                                                                                  #        LST1 AND LST2 UPDATED WITH NEW TASK AND ASSOCIATED TIME.
053290,000105: 
053291,000106:                                                                                                  #  DEBRIS -
053292,000107:                                                                                                  #        CENTRALS - A,Q,L
053293,000108:                                                                                                  #        OTHER    - WAITEXIT, WAITADR, WAITTEMP, WAITBANK
053294,000109: 
053295,000110:                                                                                                  #  DETAILED ANALYSIS OF TIMING -
053296,000111:                                                                                                  #        CONTROL WILL NOT BE RETURNED TO THE SPECIFIED ADDRESS (2CADR) IN EXACTLY DELTA T CENTISECONDS.
053297,000112:                                                                                                  #        THE APPROXIMATE TIME MAY BE CALCULATED AS FOLLOWS
053298,000113:                                                                                                  #                LET T0 = THE TIME OF THE TC WAITLIST
053299,000114:                                                                                                  #                LET TS = T0 +147U + COUNTER INCREMENTS (SET UP TIME)
053300,000115:                                                                                                  #                LET X  = TS -(100TS)/100  (VARIANCE FROM COUNTERS)
053301,000116:                                                                                                  #                LET Y  = LENGTH OF TIME OF INHIBIT INTERRUPT AFTER T3RUPT
053302,000117:                                                                                                  #                LET Z  = LENGTH OF TIME TO PROCESS TASKS WHICH ARE DUE THIS T3RUPT BUT DISPATCHED EARLIER.
053303,000118:                                                                                                  #                (Z=0, USUALLY)
053304,000119:                                                                                                  #                LET DELTD = THE ACTUAL TIME TAKEN TO GIVE CONTROL TO 2CADR
053305,000120:                                                                                                  #        THEN DELTD = TS+DELTA T -X +Y +Z +1.05MS* +COUNTERS*
053306,000121:                                                                                                  #                *-THE TIME TAKEN BY WAITLIST ITSELF AND THE COUNTER TICKING DURING THIS WAITLIST TIME.
053307,000122:                                                                                                  #        IN SHORT, THE ACTUAL TIME TO RETURN CONTROL TO A 2CADR IS AUGMENTED BY THE TIME TO SET UP THE TASK:S
053308,000123:                                                                                                  #        INTERRUPT, ALL COUNTERS TICKING, THE T3RUPT PROCESSING TIME, THE WAITLIST PROCESSING TIME AND THE POSSIBILITY
053309,000124:                                                                                                  #        OF OTHER TASKS INHIBITING THE INTERRUPT.
