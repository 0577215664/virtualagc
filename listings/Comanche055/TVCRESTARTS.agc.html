Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Colossus 249 (Apollo 9) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 055 OF AGC PROGRAM COMANCHE BY NASA 2021113-051
	10:28 APR.  1,1969
Note that the date is the date of the printout, not the date of the program revision.

042503,000038:                                                                                                  #  NAME....TVCRESTART PACKAGE, CONSISTING OF REDOTVC, ENABL1, 2, CMDSOUT, PHSCHK2, ETC.
042504,000039:                                                                                                  #  LOG SECTION....TVCRESTARTS                    SUBROUTINE....DAPCSM
042505,000040:                                                                                                  #  MODIFIED BY SCHLUNDT                          21 OCTOBER 1968
042506,000041: 
042507,000042:                                                                                                  #  FUNCTIONAL DESCRIPTION....
042508,000043: 
042509,000044:                                                                                                  #       *RESTART-PROCESS THE TVC DAPS, INCLUDING PITCHDAP, YAWDAP,
042510,000045:                                                                                                  #        TVCEXECUTIVE, ROLLDAP, TVCINIT4, TVCDAPON, AND CSM/.M V46 SWTCHOVR.
042511,000046: 
042512,000047:                                                                                                  #       *TVC RESTARTS DESERVE SPECIAL CONSIDERATION IN SEVERAL AREAS.
042513,000048:                                                                                                  #        RESTART DOWN-TIME IS IMPORTANT BECAUSE OF THE TRANSIENTS INTRODUCED
042514,000049:                                                                                                  #        BY THE THRUST VECTOR RETURN TO THE ACTUATOR MECHANICAL NULLS
042515,000050:                                                                                                  #        FOLLOWING TVC- AND OPTICS-ERROR-COUNTER-DISENABLES (CHANNEL 12).
042516,000051:                                                                                                  #        TVC USES A MIXTURE OF WAITLIST, T5, T6, AND JOB CALLS.  THERE IS
042517,000052:                                                                                                  #        FILTER MEMORY (UP TO 6TH ORDER) TO BE PROTECTED IF WILD TRANSIENTS
042518,000053:                                                                                                  #        ARE TO BE AVOIDED.  COUNTERS ARE INVOLVED FOR ONE-SHOT
042519,000054:                                                                                                  #        CORRECTIONS AND GAIN UPDATES.  THE GIMBAL TRIM ESTIMATORS AND THE
042520,000055:                                                                                                  #        BODY AXIS ATTITUDE ERROR INTEGRATORS INVOLVE DIGITAL SUMMATION.
042521,000056:                                                                                                  #        DIGITAL DIFFERENTIATORS ARE INVOLVED IN THE BODY AXIS RATE ESTIMA-
042522,000057:                                                                                                  #        TIONS AND IN THE OUTPUTTING OF ACTUATOR COMMANDS.  THERE IS AN
042523,000058:                                                                                                  #        OFFSET-TRACKER-FILTER TO PROTECT.  ETC., ETC.
042524,000059: 
042525,000060:                                                                                                  #       *THOSE QUANTITIES WHICH MUST BE PROTECTED ARE STORED IN TEMPORARY
042526,000061:                                                                                                  #        REGISTERS AS THEY ARE COMPUTED, FOR UPDATING THE REAL REGISTERS
042527,000062:                                                                                                  #        DURING COPYCYCLES.
042528,000063: 
042529,000064:                                                                                                  #       *THE SEVERAL COPYCYCLES ARE EACH PROTECTED BY PHASE POINTS AT THEIR
042530,000065:                                                                                                  #        BEGINNING AND AT THEIR TERMINATION.  THE PHASE POINTS ARE SIMPLY
042531,000066:                                                                                                  #        "INCR" INSTRUCTIONS, EITHER "INCR TVCEXPHS" FOR COPYCYCLES
042532,000067:                                                                                                  #        IN THE TVCEXECUTIVE, OR "INCR TVCPHASE" FOR THE PITCH AND YAW
042533,000068:                                                                                                  #        COPYCYCLES.  INDEXING ON EACH OF THESE POINTERS THEN PERMITS A
042534,000069:                                                                                                  #        RETURN TO THE APPROPRIATE RESTART POINTS.
042535,000070: 
042536,000071:                                                                                                  #       *IF A RESTART OCCURS DURING EITHER COPYCYCLE, THAT COPYCYCLE IS
042537,000072:                                                                                                  #        COMPLETED.  THEN THE NORMAL TVCINIT4....DAPINIT....PITCHDAP STARTUP
042538,000073:                                                                                                  #        SEQUENCE IS CALLED UPON TO GET THINGS GOING AGAIN.
042539,000074: 
042540,000075:                                                                                                  #       *TVC-ENABLE AND OPTICS-ERROR-COUNTER ENABLE MUST BE SET ASAP
042541,000076:                                                                                                  #        (ALLOWING FOR PROCEDURAL DELAYS).  THEN THE ENGINES ARE COMMANDED
042542,000077:                                                                                                  #        TO THE P,YACTOFF TRIM VALUES.  THE DAPS ARE THEN READY TO GO ON THE
042543,000078:                                                                                                  #        AIR, WITH THE REGULAR STARTUP SEQUENCE, EITHER AT MRCLEAN FOR A
042544,000079:                                                                                                  #        COMPLETE INITIALIZATION OR AT TVCINIT4 FOR A PARTIAL INITIALIZATION.
042545,000080: 
042546,000081:                                                                                                  #       *FOR RESTARTS PRIOR TO THE SETTING OF THE T5 BITS AT DOTVCON THE
042547,000082:                                                                                                  #        PRE40.6 SECTION OF S40.6 TAKES CARE OF RE-ESTABLISHING TRIMS.
042548,000083: 
042549,000084:                                                                                                  #       *IF A RESTART OCCURS DURING THE TVCEXEC....TVCEXFIN SEQUENCE THE
042550,000085:                                                                                                  #        COMPUTATIONS WILL BE COMPLETED, STARTING AT THE APPROPRIATE RESTART
042551,000086:                                                                                                  #        POINT, AFTER THE DAPS ARE READY TO GO ON THE AIR.
042552,000087: 
042553,000088:                                                                                                  #       *IF A RESTART OCCURS PRIOR TO TVCINIT4 (TVCPHAS = -1) E.G. DURING
042554,000089:                                                                                                  #        THE EARLY DAP INITIALIZATION PHASE, THE DAP STARTUP SEQUENCE IS
042555,000090:                                                                                                  #        ENTERED AT MRCLEAN FOR A FULL INITIALIZATION.
042556,000091: 
042557,000092:                                                                                                  #       *FOR RESTARTS DURING CSM/LM V46 SWITCH-VER, TVCPHASE IS SET TO -2.
042558,000093:                                                                                                  #        AND THE RESTART LOGIC GOES BACK TO REDO SWITCH-OVER (AFTER THE
042559,000094:                                                                                                  #        NORMAL DAP RESTART SEQUENCE IS FOLLOWED.)
042560,000095: 
042561,000096:                                                                                                  #       *RESTARTS ARE NOT CRITICAL TO THE ROLL DAP PERFORMANCES HENCE THE
042562,000097:                                                                                                  #        ROLL DAP IS MERELY RESTARTED.
042563,000098: 
042564,000099:                                                                                                  #       *RESTARTS DURING A STROKE TEST (STROKER IS NON-ZERO) WILL CAUSE THE
042565,000100:                                                                                                  #  Page 957
042566,000101:                                                                                                  #        STROKE TEST TO BE TERMINATED.  A NEW V68 ENTRY WILL BE REQUIRED
042567,000102:                                                                                                  #        TO GET IT GOING AGAIN (NO AUTOMATIC RESTART).
042568,000103:                                                                                                  #   
042569,000104:                                                                                                  #       *REDOTVC IS REACHED FOLLOWING ANY RESTART WHICH FINDS THE T5 BITS
042570,000105:                                                                                                  #        (BITS 15,14 OF FLAGWRD6) SET FOR TVC.  DOTVCON TVCPHASE = -1
042571,000106:                                                                                                  #        AND TVC EXPHS = 0 JUST BEFORE SETTING THESE BITS, JUST BEFORE
042572,000107:                                                                                                  #        MAKING THE T5 CALL TO TVCDAPON.  ON A NORMAL SHUTDOWN DOTVCRCS
042573,000108:                                                                                                  #        CALLS RCSDAPON, WHICH RESETS THE T5 BIT FOR RCS
042574,000109: 
042575,000110:                                                                                                  #  CALLING SEQUENCE....T5, IN PARTICULAR BY ELRSKIP OF FRESH START/RESTART
042576,000111: 
042577,000112:                                                                                                  #  NORMAL EXIT MODES....RESUME, NOQRSM, POSTJUMP (TO TVCINIT4 OR MRCLEAN)
042578,000113: 
042579,000114:                                                                                                  #  ALARM OR ABORT EXIT MODES....NONE
042580,000115: 
042581,000116:                                                                                                  #  SUBROUTINES CALLED....
042582,000117: 
042583,000118:                                                                                                  #       *PCOPY+1, YCOPY+1 (PITCH AND YAW COPYCYCLES)
042584,000119:                                                                                                  #       *ENABLE1,2, CMDSOUT (RE-ESTABLISH ACTUATOR TRIMS)
042585,000120:                                                                                                  #       *MRCLEAN OR TVCINIT4 (TVCDAP INITIALIZATIONS)
042586,000121:                                                                                                  #       *SWITCHOVR +5 (CSM/LM V46 SWITCH-OVER)
042587,000122:                                                                                                  #       *EXRSTRT AND TVCEXECUTIVE PHASE POINTS 1 THRU 6
042588,000123:                                                                                                  #       *WAITLIST, IBNKCALL, POSTJUMP, ISWCALL
042589,000124: 
042590,000125:                                                                                                  #  OTHER INTERFACES....DOTVCON AND RCSDAPON (T5 BITS), ELRSKIP (CALLS IT)
042591,000126: 
042592,000127:                                                                                                  #  ERASABLE ININTIALIZATION REQUIRED....
042593,000128: 
042594,000129:                                                                                                  #       *T5 BITS (1,0), TVCPHASE (-2,-1,0,1,2,3), TVCEXPHS (1 THRU 6)
042595,000130:                                                                                                  #       *TVC DAP VARIABLES
042596,000131:                                                                                                  #       *OPERATIONS PERFORMED BY REDOTVC ARE BASED ON THE ASSUMPTION THAT
042597,000132:                                                                                                  #        THE TVC DAPS ARE RUNNING NORMALLY
