042499,000036:                                                                                                 #  NAME....TVCRESTART PACKAGE, CONSISTING OF REDOTVC, ENABL1, 2, CMDSOUT, PHSCHK2, ETC.
042500,000037:                                                                                                 #  LOG SECTION....TVCRESTARTS                     SUBROUTINE....DAPCSM
042501,000038:                                                                                                 #  MODIFIED BY SCHLUNDT                           21 OCTOBER 1968
042502,000039: 
042503,000040:                                                                                                 #  FUNCTIONAL DESCRIPTION....
042504,000041: 
042505,000042:                                                                                                 #       *RESTART-PROCESS THE TVC DAPS, INCLUDING PITCHDAP, YAWDAP,
042506,000043:                                                                                                 #         TVCEXECUTIVE, ROLLDAP, TVCINIT4, TVCDAPON, AND CSM/.M V46 SWTCHOVR.
042507,000044: 
042508,000045:                                                                                                 #       *TVC RESTARTS DESERVE SPECIAL CONSIDERATION IN SEVERAL AREAS.
042509,000046:                                                                                                 #         RESTART DOWN-TIME IS IMPORTANT BECAUSE OF THE TRANSIENTS INTRODUCED
042510,000047:                                                                                                 #         BY THE THRUST VECTOR RETURN TO THE ACTUATOR MECHANICAL NULLS
042511,000048:                                                                                                 #         FOLLOWING TVC- AND OPTICS-ERROR-COUNTER-DISENABLES (CHANNEL 12).
042512,000049:                                                                                                 #         TVC USES A MIXTURE OF WAITLIST, T5, T6, AND JOB CALLS.  THERE IS
042513,000050:                                                                                                 #         FILTER MEMORY (UP TO 6TH ORDER) TO BE PROTECTED IF WILD TRANSIENTS
042514,000051:                                                                                                 #         ARE TO BE AVOIDED.  COUNTERS ARE INVOLVED FOR ONE-SHOT
042515,000052:                                                                                                 #         CORRECTIONS AND GAIN UPDATES.  THE GIMBAL TRIM ESTIMATORS AND THE
042516,000053:                                                                                                 #         BODY AXIS ATTITUDE ERROR INTEGRATORS INVOLVE DIGITAL SUMMATION.
042517,000054:                                                                                                 #         DIGITAL DIFFERENTIATORS ARE INVOLVED IN THE BODY AXIS RATE ESTIMA-
042518,000055:                                                                                                 #         TIONS AND IN THE OUTPUTTING OF ACTUATOR COMMANDS.  THERE IS AN
042519,000056:                                                                                                 #         OFFSET-TRACKER-FILTER TO PROTECT.  ETC., ETC.
042520,000057: 
042521,000058:                                                                                                 #       *THOSE QUANTITIES WHICH MUST BE PROTECTED ARE STORED IN TEMPORARY
042522,000059:                                                                                                 #         REGISTERS AS THEY ARE COMPUTED, FOR UPDATING THE REAL REGISTERS
042523,000060:                                                                                                 #         DURING COPYCYCLES.
042524,000061: 
042525,000062:                                                                                                 #       *THE SEVERAL COPYCYCLES ARE EACH PROTECTED BY PHASE POINTS AT THEIR
042526,000063:                                                                                                 #         BEGINNING AND AT THEIR TERMINATION.  THE PHASE POINTS ARE SIMPLY
042527,000064:                                                                                                 #         "INCR" INSTRUCTIONS, EITHER "INCR TVCEXPHS" FOR COPYCYCLES
042528,000065:                                                                                                 #         IN THE TVCEXECUTIVE, OR "INCR TVCPHASE" FOR THE PITCH AND YAW
042529,000066:                                                                                                 #         COPYCYCLES.  INDEXING ON EACH OF THESE POINTERS THEN PERMITS A
042530,000067:                                                                                                 #         RETURN TO THE APPROPRIATE RESTART POINTS.
042531,000068: 
042532,000069:                                                                                                 #       *IF A RESTART OCCURS DURING EITHER COPYCYCLE, THAT COPYCYCLE IS
042533,000070:                                                                                                 #         COMPLETED.  THEN THE NORMAL TVCINIT4....DAPINIT....PITCHDAP STARTUP
042534,000071:                                                                                                 #         SEQUENCE IS CALLED UPON TO GET THINGS GOING AGAIN.
042535,000072: 
042536,000073:                                                                                                 #       *TVC-ENABLE AND OPTICS-ERROR-COUNTER ENABLE MUST BE SET ASAP
042537,000074:                                                                                                 #         (ALLOWING FOR PROCEDURAL DELAYS).  THEN THE ENGINES ARE COMMANDED
042538,000075:                                                                                                 #         TO THE P,YACTOFF TRIM VALUES.  THE DAPS ARE THEN READY TO GO ON THE
042539,000076:                                                                                                 #         AIR, WITH THE REGULAR STARTUP SEQUENCE, EITHER AT MRCLEAN FOR A
042540,000077:                                                                                                 #         COMPLETE INITIALIZATION OR AT TVCINIT4 FOR A PARTIAL INITIALIZATION.
042541,000078: 
042542,000079:                                                                                                 #       *FOR RESTARTS PRIOR TO THE SETTING OF THE T5 BITS AT DOTVCON THE
042543,000080:                                                                                                 #         PRE40.6 SECTION OF S40.6 TAKES CARE OF RE-ESTABLISHING TRIMS.
042544,000081: 
042545,000082:                                                                                                 #       *IF A RESTART OCCURS DURING THE TVCEXEC....TVCEXFIN SEQUENCE THE
042546,000083:                                                                                                 #         COMPUTATIONS WILL BE COMPLETED, STARTING AT THE APPROPRIATE RESTART
042547,000084:                                                                                                 #         POINT, AFTER THE DAPS ARE READY TO GO ON THE AIR.
042548,000085: 
042549,000086:                                                                                                 #       *IF A RESTART OCCURS PRIOR TO TVCINIT4 (TVCPHAS = -1) E.G. DURING
042550,000087:                                                                                                 #         THE EARLY DAP INITIALIZATION PHASE, THE DAP STARTUP SEQUENCE IS
042551,000088:                                                                                                 #         ENTERED AT MRCLEAN FOR A FULL INITIALIZATION.
042552,000089: 
042553,000090:                                                                                                 #       *FOR RESTARTS DURING CSM/LM V46 SWITCH-VER, TVCPHASE IS SET TO -2.
042554,000091:                                                                                                 #         AND THE RESTART LOGIC GOES BACK TO REDO SWITCH-OVER (AFTER THE
042555,000092:                                                                                                 #         NORMAL DAP RESTART SEQUENCE IS FOLLOWED.)
042556,000093: 
042557,000094:                                                                                                 #       *RESTARTS ARE NOT CRITICAL TO THE ROLL DAP PERFORMANCES HENCE THE
042558,000095:                                                                                                 #         ROLL DAP IS MERELY RESTARTED.
042559,000096: 
042560,000097:                                                                                                 #       *RESTARTS DURING A STROKE TEST (STROKER IS NON-ZERO) WILL CAUSE THE
042561,000098:                                                                                                 #  Page 957
042562,000099:                                                                                                 #         STROKE TEST TO BE TERMINATED.  A NEW V68 ENTRY WILL BE REQUIRED
042563,000100:                                                                                                 #         TO GET IT GOING AGAIN (NO AUTOMATIC RESTART).
042564,000101:                                                                                                 #   
042565,000102:                                                                                                 #       *REDOTVC IS REACHED FOLLOWING ANY RESTART WHICH FINDS THE T5 BITS
042566,000103:                                                                                                 #         (BITS 15,14 OF FLAGWRD6) SET FOR TVC.  DOTVCON TVCPHASE = -1
042567,000104:                                                                                                 #         AND TVC EXPHS = 0 JUST BEFORE SETTING THESE BITS, JUST BEFORE
042568,000105:                                                                                                 #         MAKING THE T5 CALL TO TVCDAPON.  ON A NORMAL SHUTDOWN DOTVCRCS
042569,000106:                                                                                                 #         CALLS RCSDAPON, WHICH RESETS THE T5 BIT FOR RCS
042570,000107: 
042571,000108:                                                                                                 #  CALLING SEQUENCE....T5, IN PARTICULAR BY ELRSKIP OF FRESH START/RESTART
042572,000109: 
042573,000110:                                                                                                 #  NORMAL EXIT MODES....RESUME, NOQRSM, POSTJUMP (TO TVCINIT4 OR MRCLEAN)
042574,000111: 
042575,000112:                                                                                                 #  ALARM OR ABORT EXIT MODES....NONE
042576,000113: 
042577,000114:                                                                                                 #  SUBROUTINES CALLED....
042578,000115: 
042579,000116:                                                                                                 #       *PCOPY+1, YCOPY+1 (PITCH AND YAW COPYCYCLES)
042580,000117:                                                                                                 #       *ENABLE1,2, CMDSOUT (RE-ESTABLISH ACTUATOR TRIMS)
042581,000118:                                                                                                 #       *MRCLEAN OR TVCINIT4 (TVCDAP INITIALIZATIONS)
042582,000119:                                                                                                 #       *SWITCHOVR +5 (CSM/LM V46 SWITCH-OVER)
042583,000120:                                                                                                 #       *EXRSTRT AND TVCEXECUTIVE PHASE POINTS 1 THRU 6
042584,000121:                                                                                                 #       *WAITLIST, IBNKCALL, POSTJUMP, ISWCALL
042585,000122: 
042586,000123:                                                                                                 #  OTHER INTERFACES....DOTVCON AND RCSDAPON (T5 BITS), ELRSKIP (CALLS IT)
042587,000124: 
042588,000125:                                                                                                 #  ERASABLE ININTIALIZATION REQUIRED....
042589,000126: 
042590,000127:                                                                                                 #       *T5 BITS (1,0), TVCPHASE (-2,-1,0,1,2,3), TVCEXPHS (1 THRU 6)
042591,000128:                                                                                                 #       *TVC DAP VARIABLES
042592,000129:                                                                                                 #       *OPERATIONS PERFORMED BY REDOTVC ARE BASED ON THE ASSUMPTION THAT
042593,000130:                                                                                                 #         THE TVC DAPS ARE RUNNING NORMALLY
042594,000131: 
042595,000132:                                                                                                 #  OUTPUT....
042596,000133: 
042597,000134:                                                                                                 #       *PITCH AND YAW TVC DAP COPYCYCLES COMPLETED IF INTERRUPTED
042598,000135:                                                                                                 #       *TVCEXECUTIVE COMPLETED IF INTERRUPTED
042599,000136:                                                                                                 #       *STROKE TEST TERMINATED IF INTERRUPTED
042600,000137:                                                                                                 #       *CSM/LM V46 SWITCH-OVER REPEATED IF INTERRUPTED
042601,000138:                                                                                                 #       *ACTUATOR TRIMS RE-ESTABLISHED (ACTUATORS BACK ON THE AIR)
042602,000139:                                                                                                 #       *TVC DAP INITIALIZATION AS REQUIRED
042603,000140:                                                                                                 #       *ALL TVC DAP OPERATIONS ON THE AIR
