007057,000052:                                                                                                 #  DOWNLIST FORMAT DEFINITIONS AND RULES --
007058,000053:                                                                                                 #  1. END OF A LIST = -XDNADR (X = 1 TO 6), -DNPTR, OR -DNCHAN.
007059,000054:                                                                                                 #  2. SNAPSHOT SUBLIST = LIST WHICH STARTS WITH A -1DNADR.
007060,000055:                                                                                                 #  3. SNAPSHOT SUBLIST CAN ONLY CONTAIN 1DNADRS.
007061,000056:                                                                                                 #  4. TIME2 1DNADR MUST BE LOCATED IN THE CONTROL LIST OF A DOWNLIST.
007062,000057:                                                                                                 #  5. ERASABLE DOWN TELEMETRY WORDS SHOULD BE GROUPED IN SEQUENTIAL
007063,000058:                                                                                                 #     LOCATIONS AS MUCH AS POSSIBLE TO SAVE STORAGE USED BY DOWNLINK LISTS.
007064,000059:                                                                                                 #  6. THE DOWNLINK LISTS (INCLUDING SUBLISTS) ARE ORGANIZED SUCH THAT THE ITEMS LISTED FIRST (IN FRONT OF FBANK) ARE
007065,000060:                                                                                                 #     SENT FIRST.  EXCEPTION--- SNAPSHOT SUBLISTS.  IN THE SNAPSHOT SUBLISTS THE DATA REPRESENTED BY THE FIRST
007066,000061:                                                                                                 #     11 1DNADRS IS PRESERVED (IN ORDER) IN DNTMBUFF AND SENT BY THE NEXT 11 DOWNRUPTS.  THE DATA REPRESENTED BY THE
007067,000062:                                                                                                 #     LIST IS SENT IMMEDIATELY.
