062789,000035:                                                                                                 #  DISPLAYS CAN BE CLASSIFIED INTO THE FOLLOWING CATEGORIES --
062790,000036:                                                                                                 #         1.  PRIORITY DISPLAYS -- DISPLAYS WHICH TAKE PRIORITY OVER ALL OTHER DISPLAYS.  USUALLY THESE DISPLAYS ARE SENT
062791,000037:                                                                                                 #             OUT UNDER CRITICAL ALARM CONDITIONS.
062792,000038:                                                                                                 #         2.  EXTENDED VERB DISPLAYS -- ALL EXTENDED VERBS AND MARK ROUTINES SHOULD USE EXTENDED VERB (MARK) DISPLAYS.
062793,000039:                                                                                                 #         3.  NORMAL DISPLAYS -- ALL MISSION PROGRAM DISPLAYS WHICH INTERFACE WITH THE ASTRONAUT DURING THE NORMAL
062794,000040:                                                                                                 #             SEQUENCE OF EVENTS.
062795,000041:                                                                                                 #         4.  MISC. DISPLAYS -- ALL DISPLAYS NOT HANDLED BY THE DISPLAY INTERFACE ROUTINES.  THESE INCLUDE SUCH DISPLAYS AS
062796,000042:                                                                                                 #             MM DISPLAYS AND SPECIAL PURPOSE DISPLAYS HANDLED BY PINBALL.
062797,000043:                                                                                                 #         5.  ASTRONAUT INITIATED DISPLAYS -- ALL DISPLAYS INITIATED EXTERNALLY.
062798,000044: 
062799,000045:                                                                                                 #  THE FOLLOWING TERMS ARE USED TO DESCRIBE THE STATUS OF DISPLAYS --
062800,000046:                                                                                                 #         1.  ACTIVE -- THE DISPLAY WHICH IS (1) BEING DISPLAYED TO THE ASTRONAUT AND WAITING FOR A RESPONSE OR
062801,000047:                                                                                                 #             (2) WAITING FIRST IN LINE FOR THE ASTRONAUT TO FINISH USING THE DSKY OR (3) BEING DISPLAYED ON THE DSKY
062802,000048:                                                                                                 #             BUT NOT WAITING FOR A RESPONSE.
062803,000049:                                                                                                 #         2.  INACTIVE -- A DISPLAY WHICH HAS (1) BEEN ACTIVE BUT WAS INTERRUPTED BY A DISPLAY OF HIGHER PRIORITY,
062804,000050:                                                                                                 #             (2) BEEN PUT INTO THE WAITING LIST AT TIME IT WAS REQUESTED DUE TO THE FACT A HIGHER PRIORITY DISPLAY
062805,000051:                                                                                                 #             WAS ALREADY DOING, (3) BEEN INTERRUPTED BY THE ASTRONAUT (CALLED A PINBRANCH CONDITION, SINCE THIS TYPE
062806,000052:                                                                                                 #             OF INACTIVE DISPLAY IS USUALLY REACTIVATED ONLY BY PINBALL) OR (4) A DISPLAY WHICH HAS FINISHED BUT STILL
062807,000053:                                                                                                 #             HAS INFO SAVED FOR RESTART PURPOSES.
062808,000054: 
062809,000055:                                                                                                 #  DISPLAY PRIORITIES WORK AS FOLLOWS --
062810,000056:                                                                                                 #         INTERRUPTS --
062811,000057:                                                                                                 #                 1.  THE ASTRONAUT CAN INTERRUPT ANY DISPLAY WITH AN EXTERNAL DISPLAY REQUEST.
062812,000058:                                                                                                 #                 2.  INTERNAL DISPLAYS CAN NOT BE SENT OUT WHEN THE ASTRONAUT IS USING THE DSKY.
062813,000059:                                                                                                 #                 3.  PRIORITY DISPLAYS INTERRUPT ALL OTHER TYPES OF INTERNAL DISPLAYS.  A PRIORITY DISPLAY INTERRUPTING ANOTHER
062814,000060:                                                                                                 #                     PRIORITY DISPLAY WILL CAUSE AN ABORT UNLESS BIT14 IS SET FOR THE LINUS ROUTINE.
062815,000061:                                                                                                 #                 4.  A MARK DISPLAY INTERRUPTS ANY NORMAL DISPLAY.
062816,000062:                                                                                                 #                 5.  A MARK THAT INTERRUPTS A MARK COMPLETELY REPLACES IT.

062821,000067:                                                                                                 #                 3.  INTERRUPTED MARK

062828,000074:                                                                                                 #  THE DISPLAY ROUTINES ARE INTENDED TO SERVE AS AN INTERFACE BETWEEN THE USER AND PINBALL.  THE
062829,000075:                                                                                                 #  FOLLOWING STATEMENTS CAN BE MADE ABOUT NORMAL DISPLAYS AND PRIORITY DISPLAYS (A DESCRIPTION OF MARK ROUTINES
062830,000076:                                                                                                 #  WILL FOLLOW LATER):
062831,000077:                                                                                                 #         1.  ALL ROUTINES THAT END IN R HAVE AN IMMEDIATE RETURN TO THE USER.  FOR ALL FLASHING DISPLAYS THIS RETURN
062832,000078:                                                                                                 #             IS TO THE USER'S CALL CADR +4.  FOR THE ONLY NON-FLASHING IMMEDIATE RETURN DISPLAY (GODSPR) THIS RETURN
062833,000079:                                                                                                 #             IS TO THE USER'S CALLING LOC +1.

062836,000082:                                                                                                 #         4.  ALL ROUTIENS NOT ENDING IN R BRANCH DIRECTLY TO MAKEPLAY WHICH MAKES THESE DISPLAYS A PART OF THE
062837,000083:                                                                                                 #             USER'S JOB.
062838,000084:                                                                                                 #         5.  ALL DISPLAY ROUTIENS ARE CALLED VIA BANKCALL.
062839,000085:                                                                                                 #         6.  TO RESTART A DISPLAY THE USER WILL GENERALLY USE A PHASE OF ONE WITH DESIRED RESTART GROUP (SEE
062840,000086:                                                                                                 #             DESCRIPTION OF RESTARTS).

062842,000088:                                                                                                 #             TO THE USER'S CALL CADR +1.  A PROCEED (V33) BRANCHES TO THE USER'S CALL CADR +2.  AN ENTER OR RECYCLE
