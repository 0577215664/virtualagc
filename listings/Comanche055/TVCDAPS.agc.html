Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Colossus 249 (Apollo 9) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 055 OF AGC PROGRAM COMANCHE BY NASA 2021113-051
	10:28 APR.  1,1969
Note that the date is the date of the printout, not the date of the program revision.

043000,000043:                                                                                                  #  LOG SECTION...TVCDAPS                 SUBROUTINE...DAPCSM
043001,000044:                                                                                                  #  MODIFIED BY SCHLUNDT                          21 OCTOBER 1968
043002,000045: 
043003,000046:                                                                                                  #  FUNCTIONAL DESCRIPTION....
043004,000047: 
043005,000048:                                                                                                  #        SELF-PERPETUATING T5 TASKS WHICH GENERATE THE COMMAND SIGNALS
043006,000049:                                                                                                  #        FOR THE PITCH AND YAW SPS GIMBAL ACTUATORS DURING TVC (SPS) BURNS,
043007,000050:                                                                                                  #        IN RESPONSE TO BODY-AXIS RATE COMMANDS FROM CROSS-PRODUCT STEERING
043008,000051:                                                                                                  #        (S40.8).  IF NO STEERING (IMPULSIVE BURNS) MAINTAINS ATTITUDE-HOLD
043009,000052:                                                                                                  #        ABOUT THE REFERENCE (INITIAL) DIRECTIONS (ZERO RATE COMMANDS).
043010,000053: 
043011,000054:                                                                                                  #        THE PITCH AND YAW LOOPS ARE SEPARATE, BUT STRUCTURED IDENTICALLY.
043012,000055:                                                                                                  #        EACH ATTITUDE-RATE LOOP INCLUDES GIMBAL ANGLE RATE DERIVATION,
043013,000056:                                                                                                  #        GIMBAL/BODY AXIS TRANSFORMATION, BODY-AXIS ATTITUDE ERROR
043014,000057:                                                                                                  #        INTEGRATION WITH ERROR LIMITING, THE GENERALIZED 6TH-ORDER FILTER
043015,000058:                                                                                                  #        FOR CSM OR CSM/LM OPERATION, A FILTER OUTPUT LIMITER,
043016,000059:                                                                                                  #        CG-OFFSET TRACKER FILTER, AND THE CG-TRACKER MINOR LOOP.
043017,000060: 
043018,000061:                                                                                                  #        THE DAPS ARE CYCLIC, CALLING EACH OTHER AT 1/2 THE DAP SAMPLE
043019,000062:                                                                                                  #        TIME, AS DETERMINED BY T5TVCDT.  THE ACTUATOR COMMANDS ARE
043020,000063:                                                                                                  #        REGENERATED AS ANALOG VOLTAGES BY THE OPTICS ERROR COUNTERS, WHICH
043021,000064:                                                                                                  #        TRANSMIT THE SIGNAL TO THE ACTUATOR SERVOS WHEN THERE IS PROPER CDU
043022,000065:                                                                                                  #        MODING.
043023,000066: 
043024,000067:                                                                                                  #  CALLING SEQUENCE.... (TYPICALLY)
043025,000068: 
043026,000069:                                                                                                  #        T5 CALL OF TVCDAPON (TVCINITIALIZE) BY DOTVCON (P40)
043027,000070:                                                                                                  #        T5 CALL OF DAPINIT (TVCDAPS) BY TVCINIT4 (TVCINITIALIZE)
043028,000071:                                                                                                  #        T5 CALL OF PITCHDAP BY DAPINIT
043029,000072:                                                                                                  #        T5 CALL OF YAWDAP BY PITCHDAP
043030,000073:                                                                                                  #        T5 CALL OF PITCHDAP BY YAWDAP
043031,000074:                                                                                                  #                ETC.
043032,000075:                                                                                                  #        (AUTOMATIC SEQUENCING FROM TVCDAPON)
043033,000076: 
043034,000077:                                                                                                  #  NORMAL EXIT MODE....RESUME
043035,000078: 
043036,000079:                                                                                                  #  ALARM OR ABORT EXIT MODES....NONE
043037,000080: 
043038,000081:                                                                                                  #  SUBROUTINES CALLED....
043039,000082: 
043040,000083:                                                                                                  #        HACK FOR STROKE TEST (V68) WAVEFORM GENERATION
043041,000084:                                                                                                  #        PCOPY, YCOPY FOR COPY-CYCLES (USED ALSO BY TVC RESTART PACKAGE)
043042,000085:                                                                                                  #        DAPINIT FOR INITIAL CDUS FOR RATE MEASUREMENTS
043043,000086:                                                                                                  #        ERRORLIM, ACTLIM FOR INPUT (ATTITUDE-ERROR INTEGRATION) AND
043044,000087:                                                                                                  #                OUTPUT (ACTUATOR COMMAND) LIMITING, COMMON TO PITCH AND
043045,000088:                                                                                                  #                YAW DAPS
043046,000089:                                                                                                  #        FWDFLTR (INCLUDING OPTVARK) AND PRECOMP, TO COMPUTE FILTER
043047,000090:                                                                                                  #                OUTPUTS AND STORAGE VALUES
043048,000091:                                                                                                  #        RESUME
