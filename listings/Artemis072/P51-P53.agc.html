Photography was by Ron Burkey, and transcription was performed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images may be available. 	GAP: ASSEMBLE REVISION 072 OF AGC PROGRAM ARTEMIS

032795,000027:                                                                                                  #  ALIGNS THE IMU TO ONE OF THREE ORIENTATIONS SELECTED BY THE ASTRONAUT. THE PRESENT IMU ORIENTATION IS KNOWN

032801,000033:                                                                                                  #        STORED BY A PREVIOUSLY SELECTED PROGRAM.
032802,000034: 
032803,000035:                                                                                                  #        (B) NOMINAL (LOCAL VERTICAL) ORIENTATION
032804,000036: 
032805,000037:                                                                                                  #                X   = UNIT ( Y   X Z   )
032806,000038:                                                                                                  #                -SM          -SM   -SM

032809,000041:                                                                                                  #                -SM         -   -
032810,000042:

032815,000047:

032817,000049:                                                                                                  #                -

032819,000051:                                                                                                  #                V = THE INERTIAL VELOCITY VECTOR AT TIME T(ALIGN) SELECTED BY THE ASTRONAUT
032820,000052:                                                                                                  #                -
032821,000053: 
032822,000054:                                                                                                  #        (C) REFSMMAT ORIENTATION
032823,000055: 
032824,000056:                                                                                                  #        THIS ORIENTATION IS SELECTED AUTOMATICALLY UNLESS THE ASTRONAUT KEYS IN A DIFFERENT OPTION CODE
032825,000057: 
032826,000058:                                                                                                  #        THIS SELECTION CORRECTS THE PRESENT IMU ORIENTATION. THE PRESENT ORIENTATION DIFFERS FROM THAT TO WHICH IT
032827,000059:                                                                                                  #        WAS LAST ALIGNED ONLY DUE TO GYRO DRIFT (I.E. NEITHER GIMBAL LOCK NOR IMU POWER INTERRUPT HAS OCCURRED
032828,000060:                                                                                                  #        SINCE THE LAST ALIGNMENT).
032829,000061: 
032830,000062:                                                                                                  #  AFTER A IMU ORIENTATION HAS BEEN SELECTED ROUTINE S52.2 IS OPERATED TO COMPUTE THE GIMBAL ANGLES USING THE
032831,000063:                                                                                                  #  NEW ORIENTATION AND THE PRESENT VEHICLE ATTITUDE.  CAL52A THEN USES THESE ANGLES, STORED IN THETAD,+1,+2, TO
032832,000064:                                                                                                  #  COARSE ALIGN THE IMU. THE STAR SELECTION ROUTINE, R56, IS THEN OPERATED. IF 2 STARS ARE NOT AVAILABLE AN ALARM
032833,000065:                                                                                                  #  IS FLASHED TO NOTIFY THE ASTRONAUT. AT THIS POINT THE ASTRONAUT WILL MANEUVER THE VEHICLE AND SELECT 2 STARS
032834,000066:                                                                                                  #  EITHER MANUALLY OR AUTOMATICALLY. AFTER 2 STARS HAVE BEEN SELECTED THE IMU IS FINE ALIGNED USING ROUTINE R51. IF
032835,000067:                                                                                                  #  THE RENDEZVOUS NAVIGATION PROCESS IS OPERATING (INDICATED BY RNDVZFLG) P20 IS DISPLAYED. OTHERWISE P00 IS
032836,000068:                                                                                                  #  REQUESTED.

032838,000070:                                                                                                  #  CALLING SEQUENCE -
032839,000071: 
032840,000072:                                                                                                  #        THE PROGRAM IS CALLED BY THE ASTRONAUT BY DSKY ENTRY.
