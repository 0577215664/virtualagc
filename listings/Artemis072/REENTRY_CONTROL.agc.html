Photography was by Ron Burkey, and transcription was performed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images may be available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 072 OF AGC PROGRAM ARTEMIS 
	BY NASA 2021114-011 11:40 FEB. 26, 1971
Note that the date is the date of the printout, not the date of the program revision.

037642,000156:                                                                                                  #  GOTOADDR CONTAINS THE ADDRESS OF THE ROLL COMMAND EQUATIONS APPROPRIATE TO THE CURRENT PHASE OF
037643,000157:                                                                                                  #  RE-ENTRY.  SEQUENCING IS AS FOLLOWS:
037644,000158: 
037645,000159:                                                                                                  #  INITROLL      ADDRESS IS SET HERE INITIALLY. HOLDS INITIAL ROLL ATTITUDE UNTIL KAT  IS EXCEEDED. THEN HOLDS NEW ROLL
037646,000160:                                                                                                  #                ATTITUDE UNTIL  VRTHRESH  IS EXCEEDED.  THEN BRANCHES TO
037647,000161: 
037648,000162:                                                                                                  #  HUNTEST       THIS SECTION CHECKS TO SEE IF THE PREDICTED RANGE AT NOMINAL   L/D FROM PRESENT CONDITIONS IS LESS
037649,000163:                                                                                                  #                THAN THE DESIRED RANGE.
037650,000164:                                                                                                  #                  IF NOT - A ROLL COMMAND IS GENERATED BY THE CONSTANT DRAG CONTROLLER.
037651,000165:                                                                                                  #                  IF SO - CONTROL AND GOTOADDR ARE SET TO  UPCONTRL.
037652,000166:                                                                                                  #                USUALLY NO ITERATION IS INVOLVED EXCEPT IF THE RANGE DESIRED IS TOO LONG ON THE FIRST PASS THROUGH
037653,000167:                                                                                                  #                HUNTEST.
037654,000168: 
037655,000169:                                                                                                  #  UPCONTRL      CONTROLS ROLL DURING THE SUPER-CIRCULAR PHASE.  UPCONTRL IS TERMINATED EITHER
037656,000170:                                                                                                  #                  (A)  WHEN THE DRAG (AS MEASURED BY THE PIPAS) FALLS BELOW Q7, OR
037657,000171:                                                                                                  #                  (B)  IF RDOT IS NEGATIVE AND REFERENCE VL EXCEEDS V.
037658,000172:                                                                                                  #                IN CASE (A),  GOTOADDR  IS SET TO  KEP2  AND IN CASE (B), TO  PREDICT3  SKIPPING THE KEPLER PHASE OF
037659,000173:                                                                                                  #                ENTRY.
037660,000174: 
037661,000175:                                                                                                  #  KEP2          GOTOADDR IS SET HERE DURING THE KEPLER PHASE TO MONITOR DRAG. THE SPACECRAFT IS INSTANTANEOUSLY
037662,000176:                                                                                                  #                TRIMMED IN PITCH AND YAW TO THE COMPUTED RELATIVE VELOCITY. THE LAST COMPUTED ROLL ANGLE IS MAINTAINED.
037663,000177:                                                                                                  #                WHEN THE MEASURED DRAG EXCEEDS Q7 +0.5,  GOTOADDR  IS SET TO
037664,000178: 
037665,000179:                                                                                                  #  PREDICT3      THIS CONTROLS THE FINAL SUB-ORBITAL PHASE. ROLL COMMANDS CEASE
037666,000180:                                                                                                  #                WHEN  V  IS LESS THAN VQUIT. AN EXIT IS MADE TO
037667,000181: 
037668,000182:                                                                                                  #  P67.1         THE LAST COMPUTED ROLL ANGLE IS MAINTAINED. RATE DAMPING IS DONE IN PITCH AND YAW. PRESENT LATITUDE
037669,000183:                                                                                                  #                AND LONGITUDE ARE COMPUTED FOR DISPLAY.
037670,000184:                                                                                                  #                ENTRY IS TERMINATED WHEN DISKY RESPONSE IS MADE TO THIS FINAL FLASHING DISPLAY.
