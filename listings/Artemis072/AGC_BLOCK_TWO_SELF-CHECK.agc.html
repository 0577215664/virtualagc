059457,000022:                                                                                                 #  FUNCTIONAL DESCRIPTION
059458,000023: 
059459,000024:                                                                                                 #  PROGRAM HAS TWO MAIN PARTS. THE FIRST IS SELF-CHECK WHICH RUNS AS A ZERO PRIORITY JOB WITH NO CORE SET, AS
059460,000025:                                                                                                 #  PART OF THE BACK-UP IDLE LOOP. THE SECOND IS SHOW-BANKSUM WHICH RUNS AS A REGULAR EXECUTIVE JOB WITH ITS OWN
059461,000026:                                                                                                 #  STARTING VERB.
059462,000027:                                                                                                 #  THE PURPOSE OF SELF-CHECK IS TO CHECK OUT VARIOUS PARTS OF THE COMPUTER AS OUTLINED BELOW IN THE OPTIONS.
059463,000028:                                                                                                 #  THE PURPOSE OF SHOW-BANKSUM IS TO DISPLAY THE SUM OF EACH BANK, ONE AT A TIME.
059464,000029:                                                                                                 #  IN ALL THERE ARE  7 POSSIBLE OPTIONS IN THIS BLOCK II VERSION OF SELF-CHECK. MORE DETAIL DESCRIPTION MAY BE
059465,000030:                                                                                                 #  FOUND IN E-2065 BLOCK II AGC SELF-CHECK AND SHOW BANKSUM BY EDWIN D. SMALLY DECEMBER 1966, AND ADDENDA 2 AND 3.
059466,000031:                                                                                                 #  THE DIFFERENT OPTIONS ARE CONTROLLED BY PUTTING DIFFERENT NUMBERS IN THE SMODE REGISTER (NOUN 27). BELOW IS
059467,000032:                                                                                                 #  A DESCRIPTION OF WHAT PARTS OF THE COMPUTER THAT ARE CHECKED BY THE OPTIONS, AND THE CORRESPONDING NUMBER, IN
059468,000033:                                                                                                 #  OCTAL, TO LOAD INTO SMODE.
059469,000034:                                                                                                 #         +-4             ERASABLE MEMORY
059470,000035:                                                                                                 #         +-5             FIXED MEMORY
059471,000036:                                                                                                 #         +-1,2,3,6,7,10  EVERYTHING IN OPTIONS 4 AND 5.
059472,000037:                                                                                                 #         -0              SAME AS +-10 UNTIL AN ERROR IS DETECTED.
059473,000038:                                                                                                 #         +0              NO CHECK, PUTS COMPUTER INTO THE BACKUP IDLE LOOP.
059474,000039: 
059475,000040:                                                                                                 #  WARNINGS
059476,000041: 
059477,000042:                                                                                                 #         USE OF E MEMORY RESERVED FOR SELF-CHECK (EVEN IN IDLE LOOP) AS TEMP STORAGE BY OTHER PROGRAMS IS DANGEROUS.
059478,000043:                                                                                                 #         SMODE SET GREATER THAN OCT 10 PUTS COMPUTER INTO BACKUP IDLE LOOP.
059479,000044: 
059480,000045:                                                                                                 #  CALLING SEQUENCE
059481,000046: 
059482,000047:                                                                                                 #         TO CALL SELF-CHECK KEY IN
059483,000048:                                                                                                 #                 V 21 N 27 E     OPTION NUMBER E
059484,000049:                                                                                                 #         TO CALL SHOW-BANKSUM KEY IN
059485,000050:                                                                                                 #                 V 91 E          DISPLAYS FIRST BANK
059486,000051:                                                                                                 #                 V 33 E          PROCEED, DISPLAYS NEXT BANK
059487,000052: 
059488,000053: 
059489,000054:                                                                                                 #  EXIT MODES, NORMAL AND ALARM
059490,000055: 
059491,000056:                                                                                                 #  SELF-CHECK NORMALLY CONTINUES INDEFINITELY UNLESS THERE IS AN ERROR DETECTED. IF SO + OPTION NUMBERS PUT
059492,000057:                                                                                                 #  COMPUTER INTO BACKUP IDLE LOOP, - OPTION NUMBERS RESTART THE OPTION.
059493,000058:                                                                                                 #  THE -0 OPTION PROCEEDS FROM THE LINE FOLLOWING THE LINE WHERE THE ERROR WAS DETECTED.
059494,000059:                                                                                                 #  SHOW-BANKSUM PROCEEDS UNTIL A TERMINATE IS KEYED IN (V 34 E). THE COMPUTER IS PUT INTO THE BACKUP IDLE LOOP.
059495,000060: 
059496,000061: 
059497,000062:                                                                                                 #  OUTPUT

059499,000064:                                                                                                 #  SELF-CHECK UPON DETECTING AN ERROR LOADS THE SELF-CHECK ALARM CONSTANT (01102) INTO THE FAILREG SET AND
059500,000065:                                                                                                 #  TURNS ON THE ALARM LIGHT. THE OPERATOR MAY THEN DISPLAY THE THREE FAILREGS BY KEYING IN V 05 N 09 E. FOR FURTHER
059501,000066:                                                                                                 #  INFORMATION HE MAY KEY IN V 05 N 08 E, THE DSKY DISPLAY IN R1 WILL BE ADDRESS+1 OF WHERE THE ERROR WAS DETECTED,
059502,000067:                                                                                                 #  IN R2 THE BBCON OF SELF-CHECK, AND IN R3 THE TOTAL NUMBER OF ERRORS DETECTED BY SELF-CHECK SINCE THE LAST MAN

059504,000069:                                                                                                 #  SHOW-BANKSUM STARTING WTIH BANK 0 DISPLAYS IN R1 THE BANK SUM (A +-NUMBER EQUAL TO THE BANK NUMBER), IN R2
059505,000070:                                                                                                 #  THE BANK NUMBER, AND IN R3 THE BUGGER WORD.
