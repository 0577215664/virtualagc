These source-code files were obtained by digitally photographing an Artemis 72 program listing from a private collection and then modifying pre-existing Comanche 55 (Apollo 11) source files to incorporate changes. Photography was by Ron Burkey, and transcription was performed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images may be available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 072 OF AGC PROGRAM ARTEMIS 
	BY NASA 2021114-011 11:40 FEB. 26, 1971
Note that the date is the date of the printout, not the date of the program revision.

013927,000026:                                                                                                  #  PROGRAM NAME - KEYBOARD AND DISPLAY PROGRAM
013928,000027:                                                                                                  #  MOD BY - BARNERT      DATE - 20 AUG 1969
013929,000028:                                                                                                  #  LOG SECTION - PINBALL GAME BUTTONS AND LIGHTS
013930,000029: 
013931,000030:                                                                                                  #  FUNCTIONAL DESCRIPTION-
013932,000031: 
013933,000032:                                                                                                  #        THE KEYBOARD AND DISPLAY SYSTEM PROGRAM OPERATES UNDER EXECUTIVE
013934,000033:                                                                                                  #        CONTROL AND PROCESSES INFORMATION EXCHANGED BETWEEN THE AGC AND THE
013935,000034:                                                                                                  #        COMPUTER OPERATOR.  THE INPUTS TO THE PROGRAM ARE FROM THE KEYBOARD,
013936,000035:                                                                                                  #        FROM INTERNAL PROGRAM, AND FROM THE UPLINK.
013937,000036: 
013938,000037:                                                                                                  #        THE LANGUAGE OF COMMUNICATION WITH THE PROGRAM IS A PAIR OF WORDS
013939,000038:                                                                                                  #        KNOWN AS VERB AND NOUN.  EACH OF THESE IS REPRESENTED BY A 2 CHARACTER
013940,000039:                                                                                                  #        DECIMAL NUMBER.  THE VERB CODE INDICATES WHAT ACTION IS TO BE TAKEN, THE
013941,000040:                                                                                                  #        NOUN CODE INDICATES TO WHAT THIS ACTION IS APPLIED.  NOUNS USUALLY
013942,000041:                                                                                                  #        REFER TO A GROUP OF ERASABLE REGISTERS.
013943,000042: 
013944,000043:                                                                                                  #        VERBS ARE GROUPED INTO DISPLAYS, LOADS, MONITORS (DISPLAYS THAT ARE
013945,000044:                                                                                                  #        UPDATED ONCE PER SECOND), SPECIAL FUNCTIONS, AND EXTENDED VERBS (THESE
013946,000045:                                                                                                  #        ARE OUTSIDE OF THE DOMAIN OF PINBALL AND CAN BE FOUND UNDER LOG SECTION
013947,000046:                                                                                                  #        'EXTENDED VERBS').
013948,000047: 
013949,000048:                                                                                                  #        A LIST OF VERBS AND NOUNS IS GIVEN IN LOG SECTION 'ASSEMBLY AND
013950,000049:                                                                                                  #        OPERATION INFORMATION'.
013951,000050: 
013952,000051:                                                                                                  #  CALLING SEQUENCES-
013953,000052: 
013954,000053:                                                                                                  #        KEYBOARD:
013955,000054:                                                                                                  #        EACH DEPRESSION OF A MAIN (NAVIGATION) KEYBOARD BUTTON ACTIVATES
013956,000055:                                                                                                  #        INTERRUPT KEYRUPT1 (KEYRUPT2) AND PLACES THE 5 BIT KEY CODE INTO
013957,000056:                                                                                                  #        CHANNEL 15 (CHANNEL 16).  KEYRUPT1 (KEYRUPT2) PLACES THE KEY
013958,000057:                                                                                                  #        CODE INTO MPAC, ENTERS AN EXECUTIVE REQUEST FOR THE KEYBOARD AND DISPLAY
013959,000058:                                                                                                  #        PROGRAM (AT 'CHARIN'), AND EXECUTES A RESUME.
013960,000059: 
013961,000060:                                                                                                  #        UPLINK:
013962,000061:                                                                                                  #        EACH WORD RECEIVED BY THE UPLINK ACTIVATES INTERRUPT UPRUPT, WHICH
013963,000062:                                                                                                  #        PLACES THE 5 BIT KEY CODE INTO MPAC, ENTERS AN EXECUTIVE REQUEST FOR THE
013964,000063:                                                                                                  #        KEYBOARD AND DISPLAY PROGRAM (AT 'CHARIN') AND EXECUTES A RESUME.
013965,000064: 
013966,000065:                                                                                                  #        INTERNAL PROGRAMS:
013967,000066:                                                                                                  #        INTERNAL PROGRAMS CALL PINBALL AT 'NVSUB' WITH THE DESIRED VERB/NOUN
013968,000067:                                                                                                  #        CODE IN A (LOW 7 BITS FOR NOUN, NEXT 7 BITS FOR VERB).  DETAILS
013969,000068:                                                                                                  #        DESCRIBED ON REMARKS CARDS JUST BEFORE 'NVSUB' AND 'NVSBWAIT' (SEE
013970,000069:                                                                                                  #        SYMBOL TABLE FOR PAGE NUMBERS).
013971,000070: 
013972,000071:                                                                                                  #  NORMAL EXIT MODES-
013973,000072: 
013974,000073:                                                                                                  #        IF PINBALL WAS CALLED BY EXTERNAL ACTION, THERE ARE FOUR EXITS:
013975,000074:                                                                                                  #                1) ALL BUT (2), (3), AND (4) EXIT DIRECTLY TO ENDOFJOB.
013976,000075:                                                                                                  #                2) EXTENDED VERBS GO TO THE EXTENDED VERB FAN AS PART OF THE

Page 319 

013978,000077:                                                                                                  #                   PINBALL EXECUTIVE JOB WITH PRIORITY 30000.  IT IS THE
013979,000078:                                                                                                  #                   RESPONSIBILITY OF THE EXTENDED VERB CALLED TO EVENTUALLY
013980,000079:                                                                                                  #                   CHANGE PRIORITY (IF NECESSARY) AD DO AN ENDOFJOB.
013981,000080:                                                                                                  #                   ALSO PINBALL IS A NOVAC JOB. EBANK SET FOR COMMON.
013982,000081:                                                                                                  #                3) VERB 37. CHANGE OF PROGRAM (MAJOR MODE) CALLS 'V37' IN THE
013983,000082:                                                                                                  #                   SERVICE ROUTINES AS PART OF THE PINBALL EXEC JOB WITH PRIO
013984,000083:                                                                                                  #                   30000.  THE NEW PROGRAM CODE (MAJOR MODE) IS LEFT IN A.
013985,000084:                                                                                                  #                4) KEY RELEASE BUTTON CALLS 'PINBRNCH' IN THE DISPLAY INTERFACE
013986,000085:                                                                                                  #                   ROUTINES AS PART OF THE PINBALL EXEC JOB WITH PRIO 30000 IF
013987,000086:                                                                                                  #                   THE KEY RELEASE LIGHT IS OFF AND 'CADRSTOR' IS NOT +0.
013988,000087: 
013989,000088:                                                                                                  #        IF PINBALL WAS CALLED BY INTERNAL PROGRAMS, EXIT FROM PINBALL IS BACK
013990,000089:                                                                                                  #        TO CALLING ROUTINE.  DETAILS DESCRIBED IN REMARKS CARDS JUST BEFORE
013991,000090:                                                                                                  #        'NVSUB'.
013992,000091: 
013993,000092:                                                                                                  #  ALARM OR ABORT EXIT MODES-
013994,000093: 
013995,000094:                                                                                                  #        EXTERNAL INITIATION:
013996,000095:                                                                                                  #        IF SOME IMPROPER SEQUENCE OF KEY CODES IS DETECTED, THE OPERATOR
013997,000096:                                                                                                  #        ERROR LIGHT IS TURNED ON AND EXIT IS TO 'ENDOFJOB'.
013998,000097: 
013999,000098:                                                                                                  #        INTERNAL PROGRAM INITIATION:
014000,000099:                                                                                                  #        IF AN ILLEGAL V/N COMBINATION IS ATTEMPTED, AN ABORT IS CAUSED
014001,000100:                                                                                                  #        (WITH OCTAL 21501).
014002,000101:                                                                                                  #        IF A SECOND ATTEMPT IS MADE TO GO TO SLEEP IN PINBALL, AN ABORT IS
014003,000102:                                                                                                  #        CAUSED (WITH OCTAL 21206). THERE ARE TWO WAYS TO GO TO SLEEP IN PINBALL:
014004,000103:                                                                                                  #                1) ENDIDLE
014005,000104:                                                                                                  #                2) NVSUBUSY
014006,000105: 
014007,000106:                                                                                                  #  OUTPUT-
014008,000107: 
014009,000108:                                                                                                  #  INFORMATION TO BE SENT TO THE DISPLAY PANEL IS LEFT IN THE 'DSPTAB'
014010,000109:                                                                                                  #  BUFFERS REGISTERS (UNDER EXEC CONTROL).  'DSPOUT' (A PART OF T4RUPT)
014011,000110:                                                                                                  #  HANDLES THE PLACING OF THE 'DSPTAB' INFORMATION INTO OUTPUT CHANNEL 10
014012,000111:                                                                                                  #  IN INTERRUPT.
014013,000112: 
014014,000113:                                                                                                  #  ERASABLE INITIALIZATION-
014015,000114: 
014016,000115:                                                                                                  #  FRESH START AND RESTART INITIALIZE THE NECESSARY E REGISTERS FOR
014017,000116:                                                                                                  #  PINBALL IN 'STARTSUB'.  REGISTERS ARE:  DSPTAB BUFFER, CADRSTOR,
014018,000117:                                                                                                  #  REQRET, CLPASS, DSPLOCK, MONSAVE, MONSAVE1, VERBREG, NOUNREG, DSPLIST,
014019,000118:                                                                                                  #  DSPCOUNT, NOUT.
