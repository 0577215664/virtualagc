Source Code

These source-code files were obtained by digitally photographing an Artemis 72 program listing from a private collection and then modifying pre-existing Comanche 55 (Apollo 11) source files to incorporate changes. Photography was by Ron Burkey, and transcription was performed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images may be available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 072 OF AGC PROGRAM ARTEMIS 
	BY NASA 2021114-011 11:40 FEB. 26, 1971
Note that the date is the date of the printout, not the date of the program revision.

047613,000018:                                                                                                  #  PROGRAM NAME- DOWN TELEMETRY PROGRAM
047614,000019:                                                                                                  #  MOD NO.- 0      TO COMPLETELY REWRITE THE DOWN TELEMETRY PROGRAM AND DOWNLINK ERASABLE DUMP PROGRAM FOR THE
047615,000020:                                                                                                  #                  PURPOSE OF SAVING APPROXIMATELY 150 WORDS OF CORE STORAGE.
047616,000021:                                                                                                  #                  THIS CHANGE REQUIRES AN ENTIRELY NEW METHOD OF SPECIFYING DOWNLINK LISTS. REFER TO DOWNLINK
047617,000022:                                                                                                  #                  LISTS LOG SECTION FOR MORE DETAILS. HOWEVER THIS CHANGE WILL NOT AFFECT THE GROUND PROCESSING
047618,000023:                                                                                                  #                  OF DOWN TELEMETRY DATA.
047619,000024:                                                                                                  #  MOD BY-  KILROY, SMITH, DEWITT
047620,000025:                                                                                                  #  MOD BY - DENSMORE - JUNE 1969 - ELIMINATE ERASABLE DUMP COUNT
047621,000026:                                                                                                  #  DATE-    02 OCT 67
047622,000027:                                                                                                  #  AUTHORS- KILROY, SMITH, DWWITT, DEWOLF, FAGIN
047623,000028:                                                                                                  #  LOG SECTION- DOWN-TELEMETRY PROGRAM
047624,000029:                                                                                                  #  FUNCTIONAL DESCRIPTION- THIS ROUTINE IS INITIATED BY TELEMETRY END
047625,000030:                                                                                                  #           PULSE FROM THE DOWNLINK TELEMETRY CONVERTER. THIS PULSE OCCURS
047626,000031:                                                                                                  #           AT 50 TIMES PER SEC (EVERY 20 MS) THEREFORE DODOWNTM IS
047627,000032:                                                                                                  #           EXECUTED AT THESE RATES. THIS ROUTINE SELECTS THE APPROPRIATE
047628,000033:                                                                                                  #           AGC DATA TO BE TRANSMITTED DOWNLINK AND LOADS IT INTO OUTPUT
047629,000034:                                                                                                  #           CHANNELS 34 AND 35. THE INFORMATION IS THEN GATED OUT FROM THE
047630,000035:                                                                                                  #           LGC IN SERIAL FASHION.
047631,000036:                                                                                                  #           THIS PROGRAM IS CODED FOR A 2 SECOND DOWNLIST. SINCE DOWNRUPTS
047632,000037:                                                                                                  #           OCCUR EVERY 20 MS AND 2 AGC COMPUTER WORDS CAN BE PLACED IN
047633,000038:                                                                                                  #           CHANNELS 34 AND 35 DURING EACH DOWNRUPT THE PROGRAM IS CAPABLE
047634,000039:                                                                                                  #           OF SENDING 200 AGC WORDS EVERY 2 SECONDS.
047635,000040:                                                                                                  #  CALLING SEQUENCE- NONE
047636,000041:                                                                                                  #           PROGRAM IS ENTERED VIA TCF DODOWNTM WHICH IS EXECUTED AS A
047637,000042:                                                                                                  #           RESULT OF A DOWNRUPT. CONTROL IS RETURNED VIA TCF RESUME WHICH
047638,000043:                                                                                                  #           IN EFFECT IS A RESUME.
047639,000044:                                                                                                  #  SUBROUTINES CALLED- NONE
047640,000045:                                                                                                  #  NORMAL EXIT MODE- TCF RESUME
047641,000046:                                                                                                  #  ALARM OR ABORT EXIT MODE- NONE
047642,000047:                                                                                                  #  RESTART PROTECTION:
047643,000048:                                                                                                  #           ON A FRESH START AND RESTART THE 'STARTSUB' SUBROUTINE WILL INITIALIZE THE DOWNLIST POINTER (ACTUALLY
047644,000049:                                                                                                  #           DNTMGOTO) TO THE BEGINNING OF THE CURRENT DOWNLIST (I.E. CURRENT CONTENTS OF DNLSTADR). THIS HAS THE
047645,000050:                                                                                                  #           EFFECT OF IGNORING THE REMAINDER OF THE DOWNLIST WHICH THE DOWN-TELEMETRY PROGRAM WAS WORKING ON WHEN
047646,000051:                                                                                                  #           THE RESTART (OR FRESH START) OCCURRED AND RESUME DOWN TELEMETRY FROM THE BEGINNING OF THE CURRENT
047647,000052:                                                                                                  #           DOWNLIST.
047648,000053:                                                                                                  #           ALSO OF INTEREST IS THE FACT THAT ON A RESTART THE AGC WILL ZERO DOWNLINK CHANNELS 13, 34 AND 35.
047649,000054:                                                                                                  #  DOWNLINK LIST SELECTION:
047650,000055:                                                                                                  #           THE APPROPRIATE DOWNLINK LISTS ARE SELECTED BY THE FOLLOWING:
047651,000056:                                                                                                  #           1. FRESH START
047652,000057:                                                                                                  #           2. V37EXXE  WHERE XX = THE MAJOR MODE BEING SELECTED.
047653,000058:                                                                                                  #           3. UPDATE PROGRAM (P27)
047654,000059:                                                                                                  #           4.   NON-V37 SELECTABLE TYPE PROGRAMS (E.G. AGS INITIALIZATION (SUNDANCE, LUMINARY) AND P61-P62
047655,000060:                                                                                                  #                TRANSITION (COLOSSUS) ETC.).
047656,000061:                                                                                                  #  DOWNLINK LIST RULES AND LIMITATIONS:
047657,000062:                                                                                                  #           READ SECTION(S) WHICH FOLLOW 'DEBRIS' WRITEUP.
047658,000063:                                                                                                  #  OUTPUT-  EVERY 2 SECONDS 100 DOUBLE PRECISION WORDS (I.E., 200 LGC
047659,000064:                                                                                                  #           COMPUTER WORDS) ARE TRANSMITTED VIA DOWNLINK.
047660,000065:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED- NONE
047661,000066:                                                                                                  #           'DNTMGOTO' AND 'DNLSTADR' ARE INITIALIZED BY THE FRESH START PROGRAM.
047662,000067:                                                                                                  #  DEBRIS (ERASABLE LOCATIONS DESTROYED BY THIS PROGRAM)-
047663,000068:                                                                                                  # # Page 1094
047664,000069:                                                                                                  #           LDATALST, DNTMBUFF TO DNTMBUFF +21D, TMINDEX, DNQ.
047665,000070:

047668,000073:                                                                                                  #  RECEIPT OF AN ENDPULSE FROM THE SPACECRAFT TELEMETRY PROGRAMMER.
047669,000074: 
047670,000075:                                                                                                  #  NOTES REGARDING DOWNLINK LISTS ASSOCIATED WITH THIS PROGRAM:
047671,000076:                                                                                                  #  1. DOWNLISTS. - DOWNLISTS MUST BE COMPILED IN THE SAME BANK AS THE
047672,000077:                                                                                                  #     DOWN-TELEMETRY PROGRAM. THIS IS DONE FOR EASE OF CODING, FASTER
047673,000078:                                                                                                  #     EXECUTION.
047674,000079:                                                                                                  #  2. EACH DOWNLINK LIST CONSISTES OF A CONTROL LIST AND A NUMBER OF
047675,000080:                                                                                                  #     SUBLISTS.
047676,000081:                                                                                                  #  3. A SUBLIST REFERS TO A SNAPSHOT OR DATA COMMON TO THE SAME OR OTHER
047677,000082:                                                                                                  #     DOWNLINK LISTS. ANY SUBLIST CONTAINING COMMON DATA NEEDS TO BE
047678,000083:                                                                                                  #     CODED ONLY ONCE FOR THE APPLICABLE DOWNLINK LISTS.
047679,000084:                                                                                                  #  4. SNAPSHOT SUBLISTS REFER SPECIFICALLY TO HOMOGENEOUS DATA WHICH MUST BE
047680,000085:                                                                                                  #     SAVED IN A BUFFER DURING ONE DOWNRUPT.
047681,000086:                                                                                                  #  5. THE 1DNADR FOR THE 1ST WORD OF SNAPSHOT DATA IS FOUND AT THE END
047682,000087:                                                                                                  #     OF EACH SNAPSHOT SUBLIST, SINCE THE PROGRAM CODING SENDS THIS DP WORD
047683,000088:                                                                                                  #     IMMEDIATELY AFTER STORING THE OTHERS IN THE SNAPSHOT BUFFER.
047684,000089:                                                                                                  #  6. ALL LISTS ARE COMBINATIONS OF CODED ERASABLE ADDRESS CONSTANTS
047685,000090:                                                                                                  #     CREATED FOR THE DOWNLIST PROGRAM.
047686,000091:                                                                                                  #     A. 1DNADR                  1-WORD DOWNLIST ADDRESS.
047687,000092:                                                                                                  #         SAME AS ECADR, BUT USED WHEN THE WORD ADDRESSED IS THE LEFT
047688,000093:                                                                                                  #         HALF OF A DOUBLE-PRECISION WORD FOR DOWN TELEMETRY.
047689,000094:                                                                                                  #     B. 2DNADR - 6DNADR         N-WORD DOWNLIST ADDRESS, N = 2 - 6.
047690,000095:                                                                                                  #         SAME AS 1DNADR, BUT WTIH THE 4 UNUSED BITS OF THE ECADR FORMAT
047691,000096:                                                                                                  #         FILLED IN WITH 0001-0101.  USED TO POINT TO A LIST OF N DOUBLE-
047692,000097:                                                                                                  #         PRECISION WORDS, STORED CONSECUTIVELY, FOR DOWN TELEMETRY.
047693,000098:                                                                                                  #     C. DNCHAN                  DOWNLIST CHANNEL ADDRESS.
047694,000099:                                                                                                  #         SAME AS 1DNADR, BUT WITH PREFIX BITS 0111.  USED TO POINT TO
047695,000100:                                                                                                  #         A PAIR OF CHANNELS FOR DOWN TELEMETRY.
047696,000101:                                                                                                  #     D. DNPTR                   DOWN-TELEMETRY SUBLIST POINTER.
047697,000102:                                                                                                  #         SAME AS CAF BUT TAGGED AS A CONSTANT. USED IN CONTROL LIST TO POINT TO A SUBLIST.
047698,000103:                                                                                                  #         CAUTION--- A DNPTR CANNOT BE USED IN A SUBLIST.
047699,000104:                                                                                                  #  7. THE WORD ORDER CODE IS SET TO ZERO AT THE BEGINNING OF EACH DOWNLIST (I.E. CONTROL LIST) AND WHEN
047700,000105:                                                                                                  #     A '1DNADR TIME2' IS DETECTED IN THE CONTROL LIST (ONLY).
047701,000106:                                                                                                  #  8. IN THE SNAPSHOT SUBLIST ONLY, THE DNADR'S CANNOT POINT TO THE FIRST WORD OF ANY EBANK.

047722,000127:

047724,000129:
