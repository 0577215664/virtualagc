Source Code

These source-code files were obtained by digitally photographing an Artemis 72 program listing from a private collection and then modifying pre-existing Comanche 55 (Apollo 11) source files to incorporate changes. Photography was by Ron Burkey, and transcription was performed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images may be available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 072 OF AGC PROGRAM ARTEMIS 
	BY NASA 2021114-011 11:40 FEB. 26, 1971
Note that the date is the date of the printout, not the date of the program revision.

010523,000020:                                                                                                  #  PROGRAM NAME     -     SXTMARK
010524,000021: 
010525,000022:                                                                                                  #  MOD. 1        8 AUG. 69       BY P.RYE
010526,000023: 
010527,000024:                                                                                                  #  FUNCTIONAL DESCRIPTION:
010528,000025: 
010529,000026:                                                                                                  #        SXTMARK IS CALLED BY P03 AND BY P22, P23, P51, and P52 VIA R53.
010530,000027:                                                                                                  #        THE REGISTER MARKINDX IS SET TO 5 IF THE CALLING PROGRAM IS P22, OR TO 1 FOR OTHER USERS.
010531,000028:                                                                                                  #           THIS REGISTER INDICATES THE NUMBER OF MARKS DESIRED.
010532,000029:                                                                                                  #        THE REGISTER EXTVBACT IS CHECKED (SUBROUTINE TESTMARK) TO INSURE THAT THE MARK DISPLAY SYSTEM
010533,000030:                                                                                                  #           IS FREE.  IF NOT, BAILOUT (31211) IS DONE.
010534,000031:                                                                                                  #        BIT 2 OF THE REGISTER EXTVBACT IS SET BY SXTMARK, RESERVING THE MARKING AND EXTENDED VERB SYSTEM.
010535,000032:                                                                                                  #        SXTMARK DISPLAYS A FLASHING VERB 51, CALLING FOR MARKS.  A PROCEED RESPONSE TO THIS DISPLAY
010536,000033:                                                                                                  #           WILL RELEASE THE MARKING SYSTEM (SUBROUTINES MKRELEAS, CLEARMARK) AND RETURN TO THE
010537,000034:                                                                                                  #           CALLING PROGRAM.
010538,000035: 
010539,000036:                                                                                                  #        WHEN SUFFICIENT MARKS HAVE BEEN MADE (MARKINDX = 0) A FLASHING V50N25, R1 = 16, IS DISPLAYED.
010540,000037:                                                                                                  #           A PROCEED RESPONSE TO THIS DISPLAY WILL RETURN TO THE CALLING PROGRAM AFTER RELEASING THE
010541,000038:                                                                                                  #           MARKING SYSTEM.
010542,000039: 
010543,000040:                                                                                                  #        MARKS ARE PROCESSED BY THE ROUTINE MARKRUPT AS FOLLOWS:
010544,000041:                                                                                                  #                IF NO MARKS ARE CALLED FOR, ALARM CODE 114 IS SENT AND THE MARKRUPT ROUTINE EXITS.
010545,000042:                                                                                                  #                IF A MARK IS ACCEPTED, MARKFLG IS SET TO ENABLE A REJECT.
010546,000043:                                                                                                  #                IF R21(P20) IS RUNNING, DATA IS MOVED FROM STORAGE MRKBUF1 INTO MRKBUF2.  NEW MARK DATA
010547,000044:                                                                                                  #                        IS STORED INTO MRKBUF1.
010548,000045:                                                                                                  #                IF P22 IS RUNNING, MARK DATA IS STORED INTO SVMRKDAT, USING THE REGISTER P22DEX AS AN
010549,000046:                                                                                                  #                        INDEX AND THE REGISTER 8NN AS A COUNTER.  MARKINDX IS DECREMENTED.
010550,000047:                                                                                                  #                IF R57 IS RUNNING, MARK DATA IS STORED INTO MARKDOWN FOR DOWNLINK, THEN PROCESSED AS FOR
010551,000048:                                                                                                  #                        R21.  MARKRUPT THEN CALLS MARKDISP (IN R57).1DNADR
010552,000049:                                                                                                  #                FOR OTHER USERS, MARK DATA IS STORED INTO MRKBUF1 AND MARKINDX IS DECREMENTED.
010553,000050: 
010554,000051:                                                                                                  #        MARK REJECTS ARE PROCESSED BY THE ROUTINE MARKRUPT AS FOLLOWS:
010555,000052:                                                                                                  #                IF MARKFLG IS CLEAR (I.E., NO MARK WAS TAKEN), ALARM CODE 110 IS SENT AND THE ROUTINE EXITS.
010556,000053:                                                                                                  #                OTHERWISE, MARKINDX IS INCREMENTED, THE P22 INDICATORS 8NN AND P22DEX ARE DECREMENTED
010557,000054:                                                                                                  #                        (IF THE USER IS P22), AND THE V51FL DISPLAY IN SXTMARK IS REESTABLISHED.
010558,000055: 
010559,000056:                                                                                                  #        IF THE ERASABLE REGISTER CDUCHKWD IS SET TO A NON-ZERO VALUE, VALIDITY OF THE MARKS IS CHECKED
010560,000057:                                                                                                  #           BY THE MARKRUPT ROUTINE AS FOLLOWS:
010561,000058:                                                                                                  #                IF THE CDU'S CHANGE BY MORE THAN 3 BITS OVER THE TIME PERIOD INDICATED BY THE VALUE OF
010562,000059:                                                                                                  #                        CDUCHKWD, ALARM CODE 121 IS SENT AND THE ROUTINE EXITS.
010563,000060: 
010564,000061:                                                                                                  #  CALLING SEQUENCE -
010565,000062: 
010566,000063:                                                                                                  #        TC      BANKCALL
010567,000064:                                                                                                  #        CADR    SXTMARK
