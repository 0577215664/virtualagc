Source Code

These source-code files were obtained by digitally photographing an Artemis 72 (Apollo 15-17 Command Module) program listing from a private collection and then modifying pre-existing Comanche 55 (Apollo 11) source files to incorporate changes. Photography was by Ron Burkey, and transcription was performed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images may be available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 072 OF AGC PROGRAM ARTEMIS 
	BY NASA 2021114-011 11:40 FEB. 26, 1971
Note that the date is the date of the printout, not the date of the program revision.

042476,000017:                                                                                                  #  PROGRAM NAME....TVCDAP, CONSISTING OF PITCHDAP, YAWDAP, ETC.
042477,000018:                                                                                                  #  LOG SECTION...TVCDAPS                         SUBROUTINE...DAPCSM
042478,000019:                                                                                                  #  MODIFIED BY SCHLUNDT                          21 OCTOBER 1968
042479,000020:                                                                                                  #  FUNCTIONAL DESCRIPTION....
042480,000021: 
042481,000022:                                                                                                  #        SELF-PERPETUATING T5 TASKS WHICH GENERATE THE COMMAND SIGNALS
042482,000023:                                                                                                  #        FOR THE PITCH AND YAW SPS GIMBAL ACTUATORS DURING TVC (SPS) BURNS,
042483,000024:                                                                                                  #        IN RESPONSE TO BODY-AXIS RATE COMMANDS FROM CROSS-PRODUCT STEERING
042484,000025:                                                                                                  #        (S40.8).  IF NO STEERING (IMPULSIVE BURNS) MAINTAINS ATTITUDE-HOLD
042485,000026:                                                                                                  #        ABOUT THE REFERENCE (INITIAL) DIRECTIONS (ZERO RATE COMMANDS).
042486,000027: 
042487,000028:                                                                                                  #        THE PITCH AND YAW LOOPS ARE SEPARATE, BUT STRUCTURED IDENTICALLY.
042488,000029:                                                                                                  #        EACH ATTITUDE-RATE LOOP INCLUDES GIMBAL ANGLE RATE DERIVATION,
042489,000030:                                                                                                  #        GIMBAL/BODY AXIS TRANSFORMATION, BODY-AXIS ATTITUDE ERROR
042490,000031:                                                                                                  #        INTEGRATION WITH ERROR LIMITING, THE GENERALIZED 6TH-ORDER FILTER
042491,000032:                                                                                                  #        FOR CSM OR CSM/LM OPERATION, A FILTER OUTPUT LIMITER,
042492,000033:                                                                                                  #        CG-OFFSET TRACKER FILTER, AND THE CG-TRACKER MINOR LOOP.
042493,000034: 
042494,000035:                                                                                                  #        THE DAPS ARE CYCLIC, CALLING EACH OTHER AT 1/2 THE DAP SAMPLE
042495,000036:                                                                                                  #        TIME, AS DETERMINED BY T5TVCDT. THE ACTUATOR COMMANDS ARE
042496,000037:                                                                                                  #        REGENERATED AS ANALOG VOLTAGES BY THE OPTICS ERROR COUNTERS, WHICH
042497,000038:                                                                                                  #        TRANSMIT THE SIGNAL TO THE ACTUATOR SERVOS WHEN THERE IS PROPER CDU
042498,000039:                                                                                                  #        MODING.
042499,000040: 
042500,000041:                                                                                                  #  CALLING SEQUENCE.... (TYPICALLY)
042501,000042: 
042502,000043:                                                                                                  #        T5 CALL OF TVCDAPON (TVCINITIALIZE) BY DOTVCON (P40)
042503,000044:                                                                                                  #        T5 CALL OF DAPINIT (TVCDAPS) BY TVCINIT4 (TVCINITIALIZE)
042504,000045:                                                                                                  #        T5 CALL OF PITCHDAP BY DAPINIT
042505,000046:                                                                                                  #        T5 CALL OF YAWDAP BY PITCHDAP
042506,000047:                                                                                                  #        T5 CALL OF PITCHDAP BY YAWDAP
042507,000048:                                                                                                  #                ETC.
042508,000049:                                                                                                  #        (AUTOMATIC SEQUENCING FROM TVCDAPON)
042509,000050: 
042510,000051:                                                                                                  #  NORMAL EXIT MODE....RESUME
042511,000052: 
042512,000053:                                                                                                  #  ALARM OR ABORT EXIT MODES....NONE
042513,000054: 
042514,000055:                                                                                                  #  SUBROUTINES CALLED....
042515,000056: 
042516,000057:                                                                                                  #        HACK FOR STROKE TEST (V68) WAVEFORM GENERATION
042517,000058:                                                                                                  #        PCOPY, YCOPY FOR COPY-CYCLES (USED ALSO BY TVC RESTART PACKAGE)
042518,000059:                                                                                                  #        DAPINIT FOR INITIAL CDUS FOR RATE MEASUREMENTS
042519,000060:                                                                                                  #        ERRORLIM, ACTLIM FOR INPUT (ATTITUDE-ERROR INTEGRATION) AND
042520,000061:                                                                                                  #                OUTPUT (ACTUATOR COMMAND) LIMITING, COMMON TO PITCH AND
042521,000062:                                                                                                  #                YAW DAPS
042522,000063:                                                                                                  #        FWDFLTR (INCLUDING OPTVARK) AND PRECOMP, TO COMPUTE FILTER
042523,000064:                                                                                                  #                OUTPUTS AND STORAGE VALUES
042524,000065:                                                                                                  #        RESUME
042525,000066:
