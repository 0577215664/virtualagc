Source Code

These source-code files were obtained by digitally photographing an Artemis 72 (Apollo 15-17 Command Module) program listing from a private collection and then modifying pre-existing Comanche 55 (Apollo 11) source files to incorporate changes. Photography was by Ron Burkey, and transcription was performed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images may be available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 072 OF AGC PROGRAM ARTEMIS 
	BY NASA 2021114-011 11:40 FEB. 26, 1971
Note that the date is the date of the printout, not the date of the program revision.

042580,000019:                                                                                                  #  PROGRAM NAME....TVCDAP, CONSISTING OF PITCHDAP, YAWDAP, ETC.
042581,000020:                                                                                                  #  LOG SECTION...TVCDAPS                         SUBROUTINE...DAPCSM
042582,000021:                                                                                                  #  MODIFIED BY SCHLUNDT                          21 OCTOBER 1968
042583,000022:                                                                                                  #  FUNCTIONAL DESCRIPTION....
042584,000023: 
042585,000024:                                                                                                  #        SELF-PERPETUATING T5 TASKS WHICH GENERATE THE COMMAND SIGNALS
042586,000025:                                                                                                  #        FOR THE PITCH AND YAW SPS GIMBAL ACTUATORS DURING TVC (SPS) BURNS,
042587,000026:                                                                                                  #        IN RESPONSE TO BODY-AXIS RATE COMMANDS FROM CROSS-PRODUCT STEERING
042588,000027:                                                                                                  #        (S40.8).  IF NO STEERING (IMPULSIVE BURNS) MAINTAINS ATTITUDE-HOLD
042589,000028:                                                                                                  #        ABOUT THE REFERENCE (INITIAL) DIRECTIONS (ZERO RATE COMMANDS).
042590,000029: 
042591,000030:                                                                                                  #        THE PITCH AND YAW LOOPS ARE SEPARATE, BUT STRUCTURED IDENTICALLY.
042592,000031:                                                                                                  #        EACH ATTITUDE-RATE LOOP INCLUDES GIMBAL ANGLE RATE DERIVATION,
042593,000032:                                                                                                  #        GIMBAL/BODY AXIS TRANSFORMATION, BODY-AXIS ATTITUDE ERROR
042594,000033:                                                                                                  #        INTEGRATION WITH ERROR LIMITING, THE GENERALIZED 6TH-ORDER FILTER
042595,000034:                                                                                                  #        FOR CSM OR CSM/LM OPERATION, A FILTER OUTPUT LIMITER,
042596,000035:                                                                                                  #        CG-OFFSET TRACKER FILTER, AND THE CG-TRACKER MINOR LOOP.
042597,000036: 
042598,000037:                                                                                                  #        THE DAPS ARE CYCLIC, CALLING EACH OTHER AT 1/2 THE DAP SAMPLE
042599,000038:                                                                                                  #        TIME, AS DETERMINED BY T5TVCDT. THE ACTUATOR COMMANDS ARE
042600,000039:                                                                                                  #        REGENERATED AS ANALOG VOLTAGES BY THE OPTICS ERROR COUNTERS, WHICH
042601,000040:                                                                                                  #        TRANSMIT THE SIGNAL TO THE ACTUATOR SERVOS WHEN THERE IS PROPER CDU
042602,000041:                                                                                                  #        MODING.
042603,000042: 
042604,000043:                                                                                                  #  CALLING SEQUENCE.... (TYPICALLY)
042605,000044: 
042606,000045:                                                                                                  #        T5 CALL OF TVCDAPON (TVCINITIALIZE) BY DOTVCON (P40)
042607,000046:                                                                                                  #        T5 CALL OF DAPINIT (TVCDAPS) BY TVCINIT4 (TVCINITIALIZE)
042608,000047:                                                                                                  #        T5 CALL OF PITCHDAP BY DAPINIT
042609,000048:                                                                                                  #        T5 CALL OF YAWDAP BY PITCHDAP
042610,000049:                                                                                                  #        T5 CALL OF PITCHDAP BY YAWDAP
042611,000050:                                                                                                  #                ETC.
042612,000051:                                                                                                  #        (AUTOMATIC SEQUENCING FROM TVCDAPON)
042613,000052: 
042614,000053:                                                                                                  #  NORMAL EXIT MODE....RESUME
042615,000054: 
042616,000055:                                                                                                  #  ALARM OR ABORT EXIT MODES....NONE
042617,000056: 
042618,000057:                                                                                                  #  SUBROUTINES CALLED....
042619,000058: 
042620,000059:                                                                                                  #        HACK FOR STROKE TEST (V68) WAVEFORM GENERATION
042621,000060:                                                                                                  #        PCOPY, YCOPY FOR COPY-CYCLES (USED ALSO BY TVC RESTART PACKAGE)
042622,000061:                                                                                                  #        DAPINIT FOR INITIAL CDUS FOR RATE MEASUREMENTS
042623,000062:                                                                                                  #        ERRORLIM, ACTLIM FOR INPUT (ATTITUDE-ERROR INTEGRATION) AND
042624,000063:                                                                                                  #                OUTPUT (ACTUATOR COMMAND) LIMITING, COMMON TO PITCH AND
042625,000064:                                                                                                  #                YAW DAPS
042626,000065:                                                                                                  #        FWDFLTR (INCLUDING OPTVARK) AND PRECOMP, TO COMPUTE FILTER
042627,000066:                                                                                                  #                OUTPUTS AND STORAGE VALUES
042628,000067:                                                                                                  #        RESUME
042629,000068:
