Source Code

These source-code files were obtained by digitally photographing an Artemis 72 program listing from a private collection and then modifying pre-existing Comanche 55 (Apollo 11) source files to incorporate changes. Photography was by Ron Burkey, and transcription was performed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images may be available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 072 OF AGC PROGRAM ARTEMIS 
	BY NASA 2021114-011 11:40 FEB. 26, 1971
Note that the date is the date of the printout, not the date of the program revision.

042417,000017:                                                                                                  #  PROGRAM NAME....TVCDAP, CONSISTING OF PITCHDAP, YAWDAP, ETC.
042418,000018:                                                                                                  #  LOG SECTION...TVCDAPS                         SUBROUTINE...DAPCSM
042419,000019:                                                                                                  #  MODIFIED BY SCHLUNDT                          21 OCTOBER 1968
042420,000020:                                                                                                  #  FUNCTIONAL DESCRIPTION....
042421,000021: 
042422,000022:                                                                                                  #        SELF-PERPETUATING T5 TASKS WHICH GENERATE THE COMMAND SIGNALS
042423,000023:                                                                                                  #        FOR THE PITCH AND YAW SPS GIMBAL ACTUATORS DURING TVC (SPS) BURNS,
042424,000024:                                                                                                  #        IN RESPONSE TO BODY-AXIS RATE COMMANDS FROM CROSS-PRODUCT STEERING
042425,000025:                                                                                                  #        (S40.8).  IF NO STEERING (IMPULSIVE BURNS) MAINTAINS ATTITUDE-HOLD
042426,000026:                                                                                                  #        ABOUT THE REFERENCE (INITIAL) DIRECTIONS (ZERO RATE COMMANDS).
042427,000027: 
042428,000028:                                                                                                  #        THE PITCH AND YAW LOOPS ARE SEPARATE, BUT STRUCTURED IDENTICALLY.
042429,000029:                                                                                                  #        EACH ATTITUDE-RATE LOOP INCLUDES GIMBAL ANGLE RATE DERIVATION,
042430,000030:                                                                                                  #        GIMBAL/BODY AXIS TRANSFORMATION, BODY-AXIS ATTITUDE ERROR
042431,000031:                                                                                                  #        INTEGRATION WITH ERROR LIMITING, THE GENERALIZED 6TH-ORDER FILTER
042432,000032:                                                                                                  #        FOR CSM OR CSM/LM OPERATION, A FILTER OUTPUT LIMITER,
042433,000033:                                                                                                  #        CG-OFFSET TRACKER FILTER, AND THE CG-TRACKER MINOR LOOP.
042434,000034: 
042435,000035:                                                                                                  #        THE DAPS ARE CYCLIC, CALLING EACH OTHER AT 1/2 THE DAP SAMPLE
042436,000036:                                                                                                  #        TIME, AS DETERMINED BY T5TVCDT. THE ACTUATOR COMMANDS ARE
042437,000037:                                                                                                  #        REGENERATED AS ANALOG VOLTAGES BY THE OPTICS ERROR COUNTERS, WHICH
042438,000038:                                                                                                  #        TRANSMIT THE SIGNAL TO THE ACTUATOR SERVOS WHEN THERE IS PROPER CDU
042439,000039:                                                                                                  #        MODING.
042440,000040: 
042441,000041:                                                                                                  #  CALLING SEQUENCE.... (TYPICALLY)
042442,000042: 
042443,000043:                                                                                                  #        T5 CALL OF TVCDAPON (TVCINITIALIZE) BY DOTVCON (P40)
042444,000044:                                                                                                  #        T5 CALL OF DAPINIT (TVCDAPS) BY TVCINIT4 (TVCINITIALIZE)
042445,000045:                                                                                                  #        T5 CALL OF PITCHDAP BY DAPINIT
042446,000046:                                                                                                  #        T5 CALL OF YAWDAP BY PITCHDAP
042447,000047:                                                                                                  #        T5 CALL OF PITCHDAP BY YAWDAP
042448,000048:                                                                                                  #                ETC.
042449,000049:                                                                                                  #        (AUTOMATIC SEQUENCING FROM TVCDAPON)
042450,000050: 
042451,000051:                                                                                                  #  NORMAL EXIT MODE....RESUME
042452,000052: 
042453,000053:                                                                                                  #  ALARM OR ABORT EXIT MODES....NONE
042454,000054: 
042455,000055:                                                                                                  #  SUBROUTINES CALLED....
042456,000056: 
042457,000057:                                                                                                  #        HACK FOR STROKE TEST (V68) WAVEFORM GENERATION
042458,000058:                                                                                                  #        PCOPY, YCOPY FOR COPY-CYCLES (USED ALSO BY TVC RESTART PACKAGE)
042459,000059:                                                                                                  #        DAPINIT FOR INITIAL CDUS FOR RATE MEASUREMENTS
042460,000060:                                                                                                  #        ERRORLIM, ACTLIM FOR INPUT (ATTITUDE-ERROR INTEGRATION) AND
042461,000061:                                                                                                  #                OUTPUT (ACTUATOR COMMAND) LIMITING, COMMON TO PITCH AND
042462,000062:                                                                                                  #                YAW DAPS
042463,000063:                                                                                                  #        FWDFLTR (INCLUDING OPTVARK) AND PRECOMP, TO COMPUTE FILTER
042464,000064:                                                                                                  #                OUTPUTS AND STORAGE VALUES
042465,000065:                                                                                                  #        RESUME
042466,000066:
