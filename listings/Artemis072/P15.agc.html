Source Code

These source-code files were obtained by digitally photographing an Artemis 72 (Apollo 15-17 Command Module) program listing from a private collection and then modifying pre-existing Comanche 55 (Apollo 11) source files to incorporate changes. Photography was by Ron Burkey, and transcription was performed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images may be available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 072 OF AGC PROGRAM ARTEMIS 
	BY NASA 2021114-011 11:40 FEB. 26, 1971
Note that the date is the date of the printout, not the date of the program revision.

023144,000022:                                                                                                  #        DISPLAYS N33 : TIME OF SIVB INJECTION SEQUENCE START (TB6)
023145,000023:                                                                                                  #        ESTABLISHES T6SET TO BE EXECUTED AT TB6 TIME
023146,000024:                                                                                                  #        SETS TLITIG = TB6 + 9:37.6
023147,000025:                                                                                                  #        DISPLAYS N14 : INERTIAL VELOCITY MAGNITUDE AT TLI C/O
023148,000026:                                                                                                  #        DISPLAYS N95 : TFI - TIME FROM TLITIG (DECREASING)
023149,000027:                                                                                                  #                       VG  - VELOCITY-TO-BE-GAINED
023150,000028:                                                                                                  #                       V   - INERTIAL VELOCITY MAGNITUDE
023151,000029:                                                                                                  #        T6SET : SETS THE SIVB INJECTION SEQUENCE START BIT,
023152,000030:                                                                                                  #        ESTABLISHES T6RESET (DT = 10 SECS), AND
023153,000031:                                                                                                  #        TURNS ON THE UPLINK ACTIVITY LIGHT
023154,000032:                                                                                                  #        T6RESET : CLEARS THE SIVB I.S.S. BIT AND
023155,000033:                                                                                                  #        TURNS OFF THE UPLINK ACTIVITY LIGHT
023156,000034:                                                                                                  #        P40S/SV : CALLS MIDTOAV1 TO INTEGRATE THE CSM
023157,000035:                                                                                                  #        STATE VECTOR TO TLITLIG - 100 AND
023158,000036:                                                                                                  #        ESTABLISHES TIGBLNK AT TLITIG - 105
023159,000037:                                                                                                  #        TIGBLNK : BLANKS THE DSKY FOR 5 SECS AND
023160,000038:                                                                                                  #        ESTABLISHES TIGAVEG FOR TLITIG - 100 
023161,000039:                                                                                                  #        TIGAVEG : STARTS READACCS AND REDISPLAYS
023162,000040:                                                                                                  #        N95 (VG AND V NOW DYNAMIC)
023163,000041:                                                                                                  #        SIVBCOMP : UPDATES VG AND V AND
023164,000042:                                                                                                  #        AFTER TLITIG + 10 CALCULATES TGO AND
023165,000043:                                                                                                  #        CHANGES THE N95 R1 DISPLAY TO TFC
023166,000044:                                                                                                  #        WHEN TGO < 4 SECS, SIVBCOMP ESTABLISHES SIVBOFF (DT = TGO)
023167,000045:                                                                                                  #        SIVBOFF : SHUTS DOWN THE SIVB AND CALLS POSTTLI
023168,000046:                                                                                                  #        POSTTLI : FLASHES N95 TO INDICATE TLI IS COMPLETED
