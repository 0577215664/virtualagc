Source Code

These source-code files were obtained by digitally photographing an Artemis 72 (Apollo 15-17 Command Module) program listing from a private collection and then modifying pre-existing Comanche 55 (Apollo 11) source files to incorporate changes. Photography was by Ron Burkey, and transcription was performed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images may be available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 072 OF AGC PROGRAM ARTEMIS 
	BY NASA 2021114-011 11:40 FEB. 26, 1971
Note that the date is the date of the printout, not the date of the program revision.

060187,000063:                                                                                                  #  PHASCHNG IS THE MAIN WAY OF MAKING PHASE CHANGES FOR RESTARTS.  THERE ARE THREE FORMS OF PHASCHNG, KNOWN AS TYPE
060188,000064:                                                                                                  #  A, TYPE B, AND TYPE C. THEY ARE ALL CALLED AS FOLLOWS, WHERE OCT XXXXX CONTAINS THE PHASE INFORMATION,
060189,000065: 
060190,000066:                                                                                                  #                TC      PHASCHNG
060191,000067:                                                                                                  #                OCT     XXXXX
060192,000068: 
060193,000069:                                                                                                  #  TYPE A IS CONCERNED WITH FIXED PHASE CHANGES, THAT IS, PHASE INFORMATION THAT IS STORED PERMANENTLY.  THESE
060194,000070:                                                                                                  #  OPTIONS ARE, WHERE G STANDS FOR A GROUP AND .X FOR THE PHASE,
060195,000071: 
060196,000072:                                                                                                  #        G.0             INACTIVE, WILLNOT PERMIT A GROUP G RESTART
060197,000073:                                                                                                  #        G.1             WILL CAUSE THE LAST DISPLAY TO BE REACTIVATED, USED MAINLY IN MANNED FLIGHTS
060198,000074:                                                                                                  #        G.EVEN          A DOUBLE TABLE RESTART, CAN CAUSE ANY COMBINATION OF TWO JOBS, TASKS, AND/OR
060199,000075:                                                                                                  #                        LONGCALL TO BE RESTARTED.
060200,000076:                                                                                                  #        G.ODD NOT .1    A SINGLE TABLE RESTART, CAN CAUSE EITHER A JOB, TASK, OR LONGCALL RESTART
060201,000077: 
060202,000078:                                                                                                  #  THIS INFORMATION IS PUT INTO THE OCTAL WORD AFTER TC PHASCHNG AS FOLLOWS
060203,000079: 
060204,000080:                                                                                                  #        TL0 00P PPP PPP GGG                                               ,
060205,000081: 
060206,000082:                                                                                                  #  WHERE EACH LETTER OR NUMBER STANDS FOR A BIT.  THE G'S STAND FOR THE GROUP, OCTAL 1 - 7, THE P'S FOR THE PHASE,
060207,000083:                                                                                                  #  OCTAL 0 - 127.  0'S MUST BE 0.              IF ONE WISHES TO HAVE THE TBASE OF GROUP G TO BE SET AT THIS TIME,
060208,000084:                                                                                                  #  T IS SET TO 1, OTHERWISE IT IS SET TO 0.  SIMIARLY IF ONE WISHES TO SET LONGBASE, THEN L IS SET TO 1, OTHERWISE
060209,000085:                                                                                                  #  IT IS SET TO 0.  SOME EXAMPLES,
060210,000086: 
060211,000087:                                                                                                  #                TC      PHASCHNG        THIS WILL CAUSE GROUP 3 TO BE SET TO 0,
060212,000088:                                                                                                  #                OCT     00003           MAKING GROUP 3 INACTIVE
060213,000089: 
060214,000090:                                                                                                  #                TC      PHASCHNG        IF A RESTART OCCURS THIS WOULD CAUSE
060215,000091:                                                                                                  #                OCT     00012           GROUP 2 TO RESTART THE LAST DISPLAY
060216,000092: 
060217,000093:                                                                                                  #                TC      PHASCHNG        THIS SETS THE TBASE OF GROUP 4 AND IN
060218,000094:                                                                                                  #                OCT     40064           CASE OF A RESTART WOULD START UP THE TWO
060219,000095:                                                                                                  #                                        THINGS LOCATED IN THE DOUBLE 4.6 RESTART
060220,000096:                                                                                                  #                                        LOCATION
060221,000097:                                                                                                  #                TC      PHASCHNG        THIS SETS LONGBASE AND UPON A RESTART
060222,000098:                                                                                                  #                OCT     20135           CAUSES 5.13 TO BE RESTARTED (SINCE
060223,000099:                                                                                                  #                                        LONGBASE WAS SET THIS SINGLE ENTRY
060224,000100:                                                                                                  #                                        SHOULD BE A LONGCALL)
060225,000101:                                                                                                  #                TC      PHASCHNG        SINCE BOTH TBASE4 AND LONGBASE ARE SET,
060226,000102:                                                                                                  #                OCT     60124           4.12 SHOULD CONTAIN BOTH A TASK AND A
060227,000103:                                                                                                  #                                        LONGCALL TO BE RESTARTED
060228,000104: 
060229,000105:                                                                                                  #  TYPE C PHASCHNG CONTAINS THE VARIABLE TYPE OF PHASCHNG INFORMATION.  INSTEAD OF THE INFORMATION BEING IN A
060230,000106:                                                                                                  #  PERMANENT FORM, ONE STORES THE DESIRED RESTART INFORMATION IN A VARIABLE LOCATION. THE BITS ARE AS FOLLOWS,
060231,000107: 
060232,000108:                                                                                                  #        TL0 1AD XXX CJW GGG
060233,000109: 
060234,000110:                                                                                                  #  WHERE EACH LETTER OR NUMBER STANDS FOR A BIT.  THE G'S STAND FOR THE GROUP, OCTAL 1 - 7.  IF THE RESTART IS TO
060235,000111:                                                                                                  #  BE BY WAITLIST, W IS SET TO 1, IF IT IS A JOB, J IS SET TO 1, IF IT IS A LONGCALL, C IS SET TO 1. ONLY ONE OF
060236,000112:                                                                                                  #  THESE THREE BITS MAY BE SET.  X'S ARE IGNORED  1 MUST BE 1, AND 0 MUST BE 0.  AGAIN T STANDS FOR THE TBASE,

060238,000114:                                                                                                  #  AND L FOR LONGBASE.  THE BITS A AND D ARE CONCERNED WITH THE VARIABLE INFORMATION. IF D IS SET TO 1, A PRIORITY
060239,000115:                                                                                                  #  OR DELTA TIME WILL BE READ FROM THE NEXT LOCATION AFTER THE OCTAL INFORMATION, IF THIS IS TO BE INDIRECT, THAT

060270,000146:                                                                                                  #  PHASE INFORMATION, OCTAL 0 - 127. 1 MUST BE 1.  AND AGAIN T STANDS FOR THE TBASE AND L FOR LONGBASE.  D THIS

060272,000148:                                                                                                  #  AGAIN A STANDS FOR THE ADDRESS OF THE LOCATION TO BE RESTARTED, 1 IF THE 2CADR IS GIVEN, OR 0 IF IT IS TO BE
