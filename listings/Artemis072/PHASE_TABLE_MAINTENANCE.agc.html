Source Code

These source-code files were obtained by digitally photographing an Artemis 72 program listing from a private collection and then modifying pre-existing Comanche 55 (Apollo 11) source files to incorporate changes. Photography was by Ron Burkey, and transcription was performed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images may be available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 072 OF AGC PROGRAM ARTEMIS 
	BY NASA 2021114-011 11:40 FEB. 26, 1971
Note that the date is the date of the printout, not the date of the program revision.

059971,000062:                                                                                                  #  PHASCHNG IS THE MAIN WAY OF MAKING PHASE CHANGES FOR RESTARTS.  THERE ARE THREE FORMS OF PHASCHNG, KNOWN AS TYPE
059972,000063:                                                                                                  #  A, TYPE B, AND TYPE C. THEY ARE ALL CALLED AS FOLLOWS, WHERE OCT XXXXX CONTAINS THE PHASE INFORMATION,
059973,000064: 
059974,000065:                                                                                                  #                TC      PHASCHNG
059975,000066:                                                                                                  #                OCT     XXXXX
059976,000067: 
059977,000068:                                                                                                  #  TYPE A IS CONCERNED WITH FIXED PHASE CHANGES, THAT IS, PHASE INFORMATION THAT IS STORED PERMANENTLY.  THESE
059978,000069:                                                                                                  #  OPTIONS ARE, WHERE G STANDS FOR A GROUP AND .X FOR THE PHASE,
059979,000070: 
059980,000071:                                                                                                  #        G.0             INACTIVE, WILL NOT PERMIT A GROUP G RESTART
059981,000072:                                                                                                  #        G.1             WILL CAUSE THE LAST DISPLAY TO BE REACTIVATED, USED MAINLY IN MANNED FLIGHTS
059982,000073:                                                                                                  #        G.EVEN          A DOUBLE TABLE RESTART, CAN CAUSE ANY COMBINATION OF TWO JOBS, TASKS, AND/OR
059983,000074:                                                                                                  #                        LONGCALL TO BE RESTARTED.
059984,000075:                                                                                                  #        G.ODD NOT .1    A SINGLE TABLE RESTART, CAN CAUSE EITHER A JOB, TASK, OR LONGCALL RESTART
059985,000076: 
059986,000077:                                                                                                  #  THIS INFORMATION IS PUT INTO THE OCTAL WORD AFTER TC PHASCHNG AS FOLLOWS
059987,000078: 
059988,000079:                                                                                                  #        TL0 00P PPP PPP GGG
059989,000080: 
059990,000081:                                                                                                  #  WHERE EACH LETTER OR NUMBER STANDS FOR A BIT.  THE G'S STAND FOR THE GROUP, OCTAL 1 - 7, THE P'S FOR THE PHASE,
059991,000082:                                                                                                  #  OCTAL 0 - 127.  0'S MUST BE 0.              IF ONE WISHES TO HAVE THE TBASE OF GROUP G TO BE SET AT THIS TIME,
059992,000083:                                                                                                  #  T IS SET TO 1, OTHERWISE IT IS SET TO 0.  SIMILARLY IF ONE WISHES TO SET LONGBASE, THEN L IS SET TO 1, OTHERWISE
059993,000084:                                                                                                  #  IT IS SET TO 0.  SOME EXAMLES,
059994,000085: 
059995,000086:                                                                                                  #                TC      PHASCHNG        # THIS WILL CAUSE GROUP 3 TO BE SET TO 0,
059996,000087:                                                                                                  #                OCT     00003           # MAKING GROUP 3 INACTIVE
059997,000088: 
059998,000089:                                                                                                  #                TC      PHASCHNG        # IF A RESTART OCCURS THIS WOULD CAUSE
059999,000090:                                                                                                  #                OCT     00012           # GROUP 2 TO RESTART THE LAST DISPLAY
060000,000091: 
060001,000092:                                                                                                  #                TC      PHASCHNG        # THIS SETS THE TBASE OF GROUP 4 AND IN
060002,000093:                                                                                                  #                OCT     40064           # CASE OF A RESTART WOULD START UP THE TWO
060003,000094:                                                                                                  #                                        # THINGS LOCATED IN THE DOUBLE 4.6 RESTART
060004,000095:                                                                                                  #                                        # LOCATION.
060005,000096:                                                                                                  #                TC      PHASCHNG        # THIS SETS LONGBASE AND UPON A RESTART
060006,000097:                                                                                                  #                OCT     20135           # CAUSES 5.13 TO BE RESTARTED (SINCE
060007,000098:                                                                                                  #                                        # LONGBASE WAS SET THIS SINGLE ENTRY
060008,000099:                                                                                                  #                                        # SHOULD BE A LONGCALL)
060009,000100:                                                                                                  #                TC      PHASCHNG        # SINCE BOTH TBASE4 AND LONGBASE ARE SET,
060010,000101:                                                                                                  #                OCT     60124           # 4.12 SHOULD CONTAIN BOTH A TASK AND A
060011,000102:                                                                                                  #                                        # LONGCALL TO BE RESTARTED
060012,000103: 
060013,000104:                                                                                                  #  TYPE C PHASCHNG CONTAINS THE VARIABLE TYPE OF PHASCHNG INFORMATION.  INSTEAD OF THE INFORMATION BEING IN A
060014,000105:                                                                                                  #  PERMANENT FORM, ONE STORES THE DESIRED RESTART INFORMATION IN A VARIABLE LOCATION. THE BITS ARE AS FOLLOWS,
060015,000106: 
060016,000107:                                                                                                  #        TL0 1AD XXX CJW GGG
060017,000108: 
060018,000109:                                                                                                  #  WHERE EACH LETTER OR NUMBER STANDS FOR A BIT.  THE G'S STAND FOR THE GROUP, OCTAL 1 - 7.  IF THE RESTART IS TO
060019,000110:                                                                                                  #  BE BY WAITLIST, W IS SET TO 1, IF IT IS A JOB, J IS SET TO 1, IF IT IS A LONGCALL, C IS SET TO 1. ONLY ONE OF
060020,000111:                                                                                                  #  THESE THREE BITS MAY BE SET.  X'S ARE IGNORED, 1 MUST BE 1, AND 0 MUST BE 0.  AGAIN T STANDS FOR THE TBASE,

060023,000114:                                                                                                  #  OR DELTA TIME WILL BE READ FROM THE NEXT LOCATION AFTER THE OCTAL INFORMATION, IF THIS IS TO BE INDIRECT, THAT
