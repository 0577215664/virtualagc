Source Code

These source-code files were obtained by digitally photographing an Artemis 72 (Apollo 15-17 Command Module) program listing from a private collection and then modifying pre-existing Comanche 55 (Apollo 11) source files to incorporate changes. Photography was by Ron Burkey, and transcription was performed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images may be available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 072 OF AGC PROGRAM ARTEMIS 
	BY NASA 2021114-011 11:40 FEB. 26, 1971
Note that the date is the date of the printout, not the date of the program revision.

041810,000018:                                                                                                  #  PROGRAM NAME.... TVCEXECUTIVE, CONSISTING OF TVCEXEC, NEEDLEUP, VARGAINS
041811,000019:                                                                                                  #                   1SHOTCHK, REPCHEK, CG.CORR, COPYCYCLES, ETC.
041812,000020:                                                                                                  #  LOG SECTION...TVCEXECUTIVE                    SUBROUTINE...DAPCSM
041813,000021:                                                                                                  #  MODIFIED BY SCHLUNDT                          21 OCTOBER 1968
041814,000022:                                                                                                  #  MODIFIED BY BEALS TO ELIMINATE CSMMASS UPDATE LOGIC (NOW DONE IN S40.8)
041815,000023:                                                                                                  #  FUNCTIONAL DESCRIPTION....
041816,000024: 
041817,000025:                                                                                                  #       *A SELF-PERPETUATING WAITLIST TASK AT 1/2 SECOND INTERVALS WHICH:
041818,000026:                                                                                                  #        PREPARES THE ROLL WITH OGA (CDUX)
041819,000027:                                                                                                  #        PREPARES THE ROLL FDAI NEEDLE (FLY-TO  OGA ERROR)
041820,000028:                                                                                                  #        PREPARES THE ROLL PHASE PLANE  OGAERR  (FLY-FROM  OGA ERROR)
041821,000029:                                                                                                  #        PREPARES THE TVC ROLLDAP TASK WAITLIST CALL (3 CS DELAY)
041822,000030:                                                                                                  #        UPDATES THE NEEDLES DISPLAY
041823,000031:                                                                                                  #        CALLS MASSPROP TO UPDATE INERTIA DATA
041824,000032:                                                                                                  #        UPDATES PITCH, YAW, AND ROLL DAP GAINS FROM MASSPROP DATA
041825,000033:                                                                                                  #        PERFORMS ONE-SHOT CORRECTION FOR TMC LOOP 0-3 SEC AFTER IGNITION
041826,000034:                                                                                                  #        PERFORMS REPETITIVE UPDATES FOR THE TMC LOOP AFTER THE ONE-SHOT CORR.
041827,000035: 
041828,000036:                                                                                                  #  CALLING SEQUENCE....
041829,000037: 
041830,000038:                                                                                                  #       *TVCEXEC CALLED AS A WAITLIST TASK, IN PARTICULAR BY TVCINIT4 AND BY
041831,000039:                                                                                                  #        ITSELF, BOTH AT 1/2 SECOND INTERVALS
041832,000040: 
041833,000041:                                                                                                  #  NORMAL EXIT MODE.... TASKOVER
041834,000042: 
041835,000043:                                                                                                  #  ALARM OR ABORT EXIT MODES.... NONE
041836,000044: 
041837,000045:                                                                                                  #  SUBROUTINES CALLED....NEEDLER, S40.15, MASSPROP, TASKOVER, IBNKCALL
041838,000046: 
041839,000047:                                                                                                  #  OTHER INTERFACES....
041840,000048: 
041841,000049:                                                                                                  #       *TVCRESTART PACKAGE FOR RESTARTS
041842,000050:                                                                                                  #       *PITCHDAP, YAWDAP FOR VARIABLE GAINS AND ENGINE TRIM ANGLES
041843,000051:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED....
041844,000052: 
041845,000053:                                                                                                  #       *SEE TVCDAPON....TVCINIT4
041846,000054:                                                                                                  #       *VARK AND 1/CONACC (S40.15 OF TVCINITIALIZE)
041847,000055:                                                                                                  #       *PAD LOAD EREPFRAC
041848,000056:                                                                                                  #       *BITS 15,14 OF FLAGWRD6 (T5 BITS)
041849,000057:                                                                                                  #       *TVCEXPHS FOR RESTARTS
041850,000058:                                                                                                  #       *ENGINE-ON BIT (11.13) FOR RESTARTS
041851,000059:                                                                                                  #       *CDUX, OGAD
041852,000060: 
041853,000061:                                                                                                  #  OUTPUT....
041854,000062: 
041855,000063:                                                                                                  #       *ROLL DAP OGANOW, FDAI NEEDLE= (AK). AND PHASE PLANE OGAERR
041856,000064:                                                                                                  #       *VARIABLE GAINS FOR PITCH/YAW AND ROLL TVC DAPS
041857,000065:                                                                                                  #       *SINGLE-SHOT AND REPETITIVE CORRECTIONS TO ENGINE TRIM ANGLES
041858,000066:                                                                                                  #        PACTOFF AND YACTOFF
041859,000067:                                                                                                  #  DEBRIS.... MUCH, BUT SHAREABLE WITH RCS/ENTRY, ALL IN EBANK6
