Source Code

These source-code files were transcribed from scans made from Don Eyles's personal copy of BURST120 (SUNBURST 120). They were scanned at archive.org's Boston facility, and the scanning was sponsored by Mike Stewart. The code was transcribed from these scans by a team of volunteers who are referenced in the program comments. Comments from the original source code are in ALL-CAPS, whereas comments added later in transcription are in Mixed-Case. In some cases, where similar code blocks exist in previously-transcribed AGC programs (primarily Luminary 99, from Apollo 11) those code blocks were used as a starting point and then corrected to agree with the BURST120 scans. The full scans are available at the Virtual AGC project's collection at archive.org, while more-convenient reduced-size (but reduced-quality) images are available at the main Virtual AGC website. Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository. Notations on the program listing read, in part:

	YUL SYSTEM FOR AGC: REVISION 0 OF PROGRAM BURST120 
	BY NASA 2021106-031 DEC 7, 1967

	THIS LISTING IS A COPY OF A VERSION OF THE PROGRAM INTENDED 
	FOR USE IN THE ON-BOARD PRIMARY GUIDANCE COMPUTER IN THE 
	UNMANNED FLIGHT OF APOLLO LUNAR MODULE 1 --- THE AS206 MISSION.
Note that the date is the date of the printout, not the date of the program revision.

034512,000017:                                                                                                  #        MOD NO. 00      MODIFICATION BY A. KOSMALA      NOV. 1966
034513,000018:                                                                                                  #        MOD NO.  1      MODIFICATION BY D. LICKLY       JAN 1967
034514,000019:                                                                                                  #    *FUNCTIONAL DESCRIPTION*
034515,000020: 
034516,000021:                                                                                                  #      THE NORMAL ENTRY TO THE THRUST MISSION CONTROL PROGRAM IS THROUGH PREREAD.  PREREAD SCHEDULES THE JOB
034517,000022:                                                                                                  #  LASTBIAS.  BOOST PHASE ENTERS THE PREREAD ROUTINE AT BIBIBIAS, BYPASSING THE LASTBIAS JOB.  PIPS ARE CLEARED,
034518,000023:                                                                                                  #  AVERAGE G FLAG IS SET, DRIFT FLAG IS UNSET.  DV MONITOR AND FINDCUD ARE INITIALIZED, THE JOB NORMLIZE IS
034519,000024:                                                                                                  #  SCHEDULED, AND A WAITLIST CALL IS MADE TO START READACCS IN TWO SECONDS.
034520,000025: 
034521,000026:                                                                                                  #      NORMLIZE PERFORMS THE SCALING AND INITIALIZATION REQUIRED PRIOR TO THE FIRST ENTRY TO AVERAGE G.
034522,000027: 
034523,000028:                                                                                                  #      IN THE READACCS SECTION, THE ACCELEROMETERS ARE READ BY THE PIPASR SUBROUTINE.  IF THE AVERAGE G FLAG IS
034524,000029:                                                                                                  #  SET, READACCS IS CALLED TO RECYCLE IN TWO SECONDS.  IF THE AVERAGE G FLAG IS DOWN, AVERAGE G EXIT (AVEGEXIT)
034525,000030:                                                                                                  #  IS SET FOR THE FINAL PASS, AND READACCS IS NOT CALLED AGAIN.  IN EITHER CASE, THE SERVICER JOB IS ESTABLISHED.
034526,000031: 
034527,000032:                                                                                                  #      THE SERVICER ROUTINE CHECKS FOR RUNAWAY PIPS (DELV GREATER THAN 3200 PULSES/SEC FOR 2 SEC) AND SENDS
034528,000033:                                                                                                  #  ALARM CODE 205 IF BAD PIP IS FOUND.  PIPS ARE COMPENSATED IN 1/PIPA SUBROUTINE, MASS IS UPDATED BY MASSMON
034529,000034:                                                                                                  #  SUBROUTINE, AND  AVERAGEG DOES THE NAVIGATION.  THE AVERAGE G ROUTINE UPDATES RN, VN, AND GDT/2 VECTORS, USING
034530,000035:                                                                                                  #  THE SUBROUTINE CALCRVG.  CONTROL IS THEN TRANSFERRED TO THE MONITOR SPECIFIED BY DVSELECT.  DVSELECT IS SET BY
034531,000036:                                                                                                  #  THE BOOST PHASE TO BOOSTMON, AND BY THE ENGINEON ROUTINE TO THE PGNCSMON.  USERS STARTING SERVICER BEFORE THE
034532,000037:                                                                                                  #  ENGINE IS ON SHOULD INSURE BYPASSING DVMON INITIALLY BY SETTING DVSELECT TO THE GENADR OF AVERAGE G.
034533,000038: 
034534,000039:                                                                                                  #      THE BOOST MONITOR (BOOSTMON) CHECKS DELV AGAINST THRSHLD+, THE THRESHOLD ACCELERATION FOR THE BOOST PHASE.
034535,000040:                                                                                                  #  IF DELV IS BELOW THRESHOLD, INDICATING TERMINATION OF BOOST, MISSION PHASE 6 IS SCHEDULED AS A JOB, DVSELECT
034536,000041:                                                                                                  #  IS ALTERED TO BYPASS BOOSTMON AND PROCEED DIRECTLY TO AVERAGE G.
034537,000042: 
034538,000043:                                                                                                  #      THE PGNCS MONITOR (PGNCSMON) COMPARES ACTUAL THRUST TO THE THRESHOLD VALUE FOR THE +X ACCELERATION
034539,000044:                                                                                                  #  (100 CM/SEC).  IF THRUST IS FOUND BELOW THIS VALUE FOR 3 CYCLES (I.E., 4 TO 6 SECONDS) AS DETERMINED BY DVCNTR,
034540,000045:                                                                                                  #  AND THE ENGINE FLAG IS REMOVED, (INDICATING THAT THE ENGINE HAS BEEN TURNED OFF) CONTROL IS BRANCHED TO THE
034541,000046:                                                                                                  #  LOCATION SPECIFIED BY DVMNEXIT, PREVIOUSLY SET BY THE USERS PROGRAM.  DVSELECT IS ALTERED TO BYPASS PNGCSMON IN
034542,000047:                                                                                                  #  SUBSEQUENT PASSES.  IF, HOWEVER, THE ENGINE FLAG IS STILL ON AT THIS TIME, CONTROL IS TRANSFERRED TO ENGNFAIL.
034543,000048:                                                                                                  #  THIS SENDS ALARM CODE 1405 AND KILLS THAT MISSION PHASE.  ANY TIME THE THRUST IS LESS THAN THE THRESHOLD VALUE,
034544,000049:                                                                                                  #  THE STEERING IS DETACHED (NO EXIT VIA AVEGEXIT) AND EXIT IS MADE THRU SERVEXIT.
034545,000050: 
034546,000051:                                                                                                  #      THE NORMAL EXIT IS THRU AVEGEXIT, WHICH MUST HAVE BEEN SET BY THE USER. THE FINAL EXIT, SET INTO AVEGEXIT BY
034547,000052:                                                                                                  #  READACCS WHEN IT  FINDS THE AVERAGE G FLAG DOWN, SETS UP FREE FALL GYRO COMPENSATION, SETS THE DRIFT FLAG ON,
034548,000053:                                                                                                  #  PERFORMS AVETOMID ROUTINE, AND TRANSFERS CONTROL TO P00H, THUS CLEARING  ALL ACTIVITY UNTIL A NEW MISSION
034549,000054:                                                                                                  #  PHASE IS DUE.
034550,000055: 
034551,000056:                                                                                                  #  ***** WARNING TO USERS *****
034552,000057: 
034553,000058:                                                                                                  #  THE USER MUST SET DVMNEXIT TO THE 2CADR OF A JOB TO BE PERFORMED WHEN ENGINE SHUTDOWN IS DETECTED BY SERVICER.
034554,000059:                                                                                                  #  IN GENERAL, THE AVERAGE G FLAG WILL BE TURNED OFF BY THE USER AT THAT TIME, ALLOWING JUST ONE MORE PASS THROUGH
034555,000060:                                                                                                  #  AVERAGE G.  ALL ACTIVITY OF THE USERS MISSION PHASE MUST HAVE BEEN COMPLETED BEFORE THIS LAST PASS THROUGH
034556,000061:                                                                                                  #  AVERAGE G, DUE TO THE PERFORMANCE OF P00H AS DESCRIBED ABOVE.
034557,000062: 
034558,000063:                                                                                                  #      AVGEXIT MUST BE SET BY THE USER TO THE 2CADR OF THE JOB (E.G., STEERING) TO BE PERFORMED AFTER EACH PASS
034559,000064:                                                                                                  #  THROUGH AVERAGE G.  IF NO OTHER JOB IS TO BE DONE, AVEGEXIT SHOULD BE SET TO SERVEXIT.
