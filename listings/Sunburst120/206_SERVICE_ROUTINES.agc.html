Source Code

These source-code files were transcribed from scans made from Don Eyles's personal copy of BURST120 (SUNBURST 120). They were scanned at archive.org's Boston facility, and the scanning was sponsored by Mike Stewart. The code was transcribed from these scans by a team of volunteers who are referenced in the program comments. Comments from the original source code are in ALL-CAPS, whereas comments added later in transcription are in Mixed-Case. In some cases, where similar code blocks exist in previously-transcribed AGC programs (primarily Luminary 99, from Apollo 11) those code blocks were used as a starting point and then corrected to agree with the BURST120 scans. The full scans are available at the Virtual AGC project's collection at archive.org, while more-convenient reduced-size (but reduced-quality) images are available at the main Virtual AGC website. Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository. Notations on the program listing read, in part:

	YUL SYSTEM FOR AGC: REVISION 0 OF PROGRAM BURST120

034707,000021:                                                                                                  #        MOD NO. 00      MODIFICATION BY A. KOSMALA      NOV. 1966

034709,000023:                                                                                                  #    *FUNCTIONAL DESCRIPTION*
034710,000024: 
034711,000025:                                                                                                  #      THE NORMAL ENTRY TO THE THRUST MISSION CONTROL PROGRAM IS THROUGH PREREAD.  PREREAD SCHEDULES THE JOB
034712,000026:                                                                                                  #  LASTBIAS.  BOOST PHASE ENTERS THE PREREAD ROUTINE AT BIBIBIAS, BYPASSING THE LASTBIAS JOB.  PIPS ARE CLEARED,
034713,000027:                                                                                                  #  AVERAGE G FLAG IS SET, DRIFT FLAG IS UNSET.  DV MONITOR AND FINDCUD ARE INITIALIZED, THE JOB NORMLIZE IS
034714,000028:                                                                                                  #  SCHEDULED, AND A WAITLIST CALL IS MADE TO START READACCS IN TWO SECONDS.
034715,000029: 
034716,000030:                                                                                                  #      NORMLIZE PERFORMS THE SCALING AND INITIALIZATION REQUIRED PRIOR TO THE FIRST ENTRY TO AVERAGE G.
034717,000031: 
034718,000032:                                                                                                  #      IN THE READACCS SECTION, THE ACCELEROMETERS ARE READ BY THE PIPASR SUBROUTINE.  IF THE AVERAGE G FLAG IS
034719,000033:                                                                                                  #  SET, READACCS IS CALLED TO RECYCLE IN TWO SECONDS.  IF THE AVERAGE G FLAG IS DOWN, AVERAGE G EXIT (AVEGEXIT)
034720,000034:                                                                                                  #  IS SET FOR THE FINAL PASS, AND READACCS IS NOT CALLED AGAIN.  IN EITHER CASE, THE SERVICER JOB IS ESTABLISHED.
034721,000035: 
034722,000036:                                                                                                  #      THE SERVICER ROUTINE CHECKS FOR RUNAWAY PIPS (DELV GREATER THAN 3200 PULSES/SEC FOR 2 SEC) AND SENDS
034723,000037:                                                                                                  #  ALARM CODE 205 IF BAD PIP IS FOUND.  PIPS ARE COMPENSATED IN 1/PIPA SUBROUTINE, MASS IS UPDATED BY MASSMON
034724,000038:                                                                                                  #  SUBROUTINE, AND  AVERAGEG DOES THE NAVIGATION.  THE AVERAGE G ROUTINE UPDATES RN, VN, AND GDT/2 VECTORS, USING
034725,000039:                                                                                                  #  THE SUBROUTINE CALCRVG.  CONTROL IS THEN TRANSFERRED TO THE MONITOR SPECIFIED BY DVSELECT.  DVSELECT IS SET BY
034726,000040:                                                                                                  #  THE BOOST PHASE TO BOOSTMON, AND BY THE ENGINEON ROUTINE TO THE PGNCSMON.  USERS STARTING SERVICER BEFORE THE
034727,000041:                                                                                                  #  ENGINE IS ON SHOULD INSURE BYPASSING DVMON INITIALLY BY SETTING DVSELECT TO THE GENADR OF AVERAGE G.
034728,000042: 
034729,000043:                                                                                                  #      THE BOOST MONITOR (BOOSTMON) CHECKS DELV AGAINST THRSHLD+, THE THRESHOLD ACCELERATION FOR THE BOOST PHASE.
034730,000044:                                                                                                  #  IF DELV IS BELOW THRESHOLD, INDICATING TERMINATION OF BOOST, MISSION PHASE 6 IS SCHEDULED AS A JOB, DVSELECT
034731,000045:                                                                                                  #  IS ALTERED TO BYPASS BOOSTMON AND PROCEED DIRECTLY TO AVERAGE G.
034732,000046: 
034733,000047:                                                                                                  #      THE PGNCS MONITOR (PGNCSMON) COMPARES ACTUAL THRUST TO THE THRESHOLD VALUE FOR THE +X ACCELERATION
034734,000048:                                                                                                  #  (100 CM/SEC).  IF THRUST IS FOUND BELOW THIS VALUE FOR 3 CYCLES (I.E., 4 TO 6 SECONDS) AS DETERMINED BY DVCNTR,
034735,000049:                                                                                                  #  AND THE ENGINE FLAG IS REMOVED, (INDICATING THAT THE ENGINE HAS BEEN TURNED OFF) CONTROL IS BRANCHED TO THE
034736,000050:                                                                                                  #  LOCATION SPECIFIED BY DVMNEXIT, PREVIOUSLY SET BY THE USERS PROGRAM.  DVSELECT IS ALTERED TO BYPASS PNGCSMON IN
034737,000051:                                                                                                  #  SUBSEQUENT PASSES.  IF, HOWEVER, THE ENGINE FLAG IS STILL ON AT THIS TIME, CONTROL IS TRANSFERRED TO ENGNFAIL.
034738,000052:                                                                                                  #  THIS SENDS ALARM CODE 1405 AND KILLS THAT MISSION PHASE.  ANY TIME THE THRUST IS LESS THAN THE THRESHOLD VALUE,
034739,000053:                                                                                                  #  THE STEERING IS DETACHED (NO EXIT VIA AVEGEXIT) AND EXIT IS MADE THRU SERVEXIT.
034740,000054: 
034741,000055:                                                                                                  #      THE NORMAL EXIT IS THRU AVEGEXIT, WHICH MUST HAVE BEEN SET BY THE USER. THE FINAL EXIT, SET INTO AVEGEXIT BY
034742,000056:                                                                                                  #  READACCS WHEN IT  FINDS THE AVERAGE G FLAG DOWN, SETS UP FREE FALL GYRO COMPENSATION, SETS THE DRIFT FLAG ON,
034743,000057:                                                                                                  #  PERFORMS AVETOMID ROUTINE, AND TRANSFERS CONTROL TO P00H, THUS CLEARING  ALL ACTIVITY UNTIL A NEW MISSION
034744,000058:                                                                                                  #  PHASE IS DUE.
034745,000059: 
034746,000060:                                                                                                  #  ***** WARNING TO USERS *****
034747,000061: 
034748,000062:                                                                                                  #  THE USER MUST SET DVMNEXIT TO THE 2CADR OF A JOB TO BE PERFORMED WHEN ENGINE SHUTDOWN IS DETECTED BY SERVICER.
034749,000063:                                                                                                  #  IN GENERAL, THE AVERAGE G FLAG WILL BE TURNED OFF BY THE USER AT THAT TIME, ALLOWING JUST ONE MORE PASS THROUGH
034750,000064:                                                                                                  #  AVERAGE G.  ALL ACTIVITY OF THE USERS MISSION PHASE MUST HAVE BEEN COMPLETED BEFORE THIS LAST PASS THROUGH
034751,000065:                                                                                                  #  AVERAGE G, DUE TO THE PERFORMANCE OF P00H AS DESCRIBED ABOVE.
034752,000066: 
034753,000067:                                                                                                  #      AVGEXIT MUST BE SET BY THE USER TO THE 2CADR OF THE JOB (E.G., STEERING) TO BE PERFORMED AFTER EACH PASS
034754,000068:                                                                                                  #  THROUGH AVERAGE G.  IF NO OTHER JOB IS TO BE DONE, AVEGEXIT SHOULD BE SET TO SERVEXIT.
