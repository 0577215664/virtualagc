Source Code

These source-code files were transcribed from scans made from Don Eyles's personal copy of BURST120 (SUNBURST 120). They were scanned at archive.org's Boston facility, and the scanning was sponsored by Mike Stewart. The code was transcribed from these scans by a team of volunteers who are referenced in the program comments. Comments from the original source code are in ALL-CAPS, whereas comments added later in transcription are in Mixed-Case. In some cases, where similar code blocks exist in previously-transcribed AGC programs (primarily Luminary 99, from Apollo 11) those code blocks were used as a starting point and then corrected to agree with the BURST120 scans. The full scans are available at the Virtual AGC project's collection at archive.org, while more-convenient reduced-size (but reduced-quality) images are available at the main Virtual AGC website. Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository. Notations on the program listing read, in part:

	YUL SYSTEM FOR AGC: REVISION 0 OF PROGRAM BURST120 
	BY NASA 2021106-031 DEC 7, 1967

	THIS LISTING IS A COPY OF A VERSION OF THE PROGRAM INTENDED 
	FOR USE IN THE ON-BOARD PRIMARY GUIDANCE COMPUTER IN THE 
	UNMANNED FLIGHT OF APOLLO LUNAR MODULE 1 --- THE AS206 MISSION.
Note that the date is the date of the printout, not the date of the program revision.

028487,000183:                                                                                                  #  PROGRAM DESCRIPTION- MATRXJOB                                           DATE: 18 JAN 1967
028488,000184:                                                                                                  #  MOD NO: 2                                                               LOG SECTION- MP 2 GRR + BOOST MONITOR
028489,000185:                                                                                                  #  MOD BY: MILLER, LICKLY, KERNAN                                          ASSEMBLY: SUNBURST REVISION 79

028491,000187:                                                                                                  #  FUNCTIONAL DESCRIPTION
028492,000188: 
028493,000189:                                                                                                  #           THIS PROGRAM CONSTRUCTS THE MATRIX WHICH RELATES THE STABLE MEMBER INERTIAL FRAME TO THE REFERENCE
028494,000190:                                                                                                  #  FRAME (Z NORTH, X ALONG THE VERNAL EQUINOX.)
028495,000191: 
028496,000192:                                                                                                  #           TWO INTERMEDIATE COORDINATE SYSTEMS ARE USED: A LOCAL, EARTH FIXED, VERTICAL, SOUTH, EAST SYSTEM AND AN
028497,000193:                                                                                                  #  EARTH REFERENCE X, Y, Z SYSTEM.  IN THIS LATTER SYSTEM, THE Z AXIS IS THE EARTH'S ROTATION AXIS, THE X AXIS IS
028498,000194:                                                                                                  #  NORMAL TO Z IN THE PLANE OF Z AND THE LOCAL VERTICAL, POSITIVE IN THE DIRECTION OF V.  Y IS Z CROSS X.
028499,000195: 
028500,000196:                                                                                                  #           THE FIRST COMPUTATION IS OF AZGR, THE ANGLE BETWEEN THE REFERENCE INERTIAL AND EARTH REFERENCE X-Z
028501,000197:                                                                                                  #  PLANES (THE Z AXES ARE COINCIDENT).  AZGR IS COMPUTED BY CONVERTING THE TIME FROM THE BEGINNING OF THE EPHEMERIS
028502,000198:                                                                                                  #  YEAR TO RELEASE (TEPHEM + TPRELTER) TO REVOLUTIONS (DAYS).  THE WHOLE REVS ARE DISCARDED AND THE INITIAL ANGLE
028503,000199:                                                                                                  #  (AZ0) BETWEEN THE GREENWICH MERIDIAN AND THE REFERENCE X-Z PLANES IS ADDED.  ADDING THE LONGITUDE YIELDS AZGR.
028504,000200: 
028505,000201:                                                                                                  #           THE FOLLOWING COMPUTATIONS ARE THEN PERFORMED.
028506,000202: 
028507,000203:                                                                                                  #  LOCAL VERTICAL(ER) = COS(LATITUDE), 0, SIN(LATITUDE)  IN EARTH REFERENCE
028508,000204: 
028509,000205:                                                                                                  #  LOCAL VERTICAL(IR) = COS(LAT)COS(AZGR), COS(LAT)SIN(AZGR), SIN(LAT)  IN INERTIAL REFERENCE
028510,000206: 
028511,000207:                                                                                                  #  LOCAL EAST(IR) = UNIT(NXV) = -SIN(AZGR), COS(AZGR), 0  IN INERTIAL REFERENCE
028512,000208: 
028513,000209:                                                                                                  #  LOCAL SOUTH(IR) = E(IR) X V(IR)
028514,000210: 
028515,000211:                                                                                                  #           THE RELATIONSHIP OF THE STABLE MEMBER AXES TO THE V. S, F AXES IS GIVEN BY ZSMAZ, THE ANGLE FROM NORTH
028516,000212:                                                                                                  #  TO ZSM, AND TILT, THE ANGLE ABOUT ZSM FROM VERTICAL TO XSM.
028517,000213: 
028518,000214:                                                                                                  #  ZSM(IR) = EAST(IR)SIN(ZSMAZ) - SOUTH(IR)COS(ZSMAZ)
028519,000215: 
028520,000216:                                                                                                  #  YSM(IR) = (ZSM(IR) X V(IR))COS(TILT) - V(IR)SIN(TILT)
028521,000217: 
028522,000218:                                                                                                  #  XSM(IR) = YSM(IR) X ZSM(IR)
028523,000219: 
028524,000220:                                                                                                  #           THESE THREE HALF-UNIT VECTORS, XSM(IR), YSM(IR), AND ZSM(IR) ARE THE SM AXES EXPRESSED IN INERTIAL
028525,000221:                                                                                                  #  REFERENCE COORDINATES AND THEY FORM REFSMMAT, THE REFERENCE TO STABLE MEMBER MATRIX.
028526,000222: 
028527,000223:                                                                                                  #  THE INPUT (PRELAUNCH ERASABLE LOAD) REQUIREMENTS ARE:
028528,000224: 
028529,000225:                                                                                                  #  1) TEPHEM       THE TRIPLE PRECISION TIME IN CENTISECONDS FROM MIDNIGHT JULY 1, OF THE EPHEMERIS YEAR TO
028530,000226:                                                                                                  #  MIDNIGHT OF THE LAUNCH DAY (SIDEREAL CONVERTED TO MEAN SOLAR.)
028531,000227: 
028532,000228:                                                                                                  #     IT IS ASSUMED THAT DURING THE LAUNCH COUNTDOWN THE LGC CLOCK (TIME2, TIME1) WILL BE ALIGNED TO REFLECT A
028533,000229:                                                                                                  #  ZERO VALUE AT MIDNIGHT OF THE LAUNCH DAY.  IF NOT, THE DIFFERENCE MUST BE ADDED TO TEPHEM.
028534,000230: 
028535,000231:                                                                                                  #  2) TILT         THE ROTATION OF XSM ABOUT ZSM (RIGHT HAND RULE) FROM VERTICAL IN REVOLUTIONS.
028536,000232:
