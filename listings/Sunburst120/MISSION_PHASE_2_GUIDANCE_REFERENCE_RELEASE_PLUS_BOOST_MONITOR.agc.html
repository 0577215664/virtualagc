Source Code

These source-code files were transcribed from scans made from Don Eyles's personal copy of BURST120 (SUNBURST 120). They were scanned at archive.org's Boston facility, and the scanning was sponsored by Mike Stewart. The code was transcribed from these scans by a team of volunteers who are referenced in the program comments. Comments from the original source code are in ALL-CAPS, whereas comments added later in transcription are in Mixed-Case. In some cases, where similar code blocks exist in previously-transcribed AGC programs (primarily Luminary 99, from Apollo 11) those code blocks were used as a starting point and then corrected to agree with the BURST120 scans. The full scans are available at the Virtual AGC project's collection at archive.org, while more-convenient reduced-size (but reduced-quality) images are available at the main Virtual AGC website. Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository. Notations on the program listing read, in part:

	YUL SYSTEM FOR AGC: REVISION 0 OF PROGRAM BURST120 
	BY NASA 2021106-031 DEC 7, 1967

	THIS LISTING IS A COPY OF A VERSION OF THE PROGRAM INTENDED 
	FOR USE IN THE ON-BOARD PRIMARY GUIDANCE COMPUTER IN THE 
	UNMANNED FLIGHT OF APOLLO LUNAR MODULE 1 --- THE AS206 MISSION.
Note that the date is the date of the printout, not the date of the program revision.

028657,000185:                                                                                                  #  PROGRAM DESCRIPTION- MATRXJOB                                           DATE: 18 JAN 1967
028658,000186:                                                                                                  #  MOD NO: 2                                                               LOG SECTION- MP 2 GRR + BOOST MONITOR
028659,000187:                                                                                                  #  MOD BY: MILLER, LICKLY, KERNAN                                          ASSEMBLY: SUNBURST REVISION 79

028661,000189:                                                                                                  #  FUNCTIONAL DESCRIPTION
028662,000190: 
028663,000191:                                                                                                  #           THIS PROGRAM CONSTRUCTS THE MATRIX WHICH RELATES THE STABLE MEMBER INERTIAL FRAME TO THE REFERENCE
028664,000192:                                                                                                  #  FRAME (Z NORTH, X ALONG THE VERNAL EQUINOX.)
028665,000193: 
028666,000194:                                                                                                  #           TWO INTERMEDIATE COORDINATE SYSTEMS ARE USED: A LOCAL, EARTH FIXED, VERTICAL, SOUTH, EAST SYSTEM AND AN
028667,000195:                                                                                                  #  EARTH REFERENCE X, Y, Z SYSTEM.  IN THIS LATTER SYSTEM, THE Z AXIS IS THE EARTH'S ROTATION AXIS, THE X AXIS IS
028668,000196:                                                                                                  #  NORMAL TO Z IN THE PLANE OF Z AND THE LOCAL VERTICAL, POSITIVE IN THE DIRECTION OF V.  Y IS Z CROSS X.
028669,000197: 
028670,000198:                                                                                                  #           THE FIRST COMPUTATION IS OF AZGR, THE ANGLE BETWEEN THE REFERENCE INERTIAL AND EARTH REFERENCE X-Z
028671,000199:                                                                                                  #  PLANES (THE Z AXES ARE COINCIDENT).  AZGR IS COMPUTED BY CONVERTING THE TIME FROM THE BEGINNING OF THE EPHEMERIS
028672,000200:                                                                                                  #  YEAR TO RELEASE (TEPHEM + TPRELTER) TO REVOLUTIONS (DAYS).  THE WHOLE REVS ARE DISCARDED AND THE INITIAL ANGLE
028673,000201:                                                                                                  #  (AZ0) BETWEEN THE GREENWICH MERIDIAN AND THE REFERENCE X-Z PLANES IS ADDED.  ADDING THE LONGITUDE YIELDS AZGR.
028674,000202: 
028675,000203:                                                                                                  #           THE FOLLOWING COMPUTATIONS ARE THEN PERFORMED.
028676,000204: 
028677,000205:                                                                                                  #  LOCAL VERTICAL(ER) = COS(LATITUDE), 0, SIN(LATITUDE)  IN EARTH REFERENCE
028678,000206: 
028679,000207:                                                                                                  #  LOCAL VERTICAL(IR) = COS(LAT)COS(AZGR), COS(LAT)SIN(AZGR), SIN(LAT)  IN INERTIAL REFERENCE
028680,000208: 
028681,000209:                                                                                                  #  LOCAL EAST(IR) = UNIT(NXV) = -SIN(AZGR), COS(AZGR), 0  IN INERTIAL REFERENCE
028682,000210: 
028683,000211:                                                                                                  #  LOCAL SOUTH(IR) = E(IR) X V(IR)
028684,000212: 
028685,000213:                                                                                                  #           THE RELATIONSHIP OF THE STABLE MEMBER AXES TO THE V. S, F AXES IS GIVEN BY ZSMAZ, THE ANGLE FROM NORTH
028686,000214:                                                                                                  #  TO ZSM, AND TILT, THE ANGLE ABOUT ZSM FROM VERTICAL TO XSM.
028687,000215: 
028688,000216:                                                                                                  #  ZSM(IR) = EAST(IR)SIN(ZSMAZ) - SOUTH(IR)COS(ZSMAZ)
028689,000217: 
028690,000218:                                                                                                  #  YSM(IR) = (ZSM(IR) X V(IR))COS(TILT) - V(IR)SIN(TILT)
028691,000219: 
028692,000220:                                                                                                  #  XSM(IR) = YSM(IR) X ZSM(IR)
028693,000221: 
028694,000222:                                                                                                  #           THESE THREE HALF-UNIT VECTORS, XSM(IR), YSM(IR), AND ZSM(IR) ARE THE SM AXES EXPRESSED IN INERTIAL
028695,000223:                                                                                                  #  REFERENCE COORDINATES AND THEY FORM REFSMMAT, THE REFERENCE TO STABLE MEMBER MATRIX.
028696,000224: 
028697,000225:                                                                                                  #  THE INPUT (PRELAUNCH ERASABLE LOAD) REQUIREMENTS ARE:
028698,000226: 
028699,000227:                                                                                                  #  1) TEPHEM       THE TRIPLE PRECISION TIME IN CENTISECONDS FROM MIDNIGHT JULY 1, OF THE EPHEMERIS YEAR TO
028700,000228:                                                                                                  #  MIDNIGHT OF THE LAUNCH DAY (SIDEREAL CONVERTED TO MEAN SOLAR.)
028701,000229: 
028702,000230:                                                                                                  #     IT IS ASSUMED THAT DURING THE LAUNCH COUNTDOWN THE LGC CLOCK (TIME2, TIME1) WILL BE ALIGNED TO REFLECT A
028703,000231:                                                                                                  #  ZERO VALUE AT MIDNIGHT OF THE LAUNCH DAY.  IF NOT, THE DIFFERENCE MUST BE ADDED TO TEPHEM.
028704,000232: 
028705,000233:                                                                                                  #  2) TILT         THE ROTATION OF XSM ABOUT ZSM (RIGHT HAND RULE) FROM VERTICAL IN REVOLUTIONS.
028706,000234:
