Source Code

These source-code files were transcribed from scans made from Don Eyles's personal copy of BURST120 (SUNBURST 120). They were scanned at archive.org's Boston facility, and the scanning was sponsored by Mike Stewart. The code was transcribed from these scans by a team of volunteers who are referenced in the program comments. Comments from the original source code are in ALL-CAPS, whereas comments added later in transcription are in Mixed-Case. In some cases, where similar code blocks exist in previously-transcribed AGC programs (primarily Luminary 99, from Apollo 11) those code blocks were used as a starting point and then corrected to agree with the BURST120 scans. The full scans are available at the Virtual AGC project's collection at archive.org, while more-convenient reduced-size (but reduced-quality) images are available at the main Virtual AGC website. Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository. Notations on the program listing read, in part:

	YUL SYSTEM FOR AGC: REVISION 0 OF PROGRAM BURST120 
	BY NASA 2021106-031 DEC 7, 1967

	THIS LISTING IS A COPY OF A VERSION OF THE PROGRAM INTENDED 
	FOR USE IN THE ON-BOARD PRIMARY GUIDANCE COMPUTER IN THE 
	UNMANNED FLIGHT OF APOLLO LUNAR MODULE 1 --- THE AS206 MISSION.
Note that the date is the date of the printout, not the date of the program revision.

013466,000022:                                                                                                  #  ASSEMBLY- REV 12 OF SUNBURST
013467,000023:                                                                                                  #  FUNCTIONAL DESCRIPTION- THIS ROUTINE IS INITIATED BY TELEMETRY END
013468,000024:                                                                                                  #           PULSE FROM THE DOWNLINK TELEMETRY CONVERTER. THIS PULSE OCCURS
013469,000025:                                                                                                  #           AT 50 TIMES PER SEC(EVERY 20 MS) THEREFORE DODOWNTM IS
013470,000026:                                                                                                  #           EXECUTED AT THESE RATES. THIS ROUTINE SELECTS THE APPROPRIATE
013471,000027:                                                                                                  #           AGC DATA TO BE TRANSMITTED DOWNLINK AND LOADS IT INTO OUTPUT
013472,000028:                                                                                                  #           CHANNELS 34 AND 35. THE INFORMATION IS THEN GATED OUT FROM THE
013473,000029:                                                                                                  #           LGC IN SERIAL FASHION.
013474,000030:                                                                                                  #           THIS PROGRAM IS CODED FOR A 2 SECOND DOWNLIST. SINCE DOWNRUPTS
013475,000031:                                                                                                  #           OCCUR EVERY 20 MS AND 2 AGC COMPUTER WORDS CAN BE PLACED IN
013476,000032:                                                                                                  #           CHANNELS 34 AND 35 DURING EACH DOWNRUPT THE PROGRAM IS CAPABLE
013477,000033:                                                                                                  #           OF SENDING 200 AGC WORDS EVERY 2 SECONDS.
013478,000034:                                                                                                  #  CALLING SEQUENCE- NONE
013479,000035:                                                                                                  #           PROGRAM IS ENTERED VIA TCF DODOWNTM WHICH IS EXECUTED AS A
013480,000036:                                                                                                  #           RESULT OF A DOWNRUPT. CONTROL IS RETURNED VIA TCF RESUME WHICH
013481,000037:                                                                                                  #           IN EFFECT IS A RESUME.
013482,000038:                                                                                                  #  SUBROUTINES CALLED- NONE
013483,000039:                                                                                                  #  NORMAL EXIT MODE- TCF RESUME
013484,000040:                                                                                                  #  ALARM OR ABORT EXIT MODE- NONE
013485,000041:                                                                                                  #  RESTART PROTECTION- NONE.
013486,000042:                                                                                                  #           EXCEPT THAT WHEN A RESTART DOES OCCUR :STARTSUB: WILL
013487,000043:                                                                                                  #           INITIALIZE THE DOWNLIST POINTER TO THE BEGINNING OF THE NOMINAL
013488,000044:                                                                                                  #           DOWNLIST. THIS HAS THE EFFECT OF IGNORING THE REMAINDER OF THE
013489,000045:                                                                                                  #           DOWNLIST WHICH THE DOWN-TELEMETRY PROGRAM WAS WORKING ON WHEN
013490,000046:                                                                                                  #           THE RESTART OCCURRED.
013491,000047:                                                                                                  #  OUTPUT-  EVERY 2 SECONDS 100 DOUBLE PRECISION WORDS(I.E., 200 LGC
013492,000048:                                                                                                  #           COMPUTER WORDS) ARE TRANSMITTED VIA DOWNLINK.
013493,000049:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED- NONE
013494,000050:                                                                                                  #           DNTMGOTO,DNLSTADR AND EBANK ARE INITIALIZED IN STARTSUB(FRESH
013495,000051:                                                                                                  #           START AND RESTART)
013496,000052:                                                                                                  #  DEBRIS (ERASABLE LOCATIONS DESTROYED BY THIS PROGRAM)-
013497,000053:                                                                                                  #           LDATALST,ITEMP1,ITEMP2,DNTMBUFF TO DNTMBUFF + 21D,TMINDEX,DNQ
