Source Code

These source-code files were transcribed from scans made from Don Eyles's personal copy of BURST120 (SUNBURST 120). They were scanned at archive.org's Boston facility, and the scanning was sponsored by Mike Stewart. The code was transcribed from these scans by a team of volunteers who are referenced in the program comments. Comments from the original source code are in ALL-CAPS, whereas comments added later in transcription are in Mixed-Case. In some cases, where similar code blocks exist in previously-transcribed AGC programs (primarily Luminary 99, from Apollo 11) those code blocks were used as a starting point and then corrected to agree with the BURST120 scans. The full scans are available at the Virtual AGC project's collection at archive.org, while more-convenient reduced-size (but reduced-quality) images are available at the main Virtual AGC website. Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository. Notations on the program listing read, in part:

	YUL SYSTEM FOR AGC: REVISION 0 OF PROGRAM BURST120 
	BY NASA 2021106-031 DEC 7, 1967

	THIS LISTING IS A COPY OF A VERSION OF THE PROGRAM INTENDED 
	FOR USE IN THE ON-BOARD PRIMARY GUIDANCE COMPUTER IN THE 
	UNMANNED FLIGHT OF APOLLO LUNAR MODULE 1 --- THE AS206 MISSION.
Note that the date is the date of the printout, not the date of the program revision.

002592,000018:                                                                                                  #  RESTART TABLES
002593,000019:                                                                                                  #   DO NOT REMOVE FROM THE BEGINNING OF THIS BANK
002594,000020:                                                                                                  #  --------------
002595,000021: 
002596,000022:                                                                                                  #  THERE ARE TWO FORMS OF RESTART TABLES FOR EACH GROUP.  THEY ARE KNOWN AS THE EVEN RESTART TABLES AND THE ODD
002597,000023:                                                                                                  #  RESTART TABLES.  THE ODD TABLES HAVE ONLY ONE ENTRY OF THREE LOCATIONS WHILE THE EVEN TABLES HAVE TWO ENTRIES
002598,000024:                                                                                                  #  EACH USING THREE LOCATIONS.  THE INFORMATION AS TO WHETHER IT IS A JOB, WAITLIST, OR A LONGCALL IS GIVEN BY THE
002599,000025:                                                                                                  #  WAY THINGS ARE PUT IN TO THE TABLES.
002600,000026:                                                                                                  #       A JOB HAS ITS PRIORITY STORED IN A PRDTTAB OF THE CORRECT PHASE SPOT WITH ITS 2CADR IN THE CADRTAB. FOR
002601,000027:                                                                                                  #  EXAMPLE,
002602,000028: 
002603,000029:                                                                                                  #                5.7SPOT         OCT     23000
002604,000030:                                                                                                  #                                2CADR   SOMEJOB
002605,000031: 
002606,000032:                                                                                                  #  A RESTART OF GROUP 5 WITH PHASE SEVEN WOULD THEN CAUSE SOMEJOB TO BE RESTARTED AS A FINDVAC WITH PRIORITY 23.
002607,000033: 
002608,000034:                                                                                                  #  A LONGCALL HAS ITS GENADR OF ITS 2CADR STORED NEGATIVELY AND ITS BBCON STORED POSITIVELY.  IN ITS PRDTTAB IS
002609,000035:                                                                                                  #  PLACED THE LOCATION OF A DP REGISTER THAT CONTAINS THE DELTA TIME THAT LONGCALL HAD BEEN ORIGINALLY STARTED
002610,000036:                                                                                                  #  WITH.  EXAMPLE,
002611,000037: 
002612,000038:                                                                                                  #                3.6SPOT         GENADR  DELTAT
002613,000039:                                                                                                  #                               -GENADR  LONGTASK
002614,000040:                                                                                                  #                                BBCON   LONGTASK
002615,000041: 
002616,000042:                                                                                                  #                                OCT     31000
002617,000043:                                                                                                  #                                2CADR   JOBAGAIN
002618,000044: 
002619,000045:                                                                                                  #  THIS WOULD START UP LONGTASK AT THE APPROPRIATE TIME, OR IMMEDIATELY IF THE TIME HAD ALREADY PASSED.  IT SHOULD
002620,000046:                                                                                                  #  BE NOTED THAT IF DELTAT IS IN A SWITCHED E BANK, THIS INFORMATION SHOULD BE IN THE BBCON OF THE 2CADR OF THE
002621,000047:                                                                                                  #  TASK.  FROM ABOVE, WE SEE THAT THE SECOND PART O THIS PHASE WOULD BE STARTED AS A JOB WITH A PRIORITY OF 31.
002622,000048: 
002623,000049:                                                                                                  #  WAITLIST CALLS ARE IDENTIFIED BY THE FCT THAT THEIR 2CADR IS STORED NEGATIVELY.  IF PRDTTAB OF THE PHASE SPOT
002624,000050:                                                                                                  #  IS POSITIVE, THEN IT CONTAINS THE DELTA TIME, IF PRDTTAB IS NEGATIVE THEN IT IS THE -GENADR OF AN ERASABLE
002625,000051:                                                                                                  #  LOCATION CONTAINING THE DELTA TIME, THAT IS, THE TIME IS STORED INDIRECTLY.  IT SHOULD BE NOTED AS ABOVE, THAT
002626,000052:                                                                                                  #  IF THE TIME IS STORED INDIRECTLY, THE BBCON MUST CONTAIN THE NECESSARY E BANK INFORMATION IF APPLICABLE.  WITH
002627,000053:                                                                                                  #  WAITLIST WE HAVE ONE FURTHER OPTION, IF -0 IS STORED IN PRDTTAB, IT WILL CAUSE AN IMMEDIATE RESTART OF THE
002628,000054:                                                                                                  #  TASK.  EXAMPLES,
002629,000055: 
002630,000056:                                                                                                  #                                OCT     77777           # THIS WILL CAUSE AN IMMEDIATE RESTART
002631,000057:                                                                                                  #                                -2CADR  ATASK           # OF THE TASK :ATASK:
002632,000058:                                                                                                  #        
002633,000059:                                                                                                  #                                DEC     200             # IF THE TIME OF THE 2 SECONDS SINCE DUMMY
002634,000060:                                                                                                  #                                -2CADR  DUMMY           # WAS PUT ON THE WAITLIST IS UP, IT WILL BEGIN
002635,000061:                                                                                                  #                                                        # IN 10 MS, OTHERWISE IT WILL BEGIN WHEN
002636,000062:                                                                                                  #                                                        # IT NORMALLY WOULD HAVE BEGUN.
002637,000063: 
002638,000064:                                                                                                  #                                -GENADR DTIME           # WHERE DTIME CONTAINS THE DELTA TIME
002639,000065:                                                                                                  #                                -2CADR  TASKTASK        # OTHERWISE THIS IS AS ABOVE
