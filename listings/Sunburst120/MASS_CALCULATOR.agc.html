Source Code

These source-code files were transcribed from scans made from Don Eyles's personal copy of BURST120 (SUNBURST 120). They were scanned at archive.org's Boston facility, and the scanning was sponsored by Mike Stewart. The code was transcribed from these scans by a team of volunteers who are referenced in the program comments. Comments from the original source code are in ALL-CAPS, whereas comments added later in transcription are in Mixed-Case. In some cases, where similar code blocks exist in previously-transcribed AGC programs (primarily Luminary 99, from Apollo 11) those code blocks were used as a starting point and then corrected to agree with the BURST120 scans. The full scans are available at the Virtual AGC project's collection at archive.org, while more-convenient reduced-size (but reduced-quality) images are available at the main Virtual AGC website. Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository. Notations on the program listing read, in part:

	YUL SYSTEM FOR AGC: REVISION 0 OF PROGRAM BURST120 
	BY NASA 2021106-031 DEC 7, 1967

	THIS LISTING IS A COPY OF A VERSION OF THE PROGRAM INTENDED 
	FOR USE IN THE ON-BOARD PRIMARY GUIDANCE COMPUTER IN THE 
	UNMANNED FLIGHT OF APOLLO LUNAR MODULE 1 --- THE AS206 MISSION.
Note that the date is the date of the printout, not the date of the program revision.

035804,000016: 
035805,000017:                                                                                                  #      THE MASS UPDATE ROUTINE EXTENDS FROM APPROXIMATELY HERE TO THE NEXT
035806,000018:                                                                                                  #  ROW OF ASTERISKS.   COMPUTATIONS ARE AS FOLLOWS:-
035807,000019:                                                                                                  #      (1) /AF/ = MAGNITUDE DV/DT, SCALED IN UNITS OF 2(-5) M/CS/CS.
035808,000020:                                                                                                  #      (2) PREFORCE = /AF/ MASS, APPROXIMATE PRESENT THRUST, SCALED AT
035809,000021:                                                                                                  #          SOME 3.3 POUNDS PER BIT; PREFORCE LATER BECOMES PREFORCE/2FMAX.
035810,000022:                                                                                                  #      (3) NOMINAL EXHAUST VELOCITY, VEXNOM = VEXSTEP + (PREFORCE-FSTEP)
035811,000023:                                                                                                  #          SLOPE, WHERE VEXSTEP, FSTEP, AND SLOPE ARE SELECTED FROM THE
035812,000024:                                                                                                  #          'VEXTABLE' BY THE ROUTINE BEGINNING AT VEXFINDD; THE FSTEPS
035813,000025:                                                                                                  #          AND VXSTEPS DEFINE CERTAIN DISCRETE POINTS ON THE CURVE OF
035814,000026:                                                                                                  #          EXHAUST VELOCITY VERSUS THRUST; THE ASSOCIATED SLOPE IS ALWAYS
035815,000027:                                                                                                  #          THAT FOR THE SEGMENT JUST PRECEEDING THOSE POINTS.   (UNITS
035816,000028:                                                                                                  #          FOR VEXNOM, AS FOR NEGVEX, ARE 2(6) M/CS.)
035817,000029:                                                                                                  #      (4) AREARATE, A MODERATELY ABSTRUSE (NOT TO SAY ABSURD) FUNCTION
035818,000030:                                                                                                  #          OF DELAREA (EROSION) AND PREFORCE.   (THE AREA IN QUESTION IS
035819,000031:                                                                                                  #          THAT OF THE THROAT OF THE ENGINE.)
035820,000032:                                                                                                  #      (5) DELAREA = DELAREA + AREARATE DELTAT, SCALED IN UNITS OF 2(7) %
035821,000033:                                                                                                  #      (6) NEGVEX = -(VEXNOM - CVEX5 DELAREA).
035822,000034:                                                                                                  #      (7) MASS = MASS + (/DELV//NEGVEX) MASS, SCALED IN UNITS OF 2(15)
035823,000035:                                                                                                  #          KILOGRAMS.
035824,000036: 
035825,000037:                                                                                                  #      THE FOLLOWING INITIALIZATIONS MUST BE MADE BEFORE LAUNCH:
035826,000038:                                                                                                  #  C(MASS) = 0, C(LEMMASS1) = SCALED MASS OF LEM CUM DPS, C(LEMMASS2) =
035827,000039:                                                                                                  #  SCALED MASS OF LEM WITHOUT DPS.   UPON LEM-SIVB SEPARATION MASS MUST
035828,000040:                                                                                                  #  BE REINITIALIZED USING LEMMASS1 AND DELAREA MUST BE SET TO DP ZERO.
035829,000041:                                                                                                  #  LATER, WHEN THE DPS PARTS COMPANY, MASS MUST BE INITIALIZED STILL
035830,000042:                                                                                                  #  AGAIN USING LEMMASS2.
035831,000043: 
035832,000044:                                                                                                  #      THE FOLLOWING IS AN INELEGANT SUPERFLUITY:-   INPUTS: MASS, DELAREA,
035833,000045:                                                                                                  #  DELV, DELTAT, EMANATIONS FROM THE VEXTABLE, A TABLESPOON OF SALT AND
035834,000046:                                                                                                  #  SEVEN BYZANTINE DODO EGGS.   OUTPUTS: /AF/ AND MASS.   DEBRIS: PREFORCE,
035835,000047:                                                                                                  #  NEGVEX, AND AREARATE (TO LEARN MORE ABOUT THESE GEMS, SEE ABOVE).
035836,000048:                                                                                                  #  SUBROUTINES CALLED: MASSMULT (A PART OF THROTTLE CONTROL) AND THE
035837,000049:                                                                                                  #  INTERPRETER.   ERASABLES: BANK 5 AT PRESENT.   EYLES WROTE THIS STUFF;
