Source Code

These source-code files derive from a printout of Luminary 210 (Apollo 15-17 Lunar Module guidance computer program), from the personal library of original AGC developer Don Eyles, digitally photographed at archive.org, financially sponsored by Jim Lawton, and transcribed to source code by a team of volunteers. This colorized, syntax-highlighted form was created by assembling that transcribed source code. Note that the full page images are available on the Virtual AGC project page at archive.org, while reduced-size images are presented at the VirtualAGC project website. Report or fix any transcription errors at the Virtual AGC project code repository.

Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 210 OF AGC PROGRAM LUMINARY 
	BY NASA 2021112-161 17:11 MAR. 19,1971
Note that the date is the date of the printout, not the date of the program revision.

003941,000037:                                                                                                  #  CONVENTIONS AND NOTATIONS UTILIZED FOR ERASABLE ASSIGNMENTS.
003942,000038: 
003943,000039:                                                                                                  #           EQUALS IS USED IN TWO WAYS.  IT IS OFTEN USED TO CHAIN A GROUP
003944,000040:                                                                                                  #                  OF ASSIGNMENTS SO THAT THE GROUP MAY BE MOVED WITH THE
003945,000041:                                                                                                  #                  CHANGING OF ONLY ONE CARD.  EXAMPLE.
003946,000042: 
003947,000043:                                                                                                  #                                          X        EQUALS  START
003948,000044:                                                                                                  #                                          Y        EQUALS  X       +SIZE.X
003949,000045:                                                                                                  #                                          Z        EQUALS  Y       +SIZE.Y
003950,000046: 
003951,000047:                                                                                                  #                  (X, Y, AND Z ARE CONSECUTIVE AND BEGIN AT START.        )
003952,000048:                                                                                                  #                  (SIZE.X AND SIZE.Y ARE THE RESPECTIVE SIZES OF X AND Y,
003953,000049:                                                                                                  #                   USUALLY NUMERIC,  IE. 1, 2, 6, 18D ETC.               )
003954,000050: 
003955,000051:                                                                                                  #           EQUALS  OFTEN IMPLIES THE SHARING OF REGISTERS (DIFFERENT NAMES
003956,000052:                                                                                                  #                   AND DIFFERENT DATA).  EXAMPLE.
003957,000053: 
003958,000054:                                                                                                  #                                           X       EQUALS  Y
003959,000055: 
003960,000056: 
003961,000057:                                                                                                  #           = MEANS THAT MULTIPLE NAMES HAVE BEEN GIVEN TO THE SAME DATA.
003962,000058:                                                                                                  #                  (THIS IS LOGICAL EQUIVALENCE, NOT SHARING)  EXAMPLE.
003963,000059: 
003964,000060:                                                                                                  #                                           X       =       Y
003965,000061: 
003966,000062:                                                                                                  #           THE SIZE AND UTILIZATION OF AN ERASABLE ARE OFTEN INCLUDED IN
003967,000063:                                                                                                  #           THE COMMENTS IN THE FOLLOWING FORM.  M(SIZE)N.
003968,000064: 
003969,000065:                                                                                                  #                  M  REFERS TO THE MOBILITY OF THE ASSIGNMENT.
003970,000066:                                                                                                  #                        B       MEANS THAT THE SYMBOL IS REFERENCED BY BASIC
003971,000067:                                                                                                  #                                INSTRUCTIONS AND THUS IS E-BANK SENSITIVE.
003972,000068:                                                                                                  #                        I       MEANS THAT THE SYMBOL IS REFERENCED ONLY BY
003973,000069:                                                                                                  #                                INTERPRETIVE INSTRUCTIONS, AND IS THUS E-BANK
003974,000070:                                                                                                  #                                INSENSITIVE AND MAY APPEAR IN ANY E-BANK.
003975,000071: 
003976,000072:                                                                                                  #                  SIZE    IS THE NUMBER OF REGISTERS INCLUDED BY THE SYMBOL.
003977,000073: 
003978,000074:                                                                                                  #                  N INDICATES THE NATURE OF PERMANENCE OF THE CONTENTS.
003979,000075:                                                                                                  #                       PL  MEANS THAT THE CONTENTS ARE PAD LOADED.
003980,000076:                                                                                                  #                       DSP MEANS THAT THE REGISTER IS USED FOR A DISPLAY.
003981,000077:                                                                                                  #                       PRM MEANS THAT THE REGISTER IS PERMANENT. IE. IT
003982,000078:                                                                                                  #                           IS USED DURING THE ENTIRE MISSION FOR ONE
003983,000079:                                                                                                  #                           PURPOSE AND CANNOT BE SHARED.
003984,000080:                                                                                                  #                       TMP MEANS THAT THE REGISTER IS USED TEMPORARILY OR
003985,000081:                                                                                                  #                           IS A SCRATCH REGISTER FOR THE ROUTINE TO WHICH
003986,000082:                                                                                                  #                           IT IS ASSIGNED.  THAT IS, IT NEED NOT BE SET
003987,000083:                                                                                                  #                           PRIOR TO INVOCATION OF THE ROUTINE NOR DOES IT
003988,000084:                                                                                                  #                           CONTAIN USEFUL OUTPUT TO ANOTHER ROUTINE.  THUS
003989,000085:
