Source Code

These source-code files derive from a printout of Luminary 210 (Apollo 15-17 Lunar Module guidance computer program), from the personal library of original AGC developer Don Eyles, digitally photographed at archive.org, financially sponsored by Jim Lawton, and transcribed to source code by a team of volunteers. This colorized, syntax-highlighted form was created by assembling that transcribed source code. Note that the full page images are available on the Virtual AGC project page at archive.org, while reduced-size images are presented at the VirtualAGC project website. Report or fix any transcription errors at the Virtual AGC project code repository.

Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 210 OF AGC PROGRAM LUMINARY 
	BY NASA 2021112-161 17:11 MAR. 19,1971
Note that the date is the date of the printout, not the date of the program revision.

055746,000072:                                                                                                  #  PHASCHNG IS THE MAIN WAY OF MAKING PHASE CHANGES FOR RESTARTS.  THERE ARE THREE FORMS OF PHASCHNG, KNOWN AS TYPE
055747,000073:                                                                                                  #  A, TYPE B, AND TYPE C. THEY ARE ALL CALLED AS FOLLOWS, WHERE OCT XXXXX   CONTAINS THE PHASE INFORMATION,
055748,000074: 
055749,000075:                                                                                                  #                                                   TC     PHASCHNG
055750,000076:                                                                                                  #                                                   OCT    XXXXX
055751,000077: 
055752,000078:                                                                                                  #  TYPE A IS CONCERNED WITH FIXED PHASE CHANGES, THAT IS, PHASE INFORMATIONTHAT IS STORED PERMANENTLY.  THESE
055753,000079:                                                                                                  #  OPTIONS ARE, WHERE G STANDS FOR A GROUP AND .X FOR THE PHASE,
055754,000080: 
055755,000081:                                                                                                  #                  G.0             INACTIVE, WILLNOT PERMIT A GROUP G RESTART
055756,000082:                                                                                                  #                  G.1             WILL CAUSE THE LAST DISPLAY TO BE REACTIVATED, USED MAINLY IN MANNED FLIGHTS
055757,000083:                                                                                                  #                  G.EVEN          A DOUBLE TABLE RESTART, CAN CAUSE ANY COMBINATION OF TWO JOBS, TASKS, AND/OR
055758,000084:                                                                                                  #                                  LONGCALL TO BE RESTARTED.
055759,000085:                                                                                                  #                  G.ODD NOT .1    A SINGLE TABLE RESTART, CAN CAUSE EITHER A JOB, TASK, OR LONGCALL RESTART
055760,000086: 
055761,000087:                                                                                                  #  THIS INFORMATION IS PUT INTO THE OCTAL WORD AFTER TC PHASCHNG AS FOLLOWS
055762,000088: 
055763,000089:                                                                                                  #                  TL0 00P PPP PPP GGG                                     ,
055764,000090: 
055765,000091:                                                                                                  #  WHERE EACH LETTER OR NUMBER STANDS FOR A BIT.  THE G:S STAND FOR THE GROUP, OCTAL 1 - 7, THE P:S FOR THE PHASE,
055766,000092:                                                                                                  #  OCTAL 0 - 127.  0:S MUST BE 0.              IF ONE WISHES TO HAVE THE TBASE OF GROUP G TO BE SET AT THIS TIME,
055767,000093:                                                                                                  #  T IS SET TO 1, OTHERWISE IT IS SET TO 0.  SIMIARLY IF ONE WISHES TO SET LONGBASE, THEN L IS SET TO 1, OTHERWISE
055768,000094:                                                                                                  #  IT IS SET TO 0.  SOME EXAMPLES,
055769,000095: 
055770,000096:                                                                                                  #                                                   TC     PHASCHNG        THIS WILL CAUSE GROUP 3 TOBE SET TO 0,
055771,000097:                                                                                                  #                                                   OCT    00003           MAKING GROUP 3 INACTIVE
055772,000098: 
055773,000099:                                                                                                  #                                                   TC     PHASCHNG        IF A RESTART OCCURS THIS WOULD CAUSE
055774,000100:                                                                                                  #                                                   OCT    00012           GROUP 2 TO RESTART THE LAST DISPLAY
055775,000101: 
055776,000102:                                                                                                  #                                                   TC     PHASCHNG        THIS SETS THE TBASE OF GROUP 4 AND IN
055777,000103:                                                                                                  #                                                   OCT    40064           CASE OF A RESTART WOULD START UP THE TWO
055778,000104:                                                                                                  #                                                                          THINGS LOCATED IN THE DOUBLE 4.6 RESTART
055779,000105:                                                                                                  #                                                                          LOCATION
055780,000106:                                                                                                  #                                                   TC     PHASCHNG        THIS SETS LONGBASE AND UPON A RESTART
055781,000107:                                                                                                  #                                                   OCT    20135           CAUSES 5.13 TO BE RESTARTED  (SINCE
055782,000108:                                                                                                  #                                                                          LONGBASE WAS SET THIS SINGLE ENTRY
055783,000109:                                                                                                  #                                                                          SHOULD BE A LONGCALL)
055784,000110:                                                                                                  #                                                   TC     PHASCHNG        SINCE BOTH TBASE4 AND LONGBASE ARE SET,
055785,000111:                                                                                                  #                                                   OCT    60124           4.12 SHOULD CONTAIN BOTH A TASK AND A
055786,000112:                                                                                                  #                                                                          LONGCALL TO BE RESTARTED

055788,000114:                                                                                                  #  TYPE C PHASCHNG CONTAINS THE VARIABLE TYPE OF PHASCHNG INFORMATION. INSTEAD OF THE INFORMATION BEING IN A
055789,000115:                                                                                                  #  PERMANENT FORM, ONE STORES THE DESIRED RESTART INFORMATION IN A VARIABLE LOCATION. THE BITS ARE AS FOLLOWS,
055790,000116: 
055791,000117:                                                                                                  #                  TL0 1AD XXX CJW GGG
055792,000118:

055794,000120:                                                                                                  #  BE BY WAITLIST, W IS SET TO 1, IF IT IS A JOB, J IS SET TO 1, IF IT IS A LONGCALL, C IS SET TO 1. ONLY ONE OF
