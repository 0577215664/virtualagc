Source Code

These source-code files derive from a printout of Luminary 210 (Apollo 15-17 Lunar Module guidance computer program), from the personal library of original AGC developer Don Eyles, digitally photographed at archive.org, financially sponsored by Jim Lawton, and transcribed to source code by a team of volunteers. This colorized, syntax-highlighted form was created by assembling that transcribed source code. Note that the full page images are available on the Virtual AGC project page at archive.org, while reduced-size images are presented at the VirtualAGC project website. Report or fix any transcription errors at the Virtual AGC project code repository.

Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 210 OF AGC PROGRAM LUMINARY 
	BY NASA 2021112-161 17:11 MAR. 19,1971
Note that the date is the date of the printout, not the date of the program revision.

055720,000072:                                                                                                  #  PHASCHNG IS THE MAIN WAY OF MAKING PHASE CHANGES FOR RESTARTS.  THERE ARE THREE FORMS OF PHASCHNG, KNOWN AS TYPE
055721,000073:                                                                                                  #  A, TYPE B, AND TYPE C. THEY ARE ALL CALLED AS FOLLOWS, WHERE OCT XXXXX   CONTAINS THE PHASE INFORMATION,
055722,000074: 
055723,000075:                                                                                                  #                                                   TC     PHASCHNG
055724,000076:                                                                                                  #                                                   OCT    XXXXX
055725,000077: 
055726,000078:                                                                                                  #  TYPE A IS CONCERNED WITH FIXED PHASE CHANGES, THAT IS, PHASE INFORMATIONTHAT IS STORED PERMANENTLY.  THESE
055727,000079:                                                                                                  #  OPTIONS ARE, WHERE G STANDS FOR A GROUP AND .X FOR THE PHASE,
055728,000080: 
055729,000081:                                                                                                  #                  G.0             INACTIVE, WILLNOT PERMIT A GROUP G RESTART
055730,000082:                                                                                                  #                  G.1             WILL CAUSE THE LAST DISPLAY TO BE REACTIVATED, USED MAINLY IN MANNED FLIGHTS
055731,000083:                                                                                                  #                  G.EVEN          A DOUBLE TABLE RESTART, CAN CAUSE ANY COMBINATION OF TWO JOBS, TASKS, AND/OR
055732,000084:                                                                                                  #                                  LONGCALL TO BE RESTARTED.
055733,000085:                                                                                                  #                  G.ODD NOT .1    A SINGLE TABLE RESTART, CAN CAUSE EITHER A JOB, TASK, OR LONGCALL RESTART
055734,000086: 
055735,000087:                                                                                                  #  THIS INFORMATION IS PUT INTO THE OCTAL WORD AFTER TC PHASCHNG AS FOLLOWS
055736,000088: 
055737,000089:                                                                                                  #                  TL0 00P PPP PPP GGG                                     ,
055738,000090: 
055739,000091:                                                                                                  #  WHERE EACH LETTER OR NUMBER STANDS FOR A BIT.  THE G:S STAND FOR THE GROUP, OCTAL 1 - 7, THE P:S FOR THE PHASE,
055740,000092:                                                                                                  #  OCTAL 0 - 127.  0:S MUST BE 0.              IF ONE WISHES TO HAVE THE TBASE OF GROUP G TO BE SET AT THIS TIME,
055741,000093:                                                                                                  #  T IS SET TO 1, OTHERWISE IT IS SET TO 0.  SIMIARLY IF ONE WISHES TO SET LONGBASE, THEN L IS SET TO 1, OTHERWISE
055742,000094:                                                                                                  #  IT IS SET TO 0.  SOME EXAMPLES,
055743,000095: 
055744,000096:                                                                                                  #                                                   TC     PHASCHNG        THIS WILL CAUSE GROUP 3 TOBE SET TO 0,
055745,000097:                                                                                                  #                                                   OCT    00003           MAKING GROUP 3 INACTIVE
055746,000098: 
055747,000099:                                                                                                  #                                                   TC     PHASCHNG        IF A RESTART OCCURS THIS WOULD CAUSE
055748,000100:                                                                                                  #                                                   OCT    00012           GROUP 2 TO RESTART THE LAST DISPLAY
055749,000101: 
055750,000102:                                                                                                  #                                                   TC     PHASCHNG        THIS SETS THE TBASE OF GROUP 4 AND IN
055751,000103:                                                                                                  #                                                   OCT    40064           CASE OF A RESTART WOULD START UP THE TWO
055752,000104:                                                                                                  #                                                                          THINGS LOCATED IN THE DOUBLE 4.6 RESTART
055753,000105:                                                                                                  #                                                                          LOCATION
055754,000106:                                                                                                  #                                                   TC     PHASCHNG        THIS SETS LONGBASE AND UPON A RESTART
055755,000107:                                                                                                  #                                                   OCT    20135           CAUSES 5.13 TO BE RESTARTED  (SINCE
055756,000108:                                                                                                  #                                                                          LONGBASE WAS SET THIS SINGLE ENTRY
055757,000109:                                                                                                  #                                                                          SHOULD BE A LONGCALL)
055758,000110:                                                                                                  #                                                   TC     PHASCHNG        SINCE BOTH TBASE4 AND LONGBASE ARE SET,
055759,000111:                                                                                                  #                                                   OCT    60124           4.12 SHOULD CONTAIN BOTH A TASK AND A
055760,000112:                                                                                                  #                                                                          LONGCALL TO BE RESTARTED
055761,000113: 
055762,000114:                                                                                                  #  TYPE C PHASCHNG CONTAINS THE VARIABLE TYPE OF PHASCHNG INFORMATION. INSTEAD OF THE INFORMATION BEING IN A
055763,000115:                                                                                                  #  PERMANENT FORM, ONE STORES THE DESIRED RESTART INFORMATION IN A VARIABLE LOCATION. THE BITS ARE AS FOLLOWS,
055764,000116: 
055765,000117:                                                                                                  #                  TL0 1AD XXX CJW GGG
055766,000118: 
055767,000119:                                                                                                  #  WHERE EACH LETTER OR NUMBER STANDS FOR A BIT.  THE G:S STAND FOR THE GROUP, OCTAL 1 - 7.  IF THE RESTART IS TO
055768,000120:                                                                                                  #  BE BY WAITLIST, W IS SET TO 1, IF IT IS A JOB, J IS SET TO 1, IF IT IS A LONGCALL, C IS SET TO 1. ONLY ONE OF
055769,000121:                                                                                                  #  THESE THREE BIT S MAY BE SET.  X:S ARE IGNORED  1 MUST BE 1, AND 0 MUST  BE 0.  AGAIN T STANDS FOR THE TBASE,
