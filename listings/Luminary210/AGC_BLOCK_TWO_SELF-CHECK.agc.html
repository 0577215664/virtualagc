Source Code

These source-code files derive from a printout of Luminary 210 (Apollo 15-17 Lunar Module guidance computer program), from the personal library of original AGC developer Don Eyles, digitally photographed at archive.org, financially sponsored by Jim Lawton, and transcribed to source code by a team of volunteers. This colorized, syntax-highlighted form was created by assembling that transcribed source code. Note that the full page images are available on the Virtual AGC project page at archive.org, while reduced-size images are presented at the VirtualAGC project website. Report or fix any transcription errors at the Virtual AGC project code repository.

Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 210 OF AGC PROGRAM LUMINARY 
	BY NASA 2021112-161 17:11 MAR. 19,1971
Note that the date is the date of the printout, not the date of the program revision.

055002,000020:                                                                                                  #  PROGRAM NAME -- SELF-CHECK                    LOG SECTION:  AGC BLOCK TWO SELF-CHECK
055003,000021:                                                                                                  #  MOD NO -- 1                                   ASSEMBLY SUBROUTINE UTILITYM REV 25
055004,000022:                                                                                                  #  MOD BY -- GAUNTT
055005,000023: 
055006,000024:                                                                                                  #  FUNCTIONAL DESCRIPTION
055007,000025:                                                                                                  #        PROGRAM HAS TWO MAIN PARTS.  THE FIRST IS SELF-CHECK WHICH RUNS AS A ZERO PRIORITY JOB WITH NO CORE SET, AS
055008,000026:                                                                                                  #        PART OF THE BACK-UP IDLE LOOP.  THE SECOND IS SHOW-BANKSUM WHICH RUNS AS A REGULAR EXECUTIVE JOB WITH ITS OWN
055009,000027:                                                                                                  #        STARTING VERB.
055010,000028: 
055011,000029:                                                                                                  #        THE PURPOSE OF SELF-CHECK IS TO CHECK OUT VARIOUS PARTS OF THE COMPUTER AS OUTLINED BELOW IN THE OPTIONS.
055012,000030: 
055013,000031:                                                                                                  #        THE PURPOSE OF SHOW-BANKSUM IS TO DISPLAY THE SUM OF EACH BANK, ONE AT A TIME.
055014,000032: 
055015,000033:                                                                                                  #        IN ALL THERE ARE 7 POSSIBLE OPTIONS IN THIS BLOCK II VERSION OF SELF-CHECK.  MORE DETAIL DESCRIPTION MAY BE
055016,000034:                                                                                                  #        FOUND IN E-2065 BLOCK II AGC SELF-CHECK AND SHOW BANKSUM BY EDWIN D. SMALLY DECEMBER 1966, AND ADDENDA 2 AND 3.
055017,000035: 
055018,000036:                                                                                                  #        THE DIFFERENT OPTIONS ARE CONTROLLED BY PUTTING DIFFERENT NUMBERS IN THE SMODE REGISTER (NOUN 27).  BELOW IS
055019,000037:                                                                                                  #        A DESCRIPTION OF WHAT PARTS OF THE COMPUTER THAT ARE CHECKED BY THE OPTIONS, AND THE CORRESPONDING NUMBER, IN
055020,000038:                                                                                                  #        OCTAL, TO LOAD INTO SMODE.
055021,000039:                                                                                                  #                +-4             ERASABLE MEMORY
055022,000040:                                                                                                  #                +-5             FIXED MEMORY
055023,000041:                                                                                                  #                +-1,2,3,6,7,10  EVERYTHING IN OPTIONS 4 AND 5.
055024,000042:                                                                                                  #                -0              SAME AS +-10 UNTIL AN ERROR IS DETECTED.
055025,000043:                                                                                                  #                +0              NO CHECK, PUTS COMPUTER INTO THE BACKUP IDLE LOOP.
055026,000044: 
055027,000045:                                                                                                  #  WARNINGS
055028,000046:                                                                                                  #        USE OF E MEMORY RESERVED FOR SELF-CHECK (EVEN IN IDLE LOOP) AS TEMP STORAGE BY OTHER PROGRAMS IS DANGEROUS.
055029,000047:                                                                                                  #        SMODE SET GREATER THAN OCT 10 PUTS COMPUTER INTO BACKUP IDLE LOOP.
055030,000048: 
055031,000049:                                                                                                  #  CALLING SEQUENCE
055032,000050:                                                                                                  #        TO CALL SELF-CHECK KEY IN
055033,000051:                                                                                                  #                V 21 N 27 E     OPTION NUMBER E
055034,000052:                                                                                                  #        TO CALL SHOW-BANKSUM KEY IN
055035,000053:                                                                                                  #                V 91 E          DISPLAYS FIRST BANK
055036,000054:                                                                                                  #                V 33 E          PROCEED, DISPLAYS NEXT BANK
055037,000055: 
055038,000056:                                                                                                  #  EXIT MODES, NORMAL AND ALARM
055039,000057:                                                                                                  #        SELF-CHECK NORMALLY CONTINUES INDEFINITELY UNLESS THERE IS AN ERROR DETECTED.  IF SO + OPTION NUMBERS PUT
055040,000058:                                                                                                  #        COMPUTER INTO BACKUP IDLE LOOP, - OPTIONS NUMBERS RESTART THE OPTION.
055041,000059: 
055042,000060:                                                                                                  #        THE -0 OPTION PROCEEDS FROM THE LINE FOLLOWING THE LINE WHERE THE ERROR WAS DETECTED.
055043,000061: 
055044,000062:                                                                                                  #        SHOW-BANKSUM PROCEEDS UNTIL A TERMINATE IS KEYED IN (V 34 E).  THE COMPUTER IS PUT INTO THE BACKUP IDLE LOOP.
055045,000063: 
055046,000064:                                                                                                  #  OUTPUT
055047,000065:

055049,000067:                                                                                                  #        SELF-CHECK UPON DETECTING AN ERROR LOADS THE SELF-CHECK ALARM CONSTANT (01102) INTO THE FAILREG SET AND
055050,000068:                                                                                                  #        TURNS ON THE ALARM LIGHT.  THE OPERATOR MAY THEN DISPLAY THE THREE FAILREGS BY KEYING IN V 05 N 09 E.  FOR FURTHER
055051,000069:                                                                                                  #        INFORMATION HE MAY KEY IN V 05 N 08 E, THE DSKY DISPLAY IN R1 WILL BE ADDRESS+1 OF WHERE THE ERROR WAS DETECTED,
055052,000070:                                                                                                  #        IN R2 THE BBCON OF SELF-CHECK, AND IN R3 THE TOTAL NUMBER OF ERRORS DETECTED BY SELF-CHECK SINCE THE LAST MAN
055053,000071:                                                                                                  #        INITIATED FRESH START (SLAP1).
055054,000072: 
055055,000073:                                                                                                  #        SHOW-BANKSUM STARTING WTIH BANK 0 DISPLAYS IN R1 THE BANK SUM (A +-NUMBER EQUAL TO THE BANK NUMBER), IN R2
055056,000074:                                                                                                  #        THE BANK NUMBER, AND IN R3 THE BUGGER WORD.
055057,000075: 
055058,000076:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED
055059,000077:                                                                                                  #        ACCOMPLISHED BY FRESH START
055060,000078:                                                                                                  #                SMODE SET TO +0
055061,000079: 
055062,000080:                                                                                                  #  DEBRIS
055063,000081:                                                                                                  #        ALL EXITS FROM THE CHECK OF ERASABLE (ERASCHK) RESTORE ORIGINAL CONTENTS TO REGISTERS UNDER CHECK.
055064,000082:                                                                                                  #        EXCEPTION IS A RESTART.  RESTART THAT OCCURS DURING ERASCHK RESTORES ERASABLE, UNLESS THERE IS EVIDENCE TO DOUBT
055065,000083:                                                                                                  #        E MEMORY, IN WHICH CASE PROGRAM THEN DOES A FRESH START (DOFSTART).
