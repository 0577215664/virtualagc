Source Code

These source-code files derive from a printout of Luminary 210 (Apollo 15-17 Lunar Module guidance computer program), from the personal library of original AGC developer Don Eyles, digitally photographed at archive.org, financially sponsored by Jim Lawton, and transcribed to source code by a team of volunteers. This colorized, syntax-highlighted form was created by assembling that transcribed source code. Note that the full page images are available on the Virtual AGC project page at archive.org, while reduced-size images are presented at the VirtualAGC project website. Report or fix any transcription errors at the Virtual AGC project code repository.

Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 210 OF AGC PROGRAM LUMINARY 
	BY NASA 2021112-161 17:11 MAR. 19,1971
Note that the date is the date of the printout, not the date of the program revision.

039896,000031:                                                                                                  #                            MODES.

039898,000033:                                                                                                  #                            AUTOPILOT COMMANDS WHENEVER
039899,000034:                                                                                                  #                            1) NOT IN PNGCS-AUTO, OR
039900,000035:                                                                                                  #                            2) ENGINE IS OFF.

039903,000038: 
039904,000039:                                                                                                  #  MOD AUTHOR:     P. S. WEISSMAN
039905,000040: 
039906,000041:                                                                                                  #  OBJECTIVE OF MOD:  TO ISSUE STEERING COMMANDS TO THE DAP EVEN WHEN AGS IS INDICATED (IN CASE THE INDICA-
039907,000042:                                                                                                  #                     TION IS FALSE).
039908,000043: 
039909,000044: 
039910,000045:                                                                                                  #  FUNCTIONAL DESCRIPTION:
039911,000046: 
039912,000047:                                                                                                  #  FINDCDUW PROVIDES THE INTERFACES BETWEEN THE VARIOUS POWERED FLITE GUIDANCE PROGRAMS
039913,000048:                                                                                                  #  AND THE DIGITAL AUTOPILOT.  THE INPUTS TO FINDCDUW ARE THE THRUST COMMAND VECTOR
039914,000049:                                                                                                  #  AND THE WINDOW COMMAND VECTOR, AND THE OUTPUTS ARE THE GIMBAL ANGLE
039915,000050:                                                                                                  #  INCREMENTS, THE COMMANDED ATTITUDE ANGLE RATES, AND THE COMMANDED
039916,000051:                                                                                                  #  ATTITUDE LAG ANGLES (WHICH ACCOUNT FOR THE ANGLES BY WHICH THE BODY WILL
039917,000052:                                                                                                  #  LAG BEHIND A RAMP COMMAND IN ATTITUDE ANGLE DUE TO THE FINITE ANGULAR
039918,000053:                                                                                                  #  ACCELERATIONS AVAILABLE).
039919,000054: 
039920,000055:                                                                                                  #  FINDCDUW ALINES THE ESTIMATED THRUST VECTOR FROM THE THRUST DIRECTION
039921,000056:                                                                                                  #  FILTER WITH THE THRUST COMMAND VECTOR, AND, WHEN  XOVINHIB  SET,
039922,000057:                                                                                                  #  ALINES THE +Z HALF OF THE LM ZX PLANE WITH THE WINDOW COMMAND VECTOR.

039925,000060:                                                                                                  #  SPECIFICATIONS:
039926,000061: 
039927,000062:                                                                                                  #  INITIALIZATION: A SINGLE INTERPRETIVE CALL TO  INITCDUW  IS REQUIRED
039928,000063:                                                                                                  #                  BEFORE EACH GUIDED MANEUVER USING FINDCDUW.
039929,000064: 
039930,000065:                                                                                                  #  CALL:           INTERPRETIVE CALL TO FINDCDUW WITH THE THRUST COMMAND
039931,000066:                                                                                                  #                  VECTOR IN MPAC.  INTERPRETIVE CALL TO FINDCDUW -2 WITH
039932,000067:                                                                                                  #                  THE THRUST COMMAND VECTOR IN UNFC/2 AND NOT IN MPAC.
039933,000068: 
039934,000069:                                                                                                  #  RETURNS:        NORMAL INTERPRETIVE IN ALL CASES
039935,000070: 
039936,000071:                                                                                                  #                  1.       NORMALLY ALL AUTOPILOT CMDS ARE ISSUED.
039937,000072: 
039938,000073:                                                                                                  #                  2.       IF NOT PNGCS AUTO, DO STOPRATE AND RETURN
039939,000074:                                                                                                  #                           WITHOUT ISSUING AUTOPILOT CMDS.
039940,000075: 
039941,000076:                                                                                                  #                  3.       IF ENGINE OFF, DO STOPRATE AND RETURN WITHOUT
039942,000077:                                                                                                  #                           ISSUING AUTOPILOT CMDS.
039943,000078: 
039944,000079:                                                                                                  #  ALARMS:         00401 IF INPUTS DETERMINE AN ATTITUDE IN GIMBAL LOCK.
039945,000080:                                                                                                  #                           FINDCDUW DRIVES CDUXD AND CDUYD TO THE RQD VALUES,
039946,000081:                                                                                                  #                           BUT DRIVES CDUZD ONLY TO THE GIMBAL LOCK CONE.
039947,000082:                                                                                                  #                  00402 IF UNFC/2 OR UNWC/2 PRODUCE OVERFLOW WHEN
039948,000083:                                                                                                  #                           UNITIZED USING NORMUNIT.  FINDCDUW ISSUES
039949,000084:                                                                                                  #                           STOPRATE AS ONLY INPUT TO AUTOPILOT.
039950,000085: 
039951,000086:                                                                                                  #  INPUTS:         UNFC/2   THRUST COMMAND VECTOR, NEED NOT BE SEMI-UNIT.
039952,000087:                                                                                                  #                  UNWC/2   WINDOW COMMAND VECTOR, NEED NOT BE SEMI-UNIT.
039953,000088:                                                                                                  #                  OGABIAS  POSSIBLE BIAS FOR OUTER GIMBAL ANGLE (ZEROED IN INITCDUW), UNITS OF PI.
039954,000089:                                                                                                  #                  XOVINHIB FLAG DENOTING X AXIS OVERRIDE INHIBITED.
039955,000090:                                                                                                  #                  CSMDOCKD FLAG DENOTING CSM DOCKED.
039956,000091: 
039957,000092:                                                                                                  #  OUTPUTS:        DELCDUX,Y,Z
039958,000093:                                                                                                  #                  OMEGAPD,+1,+2
039959,000094:                                                                                                  #                  DELPEROR,+1,+2
039960,000095:                                                                                                  #                  CPHI,+1,+2 FOR NOUN22
039961,000096: 
039962,000097:                                                                                                  #  DEBRIS:         FINDCDUW DESTROYS SINCDUX,Y,Z AND COSCDUX,Y,Z BY
039963,000098:                                                                                                  #                  WRITING INTO THESE LOCATIONS THE SINES AND COSINES
039964,000099:                                                                                                  #                  OF THE CDUD'S IN PNGCS-AUTO, OF THE CDU'S OTHERWISE.
