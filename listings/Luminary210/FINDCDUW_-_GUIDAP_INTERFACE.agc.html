Source Code

These source-code files derive from a printout of Luminary 210 (Apollo 15-17 Lunar Module guidance computer program), from the personal library of original AGC developer Don Eyles, digitally photographed at archive.org, financially sponsored by Jim Lawton, and transcribed to source code by a team of volunteers. This colorized, syntax-highlighted form was created by assembling that transcribed source code. Note that the full page images are available on the Virtual AGC project page at archive.org, while reduced-size images are presented at the VirtualAGC project website. Report or fix any transcription errors at the Virtual AGC project code repository.

Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 210 OF AGC PROGRAM LUMINARY 
	BY NASA 2021112-161 17:11 MAR. 19,1971
Note that the date is the date of the printout, not the date of the program revision.

040082,000023:                                                                                                  #  PROGRAM NAME:   FINDCDUW
040083,000024: 
040084,000025:                                                                                                  #  MOD NUMBER:     1         68 07 15

040086,000027:                                                                                                  #  MOD AUTHOR:     KLUMPP
040087,000028: 
040088,000029:                                                                                                  #  OBJECTS OF MOD: 1.        TO SUPPLY COMMANDED GIMBAL ANGLES FOR NOUN 22.
040089,000030:                                                                                                  #                  2.        TO MAINTAIN CORRECT AND CURRENT THRUST
040090,000031:                                                                                                  #                            DIRECTION DATA IN ALL MODES.  THIS IS DONE BY
040091,000032:                                                                                                  #                            FETCHING FOR THE THRUST DIRECTION FILTER THE
040092,000033:                                                                                                  #                            CDUD'S IN PNGCS-AUTO, THE CDU'S IN ALL OTHER
040093,000034:                                                                                                  #                            MODES.
040094,000035:                                                                                                  #                  3.        TO SUBSTITUTE A STOPRATE FOR THE NORMAL
040095,000036:                                                                                                  #                            AUTOPILOT COMMANDS WHENEVER
040096,000037:                                                                                                  #                            1) NOT IN PNGCS-AUTO, OR
040097,000038:                                                                                                  #                            2) ENGINE IS OFF.
040098,000039: 
040099,000040:                                                                                                  #  MOD NUMBER:     2         10 DECEMBER 1970
040100,000041: 
040101,000042:                                                                                                  #  MOD AUTHOR:     P. S. WEISSMAN
040102,000043: 
040103,000044:                                                                                                  #  OBJECTIVE OF MOD:  TO ISSUE STEERING COMMANDS TO THE DAP EVEN WHEN AGS IS INDICATED (IN CASE THE INDICA-
040104,000045:                                                                                                  #                     TION IS FALSE).
040105,000046: 
040106,000047: 
040107,000048:                                                                                                  #  FUNCTIONAL DESCRIPTION:
040108,000049: 
040109,000050:                                                                                                  #  FINDCDUW PROVIDES THE INTERFACES BETWEEN THE VARIOUS POWERED FLITE GUIDANCE PROGRAMS
040110,000051:                                                                                                  #  AND THE DIGITAL AUTOPILOT.  THE INPUTS TO FINDCDUW ARE THE THRUST COMMAND VECTOR
040111,000052:                                                                                                  #  AND THE WINDOW COMMAND VECTOR, AND THE OUTPUTS ARE THE GIMBAL ANGLE
040112,000053:                                                                                                  #  INCREMENTS, THE COMMANDED ATTITUDE ANGLE RATES, AND THE COMMANDED
040113,000054:                                                                                                  #  ATTITUDE LAG ANGLES (WHICH ACCOUNT FOR THE ANGLES BY WHICH THE BODY WILL
040114,000055:                                                                                                  #  LAG BEHIND A RAMP COMMAND IN ATTITUDE ANGLE DUE TO THE FINITE ANGULAR
040115,000056:                                                                                                  #  ACCELERATIONS AVAILABLE).
040116,000057: 
040117,000058:                                                                                                  #  FINDCDUW ALINES THE ESTIMATED THRUST VECTOR FROM THE THRUST DIRECTION
040118,000059:                                                                                                  #  FILTER WITH THE THRUST COMMAND VECTOR, AND, WHEN  XOVINHIB  SET,
040119,000060:                                                                                                  #  ALINES THE +Z HALF OF THE LM ZX PLANE WITH THE WINDOW COMMAND VECTOR.

040122,000063:                                                                                                  #  SPECIFICATIONS:
040123,000064: 
040124,000065:                                                                                                  #  INITIALIZATION: A SINGLE INTERPRETIVE CALL TO  INITCDUW  IS REQUIRED
040125,000066:                                                                                                  #                  BEFORE EACH GUIDED MANEUVER USING FINDCDUW.
040126,000067: 
040127,000068:                                                                                                  #  CALL:           INTERPRETIVE CALL TO FINDCDUW WITH THE THRUST COMMAND
040128,000069:                                                                                                  #                  VECTOR IN MPAC.  INTERPRETIVE CALL TO FINDCDUW -2 WITH
040129,000070:                                                                                                  #                  THE THRUST COMMAND VECTOR IN UNFC/2 AND NOT IN MPAC.
040130,000071: 
040131,000072:                                                                                                  #  RETURNS:        NORMAL INTERPRETIVE IN ALL CASES
040132,000073: 
040133,000074:                                                                                                  #                  1.       NORMALLY ALL AUTOPILOT CMDS ARE ISSUED.
040134,000075: 
040135,000076:                                                                                                  #                  2.       IF NOT PNGCS AUTO, DO STOPRATE AND RETURN
040136,000077:                                                                                                  #                           WITHOUT ISSUING AUTOPILOT CMDS.
040137,000078: 
040138,000079:                                                                                                  #                  3.       IF ENGINE OFF, DO STOPRATE AND RETURN WITHOUT
040139,000080:                                                                                                  #                           ISSUING AUTOPILOT CMDS.
040140,000081: 
040141,000082:                                                                                                  #  ALARMS:         00401 IF INPUTS DETERMINE AN ATTITUDE IN GIMBAL LOCK.
040142,000083:                                                                                                  #                           FINDCDUW DRIVES CDUXD AND CDUYD TO THE RQD VALUES,
040143,000084:                                                                                                  #                           BUT DRIVES CDUZD ONLY TO THE GIMBAL LOCK CONE.
040144,000085:                                                                                                  #                  00402 IF UNFC/2 OR UNWC/2 PRODUCE OVERFLOW WHEN
040145,000086:                                                                                                  #                           UNITIZED USING NORMUNIT.  FINDCDUW ISSUES
040146,000087:                                                                                                  #                           STOPRATE AS ONLY INPUT TO AUTOPILOT.
040147,000088: 
040148,000089:                                                                                                  #  INPUTS:         UNFC/2   THRUST COMMAND VECTOR, NEED NOT BE SEMI-UNIT.
040149,000090:                                                                                                  #                  UNWC/2   WINDOW COMMAND VECTOR, NEED NOT BE SEMI-UNIT.
040150,000091:                                                                                                  #                  OGABIAS  POSSIBLE BIAS FOR OUTER GIMBAL ANGLE (ZEROED IN INITCDUW), UNITS OF PI.
040151,000092:                                                                                                  #                  XOVINHIB FLAG DENOTING X AXIS OVERRIDE INHIBITED.
040152,000093:                                                                                                  #                  CSMDOCKD FLAG DENOTING CSM DOCKED.
040153,000094: 
040154,000095:                                                                                                  #  OUTPUTS:        DELCDUX,Y,Z
040155,000096:                                                                                                  #                  OMEGAPD,+1,+2
040156,000097:                                                                                                  #                  DELPEROR,+1,+2
040157,000098:                                                                                                  #                  CPHI,+1,+2 FOR NOUN22
040158,000099: 
040159,000100:                                                                                                  #  DEBRIS:         FINDCDUW DESTROYS SINCDUX,Y,Z AND COSCDUX,Y,Z BY
040160,000101:                                                                                                  #                  WRITING INTO THESE LOCATIONS THE SINES AND COSINES
040161,000102:                                                                                                  #                  OF THE CDUD'S IN PNGCS-AUTO, OF THE CDU'S OTHERWISE.
