Source Code

These source-code files were derived from scans of the Colossus 237 (Apollo 8 Command Module) program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9 CM) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

061622,000060:                                                                                                  #      UPFLAG AND DOWNFLAG ARE ENTIRELY GENERAL FLAG SETTING AND CLEARING SUBROUTINES.   USING THEM, WHETHER OR
061623,000061:                                                                                                  #  NOT IN INTERRUPT, ONE MAY SET OR CLEAR ANY SINGLE, NAMED BIT IN ANY ERASABLE REGISTER, SUBJECT OF COURSE TO
061624,000062:                                                                                                  #  EBANK SETTING.   A "NAMED" BIT, AS THE WORD IS USED HERE, IS ANY BIT WITH A NAME FORMALLY ASSIGNED BY THE YUL
061625,000063:                                                                                                  #  ASSEMBLER.
061626,000064: 
061627,000065:                                                                                                  #      AT PRESENT THE ONLY NAMED BITS ARE THOSE IN THE FLAGWORDS.   ASSEMBLER CHANGES WILL MAKE IT POSSIBLE TO

061742,000180:                                                                                                  #  GENTRAN, A BLOCK TRANSFER ROUTINE.
061743,000181: 
061744,000182:                                                                                                  #  WRITTEN BY D. EYLES
061745,000183:                                                                                                  #  MOD 1 BY KERNAN                                                         UTILITYM REV 17 11/18/67
061746,000184: 
061747,000185:                                                                                                  #  MOD 2 BY SCHULENBERG  (REMOVE RELINT)   SKIPPER REV 4 2/28/68
061748,000186: 
061749,000187:                                                                                                  #           THIS ROUTINE IS USEFULL FOR TRANSFERING N CONSECUTIVE ERASABLE OR FIXED QUANTITIES TO SOME OTHER N
061750,000188:                                                                                                  #  CONSECUTIVE ERASABLE LOCATIONS.  IF BOTH BLOCKS OF DATA ARE IN SWITCHABLE EBANKS, THEY MUST BE IN THE SAME ONE.
061751,000189: 
061752,000190:                                                                                                  #           GENTRAN IS CALLABLE IN A JOB AS WELL AS A RUPT.  THE CALLING SEQUENCE IS:
061753,000191: 
061754,000192:                                                                                                  #                                          I        CA     N-1             # OF QUANTITIES MINUS ONE.
061755,000193:                                                                                                  #                                          I +1     TC     GENTRAN         IN FIXED-FIXED.
061756,000194:                                                                                                  #                                          I +2     ADRES  L               STARTING ADRES OF DATA TO BE MOVED.
061757,000195:                                                                                                  #                                          I +3     ADRES  M               STARTING ADRES OF DUPLICATION BLOCK.
061758,000196:                                                                                                  #                                          I +4                            RETURNS HERE.
061759,000197: 
061760,000198:                                                                                                  #           GENTRAN TAKES 25 MCT'S (300 MICROSECONDS) PER ITEM + 5 MCT'S (60 MICS) FOR ENTERING AND EXITING.
061761,000199: 
061762,000200:                                                                                                  #           A, L AND ITEMP1 ARE NOT PRESERVED.
061763,000201: 
061764,000202:    5475                                           BLOCK    02                                    
061765,000203:    4000                                           SETLOC   FFTAG4                                
061766,000204:    4000                                           BANK                                           
061767,000205: 
061768,000206:    5475     0061                                  EBANK=   ITEMP1                                
061769,000207: 
061770,000208:    5475                                           COUNT*   $$/TRAN                               
061771,000209: 
061772,000210:    5475           00004        GENTRAN            INHINT                                         
061773,000211:    5476           54061                           TS       ITEMP1                                #  SAVE N-1.
061774,000212:    5477           50002                           INDEX    Q                                     #  C(Q) = ADRES L.
061775,000213:    5500           60000                           AD       0                                     #  ADRES (L + N - 1).
061776,000214:    5501           50000                           INDEX    A                                     
061777,000215:    5502           30000                           CA       0                                     #  C(ABOVE).
061778,000216:    5503           54001                           TS       L                                     #  SAVE DATA.
