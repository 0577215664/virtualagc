Source Code

These source-code files were derived from scans of the Colossus 237 program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

007167,000039:                                                                                                  #  DOWNLIST FORMAT DEFINITIONS AND RULES-
007168,000040:                                                                                                  #  1. END OF A LIST = -XDNADR (X = 1 TO 6), -DNPTR, OR -DNCHAN.
007169,000041:                                                                                                  #  2. SNAPSHOT SUBLIST = LIST WHICH STARTS WITH A -1DNADR.
007170,000042:                                                                                                  #  3. SNAPSHOT SUBLIST CAN ONLY CONTAIN 1DNADRS.
007171,000043:                                                                                                  #  4. TIME2 1DNADR MUST BE LOCATED IN THE CONTROL LIST OF A DOWNLIST.
007172,000044:                                                                                                  #  5. ERASABLE DOWN TELEMETRY WORDS SHOULD BE GROUPED IN SEQUENTIAL
007173,000045:                                                                                                  #    LOCATIONS AS MUCH AS POSSIBLE TO SAVE STORAGE USED BY DOWNLINK LISTS.
007174,000046:                                                                                                  #  6. THE DOWNLINK LISTS(INCLUDING SUBLISTS) ARE ORGANIZED SUCH THAT THE ITEMS LISTED FIRST(IN FRONT OF FBANK) ARE
007175,000047:                                                                                                  #     SENT FIRST.  EXCEPTION--- SNAPSHOT SUBLISTS. IN THE SNAPSHOT SUBLISTS THE DATA REPRESENTED BY THE FIRST
007176,000048:                                                                                                  #     11 1DNADRS IS PRESERVED (IN ORDER) IN DNTMBUFF AND SENT BY THE NEXT 11 DOWNRUPTS. THE DATA REPRESENTED BY THE
007177,000049:                                                                                                  #     LIST IS SENT IMMEDIATELY.
