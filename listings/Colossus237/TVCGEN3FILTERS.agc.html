Source Code

These source-code files were derived from scans of the Colossus 237 program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

041578,000018:                                                                                                  #  FUNCTIONAL DESCRIPTION....
041579,000019: 
041580,000020:                                                                                                  #       THE GEN3DAP FILTER PACKAGE IS DESIGNED TO PROVIDE FLEXIBLE, LAST-MINUTE CHANGEABLE DIGITAL AUTOPILOT
041581,000021:                                                                                                  #  FILTERS FOR LEM-OFF FLIGHT.  GROUNDRULES FOR THE DESIGN AND USE OF THE PACKAGE ARE AS FOLLOWS.....
041582,000022: 
041583,000023:                                                                                                  #       1. FILTER COEFFICIENTS AND GAINS IN ERASABLE MEMORY
041584,000024:                                                                                                  #       2. UP TO THIRD-ORDER NUMERATOR OR DENOMINATOR
041585,000025:                                                                                                  #       3. OPERATIONAL FIT WITHIN THE STRUCTURE OF TEH REGULAR LEM-ON DAP CODING
041586,000026:                                                                                                  #       4. DENOMINATOR POLES INSIDE THE Z-PLANE UNIT CIRCLE
041587,000027:                                                                                                  #       5. NUMERATOR ZEROES INSIDE THE Z-PLANE DOUBLE-UNIT CIRCLE
041588,000028:                                                                                                  #       6. HIGH FREQUENCY (BODE) GAIN LESS THAN 8ASCREVS, OR 8.6380088 DEG/DEG
041589,000029: 
041590,000030:                                                                                                  #  THE FILTERS ARE SHOWN IN THE FOLLOWING DIAGRAMS.....
041591,000031:

041622,000062:                                                                                                  # # Page 959
041623,000063:                                                                                                  #  THE IMPLEMENTING EQUATIONS FOR THESE FILTERS ARE AS FOLLOWS.....
041624,000064: 
041625,000065:                                                                                                  #       PITCH GEN3DAP....                                 YAW GEN3DAP....
041626,000066:                                                                                                  #           NPD = (B+4) KPGEN3 NP0                            NYD = (B+4) KYGEN3 NY0
041627,000067:                                                                                                  #           NP0 = AP0 EP           +4(Z-1) NP1                NY0 = AY0 EY           +4(Z-1) NY1
041628,000068:                                                                                                  #           NP1 = AP1 EP - BP1 NP0 + (Z-1) NP2                NY1 = AY1 EY - BY1 NY0 + (Z-1) NY2
041629,000069:                                                                                                  #           NP2 = AP3 EP - BP2 NP0 + (Z-1) NP3                NY2 = AY2 EY - BY2 NY0 + (Z-1) NY3
041630,000070:                                                                                                  #           NP3 = AP3 EP - BP3 NP0                            NY3 = AY3 EY - BY3 NY0
041631,000071: 
041632,000072: 
041633,000073:                                                                                                  #       FILTER INPUTS EP AND EY ARE PICKED UP FROM REGULAR LEM-ON CODING AT ERRBTMP  (UPPER WORD ONLY),  THUS ARE
041634,000074:                                                                                                  #  SINGLE PRECISION QUANTITIES SCALED AT B-1 REVS. FILTER OUTPUTS NPD AND NYD ARE LEFT IN DOUBLE PRECISION AT
041635,000075:                                                                                                  #  CMDTMP, SCALED AT 1 ASCREV, READY FOR OUTPUT PROCESSING VIA REGULAR LEM-ON CODING AT ..P,YOFFSET..
041636,000076:                                                                                                  #  FOLLOWING OUTPUT PROCESSING, RETURN TO THE GEN3DAP FILTERS IS MADE FOR CALCULATION OF THE REMAINING NODES
041637,000077:                                                                                                  #  NP1 TO NP3, OR NY1 TO NY3.  GEN3DAP FILTERS THEN RETURN TO THE LEM-ON CODING AT  ..DELBARP,Y..   FOR RESPECTIVE
041638,000078:                                                                                                  #  OFFSET-TRACKER-FILTER COMPUTATIONS AND COPYCYCLES. NOTE THE EQUIVALENCES...NP1TMP=J5TMP, NP1=J5,
041639,000079:                                                                                                  #  NP2TMP=NSUMTMP, NP2=PNSUM, NP3TMP=DSUMTMP, NP3=PDSUM, WITH CORRESPONDING RELATIONS FOR YAW.  THUS THE COPY-
041640,000080:                                                                                                  #  CYCLE  PCOPY, FROM THE GEN3DAP STANDPOINT, IS EFFECTIVE FROM PMISC-3 TO ITS END AT TC Q.  YCOPY FROM YMISC-3.
041641,000081: 
041642,000082: 
041643,000083:                                                                                                  #  SCALING OF THE FILTER NODES, COEFFICIENTS, AND GAINS WITHIN THE AGC IS AS FOLLOWS.....
041644,000084: 
041645,000085:                                                                                                  #           QUANTITY   QUANTITY      PHYS.UNITS     MAX.VALUE      SCALE AT (FOR)
041646,000086: 
041647,000087:                                                                                                  #           EP         EY            REVS              1/8           B-1 REV           (CDU SCALING)
041648,000088: 
041649,000089:                                                                                                  #           NP0        NY0           REVS             (B+1)          B+1 REV
041650,000090:                                                                                                  #           NP1        NY1           REVS             (B+3)          B+3 REV
041651,000091:                                                                                                  #           NP2        NY2           REVS             (B+3)          B+3 REV
041652,000092:                                                                                                  #           NP3        NY3           REVS             (B+3)          B+3 REV
041653,000093: 
041654,000094:                                                                                                  #           NPD        NYD         ASC REVS            (1)           1 ASCREV          (ACTUATOR CDU SCALING)
041655,000095: 
041656,000096:                                                                                                  #           KPGEN3     KYGEN3     ASCREV/REV           (8)         B+3 ASCREV/REV
041657,000097: 
041658,000098:                                                                                                  #           AP0        AY0         DIMLESS.             1              B+2
041659,000099:                                                                                                  #           AP1        AY1         DIMLESS.             6              B+4
041660,000100:                                                                                                  #           AP2        AY2         DIMLESS.             12             B+4
041661,000101:                                                                                                  #           AP3        AY3         DIMLESS.             8              B+4
041662,000102: 
041663,000103:                                                                                                  #           BP1        BY1         DIMLESS.             3              B+2
041664,000104:                                                                                                  #           BP2        BY2         DIMLESS.             3              B+2
041665,000105:                                                                                                  #           BP3        BY3         DIMLESS.             1              B+2
041666,000106: 
041667,000107:                                                                                                  #  FILTER COEFFICIENTS, GAINS, AND NODES ARE HELD IN DOUBLE PRECISION (ERASABLE) TO PERMIT CONSERVATIVE
041668,000108:                                                                                                  #  SCALING AND YET OFFSET TRUNCATION LOSSES.  THIS APPEARS NECESSARY IF FILTER FLEXIBILITY IS TO BE MAINTAINED.
041669,000109:                                                                                                  #  COMPUTATION TIME IS NOT CRITICAL.
