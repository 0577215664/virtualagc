Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. 041569,000018:                                                                                                 #  FUNCTIONAL DESCRIPTION....
041570,000019: 
041571,000020:                                                                                                 #       THE GEN3DAP FILTER PACKAGE IS DESIGNED TO PROVIDE FLEXIBLE, LAST-MINUTE CHANGEABLE DIGITAL AUTOPILOT
041572,000021:                                                                                                 #  FILTERS FOR LEM-OFF FLIGHT.  GROUNDRULES FOR THE DESIGN AND USE OF THE PACKAGE ARE AS FOLLOWS.....
041573,000022: 
041574,000023:                                                                                                 #       1. FILTER COEFFICIENTS AND GAINS IN ERASABLE MEMORY
041575,000024:                                                                                                 #       2. UP TO THIRD-ORDER NUMERATOR OR DENOMINATOR
041576,000025:                                                                                                 #       3. OPERATIONAL FIT WITHIN THE STRUCTURE OF TEH REGULAR LEM-ON DAP CODING
041577,000026:                                                                                                 #       4. DENOMINATOR POLES INSIDE THE Z-PLANE UNIT CIRCLE
041578,000027:                                                                                                 #       5. NUMERATOR ZEROES INSIDE THE Z-PLANE DOUBLE-UNIT CIRCLE
041579,000028:                                                                                                 #       6. HIGH FREQUENCY (BODE) GAIN LESS THAN 8ASCREVS, OR 8.6380088 DEG/DEG
041580,000029: 
041581,000030:                                                                                                 #  THE FILTERS ARE SHOWN IN THE FOLLOWING DIAGRAMS.....

041614,000063:                                                                                                 #  THE IMPLEMENTING EQUATIONS FOR THESE FILTERS ARE AS FOLLOWS.....
041615,000064: 
041616,000065:                                                                                                 #       PITCH GEN3DAP....                                 YAW GEN3DAP....
041617,000066:                                                                                                 #           NPD = (B+4) KPGEN3 NP0                            NYD = (B+4) KYGEN3 NY0
041618,000067:                                                                                                 #           NP0 = AP0 EP            +4(Z-1) NP1                NY0 = AY0 EY           +4(Z-1) NY1
041619,000068:                                                                                                 #           NP1 = AP1 EP - BP1 NP0 + (Z-1) NP2                NY1 = AY1 EY - BY1 NY0 + (Z-1) NY2
041620,000069:                                                                                                 #           NP2 = AP3 EP - BP2 NP0 + (Z-1) NP3                NY2 = AY2 EY - BY2 NY0 + (Z-1) NY3
041621,000070:                                                                                                 #           NP3 = AP3 EP - BP3 NP0                            NY3 = AY3 EY - BY3 NY0
041622,000071: 
041623,000072: 
041624,000073:                                                                                                 #       FILTER INPUTS EP AND EY ARE PICKED UP FROM REGULAR LEM-ON CODING AT ERRBTMP  (UPPER WORD ONLY),  THUS ARE
041625,000074:                                                                                                 #  SINGLE PRECISION QUANTITIES SCALED AT B-1 REVS. FILTER OUTPUTS NPD AND NYD ARE LEFT IN DOUBLE PRECISION AT
041626,000075:                                                                                                 #  CMDTMP, SCALED AT 1 ASCREV, READY FOR OUTPUT PROCESSING VIA REGULAR LEM-ON CODING AT ..P,YOFFSET..
041627,000076:                                                                                                 #  FOLLOWING OUTPUT PROCESSING, RETURN TO THE GEN3DAP FILTERS IS MADE FOR CALCULATION OF THE REMAINING NODES
041628,000077:                                                                                                 #  NP1 TO NP3, OR NY1 TO NY3.  GEN3DAP FILTERS THEN RETURN TO THE LEM-ON CODING AT  ..DELBARP,Y..   FOR RESPECTIVE
041629,000078:                                                                                                 #  OFFSET-TRACKER-FILTER COMPUTATIONS AND COPYCYCLES. NOTE THE EQUIVALENCES...NP1TMP=J5TMP, NP1=J5,
041630,000079:                                                                                                 #  NP2TMP=NSUMTMP, NP2=PNSUM, NP3TMP=DSUMTMP, NP3=PDSUM, WITH CORRESPONDING RELATIONS FOR YAW.  THUS THE COPY-
041631,000080:                                                                                                 #  CYCLE  PCOPY, FROM THE GEN3DAP STANDPOINT, IS EFFECTIVE FROM PMISC-3 TO ITS END AT TC Q.  YCOPY FROM YMISC-3.
041632,000081: 
041633,000082:

041641,000090:                                                                                                 #           NP1        NY1           REVS             (B+3)          B+3 REV
041642,000091:                                                                                                 #           NP2        NY2           REVS             (B+3)          B+3 REV

041655,000104:                                                                                                 #           BP2        BY2         DIMLESS.             3              B+2
