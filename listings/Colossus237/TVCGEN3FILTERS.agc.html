Source Code

These source-code files were derived from scans of the Colossus 237 (Apollo 8 Command Module) program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9 CM) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

041758,000018:                                                                                                  #  PROGRAM NAME.... GEN3DAP FILTERS, CONSISTING OF NP0NODE, NP1NODE, NY0NODE, NY1NODE, ETC.
041759,000019:                                                                                                  #  LOG SECTION.... GEN3DAP FILTERS   SUBROUTINE....DAPCSM
041760,000020:                                                                                                  #  MOD BY ENGEL                      20 OCT, 1967
041761,000021: 
041762,000022:                                                                                                  #  FUNCTIONAL DESCRIPTION....
041763,000023: 
041764,000024:                                                                                                  #       THE GEN3DAP FILTER PACKAGE IS DESIGNED TO PROVIDE FLEXIBLE, LAST-MINUTE CHANGEABLE DIGITAL AUTOPILOT
041765,000025:                                                                                                  #  FILTERS FOR LEM-OFF FLIGHT.  GROUNDRULES FOR THE DESIGN AND USE OF THE PACKAGE ARE AS FOLLOWS.....
041766,000026: 
041767,000027:                                                                                                  #       1. FILTER COEFFICIENTS AND GAINS IN ERASABLE MEMORY
041768,000028:                                                                                                  #       2. UP TO THIRD-ORDER NUMERATOR OR DENOMINATOR
041769,000029:                                                                                                  #       3. OPERATIONAL FIT WITHIN THE STRUCTURE OF THE REGULAR LEM-ON DAP CODING
041770,000030:                                                                                                  #       4. DENOMINATOR POLES INSIDE THE Z-PLANE UNIT CIRCLE
041771,000031:                                                                                                  #       5. NUMERATOR ZEROS INSIDE THE Z-PLANE DOUBLE-UNIT CIRCLE
041772,000032:                                                                                                  #       6. HIGH FREQUENCY (BODE) GAIN LESS THAN 8ASCREVS, OR 8.6380088 DEG/DEG
041773,000033: 
041774,000034:                                                                                                  #  THE FILTERS ARE SHOWN IN THE FOLLOWING DIAGRAMS.....
041775,000035: 
041776,000036: 
041777,000037:                                                                                                  #  PITCH GEN3DAP FILTER..
041778,000038: 
041779,000039:                                                                                                  #                                                                   KPGEN3
041780,000040:                                                                                                  #                                                                  ********
041781,000041:                                                                                                  #                     *****************************************           *
041782,000042:                                                                                                  #                     *                                       *           *
041783,000043:                                                                                                  #                     *              -1        -2        -3   *           *
041784,000044:                                                                                                  #        EP = ERRBTMP *    APO + AP1 Z   + AP2 Z   + AP3 Z     *   NPO          NPD = CMDTMP  **
041785,000045:                                                                                                  #       ***************  -----------------------------------  **********( X )*********************
041786,000046:                                                                                                  #                     *              -1        -2        -3   *                               **
041787,000047:                                                                                                  #                     *    1  + BP1 Z   + BP2 Z   + BP3 Z     *
041788,000048:                                                                                                  #                     *                                       *
041789,000049:                                                                                                  #                     *****************************************

041807,000067:                                                                                                  #  THE IMPLEMENTING EQUATIONS FOR THESE FILTERS ARE AS FOLLOWS.....
041808,000068: 
041809,000069:                                                                                                  #       PITCH GEN3DAP....                                 YAW GEN3DAP....
041810,000070:                                                                                                  #           NPD = (B+4) KPGEN3 NP0                            NYD = (B+4) KYGEN3 NY0
041811,000071:                                                                                                  #           NP0 = AP0 EP           +4(Z-1) NP1                NY0 = AY0 EY           +4(Z-1) NY1
041812,000072:                                                                                                  #           NY1 = AP1 EP - BP1 NP0 + (Z-1) NP2                NY1 = AY1 EY - BY1 NY0 + (Z-1) NY2
041813,000073:                                                                                                  #           NP2 = AP2 EP - BP2 NP0 + (Z-1) NP3                NY2 = AY2 EY - BY2 NY0 + (Z-1) NY3
041814,000074:                                                                                                  #           NP3 = AP3 EP - BP3 NP0                            NY3 = AY3 EY - BY3 NY0
041815,000075: 
041816,000076: 
041817,000077:                                                                                                  #       FILTER INPUTS EP AND EY ARE PICKED UP FROM REGULAR LEM-ON CODING AT ERRBTMP  (UPPER WORD ONLY),  THUS ARE
041818,000078:                                                                                                  #  SINGLE PRECISION QUANTITIES SCALED AT B-1 REVS. FILTER OUTPUTS NPD AND NYD ARE LEFT IN DOUBLE PRECISION AT
041819,000079:                                                                                                  #  CMDTMP, SCALED AT 1 ASCREV, READY FOR OUTPUT PROCESSING VIA REGULAR LEM-ON CODING AT ..P,YOFFSET..
041820,000080:                                                                                                  #  FOLLOWING OUTPUT PROCESSING, RETURN TO THE GEN3DAP FILTERS IS MADE FOR CALCULATION OF THE REMAINING NODES
041821,000081:                                                                                                  #  NP1 TO NP3, OR NY1 TO NY3.  GEN3DAP FILTERS THEN RETURN TO THE LEM-ON CODING AT  ..DELBARP,Y..   FOR RESPECTIVE
041822,000082:                                                                                                  #  OFFSET-TRACKER-FILTER COMPUTATIONS AND COPYCYCLES. NOTE THE EQUIVALENCES...NP1TMP=J5TMP, NP1=J5,
041823,000083:                                                                                                  #  NP2TMP=NSUMTMP, NP2=PNSUM, NP3TMP=DSUMTMP, NP3=PDSUM, WITH CORRESPONDING RELATIONS FOR YAW.  THUS THE COPY-
041824,000084:                                                                                                  #  CYCLE  PCOPY, FROM THE GEN3DAP STANDPOINT, IS EFFECTIVE FROM PMISC-3 TO ITS END AT TC Q.  YCOPY FROM YMISC-3.
041825,000085: 
041826,000086: 
041827,000087:                                                                                                  #  SCALING OF THE FILTER NODES, COEFFICIENTS, AND GAINS WITHIN THE AGC IS AS FOLLOWS.....
041828,000088: 
041829,000089:                                                                                                  #           QUANTITY   QUANTITY      PHYS.UNITS     MAX.VALUE      SCALE AT (FOR)
041830,000090: 
041831,000091:                                                                                                  #           EP         EY            REVS              1/8           B-1 REV           (CDU SCALING)
041832,000092: 
041833,000093:                                                                                                  #           NP0        NY0           REVS             (B+1)          B+1 REV
041834,000094:                                                                                                  #           NP1        NY1           REVS             (B+3)          B+3 REV
041835,000095:                                                                                                  #           NP2        NY2           REVS             (B+3)          B+3 REV
041836,000096:                                                                                                  #           NP3        NY3           REVS             (B+3)          B+3 REV
041837,000097: 
041838,000098:                                                                                                  #           NPD        NYD         ASC REVS            (1)           1 ASCREV          (ACTUATOR CDU SCALING)
041839,000099: 
041840,000100:                                                                                                  #           KPGEN3     KYGEN3     ASCREV/REV           (8)         B+3 ASCREV/REV
041841,000101: 
041842,000102:                                                                                                  #           AP0        AY0         DIMLESS.             1              B+2
041843,000103:                                                                                                  #           AP1        AY1         DIMLESS.             6              B+4
041844,000104:                                                                                                  #           AP2        AY2         DIMLESS.             12             B+4
041845,000105:                                                                                                  #           AP3        AY3         DIMLESS.             8              B+4
041846,000106: 
041847,000107:                                                                                                  #           BP1        BY1         DIMLESS.             3              B+2
041848,000108:                                                                                                  #           BP2        BY2         DIMLESS.             3              B+2
041849,000109:                                                                                                  #           BP3        BY3         DIMLESS.             1              B+2
041850,000110: 
041851,000111:                                                                                                  #  FILTER COEFFICIENTS, GAINS, AND NODES ARE HELD IN DOUBLE PRECISION (ERASABLE) TO PERMIT CONSERVATIVE
041852,000112:                                                                                                  #  SCALING AND YET OFFSET TRUNCATION LOSSES.  THIS APPEARS NECESSARY IF FILTER FLEXIBILITY IS TO BE MAINTAINED.
041853,000113:                                                                                                  #  COMPUTATION TIME IS NOT CRITICAL.
