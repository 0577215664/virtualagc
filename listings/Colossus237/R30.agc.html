Source Code

These source-code files were derived from scans of the Colossus 237 program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

021760,000023:                                                                                                  #  1. IF AVERAGE G IS OFF:
021761,000024:                                                                                                  #           FLASH DISPLAY V04N06. R2 INDICATES WHICH SHIP'S STATE VECTOR IS
021762,000025:                                                                                                  #            TO BE UPDATED. INITIAL CHOICE IS THIS SHIP (R2=1). ASTRONAUT
021763,000026:                                                                                                  #            CAN CHANGE TO OTHER SHIP BY V22EXE, WHERE X IS NOT EQ 1.
021764,000027:                                                                                                  #           SELECTED STATE VECTOR UPDATED BY THISPREC (OTHPREC).
021765,000028:                                                                                                  #           CALLS SR30.1 (WHICH CALLS TFFCONMU + TFFRP/RA) TO CALCULATE
021766,000029:                                                                                                  #            RPER (PERIGEE RADIUS), RAPO (APOGEE RADIUS), HPER (PERIGEE
021767,000030:                                                                                                  #            HEIGHT ABOVE LAUNCH PAD OR LAUNAR LANDING SITE), HARD (APOGEE
021768,000031:                                                                                                  #            HEIGHT AS ABOVE), TPER (TIME TO PERIGEE), TFF (TIME TO
021769,000032:                                                                                                  #            INTERSECT 300 KFT ABOVE PAD OR 36KFT ABOVE LANDING SITE).
021770,000033:                                                                                                  #           FLASH MONITOR V16N44 (HAPO, HPER, TFF).TFF IS -59MS59S IF IT WAS
021771,000034:                                                                                                  #            NOT COMPUTABLE, OTHERWISE IT INCREMENTS ONCE PER SECOND.
021772,000035:                                                                                                  #            ASTRONAUT HAS OPTION TO MONITOR TPER BY KEYING IN N 32 E.
021773,000036:                                                                                                  #            DISPLAY IS IN HMS, IS NEGATIVE (AS WAS TFF), AND INCREMENTS
021774,000037:                                                                                                  #            ONCE PER SECOND ONLY IF TFF DISPLAY WAS -59M59S.
021775,000038: 
021776,000039:                                                                                                  #  2. IF AVERAGE G IS ON:
021777,000040:                                                                                                  #           CALLS SR30.1 APPROX EVERY TWO SECS.  STATE VECTOR IS ALWAYS
021778,000041:                                                                                                  #            FOR THIS VEHICLE. V82 DOES NOT DISTURB STATE VECTOR.  RESULTS
021779,000042:                                                                                                  #            OF SR30.1 ARE RAPO, RPER, HAPO, HPER, TPER, TFF.
021780,000043:                                                                                                  #           FLASH MONITOR V16N44 (HAPO, HPER, TFF).
021781,000044:                                                                                                  #           IF MODE IS P11, THEN CALL DELRSPL SO ASTRONAUT CAN MONITOR
021782,000045:                                                                                                  #            RESULTS BY N50E. SPLASH COMPUTATION DONE ONCE FOR TWO SECS.
021783,000046:
