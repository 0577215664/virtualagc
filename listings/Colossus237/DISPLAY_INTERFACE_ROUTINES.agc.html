Source Code

These source-code files were derived from scans of the Colossus 237 (Apollo 8 Command Module) program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9 CM) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

060035,000019:                                                                                                  #  DISPLAYS CAN BE CLASSIFIED INTO THE FOLLOWING CATEGORIES-
060036,000020: 
060037,000021:                                                                                                  #      1. PRIORITY DISPLAYS- DISPLAYS WHICH TAKE PRIORITY OVER ALL OTHER DISPLAYS. USUALLY THESE DISPLAYS ARE SENT
060038,000022:                                                                                                  #         OUT UNDER CRITICAL ALARM CONDITIONS.
060039,000023:                                                                                                  #      2. EXTENDED VERB DISPLAYS- ALL EXTENDED VERBS AND MARK ROUTINES SHOULD USE EXTENDED VERB (MARK) DISPLAYS.
060040,000024:                                                                                                  #      3. NORMAL DISPLAYS- ALL MISSION PROGRAM DISPLAYS WHICH INTERFACE WITH THE ASTRONAUT DURING THE NORMAL
060041,000025:                                                                                                  #         SEQUENCE OF EVENTS.
060042,000026:                                                                                                  #      4. MISC. DISPLAYS- ALL DISPLAYS NOT HANDLED BY THE DISPLAY INTERFACEROUTINES. THESE INCLUDE SUCH DISPLAYS AS
060043,000027:                                                                                                  #         MM DISPLAYS AND SPECIAL PURPOSE DISPLAYS HANDLED BY PINBALL.
060044,000028: 
060045,000029: 
060046,000030:                                                                                                  #      5. ASTRONAUT INITIATED DISPLAYS- ALL DISPLAYS INITIATED EXTERNALLY.
060047,000031:                                                                                                  #  THE FOLLOWING TERMS ARE USED TO DESCRIBE THE STATUS OF DISPLAYS-
060048,000032: 
060049,000033:                                                                                                  #      1. ACTIVE-THE DISPLAY WHICH IS (1) BEING DISPLAYED TO THE ASTRONAUT AND WAITING FOR A RESPONSE OR
060050,000034:                                                                                                  #         (2) WAITING FIRST IN LINE FOR THE ASTRONAUT TO FINISH USING THE DSKY OR (3) BEING DISPLAYED ON THE DSKY
060051,000035:                                                                                                  #         BUT NOT WAITING FOR A RESPONSE.
060052,000036:                                                                                                  #      2. INACTIVE -A DISPLAY WHICH HAS (1) BEEN ACTIVE BUT WAS INTERRUPTEDBY A DISPLAY OF HIGHER PRIORITY,
060053,000037:                                                                                                  #         (2) BEEN PUT INTO THE WAITING LIST AT TIME IT WAS REQUESTED DUE TO THE FACT A HIGHER PRIORITY DISPLAY
060054,000038:                                                                                                  #         WAS ALREADY GOING, (3) BEEN INTERRUPTED BY THE ASTRONAUT (CALLED A PINBRANCH CONDITION, SINCE THIS TYPE
060055,000039:                                                                                                  #         OF INACTIVE DISPLAY IS USUALLY REACTIVATED ONLY BY PINBALL) OR (4) A DISPLAY WHICH HAS FINISHED BUT STILL
060056,000040:                                                                                                  #         HAS INFO SAVED FOR RESTART PURPOSES.
060057,000041: 
060058,000042: 
060059,000043:                                                                                                  #  DISPLAY PRIORITIES WORK AS FOLLOWS-
060060,000044: 
060061,000045:                                                                                                  #    INTERRUPTS-
060062,000046:                                                                                                  #      1. THE ASTRONAUT CAN INTERRUPT ANY DISPLAY WITH AN EXTERNAL DISPLAY REQUEST.
060063,000047:                                                                                                  #      2. INTERNAL DISPLAYS CAN NOT BE SENT OUT WHEN THE ASTRONAUT IS USING THE DSKY.
060064,000048:                                                                                                  #      3. PRIORITY DISPLAYS INTERRUPT ALL OTHER TYPES OF INTERNAL DISPLAYS. A PRIORITY DISPLAY INTERRUPTING ANOTHER
060065,000049:                                                                                                  #         PRIORITY DISPLAY WILL CAUSE AN ABORT UNLESS BIT14 IS SET FOR THE LINUS ROUTINE.
060066,000050:                                                                                                  #      4. A MARK DISPLAY INTERRUPTS ANY NORMAL DISPLAY.
060067,000051:                                                                                                  #      5. A MARK THAT INTERRUPTS A MARK COMPLETELY REPLACES IT.
060068,000052: 
060069,000053:                                                                                                  #    ORDER OF WAITING DISPLAYS-
060070,000054:                                                                                                  #      1. ASTRONAUT EXTERNAL USE
060071,000055:                                                                                                  #      2. PRIORITY
060072,000056:                                                                                                  #      3. INTERRUPTED MARK
060073,000057:                                                                                                  #      4. INTERRUPTED NORMAL
060074,000058: 
060075,000059:                                                                                                  #      5. MARK TO BE REQUESTED (SEE DESCRIPTION OF ENDMARK)
060076,000060:                                                                                                  #      6. MARK WAITING
060077,000061:                                                                                                  #      7. NORMAL WAITING

060080,000064:                                                                                                  #  THE DISPLAY ROUTINES ARE INTENDED TO SERVE AS AN INTERFACE BETWEEN THE USER AND PINBALL.  THE
060081,000065:                                                                                                  #  FOLLOWING STATEMENTS CAN BE MADE ABOUT NORMAL DISPLAYS AND PRIORITY DISPLAYS (A DESCRIPTION OF MARK ROUTINES
060082,000066:                                                                                                  #  WILL FOLLOW LATER):
060083,000067: 
060084,000068:                                                                                                  #       1.  ALL ROUTINES THAT END IN R HAVE AN IMMEDIATE RETURN TO THE USER.  FOR ALL FLASHING DISPLAYS THIS RETURN
060085,000069:                                                                                                  #           IS TO THE USERS CALL CADR +4.  FOR THE ONLY NON FLASHING IMMEDIATE RETURN DISPLAY (GODSPR) THIS RETURN
060086,000070:                                                                                                  #           IS TO THE USERS CALLING LOC +1.
060087,000071:                                                                                                  #       2.  ALL ROUTINES NOT ENDING IN R DO NOT DO AN IMMEDIATE RETURN TO THE USER.
060088,000072:                                                                                                  #       3.  ALL ROUTINES THAT END IN R START A SEPARATE JOB (MAKEPLAY) WITH USERS JOB PRIORITY.
060089,000073:                                                                                                  #       4.  ALL ROUTINES NOT ENDING IN R BRANCH DIRECTLY TO MAKEPLAY WHICH MAKES THESE DISPLAYS A PART OF THE
060090,000074:                                                                                                  #           USERS JOB.
060091,000075:                                                                                                  #       5.  ALL DISPLAY ROUTINES ARE CALLED VIA BANKCALL.
060092,000076:                                                                                                  #       6.  TO RESTART A DISPLAY THE USER WILL GENERALLY USE A PHASE OF ONE WITH DESIRED RESTART GROUP (SEE
060093,000077:                                                                                                  #           DESCRIPTION OF RESTARTS).
060094,000078:                                                                                                  #       7.  ALL FLASHING DISPLAYS HAVE 3 RETURNS TO THE USER FROM ASTRONAUT RESPONSES.  A TERMINATE (V34) BRANCHES
060095,000079:                                                                                                  #           TO THE USERS CALL CADR +1.  A PROCEED (V33) BRANCHES TO THE USERS CALL CADR +2.  AN ENTER OR RECYCLE
060096,000080:                                                                                                  #           (V32) BRANCHES TO THE USERS CALL CADR +3.
060097,000081:                                                                                                  #       8.  ALL ROUTINES MUST BE USED UNDER EXECUTIVE CONTROL.
