Source Code

These source-code files were derived from scans of the Colossus 237 (Apollo 8 Command Module) program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9 CM) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

060078,000022:                                                                                                  #  DISPLAYS CAN BE CLASSIFIED INTO THE FOLLOWING CATEGORIES-
060079,000023: 
060080,000024:                                                                                                  #      1. PRIORITY DISPLAYS- DISPLAYS WHICH TAKE PRIORITY OVER ALL OTHER DISPLAYS. USUALLY THESE DISPLAYS ARE SENT
060081,000025:                                                                                                  #         OUT UNDER CRITICAL ALARM CONDITIONS.
060082,000026:                                                                                                  #      2. EXTENDED VERB DISPLAYS- ALL EXTENDED VERBS AND MARK ROUTINES SHOULD USE EXTENDED VERB (MARK) DISPLAYS.
060083,000027:                                                                                                  #      3. NORMAL DISPLAYS- ALL MISSION PROGRAM DISPLAYS WHICH INTERFACE WITH THE ASTRONAUT DURING THE NORMAL
060084,000028:                                                                                                  #         SEQUENCE OF EVENTS.
060085,000029:                                                                                                  #      4. MISC. DISPLAYS- ALL DISPLAYS NOT HANDLED BY THE DISPLAY INTERFACEROUTINES. THESE INCLUDE SUCH DISPLAYS AS
060086,000030:                                                                                                  #         MM DISPLAYS AND SPECIAL PURPOSE DISPLAYS HANDLED BY PINBALL.
060087,000031: 
060088,000032: 
060089,000033:                                                                                                  #      5. ASTRONAUT INITIATED DISPLAYS- ALL DISPLAYS INITIATED EXTERNALLY.
060090,000034:                                                                                                  #  THE FOLLOWING TERMS ARE USED TO DESCRIBE THE STATUS OF DISPLAYS-
060091,000035: 
060092,000036:                                                                                                  #      1. ACTIVE-THE DISPLAY WHICH IS (1) BEING DISPLAYED TO THE ASTRONAUT AND WAITING FOR A RESPONSE OR
060093,000037:                                                                                                  #         (2) WAITING FIRST IN LINE FOR THE ASTRONAUT TO FINISH USING THE DSKY OR (3) BEING DISPLAYED ON THE DSKY
060094,000038:                                                                                                  #         BUT NOT WAITING FOR A RESPONSE.
060095,000039:                                                                                                  #      2. INACTIVE -A DISPLAY WHICH HAS (1) BEEN ACTIVE BUT WAS INTERRUPTEDBY A DISPLAY OF HIGHER PRIORITY,
060096,000040:                                                                                                  #         (2) BEEN PUT INTO THE WAITING LIST AT TIME IT WAS REQUESTED DUE TO THE FACT A HIGHER PRIORITY DISPLAY
060097,000041:                                                                                                  #         WAS ALREADY GOING, (3) BEEN INTERRUPTED BY THE ASTRONAUT (CALLED A PINBRANCH CONDITION, SINCE THIS TYPE
060098,000042:                                                                                                  #         OF INACTIVE DISPLAY IS USUALLY REACTIVATED ONLY BY PINBALL) OR (4) A DISPLAY WHICH HAS FINISHED BUT STILL
060099,000043:                                                                                                  #         HAS INFO SAVED FOR RESTART PURPOSES.
060100,000044: 
060101,000045: 
060102,000046:                                                                                                  #  DISPLAY PRIORITIES WORK AS FOLLOWS-
060103,000047: 
060104,000048:                                                                                                  #    INTERRUPTS-
060105,000049:                                                                                                  #      1. THE ASTRONAUT CAN INTERRUPT ANY DISPLAY WITH AN EXTERNAL DISPLAY REQUEST.
060106,000050:                                                                                                  #      2. INTERNAL DISPLAYS CAN NOT BE SENT OUT WHEN THE ASTRONAUT IS USING THE DSKY.
060107,000051:                                                                                                  #      3. PRIORITY DISPLAYS INTERRUPT ALL OTHER TYPES OF INTERNAL DISPLAYS. A PRIORITY DISPLAY INTERRUPTING ANOTHER
060108,000052:                                                                                                  #         PRIORITY DISPLAY WILL CAUSE AN ABORT UNLESS BIT14 IS SET FOR THE LINUS ROUTINE.
060109,000053:                                                                                                  #      4. A MARK DISPLAY INTERRUPTS ANY NORMAL DISPLAY.
060110,000054:                                                                                                  #      5. A MARK THAT INTERRUPTS A MARK COMPLETELY REPLACES IT.
060111,000055: 
060112,000056:                                                                                                  #    ORDER OF WAITING DISPLAYS-
060113,000057:                                                                                                  #      1. ASTRONAUT EXTERNAL USE
060114,000058:                                                                                                  #      2. PRIORITY
060115,000059:                                                                                                  #      3. INTERRUPTED MARK

060123,000067:                                                                                                  #  THE DISPLAY ROUTINES ARE INTENDED TO SERVE AS AN INTERFACE BETWEEN THE USER AND PINBALL.  THE
060124,000068:                                                                                                  #  FOLLOWING STATEMENTS CAN BE MADE ABOUT NORMAL DISPLAYS AND PRIORITY DISPLAYS (A DESCRIPTION OF MARK ROUTINES
060125,000069:                                                                                                  #  WILL FOLLOW LATER):
060126,000070: 
060127,000071:                                                                                                  #       1.  ALL ROUTINES THAT END IN R HAVE AN IMMEDIATE RETURN TO THE USER.  FOR ALL FLASHING DISPLAYS THIS RETURN
060128,000072:                                                                                                  #           IS TO THE USERS CALL CADR +4.  FOR THE ONLY NON FLASHING IMMEDIATE RETURN DISPLAY (GODSPR) THIS RETURN
060129,000073:                                                                                                  #           IS TO THE USERS CALLING LOC +1.
060130,000074:                                                                                                  #       2.  ALL ROUTINES NOT ENDING IN R DO NOT DO AN IMMEDIATE RETURN TO THE USER.
060131,000075:                                                                                                  #       3.  ALL ROUTINES THAT END IN R START A SEPARATE JOB (MAKEPLAY) WITH USERS JOB PRIORITY.
060132,000076:                                                                                                  #       4.  ALL ROUTINES NOT ENDING IN R BRANCH DIRECTLY TO MAKEPLAY WHICH MAKES THESE DISPLAYS A PART OF THE
060133,000077:                                                                                                  #           USERS JOB.
060134,000078:                                                                                                  #       5.  ALL DISPLAY ROUTINES ARE CALLED VIA BANKCALL.
060135,000079:                                                                                                  #       6.  TO RESTART A DISPLAY THE USER WILL GENERALLY USE A PHASE OF ONE WITH DESIRED RESTART GROUP (SEE
060136,000080:                                                                                                  #           DESCRIPTION OF RESTARTS).
060137,000081:                                                                                                  #       7.  ALL FLASHING DISPLAYS HAVE 3 RETURNS TO THE USER FROM ASTRONAUT RESPONSES.  A TERMINATE (V34) BRANCHES
060138,000082:                                                                                                  #           TO THE USERS CALL CADR +1.  A PROCEED (V33) BRANCHES TO THE USERS CALL CADR +2.  AN ENTER OR RECYCLE
060139,000083:                                                                                                  #           (V32) BRANCHES TO THE USERS CALL CADR +3.
060140,000084:                                                                                                  #       8.  ALL ROUTINES MUST BE USED UNDER EXECUTIVE CONTROL.
