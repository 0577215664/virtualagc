Source Code

These source-code files were derived from scans of the Colossus 237 program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

059765,000015:                                                                                                  #  DISPLAYS CAN BE CLASSIFIED INTO THE FOLLOWING CATEGORIES-
059766,000016: 
059767,000017:                                                                                                  #      1. PRIORITY DISPLAYS- DISPLAYS WHICH TAKE PRIORITY OVER ALL OTHER DISPLAYS. USUALLY THESE DISPLAYS ARE SENT
059768,000018:                                                                                                  #         OUT UNDER CRITICAL ALARM CONDITIONS.
059769,000019:                                                                                                  #      2. EXTENDED VERB DISPLAYS- ALL EXTENDED VERBS AND MARK ROUTINES SHOULD USE EXTENDED VERB (MARK) DISPLAYS.
059770,000020:                                                                                                  #      3. NORMAL DISPLAYS- ALL MISSION PROGRAM DISPLAYS WHICH INTERFACE WITH THE ASTRONAUT DURING THE NORMAL
059771,000021:                                                                                                  #         SEQUENCE OF EVENTS.
059772,000022:                                                                                                  #      4. MISC. DISPLAYS- ALL DISPLAYS NOT HANDLED BY THE DISPLAY INTERFACEROUTINES. THESE INCLUDE SUCH DISPLAYS AS
059773,000023:                                                                                                  #         MM DISPLAYS AND SPECIAL PURPOSE DISPLAYS HANDLED BY PINBALL.
059774,000024: 
059775,000025: 
059776,000026:                                                                                                  #      5. ASTRONAUT INITIATED DISPLAYS- ALL DISPLAYS INITIATED EXTERNALLY.
059777,000027:                                                                                                  #  THE FOLLOWING TERMS ARE USED TO DESCRIBE THE STATUS OF DISPLAYS-
059778,000028: 
059779,000029:                                                                                                  #      1. ACTIVE-THE DISPLAY WHICH IS (1) BEING DISPLAYED TO THE ASTRONAUT AND WAITING FOR A RESPONSE OR
059780,000030:                                                                                                  #         (2) WAITING FIRST IN LINE FOR THE ASTRONAUT TO FINISH USING THE DSKY OR (3) BEING DISPLAYED ON THE DSKY
059781,000031:                                                                                                  #         BUT NOT WAITING FOR A RESPONSE.
059782,000032:                                                                                                  #      2. INACTIVE -A DISPLAY WHICH HAS (1) BEEN ACTIVE BUT WAS INTERRUPTEDBY A DISPLAY OF HIGHER PRIORITY,
059783,000033:                                                                                                  #         (2) BEEN PUT INTO THE WAITING LIST AT TIME IT WAS REQUESTED DUE TO THE FACT A HIGHER PRIORITY DISPLAY
059784,000034:                                                                                                  #         WAS ALREADY DOING, (3) BEEN INTERRUPTED BY THE ASTRONAUT (CALLED A PINBRANCH CONDITION, SINCE THIS TYPE
059785,000035:                                                                                                  #         OF INACTIVE DISPLAY IS USUALLY REACTIVATED ONLY BY PINBALL) OR (4) A DISPLAY WHICH HAS FINISHED BUT STILL
059786,000036:                                                                                                  #         HAS INFO SAVED FOR RESTART PURPOSES.
059787,000037: 
059788,000038: 
059789,000039:                                                                                                  #  DISPLAY PRIORITIES WORK AS FOLLOWS-
059790,000040: 
059791,000041:                                                                                                  #    INTERRUPTS-
059792,000042:                                                                                                  #      1. THE ASTRONAUT CAN INTERRUPT ANY DISPLAY WITH AN EXTERNAL DISPLAY REQUEST.
059793,000043:                                                                                                  #      2. INTERNAL DISPLAYS CAN NOT BE SENT OUT WHEN THE ASTRONAUT IS USING THE DSKY.
059794,000044:                                                                                                  #      3. PRIORITY DISPLAYS INTERRUPT ALL OTHER TYPES OF INTERNAL DISPLAYS. A PRIORITY DISPLAY INTERRUPTING ANOTHER
059795,000045:                                                                                                  #         PRIORITY DISPLAY WILL CAUSE AN ABORT UNLESS BIT14 IS SET FOR THE LINUS ROUTINE.
059796,000046:                                                                                                  #      4. A MARK DISPLAY INTERRUPTS ANY NORMAL DISPLAY.
059797,000047:                                                                                                  #      5. A MARK THAT INTERRUPTS A MARK COMPLETELY REPLACES IT.
059798,000048: 
059799,000049:                                                                                                  #    ORDER OF WAITING DISPLAYS-

059801,000051:                                                                                                  #      2. PRIORITY

059810,000060:                                                                                                  #  THE DISPLAY ROUTINES ARE INTENDED TO SERVE AS AN INTERFACE BETWEEN THE USER AND PINBALL.  THE
