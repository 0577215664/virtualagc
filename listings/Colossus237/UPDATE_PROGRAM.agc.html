Source Code

These source-code files were derived from scans of the Colossus 237 (Apollo 8 Command Module) program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9 CM) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

062059,000028:                                                                                                  #  FUNCT. DESCR:   P27 (THE UPDATE PROGRAM) PROCESSES COMMANDS AND DATA
062060,000029:                                                                                                  #                      INSERTIONS REQUESTED BY THE GROUND VIA UPLINK.
062061,000030:                                                                                                  #                      THE P27 PROGRAM WILL ACCEPT UPDATES
062062,000031:                                                                                                  #                      ONLY DURING P00 FOR THE LM, AND ONLY DURING P00,
062063,000032:                                                                                                  #                  P02, AND FRESH START FOR THE CSM
062064,000033: 
062065,000034:                                                                                                  #  CALLING SEQ:    PROGRAM IS INITIATED BY UPLINK ENTRY OF VERBS 70, 71, 72 AND 73.
062066,000035:

062074,000043:                                                                                                  #  RESTARTS:       P27 IS RESTART PROTECTED IN TWO WAYS...
062075,000044:                                                                                                  #                  1. PRIOR TO VERIFLAG INVERSION(WHICH IS CAUSED BY THE GROUND/ASTRONAUT'S VERIFICATION OF UPDATE
062076,000045:                                                                                                  #                     DATA BY SENDING A V33E WHEN V21N02 IS FLASHING)---
062077,000046:                                                                                                  #                     NO PROTECTION EXCEPT PRE-P27 MODE IS RESTORED, COAST + ALIGN DOWNLIST IS SELECTED AND UPLINK
062078,000047:                                                                                                  #                     ACTIVITY LIGHT IS TURNED OFF.(JUST AS IF A V34E WAS SENT DURING P27 DATA LOADS).
062079,000048:                                                                                                  #                     V70,V71,V72 OR V73 WILL HAVE TO BE COMPLETELY RESENT BY USER.
062080,000049:                                                                                                  #                  2. AFTER VERIFLAG INVERSION(WHEN UPDATE OF THE SPECIFIED ERASABLES IS BEING PERFORMED)---
062081,000050:                                                                                                  #                     PROTECTED AGAINST RESTARTS.
062082,000051: 
062083,000052:                                                                                                  #  DEBRIS:         UPBUFF   (20D)  TEMP STORAGE FOR ADDRESSES AND CONTENTS.
062084,000053:                                                                                                  #                  UPVERB   (1)    VERB NUMBER MINUS 70D (E.G. FOR V72, UPVERB = 72D - 70D = 2)
062085,000054:                                                                                                  #                  UPOLDMOD (1)    FOR MAJOR MODE INTERRUPTED BY P27.
062086,000055:                                                                                                  #                  COMPNUMB (1)    TOTAL NUMBER OF COMPONENTS TO BE TRANSMITTED.
062087,000056:                                                                                                  #                  UPCOUNT  (1)    ACTUAL NUMBER OF COMPONENTS RECEIVED.
062088,000057:                                                                                                  #                  UPTEMP   (1)    SCRATCH, BUT USUALLY CONTAINS COMPONENT NUMBER TO BE CHANGED DURING VERIFY CYCLE
