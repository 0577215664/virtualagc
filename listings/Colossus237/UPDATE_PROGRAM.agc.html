Source Code

These source-code files were derived from scans of the Colossus 237 program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

061756,000023:                                                                                                  #  FUNCT. DESCR:   P27 (THE UPDATE PROGRAM) PROCESSES COMMANDS AND DATA
061757,000024:                                                                                                  #                      INSERTIONS REQUESTED BY THE GROUND VIA UPLINK.
061758,000025:                                                                                                  #                      THE P27 PROGRAM WILL ACCEPT UPDATES
061759,000026:                                                                                                  #                      ONLY DURING P00 FOR THE LM, AND ONLY DURINT P00,
061760,000027:                                                                                                  #                  P02, AND FRESH START FOR THE CSM
061761,000028:

061763,000030:

061770,000037: 
061771,000038:                                                                                                  #  RESTARTS:       P27 IS RESTART PROTECTED IN TWO WAYS...
061772,000039:                                                                                                  #                  1. PRIOR TO VERIFLAG INVERSION(WHICH IS CAUSED BY THE GROUND/ASTRONAUT'S VERIFICATION OF UPDATE
061773,000040:                                                                                                  #                     DATA BY SENDING A V33E WHEN V21N02 IS FLASHING)---
061774,000041:                                                                                                  #                     NO PROTECTION EXCEPT PRE-P27 MODE IS RESTROED, COAST + ALIGN DOWNLIST IS SELECTED AND UPLINE
061775,000042:                                                                                                  #                     ACTIVITY LIGHT IS TURNED OFF.(JUST AS IF A V34E WAS SENT DURING P27 DATA LOADS).
061776,000043:                                                                                                  #                     V70,V71,V72 OR V73 WILL HAVE TO BE COMPLETELY RESENT BY USER.
061777,000044:                                                                                                  #                  2. AFTER VERIFLAG INVERSION(WHEN UPDATE OF THE SPECIFIED ERASABLES IS BEING PERFORMED)---
061778,000045:                                                                                                  #                     PROTECTED AGAINST RESTARTS.
061779,000046: 
061780,000047:                                                                                                  #  DEBRIS:         UPBUFF   (20D)  TEMP STORAGE FOR ADDRESSES AND CONTENTS.
061781,000048:                                                                                                  #                  UPVERB   (1)    VERB NUMBER MINUS 70D (E.G. FOR V72, UPVERB = 72D - 70D = 2)
061782,000049:                                                                                                  #                  UPOLDMOD (1)    FOR MAJOR MODE INTERRUPTED BY P27.
061783,000050:                                                                                                  #                  COMPNUMB (1)    TOTAL NUMBER OF COMPONENTS TO BE TRANSMITTED.
061784,000051:                                                                                                  #                  UPCOUNT  (1)    ACTUAL NUMBER OF COMPONENTS RECEIVED.
061785,000052:                                                                                                  #                  UPTEMP   (1)    SCRATCH, BUT USUALLY CONTAINS COMPONENT NUMBER TO BE CHANGED DURING VERIFY CYCLE
