Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. 057914,000069:                                                                                                 #  PHASCHNG IS THE MAIN WAY OF MAKING PHASE CHANGES FOR RESTARTS.  THERE ARE THREE FORMS OF PHASCHNG, KNOWN AS TYPE
057915,000070:                                                                                                 #  A, TYPE B, AND TYPE C. THEY ARE ALL CALLED AS FOLLOWS, WHERE OCT XXXXX   CONTAINS THE PHASE INFORMATION,
057916,000071: 
057917,000072:                                                                                                 #                                                   TC     PHASCHNG
057918,000073:                                                                                                 #                                                   OCT    XXXXX
057919,000074: 
057920,000075:                                                                                                 #  TYPE A IS CONCERNED WITH FIXED PHASE CHANGES, THAT IS, PHASE INFORMATIONTHAT IS STORED PERMANENTLY.  THESE
057921,000076:                                                                                                 #  OPTIONS ARE, WHERE G STANDS FOR A GROUP AND .X FOR THE PHASE,
057922,000077: 
057923,000078:                                                                                                 #                  G.0             INACTIVE, WILL NOT PERMIT A GROUP G RESTART
057924,000079:                                                                                                 #                  G.1             WILL CAUSE THE LAST DISPLAY TO BE REACTIVATED, USED MAINLY IN MANNED FLIGHTS
057925,000080:                                                                                                 #                  G.EVEN          A DOUBLE TABLE RESTART, CAN CAUSE ANY COMBINATION OF TWO JOBS, TASKS, AND/OR
057926,000081:                                                                                                 #                                  LONGCALL TO BE RESTARTED.
057927,000082:                                                                                                 #                  G.ODD NOT .1    A SINGLE TABLE RESTART, CAN CAUSE EITHER A JOB, TASK, OR LONGCALL RESTART
057928,000083: 
057929,000084:                                                                                                 #  THIS INFORMATION IS PUT INTO THE OCTAL WORD AFTER TC PHASCHNG AS FOLLOWS
057930,000085: 
057931,000086:                                                                                                 #                  TL0 00P PPP PPP GGG
057932,000087: 
057933,000088:                                                                                                 #  WHERE EACH LETTER OR NUMBER STANTS FOR A BIT.  THE G:S STAND FOR THE GROUP, OCTAL 1 - 7, THE P:S FOR THE PHASE,
057934,000089:                                                                                                 #  OCTAL 0 - 127.  0:S MUST BE 0.              IF ONE WISHES TO HAVE THE TBASE OF GROUP G TO BE SET AT THIS TIME,
057935,000090:                                                                                                 #  T IS SET TO 1, OTHERWISE IT IS SET TO 0.  SIMIARLY IF ONE WISHES TO SET LONGBASE, THEN L IS SET TO 1, OTHERWISE
057936,000091:                                                                                                 #  IT IS SET TO 0.  SOME EXAMLES,
057937,000092: 
057938,000093:                                                                                                 #                                                   TC     PHASCHNG        THIS WILL CAUSE GROUP 3 TOBE SET TO 0,
057939,000094:                                                                                                 #                                                   OCT    00003           MAKING GROUP 3 INACTIVE
057940,000095: 
057941,000096:                                                                                                 #                                                   TC     PHASCHNG        IF A RESTART OCCURS THIS WOULD CAUSE
057942,000097:                                                                                                 #                                                   OCT    00012           GROUP 2 TO RESTART THE LAST DISPLAY
057943,000098: 
057944,000099:                                                                                                 #                                                   TC     PHASCHNG        THIS SETS THE TBASE OF GROUP 4 AND IN
057945,000100:                                                                                                 #                                                   OCT    40064           CASE OF A RESTART WOULD START UP THE TWO
057946,000101:                                                                                                 #                                                                          THINGS LOCATED IN THE DOUBLE 4.6 RESTART
057947,000102:                                                                                                 #                                                                          LOCATION
057948,000103:                                                                                                 #                                                   TC     PHASCHNG        THIS SETS LONGBASE AND UPON A RESTART
057949,000104:                                                                                                 #                                                   OCT    20135           CAUSES 5.13 TO BE RESTARTED (SINCE
057950,000105:                                                                                                 #                                                                          LONGBASE WAS SET THIS SINGLE ENTRY
057951,000106:                                                                                                 #                                                                          SHOULD BE A LONGCALL)

057956,000111:                                                                                                 #  TYPE C PHASCHNG CONTAINS THE VARIABLE TYPE OF PHASCHNG INFORMATION. INSTEAD OF THE INFORMATION BEING IN A
057957,000112:                                                                                                 #  PERMANENT FORM, ONE STORES THE DESIRED RESTART INFORMATION IN A VARIABKE LOCATION. THE BITS ARE AS FOLLOWS,

057959,000114:                                                                                                 #                  TL0 1AD XXX CJW GGG
057960,000115: 
057961,000116:                                                                                                 #  WHERE EACH LETTER OR NUMBER STANDS FOR A BIT.  THE G:S STAND FOR THE GROUP, OCTAL 1 - 7.  IF THE RESTART IS TO
057962,000117:                                                                                                 #  BE BY WAITLIST, W IS SET TO 1, IF IT IS A JOB, J IS SET TO 1, IF IT IS A LONGCALL, C IS SET TO 1. ONLY ONE OF
057963,000118:                                                                                                 #  THESE THREE BIT S MAY BE SET.  X:S ARE IGNORED  1 MUST BE 1, AND 0 MUST  BE 0.  AGAIN T STANDS FOR THE TBASE,
057964,000119:                                                                                                 # # Page 1371
057965,000120:                                                                                                 #  AND L FOR LONGBASE.  THE BITS A AND D ARE CONCERNED WITH THE VARIABLE INFORMATION. IF D IS SET TO 1, A PRIORITY
057966,000121:                                                                                                 #  OR DELTA TIME WILL BE READ FROM THE NEXT LOCATION AFTER THE OCTAL INFORMATION, IF THIS IS TO BE INDIRECT, THAT
057967,000122:                                                                                                 #  IS, THE NAME OF A LOCATION CONT+INING THE INFORMATION (DELTA TIME ONLY), THEN THIS IS GIVEN AS THE -GENADR OF
057968,000123:                                                                                                 #  THAT LOCATION WHICH CONTAINS THE DELTA TIME.  IF THE OLD PRIORITY OR DELTA TIME IS TO BE USED, THAT WHICH IS
057969,000124:                                                                                                 #  ALREADY IN THE VARIABLE STORAGE, THEN D IS SET TO 0. NEXT THE A BIT IS USED.  IF IT IS SET TO 0, THE ADDRESS
057970,000125:                                                                                                 #  THAT WOULD BE RESTARTED DURING A RESTART IS THE NEXT LOCATION AFTER THE  PHASE INFORMATION, THAT IS, EITHER
