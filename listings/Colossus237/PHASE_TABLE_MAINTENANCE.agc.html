Source Code

These source-code files were derived from scans of the Colossus 237 program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

057930,000069:                                                                                                  #  PHASCHNG IS THE MAIN WAY OF MAKING PHASE CHANGES FOR RESTARTS.  THERE ARE THREE FORMS OF PHASCHNG, KNOWN AS TYPE
057931,000070:                                                                                                  #  A, TYPE B, AND TYPE C. THEY ARE ALL CALLED AS FOLLOWS, WHERE OCT XXXXX   CONTAINS THE PHASE INFORMATION,
057932,000071: 
057933,000072:                                                                                                  #                                                   TC     PHASCHNG
057934,000073:                                                                                                  #                                                   OCT    XXXXX
057935,000074: 
057936,000075:                                                                                                  #  TYPE A IS CONCERNED WITH FIXED PHASE CHANGES, THAT IS, PHASE INFORMATIONTHAT IS STORED PERMANENTLY.  THESE
057937,000076:                                                                                                  #  OPTIONS ARE, WHERE G STANDS FOR A GROUP AND .X FOR THE PHASE,
057938,000077: 
057939,000078:                                                                                                  #                  G.0             INACTIVE, WILL NOT PERMIT A GROUP G RESTART
057940,000079:                                                                                                  #                  G.1             WILL CAUSE THE LAST DISPLAY TO BE REACTIVATED, USED MAINLY IN MANNED FLIGHTS
057941,000080:                                                                                                  #                  G.EVEN          A DOUBLE TABLE RESTART, CAN CAUSE ANY COMBINATION OF TWO JOBS, TASKS, AND/OR
057942,000081:                                                                                                  #                                  LONGCALL TO BE RESTARTED.
057943,000082:                                                                                                  #                  G.ODD NOT .1    A SINGLE TABLE RESTART, CAN CAUSE EITHER A JOB, TASK, OR LONGCALL RESTART
057944,000083: 
057945,000084:                                                                                                  #  THIS INFORMATION IS PUT INTO THE OCTAL WORD AFTER TC PHASCHNG AS FOLLOWS
057946,000085: 
057947,000086:                                                                                                  #                  TL0 00P PPP PPP GGG
057948,000087: 
057949,000088:                                                                                                  #  WHERE EACH LETTER OR NUMBER STANTS FOR A BIT.  THE G:S STAND FOR THE GROUP, OCTAL 1 - 7, THE P:S FOR THE PHASE,
057950,000089:                                                                                                  #  OCTAL 0 - 127.  0:S MUST BE 0.              IF ONE WISHES TO HAVE THE TBASE OF GROUP G TO BE SET AT THIS TIME,
057951,000090:                                                                                                  #  T IS SET TO 1, OTHERWISE IT IS SET TO 0.  SIMIARLY IF ONE WISHES TO SET LONGBASE, THEN L IS SET TO 1, OTHERWISE
057952,000091:                                                                                                  #  IT IS SET TO 0.  SOME EXAMLES,
057953,000092: 
057954,000093:                                                                                                  #                                                   TC     PHASCHNG        THIS WILL CAUSE GROUP 3 TOBE SET TO 0,
057955,000094:                                                                                                  #                                                   OCT    00003           MAKING GROUP 3 INACTIVE
057956,000095: 
057957,000096:                                                                                                  #                                                   TC     PHASCHNG        IF A RESTART OCCURS THIS WOULD CAUSE
057958,000097:                                                                                                  #                                                   OCT    00012           GROUP 2 TO RESTART THE LAST DISPLAY
057959,000098: 
057960,000099:                                                                                                  #                                                   TC     PHASCHNG        THIS SETS THE TBASE OF GROUP 4 AND IN
057961,000100:                                                                                                  #                                                   OCT    40064           CASE OF A RESTART WOULD START UP THE TWO
057962,000101:                                                                                                  #                                                                          THINGS LOCATED IN THE DOUBLE 4.6 RESTART
057963,000102:                                                                                                  #                                                                          LOCATION
057964,000103:                                                                                                  #                                                   TC     PHASCHNG        THIS SETS LONGBASE AND UPON A RESTART
057965,000104:                                                                                                  #                                                   OCT    20135           CAUSES 5.13 TO BE RESTARTED (SINCE
057966,000105:                                                                                                  #                                                                          LONGBASE WAS SET THIS SINGLE ENTRY
057967,000106:                                                                                                  #                                                                          SHOULD BE A LONGCALL)
057968,000107:                                                                                                  #                                                   TC     PHASCHNG        SINCE BOTH TBASE4 AND LONGBASE ARE SET,
057969,000108:                                                                                                  #                                                   OCT    60124           4.12 SHOULD CONTAIN BOTH A TASK AND A
057970,000109:                                                                                                  #                                                                          LONGCALL TO BE RESTARTED

057972,000111:                                                                                                  #  TYPE C PHASCHNG CONTAINS THE VARIABLE TYPE OF PHASCHNG INFORMATION. INSTEAD OF THE INFORMATION BEING IN A
057973,000112:                                                                                                  #  PERMANENT FORM, ONE STORES THE DESIRED RESTART INFORMATION IN A VARIABKE LOCATION. THE BITS ARE AS FOLLOWS,
057974,000113: 
057975,000114:                                                                                                  #                  TL0 1AD XXX CJW GGG
057976,000115: 
057977,000116:                                                                                                  #  WHERE EACH LETTER OR NUMBER STANDS FOR A BIT.  THE G:S STAND FOR THE GROUP, OCTAL 1 - 7.  IF THE RESTART IS TO
057978,000117:                                                                                                  #  BE BY WAITLIST, W IS SET TO 1, IF IT IS A JOB, J IS SET TO 1, IF IT IS A LONGCALL, C IS SET TO 1. ONLY ONE OF
057979,000118:                                                                                                  #  THESE THREE BIT S MAY BE SET.  X:S ARE IGNORED  1 MUST BE 1, AND 0 MUST  BE 0.  AGAIN T STANDS FOR THE TBASE,
