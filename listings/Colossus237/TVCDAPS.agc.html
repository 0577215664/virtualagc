Source Code

These source-code files were derived from scans of the Colossus 237 (Apollo 8 Command Module) program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9 CM) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

040032,000016:                                                                                                  #  PROGRAM NAME....TVCDAP, CONSISTING OF PITCHDAP, YAWDAP, ETC.
040033,000017:                                                                                                  #  LOG SECTION....TVCDAP           SUBROUTINE....DAPCSM
040034,000018:                                                                                                  #  MOD BY ENGEL                    DATE....27 OCT, 1967
040035,000019: 
040036,000020:                                                                                                  #  FUNCTIONAL DESCRIPTION....
040037,000021: 
040038,000022:                                                                                                  #       SELF-PERPETUATING T5 TASKS WHICH GENERATE THE COMMAND SIGNALS
040039,000023:                                                                                                  #       FOR THE PITCH AND YAW SPS GIMBAL ACTUATORS DURING TVC (SPS) BURNS,
040040,000024:                                                                                                  #       IN RESPONSE TO BODY-AXIS RATE COMMANDS FROM CROSS-PRODUCT STEERING
040041,000025:                                                                                                  #       (S40.8).  IF NO STEERING (IMPULSIVE BURNS) MAINTAINS ATTITUDE-HOLD
040042,000026:                                                                                                  #       ABOUT THE REFERENCE (INITIAL) DIRECTIONS (ZERO RATE COMMANDS).
040043,000027: 
040044,000028:                                                                                                  #       THE PITCH AND YAW LOOPS ARE SEPARATE, BUT STRUCTURED IDENTICALLY.
040045,000029:                                                                                                  #       EACH ATTITUDE-RATE LOOP INCLUDES GIMBAL ANGLE RATE DERIVATION,
040046,000030:                                                                                                  #       GIMBAL/BODY AXIS TRANSFORMATION, BODY-AXIS ATTITUDE ERROR
040047,000031:                                                                                                  #       INTEGRATION WITH ERROR LIMITING, THE CSM/LEM FILTER  OR THE BRANCH
040048,000032:                                                                                                  #       POINTS FOR THE CSM-ALONE (GEN3DAP) FILTER,  OUTPUT LIMITER,
040049,000033:                                                                                                  #       CG-OFFSET TRACKER FILTER, AND THE CG-TRACKER MINOR LOOP.
040050,000034: 
040051,000035:                                                                                                  #       THE DAPS ARE CYCLIC, CALLING EACH OTHER AT 1/2  THE DAP SAMPLE
040052,000036:                                                                                                  #       TIME, AS DETERMINED BY T5TVCDT. THE ACTUATOR COMMANDS ARE
040053,000037:                                                                                                  #       REGENERATED AS ANALOG VOLTAGES BY THE OPTICS ERROR COUNTERS, WHICH
040054,000038:                                                                                                  #       TRANSMIT THE SIGNAL TO THE ACTUATOR SERVOS WHEN THERE IS PROPER CDU
040055,000039:                                                                                                  #       MODING.
040056,000040: 
040057,000041:                                                                                                  #       REFERENCES FOR THE CSM/LEM FILTER DESIGN INCLUDE R503 BY  STUBBS
040058,000042:                                                                                                  #       (MIT IL OCT 1965) AND SGA MEMO R26-65 BY MARTIN (MIT IL OCT 1965).
040059,000043:                                                                                                  #       REFERENCES FOR THE CSM FILTER DESIGN (SEE GEN3DAP) INCLUDE R533 BY
040060,000044:                                                                                                  #       LU  (MIT IL JUNE 1966).
040061,000045: 
040062,000046:                                                                                                  #       OPERATIONAL ASPECTS OF THE INTEGRATED CONTROL PACKAGE, WITH DESIGN-
040063,000047:                                                                                                  #       NOMINAL PARAMETER VALUES ARE DISCUSSED IN AG R336-67 BY ENGEL
040064,000048:                                                                                                  #       (MIT IL OCT 1967) AND SGA MEMO R18-67 BY SCHLUNDT (MIT IL OCT 1967)
040065,000049: 
040066,000050:                                                                                                  #  CALLING SEQUENCE.... (TYPICALLY)
040067,000051: 
040068,000052:                                                                                                  #       T5 CALL OF TVCDAPON (P40-P47) BY IGNOVER (P40-P47)
