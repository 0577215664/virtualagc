Source Code

These source-code files were derived from scans of the Colossus 237 program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

039625,000013:                                                                                                  # # Page 913
039626,000014:                                                                                                  #  NAME....TVCRESTART PACKAGE,  CONSISTING OF REDOTVC, ENABL1, 2, CMDSOUT, PHSCHK2, ETC.
039627,000015:                                                                                                  #  LOG SECTION....TVCRESTART PACKAGE    SUBROUTINE....DAPCSM
039628,000016:                                                                                                  #  MOD BY ENGEL                         DATE....19 OCT, 1967
039629,000017: 
039630,000018:                                                                                                  #  FUNCTIONAL DESCRIPTION....
039631,000019: 
039632,000020:                                                                                                  #      *RESTART-PROOFS THE TVC DAPS, INCLUDING PITCHDAP, YAWDAP,
039633,000021:                                                                                                  #       TVCEXECUTIVE, ROLL DAP, TVCINIT4, TVCDAPON, AND STROKE TEST
039634,000022:                                                                                                  #      *TVC RESTARTS DESERVE SPECIAL CONSIDERATION IN SEVERAL AREAS.
039635,000023:                                                                                                  #       RESTART DOWN-TIME IS IMPORTANT BECAUSE OF THE TRANSIENTS INTRODUCED
039636,000024:                                                                                                  #       BY THE THRUST VECTOR RETURN TO THE ACTUATOR MECHANICAL NULLS
039637,000025:                                                                                                  #       FOLLOWING TVC- AND OPTICS-ERROR-COUNTER-DISENABLES (CHANNEL 12).
039638,000026:                                                                                                  #       TVC    USES A MIXTURE OF WAITLIST, T5, T6, AND JOB CALLS. THERE IS
039639,000027:                                                                                                  #       FILTER MEMORY (UP TO 7TH ORDER) TO BE PROTECTED IF WILD TRANSIENTS
039640,000028:                                                                                                  #       ARE TO BE AVOIDED. SEVERAL COUNTERS ARE INVOLVED FOR TIMING TVC
039641,000029:                                                                                                  #       EVENTS SUCH AS SWITCHOVER AND STROKE TEST STARTUPS AND RE-STARTUPS.
039642,000030:                                                                                                  #       THE TVC GAINS ARE DECREMENTED. THE GIMBAL TRIM ESTIMATORS AND THE
039643,000031:                                                                                                  #       BODY AXIS ATTITUDE ERROR INTEGRATORS INVOLVE DIGITAL SUMMATION.
039644,000032:                                                                                                  #       DIGITAL DIFFERENTIATORS ARE INVOLVED IN THE BODY AXIS RATE ESTIMA-
039645,000033:                                                                                                  #       TIONS AND IN THE OUTPUTTING OF ACTUATOR COMMANDS. THERE IS AN
039646,000034:                                                                                                  #       OFFSET-TRACKER-FILTER TO PROTECT. ETC., ETC.
039647,000035:                                                                                                  #      *THOSE QUANTITIES WHICH MUST BE PROTECTED ARE STORED IN TEMPORARY
039648,000036:                                                                                                  #       REGISTERS AS THEY ARE COMPUTED, FOR UPDATING THE REAL REGISTERS
039649,000037:                                                                                                  #       DURING COPYCYCLES.
039650,000038:                                                                                                  #      *THE SEVERAL COPYCYCLES ARE EACH PROTECTED BY PHASE POINTS AT THEIR
039651,000039:                                                                                                  #       BEGINNING AND AT THEIR TERMINATION. THE PHASE POINTS ARE SIMPLY
039652,000040:                                                                                                  #       ..INCR.. INSTRUCTIONS,   EITHER ..INCR TVCEXPHS.. FOR COPYCYCLES
039653,000041:                                                                                                  #       IN THE TVCEXECUTIVE, OR ..INCR TVCPHASE.. FOR THE PITCH AND YAW
039654,000042:                                                                                                  #       COPYCYCLES. INDEXING ON EACH OF THESE POINTERS THEN PERMITS A
039655,000043:                                                                                                  #       RETURN TO THE APPROPRIATE RESTART POINTS.
039656,000044:                                                                                                  #      *IF A RESTART OCCURS DURING EITHER COPYCYCLE, THAT COPYCYCLE IS
039657,000045:                                                                                                  #       COMPLETED. THEN THE NORMAL TVCINIT4....DAPINIT....PITCHDAP STARTUP
039658,000046:                                                                                                  #       SEQUENCE IS CALLED UPON TO GET THINGS GOING AGAIN.
039659,000047:                                                                                                  #      *TVC-ENABLE AND OPTICS-ERROR-COUNTER ENABLE MUST BE SET ASAP
039660,000048:                                                                                                  #       (ALLOWING FOR PROCEDURAL DELAYS). THEN THE ENGINES ARE COMMANDED
039661,000049:                                                                                                  #       TO THE P,YACTOFF TRIM VALUES. THE DAPS ARE THEN READY TO GO ON THE
039662,000050:                                                                                                  #       AIR, WITH THE REGULAR STARTUP SEQUENCE, EITHER AT MRCLEAN FOR A
039663,000051:                                                                                                  #       COMPLETE INITIALIZATION OR AT TVCINIT4 FOR A PARTIAL INITIALIZATION
039664,000052:                                                                                                  #      *FOR RESTARTS PRIOR TO THE SETTING OF THE T5 BITS IN IGNOVER THE
039665,000053:                                                                                                  #       PRE40.6 SECTION OF S40.6 TAKES CARE OF RE-ESTABLISHING TRIMS.
039666,000054:                                                                                                  #      *IF A RESTART OCCURS DURING THE TVCEXEC....TVCEXFIN SEQUENCE THE
039667,000055:                                                                                                  #       COMPUTATIONS WILL BE COMPLETED, STARTING AT THE APPROPRIATE RESTART
039668,000056:                                                                                                  #       POINT, AFTER THE DAPS ARE READY TO GO ON THE AIR.
039669,000057:                                                                                                  #      *IF A RESTART OCCURS PRIOR TO TVCINIT4 (TVCPHAS = -1) E.G. DURING
039670,000058:                                                                                                  #       THE EARLY DAP INITIALIZATION PHASE, THE DAP STARTUP SEQUENCE IS
039671,000059:                                                                                                  #       ENTERED AT MRCLEAN FOR A FULL INITIALIZATION.
039672,000060:                                                                                                  #      *RESTARTS ARE NOT CRITICAL TO THE ROLL DAP PERFORMANCES HENCE THE
039673,000061:                                                                                                  #       ROLL DAP IS MERELY RESTARTED.
039674,000062:                                                                                                  #      *RESTARTS DURING A STROKE TEST (STROKER IS NON-ZERO) WILL CAUSE THE
039675,000063:                                                                                                  #       STROKE TEST TO BE TERMINATED. A NEW V68 ENTRY WILL BE REQUIRED
039676,000064:                                                                                                  # # Page 914
039677,000065:                                                                                                  #       TO GET IT GOING AGAIN (NO AUTOMATIC RESTART).
039678,000066:                                                                                                  #      *REDOTVC IS REACHED FOLLOWING ANY RESTART WHICH FINDS THE T5 BITS
039679,000067:                                                                                                  #       (BITS 15,14 OF FLAGWRD6) SET FOR TVC. IGNOVER PREPARES TVCPHASE =-1
039680,000068:                                                                                                  #       AND TVC EXPHS = 0 JUST BEFORE SETTING THESE BITS, JUST BEFORE
039681,000069:                                                                                                  #       MAKING THE T5 CALL TO TVCDAPON.  T.V.N.G. TAKES OVER THE T5 CLOCK
039682,000070:                                                                                                  #       TO CALL RCSUP/RCSDAPON WHICH RESETS THE T5 BITS(FOR RCS) ON A
039683,000071:                                                                                                  #       NORMAL SHUTDOWN.
039684,000072: 
039685,000073:                                                                                                  #  CALLING SEQUENCE....T5, IN PARTICULAR BY ELRSKIP OF FRESH START/RESTART
039686,000074: 
039687,000075:                                                                                                  #  NORMAL EXIT MODES....RESUME, NOQRSM, POSTJUMP (TO TVCINIT4 OR MRCLEAN)
039688,000076: 
039689,000077:                                                                                                  #  ALARM OR ABORT EXIT MODES....NONE
