Source Code

These source-code files were derived from scans of the Colossus 237 program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

040748,000014:                                                                                                  #  NAME     STROKE TEST PACKAGE   (INCLUDING INITIALIZATION PACKAGE)
040749,000015:                                                                                                  #  ORIGINAL CODING BY OLSSON       LOG SECTION....STROKE TEST PACKAGE
040750,000016:                                                                                                  #  MOD BY ENGEL                    DATE....21 MARCH, 1967
040751,000017: 
040752,000018:                                                                                                  #  FUNCTIONAL DESCRIPTION....
040753,000019:                                                                                                  #       STROKE TEST PACKAGE GENERATES A WAVEFORM DESIGNED TO EXCITE BENDING
040754,000020:                                                                                                  #       STRKTSTI (STROKE TEST INITIALIZATION) IS CALLED AS A JOB BY VB68.
040755,000021:                                                                                                  #                IT INITIALIZES ALL ERASABLES REQD FOR A STROKE TEST, AND
040756,000022:                                                                                                  #                THEN TESTS FOR AN 80MS DAP. IF 80MS IT SETS STROKER = ESTROKER
040757,000023:                                                                                                  #                FOR AN IMMEDIATE STROKE TEST, OTEHRWISE IT MERELY ENABLES
040758,000024:                                                                                                  #                A STROKE TEST BY SETTING STROKER TO -0. THE STROKE TEST
040759,000025:                                                                                                  #                THEN AWAITS SWITCHOVER TO THE 80MS DAP WHEREUPON IT IS
040760,000026:                                                                                                  #                ENABLED AFTER AN ADDITIONAL 4 SECOND DELAY TO AVOID
040761,000027:                                                                                                  #                THE SWITCHOVER TRANSIENTS (SEE STRKCALL, STRKUP IN
040762,000028:                                                                                                  #                TVCEXECUTIVE)
040763,000029:                                                                                                  #       HACK (STROKE TEST) GENERATES THE WAVEFORM BY DUMPING PULSE BURSTS
040764,000030:                                                                                                  #                OF PROPER SIGN AND IN PROPER SEQUENCE DIRECTLY INTO
040765,000031:                                                                                                  #                TVCPITCH, WORKING IN CONJUNCITON WITH BOTH PITCH AND YAW
040766,000032:                                                                                                  #                TVC DAPS, WITH INTERMEDIAT WAITLIST CALLS. NOTE, HOWEVER
040767,000033:                                                                                                  #                THAT THE STROKE TEST IS PERFORMED ONLY IN THE PITCH AXIS.
040768,000034:                                                                                                  #                AN EXAMPLE WAVEFORM IS GIVEN BELOW, TO DEMONSTRATE STROKE-
040769,000035:                                                                                                  #                TEST PARAMETER SELECTION
040770,000036:                                                                                                  #       RESTARTS CAUSE TEST TO BE TERMINATED. ANOTHER V68 REQD IF TEST
040771,000037:                                                                                                  #                IS TO BE RE-RUN.
040772,000038:                                                                                                  #       PULSE BURST SIZE IS PAD-LOADED (ESTROKER) SO THAT AMPLITUDE OF
040773,000039:                                                                                                  #                WAVEFORM CAN BE CHANGED.  THERE ARE TEN PULSE BURSTS IN
040774,000040:                                                                                                  #                THE HALF-AMPLITUDE OF THE FIRST FREQUENCY SET IN THE
040775,000041:                                                                                                  #                STANDARD WAVEFORM.  AMPLITUDE IS 10(ESTROKER)(1/42.15),
040776,000042:                                                                                                  #                NOMINALLY 50/42.15  =  1.185 DEG
040777,000043:                                                                                                  #  CALLING SEQUENCE....
040778,000044:                                                                                                  #       EXTENDED VERB 68 SETS UP STRKTSTI JOB
040779,000045:                                                                                                  #       PITCH AND YAW TVCDAPS, FINDING STROKER NON-ZERO, DO A  ..TC HACK..
040780,000046:                                                                                                  #       AN INTERNALLY-GENERATED WAITLIST CALL ENTERS AT  ..HACKWLST..
040781,000047:                                                                                                  #  NORMAL EXIT MODES....
040782,000048:                                                                                                  #       TC BUNKER  (..Q.. IF ENTRY FROM DAP, ..TCTSKOVR.. IF FROM WAITLIST)LIST
040783,000049:                                                                                                  #  SUBROUTINES CALLED....
040784,000050:                                                                                                  #       WAITLIST
040785,000051:                                                                                                  #  ALARM OR ABORT EXIT MODES....
040786,000052:                                                                                                  #       NONE
040787,000053:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED....
040788,000054:                                                                                                  #       ESTROKER (PAD-LOAD)
040789,000055:                                                                                                  #       STROKER,  CADDY,  REVS,  CARD,  N
040790,000056:                                                                                                  #  OUTPUT....
040791,000057:                                                                                                  #       STRKTSTI...INITIALIZATION FOR STROKE TEST
040792,000058:                                                                                                  #       HACK, HACKWLST...PULSE BURSTS INTO TVCPITCH VIA..ADS..
040793,000059:                                                                                                  #                        RESETS STROKER = +0 WHEN TEST COMPLETED
040794,000060:                                                                                                  #  DEBRIS....
040795,000061:                                                                                                  #        N = CADDY = +0, CARD = -0, REVS = -1
040796,000062:                                                                                                  #        BUNKER
