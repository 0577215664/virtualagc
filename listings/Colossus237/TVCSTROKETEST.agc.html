040741,000018:                                                                                                 #  FUNCTIONAL DESCRIPTION....
040742,000019:                                                                                                 #       STROKE TEST PACKAGE GENERATES A WAVEFORM DESIGNED TO EXCITE BENDING
040743,000020:                                                                                                 #       STRKTSTI (STROKE TEST INITIALIZATION) IS CALLED AS A JOB BY VB68.
040744,000021:                                                                                                 #                IT INITIALIZES ALL ERASABLES REQD FOR A STROKE TEST, AND
040745,000022:                                                                                                 #                 THEN TESTS FOR AN 80MS DAP. IF 80MS IT SETS STROKER = ESTROKER
040746,000023:                                                                                                 #                 FOR AN IMMEDIATE STROKE TEST, OTEHRWISE IT MERELY ENABLES
040747,000024:                                                                                                 #                 A STROKE TEST BY SETTING STROKER TO -0. THE STROKE TEST
040748,000025:                                                                                                 #                 THEN AWAITS SWITCHOVER TO THE 80MS DAP WHEREUPON IT IS
040749,000026:                                                                                                 #                 ENABLED AFTER AN ADDITIONAL 4 SECOND DELAY TO AVOID
040750,000027:                                                                                                 #                 THE SWITCHOVER TRANSIENTS (SEE STRKCALL, STRKUP IN
040751,000028:                                                                                                 #                 TVCEXECUTIVE)
040752,000029:                                                                                                 #       HACK (STROKE TEST) GENERATES THE WAVEFORM BY DUMPING PULSE BURSTS
040753,000030:                                                                                                 #                 OF PROPER SIGN AND IN PROPER SEQUENCE DIRECTLY INTO
040754,000031:                                                                                                 #                 TVCPITCH, WORKING IN CONJUNCITON WITH BOTH PITCH AND YAW
040755,000032:                                                                                                 #                 TVC DAPS, WITH INTERMEDIAT WAITLIST CALLS. NOTE, HOWEVER
040756,000033:                                                                                                 #                 THAT THE STROKE TEST IS PERFORMED ONLY IN THE PITCH AXIS.
040757,000034:                                                                                                 #                 AN EXAMPLE WAVEFORM IS GIVEN BELOW, TO DEMONSTRATE STROKE-
040758,000035:                                                                                                 #                 TEST PARAMETER SELECTION
040759,000036:                                                                                                 #       RESTARTS CAUSE TEST TO BE TERMINATED. ANOTHER V68 REQD IF TEST
040760,000037:                                                                                                 #                 IS TO BE RE-RUN.
040761,000038:                                                                                                 #       PULSE BURST SIZE IS PAD-LOADED (ESTROKER) SO THAT AMPLITUDE OF
040762,000039:                                                                                                 #                 WAVEFORM CAN BE CHANGED.  THERE ARE TEN PULSE BURSTS IN
040763,000040:                                                                                                 #                 THE HALF-AMPLITUDE OF THE FIRST FREQUENCY SET IN THE
040764,000041:                                                                                                 #                 STANDARD WAVEFORM.  AMPLITUDE IS 10(ESTROKER)(1/42.15),
040765,000042:                                                                                                 #                 NOMINALLY 50/42.15  =  1.185 DEG
040766,000043:                                                                                                 #  CALLING SEQUENCE....
040767,000044:                                                                                                 #       EXTENDED VERB 68 SETS UP STRKTSTI JOB
040768,000045:                                                                                                 #       PITCH AND YAW TVCDAPS, FINDING STROKER NON-ZERO, DO A  ..TC HACK..
040769,000046:                                                                                                 #       AN INTERNALLY-GENERATED WAITLIST CALL ENTERS AT  ..HACKWLST..
040770,000047:                                                                                                 #  NORMAL EXIT MODES....
040771,000048:                                                                                                 #       TC BUNKER  (..Q.. IF ENTRY FROM DAP, ..TCTSKOVR.. IF FROM WAITLIST)LIST
040772,000049:                                                                                                 #  SUBROUTINES CALLED....
040773,000050:                                                                                                 #       WAITLIST
040774,000051:                                                                                                 #  ALARM OR ABORT EXIT MODES....
040775,000052:                                                                                                 #       NONE
040776,000053:                                                                                                 #  ERASABLE INITIALIZATION REQUIRED....
040777,000054:                                                                                                 #       ESTROKER (PAD-LOAD)
040778,000055:                                                                                                 #       STROKER,  CADDY,  REVS,  CARD,  N
040779,000056:                                                                                                 #  OUTPUT....
040780,000057:                                                                                                 #       STRKTSTI...INITIALIZATION FOR STROKE TEST
040781,000058:                                                                                                 #       HACK, HACKWLST...PULSE BURSTS INTO TVCPITCH VIA..ADS..
040782,000059:                                                                                                 #                        RESETS STROKER = +0 WHEN TEST COMPLETED
040783,000060:                                                                                                 #  DEBRIS....
040784,000061:                                                                                                 #         N = CADDY = +0, CARD = -0, REVS = -1
040785,000062:                                                                                                 #         BUNKER
