053910,000017:                                                                                                 #  FROM A USERS POINT OF VIEW, ORBITAL INTEGRATION IS ESSENTIALLY THE SAME AS THE 278 INTEGRATION
053911,000018:                                                                                                 #  PROGRAM. THE SAME ENTRANCES TO THE PROGRAM WILL BE MAINTAINED, THE SAME STALLING ROUTINE WILL BE USED AND
053912,000019:                                                                                                 #  OUTPUT WILL STILL BE VIA THE PUSHLIST. THE PRIMARY DIFFERENCES TO A USER INVOLVE THE ADDED CAPABILITY OF
053913,000020:                                                                                                 #  TERMINATING INTEGRATION AT A SPECIFIC FINAL RADIUS AND THE DIFFERENCE IN STATE VECTOR SCALING INSIDE AND OUT-
053914,000021:                                                                                                 #  SIDE THE LUNAR SPHERE OF INFLUENCE.
053915,000022: 
053916,000023:                                                                                                 #  IN ORDER TO MAKE THE CSM(LEM)PREC AND CSM(LEM)CONIC ENTRANCES SIMILAR TO FLIGHT 278, THE INTEGRATION PROGRAM
053917,000024:                                                                                                 #  WILL ITSELF SET THE FINAL RADIUS (RFINAL) TO 0 SO THAT REACHING THE DESIRED TIME ONLY WILL TERMINATE
053918,000025:                                                                                                 #  INTEGRATION. THE DP REGISTER RFINAL MUST BE SET BY USERS OF INTEGRVS AND INTEGRV, AND MUST BE DONE AFTER THE
053919,000026:                                                                                                 #  CALL TC INTSTALL.
053920,000027: 
053921,000028:                                                                                                 #  WHEN THE LM IS ON THE LUNAR SURFACE (INDICATED BY LUNAR SURFACE FLAG SET) CALLS TO LEMCONIC, LEMPREC, AND
053922,000029:                                                                                                 #  INTEGRV WITH VINFLAG = 0 WILL RESULT IN THE USE OF THE PLANETARY INERTIAL ORIENTATION SUBROUTINES TO PROVIDE
053923,000030:                                                                                                 #  BOTH THE LMS POSITION AND VELOCITY IN THE REFERENCE COORDINATE  SYSTEM.
053924,000031:                                                                                                 #  THE PROGRAM WILL PROVIDE OUTPUT AS IF INTEGRATION WAS USED. THAT IS, THE PUSHLIST WILL BE SET AS NOTED BELOW AND
053925,000032:                                                                                                 #  THE PERMANENT STATE VECTOR UPDATED WHEN SPECIFIED BY AN INTEGRV CALL.
053926,000033: 
053927,000034:                                                                                                 #  USERS OF INTEGRVS DESIRING INTEGRATION (INTYPFLG = 0) SHOULD NOTE THAT THE OBLATENESS PERTURBATION COMPUTATION
053928,000035:                                                                                                 #  IN LUNAR ORBIT IS TIME DEPENDENT. THEREFORE, THE USER SHOULD SUPPLY AN INITIAL STATE VECTOR VALID AT SOME REAL
053929,000036:                                                                                                 #  TIME AND THE DESIRED TIME (TDEC1) ALSO AT SOME REAL TIME. FOR CONIC ,,INTEGRATION,, THE USER MAY STILL USE ZERO
053930,000037:                                                                                                 #  AS THE INITIAL TIME AND DELTA TIME AS THE DESIRED TIME.

053932,000039:                                                                                                 #  2.0 CENTRAL DESCRIPTION
053933,000040:                                                                                                 #  -----------------------
053934,000041: 
053935,000042:                                                                                                 #  THE INTEGRATION PROGRAM OPERATES AS A CLOSED INTERPRETIVE SUBROUTINE AND PERFORMS THESE FUNCTIONS---
053936,000043:                                                                                                 #     1) INTEGRATES (PRECISION OR CONIC) EITHER CSM OR LM STATE VECTOR
053937,000044:                                                                                                 #     2) INTEGRATES THE W-MATRIX
053938,000045:                                                                                                 #     3) PERMANENT OR TEMPORARY UPDATE OF THE STATE VECTOR
053939,000046: 
053940,000047:                                                                                                 #  THERE ARE SIX ENTRANCES TO THE INTEGRATION PROGRAM. FOUR OF THESE (CSMPREC, LEMPREC, CSMCONIC, LEMCONIC) SET
053941,000048:                                                                                                 #  ALL THE FLAGS REQUIRED IN THE INTEGRATION PROGRAM ITSELF TO CAUSE THE PRECISION OR CONIC INTEGRATION (KEPLER) OF
053942,000049:                                                                                                 #  THE LM OR CSM STATE VECTOR, AS THE NAMES SUGGEST. ONE ENTRANCE (INTEGRVS) PERMITS THE CALLING PROGRAM TO
053943,000050:                                                                                                 #  PROVIDE A STATE VECTOR TO BE INTEGRATED. THE CALLING PROGRAM MUST SET THE FLAGS INDICATING (1) PRECISION OR
053944,000051:                                                                                                 #  CONIC INTEGRATION, (2) IN OR OUT OF LUNAR SPHERE, (3) MIDCOURSE OR NOT, AND THE INTEGRATION PROGRAM COMPLETES
053945,000052:                                                                                                 #  THE FLAG SETTING TO BYPASS W-MATRIX INTEGRATION. THE LAST ENTRANCE (INTEGRV, USED IN GENERAL BY THE
053946,000053:                                                                                                 #  NAVIGATION PROGRAMS) PERMITS THE CALLER TO SET FIVE FLAGS (NOT MOONFLAG OR MIDFLAG) BUT NOT TO INPUT A STATE
053947,000054:                                                                                                 #  VECTOR. ANY PROGRAM WHICH CALLS INTEGRVS OR INTEGRV MUST CALL INTSTALL BEFORE IT SETS THE INTEGRATION FLAGS
053948,000055:                                                                                                 #  AND/OR STATE VECTOR.
053949,000056: 
053950,000057:                                                                                                 #  THREE SETS OF 42 REGISTERS AND 2 FLAGS ARE USED FOR THE STATE VECTORS. TWO SETS, WHICH MAY NOT BE OVERLAYED, ARE
053951,000058:                                                                                                 #  USED FOR THE PERMANENT STATE VECTORS FOR THE CSM AND LM. THE THIRD SET, WHICH MAY BE OVERLAYED WHEN INTEGRATION
053952,000059:                                                                                                 #  IS NOT BEING DONE, IS USED IN THE COMPUTATIONS.
053953,000060: 
053954,000061:                                                                                                 #  THE PERMANENT STATE VECTORS WILL BE PERIODICALLY UPDATED SO THAT THE VECTORS WILL NOT BE OLDER THAN 4 TIMESTEPS.
053955,000062:                                                                                                 #  THE PERMANENT STATE VECTORS WILL ALSO BE UPDATED WHENEVER THE W-MATRIX IS INTEGRATED OR WHEN A CALLER OF INTEGRV
053956,000063:                                                                                                 #  SETS STATEFLG (THE NAVIGATION PROGRAMS P20, P22.)
053957,000064:                                                                                                 # # Page 1276
053958,000065:                                                                                                 #  APPENDIX B OF THE USERS GUIDE LISTS THE STATE VECTOR QUANTITIES.
053959,000066: 
053960,000067:                                                                                                 #  2.1 RESTARTS
053961,000068: 
053962,000069:                                                                                                 #  PHASE CHANGES WILL BE MADE IN THE INTEGRATION PROGRAM ONLY FOR THE INTEGRV ENTRANCE (I.E., WHEN THE W-MATRIX IS
053963,000070:                                                                                                 #  INTEGRATED OR PERMANENT STATE VECTOR IS UPDATED.) THE GROUP NUMBER USED WILL BE THAT FOR THE P20-25 PROGRAMS
053964,000071:                                                                                                 #  (I.E., GROUP2) WINCE THE INTEGRV ENTRANCE WILL ONLY BE USED BY THESE PROGRAMS. IF A RESTART OCCURS DURING AN
053965,000072:                                                                                                 #  INTEGRATION OF THE STATE VECTOR ONLY, THE RECOVERY WILL BE TO THE LAST PHASE IN THE CALLING PROGRAM. CALLING
053966,000073:                                                                                                 #  PROGRAMS WHICH USE THE INTEGRV OR INTEGRVS ENTRANCE OF INTEGRATION SHOULD ENSURE THAT IF PHASE CHANGING IS DONE
053967,000074:                                                                                                 #  THAT IT IS PRIOR TO SETTING THE INTEGRATION INPUTS IN THE PUSHLIST.
053968,000075:                                                                                                 #  THIS IS BECAUSE THE PUSHLIST IS LOST DURING A RESTART.
053969,000076: 
053970,000077:                                                                                                 #  2.2 SCALING
053971,000078: 
053972,000079:                                                                                                 #  THE INTEGRATION ROUTINE WILL MAINTAIN THE PERMANENT MEMORY STATE VECTORS IN THE SCALING AND UNITS DEFINED IN
053973,000080:                                                                                                 #  APPENDIX B OF THE USERS GUIDE. THE SCALING OF THE OUTPUT POSITION VECTORDEPENDS ON THE ORIGIN OF THE COORDINATE
053974,000081:                                                                                                 #   SYSTEM AT THE DESIRED INTEGRATION TIME. THE COORDINATE SYSTEM TRANSFORMATION WILL BE DONE AUTOMATICALLY ON
053975,000082:                                                                                                 #  MULTIPLE TIMESTEP ENCKE INTEGRATION ONLY. THUS IT IS POSSIBLE TO HAVE OUTPUT FROM SUCCESSIVE INTEGRATIONS IN
053976,000083:                                                                                                 #  DIFFERENT SCALING.
053977,000084:                                                                                                 #  HOWEVER, RATT, VATT WILL ALWAYS BE SCALED THE SAME.
