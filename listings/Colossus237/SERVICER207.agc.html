Source Code

These source-code files were derived from scans of the Colossus 237 program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

033599,000020:                                                                                                  #  FUNCTIONAL DESCRIPTION
033600,000021: 
033601,000022:                                                                                                  #  THE ROUTINES DESCRIBED BELOW ARE USED TO CALCULATE VALUES OF RN, VN, AND GDT/2 DURING ACCELERATED FLIGHT.
033602,000023:                                                                                                  #  THE SEVERAL ROUTINES COMPRISE A PACKAGE AND ARE NOT MEANT TO BE USED AS SEPARATE SUBROUTINES.
033603,000024: 
033604,000025:                                                                                                  #  GENERAL REFERENCES TO  SERVICER  OR  AVERAGE G  ARE UNDERSTOOD TO REFER TO THE ENTIRE SET OF ROUTINES INCLUDING
033605,000026:                                                                                                  #  READACCS, SERVICER, AVERAGE G, INTEREAD, SMOOTHER, AND ANY ADDITIONAL ROUTINES ATTACHED AT AVGEXIT (SEE BELOW).
033606,000027: 
033607,000028:                                                                                                  #  PROGRAMS INITIATING SERVICER ARE REQUIRED TO MAKE A WAITLIST CALL FOR PREREAD (OR, IF LIFTOFF, FOR BIBIBIAS)
033608,000029:                                                                                                  #  AT 2 SECONDS BEFORE THE FIRST AVERAGE G UPDATE IN ORDER TO INITIALIZE THE SEQUENCE, WHICH WILL RECUR EVERY
033609,000030:                                                                                                  #  2 SECONDS FROM THAT TIME ON AS LONG AS AVEGFLAG REMAINS SET.
033610,000031: 
033611,000032:                                                                                                  #  THE USE OF ERASABLE AVGEXIT ALLOWS VARIOUS ROUTINES TO BE PERFORMED AS PART OF THE NORMAL CYCLE (SEE
033612,000033:                                                                                                  #  EXPLANATION OF AVGEXIT BELOW).
033613,000034: 
033614,000035:                                                                                                  #  DESCRIPTIONS OF INDIVIDUAL ROUTINES FOLLOW.
033615,000036: 
033616,000037: 
033617,000038:                                                                                                  #     PREREAD
033618,000039: 
033619,000040:                                                                                                  #           PREVIOUSLY EXTRAPOLATED VALUES COPIED FROM RN1, VN1, AND PIPTIME1 INTO RN, VN, AND PIPTIME.
033620,000041:                                                                                                  #           LASTBIAS JOB SCHEDULED.
033621,000042:                                                                                                  #           PIPS READ AND CLEARED VIA PIPASR SUBROUTINE.
033622,000043:                                                                                                  #           AVERAGE G FLAG SET ON.
033623,000044:                                                                                                  #           DRIFT FLAG SET OFF.
033624,000045:                                                                                                  #           V37 FLAG SET ON.
033625,000046:                                                                                                  #           INITIALIZATION OF   1) THRUST MONITOR (DVMON) - DVCNTR SET TO ONE.
033626,000047:                                                                                                  #                               2) TOTAL ACCUMULATED DELV VALUE (DVTOTAL) - SET TO ZERO.
033627,000048:                                                                                                  #                               3) AXIS VECTOR (AXIS) - SET TO (.5,0,0).
033628,000049:                                                                                                  #           NORMLIZE JOB SCHEDULED.
033629,000050:                                                                                                  #           READACCS TASH CALLED IN 2 SECONDS.
033630,000051: 
033631,000052: 
033632,000053:                                                                                                  #     NORMLIZE
033633,000054: 
033634,000055:                                                                                                  #           GDT/2 INITIALIZED VIA CALCGRAV SUBROUTINE.
033635,000056: 
033636,000057: 
033637,000058:                                                                                                  #     READACCS
033638,000059: 
033639,000060:                                                                                                  #           IF ONMON FLAG SET QUIKREAD ROUTINE IS PERFORMED BEFORE PIPASR ZEROS THE PIPA REGISTERS, AND THE 1/2 SEC
033640,000061:                                                                                                  #           ONMONITOR LOOP IS INITIATED TO PROVIDE DOWNLINK INFORMATION DURING ENTRY.
033641,000062:                                                                                                  #           PIPS READ AND CLEARED BY PIPASR SUBROUTINE.
033642,000063:                                                                                                  #           IF CM/DSTBY IS ON, ENTRY VARIABLES INITIALIZED AND SETJTAG TASK CALLED.
