Source Code

These source-code files were derived from scans of the Colossus 237 (Apollo 8 Command Module) program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9 CM) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

033728,000018:                                                                                                  #  PROGRAM NAME - PREREAD, READACCS, SERVICER, AVERAGE G.
033729,000019: 
033730,000020: 
033731,000021:                                                                                                  #  MOD NO.  00   BY M.HAMILTON   DEC.12, 1966
033732,000022: 
033733,000023: 
033734,000024:                                                                                                  #  FUNCTIONAL DESCRIPTION
033735,000025: 
033736,000026:                                                                                                  #  THE ROUTINES DESCRIBED BELOW ARE USED TO CALCULATE VALUES OF RN, VN, AND GDT/2 DURING ACCELERATED FLIGHT.
033737,000027:                                                                                                  #  THE SEVERAL ROUTINES COMPRISE A PACKAGE AND ARE NOT MEANT TO BE USED AS SEPARATE SUBROUTINES.
033738,000028: 
033739,000029:                                                                                                  #  GENERAL REFERENCES TO  SERVICER  OR  AVERAGE G  ARE UNDERSTOOD TO REFER TO THE ENTIRE SET OF ROUTINES INCLUDING
033740,000030:                                                                                                  #  READACCS, SERVICER, AVERAGE G, INTEREAD, SMOOTHER, AND ANY ADDITIONAL ROUTINES ATTACHED AT AVGEXIT (SEE BELOW).
033741,000031: 
033742,000032:                                                                                                  #  PROGRAMS INITIATING SERVICER ARE REQUIRED TO MAKE A WAITLIST CALL FOR PREREAD (OR, IF LIFTOFF, FOR BIBIBIAS)
033743,000033:                                                                                                  #  AT 2 SECONDS BEFORE THE FIRST AVERAGE G UPDATE IN ORDER TO INITIALIZE THE SEQUENCE, WHICH WILL RECUR EVERY
033744,000034:                                                                                                  #  2 SECONDS FROM THAT TIME ON AS LONG AS AVEGFLAG REMAINS SET.
033745,000035: 
033746,000036:                                                                                                  #  THE USE OF ERASABLE AVGEXIT ALLOWS VARIOUS ROUTINES TO BE PERFORMED AS PART OF THE NORMAL CYCLE (SEE
033747,000037:                                                                                                  #  EXPLANATION OF AVGEXIT BELOW).
033748,000038: 
033749,000039:                                                                                                  #  DESCRIPTIONS OF INDIVIDUAL ROUTINES FOLLOW.
033750,000040: 
033751,000041: 
033752,000042:                                                                                                  #     PREREAD
033753,000043: 
033754,000044:                                                                                                  #           PREVIOUSLY EXTRAPOLATED VALUES COPIED FROM RN1, VN1, AND PIPTIME1 INTO RN, VN, AND PIPTIME.
033755,000045:                                                                                                  #           LASTBIAS JOB SCHEDULED.
033756,000046:                                                                                                  #           PIPS READ AND CLEARED VIA PIPASR SUBROUTINE.
033757,000047:                                                                                                  #           AVERAGE G FLAG SET ON.
033758,000048:                                                                                                  #           DRIFT FLAG SET OFF.
033759,000049:                                                                                                  #           V37 FLAG SET ON.
033760,000050:                                                                                                  #           INITIALIZATION OF   1) THRUST MONITOR (DVMON) - DVCNTR SET TO ONE.
033761,000051:                                                                                                  #                               2) TOTAL ACCUMULATED DELV VALUE (DVTOTAL) - SET TO ZERO.
033762,000052:                                                                                                  #                               3) AXIS VECTOR (AXIS) - SET TO (.5,0,0).
033763,000053:                                                                                                  #           NORMLIZE JOB SCHEDULED.
033764,000054:                                                                                                  #           READACCS TASK CALLED IN 2 SECONDS.
033765,000055: 
033766,000056: 
033767,000057:                                                                                                  #     NORMLIZE
033768,000058: 
033769,000059:                                                                                                  #           GDT/2 INITIALIZED VIA CALCGRAV SUBROUTINE.
033770,000060: 
033771,000061: 
033772,000062:                                                                                                  #     READACCS
033773,000063: 
033774,000064:                                                                                                  #           IF ONMON FLAG SET QUIKREAD ROUTINE IS PERFORMED BEFORE PIPASR ZEROS THE PIPA REGISTERS, AND THE 1/2 SEC
033775,000065:                                                                                                  #           ONMONITOR LOOP IS INITIATED TO PROVIDE DOWNLINK INFORMATION DURING ENTRY.
033776,000066:                                                                                                  #           PIPS READ AND CLEARED BY PIPASR SUBROUTINE.
033777,000067:                                                                                                  #           IF CM/DSTBY IS ON, ENTRY VARIABLES INITIALIZED AND SETJTAG TASK CALLED.

033798,000088:                                                                                                  #                  IF BAD PIP   1) ALARM SENT.
033799,000089:                                                                                                  #                               2) COMPENSATION, DVTOTAL ACCUMULATION, AND DVMON BYPASSED.  CONTROL

033804,000094:                                                                                                  #           CONTROL TRANSFERRED TO AVERAGE G.
033805,000095: 
033806,000096: 
033807,000097:                                                                                                  #           DVMON
033808,000098: 
033809,000099:                                                                                                  #              THRESHOLD VALUE (PLACED IN DVTHRUSH BY USER) CHECKED AGAINST ABSOLUTE VALUE OF DELV TO CHECK
033810,000100:                                                                                                  #              THRUST LEVEL.
033811,000101:                                                                                                  #                  IF THRUST    1) ULLAGE OFF ROUTINE PERFORMED.
033812,000102:                                                                                                  #                               2) STEERING FLAG TURNED ON AT FIRST DETECTION OF THRUST.
033813,000103:                                                                                                  #                               3) CONTROL TRANSFERRED TO AVERAGE G.
033814,000104:                                                                                                  #                  IF NO THRUST 1) ON FIRST PASS THROUGH MONITOR, CONTROL TRANSFERRED TO AVERAGE G.
033815,000105:                                                                                                  #                               2) ON SUBSEQUENT PASSES, CONTROL TRANSFERRED TO ENGINE FAIL ROUTINE IF THRUST
033816,000106:                                                                                                  #                                  HAS FAILED FOR 3 CONSECUTIVE PASSES.
033817,000107: 
033818,000108: 
033819,000109:                                                                                                  #                ENGINE FAIL

033822,000112:                                                                                                  #                       SEQUENCE MAY BE REPEATED.
033823,000113:                                                                                                  #                   ENGINOF3 PERFORMED.
033824,000114:                                                                                                  #                   DAP SET UP FOR RCS.
