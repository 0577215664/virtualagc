Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. 033589,000020:                                                                                                 #  FUNCTIONAL DESCRIPTION
033590,000021: 
033591,000022:                                                                                                 #  THE ROUTINES DESCRIBED BELOW ARE USED TO CALCULATE VALUES OF RN, VN, AND GDT/2 DURING ACCELERATED FLIGHT.
033592,000023:                                                                                                 #  THE SEVERAL ROUTINES COMPRISE A PACKAGE AND ARE NOT MEANT TO BE USED AS SEPARATE SUBROUTINES.
033593,000024: 
033594,000025:                                                                                                 #  GENERAL REFERENCES TO  SERVICER  OR  AVERAGE G  ARE UNDERSTOOD TO REFER TO THE ENTIRE SET OF ROUTINES INCLUDING
033595,000026:                                                                                                 #  READACCS, SERVICER, AVERAGE G, INTEREAD, SMOOTHER, AND ANY ADDITIONAL ROUTINES ATTACHED AT AVGEXIT (SEE BELOW).
033596,000027: 
033597,000028:                                                                                                 #  PROGRAMS INITIATING SERVICER ARE REQUIRED TO MAKE A WAITLIST CALL FOR PREREAD (OR, IF LIFTOFF, FOR BIBIBIAS)
033598,000029:                                                                                                 #  AT 2 SECONDS BEFORE THE FIRST AVERAGE G UPDATE IN ORDER TO INITIALIZE THE SEQUENCE, WHICH WILL RECUR EVERY
033599,000030:                                                                                                 #  2 SECONDS FROM THAT TIME ON AS LONG AS AVEGFLAG REMAINS SET.
033600,000031: 
033601,000032:                                                                                                 #  THE USE OF ERASABLE AVGEXIT ALLOWS VARIOUS ROUTINES TO BE PERFORMED AS PART OF THE NORMAL CYCLE (SEE
033602,000033:                                                                                                 #  EXPLANATION OF AVGEXIT BELOW).
033603,000034:

033612,000043:                                                                                                 #           AVERAGE G FLAG SET ON.
033613,000044:                                                                                                 #           DRIFT FLAG SET OFF.
033614,000045:                                                                                                 #           V37 FLAG SET ON.
033615,000046:                                                                                                 #           INITIALIZATION OF   1) THRUST MONITOR (DVMON) - DVCNTR SET TO ONE.
033616,000047:                                                                                                 #                               2) TOTAL ACCUMULATED DELV VALUE (DVTOTAL) - SET TO ZERO.
033617,000048:                                                                                                 #                               3) AXIS VECTOR (AXIS) - SET TO (.5,0,0).
033618,000049:                                                                                                 #           NORMLIZE JOB SCHEDULED.
033619,000050:                                                                                                 #           READACCS TASH CALLED IN 2 SECONDS.

033622,000053:                                                                                                 #     NORMLIZE

033625,000056: 
033626,000057:

033628,000059: 
033629,000060:                                                                                                 #           IF ONMON FLAG SET QUIKREAD ROUTINE IS PERFORMED BEFORE PIPASR ZEROS THE PIPA REGISTERS, AND THE 1/2 SEC
033630,000061:                                                                                                 #           ONMONITOR LOOP IS INITIATED TO PROVIDE DOWNLINK INFORMATION DURING ENTRY.
033631,000062:                                                                                                 #           PIPS READ AND CLEARED BY PIPASR SUBROUTINE.
033632,000063:                                                                                                 #           IF CM/DSTBY IS ON, ENTRY VARIABLES INITIALIZED AND SETJTAG TASK CALLED.
033633,000064:                                                                                                 # # Page 771
033634,000065:                                                                                                 #           IF AVERAGEG FLAG ON    READACCS CALLED TO RECYCLE IN 2 SECONDS.
033635,000066:                                                                                                 #           IF AVERAGEG FLAG OFF   AVERAGE G EXIT (AVGEXIT) SET TO 2CADR AVGEND FOR FINAL PASS.
033636,000067:                                                                                                 #           SERVICER JOB SCHEDULED.
033637,000068: 
033638,000069: 
033639,000070:                                                                                                 #           TEST CONNECTOR OUTBIT TURNED ON.
033640,000071: 
033641,000072: 
033642,000073:                                                                                                 #  ONMNITOR
033643,000074: 
033644,000075:                                                                                                 #           A SEQUENCE OF THREE PASSES THROUGH QUICKREAD FOLLOWING A CALL TO READACCS WITH ONMONFLG SET AT 1/2 
033645,000076:                                                                                                 #           SEC INTERVALS. INTERVALS ARE COUNTED OUT BY PIPCTR, INITIALIZED AT 3 BY READACCS
033646,000077: 
033647,000078:                                                                                                 #     QUIKREAD
033648,000079: 
033649,000080:                                                                                                 #         READS CURRENT PIPS INTO X,Y,ZPIPBUF. READS OLD X,Y,ZPIPBUF INTO X,Y,ZOLDBUF. VALUES ARE SENT TO
033650,000081:                                                                                                 #           DOWNLIST DURING ENTRY.
033651,000082:                                                                                                 #     SERVICER
033652,000083:                                                                                                 #           DELV VALUES CHECKED TO DETECT RUNAWAY PIP -
033653,000084:                                                                                                 #                  IF BAD PIP   1) ALARM SENT.
033654,000085:                                                                                                 #                               2) COMPENSATION, DVTOTAL ACCUMULATION, AND DVMON BYPASSED.  CONTROL
033655,000086:                                                                                                 #                                  TRANSFERRED TO AVERAGE G.
033656,000087:                                                                                                 #           PIPS COMPENSATED VIA 1/PIPA SUBROUTINE.
033657,000088:                                                                                                 #           DVTOTAL INCREMENTED BY ABSOLUTE VALUE OF DELV.
033658,000089:                                                                                                 #           THRUST MONITOR (DVMON) PERFORMED UNLESS IDLE FLAG IS ON.
033659,000090:                                                                                                 #           CONTROL TRANSFERRED TO AVERAGE Q.
033660,000091: 
033661,000092: 
033662,000093:                                                                                                 #           DVMON
033663,000094: 
033664,000095:                                                                                                 #              THRESHOLD VALUE (PLACED IN DVTHRUSH BY USER) CHECKED AGAINST ABSOLUTE VALUE OF DELV TO CHECK
033665,000096:                                                                                                 #              THRUST LEVEL.
033666,000097:                                                                                                 #                  IF THRUST    1) ULLAGE OFF ROUTINE PERFORMED.
033667,000098:                                                                                                 #                               2) STEERING FLAG TURNED ON AT FIRST DETECTION OF THRUST.
033668,000099:                                                                                                 #                               3) CONTROL TRANSFERRED TO AVERAGE G.
033669,000100:                                                                                                 #                  IF NO THRUST 1) ON FIRST PASS THROUGH MONITOR, CONTROL TRANSFERRED TO AVERAGE G.
033670,000101:                                                                                                 #                               2) ON SUBSEQUENT PASSES, CONTROL TRANSFERRED TO ENGINE FAIL ROUTINE IF THRUST
033671,000102:                                                                                                 #                                  HAS FAILED FOR 3 CONSECUTIVE PASSES.
033672,000103: 
033673,000104:

033679,000110:                                                                                                 #                   DAP SET UP FOR RCS.
