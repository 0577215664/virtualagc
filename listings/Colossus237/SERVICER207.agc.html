Source Code

These source-code files were derived from scans of the Colossus 237 program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

033598,000020:                                                                                                  #  FUNCTIONAL DESCRIPTION
033599,000021: 
033600,000022:                                                                                                  #  THE ROUTINES DESCRIBED BELOW ARE USED TO CALCULATE VALUES OF RN, VN, AND GDT/2 DURING ACCELERATED FLIGHT.
033601,000023:                                                                                                  #  THE SEVERAL ROUTINES COMPRISE A PACKAGE AND ARE NOT MEANT TO BE USED AS SEPARATE SUBROUTINES.
033602,000024: 
033603,000025:                                                                                                  #  GENERAL REFERENCES TO  SERVICER  OR  AVERAGE G  ARE UNDERSTOOD TO REFER TO THE ENTIRE SET OF ROUTINES INCLUDING
033604,000026:                                                                                                  #  READACCS, SERVICER, AVERAGE G, INTEREAD, SMOOTHER, AND ANY ADDITIONAL ROUTINES ATTACHED AT AVGEXIT (SEE BELOW).
033605,000027: 
033606,000028:                                                                                                  #  PROGRAMS INITIATING SERVICER ARE REQUIRED TO MAKE A WAITLIST CALL FOR PREREAD (OR, IF LIFTOFF, FOR BIBIBIAS)
033607,000029:                                                                                                  #  AT 2 SECONDS BEFORE THE FIRST AVERAGE G UPDATE IN ORDER TO INITIALIZE THE SEQUENCE, WHICH WILL RECUR EVERY
033608,000030:                                                                                                  #  2 SECONDS FROM THAT TIME ON AS LONG AS AVEGFLAG REMAINS SET.
033609,000031: 
033610,000032:                                                                                                  #  THE USE OF ERASABLE AVGEXIT ALLOWS VARIOUS ROUTINES TO BE PERFORMED AS PART OF THE NORMAL CYCLE (SEE
033611,000033:                                                                                                  #  EXPLANATION OF AVGEXIT BELOW).
033612,000034:

033625,000047:                                                                                                  #                               2) TOTAL ACCUMULATED DELV VALUE (DVTOTAL) - SET TO ZERO.
033626,000048:                                                                                                  #                               3) AXIS VECTOR (AXIS) - SET TO (.5,0,0).
033627,000049:                                                                                                  #           NORMLIZE JOB SCHEDULED.
