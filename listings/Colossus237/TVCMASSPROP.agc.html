Source Code

These source-code files were derived from scans of the Colossus 237 program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

039400,000014:                                                                                                  #  PROGRAM NAME....MASSPROP
039401,000015:                                                                                                  #  LOG SECTION....TVCMASSPROP      PROGRAMMER...MELANSON (ENGEL , SCHLUNDT)

039404,000018:                                                                                                  #  FUNCTIONAL DESCRIPTION:
039405,000019: 
039406,000020:                                                                                                  #       MASSPROP OPERATES IN TWO MODES:(1)IF LEM MASS OR CONFIGURATION ARE UPDATED (MASSPROP DOES NOT TEST
039407,000021:                                                                                                  #  FOR THIS) THE ENTIRE PROGRAM MUST BE RUN THROUGH, BREAKPOINT VALUES AND DERIVATIVES OF THE OUTPUTS WITH
039408,000022:                                                                                                  #  RESPECT TO CSM MASS BEING CALCULATED PRIOR TO CALCULATION OF THE OUTPUTS. (2)OTHERWISE, THE OUTPUTS CAN BE
039409,000023:                                                                                                  #  CALCULATED USING PREVIOUSLY COMPUTED BREAKPOINT VALUES AND DERIVATIVES.
039410,000024: 
039411,000025:                                                                                                  #  CALLING SEQUENCES
039412,000026: 
039413,000027:                                                                                                  #     IF LEM MASS OR CONFIGURATION HAS BEEN UPDATED, TRANSFER TO MASSPROP, OTHERWISE TRANSFER TO FIXCW.
039414,000028:                                                                                                  #           L      TC     BANKCALL OR IBNKCALL
039415,000029:                                                                                                  #           L+1    CADR   MASSPROP
039416,000030:                                                                                                  #                     OR
039417,000031:                                                                                                  #           L+1    CADR   FIXCW
039418,000032: 
039419,000033:                                                                                                  #           L+2    RETURNS VIA Q
039420,000034: 
039421,000035:                                                                                                  #  CALLED   IN PARTICULAR BY DONOUN47 (JOB) AND TVCEXECUTIVE (TASK)
039422,000036: 
039423,000037:                                                                                                  #  JOBS OR TASKS INITIATED - NONE
039424,000038: 
039425,000039:                                                                                                  #  SUBROUTINES CALLED - NONE
039426,000040: 
039427,000041:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED
039428,000042: 
039429,000043:                                                                                                  #       LEMMASS MUST CONTAIN LEM MASS SCALED AT B+16 KILOGRAMS
039430,000044:                                                                                                  #       CSMMASS MUST CONTAIN CSM MASS SCALED AT B+16 KILOGRAMS
039431,000045: 
039432,000046:                                                                                                  #       DAPDATR1 MUST BE SET TO INDICATE VEHICLE CONFIGURATION.
039433,000047:                                                                                                  #                BITS (15,14,13)  =  ( 0 , 0 , 1 )     LEM OFF
039434,000048:                                                                                                  #                                    ( 0 , 1 , 0 )     LEM ON (ASCNT,DSCNT)
039435,000049:                                                                                                  #                                    ( 1 , 1 , 0 )     LEM ON (ASCNT ONLY)
039436,000050: 
039437,000051: 
039438,000052:                                                                                                  #  ALARMS - NONE
039439,000053: 
039440,000054:                                                                                                  #  EXIT -   TC     Q
039441,000055: 
039442,000056:                                                                                                  #  OUTPUTS:
039443,000057: 
039444,000058:                                                                                                  #       (1)IXX, SINGLE PRECISION SCALED AT B+20 IN KG-M SQ.
039445,000059:                                                                                                  #       (2)IAVG, SINGLE PRECISION SCALED AT B+20 IN KG-M SQ.
039446,000060:                                                                                                  #       (3)IAVG/TLX, SINGLE PRECISION, SCALED AT B+2 SEC-SQD
039447,000061:                                                                                                  #       THEY ARE STORED IN CONSECUTIVE REGISTERS IXX0, IXX1, IXX2
039448,000062: 
039449,000063:                                                                                                  #       CONVERSION FACTOR :  (SLUG-FTSQ) = 0.737562 (KG-MSQ)

039451,000065:                                                                                                  #  OUTPUTS ARE CALCULATED AS FOLLOWS:
039452,000066: 
039453,000067:                                                                                                  #   (1) IF LEM DOCKED, LEMMASS IS FIRST ELIMINATED AS A PARAMETER
039454,000068: 
039455,000069:                                                                                                  #       VARST0 = INTVALUE0 + LEMMASS(SLOPEVAL0)       IXX       BREAKPOINT VALUE
039456,000070:                                                                                                  #       VARST1 = INTVALUE1 + LEMMASS(SLOPEVAL1)       IAVG      BREAKPOINT VALUE
039457,000071:                                                                                                  #       VARST2 = INTVALUE2 + LEMMASS(SLOPEVAL2)       IAVG/TLX  BREAKPOINT VALUE
039458,000072: 
039459,000073:                                                                                                  #       VARST3 = INTVALUE3 + LEMMASS(SLOPEVAL3)       IAVG/TLX  SLOPE  FOR CSMMASS > 33956 LBS  ( SPS > 10000 LBS)
039460,000074:                                                                                                  #       VARST4 = INTVALUE4 + LEMMASS(SLOPEVAL4)       IAVG      SLOPE  FOR CSMMASS > 33956 LBS  ( SPS > 10000 LBS)
039461,000075: 
039462,000076:                                                                                                  #       VARST5 = INTVALUE5 + LEMMASS(SLOPEVAL5)       IXX       SLOPE  FOR ALL VALUES OF CSMMASS
039463,000077: 
039464,000078:                                                                                                  #       VARST6 = INTVALUE6 + LEMMASS(SLOPEVAL6)       IAVG      SLOPE  FOR CSMMASS < 33956 LBS ( SPS < 10000 LBS)
039465,000079:                                                                                                  #       VARST7 = INTVALUE7 + LEMMASS(SLOPEVAL7)       IAVG/TLX  SLOPE  FOR CSMMASS < 33956 LBS ( SPS < 10000 LBS)
039466,000080: 
039467,000081:                                                                                                  #       VARST8 = INTVALUE8 + LEMMASS(SLOPEVAL8)       IAVG      DECREMENT  TO BRKPT VALUE WHEN LEM DSCNT STAGE OFF
039468,000082:                                                                                                  #       VARST9 = INTVALUE9 + LEMMASS(SLOPEVAL9)       IAVG/TLX  DECREMENT  TO BRKPT VALUE WHEN LEM DSCNT STAGE OFF
039469,000083: 
039470,000084: 
039471,000085:                                                                                                  #   (2) IF LEM NOT DOCKED
039472,000086: 
039473,000087:                                                                                                  #       VARST0 = NOLEMVAL0    WHERE THE MEANING AND SCALING OF VARST0
039474,000088:                                                                                                  #          .                  TO VARST9 ARE THE SAME AS GIVEN ABOVE
039475,000089:                                                                                                  #          .
039476,000090:                                                                                                  #          .                  NOTE... FOR THIS CASE, VARST8,9 HAVE NO
039477,000091:                                                                                                  #       VARST9 = NOLEMVAL9    MEANING (THEY ARE COMPUTED BUT NOT USED)
039478,000092: 
039479,000093: 
039480,000094:                                                                                                  #   (3) THE FINAL OUTPUT CALCULATIONS ARE THEN DONE
039481,000095: 
039482,000096:                                                                                                  #       IXX0 =  VARST0 + (CSMMASS + NEGBPW)VARST5            IXX
039483,000097: 
039484,000098:                                                                                                  #       IXX1 =  VARST1 + (CSMMASS + NEGBPW)VARST(4 OR 6)     IAVG
039485,000099: 
039486,000100:                                                                                                  #       IXX2 =  VARST2 + (CSMMASS + NEGBPW)VARST(3 OR 7)     IAVG/TLX
