Source Code

These source-code files were derived from scans of the Colossus 237 (Apollo 8 Command Module) program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9 CM) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

039552,000017:                                                                                                  #  LOG SECTION....TVCMASSPROP      PROGRAMMER...MELANSON (ENGEL , SCHLUNDT)

039555,000020:                                                                                                  #  FUNCTIONAL DESCRIPTION:
039556,000021: 
039557,000022:                                                                                                  #       MASSPROP OPERATES IN TWO MODES:(1)IF LEM MASS OR CONFIGURATION ARE UPDATED (MASSPROP DOES NOT TEST
039558,000023:                                                                                                  #  FOR THIS) THE ENTIRE PROGRAM MUST BE RUN THROUGH, BREAKPOINT VALUES AND DERIVATIVES OF THE OUTPUTS WITH
039559,000024:                                                                                                  #  RESPECT TO CSM MASS BEING CALCULATED PRIOR TO CALCULATION OF THE OUTPUTS. (2)OTHERWISE, THE OUTPUTS CAN BE
039560,000025:                                                                                                  #  CALCULATED USING PREVIOUSLY COMPUTED BREAKPOINT VALUES AND DERIVATIVES.
039561,000026: 
039562,000027:                                                                                                  #  CALLING SEQUENCES
039563,000028: 
039564,000029:                                                                                                  #     IF LEM MASS OR CONFIGURATION HAS BEEN UPDATED, TRANSFER TO MASSPROP, OTHERWISE TRANSFER TO FIXCW.
039565,000030:                                                                                                  #           L      TC     BANKCALL OR IBNKCALL
039566,000031:                                                                                                  #           L+1    CADR   MASSPROP
039567,000032:                                                                                                  #                     OR
039568,000033:                                                                                                  #           L+1    CADR   FIXCW
039569,000034: 
039570,000035:                                                                                                  #           L+2    RETURNS VIA Q
039571,000036: 
039572,000037:                                                                                                  #  CALLED   IN PARTICULAR BY DONOUN47 (JOB) AND TVCEXECUTIVE (TASK)
039573,000038: 
039574,000039:                                                                                                  #  JOBS OR TASKS INITIATED - NONE
039575,000040: 
039576,000041:                                                                                                  #  SUBROUTINES CALLED - NONE
039577,000042: 
039578,000043:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED
039579,000044: 
039580,000045:                                                                                                  #       LEMMASS MUST CONTAIN LEM MASS SCALED AT B+16 IN KILOGRAMS
039581,000046:                                                                                                  #       CSMMASS MUST CONTAIN CSM MASS SCALED AT B+16 IN KILOGRAMS
039582,000047: 
039583,000048:                                                                                                  #       DAPDATR1 MUST BE SET TO INDICATE VEHICLE CONFIGURATION.
039584,000049:                                                                                                  #                BITS (15,14,13)  =  ( 0 , 0 , 1 )     LEM OFF
039585,000050:                                                                                                  #                                    ( 0 , 1 , 0 )     LEM ON (ASCNT,DSCNT)
039586,000051:                                                                                                  #                                    ( 1 , 1 , 0 )     LEM ON (ASCNT ONLY)
039587,000052: 
039588,000053: 
039589,000054:                                                                                                  #  ALARMS - NONE
039590,000055: 
039591,000056:                                                                                                  #  EXIT -   TC     Q
039592,000057: 
039593,000058:                                                                                                  #  OUTPUTS:
039594,000059: 
039595,000060:                                                                                                  #       (1)IXX, SINGLE PRECISION SCALED AT B+20 IN KG-M SQ.
039596,000061:                                                                                                  #       (2)IAVG, SINGLE PRECISION SCALED AT B+20 IN KG-M SQ.
039597,000062:                                                                                                  #       (3)IAVG/TLX, SINGLE PRECISION, SCALED AT B+2 SEC-SQD
039598,000063:                                                                                                  #       THEY ARE STORED IN CONSECUTIVE REGISTERS IXX0, IXX1, IXX2
039599,000064: 
039600,000065:                                                                                                  #       CONVERSION FACTOR :  (SLUG-FTSQ) = 0.737562 (KG-MSQ)

039602,000067:                                                                                                  #  OUTPUTS ARE CALCULATED AS FOLLOWS:
039603,000068: 
039604,000069:                                                                                                  #   (1) IF LEM DOCKED, LEMMASS IS FIRST ELIMINATED AS A PARAMETER
039605,000070: 
039606,000071:                                                                                                  #       VARST0 = INTVALUE0 + LEMMASS(SLOPEVAL0)       IXX       BREAKPOINT VALUE
039607,000072:                                                                                                  #       VARST1 = INTVALUE1 + LEMMASS(SLOPEVAL1)       IAVG      BREAKPOINT VALUE
039608,000073:                                                                                                  #       VARST2 = INTVALUE2 + LEMMASS(SLOPEVAL2)       IAVG/TLX  BREAKPOINT VALUE
039609,000074: 
039610,000075:                                                                                                  #       VARST3 = INTVALUE3 + LEMMASS(SLOPEVAL3)       IAVG/TLX  SLOPE  FOR CSMMASS > 33956 LBS  ( SPS > 10000 LBS)
039611,000076:                                                                                                  #       VARST4 = INTVALUE4 + LEMMASS(SLOPEVAL4)       IAVG      SLOPE  FOR CSMMASS > 33956 LBS  ( SPS > 10000 LBS)
039612,000077: 
039613,000078:                                                                                                  #       VARST5 = INTVALUE5 + LEMMASS(SLOPEVAL5)       IXX       SLOPE  FOR ALL VALUES OF CSMMASS
039614,000079: 
039615,000080:                                                                                                  #       VARST6 = INTVALUE6 + LEMMASS(SLOPEVAL6)       IAVG      SLOPE  FOR CSMMASS < 33956 LBS ( SPS < 10000 LBS)
039616,000081:                                                                                                  #       VARST7 = INTVALUE7 + LEMMASS(SLOPEVAL7)       IAVG/TLX  SLOPE  FOR CSMMASS < 33956 LBS ( SPS < 10000 LBS)
039617,000082: 
039618,000083:                                                                                                  #       VARST8 = INTVALUE8 + LEMMASS(SLOPEVAL8)       IAVG      DECREMENT  TO BRKPT VALUE WHEN LEM DSCNT STAGE OFF
039619,000084:                                                                                                  #       VARST9 = INTVALUE9 + LEMMASS(SLOPEVAL9)       IAVG/TLX  DECREMENT  TO BRKPT VALUE WHEN LEM DSCNT STAGE OFF
039620,000085: 
039621,000086: 
039622,000087:                                                                                                  #   (2) IF LEM NOT DOCKED
039623,000088: 
039624,000089:                                                                                                  #       VARST0 = NOLEMVAL0    WHERE THE MEANING AND SCALING OF VARST0
039625,000090:                                                                                                  #          .          .       TO VARST9 ARE THE SAME AS GIVEN ABOVE
039626,000091:                                                                                                  #          .          .
039627,000092:                                                                                                  #          .          .       NOTE... FOR THIS CASE, VARST8,9 HAVE NO
039628,000093:                                                                                                  #       VARST9 = NOLEMVAL9    MEANING (THEY ARE COMPUTED BUT NOT USED)
039629,000094: 
039630,000095: 
039631,000096:                                                                                                  #   (3) THE FINAL OUTPUT CALCULATIONS ARE THEN DONE
039632,000097: 
039633,000098:                                                                                                  #       IXX0 =  VARST0 + (CSMMASS + NEGBPW)VARST5            IXX
039634,000099: 
039635,000100:                                                                                                  #       IXX1 =  VARST1 + (CSMMASS + NEGBPW)VARST(4 OR 6)     IAVG
039636,000101: 
039637,000102:                                                                                                  #       IXX2 =  VARST2 + (CSMMASS + NEGBPW)VARST(3 OR 7)     IAVG/TLX
