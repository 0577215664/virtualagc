Source Code

These source-code files were derived from scans of the Colossus 237 (Apollo 8 Command Module) program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9 CM) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

057668,000027:                                                                                                  #  FUNCTIONAL DESCRIPTION
057669,000028: 
057670,000029:                                                                                                  #       PROGRAM HAS TWO MAIN PARTS. THE FIRST IS SELF-CHECK WHICH RUNS AS A ZERO PRIORITY JOB WITH NO CORE SET, AS
057671,000030:                                                                                                  #  PART OF THE BACK-UP IDLE LOOP. THE SECOND IS SHOW-BANKSUM WHICH RUNS AS A REGULAR EXECUTIVE JOB WITH ITS OWN
057672,000031:                                                                                                  #  STARTING VERB.
057673,000032:                                                                                                  #       THE PURPOSE OF SELF-CHECK IS TO CHECK OUT VARIOUS PARTS OF THE COMPUTER AS OUTLINED BELOW IN THE OPTIONS.
057674,000033:                                                                                                  #       THE PURPOSE OF SHOW-BANKSUM IS TO DISPLAY THE SUM OF EACH BANK , ONE AT A TIME.
057675,000034:                                                                                                  #       IN ALL THERE ARE  7 POSSIBLE OPTIONS IN THIS BLOCK II VERSION OF SELF-CHECK. MORE DETAIL DESCRIPTION MAY BE
057676,000035:                                                                                                  #  FOUND IN E-2065 BLOCK II AGC SELF-CHECK AND SHOW BANKSUM BY EDWIN D. SMALLY DECEMBER 1966, AND ADDENDA 2 AND 3.
057677,000036:                                                                                                  #       THE DIFFERENT OPTIONS ARE CONTROLLED BY PUTTING DIFFERENT NUMBERS IN THE SMODE REGISTER (NOUN 27). BELOW IS
057678,000037:                                                                                                  #  A DESCRIPTION OF WHAT PARTS OF THE COMPUTER THAT ARE CHECKED BY THE OPTIONS, AND THE CORRESPONDING NUMBER, IN
057679,000038:                                                                                                  #  OCTAL, TO LOAD INTO SMODE.
057680,000039:                                                                                                  #  +-4   ERASABLE MEMORY
057681,000040:                                                                                                  #  +-5   FIXED MEMORY
057682,000041:                                                                                                  #  +-1,2,3,6,7,10   EVERYTHING IN OPTIONS 4 AND 5.
057683,000042:                                                                                                  #  -0    SAME AS +-10 UNTIL AN ERROR IS DETECTED.
057684,000043:                                                                                                  #  +0    NO CHECK, PUTS COMPUTER INTO THE BACKUP IDLE LOOP.
057685,000044: 
057686,000045: 
057687,000046:                                                                                                  #  WARNINGS
057688,000047: 
057689,000048:                                                                                                  #       USE OF E MEMORY RESERVED FOR SELF-CHECK (EVEN IN IDLE LOOP) AS TEMP STORAGE BY OTHER PROGRAMS IS DANGEROUS.
057690,000049:                                                                                                  #       SMODE SET GREATER THAN OCT 10 PUTS COMPUTER INTO BACKUP IDLE LOOP.
057691,000050: 
057692,000051: 
057693,000052:                                                                                                  #  CALLING SEQUENCE
057694,000053: 
057695,000054:                                                                                                  #       TO CALL SELF-CHECK KEY IN
057696,000055:                                                                                                  #            V 21 N 27 E  OPTION NUMBER E
057697,000056:                                                                                                  #       TO CALL SHOW-BANKSUM KEY IN
057698,000057:                                                                                                  #            V 91 E         DISPLAYS FIRST BANK
057699,000058:                                                                                                  #            V 33 E         PROCEED, DISPLAYS NEXT BANK
057700,000059: 
057701,000060: 
057702,000061:                                                                                                  #  EXIT MODES, NORMAL AND ALARM
057703,000062: 
057704,000063:                                                                                                  #       SELF-CHECK NORMALLY CONTINUES INDEFINITELY UNLESS THERE IS AN ERROR DETECTED. IF SO + OPTION NUMBERS PUT
057705,000064:                                                                                                  #  COMPUTER INTO BACKUP IDLE LOOP, - OPTION NUMBERS RESTART THE OPTION.
057706,000065:                                                                                                  #       THE -0 OPTION PROCEEDS FROM THE LINE FOLLOWING THE LINE WHERE THE ERROR WAS DETECTED.
057707,000066:                                                                                                  #       SHOW-BANKSUM PROCEEDS UNTIL A TERMINATE IS KEYED IN (V 34 E). THE COMPUTER IS PUT INTO THE BACKUP IDLE LOOP
057708,000067: 
057709,000068: 
057710,000069: 
057711,000070:                                                                                                  #  OUTPUT

057714,000073:                                                                                                  #       SELF-CHECK UPON DETECTING AN ERROR LOADS THE SELF-CHECK ALARM CONSTANT (01102) INTO THE FAILREG SET AND
057715,000074:                                                                                                  #  TURNS ON THE ALARM LIGHT. THE OPERATOR MAY THEN DISPLAY THE THREE FAILREGS BY KEYING IN V 05 N 09 E. FOR FURTHER
057716,000075:                                                                                                  #  INFORMATION HE MAY KEY IN V 05 N 08 E, THE DSKY DISPLAY IN R1 WILL BE ADDRESS+1 OF WHERE THE ERROR WAS DETECTED,
057717,000076:                                                                                                  #  IN R2 THE BBCON OF SELF-CHECK, AND IN R3 THE TOTAL NUMBER OF ERRORS DETECTED BY SELF-CHECK SINCE THE LAST MAN
057718,000077:                                                                                                  #  INITIATED FRESH START (SLAP1).
057719,000078:                                                                                                  #       SHOW-BANKSUM STARTING WITH BANK 0 DISPLAYS IN R1 THE BANK SUM (A +-NUMBER EQUAL TO THE BANK NUMBER), IN R2
057720,000079:                                                                                                  #  THE BANK NUMBER, AND IN R3 THE BUGGER WORD.
057721,000080: 
057722,000081: 
057723,000082:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED
057724,000083: 
057725,000084:                                                                                                  #       ACCOMPLISHED BY FRESH START
057726,000085:                                                                                                  #            SMODE SET TO +0
057727,000086: 
057728,000087: 
057729,000088:                                                                                                  #  DEBRIS
057730,000089: 
057731,000090:                                                                                                  #       ALL EXITS FROM THE CHECK OF ERASABLE (ERASCHK) RESTORE ORIGINAL CONTENTS TO REGISTERS UNDER CHECK.
057732,000091:                                                                                                  #  EXCEPTION IS A RESTART. RESTART THAT OCCURS DURING ERASCHK RESTORES ERASABLE, UNLESS THERE IS EVIDENCE TO DOUBT
057733,000092:                                                                                                  #  E MEMORY, IN WHICH CASE PROGRAM THEN DOES A FRESH START (DOFSTART).
