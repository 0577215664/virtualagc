057364,000023:                                                                                                 #  FUNCTIONAL DESCRIPTION
057365,000024: 
057366,000025:                                                                                                 #       PROGRAM HAS TWO MAIN PARTS. THE FIRST IS SELF-CHECK WHICH RUNS AS A ZERO PRIORITY JOB WITH NO CORE SET, AS
057367,000026:                                                                                                 #  PART OF THE BACK-UP IDLE LOOP. THE SECOND IS SHOW-BANKSUM WHICH RUNS AS A REGULAR EXECUTIVE JOB WITH ITS OWN
057368,000027:                                                                                                 #  STARTING VERB.
057369,000028:                                                                                                 #       THE PURPOSE OF SELF-CHECK IS TO CHECK OUT VARIOUS PARTS OF THE COMPUTER AS OUTLINED BELOW IN THE OPTIONS.
057370,000029:                                                                                                 #       THE PURPOSE OF SHOW-BANKSUM IS TO DISPLAY THE SUM OF EACH BANK , ONE AT A TIME.
057371,000030:                                                                                                 #       IN ALL THERE ARE  7 POSSIBLE OPTIONS IN THIS BLOCK II VERSION OF SELF-CHECK. MORE DETAIL DESCRIPTION MAY BE
057372,000031:                                                                                                 #  FOUND IN E-2065 BLOCK II AGC SELF-CHECK AND SHOW BANKSUM BY EDWIN D. SMALLY DECEMBER 1966, AND ADDENDA 2 AND 3.
057373,000032:                                                                                                 #       THE DIFFERENT OPTIONS ARE CONTROLLED BY PUTTING DIFFERENT NUMBERS IN THE SMODE REGISTER (NOUN 27). BELOW IS
057374,000033:                                                                                                 #  A DESCRIPTION OF WHAT PARTS OF THE COMPUTER THAT ARE CHECKED BY THE OPTIONS, AND THE CORRESPONDING NUMBER, IN
057375,000034:                                                                                                 #  OCTAL, TO LOAD INTO SMODE.
057376,000035:                                                                                                 #  +-4   ERASABLE MEMORY
057377,000036:                                                                                                 #  +-5   FIXED MEMORY
057378,000037:                                                                                                 #  +-1,2,3,6,7,10   EVERYTHING IN OPTIONS 4 AND 5.
057379,000038:                                                                                                 #  -0    SAME AS +-10 UNTIL AN ERROR IS DETECTED.
057380,000039:                                                                                                 #  +0    NO CHECK, PUTS COMPUTER INTO THE BACKUP IDLE LOOP.
057381,000040: 
057382,000041: 
057383,000042:                                                                                                 #  WARNINGS
057384,000043: 
057385,000044:                                                                                                 #       USE OF E MEMORY RESERVED FOR SELF-CHECK (EVEN IN IDLE LOOP) AS TEMP STORAGE BY OTHER PROGRAMS IS DANGEROUS.
057386,000045:                                                                                                 #       SMODE SET GREATER THAN OCT 10 PUTS COMPUTER INTO BACKUP IDLE LOOP.
057387,000046: 
057388,000047: 
057389,000048:                                                                                                 #  CALLING SEQUENCE
057390,000049: 
057391,000050:                                                                                                 #       TO CALL SELF-CHECK KEY IN
057392,000051:                                                                                                 #            V 21 N 27 E   OPTION NUMBER E
057393,000052:                                                                                                 #       TO CALL SHOW-BANKSUM KEY IN
057394,000053:                                                                                                 #            V 91 E         DISPLAYS FIRST BANK
057395,000054:                                                                                                 #            V 33 E         PROCEED, DISPLAYS NEXT BANK
057396,000055: 
057397,000056: 
057398,000057:                                                                                                 #  EXIT MODES, NORMAL AND ALARM
057399,000058: 
057400,000059:                                                                                                 #       SELF-CHECK NORMALLY CONTINUES INDEFINITELY UNLESS THERE IS AN ERROR DETECTED. IF SO + OPTION NUMBERS PUT
057401,000060:                                                                                                 #  COMPUTER INTO BACKUP IDLE LOOP, - OPTION NUMBERS RESTART THE OPTION.
057402,000061:                                                                                                 #       THE -0 OPTION PROCEEDS FROM THE LINE FOLLOWING THE LINE WHERE THE ERROR WAS DETECTED.
057403,000062:                                                                                                 #       SHOW-BANKSUM PROCEEDS UNTIL A TERMINATE IS KEYED IN (V 34 E). THE COMPUTER IS PUT INTO THE BACKUP IDLE LOOP
057404,000063: 
057405,000064: 
057406,000065: 
057407,000066:                                                                                                 #  OUTPUT
057408,000067: 
057409,000068:                                                                                                 # # Page 1359
057410,000069:                                                                                                 #       SELF-CHECK UPON DETECTING AN ERROR LOADS THE SELF-CHECK ALARM CONSTANT (01102) INTO THE FAILREG SET AND
057411,000070:                                                                                                 #  TURNS ON THE ALARM LIGHT. THE OPERATOR MAY THEN DISPLAY THE THREE FAILREGS BY KEYING IN V 05 N 09 E. FOR FURTHER
057412,000071:                                                                                                 #  INFORMATION HE MAY KEY IN V 05 N 08 E, THE DSKY DISPLAY IN R1 WILL BE ADDRESS+1 OF WHERE THE ERROR WAS DETECTED,
057413,000072:                                                                                                 #  IN R2 THE BBCON OF SELF-CHECK, AND IN R3 THE TOTAL NUMBER OF ERRORS DETECTED BY SELF-CHECK SINCE THE LAST MAN
057414,000073:                                                                                                 #  INITIATED FRESH START (SLAP1).
057415,000074:                                                                                                 #       SHOW-BANKSUM STARTING WTIH BANK 0 DISPLAYS IN R1 THE BANK SUM (A +-NUMBER EQUAL TO THE BANK NUMBER), IN R2
057416,000075:                                                                                                 #  THE BANK NUMBER, AND IN R3 THE BUGGER WORD.
057417,000076: 
057418,000077: 
057419,000078:                                                                                                 #  ERASABLE INITIALIZATION REQUIRED
057420,000079: 
057421,000080:                                                                                                 #       ACCOMPLISHED BY FRESH START
057422,000081:                                                                                                 #            SMODE SET TO +0
057423,000082: 
057424,000083: 
057425,000084:                                                                                                 #  DEBRIS
057426,000085: 
057427,000086:                                                                                                 #       ALL EXITS FROM THE CHECK OF ERASABLE (ERASCHK) RESTORE ORIGINAL CONTENTS TO REGISTERS UNDER CHECK.
057428,000087:                                                                                                 #  EXCEPTION IS A RESTART. RESTART THAT OCCURS DURING ERASCHK RESTORES ERASABLE, UNLESS THERE IS EVIDENCE TO DOUBT
057429,000088:                                                                                                 #  E MEMORY, IN WHICH CASE PROGRAM THEN DOES A FRESH START (DOFSTART).
