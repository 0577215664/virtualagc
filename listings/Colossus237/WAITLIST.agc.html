These source-code files were obtained by digitally photographing the Colossus 237 program listing from the private collection of original AGC developer Fred Martin, and then modifying pre-existing Colossus 249 (Apollo 9) source files to incorporate changes. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. 050332,000097:                                                                                                 #     THE APPROXIMATE TIME MAY BE CALCULATED AS FOLLOWS:
050333,000098: 
050334,000099:                                                                                                 #               LET T0 = THE TIME OF THE TC WAITLIST
050335,000100:                                                                                                 #               LET TS = T0 +147U + COUNTER INCREMENTS (SET UP TIME)
050336,000101:                                                                                                 #               LET X  = TS -(100TS)/100  (VARIANCE FROM COUNTERS)
050337,000102:                                                                                                 #               LET Y  = LENGTH OF TIME OF INHIBIT INTERRUPT AFTER T3RUPT
050338,000103:                                                                                                 #               LET Z  = LENGTH OF TIME TO PROCESS TASKS WHICH ARE DUE THIS T3RUPT BUT DISPATCHED EARLIER.
050339,000104:                                                                                                 #     (Z=0, USUALLY).

050341,000106:                                                                                                 #            THEN   DELTD = TS+DELTA T -X +Y +Z +1.05MS* +COUNTERS*
050342,000107:                                                                                                 #     *-THE TIME TAKEN BY WAITLIST ITSELF AND THE COUNTER TICKING DURING THIS WAITLIST TIME.
050343,000108: 
050344,000109: 
050345,000110:                                                                                                 #          IN SHORT, THE ACTUAL TIME TO RETURN CONTROL TO A 2CADR IS AUGMENTED BY THE TIME TO SET UP THE TASK:S
050346,000111:                                                                                                 #     INTERRUPT, ALL COUNTERS TICKING, THE T3RUPT PROCESSING TIME, THE WAITLIST PROCESSING TIME AND THE POSSIBILITY
050347,000112:                                                                                                 #     OF OTHER TASKS INHIBITING THE INTERRUPT.
