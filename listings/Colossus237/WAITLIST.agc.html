Source Code

These source-code files were derived from scans of the Colossus 237 program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

050274,000025:                                                                                                  #  FUNCTIONAL DESCRIPTION-
050275,000026:                                                                                                  #          PART OF A SECTION OF PROGRAMS,-WAITLIST, TASKOVER, T3RUPT, USED  TO CALL A PROGRAM, (CALLED A TASK),
050276,000027:                                                                                                  #     WHICH IS TO BEGIN IN C(A) CENTISECONDS.  WAITLIST UPDATES TIME3, LST1, AND LST2. THE MEANING OF THESE LISTS
050277,000028:                                                                                                  #     FOLLOW.
050278,000029: 
050279,000030:                                                                                                  #                  C(TIME3)  = 16384 -(T1-T) CENTISECONDS, (T=PRESENT TIME, T1-TIME FOR TASK1)
050280,000031: 
050281,000032: 
050282,000033:                                                                                                  #                               C(LST1)    = -(T2-T1)+1
050283,000034:                                                                                                  #                               C(LST1 +1) = -(T3-T2)+1
050284,000035:                                                                                                  #                               C(LST1 +2) = -(T4-T3)+1
050285,000036:                                                                                                  #                                          .
050286,000037:                                                                                                  #                                          .
050287,000038:                                                                                                  #                               C(LST1 +6) = -(T8-T7)+1
050288,000039:                                                                                                  #                               C(LST1 +7) = -(T9-T8)+1
050289,000040: 
050290,000041:                                                                                                  #                               C(LST2)    = 2CADR OF TASK1
050291,000042:                                                                                                  #                               C(LST2 +2) = 2CADR OF TASK2
050292,000043:                                                                                                  #                                          .
050293,000044:                                                                                                  #                                          .
050294,000045:                                                                                                  #                               C(LST2 +14)= 2CADR OF TASK8
050295,000046:                                                                                                  #                               C(LST2 +16)= 2CADR OF TASK9
050296,000047: 
050297,000048:                                                                                                  #  WARNINGS-
050298,000049:                                                                                                  #  --------
050299,000050:                                                                                                  #               1) 1 <= C(A) <= 16250D (1 CENTISCOND TO 162.5 SEC)
050300,000051:                                                                                                  #               2) 9 TASKS MAXIMUM
050301,000052:                                                                                                  #               3) TASKS CALLED UNDER INTERRUPT INHIBITED
050302,000053:                                                                                                  #               4) TASKS END BY TC TASKOVER
050303,000054: 
050304,000055:                                                                                                  #  CALLING SEQUENCE-
050305,000056: 
050306,000057:                                                                                                  #               L-1    CA     DELTAT   (TIME IN CENTISECONDS TO TASK START)
050307,000058:                                                                                                  #               L      TC     WAITLIST
050308,000059:                                                                                                  #               L+1    2CADR  DESIRED TASK
050309,000060:                                                                                                  #               L+2    (MINOR OF 2CADR)
050310,000061:                                                                                                  #               L+3    RELINT          (RETURNS HERE)

050313,000064:                                                                                                  #  -------
050314,000065:                                                                                                  #               TWIDDLE IS FOR USE WHEN THE TASK BEING SET UP IS IN THE SAME EBANK AND FBANK AS THE USER.   IN
050315,000066:                                                                                                  #               SUCH CASES, IT IMPROVES UPON WAITLIST BY ELIMINATING THE NEED FOR THE BBCON HALF OF THE 2CADR,

050317,000068:                                                                                                  #               SAVING A WORD.   TWIDDLE IS LIKE WAITLIST IN EVERY RESPECT EXCEPT CALLING SEQUENCE, TO WIT-
050318,000069: 
050319,000070:                                                                                                  #               L-1    CA     DELTAT
050320,000071:                                                                                                  #               L      TC     TWIDDLE
050321,000072:                                                                                                  #               L+1    ADRES  DESIRED TASK
050322,000073:                                                                                                  #               L+2    RELINT               (RETURNS HERE)

050325,000076: 
050326,000077:                                                                                                  #               AT L+3 OF CALLING SEQUENCE.
050327,000078: 
050328,000079:                                                                                                  #  ALARM OR ABORT EXIT MODES-
050329,000080: 
050330,000081:                                                                                                  #               TC     ABORT
050331,000082:                                                                                                  #               OCT    1203   (WAITLIST OVERFLOW - TOO MANY TASKS)
050332,000083: 
050333,000084:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED-
050334,000085: 
050335,000086:                                                                                                  #               ACCOMPLISHED BY FRESH START,--LST2,..., LST2 +16 =ENDTASK
050336,000087:                                                                                                  #                                             LST1,..., LST1 +7  =NEG1/2
050337,000088: 
050338,000089:                                                                                                  #  OUTPUT--
050339,000090: 
050340,000091:                                                                                                  #               LST1 AND LST2 UPDATED WTIH NEW TASK AND ASSOCIATED TIME.
050341,000092:                                                                                                  #  DEBRIS-
050342,000093: 
050343,000094:                                                                                                  #               CENTRALS- A,Q,L
050344,000095:                                                                                                  #               OTHER   - WAITEXIT, WAITADR, WAITTEMP, WAITBANK
050345,000096:                                                                                                  #  DETAILED ANALYSIS OF TIMING-
050346,000097:                                                                                                  #          CONTROL WILL NOT BE RETURNED TO THE SPECIFIED ADDRESS (2CADR)   IN EXACTLY DELTA T CENTISECONDS.
050347,000098:                                                                                                  #     THE APPROXIMATE TIME MAY BE CALCULATED AS FOLLOWS:
050348,000099: 
050349,000100:                                                                                                  #               LET T0 = THE TIME OF THE TC WAITLIST
050350,000101:                                                                                                  #               LET TS = T0 +147U + COUNTER INCREMENTS (SET UP TIME)
050351,000102:                                                                                                  #               LET X  = TS -(100TS)/100  (VARIANCE FROM COUNTERS)
050352,000103:                                                                                                  #               LET Y  = LENGTH OF TIME OF INHIBIT INTERRUPT AFTER T3RUPT
050353,000104:                                                                                                  #               LET Z  = LENGTH OF TIME TO PROCESS TASKS WHICH ARE DUE THIS T3RUPT BUT DISPATCHED EARLIER.
050354,000105:                                                                                                  #     (Z=0, USUALLY).
050355,000106:                                                                                                  #               LET DELTD = THE ACTUAL TIME TAKEN TO GIVE CONTROL TO 2CADR
050356,000107:                                                                                                  #            THEN   DELTD = TS+DELTA T -X +Y +Z +1.05MS* +COUNTERS*
050357,000108:                                                                                                  #     *-THE TIME TAKEN BY WAITLIST ITSELF AND THE COUNTER TICKING DURING THIS WAITLIST TIME.
050358,000109: 
050359,000110: 
050360,000111:                                                                                                  #          IN SHORT, THE ACTUAL TIME TO RETURN CONTROL TO A 2CADR IS AUGMENTED BY THE TIME TO SET UP THE TASK:S
050361,000112:                                                                                                  #     INTERRUPT, ALL COUNTERS TICKING, THE T3RUPT PROCESSING TIME, THE WAITLIST PROCESSING TIME AND THE POSSIBILITY
050362,000113:                                                                                                  #     OF OTHER TASKS INHIBITING THE INTERRUPT.
