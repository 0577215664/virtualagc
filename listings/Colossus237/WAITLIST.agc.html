Source Code

These source-code files were derived from scans of the Colossus 237 (Apollo 8 Command Module) program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9 CM) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

050488,000029:                                                                                                  #  FUNCTIONAL DESCRIPTION-
050489,000030:                                                                                                  #          PART OF A SECTION OF PROGRAMS,-WAITLIST, TASKOVER, T3RUPT, USED  TO CALL A PROGRAM, (CALLED A TASK),
050490,000031:                                                                                                  #     WHICH IS TO BEGIN IN C(A) CENTISECONDS.  WAITLIST UPDATES TIME3, LST1 AND LST2. THE MEANING OF THESE LISTS
050491,000032:                                                                                                  #     FOLLOW.
050492,000033: 
050493,000034:                                                                                                  #                  C(TIME3)  = 16384 -(T1-T) CENTISECONDS, (T=PRESENT TIME, T1-TIME FOR TASK1)
050494,000035: 
050495,000036: 
050496,000037:                                                                                                  #                               C(LST1)    = -(T2-T1)+1
050497,000038:                                                                                                  #                               C(LST1 +1) = -(T3-T2)+1
050498,000039:                                                                                                  #                               C(LST1 +2) = -(T4-T3)+1
050499,000040:                                                                                                  #                                          .
050500,000041:                                                                                                  #                                          .
050501,000042:                                                                                                  #                               C(LST1 +6) = -(T8-T7)+1
050502,000043:                                                                                                  #                               C(LST1 +7) = -(T9-T8)+1
050503,000044: 
050504,000045:                                                                                                  #                               C(LST2)    = 2CADR OF TASK1
050505,000046:                                                                                                  #                               C(LST2 +2) = 2CADR OF TASK2
050506,000047:                                                                                                  #                                          .
050507,000048:                                                                                                  #                                          .
050508,000049:                                                                                                  #                               C(LST2 +14)= 2CADR OF TASK8
050509,000050:                                                                                                  #                               C(LST2 +16)= 2CADR OF TASK9
050510,000051: 
050511,000052:                                                                                                  #  WARNINGS-
050512,000053:                                                                                                  #  --------
050513,000054:                                                                                                  #               1) 1 <= C(A) <= 16250D (1 CENTISECOND TO 162.5 SEC)
050514,000055:                                                                                                  #               2) 9 TASKS MAXIMUM
050515,000056:                                                                                                  #               3) TASKS CALLED UNDER INTERRUPT INHIBITED
050516,000057:                                                                                                  #               4) TASKS END BY TC TASKOVER

050519,000060:

050521,000062:                                                                                                  #               L      TC     WAITLIST
050522,000063:                                                                                                  #               L+1    2CADR  DESIRED TASK
050523,000064:                                                                                                  #               L+2    (MINOR OF 2CADR)
050524,000065:                                                                                                  #               L+3    RELINT          (RETURNS HERE)

050559,000100:                                                                                                  #  DETAILED ANALYSIS OF TIMING-
050560,000101:                                                                                                  #          CONTROL WILL NOT BE RETURNED TO THE SPECIFIED ADDRESS (2CADR)   IN EXACTLY DELTA T CENTISECONDS.
050561,000102:                                                                                                  #     THE APPROXIMATE TIME MAY BE CALCULATED AS FOLLOWS
050562,000103: 
050563,000104:                                                                                                  #               LET T0 = THE TIME OF THE TC WAITLIST
050564,000105:                                                                                                  #               LET TS = T0 +147U + COUNTER INCREMENTS (SET UP TIME)
050565,000106:                                                                                                  #               LET X  = TS -(100TS)/100  (VARIANCE FROM COUNTERS)
050566,000107:                                                                                                  #               LET Y  = LENGTH OF TIME OF INHIBIT INTERRUPT AFTER T3RUPT
050567,000108:                                                                                                  #               LET Z  = LENGTH OF TIME TO PROCESS TASKS WHICH ARE DUE THIS T3RUPT BUT DISPATCHED EARLIER.
050568,000109:                                                                                                  #     (Z=0, USUALLY)
050569,000110:                                                                                                  #               LET DELTD = THE ACTUAL TIME TAKEN TO GIVE CONTROL TO 2CADR
050570,000111:                                                                                                  #            THEN   DELTD = TS+DELTA T -X +Y +Z +1.05MS* +COUNTERS*
050571,000112:                                                                                                  #     *-THE TIME TAKEN BY WAITLIST ITSELF AND THE COUNTER TICKING DURING THIS WAITLIST TIME.
050572,000113: 
050573,000114: 
050574,000115:                                                                                                  #          IN SHORT, THE ACTUAL TIME TO RETURN CONTROL TO A 2CADR IS AUGMENTED BY THE TIME TO SET UP THE TASK:S
050575,000116:                                                                                                  #     INTERRUPT, ALL COUNTERS TICKING, THE T3RUPT PROCESSING TIME, THE WAITLIST PROCESSING TIME AND THE POSSIBILITY
050576,000117:                                                                                                  #     OF OTHER TASKS INHIBITING THE INTERRUPT.
