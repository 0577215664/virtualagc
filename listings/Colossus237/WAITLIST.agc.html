Source Code

These source-code files were derived from scans of the Colossus 237 program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

050268,000024:                                                                                                  #  FUNCTIONAL DESCRIPTION-
050269,000025:                                                                                                  #          PART OF A SECTION OF PROGRAMS,-WAITLIST, TASKOVER, T3RUPT, USED  TO CALL A PROGRAM, (CALLED A TASK),
050270,000026:                                                                                                  #     WHICH IS TO BEGIN IN C(A) CENTISECONDS.  WAITLIST UPDATES TIME3, LST1, AND LST2. THE MEANING OF THESE LISTS
050271,000027:                                                                                                  #     FOLLOW.
050272,000028: 
050273,000029:                                                                                                  #                  C(TIME3)  = 16384 -(T1-T) CENTISECONDS, (T=PRESENT TIME, T1-TIME FOR TASK1)
050274,000030: 
050275,000031: 
050276,000032:                                                                                                  #                               C(LST1)    = -(T2-T1)+1
050277,000033:                                                                                                  #                               C(LST1 +1) = -(T3-T2)+1
050278,000034:                                                                                                  #                               C(LST1 +2) = -(T4-T3)+1
050279,000035:                                                                                                  #                                          .
050280,000036:                                                                                                  #                                          .
050281,000037:                                                                                                  #                               C(LST1 +6) = -(T8-T7)+1
050282,000038:                                                                                                  #                               C(LST1 +7) = -(T9-T8)+1
050283,000039: 
050284,000040:                                                                                                  #                               C(LST2)    = 2CADR OF TASK1
050285,000041:                                                                                                  #                               C(LST2 +2) = 2CADR OF TASK2
050286,000042:                                                                                                  #                                          .
050287,000043:                                                                                                  #                                          .
050288,000044:                                                                                                  #                               C(LST2 +14)= 2CADR OF TASK8
050289,000045:                                                                                                  #                               C(LST2 +16)= 2CADR OF TASK9
050290,000046:

050292,000048:                                                                                                  #  --------
050293,000049:                                                                                                  #               1) 1 <= C(A) <= 16250D (1 CENTISCOND TO 162.5 SEC)
050294,000050:                                                                                                  #               2) 9 TASKS MAXIMUM
050295,000051:                                                                                                  #               3) TASKS CALLED UNDER INTERRUPT INHIBITED
050296,000052:                                                                                                  #               4) TASKS END BY TC TASKOVER

050338,000094:                                                                                                  #               OTHER   - WAITEXIT, WAITADR, WAITTEMP, WAITBANK
050339,000095:                                                                                                  #  DETAILED ANALYSIS OF TIMING-
050340,000096:                                                                                                  #          CONTROL WILL NOT BE RETURNED TO THE SPECIFIED ADDRESS (2CADR)   IN EXACTLY DELTA T CENTISECONDS.
050341,000097:                                                                                                  #     THE APPROXIMATE TIME MAY BE CALCULATED AS FOLLOWS:
050342,000098: 
050343,000099:                                                                                                  #               LET T0 = THE TIME OF THE TC WAITLIST
050344,000100:                                                                                                  #               LET TS = T0 +147U + COUNTER INCREMENTS (SET UP TIME)
050345,000101:                                                                                                  #               LET X  = TS -(100TS)/100  (VARIANCE FROM COUNTERS)
050346,000102:                                                                                                  #               LET Y  = LENGTH OF TIME OF INHIBIT INTERRUPT AFTER T3RUPT
050347,000103:                                                                                                  #               LET Z  = LENGTH OF TIME TO PROCESS TASKS WHICH ARE DUE THIS T3RUPT BUT DISPATCHED EARLIER.
050348,000104:                                                                                                  #     (Z=0, USUALLY).
050349,000105:                                                                                                  #               LET DELTD = THE ACTUAL TIME TAKEN TO GIVE CONTROL TO 2CADR
050350,000106:                                                                                                  #            THEN   DELTD = TS+DELTA T -X +Y +Z +1.05MS* +COUNTERS*
050351,000107:                                                                                                  #     *-THE TIME TAKEN BY WAITLIST ITSELF AND THE COUNTER TICKING DURING THIS WAITLIST TIME.
050352,000108: 
050353,000109: 
050354,000110:                                                                                                  #          IN SHORT, THE ACTUAL TIME TO RETURN CONTROL TO A 2CADR IS AUGMENTED BY THE TIME TO SET UP THE TASK:S
050355,000111:                                                                                                  #     INTERRUPT, ALL COUNTERS TICKING, THE T3RUPT PROCESSING TIME, THE WAITLIST PROCESSING TIME AND THE POSSIBILITY
050356,000112:                                                                                                  #     OF OTHER TASKS INHIBITING THE INTERRUPT.
