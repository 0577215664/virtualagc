Source Code

These source-code files were derived from scans of the Colossus 237 (Apollo 8 Command Module) program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9 CM) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

050508,000030:                                                                                                  #  FUNCTIONAL DESCRIPTION-
050509,000031:                                                                                                  #          PART OF A SECTION OF PROGRAMS,-WAITLIST, TASKOVER, T3RUPT, USED  TO CALL A PROGRAM, (CALLED A TASK),
050510,000032:                                                                                                  #     WHICH IS TO BEGIN IN C(A) CENTISECONDS.  WAITLIST UPDATES TIME3, LST1 AND LST2. THE MEANING OF THESE LISTS
050511,000033:                                                                                                  #     FOLLOW.
050512,000034: 
050513,000035:                                                                                                  #                  C(TIME3)  = 16384 -(T1-T) CENTISECONDS, (T=PRESENT TIME, T1-TIME FOR TASK1)
050514,000036: 
050515,000037: 
050516,000038:                                                                                                  #                               C(LST1)    = -(T2-T1)+1
050517,000039:                                                                                                  #                               C(LST1 +1) = -(T3-T2)+1
050518,000040:                                                                                                  #                               C(LST1 +2) = -(T4-T3)+1
050519,000041:                                                                                                  #                                          .
050520,000042:                                                                                                  #                                          .
050521,000043:                                                                                                  #                               C(LST1 +6) = -(T8-T7)+1
050522,000044:                                                                                                  #                               C(LST1 +7) = -(T9-T8)+1
050523,000045: 
050524,000046:                                                                                                  #                               C(LST2)    = 2CADR OF TASK1
050525,000047:                                                                                                  #                               C(LST2 +2) = 2CADR OF TASK2
050526,000048:                                                                                                  #                                          .
050527,000049:                                                                                                  #                                          .
050528,000050:                                                                                                  #                               C(LST2 +14)= 2CADR OF TASK8
050529,000051:                                                                                                  #                               C(LST2 +16)= 2CADR OF TASK9
050530,000052: 
050531,000053:                                                                                                  #  WARNINGS-
050532,000054:                                                                                                  #  --------
050533,000055:                                                                                                  #               1) 1 <= C(A) <= 16250D (1 CENTISECOND TO 162.5 SEC)
050534,000056:                                                                                                  #               2) 9 TASKS MAXIMUM
050535,000057:                                                                                                  #               3) TASKS CALLED UNDER INTERRUPT INHIBITED
050536,000058:                                                                                                  #               4) TASKS END BY TC TASKOVER
050537,000059: 
050538,000060:                                                                                                  #  CALLING SEQUENCE-
050539,000061: 
050540,000062:                                                                                                  #               L-1    CA     DELTAT   (TIME IN CENTISECONDS TO TASK START)
050541,000063:                                                                                                  #               L      TC     WAITLIST
050542,000064:                                                                                                  #               L+1    2CADR  DESIRED TASK
050543,000065:                                                                                                  #               L+2    (MINOR OF 2CADR)
050544,000066:                                                                                                  #               L+3    RELINT          (RETURNS HERE)
050545,000067: 
050546,000068:                                                                                                  #  TWIDDLE-
050547,000069:                                                                                                  #  -------
050548,000070:                                                                                                  #               TWIDDLE IS FOR USE WHEN THE TASK BEING SET UP IS IN THE SAME EBANK AND FBANK AS THE USER.   IN
050549,000071:                                                                                                  #               SUCH CASES, IT IMPROVES UPON WAITLIST BY ELIMINATING THE NEED FOR THE BBCON HALF OF THE 2CADR,

050578,000100:                                                                                                  #               OTHER   - WAITEXIT, WAITADR, WAITTEMP, WAITBANK
050579,000101:                                                                                                  #  DETAILED ANALYSIS OF TIMING-
050580,000102:                                                                                                  #          CONTROL WILL NOT BE RETURNED TO THE SPECIFIED ADDRESS (2CADR)   IN EXACTLY DELTA T CENTISECONDS.
050581,000103:                                                                                                  #     THE APPROXIMATE TIME MAY BE CALCULATED AS FOLLOWS
050582,000104: 
050583,000105:                                                                                                  #               LET T0 = THE TIME OF THE TC WAITLIST
050584,000106:                                                                                                  #               LET TS = T0 +147U + COUNTER INCREMENTS (SET UP TIME)
050585,000107:                                                                                                  #               LET X  = TS -(100TS)/100  (VARIANCE FROM COUNTERS)
050586,000108:                                                                                                  #               LET Y  = LENGTH OF TIME OF INHIBIT INTERRUPT AFTER T3RUPT
050587,000109:                                                                                                  #               LET Z  = LENGTH OF TIME TO PROCESS TASKS WHICH ARE DUE THIS T3RUPT BUT DISPATCHED EARLIER.
050588,000110:                                                                                                  #     (Z=0, USUALLY)
050589,000111:                                                                                                  #               LET DELTD = THE ACTUAL TIME TAKEN TO GIVE CONTROL TO 2CADR
050590,000112:                                                                                                  #            THEN   DELTD = TS+DELTA T -X +Y +Z +1.05MS* +COUNTERS*
050591,000113:                                                                                                  #     *-THE TIME TAKEN BY WAITLIST ITSELF AND THE COUNTER TICKING DURING THIS WAITLIST TIME.
050592,000114: 
050593,000115: 
050594,000116:                                                                                                  #          IN SHORT, THE ACTUAL TIME TO RETURN CONTROL TO A 2CADR IS AUGMENTED BY THE TIME TO SET UP THE TASK:S
050595,000117:                                                                                                  #     INTERRUPT, ALL COUNTERS TICKING, THE T3RUPT PROCESSING TIME, THE WAITLIST PROCESSING TIME AND THE POSSIBILITY
050596,000118:                                                                                                  #     OF OTHER TASKS INHIBITING THE INTERRUPT.
