Source Code

These source-code files were derived from scans of the Colossus 237 (Apollo 8 Command Module) program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9 CM) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

050510,000030:                                                                                                  #  FUNCTIONAL DESCRIPTION-
050511,000031:                                                                                                  #          PART OF A SECTION OF PROGRAMS,-WAITLIST, TASKOVER, T3RUPT, USED  TO CALL A PROGRAM, (CALLED A TASK),
050512,000032:                                                                                                  #     WHICH IS TO BEGIN IN C(A) CENTISECONDS.  WAITLIST UPDATES TIME3, LST1 AND LST2. THE MEANING OF THESE LISTS
050513,000033:                                                                                                  #     FOLLOW.
050514,000034: 
050515,000035:                                                                                                  #                  C(TIME3)  = 16384 -(T1-T) CENTISECONDS, (T=PRESENT TIME, T1-TIME FOR TASK1)
050516,000036: 
050517,000037: 
050518,000038:                                                                                                  #                               C(LST1)    = -(T2-T1)+1
050519,000039:                                                                                                  #                               C(LST1 +1) = -(T3-T2)+1
050520,000040:                                                                                                  #                               C(LST1 +2) = -(T4-T3)+1
050521,000041:                                                                                                  #                                          .
050522,000042:                                                                                                  #                                          .
050523,000043:                                                                                                  #                               C(LST1 +6) = -(T8-T7)+1
050524,000044:                                                                                                  #                               C(LST1 +7) = -(T9-T8)+1
050525,000045: 
050526,000046:                                                                                                  #                               C(LST2)    = 2CADR OF TASK1
050527,000047:                                                                                                  #                               C(LST2 +2) = 2CADR OF TASK2
050528,000048:                                                                                                  #                                          .
050529,000049:                                                                                                  #                                          .
050530,000050:                                                                                                  #                               C(LST2 +14)= 2CADR OF TASK8
050531,000051:                                                                                                  #                               C(LST2 +16)= 2CADR OF TASK9
050532,000052: 
050533,000053:                                                                                                  #  WARNINGS-
050534,000054:                                                                                                  #  --------
050535,000055:                                                                                                  #               1) 1 <= C(A) <= 16250D (1 CENTISECOND TO 162.5 SEC)
050536,000056:                                                                                                  #               2) 9 TASKS MAXIMUM
050537,000057:                                                                                                  #               3) TASKS CALLED UNDER INTERRUPT INHIBITED
050538,000058:                                                                                                  #               4) TASKS END BY TC TASKOVER
050539,000059: 
050540,000060:                                                                                                  #  CALLING SEQUENCE-
050541,000061: 
050542,000062:                                                                                                  #               L-1    CA     DELTAT   (TIME IN CENTISECONDS TO TASK START)
050543,000063:                                                                                                  #               L      TC     WAITLIST
050544,000064:                                                                                                  #               L+1    2CADR  DESIRED TASK
050545,000065:                                                                                                  #               L+2    (MINOR OF 2CADR)
050546,000066:                                                                                                  #               L+3    RELINT          (RETURNS HERE)
050547,000067: 
050548,000068:                                                                                                  #  TWIDDLE-
050549,000069:                                                                                                  #  -------
050550,000070:                                                                                                  #               TWIDDLE IS FOR USE WHEN THE TASK BEING SET UP IS IN THE SAME EBANK AND FBANK AS THE USER.   IN
050551,000071:                                                                                                  #               SUCH CASES, IT IMPROVES UPON WAITLIST BY ELIMINATING THE NEED FOR THE BBCON HALF OF THE 2CADR,

050553,000073:                                                                                                  #               SAVING A WORD.   TWIDDLE IS LIKE WAITLIST IN EVERY RESPECT EXCEPT CALLING SEQUENCE, TO WIT-
050554,000074: 
050555,000075:                                                                                                  #               L-1    CA     DELTAT
050556,000076:                                                                                                  #               L      TC     TWIDDLE
050557,000077:                                                                                                  #               L+1    ADRES  DESIRED TASK
050558,000078:                                                                                                  #               L+2    RELINT               (RETURNS HERE)

050561,000081:

050563,000083: 
050564,000084:                                                                                                  #  ALARM OR ABORT EXIT MODES-
050565,000085: 
050566,000086:                                                                                                  #               TC     ABORT

050568,000088: 
050569,000089:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED-
050570,000090: 
050571,000091:                                                                                                  #               ACCOMPLISHED BY FRESH START,--LST2,..., LST2 +16 =ENDTASK
050572,000092:                                                                                                  #                                             LST1,..., LST1 +7  =NEG1/2
050573,000093: 
050574,000094:                                                                                                  #  OUTPUT--
050575,000095: 
050576,000096:                                                                                                  #               LST1 AND LST2 UPDATED WITH NEW TASK AND ASSOCIATED TIME.
050577,000097:                                                                                                  #  DEBRIS-
050578,000098: 
050579,000099:                                                                                                  #               CENTRALS- A,Q,L
050580,000100:                                                                                                  #               OTHER   - WAITEXIT, WAITADR, WAITTEMP, WAITBANK
050581,000101:                                                                                                  #  DETAILED ANALYSIS OF TIMING-
050582,000102:                                                                                                  #          CONTROL WILL NOT BE RETURNED TO THE SPECIFIED ADDRESS (2CADR)   IN EXACTLY DELTA T CENTISECONDS.
050583,000103:                                                                                                  #     THE APPROXIMATE TIME MAY BE CALCULATED AS FOLLOWS
050584,000104: 
050585,000105:                                                                                                  #               LET T0 = THE TIME OF THE TC WAITLIST
050586,000106:                                                                                                  #               LET TS = T0 +147U + COUNTER INCREMENTS (SET UP TIME)
050587,000107:                                                                                                  #               LET X  = TS -(100TS)/100  (VARIANCE FROM COUNTERS)
050588,000108:                                                                                                  #               LET Y  = LENGTH OF TIME OF INHIBIT INTERRUPT AFTER T3RUPT
050589,000109:                                                                                                  #               LET Z  = LENGTH OF TIME TO PROCESS TASKS WHICH ARE DUE THIS T3RUPT BUT DISPATCHED EARLIER.
050590,000110:                                                                                                  #     (Z=0, USUALLY)
050591,000111:                                                                                                  #               LET DELTD = THE ACTUAL TIME TAKEN TO GIVE CONTROL TO 2CADR
050592,000112:                                                                                                  #            THEN   DELTD = TS+DELTA T -X +Y +Z +1.05MS* +COUNTERS*
050593,000113:                                                                                                  #     *-THE TIME TAKEN BY WAITLIST ITSELF AND THE COUNTER TICKING DURING THIS WAITLIST TIME.
050594,000114: 
050595,000115: 
050596,000116:                                                                                                  #          IN SHORT, THE ACTUAL TIME TO RETURN CONTROL TO A 2CADR IS AUGMENTED BY THE TIME TO SET UP THE TASK:S
050597,000117:                                                                                                  #     INTERRUPT, ALL COUNTERS TICKING, THE T3RUPT PROCESSING TIME, THE WAITLIST PROCESSING TIME AND THE POSSIBILITY
050598,000118:                                                                                                  #     OF OTHER TASKS INHIBITING THE INTERRUPT.
