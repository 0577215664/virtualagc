Source Code

These source-code files were derived from scans of the Colossus 237 program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

041005,000018:                                                                                                  #  FUNCTIONAL DESCRIPTION....
041006,000019: 
041007,000020:                                                                                                  #      *AN ADAPTATION OF THE LEM P-AXIS CONTROLLER
041008,000021:                                                                                                  #      *MAINTAIN OGA WITHIN 5 DEG DEADBND OF OGAD, WHERE OGAD = OGA AS SEEN
041009,000022:                                                                                                  #       BY IGNOVER  (IGNITION)
041010,000023:                                                                                                  #      *MAINTAIN OGA RATE LESS THAN 0.1 DEG/SEC LIMIT CYCLE RATE
041011,000024:                                                                                                  #      *SWITCHING LOGIC IN PHASE PLANE.... SEE GSOP CHAPTER 3
041012,000025:                                                                                                  #      *USES T6 CLOCK TO TIME JET FIRINGS.
041013,000026:                                                                                                  #      *MAXIMUM JET FIRING TIME = 2.56 SECONDS, LIMITED TO 2.5 IF GREATER
041014,000027:                                                                                                  #      *MINIMUM JET FIRING TIME = 15 MS
041015,000028:                                                                                                  #      *JET PAIRS FIRE ALTERNATELY
041016,000029:                                                                                                  #      *AT LEAST 1/2 SECOND DELAY BEFORE A NEW JET PAIR IS FIRED
041017,000030:                                                                                                  #      *JET FIRINGS MAY NOT BE EXTENDED, ONLY SHORTENED, WHEN RE-EVALUATION
041018,000031:                                                                                                  #       OF A JET FIRING TIME IS MADE ON A LATER PASS
041019,000032: 
041020,000033:                                                                                                  #  CALLING SEQUENCE....
041021,000034: 
041022,000035:                                                                                                  #      *ROLLDAP CALL VIA WAITLIST, IN PARTICULAR BY TVCEXEC (EVERY 1/2 SEC)
041023,000036:                                                                                                  #       WITH A 3CS DELAY TO ALLOW FREE TIME FOR OTHER RUPTS (DWNRPT, ETC.)
041024,000037: 
041025,000038:                                                                                                  #  NORMAL EXIT MODES.... ENDOFJOG
041026,000039:

041101,000114:                                                                                                  #  GIVEN THE OPERATING POINT NOT IN THE COAST REGION, THE DESIRED OGARATE
041102,000115:                                                                                                  #       IS AT THE POINT OF PENETRATION OF THE COAST REGION BY THE CONTROL
041103,000116:                                                                                                  #       PARABOLA WHICH PASSES THROUGH THE OPERATING POINT. FOR REGION 3
041104,000117:                                                                                                  #       DESIRED OGARATE IS SIMPLY +-MAXLIM. FOR REGIONS 1 OR 6 THE SOLUTION
041105,000118:                                                                                                  #       TO A QUADRATIC IS REQUIRED (THE PENETRATION IS ALONG THE STRAIGHT
041106,000119:                                                                                                  #       LINE OR MINLIM BOUNDRY SWITCH LINES). AN APPROXIMATION IS MADE
041107,000120:                                                                                                  #       INSTEAD. CONSIDER AN OPERATING POINT IN REGION 6. PASS A TANGENT TO
041108,000121:                                                                                                  #       THE CONTROL PARABOLA THROUGH THE OPERATING POINT, AND FIND ITS
041109,000122:                                                                                                  #       INTERSECTION WITH THE STRAIGHT LINE SECTION OF THE SWITCH CURVE...
041110,000123:                                                                                                  #       THE INTERSECTION DEFINES DESIRED OGARATE. IF THE OPERATING POINT IS
041111,000124:                                                                                                  #       CLOSE TO THE SWITCH LINE, THE APPROXIMATION IS QUITE GOOD (INDEED
041112,000125:                                                                                                  #       THE APPROXIMATE AND QUADRATIC SOLUTIONS CONVERGE IN THE LIMIT AS
041113,000126:                                                                                                  #       THE SWITCH LINE IS APPROACHED). IF THE OPERATING POINT IS NOT CLOSE
041114,000127:                                                                                                  #       TO THE SWITCH LINE, THE APPROXIMATE SOLUTION GIVES VALID TREND
041115,000128:                                                                                                  #       INFORMATION (DIRECTION OF DESIRED OGARATE) AT LEAST. THE 
041116,000129:                                                                                                  #       RE-EVALUATION OF DESIRED OGARATE IN SUBSEQUENT ROLL DAP PASSES (1/2
041117,000130:                                                                                                  #       SECOND INTERVALS) WILL BENEFIT FROM THE CONVERGENT NATURE OF THE
041118,000131:                                                                                                  #       APPROXIMATION.
041119,000132: 
041120,000133:                                                                                                  #  FOR LARGE OGAERROR THE TANGENT INTERSECTS +-MINLIM SWITCH BOUNDRY BEFORE
041121,000134:                                                                                                  #       INTERSECTING THE STRAIGHT LINE SWITCH. HOWEVER THE MINLIM IS
041122,000135:                                                                                                  #       IGNORED IN COMPUTING THE FIRING TIME, SO THAT THE EXTENSION (INTO
041123,000136:                                                                                                  #       THE COAST REGION) OF THE STRAIGHT LINE SWITCH IS WHAT IS FIRED TO.
041124,000137:                                                                                                  #       IF THE ROLL DAP FINDS ITSELF IN THE COAST REGION BEFORE REACHING
041125,000138:                                                                                                  #       THE DESIRED INTERSECTION (IE, IN THE REGION BETWEEN THE MINLIM
041126,000139:                                                                                                  #       AND THE STRAIGHT LINE SWITCH) IT WILL EXIBIT NORMAL COAST-REGION
041127,000140:                                                                                                  #       BEHAVIOR AND TURN OFF THE JETS. THE PURPOSE OF THIS FIRING POLICY
041128,000141:                                                                                                  #       IS TO MAINTAIN STATIC ROLL STABILITY IN THE EVENT OF A JET
041129,000142:                                                                                                  #       FAILED-ON.
041130,000143: 
041131,000144:                                                                                                  #  WHEN THE OPERATING POINT IS IN REGION 1 THE SAME APPROXIMATION IS
041132,000145:                                                                                                  #       MADE, BUT AT AN ARTIFICIALLY-CREATED OR DUMMY OPERATING POINT,
041133,000146:                                                                                                  #       DEFINED BY.. OGAERROR = INTERSECTION OF CONTROL PARABOLA AND
041134,000147:                                                                                                  #       OGAERROR AXIS, OGARATE = +-LMCRATE WHERE SIGN IS OPPOSITE THAT OF
041135,000148:                                                                                                  #       REAL OPERATING POINT RATE.  WHEN THE OPERATING POINT HAS PASSED
041136,000149:                                                                                                  #       FROM REGION 1 TO REGION 6, THE DUMMY POINT IS NO LONGER REQUIRED,
041137,000150:                                                                                                  #       AND THE SOLUTION REVERTS TO THAT OF A REGULAR REGION 6 POINT.
041138,000151: 
041139,000152: 
041140,000153:                                                                                                  #  EQUATION FOR SWITCHING PARABOLA (SEE FIGURE ABOVE)....
041141,000154:                                                                                                  #                                  2
041142,000155:                                                                                                  #      SOGAERROR = (DB - (SOGARATE) (1/CONACC)/2) SGN(SOGARATE)
041143,000156: 
041144,000157: 
041145,000158:                                                                                                  #  EQUATION FOR SWITCHING STRAIGHT LINE SEGMENT....
041146,000159: 
041147,000160:                                                                                                  #       SOGARATE = -(-SLOPE)(SOGAERROR) - SGN(SOGARATE) INTERCEP
041148,000161: 
041149,000162:                                                                                                  #                WHERE  INTERCEP = DB(-SLOPE) - LMCRATE
041150,000163:

041152,000165:                                                                                                  #  EQUATION FOR INTERSECTION, CONTROL PARABOLA, AND STRAIGHT SWITCH LINE....
041153,000166: 
041154,000167:                                                                                                  #       DOGADOT = NUM/DEN, WHERE
041155,000168:                                                                                                  #                                  2
041156,000169:                                                                                                  #           NUM = (-SLOPE)(OGARATE) (1/CONACC)
041157,000170:                                                                                                  #                   +SGN(DELOGA)(-SLOPE)(OGAERROR - SGN(DELOGA)(DB))
041158,000171:                                                                                                  #                   +LMCRATE
041159,000172: 
041160,000173:                                                                                                  #           DEN = (-SLOPE)(OGARATE)(1/CONACC) - SGN(DELOGA)
041161,000174:                                                                                                  #                                             2
041162,000175:                                                                                                  #           DELOGA = OGAERROR - (DB - (OGADOT) (1/CONACC)/2)SGN(OGADOT)
041163,000176: 
041164,000177: 
041165,000178:                                                                                                  #  FOR REGIONS 6 AND 6-PRIME  USE ACTUAL OPEATING POINT  (OGA, OGARATE)
041166,000179:                                                                                                  #       FOR OGAERROR AND OGARATE IN THE INTERSECTION EQUATIONS ABOVE.
041167,000180:                                                                                                  #       FOR REGIONS 1 AND 1-PRIME  USE DUMMY OPERATING POINT FOR OGAERROR
041168,000181:                                                                                                  #       AND OGARATE,  WHERE THE DUMMY POINT IS GIVEN BY....
041169,000182:                                                                                                  #       OGAERROR= DELOGA + DB SGN(OGARATE)
041170,000183:                                                                                                  #       OGARATE= -LMCRATE SGN(OGARATE)
041171,000184: 
041172,000185:                                                                                                  #  NOTE, OGAERROR = OGA - OGAD  USES DUMMY REGISTER  OGA IN ROLL DAP CODING
041173,000186:                                                                                                  #       ALSO,AT POINT WHERE DOGADOT IS COMPUTED, REGISTER DELOGA IS USED
041174,000187:                                                                                                  #       AS A DUMMY REGISTER FOR THE OGAERROR IN THE NUM EQUATION ABOVE
