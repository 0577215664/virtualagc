Source Code

These source-code files were derived from scans of the Colossus 237 program listing from the private collection of original AGC developer Fred Martin. Scanning was by Ron Burkey. Pre-existing Colossus 249 (Apollo 9) source files were used as a template, and changes between Colossus 249 and Colossus 237 were manually transcribed by a team of volunteers. Note that the page images presented online are of reduced quality, and that additional images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP: ASSEMBLE REVISION 237 OF AGC PROGRAM COLOSSUS 
	BY NASA 2021111-031  1:17 AUG. 24, 1968
Note that the date is the date of the printout, not the date of the program revision.

041004,000019: 
041005,000020:                                                                                                  #      *AN ADAPTATION OF THE LEM P-AXIS CONTROLLER

041007,000022:                                                                                                  #       BY IGNOVER  (IGNITION)

041014,000029:                                                                                                  #      *AT LEAST 1/2 SECOND DELAY BEFORE A NEW JET PAIR IS FIRED
041015,000030:                                                                                                  #      *JET FIRINGS MAY NOT BE EXTENDED, ONLY SHORTENED, WHEN RE-EVALUATION
041016,000031:                                                                                                  #       OF A JET FIRING TIME IS MADE ON A LATER PASS

041019,000034: 
041020,000035:                                                                                                  #      *ROLLDAP CALL VIA WAITLIST, IN PARTICULAR BY TVCEXEC (EVERY 1/2 SEC)
041021,000036:                                                                                                  #       WITH A 3CS DELAY TO ALLOW FREE TIME FOR OTHER RUPTS (DWNRPT, ETC.)
041022,000037:

041024,000039:

041099,000114:                                                                                                  #  GIVEN THE OPERATING POINT NOT IN THE COAST REGION, THE DESIRED OGARATE
041100,000115:                                                                                                  #       IS AT THE POINT OF PENETRATION OF THE COAST REGION BY THE CONTROL
041101,000116:                                                                                                  #       PARABOLA WHICH PASSES THROUGH THE OPERATING POINT. FOR REGION 3
041102,000117:                                                                                                  #       DESIRED OGARATE IS SIMPLY +-MAXLIM. FOR REGIONS 1 OR 6 THE SOLUTION
041103,000118:                                                                                                  #       TO A QUADRATIC IS REQUIRED (THE PENETRATION IS ALONG THE STRAIGHT
041104,000119:                                                                                                  #       LINE OR MINLIM BOUNDRY SWITCH LINES). AN APPROXIMATION IS MADE
041105,000120:                                                                                                  #       INSTEAD. CONSIDER AN OPERATING POINT IN REGION 6. PASS A TANGENT TO
041106,000121:                                                                                                  #       THE CONTROL PARABOLA THROUGH THE OPERATING POINT, AND FIND ITS
041107,000122:                                                                                                  #       INTERSECTION WITH THE STRAIGHT LINE SECTION OF THE SWITCH CURVE...
041108,000123:                                                                                                  #       THE INTERSECTION DEFINES DESIRED OGARATE. IF THE OPERATING POINT IS
041109,000124:                                                                                                  #       CLOSE TO THE SWITCH LINE, THE APPROXIMATION IS QUITE GOOD (INDEED
041110,000125:                                                                                                  #       THE APPROXIMATE AND QUADRATIC SOLUTIONS CONVERGE IN THE LIMIT AS
041111,000126:                                                                                                  #       THE SWITCH LINE IS APPROACHED). IF THE OPERATING POINT IS NOT CLOSE
041112,000127:                                                                                                  #       TO THE SWITCH LINE, THE APPROXIMATE SOLUTION GIVES VALID TREND
041113,000128:                                                                                                  #       INFORMATION (DIRECTION OF DESIRED OGARATE) AT LEAST. THE 
041114,000129:                                                                                                  #       RE-EVALUATION OF DESIRED OGARATE IN SUBSEQUENT ROLL DAP PASSES (1/2
041115,000130:                                                                                                  #       SECOND INTERVALS) WILL BENEFIT FROM THE CONVERGENT NATURE OF THE
041116,000131:                                                                                                  #       APPROXIMATION.
041117,000132: 
041118,000133:                                                                                                  #  FOR LARGE OGAERROR THE TANGENT INTERSECTS +-MINLIM SWITCH BOUNDRY BEFORE
041119,000134:                                                                                                  #       INTERSECTING THE STRAIGHT LINE SWITCH. HOWEVER THE MINLIM IS
041120,000135:                                                                                                  #       IGNORED IN COMPUTING THE FIRING TIME, SO THAT THE EXTENSION (INTO
041121,000136:                                                                                                  #       THE COAST REGION) OF THE STRAIGHT LINE SWITCH IS WHAT IS FIRED TO.
041122,000137:                                                                                                  #       IF THE ROLL DAP FINDS ITSELF IN THE COAST REGION BEFORE REACHING
041123,000138:                                                                                                  #       THE DESIRED INTERSECTION (IE, IN THE REGION BETWEEN THE MINLIM
041124,000139:                                                                                                  #       AND THE STRAIGHT LINE SWITCH) IT WILL EXIBIT NORMAL COAST-REGION
041125,000140:                                                                                                  #       BEHAVIOR AND TURN OFF THE JETS. THE PURPOSE OF THIS FIRING POLICY
041126,000141:                                                                                                  #       IS TO MAINTAIN STATIC ROLL STABILITY IN THE EVENT OF A JET
041127,000142:                                                                                                  #       FAILED-ON.
041128,000143: 
041129,000144:                                                                                                  #  WHEN THE OPERATING POINT IS IN REGION 1 THE SAME APPROXIMATION IS
041130,000145:                                                                                                  #       MADE, BUT AT AN ARTIFICIALLY-CREATED OR DUMMY OPERATING POINT,
041131,000146:                                                                                                  #       DEFINED BY.. OGAERROR = INTERSECTION OF CONTROL PARABOLA AND
041132,000147:                                                                                                  #       OGAERROR AXIS, OGARATE = +-LMCRATE WHERE SIGN IS OPPOSITE THAT OF
041133,000148:                                                                                                  #       REAL OPERATING POINT RATE.  WHEN THE OPERATING POINT HAS PASSED
041134,000149:                                                                                                  #       FROM REGION 1 TO REGION 6, THE DUMMY POINT IS NO LONGER REQUIRED,
041135,000150:                                                                                                  #       AND THE SOLUTION REVERTS TO THAT OF A REGULAR REGION 6 POINT.
041136,000151: 
041137,000152: 
041138,000153:                                                                                                  #  EQUATION FOR SWITCHING PARABOLA (SEE FIGURE ABOVE)....
041139,000154:                                                                                                  #                                  2
041140,000155:                                                                                                  #      SOGAERROR = (DB - (SOGARATE) (1/CONACC)/2) SGN(SOGARATE)
041141,000156: 
041142,000157: 
041143,000158:                                                                                                  #  EQUATION FOR SWITCHING STRAIGHT LINE SEGMENT....
041144,000159: 
041145,000160:                                                                                                  #       SOGARATE = -(-SLOPE)(SOGAERROR) - SGN(SOGARATE) INTERCEP
041146,000161: 
041147,000162:                                                                                                  #                WHERE  INTERCEP = DB(-SLOPE) - LMCRATE
041148,000163:

041150,000165:                                                                                                  #  EQUATION FOR INTERSECTION, CONTROL PARABOLA, AND STRAIGHT SWITCH LINE....
041151,000166: 
041152,000167:                                                                                                  #       DOGADOT = NUM/DEN, WHERE
041153,000168:                                                                                                  #                                  2
041154,000169:                                                                                                  #           NUM = (-SLOPE)(OGARATE) (1/CONACC)
041155,000170:                                                                                                  #                   +SGN(DELOGA)(-SLOPE)(OGAERROR - SGN(DELOGA)(DB))
041156,000171:                                                                                                  #                   +LMCRATE
041157,000172: 
041158,000173:                                                                                                  #           DEN = (-SLOPE)(OGARATE)(1/CONACC) - SGN(DELOGA)
041159,000174:                                                                                                  #                                             2
