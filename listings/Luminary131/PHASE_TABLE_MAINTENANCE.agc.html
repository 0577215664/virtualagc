Source Code

This is the source code for the Apollo 13 Lunar Module guidance computer. These files were derived originally from the scan from MIT's Dibner Institute's now-discontinued website titled "History of Recent Science and Technology". It is believed that the original listing was preserved by AGC developer Don Eyles, copied/reduced by AGC developer Hugh Blair-Smith, passed by AGC developer James Kernan to collector David Craig, scanned by Gary Neff, and then passed to the Dibner Institute where unfortunately the high-quality scans were reduced in legibility as part of the online-presentation process, and the original scans discarded. The source code was transcribed from those images by Ron Burkey. The images from the Dibner Institute website, and unfortunately not the discarded original high-quality scans, though with a few high-quality replacement scans provided directly by Gary Neff, are now available at the Virtual AGC project website. Report any conversion errors to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 131 OF AGC PROGRAM LUMINARY BY NASA 2021112-091
	17:53 DEC. 19, 1969
Note that the date is the date of the printout, not the date of the program revision.

053884,000073:                                                                                                  #  PHASCHNG IS THE MAIN WAY OF MAKING PHASE CHANGES FOR RESTARTS.  THERE ARE THREE FORMS OF PHASCHNG, KNOWN AS TYPE
053885,000074:                                                                                                  #  A, TYPE B, AND TYPE C.  THEY ARE ALL CALLED AS FOLLOWS, WHERE OCT XXXXX CONTAINS THE PHASE INFORMATION,
053886,000075:                                                                                                  #                TC      PHASCHNG
053887,000076:                                                                                                  #                OCT     XXXXX
053888,000077:                                                                                                  #  TYPE A IS CONCERNED WITH FIXED PHAS CHANGES, THAT IS, PHASE INFORMATION THAT IS STORED PERMANENTLY.  THESE
053889,000078:                                                                                                  #  OPTIONS ARE, WEHRE G STANDS FOR A GROUP AND .X FOR THE PHASE,
053890,000079:                                                                                                  #        G.0             INACTIVE, WILL NOT PERMIT A GROUP G RESTART
053891,000080:                                                                                                  #        G.1             WILL CAUSE THE LAST DISPLAY TO BE REACTIVATED, USED MAINLY IN MANNED FLIGHTS
053892,000081:                                                                                                  #        G.EVEN          A DOUBLE TABLE RESTART, CAN CAUSE ANY COMBINATION OF TWO JOBS, TASKS, AND/OR
053893,000082:                                                                                                  #                        LONGCALL TO BE RESTARTED.
053894,000083:                                                                                                  #        G.ODD NOT .1    A SINGLE TABLE RESTART, CAN CAUSE EITHER A JOB, TASK, OR LONGCALL RESTART.
053895,000084: 
053896,000085:                                                                                                  #  THIS INFORMATION IS PUT INTO THE OCTAL WORD AFTER TC PHASCHNG AS FOLLOWS
053897,000086:                                                                                                  #        TL0 00P PPP PPP GGG
053898,000087:                                                                                                  #  WHERE EACH LETTER OR NUMBER STANTS FOR A BIT.  THE G'S STAND FOR THE GROUP, OCTAL 1-7, THE P'S FOR THE PHASE,
053899,000088:                                                                                                  #  OCTAL 0 - 127.  0'S MUST BE 0.  IF ONE WISHES TO HAVE THE TBASE OF GROUP G TO BE SET AT THIS TIME,
053900,000089:                                                                                                  #  T IS SET TO 1, OTHERWISE IT IS SET TO 0.  SIMILARLY IF ONE WISHES TO SET LONGBASE, THEN L IS SET TO 1, OTHERWISE
053901,000090:                                                                                                  #  IT IS SET TO 0.  SOME EXAMLES,
053902,000091:                                                                                                  #                TC      PHASCHNG        # THIS WILL CAUSE GROUP 3 TO BE SET TO 0,
053903,000092:                                                                                                  #                OCT     00003           # MAKING GROUP 3 INACTIVE
053904,000093: 
053905,000094:                                                                                                  #                TC      PHASCHNG        # IF A RESTART OCCURS THIS WOULD CAUSE
053906,000095:                                                                                                  #                OCT     00012           # GROUP 2 TO RESTART THE LAST DISPLAY
053907,000096: 
053908,000097:                                                                                                  #                TC      PHASCHNG        # THIS SETS THE TBASE OF GROUP 4 AND IN
053909,000098:                                                                                                  #                OCT     40064           # CASE OF A RESTART WOULD START UP THE TWO
053910,000099:                                                                                                  #                                        # THINGS LOCATED IN THE DOUBLE 4.6 RESTART
053911,000100:                                                                                                  #                                        # LOCATION.
053912,000101: 
053913,000102:                                                                                                  #                TC      PHASCHNG        # THIS SETS LONGBASE AND UPON A RESTART
053914,000103:                                                                                                  #                OCT     20135           # CAUSES 5.13 TO BE RESTARTED (SINCE
053915,000104:                                                                                                  #                                        # LONGBASE WAS SET THIS SINGLE ENTRY
053916,000105:                                                                                                  #                                        # SHOULD BE A LONGCALL)
053917,000106: 
053918,000107:                                                                                                  #                TC      PHASCHNG        # SINCE BOTH TBASE4 AND LONGBASE ARE SET,
053919,000108:                                                                                                  #                OCT     60124           # 4.12 SHOULD CONTAIN BOTH A TASK AND A
053920,000109:                                                                                                  #                                        # LONGCALL TO BE RESTARTED
053921,000110: 
053922,000111:                                                                                                  #  TYPE C PHASCHNG CONTAINS THE VARIABLE TYPE OF PHASCHNG INFORMATION.  INSTEAD OF THE INFORMATION BEING IN A
053923,000112:                                                                                                  #  PERMANENT FORM, ONE STORES THE DESIRED RESTART INFORMATION IN A VARIABLE LOCATION.  THE BITS ARE AS FOLLOWS,
053924,000113:                                                                                                  #        TL0 1AD XXX CJW GGG
053925,000114:                                                                                                  #  WHERE EACH LETTER OR NUMBER STANDS FOR A BIT.  THE G'S STAND FOR THE GROUP, OCTAL 1 - 7.  IF THE RESTART IS TO
053926,000115:                                                                                                  #  BE BY WAITLIST, W IS SET TO 1, IF IT IS A JOB, J IS SET TO 1, IF IT IS A LONGCALL, C IS SET TO 1.  ONLY ONE OF
053927,000116:                                                                                                  #  THESE THREE BITS MAY BE SET.  X'S ARE IGNORED, 1 MUST BE 1, AND 0 MUST BE 0.  AGAIN T STANDS FOR THE TBASE,

053957,000146:                                                                                                  #  WHERE EACH LETTER OR NUMBER STANDS FOR A BIT.  THE G'S STAND FOR THE GROUP, OCTAL 1 - 7, THE P'S FOR THE FIXED
053958,000147:                                                                                                  #  PHASE INFORMATION, OCTAL 0 - 127.  1 MUST BE 1.  AND AGAIN T STANDS FOR THE TBASE AND L FOR LONGBASE.  D THIS
053959,000148:                                                                                                  #  TIME STANDS ONLY FOR PRIORITY SINCE THIS WILL BE CONSIDERED A JOB, AND IT MUST BE GIVEN DIRECTLY IF GIVEN.
053960,000149:                                                                                                  #  AGAIN A STANDS FOR THE ADDRESS OF THE LOCATION TO BE RESTARTED, 1 IF THE 2CADR IS GIVEN, OR 0 IF IT IS TO BE
053961,000150:                                                                                                  #  THE NEXT LOCATION.  (THE RETURN LOCATION OF PHASCHNG) EXAMPLES,
