Source Code

These source-code files were derived originally from the scan from MIT's Dibner Institute's now-discontinued website titled "History of Recent Science and Technology". It is believed that the original listing was preserved by AGC developer Don Eyles, copied/reduced by AGC developer Hugh Blair-Smith, passed by AGC developer James Kernan to collector David Craig, scanned by Gary Neff, and then passed to the Dibner Institute where unfortunately the high-quality scans were reduced in legibility as part of the online-presentation process, and the original scans discarded. The source code was transcribed from those images by Ron Burkey. The images from the Dibner Institute website, and unfortunately not the discarded original high-quality scans, though with a few high-quality replacement scans provided directly by Gary Neff, are now available at the Virtual AGC project website. Report any conversion errors to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 131 OF AGC PROGRAM LUMINARY BY NASA 2021112-091
	17:53 DEC. 19, 1969
Note that the date is the date of the printout, not the date of the program revision.

038358,000019:                                                                                                  #  MOD NUMBER:  1        68-07-15
038359,000020:                                                                                                  #  MOD AUTHOR:  KLUMPP
038360,000021: 
038361,000022:                                                                                                  #  OBJECTS OF MOD:       1.      TO SUPPLY COMMANDED GIMBAL ANGLES FOR NOUN 22.
038362,000023:                                                                                                  #                        2.      TO MAINTAIN CORRECT AND CURRENT THRUST
038363,000024:                                                                                                  #                                DIRECTION DATA IN ALL MODES.  THIS IS DONE BY
038364,000025:                                                                                                  #                                FETCHING FOR THE THRUST DIRECTION FILTER THE
038365,000026:                                                                                                  #                                CDUD'S IN PNGCS-AUTO, THE CDU'S IN ALL OTHER
038366,000027:                                                                                                  #                                MODES.
038367,000028:                                                                                                  #                        3.      TO SUBSTITUDE A STOPRATE FOR THE NORMAL
038368,000029:                                                                                                  #                                AUTOPILOT COMMANDS WHENEVER
038369,000030:                                                                                                  #                                1) NOT IN PNGCS-AUTO, OR
038370,000031:                                                                                                  #                                2) ENGINE IS OFF.
038371,000032: 
038372,000033:                                                                                                  #  FUNCTIONAL DESCRIPTION:
038373,000034: 
038374,000035:                                                                                                  #  FINDCDUW PROVIDES THE INTERFACES BETWEEN THE VARIOUS POWERED FLITE GUIDANCE PROGRAMS
038375,000036:                                                                                                  #  AND THE DIGITAL AUTOPILOT.  THE INPUTS TO FINDCDUW ARE THE THRUST COMMAND VECTOR
038376,000037:                                                                                                  #  AND THE WINDOW COMMAND VECTOR, AND THE OUTPUTS ARE THE GIMBAL ANGLE
038377,000038:                                                                                                  #  INCRMENTS, THE COMMANDED ATTITUDE ANGLE RATES, AND THE COMMANDED
038378,000039:                                                                                                  #  ATTITUDE LAG ANGLES (WHICH ACCOUNT FOR TH ANGLES BY WHICH THE BODY WILL
038379,000040:                                                                                                  #  LAG BEHIND A RAMP COMMAND IN ATTITUDE ANGLE DUE TO THE FINITE ANGULAR
038380,000041:                                                                                                  #  ACCELERATIONS AVAILABLE).
038381,000042: 
038382,000043:                                                                                                  #  FINDCDUW ALIGNS THE ESTIMATED THRUST VECTOR FROM THE THRUST DIRECTION
038383,000044:                                                                                                  #  FILTER WITH THE THRUST COMMAND VECTOR, AND, WHEN XDVINHIB SET,
038384,000045:                                                                                                  #  ALIGNS THE +Z HALF OF THE LM ZX PLANE WITH THE WINDOW COMMAND VECTOR.
038385,000046: 
038386,000047:                                                                                                  #  Page 908
038387,000048:                                                                                                  #  SPECIFICATIONS:
038388,000049: 
038389,000050:                                                                                                  #  INITIALIZATION:       A SINGLE INTERPRETIVE CALL TO INITCDUW IS REQUIRED
038390,000051:                                                                                                  #                        BEFORE EACH GUIDED MANEUVER USING FINDCDUW.
038391,000052: 
038392,000053:                                                                                                  #  CALL:                 INTERPRETIVE CALL TO FINDCDUW WITH THE THRUST COMMAND
038393,000054:                                                                                                  #                        VECTOR IN MPAC.  INTERPRETIVE CALL TO FINDCDUW -2 WITH
038394,000055:                                                                                                  #                        THE THRUST COMMAND VECTOR IN UNFC/2 AND NOT IN MPAC.
038395,000056: 
038396,000057:                                                                                                  #  RETURNS:              NORMAL INTERPRETIVE IN ALL CASES
038397,000058:                                                                                                  #                        1.      NORMALLY ALL AUTOPILOT CMDS ARE ISSUED.
038398,000059:                                                                                                  #                        2.      IF NOT PNGCS AUTO, DO STOPRATE AND RETURN
038399,000060:                                                                                                  #                                WITHOUT ISSUING AUTOPILOT CMDS.
038400,000061:                                                                                                  #                        3.      IF ENGINE OFF, DO STOPRATE AND RETURN WITHOUT
038401,000062:                                                                                                  #                                ISSUING AUTOPILOT CMDS.
038402,000063: 
038403,000064:                                                                                                  #  ALARMS:               00401   IF INPUTS DETERMINE AN ATTITUDE IN GIMBAL LOCK.
038404,000065:                                                                                                  #                                FINDCDUW DRIVES CDUXD AND CDUYD TO THE RQD VALUES,
038405,000066:                                                                                                  #                                BUT DRIVES CDUZD ONLY TO THE GIMBAL LOCK CONE.
038406,000067:

038408,000069:                                                                                                  #                                UNITIZED USING NORMUNIT.  FINDCDUW ISSUES
038409,000070:                                                                                                  #                                STOPRATE AS ONLY INPUT TO AUTOPILOT.
