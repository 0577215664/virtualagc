Source Code

This is the source code for the Apollo 13 Lunar Module guidance computer. These files were derived originally from the scan from MIT's Dibner Institute's now-discontinued website titled "History of Recent Science and Technology". It is believed that the original listing was preserved by AGC developer Don Eyles, copied/reduced by AGC developer Hugh Blair-Smith, passed by AGC developer James Kernan to collector David Craig, scanned by Gary Neff, and then passed to the Dibner Institute where unfortunately the high-quality scans were reduced in legibility as part of the online-presentation process, and the original scans discarded. The source code was transcribed from those images by Ron Burkey. The images from the Dibner Institute website, and unfortunately not the discarded original high-quality scans, though with a few high-quality replacement scans provided directly by Gary Neff, are now available at the Virtual AGC project website. Report any conversion errors to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 131 OF AGC PROGRAM LUMINARY BY NASA 2021112-091
	17:53 DEC. 19, 1969
Note that the date is the date of the printout, not the date of the program revision.

049886,000023:                                                                                                  #  ----------------
049887,000024: 
049888,000025:                                                                                                  #  FROM A USER'S POINT OF VIEW, ORBITAL INTEGRATION IS ESSENTIALLY THE SAME AS THE 278 INTEGRATION
049889,000026:                                                                                                  #  PROGRAM.  THE SAME ENTRANCES TO THE PROGRAM WILL BE MAINTAINED, THE SAME STALLING ROUTINE WILL BE USED AND
049890,000027:                                                                                                  #  OUTPUT WILL STILL BE VIA THE PUSHLIST.  THE PRIMARY DIFFERENCES TO A USER INVOLVE THE ADDED CAPABILITY OF
049891,000028:                                                                                                  #  TERMINATING INTEGRATION AT A SPECIFIC FINAL RADIUS AND THE DIFFERENCE IN STATE VECTOR SCALING INSIDE AND OUTSIDE
049892,000029:                                                                                                  #  THE LUNAR SPHERE OF INFLUENCE.
049893,000030: 
049894,000031:                                                                                                  #  IN ORDER TO MAKE THE CSM(LEM)PREC AND CSM(LEM)CONIC ENTRANCES SIMILAR TO FLIGHT 278, THE INTEGRATION PROGRAM
049895,000032:                                                                                                  #  WILL ITSELF SET THE FINAL RADIUS (RFINAL) TO 0 SO THAT REACHING THE DESIRED TIME ONLY WILL TERMINATE
049896,000033:                                                                                                  #  INTEGRATION.  THE DP REGISTER RFINAL MUST BE SET BY USERS OF INTEGRVS AND INTEGRV, AND MUST BE DONE AFTER THE
049897,000034:                                                                                                  #  CALL TC INTSTALL.
049898,000035: 
049899,000036:                                                                                                  #  WHEN THE LM IS ON THE LUNAR SURFACE (INDICATED BY LUNAR SURFACE FLAG SET) CALLS TO LEMCONIC, LEMPREC, AND
049900,000037:                                                                                                  #  INTEGRV WITH VINFLAG = 0 WILL RESULT IN THE USE OF THE PLANETARY INERTIAL ORIENTATION SUBROUTINES TO PROVIDE
049901,000038:                                                                                                  #  BOTH THE LM'S POSITION AND VELOCITY IN THE REFERENCE COORDINATE SYSTEM.
049902,000039:                                                                                                  #  THE PROGRAM WILL PROVIDE OUTPUT AS IF INTEGRATION WAS USED.  THAT IS, THE PUSHLIST WILL BE SET AS NOTED BELOW AND
049903,000040:                                                                                                  #  THE PERMANENT STATE VECTOR UPDATED WHEN SPECIFIED BY AN INTEGRV CALL.
049904,000041: 
049905,000042:                                                                                                  #  USERS OF INTEGRVS DESIRING INTEGRATION (INTYPFLG = 0) SHOULD NOTE THAT THE OBLATENESS PERTURBATION COMPUTATION
049906,000043:                                                                                                  #  IN LUNAR ORBIT IS TIME DEPENDENT.  THEREFORE, THE USER SHOULD SUPPLY AN INITIAL STATE VECTOR VALID AT SOME REAL
049907,000044:                                                                                                  #  TIME AND THE DESIRED TIME (TDEC1) ALSO AT SOME REAL TIME.  FOR CONIC "INTEGRATION" THE USER MAY STILL USE ZERO
049908,000045:                                                                                                  #  AS THE INITIAL TIME AND DELTA TIME AS THE DESIRED TIME.

049910,000047:                                                                                                  #  2.0 CENTRAL DESCRIPTION
049911,000048:                                                                                                  #  -----------------------
049912,000049: 
049913,000050:                                                                                                  #  THE INTEGRATION PROGRAM OPERATES AS A CLOSED INTERPRETIVE SUBROUTINE AND PERFORMS THESE FUNCTIONS --
049914,000051:                                                                                                  #        1) INTEGRATES (PRECISION OR CONIC) EITHER CSM OR LM STATE VECTOR
049915,000052:                                                                                                  #        2) INTEGRATES THE W-MATRIX
049916,000053:                                                                                                  #        3) PERMANENT OR TEMPORARY UPDATE OF THE STATE VECTOR
049917,000054: 
049918,000055:                                                                                                  #  THERE ARE SIX ENTRANCES TO THE INTEGRATION PROGRAM.  FOUR OF THESE (CSMPREC, LEMPREC, CSMCONIC, LEMCONIC) SET
049919,000056:                                                                                                  #  ALL THE FLAGS REQUIRED IN THE INTEGRATION PROGRAM ITSELF TO CAUSE THE PRECISION OR CONIC INTEGRATION (KEPLER) OF
049920,000057:                                                                                                  #  THE LM OR CSM STATE VECTOR, AS THE NAMES SUGGEST.  ONE ENTRANCE (INTEGRVS) PERMITS THE CALLING PROGRAM TO
049921,000058:                                                                                                  #  PROVIDE A STATE VECTOR TO BE INTEGRATED.  THE CALLING PROGRAM MUST SET THE FLAGS INDICATING (1) PRECISION OR
049922,000059:                                                                                                  #  CONIC INTEGRATION, (2) IN OR OUT OF LUNAR SPHERE, (3) MIDCOURSE OR NOT, AND THE INTEGRATION PROGRAM COMPLETES
049923,000060:                                                                                                  #  THE FLAG SETTING TO BYPASS W-MATRIX INTEGRATION.  THE LAST ENTRANCE (INTEGRV, USED IN GENERAL BY THE
049924,000061:                                                                                                  #  NAVIGATION PROGRAMS) PERMITS THE CALLER TO SET FIVE FLAGS (NOT MOONFLAG OR MIDFLAG) BUT NOT TO INPUT A STATE
049925,000062:                                                                                                  #  VECTOR.  ANY PROGRAM WHICH CALLS INTEGRVS OR INTEGRV MUST CALL INTSTALL BEFORE IT SETS THE INTEGRATION FLAGS
049926,000063:                                                                                                  #  AND/OR STATE VECTOR.
049927,000064: 
049928,000065:                                                                                                  #  THREE SETS OF 42 REGISTERS AND 2 FLAGS ARE USED FOR THE STATE VECTORS.  TWO SETS, WHICH MAY NOT BE OVERLAYED, ARE
049929,000066:                                                                                                  #  USED FOR THE PERMANENT STATE VECTORS FOR THE CSM AND LM.  THE THIRD SET, WHICH MAY BE OVERLAYED WHEN INTEGRATION
049930,000067:                                                                                                  #  IS NOT BEING DONE, IS USED IN THE COMPUTATIONS.
049931,000068:                                                                                                  #  THE PERMANENT STATE VECTORS WILL BE PERIODICALLY UPDATED SO THAT THE VECTORS WILL NOT BE ODER THAN 4 TIMESTEPS.
049932,000069:                                                                                                  #  THE PERMANENT STATE VECTORS WILL ALSO BE UPDATED WHENEVER THE W-MATRIX IS INTEGRATED OR WHEN A CALLER OF INTEGRV
049933,000070:                                                                                                  #  SETS STATEFLG (THE NAVIGATION PROGRAMS P20, P22.)
