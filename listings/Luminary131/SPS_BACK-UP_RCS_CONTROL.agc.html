Source Code

These source-code files were derived originally from the scan from MIT's Dibner Institute's now-discontinued website titled "History of Recent Science and Technology". It is believed that the original listing was preserved by AGC developer Don Eyles, copied/reduced by AGC developer Hugh Blair-Smith, passed by AGC developer James Kernan to collector David Craig, scanned by Gary Neff, and then passed to the Dibner Institute where unfortunately the high-quality scans were reduced in legibility as part of the online-presentation process, and the original scans discarded. The source code was transcribed from those images by Ron Burkey. The images from the Dibner Institute website, and unfortunately not the discarded original high-quality scans, though with a few high-quality replacement scans provided directly by Gary Neff, are now available at the Virtual AGC project website. Report any conversion errors to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 131 OF AGC PROGRAM LUMINARY BY NASA 2021112-091
	17:53 DEC. 19, 1969
Note that the date is the date of the printout, not the date of the program revision.

063139,000019:                                                                                                  #  PROGRAM NAME:         SPSRCS
063140,000020:                                                                                                  #  AUTHOR:               EDGAR M. CSHIKA (AC ELECTRONICS)
063141,000021:                                                                                                  #  MODIFIED:             TO RETURN TO ALL AXES VIA Q BY P. S. WEISSMAN, OCT 7, 1968
063142,000022:                                                                                                  #  MODIFIED TO IMPROVE BENDING STABILITY BY G. KALAN, FEB. 14, 1969
063143,000023: 
063144,000024:                                                                                                  #  FUNCTIONAL DESCRIPTION:
063145,000025:                                                                                                  #        THE PROGRAM CONTROLS THE FIRING OF ALL RCS JETS IN THE DOCKED CONFIGURATION ACCORDING TO THE FOLLOWING PHASE
063146,000026:                                                                                                  #        PLANE LOGIC.
063147,000027: 
063148,000028:                                                                                                  #        1. JET SENSE TEST (SPSRCS)
063149,000029:                                                                                                  #                IF JETS ARE FIRING NEGATIVELY, SET OLDSENSE NEGATIVE AND CONTINUE
063150,000030:                                                                                                  #                IF JETS ARE FIRING POSITIVLEY, SET OLDSENCE POSITIVE AND CONTINUE
063151,000031:                                                                                                  #                IF JETS ARE NOT FIRING, SET OLDSENSE TO ZERO AND GO TO OUTER RATE LIMIT TEST
063152,000032: 
063153,000033:                                                                                                  #        2. RATE DEAD BAND TEST
063154,000034:                                                                                                  #                IF JETS ARE FIRING NEGATIVELY AND RATE IS GREATER THAN TARGET RATE, LEAVE
063155,000035:                                                                                                  #                JETS ON AND GO TO INHIBITION LOGIC.  OTHERWISE, CONTINUE.
063156,000036:                                                                                                  #                IF JETS ARE FIRING POSITIVELY AND RATE IS LESS THAN TARGET RATE, LEAVE
063157,000037:                                                                                                  #                JETS ON AND GO TO INHIBITION LOGIC.  OTHERWISE, CONTINUE.
063158,000038: 
063159,000039:                                                                                                  #        3. OUTER RATE LIMIT TEST (SPSSTART)
063160,000040:                                                                                                  #                IF MAGNITUDE OF EDOT IS GREATER THAN 1.73 DEG/SEC SET JET FIRING TIME
063161,000041:                                                                                                  #                TO REDUCE RATE AND GO TO INHIBITION LOGIC.  OTHERWISE, CONTINUE.
063162,000042: 
063163,000043:                                                                                                  #        4. COAST ZONE TEST
063164,000044:                                                                                                  #                IF STATE (E,EDOT) IS BELOW LINE E + 4 X EDOT > -1.4 DEG AND EDOT IS LESS THAN 1.30 DEG/SEC SET JET TIME
063165,000045:                                                                                                  #                POSITIVE AND CONTINUE.  OTHERWISE, SET JET FIRING TIME TO ZERO AND CONTINUE.
063166,000046:                                                                                                  #                IF STATE IS ABOVE LINE E + 4 X EDOT > +1.4 DEG AND EDOT IS GREATER THAN -1.30 DEG/SEC, SET JET TIME NEGATIVE
063167,000047:                                                                                                  #                AND CONTINUE.  OTHERWISE, SET JET FIRING TIME TO ZERO AND CONTINUE.
063168,000048: 
063169,000049:                                                                                                  #        5. INHIBITION LOGIC
063170,000050:                                                                                                  #                IF OLDSENSE IS NON-ZERO:
063171,000051:                                                                                                  #                        A) RETURN IF JET TIME AS THE SAME SIGN AS OLDSENSE
063172,000052:                                                                                                  #                        B) SET INHIBITION COUNTER* AND RETURN IF JET TIME IS ZERO
063173,000053:                                                                                                  #                        C) SET INHIBITION COUNTER,* SET JET TIME TO ZERO AND RETURN IF SIGN
063174,000054:                                                                                                  #                           OF JET TIME IS OPPOSITE TO THAT OF OLDSENSE
063175,000055:                                                                                                  #                IF OLDSENSE IS ZERO:
063176,000056:                                                                                                  #                        A) RETURN IF INHIBITION COUNTER IS NOT POSITIVE
063177,000057:                                                                                                  #                        B) SET JET TIME TO ZERO AND RETURN IF INHIBITION COUNTER IS POSITIVE
063178,000058:                                                                                                  #                *NOTE: INHIBITION COUNTERS CAN BE SET TO 4 OR 10 FOR THE P AND UV AXES,
063179,000059:                                                                                                  #                RESPECTIVELY, IN SPSRCS.  THEY ARE DECREMENTED BY ONE AT THE BEGINNING OF

063184,000064:                                                                                                  #        AND IS NOMINALLY 100 MS FOR ALL AXES IN DRIFTING FLIGHT.  DURING POWERED FLIGHT THE MINIMUM IS 100 MS FOR THE
