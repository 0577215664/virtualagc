Source Code

This is the source code for the Apollo 13 Lunar Module guidance computer. These files were derived originally from the scan from MIT's Dibner Institute's now-discontinued website titled "History of Recent Science and Technology". It is believed that the original listing was preserved by AGC developer Don Eyles, copied/reduced by AGC developer Hugh Blair-Smith, passed by AGC developer James Kernan to collector David Craig, scanned by Gary Neff, and then passed to the Dibner Institute where unfortunately the high-quality scans were reduced in legibility as part of the online-presentation process, and the original scans discarded. The source code was transcribed from those images by Ron Burkey. The images from the Dibner Institute website, and unfortunately not the discarded original high-quality scans, though with a few high-quality replacement scans provided directly by Gary Neff, are now available at the Virtual AGC project website. Report any conversion errors to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 131 OF AGC PROGRAM LUMINARY BY NASA 2021112-091
	17:53 DEC. 19, 1969
Note that the date is the date of the printout, not the date of the program revision.

041574,000021:                                                                                                  #  NAME -- LSPOS -- LOCATE SUN AND MOON                  DATE -- 25 OCT 67
041575,000022:                                                                                                  #  MOD NO. 1
041576,000023:                                                                                                  #  MOD BY NEVILLE                                        ASSEMBLY SUNDANCE
041577,000024: 
041578,000025:                                                                                                  #  FUNCTIONAL DESCRIPTION
041579,000026: 
041580,000027:                                                                                                  #  COMPUTES UNIT POSITION VECTOR OF THE SUN AND MOON IN THE BASIC REFERENCE SYSTEM.  THE SUN VECTOR S IS
041581,000028:                                                                                                  #  LOCATED VIA TWO ANLES.  THE FIRST ANGLE (OBLIQUITY) IS THE ANGLE BETWEEN THE EARTH EQUATOR AND THE ECLIPTIC.  THE
041582,000029:                                                                                                  #  POSITION VECTOR OF THE SUN IS
041583,000030:                                                                                                  #        _
041584,000031:                                                                                                  #        S = (COS(LOS), COS(OBL)*SIN(LOS), SIN(OBL)*SIN(LOS)), WHERE
041585,000032: 
041586,000033:                                                                                                  #        LOS = LOS +LOS *T-(C *SIN(2PI*T)/365.24 +C *COS(2PI*T)/365.24)
041587,000034:                                                                                                  #                 0    R     0                     1
041588,000035:                                                                                                  #        LOS  (RAD) IS THE LONGITUDE OF THE SUN FOR MIGNIGHT JUNE 30TH OF THE PARTICULAR YEAR.
041589,000036:                                                                                                  #           0
041590,000037:                                                                                                  #        LOS  (RAD/DAY) IS THE MEAN RATE FOR THE PARTICULAR YEAR.
041591,000038:                                                                                                  #           R
041592,000039: 
041593,000040:                                                                                                  #  LOS  AND LOS  ARE STORED AS LOSC AND LOSR IN RATESP.
041594,000041:                                                                                                  #     0        R
041595,000042:                                                                                                  #  COS(OBL) AND SIN(OBL) ARE STORED IN THE MATRIX KONMAT.
041596,000043: 
041597,000044:                                                                                                  #  T, TIME MEASURED IN DAYS (24 HOURS) IS STORED IN TIMEP.
041598,000045: 
041599,000046:                                                                                                  #  C  AND C  ARE FUDGE FACTORS TO MINIMIZE THE DEVIATION.  THEY ARE STORED AS ONE CONSTANT (CMOD), SINCE
041600,000047:                                                                                                  #   0      1                               2  2 1/2
041601,000048:                                                                                                  #  C *SIN(X)+C *COS(X) CAN BE WRITTEN AS (C +C )   *SIN(X+PHI), WHERE PHI=ARCTAN(C /C ).
041602,000049:                                                                                                  #                                          0  1                                   1  0
041603,000050: 
041604,000051:                                                                                                  #  THE MOON IS LOCATED VIA FOUR ANGLES, THE FIRST IS THE OBLIQUITY. THE SECOND IS THE MEAN LONGITUDE OF THE MOON,
041605,000052:                                                                                                  #  MEASURED IN THE ECLIPTIC FROM THE MEAN EQUINOX TO THE MEAN ASCENDING NODE OF THE LUNAR ORBIT, AND THEN ALONG THE
041606,000053:                                                                                                  #  ORBIT.  THE THIRD ANGLE IS THE ANGLE BETWEEN THE ECLIPTIC AND THE LUNAR ORBIT.  THE FOURTH ANGLE IS THE LONGITUDE
041607,000054:                                                                                                  #  OF THE NODE OF THE MOON, MEASURED IN THE LUNAR ORBIT.  LET THESE ANGLES BE OBL,LOM,IM, AND LON RESPECTIVELY.
041608,000055: 
041609,000056:                                                                                                  #  THE SIMPLIFIED POSITION VECTOR OF THE MOON IS
041610,000057:                                                                                                  #        _
041611,000058:                                                                                                  #        M=(COS(LOM), COS(OBL)*SIN(LOM)-SIN(OBL)*SIN(IM)*SIN(LOM-LON), SIN(OBL)*SIN(LOM)+COS(OBL)*SIN(IM)*SIN(LOM-LON))
041612,000059: 
041613,000060:                                                                                                  #  WHERE
041614,000061:                                                                                                  #        LOM=LOM +LOM *T-(A *SIN(2PI*T/27.5545)+A *COS(2PI*T/27.5545)+B *SIN(2PI*T/32)+B *COS(2PI*T/32)), AND
041615,000062:                                                                                                  #               0    R     0                     1                     0                     1
041616,000063:                                                                                                  #        LON=LON +LON
041617,000064:                                                                                                  #               0    R
041618,000065:                                                                                                  #  A , A , B  AND B  ARE STORE AS AMOD AND BMOD (SEE DESCRIPTION OF CMOD, ABOVE).  COS(OBL), SIN(OBL)*SIN(IM),
041619,000066:                                                                                                  #   0   1   0      1
041620,000067:                                                                                                  #  SIN(OBL), AND COS(OBL)*SIN(IM) ARE STORED IN KONMAT AS K1, K2, K3, AND K4, RESPECTIVELY.  LOM , LOM , LON , LON
041621,000068:                                                                                                  #                                                                                               0     R     0     R
041622,000069:                                                                                                  #  ARE STORED AS LOM0, LOMR, LON0, AND LONR IN RATESP.
041623,000070: 
041624,000071:                                                                                                  #  THE THREE PHIS ARE STORED AS AARG, BARG, AND CARG(SUN).  ALL CONSTANTS ARE UPDATED BY YEAR.
041625,000072:
