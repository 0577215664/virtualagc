Source Code

These source-code files were derived originally from the scan from MIT's Dibner Institute's now-discontinued website titled "History of Recent Science and Technology". It is believed that the original listing was preserved by AGC developer Don Eyles, copied/reduced by AGC developer Hugh Blair-Smith, passed by AGC developer James Kernan to collector David Craig, scanned by Gary Neff, and then passed to the Dibner Institute where unfortunately the high-quality scans were reduced in legibility as part of the online-presentation process, and the original scans discarded. The source code was transcribed from those images by Ron Burkey. The images from the Dibner Institute website, and unfortunately not the discarded original high-quality scans, though with a few high-quality replacement scans provided directly by Gary Neff, are now available at the Virtual AGC project website. Report any conversion errors to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 131 OF AGC PROGRAM LUMINARY BY NASA 2021112-091
	17:53 DEC. 19, 1969
Note that the date is the date of the printout, not the date of the program revision.

041425,000018:                                                                                                  #  NAME -- LSPOS -- LOCATE SUN AND MOON                  DATE -- 25 OCT 67
041426,000019:                                                                                                  #  MOD NO. 1
041427,000020:                                                                                                  #  MOD BY NEVILLE                                        ASSEMBLY SUNDANCE
041428,000021: 
041429,000022:                                                                                                  #  FUNCTIONAL DESCRIPTION
041430,000023: 
041431,000024:                                                                                                  #  COMPUTES UNIT POSITION VECTOR OF THE SUN AND MOON IN THE BASIC REFERENCE SYSTEM.  THE SUN VECTOR S IS
041432,000025:                                                                                                  #  LOCATED VIA TWO ANLES.  THE FIRST ANGLE (OBLIQUITY) IS THE ANGLE BETWEEN THE EARTH EQUATOR AND THE ECLIPTIC.  THE
041433,000026:                                                                                                  #  POSITION VECTOR OF THE SUN IS
041434,000027:                                                                                                  #        _
041435,000028:                                                                                                  #        S = (COS(LOS), COS(OBL)*SIN(LOS), SIN(OBL)*SIN(LOS)), WHERE
041436,000029: 
041437,000030:                                                                                                  #        LOS = LOS +LOS *T-(C *SIN(2PI*T)/365.24 +C *COS(2PI*T)/365.24)
041438,000031:                                                                                                  #                 0    R     0                     1
041439,000032:                                                                                                  #        LOS  (RAD) IS THE LONGITUDE OF THE SUN FOR MIGNIGHT JUNE 30TH OF THE PARTICULAR YEAR.
041440,000033:                                                                                                  #           0
041441,000034:                                                                                                  #        LOS  (RAD/DAY) IS THE MEAN RATE FOR THE PARTICULAR YEAR.
041442,000035:                                                                                                  #           R
041443,000036: 
041444,000037:                                                                                                  #  LOS  AND LOS  ARE STORED AS LOSC AND LOSR IN RATESP.
041445,000038:                                                                                                  #     0        R
041446,000039:                                                                                                  #  COS(OBL) AND SIN(OBL) ARE STORED IN THE MATRIX KONMAT.
041447,000040: 
041448,000041:                                                                                                  #  T, TIME MEASURED IN DAYS (24 HOURS) IS STORED IN TIMEP.
041449,000042: 
041450,000043:                                                                                                  #  C  AND C  ARE FUDGE FACTORS TO MINIMIZE THE DEVIATION.  THEY ARE STORED AS ONE CONSTANT (CMOD), SINCE
041451,000044:                                                                                                  #   0      1                               2  2 1/2
041452,000045:                                                                                                  #  C *SIN(X)+C *COS(X) CAN BE WRITTEN AS (C +C )   *SIN(X+PHI), WHERE PHI=ARCTAN(C /C ).
041453,000046:                                                                                                  #                                          0  1                                   1  0
041454,000047: 
041455,000048:                                                                                                  #  THE MOON IS LOCATED VIA FOUR ANGLES, THE FIRST IS THE OBLIQUITY. THE SECOND IS THE MEAN LONGITUDE OF THE MOON,
041456,000049:                                                                                                  #  MEASURED IN THE ECLIPTIC FROM THE MEAN EQUINOX TO THE MEAN ASCENDING NODE OF THE LUNAR ORBIT, AND THEN ALONG THE
041457,000050:                                                                                                  #  ORBIT.  THE THIRD ANGLE IS THE ANGLE BETWEEN THE ECLIPTIC AND THE LUNAR ORBIT.  THE FOURTH ANGLE IS THE LONGITUDE
041458,000051:                                                                                                  #  OF THE NODE OF THE MOON, MEASURED IN THE LUNAR ORBIT.  LET THESE ANGLES BE OBL,LOM,IM, AND LON RESPECTIVELY.
041459,000052: 
041460,000053:                                                                                                  #  THE SIMPLIFIED POSITION VECTOR OF THE MOON IS
041461,000054:                                                                                                  #        _
041462,000055:                                                                                                  #        M=(COS(LOM), COS(OBL)*SIN(LOM)-SIN(OBL)*SIN(IM)*SIN(LOM-LON), SIN(OBL)*SIN(LOM)+COS(OBL)*SIN(IM)*SIN(LOM-LON))
041463,000056: 
041464,000057:                                                                                                  #  WHERE
041465,000058:                                                                                                  #        LOM=LOM +LOM *T-(A *SIN(2PI*T/27.5545)+A *COS(2PI*T/27.5545)+B *SIN(2PI*T/32)+B *COS(2PI*T/32)), AND
041466,000059:                                                                                                  #               0    R     0                     1                     0                     1
041467,000060:                                                                                                  #        LON=LON +LON
041468,000061:                                                                                                  #               0    R
041469,000062:                                                                                                  #  A , A , B  AND B  ARE STORE AS AMOD AND BMOD (SEE DESCRIPTION OF CMOD, ABOVE).  COS(OBL), SIN(OBL)*SIN(IM),
041470,000063:                                                                                                  #   0   1   0      1
041471,000064:                                                                                                  #  SIN(OBL), AND COS(OBL)*SIN(IM) ARE STORED IN KONMAT AS K1, K2, K3, AND K4, RESPECTIVELY.  LOM , LOM , LON , LON
041472,000065:                                                                                                  #                                                                                               0     R     0     R
041473,000066:                                                                                                  #  ARE STORED AS LOM0, LOMR, LON0, AND LONR IN RATESP.
041474,000067: 
041475,000068:                                                                                                  #  THE THREE PHIS ARE STORED AS AARG, BARG, AND CARG(SUN).  ALL CONSTANTS ARE UPDATED BY YEAR.
041476,000069:
