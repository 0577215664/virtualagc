Source Code

These source-code files were derived originally from the scan from MIT's Dibner Institute's now-discontinued website titled "History of Recent Science and Technology". It is believed that the original listing was preserved by AGC developer Don Eyles, copied/reduced by AGC developer Hugh Blair-Smith, passed by AGC developer James Kernan to collector David Craig, scanned by Gary Neff, and then passed to the Dibner Institute where unfortunately the high-quality scans were reduced in legibility as part of the online-presentation process, and the original scans discarded. The source code was transcribed from those images by Ron Burkey. The images from the Dibner Institute website, and unfortunately not the discarded original high-quality scans, though with a few high-quality replacement scans provided directly by Gary Neff, are now available at the Virtual AGC project website. Report any conversion errors to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 131 OF AGC PROGRAM LUMINARY BY NASA 2021112-091
	17:53 DEC. 19, 1969
Note that the date is the date of the printout, not the date of the program revision.

053134,000018:                                                                                                  #  PROGRAM DESCRIPTION                           DATE:  20 DECEMBER 1967
053135,000019:                                                                                                  #  PROGRAM NAME -- SELF-CHECK                    LOG SECTION:  AGC BLOCK TWO SELF-CHECK
053136,000020:                                                                                                  #  MOD NO -- 1                                   ASSEMBLY SUBROUTINE UTILITYM REV 25
053137,000021:                                                                                                  #  MOD BY -- GAUNTT

053139,000023:                                                                                                  #  FUNCTIONAL DESCRIPTION
053140,000024:                                                                                                  #        PROGRAM HAS TWO MAIN PARTS.  THE FIRST IS SELF-CHECK WHICH RUNS AS A ZERO PRIORITY JOB WITH NO CORE SET, AS
053141,000025:                                                                                                  #        PART OF THE BACK-UP IDLE LOOP.  THE SECOND IS SHOW-BANKSUM WHICH RUNS AS A REGULAR EXECUTIVE JOB WITH ITS OWN
053142,000026:                                                                                                  #        STARTING VERB.
053143,000027: 
053144,000028:                                                                                                  #        THE PURPOSE OF SELF-CHECK IS TO CHECK OUT VARIOUS PARTS OF THE COMPUTER AS OUTLINED BELOW IN THE OPTIONS.
053145,000029: 
053146,000030:                                                                                                  #        THE PURPOSE OF SHOW-BANKSUM IS TO DISPLAY THE SUM OF EACH BANK, ONE AT A TIME.
053147,000031: 
053148,000032:                                                                                                  #        IN ALL THERE ARE 7 POSSIBLE OPTIONS IN THIS BLOCK II VERSION OF SELF-CHECK.  MORE DETAIL DESCRIPTION MAY BE
053149,000033:                                                                                                  #        FOUND IN E-2065 BLOCK II AGC SELF-CHECK AND SHOW BANKSUM BY EDWIN D. SMALLY DECEMBER 1966, AND ADDENDA 2 AND 3.
053150,000034: 
053151,000035:                                                                                                  #        THE DIFFERENT OPTIONS ARE CONTROLLED BY PUTTING DIFFERENT NUMBERS IN THE SMODE REGISTER (NOUN 27).  BELOW IS
053152,000036:                                                                                                  #        A DESCRIPTION OF WHAT PARTS OF THE COMPUTER THAT ARE CHECKED BY THE OPTIONS, AND THE CORRESPONDING NUMBER, IN
053153,000037:                                                                                                  #        OCTAL, TO LOAD INTO SMODE.
053154,000038:                                                                                                  #                +-4             ERASABLE MEMORY
053155,000039:                                                                                                  #                +-5             FIXED MEMORY
053156,000040:                                                                                                  #                +-1,2,3,6,7,10  EVERYTHING IN OPTIONS 4 AND 5.
053157,000041:                                                                                                  #                -0              SAME AS +-10 UNTIL AN ERROR IS DETECTED.
053158,000042:                                                                                                  #                +0              NO CHECK, PUTS COMPUTER INTO THE BACKUP IDLE LOOP.
053159,000043: 
053160,000044:                                                                                                  #  WARNINGS
053161,000045:                                                                                                  #        USE OF E MEMORY RESERVED FOR SELF-CHECK (EVEN IN IDLE LOOP) AS TEMP STORAGE BY OTHER PROGRAMS IS DANGEROUS.
053162,000046:                                                                                                  #        SMODE SET GREATER THAN OCT 10 PUTS COMPUTER INTO BACKUP IDLE LOOP.
053163,000047: 
053164,000048:                                                                                                  #  CALLING SEQUENCE
053165,000049:                                                                                                  #        TO CALL SELF-CHECK KEY IN
053166,000050:                                                                                                  #                V 21 N 27 E     OPTION NUMBER E
053167,000051:                                                                                                  #        TO CALL SHOW-BANKSUM KEY IN
053168,000052:                                                                                                  #                V 91 E          DISPLAYS FIRST BANK
053169,000053:                                                                                                  #                V 33 E          PROCEED, DISPLAYS NEXT BANK
053170,000054: 
053171,000055:                                                                                                  #  EXIT MODES, NORMAL AND ALARM
053172,000056:                                                                                                  #        SELF-CHECK NORMALLY CONTINUES INDEFINITELY UNLESS THERE IS AN ERROR DETECTED.  IF SO + OPTION NUMBERS PUT
053173,000057:                                                                                                  #        COMPUTER INTO BACKUP IDLE LOOP, - OPTIONS NUMBERS RESTART THE OPTION.
053174,000058: 
053175,000059:                                                                                                  #        THE -0 OPTION PROCEEDS FROM THE LINE FOLLOWING THE LINE WHERE THE ERROR WAS DETECTED.
053176,000060: 
053177,000061:                                                                                                  #        SHOW-BANKSUM PROCEEDS UNTIL A TERMINATE IS KEYED IN (V 34 E).  THE COMPUTER IS PUT INTO THE BACKUP IDLE LOOP.
053178,000062: 
053179,000063:                                                                                                  #  OUTPUT

053181,000065:                                                                                                  #        SELF-CHECK UPON DETECTING AN ERROR LOADS THE SELF-CHECK ALARM CONSTANT (01102) INTO THE FAILREG SET AND
053182,000066:                                                                                                  #        TURNS ON THE ALARM LIGHT.  THE OPERATOR MAY THEN DISPLAY THE THREE FAILREGS BY KEYING IN V 05 N 09 E.  FOR FURTHER
053183,000067:                                                                                                  #        INFORMATION HE MAY KEY IN V 05 N 08 E, THE DSKY DISPLAY IN R1 WILL BE ADDRESS+1 OF WHERE THE ERROR WAS DETECTED,
053184,000068:                                                                                                  #        IN R2 THE BBCON OF SELF-CHECK, AND IN R3 THE TOTAL NUMBER OF ERRORS DETECTED BY SELF-CHECK SINCE THE LAST MAN
053185,000069:                                                                                                  #        INITIATED FRESH START (SLAP1).
053186,000070: 
053187,000071:                                                                                                  #        SHOW-BANKSUM STARTING WTIH BANK 0 DISPLAYS IN R1 THE BANK SUM (A +-NUMBER EQUAL TO THE BANK NUMBER), IN R2
053188,000072:                                                                                                  #        THE BANK NUMBER, AND IN R3 THE BUGGER WORD.
053189,000073: 
053190,000074:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED
053191,000075:                                                                                                  #        ACCOMPLISHED BY FRESH START
053192,000076:                                                                                                  #                SMODE SET TO +0
053193,000077: 
053194,000078:                                                                                                  #  DEBRIS
053195,000079:                                                                                                  #        ALL EXITS FROM THE CHECK OF ERASABLE (ERASCHK) RESTORE ORIGINAL CONTENTS TO REGISTERS UNDER CHECK.
053196,000080:                                                                                                  #        EXCEPTION IS A RESTART.  RESTART THAT OCCURS DURING ERASCHK RESTORES ERASABLE, UNLESS THERE IS EVIDENCE TO DOUBT
053197,000081:                                                                                                  #        E MEMORY, IN WHICH CASE PROGRAM THEN DOES A FRESH START (DOFSTART).
