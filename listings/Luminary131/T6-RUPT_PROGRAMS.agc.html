Source Code

These source-code files were derived originally from the scan from MIT's Dibner Institute's now-discontinued website titled "History of Recent Science and Technology". It is believed that the original listing was preserved by AGC developer Don Eyles, copied/reduced by AGC developer Hugh Blair-Smith, passed by AGC developer James Kernan to collector David Craig, scanned by Gary Neff, and then passed to the Dibner Institute where unfortunately the high-quality scans were reduced in legibility as part of the online-presentation process, and the original scans discarded. The source code was transcribed from those images by Ron Burkey. The images from the Dibner Institute website, and unfortunately not the discarded original high-quality scans, though with a few high-quality replacement scans provided directly by Gary Neff, are now available at the Virtual AGC project website. Report any conversion errors to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 131 OF AGC PROGRAM LUMINARY BY NASA 2021112-091
	17:53 DEC. 19, 1969
Note that the date is the date of the printout, not the date of the program revision.

058169,000018:                                                                                                  #  PROGRAM NAMES:        (1) T6JOBCHK    MOD. NO. 5      OCTOBER 2, 1967
058170,000019:                                                                                                  #                        (2) DOT6RUPT
058171,000020:                                                                                                  #  MODIFICATION BY:      LOWELL G. HULL (A.C.ELECTRONICS)
058172,000021: 
058173,000022:                                                                                                  #  THESE PROGRAMS ENABLE THE LM DAP TO CONTROL THE THRUST TIMES OF THE REACTION CONTROL SYSTEM JETS BY USING TIME6.
058174,000023:                                                                                                  #  SINCE THE LM DAP MAINTAINS EXCLUSIVE CONTROL OVER TIME6 AND ITS INTERRUPTS, THE FOLLOWING CONVENTIONS HAVE BEEN
058175,000024:                                                                                                  #  ESTABLISHED AND MUST NOT BE TAMPERED WITH:
058176,000025:                                                                                                  #        1.      NO NUMBER IS EVER PLACED INTO TIME6 EXCEPT BY LM DAP.
058177,000026:                                                                                                  #        2.      NO PROGRAM OTHER THAN LM DAP ENABLES THE TIME6 COUNTER.
058178,000027:                                                                                                  #        3.      TO USE TIME6, THE FOLLOWING SEQUENCE IS ALWAYS EMPLOYED:
058179,000028:                                                                                                  #                A.      A POSITIVE (NON-ZERO) NUMBER IS STORED IN TIME6.
058180,000029:                                                                                                  #                B.      THE TIME6 CLOCK IS ENABLED.
058181,000030:                                                                                                  #                C.      TIME6 IS INTERROGATED AND IS:
058182,000031:                                                                                                  #                        I.      NEVER FOUND NEGATIVE (NON-ZERO) OR +0.
058183,000032:                                                                                                  #                        II.     SOMETIMES FOUND POSITIVE (BETWEEN 1 AND 240D) INDICATING THAT IT IS ACTIVE.
058184,000033:                                                                                                  #                        III.    SOMETIMES FOUND POSMAX INDICATING THAT IT IS INACTIVE AND NOT ENABLED.
058185,000034:                                                                                                  #                        IV.     SOMETIMES FOUND NEGATIVE ZERO INDICATING THAT:
058186,000035:                                                                                                  #                                A.      A T6RUPT IS ABOUT TO OCCUR AT THE NEXT DINC, OR
058187,000036:                                                                                                  #                                B.      A T6RUPT IS WAITING IN THE PRIORITY CHAIN, OR
058188,000037:                                                                                                  #                                C.      A T6RUPT IS IN PROCESS NOW.
058189,000038:                                                                                                  #        4.      ALL PROGRAMS WHICH OPERATE IN EITHER INTERRUPT MODE OR WITH INTERRUPT INHIBITED MUST CALL T6JOBCHK
058190,000039:                                                                                                  #                EVERY 5 MILLISECONDS TO PROCESS A POSSIBLE WAITING T6RUPT BEFORE IT CAN BE HONORED BY THE HARDWARE.
058191,000040:                                                                                                  #       (5.      PROGRAM JTLST, IN Q,R-AXES, HANDLES THE INPUT LIST.)
058192,000041: 
058193,000042:                                                                                                  #  T6JOBCHK CALLING SEQUENCE:
058194,000043:                                                                                                  #                L       TC      T6JOBCHK
058195,000044:                                                                                                  #                L+1     (RETURN)
058196,000045: 
058197,000046:                                                                                                  #  DOT6RUPT CALLING SEQUENCE:
058198,000047:                                                                                                  #                        DXCH    ARUPT           # T6RUPT LEAD IN AT LOCATION 4004.
058199,000048:                                                                                                  #                        EXTEND
058200,000049:                                                                                                  #                        DCA     T6ADR
058201,000050:                                                                                                  #                        DTCB
058202,000051: 
058203,000052:                                                                                                  #  SUBROUTINES CALLED:   DOT6RUPT CALLS T6JOBCHK.
058204,000053: 
058205,000054:                                                                                                  #  NORMAL EXIT MODES:    T6JOBCHK RETURNS TO L +1.
058206,000055:                                                                                                  #                        DOT6RUPT TRANSFERS CONTROL TO RESUME.
058207,000056: 
058208,000057:                                                                                                  #  ALARM/ABORT MODES:    NONE.
058209,000058:
