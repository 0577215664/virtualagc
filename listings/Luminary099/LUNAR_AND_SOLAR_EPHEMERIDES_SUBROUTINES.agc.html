Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Luminary 131 (Apollo 13) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 001 OF AGC PROGRAM LMY99 BY NASA 2021112-061
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

042662,000031:                                                                                                  #  MOD BY NEVILLE                                        ASSEMBLY SUNDANCE
042663,000032: 
042664,000033:                                                                                                  #  FUNCTIONAL DESCRIPTION
042665,000034: 
042666,000035:                                                                                                  #  COMPUTES UNIT POSITION VECTOR OF THE SUN AND MOON IN THE BASIC REFERENCE SYSTEM.  THE SUN VECTOR S IS
042667,000036:                                                                                                  #  LOCATED VIA TWO ANGLES.  THE FIRST ANGLE (OBLIQUITY) IS THE ANGLE BETWEEN THE EARTH EQUATOR AND THE ECLIPTIC.  THE
042668,000037:                                                                                                  #  POSITION VECTOR OF THE SUN IS
042669,000038:                                                                                                  #        _
042670,000039:                                                                                                  #        S = (COS(LOS), COS(OBL)*SIN(LOS), SIN(OBL)*SIN(LOS)), WHERE
042671,000040: 
042672,000041:                                                                                                  #        LOS = LOS +LOS *T-(C *SIN(2PI*T)/365.24 +C *COS(2PI*T)/365.24)
042673,000042:                                                                                                  #                 0    R     0                     1
042674,000043:                                                                                                  #        LOS  (RAD) IS THE LONGITUDE OF THE SUN FOR MIGNIGHT JUNE 30TH OF THE PARTICULAR YEAR.
042675,000044:                                                                                                  #           0
042676,000045:                                                                                                  #        LOS  (RAD/DAY) IS THE MEAN RATE FOR THE PARTICULAR YEAR.
042677,000046:                                                                                                  #           R
042678,000047: 
042679,000048:                                                                                                  #  LOS  AND LOS  ARE STORED AS LOSC AND LOSR IN RATESP.
042680,000049:                                                                                                  #     0        R
042681,000050:                                                                                                  #  COS(OBL) AND SIN(OBL) ARE STORED IN THE MATRIX KONMAT.
042682,000051: 
042683,000052:                                                                                                  #  T, TIME MEASURED IN DAYS (24 HOURS) IS STORED IN TIMEP.
042684,000053: 
042685,000054:                                                                                                  #  C  AND C  ARE FUDGE FACTORS TO MINIMIZE THE DEVIATION.  THEY ARE STORED AS ONE CONSTANT (CMOD), SINCE
042686,000055:                                                                                                  #   0      1                               2  2 1/2
042687,000056:                                                                                                  #  C *SIN(X)+C *COS(X) CAN BE WRITTEN AS (C +C )   *SIN(X+PHI), WHERE PHI=ARCTAN(C /C ).
042688,000057:                                                                                                  #                                          0  1                                   1  0
042689,000058: 
042690,000059:                                                                                                  #  THE MOON IS LOCATED VIA FOUR ANGLES, THE FIRST IS THE OBLIQUITY. THE SECOND IS THE MEAN LONGITUDE OF THE MOON,
042691,000060:                                                                                                  #  MEASURED IN THE ECLIPTIC FROM THE MEAN EQUINOX TO THE MEAN ASCENDING NODE OF THE LUNAR ORBIT, AND THEN ALONG THE
042692,000061:                                                                                                  #  ORBIT.  THE THIRD ANGLE IS THE ANGLE BETWEEN THE ECLIPTIC AND THE LUNAR ORBIT.  THE FOURTH ANGLE IS THE LONGITUDE
042693,000062:                                                                                                  #  OF THE NODE OF THE MOON, MEASURED IN THE LUNAR ORBIT.  LET THESE ANGLES BE OBL,LOM,IM, AND LON RESPECTIVELY.
042694,000063: 
042695,000064:                                                                                                  #  THE SIMPLIFIED POSITION VECTOR OF THE MOON IS
042696,000065:                                                                                                  #        _
042697,000066:                                                                                                  #        M=(COS(LOM), COS(OBL)*SIN(LOM)-SIN(OBL)*SIN(IM)*SIN(LOM-LON), SIN(OBL)*SIN(LOM)+COS(OBL)*SIN(IM)*SIN(LOM-LON))
042698,000067: 
042699,000068:                                                                                                  #  WHERE
042700,000069:                                                                                                  #        LOM=LOM +LOM *T-(A *SIN(2PI*T/27.5545)+A *COS(2PI*T/27.5545)+B *SIN(2PI*T/32)+B *COS(2PI*T/32)), AND
042701,000070:                                                                                                  #               0    R     0                     1                     0                     1
042702,000071:                                                                                                  #        LON=LON +LON
042703,000072:                                                                                                  #               0    R
042704,000073:                                                                                                  #  A , A , B  AND B  ARE STORE AS AMOD AND BMOD (SEE DESCRIPTION OF CMOD, ABOVE).  COS(OBL), SIN(OBL)*SIN(IM),
042705,000074:                                                                                                  #   0   1   0      1
042706,000075:                                                                                                  #  SIN(OBL), AND COS(OBL)*SIN(IM) ARE STORED IN KONMAT AS K1, K2, K3, AND K4, RESPECTIVELY.  LOM , LOM , LON , LON
042707,000076:                                                                                                  #                                                                                               0     R     0     R
042708,000077:                                                                                                  #  ARE STORED AS LOM0, LOMR, LON0, AND LONR IN RATESP.
042709,000078: 
042710,000079:                                                                                                  #  THE THREE PHIS ARE STORED AS AARG, BARG, AND CARG(SUN).  ALL CONSTANTS ARE UPDATED BY YEAR.
042711,000080:

042713,000082:                                                                                                  #  Page 985
