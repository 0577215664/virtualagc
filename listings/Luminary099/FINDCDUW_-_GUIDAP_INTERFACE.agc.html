These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Luminary 131 (Apollo 13) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 001 OF AGC PROGRAM LMY99 BY NASA 2021112-061
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

039688,000031:                                                                                                  #  PROGRAM NAME:  FINDCDUW
039689,000032:                                                                                                  #  MOD NUMBER:  1        68 07 15
039690,000033:                                                                                                  #  MOD AUTHOR:  KLUMPP
039691,000034: 
039692,000035:                                                                                                  #  OBJECTS OF MOD:       1.      TO SUPPLY COMMANDED GIMBAL ANGLES FOR NOUN 22.
039693,000036:                                                                                                  #                        2.      TO MAINTAIN CORRECT AND CURRENT THRUST
039694,000037:                                                                                                  #                                DIRECTION DATA IN ALL MODES.  THIS IS DONE BY
039695,000038:                                                                                                  #                                FETCHING FOR THE THRUST DIRECTION FILTER THE
039696,000039:                                                                                                  #                                CDUD'S IN PNGCS-AUTO, THE CDU'S IN ALL OTHER
039697,000040:                                                                                                  #                                MODES.
039698,000041:                                                                                                  #                        3.      TO SUBSTITUTE A STOPRATE FOR THE NORMAL
039699,000042:                                                                                                  #                                AUTOPILOT COMMANDS WHENEVER
039700,000043:                                                                                                  #                                1) NOT IN PNGCS-AUTO, OR
039701,000044:                                                                                                  #                                2) ENGINE IS OFF.
039702,000045: 
039703,000046:                                                                                                  #  FUNCTIONAL DESCRIPTION:
039704,000047: 
039705,000048:                                                                                                  #  FINDCDUW PROVIDES THE INTERFACES BETWEEN THE VARIOUS POWERED FLITE GUIDANCE PROGRAMS
039706,000049:                                                                                                  #  AND THE DIGITAL AUTOPILOT.  THE INPUTS TO FINDCDUW ARE THE THRUST COMMAND VECTOR
039707,000050:                                                                                                  #  AND THE WINDOW COMMAND VECTOR, AND THE OUTPUTS ARE THE GIMBAL ANGLE
039708,000051:                                                                                                  #  INCREMENTS, THE COMMANDED ATTITUDE ANGLE RATES, AND THE COMMANDED
039709,000052:                                                                                                  #  ATTITUDE LAG ANGLES (WHICH ACCOUNT FOR THE ANGLES BY WHICH THE BODY WILL
039710,000053:                                                                                                  #  LAG BEHIND A RAMP COMMAND IN ATTITUDE ANGLE DUE TO THE FINITE ANGULAR
039711,000054:                                                                                                  #  ACCELERATIONS AVAILABLE).
039712,000055: 
039713,000056:                                                                                                  #  FINDCDUW ALINES THE ESTIMATED THRUST VECTOR FROM THE THRUST DIRECTION
039714,000057:                                                                                                  #  FILTER WITH THE THRUST COMMAND VECTOR, AND, WHEN XDVINHIB SET,
039715,000058:                                                                                                  #  ALINES THE +Z HALF OF THE LM ZX PLANE WITH THE WINDOW COMMAND VECTOR.

039718,000061:                                                                                                  #  SPECIFICATIONS:
039719,000062: 
039720,000063:                                                                                                  #  INITIALIZATION:       A SINGLE INTERPRETIVE CALL TO INITCDUW IS REQUIRED
039721,000064:                                                                                                  #                        BEFORE EACH GUIDED MANEUVER USING FINDCDUW.
039722,000065: 
039723,000066:                                                                                                  #  CALL:                 INTERPRETIVE CALL TO FINDCDUW WITH THE THRUST COMMAND
039724,000067:                                                                                                  #                        VECTOR IN MPAC.  INTERPRETIVE CALL TO FINDCDUW -2 WITH
039725,000068:                                                                                                  #                        THE THRUST COMMAND VECTOR IN UNFC/2 AND NOT IN MPAC.
039726,000069: 
039727,000070:                                                                                                  #  RETURNS:              NORMAL INTERPRETIVE IN ALL CASES
039728,000071:                                                                                                  #                        1.      NORMALLY ALL AUTOPILOT CMDS ARE ISSUED.
039729,000072:                                                                                                  #                        2.      IF NOT PNGCS AUTO, DO STOPRATE AND RETURN
039730,000073:                                                                                                  #                                WITHOUT ISSUING AUTOPILOT CMDS.
039731,000074:                                                                                                  #                        3.      IF ENGINE OFF, DO STOPRATE AND RETURN WITHOUT
039732,000075:                                                                                                  #                                ISSUING AUTOPILOT CMDS.
039733,000076: 
039734,000077:                                                                                                  #  ALARMS:               00401   IF INPUTS DETERMINE AN ATTITUDE IN GIMBAL LOCK.
039735,000078:                                                                                                  #                                FINDCDUW DRIVES CDUXD AND CDUYD TO THE RQD VALUES,
039736,000079:                                                                                                  #                                BUT DRIVES CDUZD ONLY TO THE GIMBAL LOCK CONE.
039737,000080: 
039738,000081:                                                                                                  #                        00402   IF UNFC/2 OR UNWC/2 PRODUCE OVERFLOW WHEN
039739,000082:                                                                                                  #                                UNITIZED USING NORMUNIT.  FINDCDUW ISSUES
039740,000083:                                                                                                  #                                STOPRATE AS ONLY INPUT TO AUTOPILOT.
039741,000084: 
039742,000085:                                                                                                  #  INPUTS:               UNFC/2          THRUST COMMAND VECTOR, NEED NOT BE SEMI-UNIT.
039743,000086:                                                                                                  #                        UNWC/2          WINDOW COMMAND VECTOR, NEED NOT BE SEMI-UNIT.
039744,000087:                                                                                                  #                        XOVINHIB        FLAG DENOTING X AXIS OVERRIDE INHIBITED.
039745,000088:                                                                                                  #                        CSMDOCKD        FLAG DENOTING CSM DOCKED.
039746,000089:                                                                                                  #                        STEERSW         FLAG DENOTING INSUFF THRUST FOR THRUST DIR FLTR.
039747,000090: 
039748,000091:                                                                                                  #  OUTPUTS:              DELCDUX,Y,Z
039749,000092:                                                                                                  #                        OMEGAPD,+1,+2
039750,000093:                                                                                                  #                        DELPEROR,+1,+2
039751,000094:                                                                                                  #                        CPHI,+1,+2 FOR NOUN22
039752,000095: 
039753,000096:                                                                                                  #  DEBRIS:               FINDCDUW DESTROYS SINCDUX,Y,Z AND COSCDUX,Y,Z BY
039754,000097:                                                                                                  #                        WRITING INTO THESE LOCATIONS THE SINES AND COSINES
039755,000098:                                                                                                  #                        OF THE CDUD'S IN PNGCS-AUTO, OF THE CDU'S OTHERWISE.
