Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Luminary 131 (Apollo 13) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 001 OF AGC PROGRAM LMY99 BY NASA 2021112-061
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

047632,000037:                                                                                                  #  MOD NO - 2                                                                    LOG SECTION - WAITLIST

047637,000042:                                                                                                  #  FUNCTIONAL DESCRIPTION -
047638,000043:                                                                                                  #        PART OF A SECTION OF PROGRAMS.- WAITLIST, TASKOVER, T3RUPT, USED TO CALL A PROGRAM, (CALLED A TASK),
047639,000044:                                                                                                  #        WHICH IS TO BEGIN IN C(A) CENTISECONDS.  WAITLIST UPDATES TIME3, LST1 AND LST2.  THE MEANING OF THESE LISTS
047640,000045:                                                                                                  #        FOLLOW.
047641,000046: 
047642,000047:                                                                                                  #                C(TIME3) = 16384 -(T1-T) CENTISECONDS, (T=PRESENT TIME, T1-TIME FOR TASK1)
047643,000048: 
047644,000049:                                                                                                  #                        C(LST1)         =       -(T2-T1)+1
047645,000050:                                                                                                  #                        C(LST1 +1)      =       -(T3-T2)+1
047646,000051:                                                                                                  #                        C(LST1 +2)      =       -(T4-T3)+1
047647,000052:                                                                                                  #                                       ...
047648,000053:                                                                                                  #                        C(LST1 +6)      =       -(T8-T7)+1
047649,000054:                                                                                                  #                        C(LST1 +7)      =       -(T9-T8)+1
047650,000055: 
047651,000056:                                                                                                  #                        C(LST2)         =       2CADR OF TASK1
047652,000057:                                                                                                  #                        C(LST2 +2)      =       2CADR OF TASK2
047653,000058:                                                                                                  #                                       ...
047654,000059:                                                                                                  #                        C(LST2 +14)     =       2CADR OF TASK8
047655,000060:                                                                                                  #                        C(LST2 +16)     =       2CADR OF TASK9
047656,000061: 
047657,000062:                                                                                                  #  WARNINGS -
047658,000063:                                                                                                  #        1)      1 <= C(A) <= 16250D (1 CENTISECOND TO 162.5 SEC)
047659,000064:                                                                                                  #        2)      9 TASKS MAXIMUM
047660,000065:                                                                                                  #        3)      TASKS CALLED UNDER INTERRUPT INHIBITED
047661,000066:                                                                                                  #        4)      TASKS END BY TC TASKOVER
047662,000067: 
047663,000068:                                                                                                  #  CALLING SEQUENCE -
047664,000069:                                                                                                  #        L-1     CA      DELTAT  (TIME IN CENTISECONDS TO TASK START)
047665,000070:                                                                                                  #        L       TC      WAITLIST
047666,000071:                                                                                                  #        L+1     2CADR   DESIRED TASK
047667,000072:                                                                                                  #        L+2     (MINOR OF 2CADR)
047668,000073:                                                                                                  #        L+3     RELINT          (RETURNS HERE)
047669,000074: 
047670,000075:                                                                                                  #  TWIDDLE -
047671,000076:                                                                                                  #        TWIDDLE IS FOR USE WHEN THE TASK BEING SET UP IS IN THE SAME EBANK AND FBANK AS THE USER.  IN
047672,000077:                                                                                                  #        SUCH CASES, IT IMPROVES UPON WAITLIST BY ELIMINATING THE NEED FOR THE BBCON HALF OF THE 2CADR,

Page 1118 

047674,000079:                                                                                                  #        SAVING A WORD.  TWIDDLE IS LIKE WAITLIST IN EVERY RESPECT EXCEPT CALLING SEQUENCE, TO WIT-
047675,000080:                                                                                                  #                L-1     CA      DELTAT
047676,000081:                                                                                                  #                L       TC      TWIDDLE
047677,000082:                                                                                                  #                L+1     ADRES   DESIRED TASK
047678,000083:                                                                                                  #                L+2     RELINT          (RETURNS HERE)
047679,000084: 
047680,000085:                                                                                                  #  NORMAL EXIT MODES -
047681,000086:                                                                                                  #        AT L+3 OF CALLING SEQUENCE
047682,000087: 
047683,000088:                                                                                                  #  ALARM OR ABORT EXIT MODES -
047684,000089:                                                                                                  #        TC      ABORT
047685,000090:                                                                                                  #        OCT     1203    (WAITLIST OVERFLOW - TOO MANY TASKS)
047686,000091: 
047687,000092:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED -
047688,000093:                                                                                                  #        ACCOMPLISHED BY FRESH START.--  LST2, ..., LST2 +16 = ENDTASK
047689,000094:                                                                                                  #                                        LST1, ..., LST1 +7  = NEG1/2
047690,000095: 
047691,000096:                                                                                                  #  OUTPUT --
047692,000097:                                                                                                  #        LST1 AND LST2 UPDATED WITH NEW TASK AND ASSOCIATED TIME.
047693,000098: 
047694,000099:                                                                                                  #  DEBRIS -
047695,000100:                                                                                                  #        CENTRALS - A,Q,L
047696,000101:                                                                                                  #        OTHER    - WAITEXIT, WAITADR, WAITTEMP, WAITBANK
047697,000102: 
047698,000103:                                                                                                  #  DETAILED ANALYSIS OF TIMING -
047699,000104:                                                                                                  #        CONTROL WILL NOT BE RETURNED TO THE SPECIFIED ADDRESS (2CADR) IN EXACTLY DELTA T CENTISECONDS.
047700,000105:                                                                                                  #        THE APPROXIMATE TIME MAY BE CALCULATED AS FOLLOWS
047701,000106:                                                                                                  #                LET T0 = THE TIME OF THE TC WAITLIST
047702,000107:                                                                                                  #                LET TS = T0 +147U + COUNTER INCREMENTS (SET UP TIME)
047703,000108:                                                                                                  #                LET X  = TS -(100TS)/100  (VARIANCE FROM COUNTERS)
047704,000109:                                                                                                  #                LET Y  = LENGTH OF TIME OF INHIBIT INTERRUPT AFTER T3RUPT
047705,000110:                                                                                                  #                LET Z  = LENGTH OF TIME TO PROCESS TASKS WHICH ARE DUE THIS T3RUPT BUT DISPATCHED EARLIER.
047706,000111:                                                                                                  #                         (Z=0, USUALLY)
047707,000112:                                                                                                  #                LET DELTD  = THE ACTUAL TIME TAKEN TO GIVE CONTROL TO 2CADR
047708,000113:                                                                                                  #                THEN DELTD = TS+DELTA T -X +Y +Z +1.05MS* +COUNTERS*
047709,000114:                                                                                                  #                *-THE TIME TAKEN BY WAITLIST ITSELF AND THE COUNTER TICKING DURING THIS WAITLIST TIME.
047710,000115:                                                                                                  #        IN SHORT, THE ACTUAL TIME TO RETURN CONTROL TO A 2CADR IS AUGMENTED BY THE TIME TO SET UP THE TASK:S
047711,000116:                                                                                                  #        INTERRUPT, ALL COUNTERS TICKING, THE T3RUPT PROCESSING TIME, THE WAITLIST PROCESSING TIME AND THE POSSIBILITY
047712,000117:                                                                                                  #        OF OTHER TASKS INHIBITING THE INTERRUPT.
