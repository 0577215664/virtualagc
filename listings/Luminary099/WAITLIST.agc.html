Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Luminary 131 (Apollo 13) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 001 OF AGC PROGRAM LMY99 BY NASA 2021112-061
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

047506,000040:                                                                                                  #        PART OF A SECTION OF PROGRAMS -- WAITLIST, TASKOVER, T3RUPT, USED TO CALL A PROGRAM (CALLED A TASK),

047508,000042:                                                                                                  #        FOLLOW.
047509,000043: 
047510,000044:                                                                                                  #                C(TIME3) = 16384 -(T1-T) CENTISECONDS, (T=PRESENT TIME, T1-TIME FOR TASK1)
047511,000045: 
047512,000046:                                                                                                  #                        C(LST1)         =       -(T2-T1)+1
047513,000047:                                                                                                  #                        C(LST1 +1)      =       -(T3-T2)+1
047514,000048:                                                                                                  #                        C(LST1 +2)      =       -(T4-T3)+1
047515,000049:                                                                                                  #                                       ...
047516,000050:                                                                                                  #                        C(LST1 +6)      =       -(T8-T7)+1
047517,000051:                                                                                                  #                        C(LST1 +7)      =       -(T9-T8)+1
047518,000052: 
047519,000053:                                                                                                  #                        C(LST2)         =       2CADR OF TASK1
047520,000054:                                                                                                  #                        C(LST2 +2)      =       2CADR OF TASK2
047521,000055:                                                                                                  #                                       ...
047522,000056:                                                                                                  #                        C(LST2 +14)     =       2CADR OF TASK8
047523,000057:                                                                                                  #                        C(LST2 +16)     =       2CADR OF TASK9
047524,000058: 
047525,000059:                                                                                                  #  WARNINGS --
047526,000060:                                                                                                  #        1)      1 <= C(A) <= 16250D (1 CENTISECOND TO 162.5 SEC)
047527,000061:                                                                                                  #        2)      9 TASKS MAXIMUM
047528,000062:                                                                                                  #        3)      TASKS CALLED UNDER INTERRUPT INHIBITED
047529,000063:                                                                                                  #        4)      TASKS END BY TC TASKOVER
047530,000064: 
047531,000065:                                                                                                  #  CALLING SEQUENCE --
047532,000066:                                                                                                  #        L-1     CA      DELTAT  (TIME IN CENTISECONDS TO TASK START)
047533,000067:                                                                                                  #        L       TC      WAITLIST
047534,000068:                                                                                                  #        L+1     2CADR   DESIRED TASK.
047535,000069:                                                                                                  #        L+2     (MINOR OF 2CADR)
047536,000070:                                                                                                  #        L+3     RELINT          (RETURNS HERE)
047537,000071: 
047538,000072:                                                                                                  #  TWIDDLE --
047539,000073:                                                                                                  #        TWIDDLE IS FOR USE WHEN THE TASK BEING SET UP IS IN THE SAME EBANK AND FBANK AS THE USER.  IN
047540,000074:                                                                                                  #        SUCH CASES, IT IMPROVES UPON WAITLIST BY ELIMINATING THE NEED FOR THE BBCON HALF OF THE 2CADR,
047541,000075:                                                                                                  #  Page 1118
047542,000076:                                                                                                  #        SAVING A WORD.  TWIDDLE IS LIKE WAITLIST IN EVERY RESPECT EXCEPT CALLING SEQUENCE, TO WIT,
047543,000077:                                                                                                  #                L-1     CA      DELTAT
047544,000078:                                                                                                  #                L       TC      TWIDDLE
047545,000079:                                                                                                  #                L+1     ADRES   DESIRED TASK
047546,000080:                                                                                                  #                L+2     RELINT          (RETURNS HERE)
047547,000081: 
047548,000082:                                                                                                  #  NORMAL EXIT MODES --
047549,000083:                                                                                                  #        AT L+3 OF CALLING SEQUENCE.
047550,000084: 
047551,000085:                                                                                                  #  ALARM OR ABORT EXIT MODES --
047552,000086:                                                                                                  #        TC      ABORT
047553,000087:                                                                                                  #        OCT     1203    (WAITLIST OVERFLOW -- TOO MANY TASKS)
047554,000088: 
047555,000089:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED --
047556,000090:                                                                                                  #        ACCOMPLISHED BY FRESH START --  LST2, ..., LST2 +16 = ENDTASK
047557,000091:                                                                                                  #                                        LST1, ..., LST1 +7  = NEG1/2

047566,000100:                                                                                                  #  DETAILED ANALYSIS OF TIMING --
047567,000101:                                                                                                  #        CONTROL WILL NOT BE RETURNED TO THE SPECIFIED ADDRESS (2CADR) IN EXACTLY DELTA T CENTISECONDS.
047568,000102:                                                                                                  #        THE APPROXIMATE TIME MAY BE CALCULATED AS FOLLOWS:
047569,000103:                                                                                                  #                LET T0 = THE TIME OF THE TC WAITLIST
047570,000104:                                                                                                  #                LET TS = T0 +147U + COUNTER INCREMENTS (SET UP TIME)
047571,000105:                                                                                                  #                LET X  = TS -(100TS)/100  (VARIANCE FROM COUNTERS)
047572,000106:                                                                                                  #                LET Y  = LENGTH OF TIME OF INHIBIT INTERRUPT AFTER T3RUPT
047573,000107:                                                                                                  #                LET Z  = LENGTH OF TIME TO PROCESS TASKS WHICH ARE DUE THIS T3RUPT BUT DISPATCHED EARLIER.
047574,000108:                                                                                                  #                         (Z=0, USUALLY).
047575,000109:                                                                                                  #                LET DELTD  = THE ACTUAL TIME TAKEN TO GIVE CONTROL TO 2CADR
047576,000110:                                                                                                  #                THEN DELTD = TS+DELTA T -X +Y +Z +1.05MS* +COUNTERS*
047577,000111:                                                                                                  #                *THE TIME TAKEN BY WAITLIST ITSELF AND THE COUNTER TICKING DURING THIS WAITLIST TIME.
047578,000112:                                                                                                  #        IN SHORT, THE ACTUAL TIME TO RETURN CONTROL TO A 2CADR IS AUGMENTED BY THE TIME TO SET UP THE TASK'S
047579,000113:                                                                                                  #        INTERRUPT, ALL COUNTERS TICKING, THE T3RUPT PROCESSING TIME, THE WAITLIST PROCESSING TIME AND THE POSSIBILITY
047580,000114:                                                                                                  #        OF OTHER TASKS INHIBITING THE INTERRUPT.
