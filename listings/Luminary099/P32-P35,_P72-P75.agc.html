Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Luminary 131 (Apollo 13) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 001 OF AGC PROGRAM LMY99 BY NASA 2021112-061
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

026623,000043:                                                                                                  #  PURPOSE
026624,000044: 
026625,000045:                                                                                                  #        (1)     TO CALCULATE PARAMETERS ASSOCIATED WITH THE FOLLOWING
026626,000046:                                                                                                  #                CONCENTRIC FLIGHT PLAN MANEUVERS - THE CO-ELLIPTIC SEQUENCE
026627,000047:                                                                                                  #                INITIATION (CSI) MANEUVER AND THE CONSTANT DELTA ALTITUDE
026628,000048:                                                                                                  #                (CDH) MANEUVER.
026629,000049: 
026630,000050:                                                                                                  #        (2)     TO CALCULATE THESE PARAMETERS BASED UPON MANEUVER DATA
026631,000051:                                                                                                  #                APPROVED AND KEYED INTO THE DSKY BY THE ASTRONAUT.
026632,000052: 
026633,000053:                                                                                                  #        (3)     TO DISPLAY TO THE ASTRONAUT AND THE GROUND DEPENDENT VARIABLES
026634,000054:                                                                                                  #                ASSOCIATED WITH THE CONCENTRIC FLIGNT PLAN MANEUVERS FOR
026635,000055:                                                                                                  #                APPROVAL BY THE ASTRONAUT/GROUND.
026636,000056: 
026637,000057:                                                                                                  #        (4)     TO STORE THE CSI TARGET PARAMETERS FOR USE BY THE DESIRED
026638,000058:                                                                                                  #                THRUSTING PROGRAM.
026639,000059: 
026640,000060:                                                                                                  #  ASSUMPTIONS
026641,000061: 
026642,000062:                                                                                                  #        (1)     AT A SELECTED TPI TIME THE LINE OF SIGHT BETWEEN THE ACTIVE
026643,000063:                                                                                                  #                AND PASSIVE VEHICLES IS SELECTED TO BE A PRESCRIBED ANGLE (E)
026644,000064:                                                                                                  #                FROM THE HORIZONTAL PLANE DEFINED BY THE ACTIVE VEHICLE
026645,000065:                                                                                                  #                POSITION.
026646,000066: 
026647,000067:                                                                                                  #        (2)     THE TIME BETWEEN CSI IGNITION AND CDH IGNITION MUST BE
026648,000068:                                                                                                  #                COMPUTED TO BE GREATER THAN 10 MINUTES FOR SUCCESSFUL
026649,000069:                                                                                                  #                COMPLETION OF THE PROGRAM.
026650,000070: 
026651,000071:                                                                                                  #        (3)     THE TIME BETWEEN CDH IGNITION AND TPI IGNITION MUST BE
026652,000072:                                                                                                  #                COMPUTED TO BE GREATER THAN 10 MINUTES FOR SUCCESSFUL
026653,000073:                                                                                                  #                COMPLETION OF THE PROGRAM.
026654,000074: 
026655,000075:                                                                                                  #        (4)     CDH DELTA V IS SELECTED TO MINIMIZE THE VARIATION OF THE
026656,000076:                                                                                                  #                ALTITUDE DIFFERENCE BETWEEN THE ORBITS.
026657,000077: 
026658,000078:                                                                                                  #        (5)     CSI BURN IS DEFINED SUCH THAT THE IMPULSIVE DELTA V IS IN THE
026659,000079:                                                                                                  #                HORIZONTAL PLANE DEFINED BY THE ACTIVE VEHICLE POSITION AT CSI
026660,000080:                                                                                                  #                IGNITION.
026661,000081: 
026662,000082:                                                                                                  #        (6)     THE PERICENTER ALTITUDE OF THE ORBIT FOLLOWING CSI AND CDH
026663,000083:                                                                                                  #                MUST BE GREATER THAN 35,000 FT (LUNAR ORBIT) OR 85 NM (EARTH
026664,000084:                                                                                                  #                ORBIT) FOR SUCCESSFUL COMPLETION OF THIS PROGRAM.
026665,000085: 
026666,000086:                                                                                                  #        (7)     THE CSI AND CDH MANEUVERS ARE ORIGINALLY ASSUMED TO BE
026667,000087:                                                                                                  #                PARALLEL TO THE PLANE OF THE CSM ORBIT.  HOWEVER CREW

026681,000101:                                                                                                  #                OR CSM STATE VECTORS FOR THIS PROGRAM.  IF RADAR USE IS

026686,000106:                                                                                                  #                MARK COUNTER IS ZEROED BY THE SELECTION OF P20 AND AFTER EACH
026687,000107:                                                                                                  #                THRUSTING MANEUVER.
026688,000108: 
026689,000109:                                                                                                  #        (11)    THE ISS NEED NOT BE ON TO COMPLETE THIS PROGRAM.
026690,000110:

026692,000112:

026694,000114:                                                                                                  #                        DOING RENDEZVOUS THRUSTING MANEUVERS TO THE PROGRAM WHICH

026699,000119:                                                                                                  #                        COMPLETED THE FINAL MANEUVER COMPUTATION AND DISPLAY
026700,000120:                                                                                                  #                        CYCLE.

026703,000123:                                                                                                  #                        STEERING REQUIRED FOR EXECUTION OF THIS MANEUVER BY THE
026704,000124:                                                                                                  #                        THRUSTING PROGRAM SELECTED AFTER COMPLETION OF THIS
026705,000125:                                                                                                  #                        PROGRAM.
026706,000126:

026708,000128:                                                                                                  #                A THRUSTING MANEUVER.
