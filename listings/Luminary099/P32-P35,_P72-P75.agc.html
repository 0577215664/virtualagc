Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Luminary 131 (Apollo 13) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 001 OF AGC PROGRAM LMY99 BY NASA 2021112-061
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

026544,000035:                                                                                                  #  COELLIPTIC SEQUENCE INITIATION (CSI) PROGRAMS (P32 AND P72)
026545,000036: 
026546,000037:                                                                                                  #  MOD NO -1             LOG SECTION - P32-P35, P72-P75

026548,000039: 
026549,000040:                                                                                                  #  PURPOSE
026550,000041: 
026551,000042:                                                                                                  #        (1)     TO CALCULATE PARAMETERS ASSOCIATED WITH THE FOLLOWING
026552,000043:                                                                                                  #                CONCENTRIC FLIGHT PLAN MANEUVERS - THE CO-ELLIPTIC SEQUENCE
026553,000044:                                                                                                  #                INITIATION (CSI) MANEUVER AND THE CONSTANT DELTA ALTITUDE
026554,000045:                                                                                                  #                (CDH) MANEUVER.
026555,000046: 
026556,000047:                                                                                                  #        (2)     TO CALCULATE THESE PARAMETERS BASED UPON MANEUVER DATA
026557,000048:                                                                                                  #                APPROVED AND KEYED INTO THE DSKY BY THE ASTRONAUT.
026558,000049: 
026559,000050:                                                                                                  #        (3)     TO DISPLAY TO THE ASTRONAUT AND THE GROUND DEPENDENT VARIABLES
026560,000051:                                                                                                  #                ASSOCIATED WITH THE CONCENTRIC FLIGNT PLAN MANEUVERS FOR
026561,000052:                                                                                                  #                APPROVAL BY THE ASTRONAUT/GROUND.
026562,000053: 
026563,000054:                                                                                                  #        (4)     TO STORE THE CSI TARGET PARAMETERS FOR USE BY THE DESIRED
026564,000055:                                                                                                  #                THRUSTING PROGRAM.
026565,000056: 
026566,000057:                                                                                                  #  ASSUMPTIONS
026567,000058: 
026568,000059:                                                                                                  #        (1)     AT A SELECTED TPI TIME THE LINE OF SIGHT BETWEEN THE ACTIVE
026569,000060:                                                                                                  #                AND PASSIVE VEHICLES IS SELECTED TO BE A PRESCRIBED ANGLE (E)
026570,000061:                                                                                                  #                FROM THE HORIZONTAL PLANE DEFINED BY THE ACTIVE VEHICLE
026571,000062:                                                                                                  #                POSITION.
026572,000063: 
026573,000064:                                                                                                  #        (2)     THE TIME BETWEEN CSI IGNITION AND CDH IGNITION MUST BE
026574,000065:                                                                                                  #                COMPUTED TO BE GREATER THAN 10 MINUTES FOR SUCCESSFUL
026575,000066:                                                                                                  #                COMPLETION OF THE PROGRAM.
026576,000067: 
026577,000068:                                                                                                  #        (3)     THE TIME BETWEEN CDH IGNITION AND TPI IGNITION MUST BE
026578,000069:                                                                                                  #                COMPUTED TO BE GREATER THAN 10 MINUTES FOR SUCCESSFUL
026579,000070:                                                                                                  #                COMPLETION OF THE PROGRAM.
026580,000071: 
026581,000072:                                                                                                  #        (4)     CDH DELTA V IS SELECTED TO MINIMIZE THE VARIATION OF THE
026582,000073:                                                                                                  #                ALTITUDE DIFFERENCE BETWEEN THE ORBITS.
026583,000074: 
026584,000075:                                                                                                  #        (5)     CSI BURN IS DEFINED SUCH THAT THE IMPULSIVE DELTA V IS IN THE
026585,000076:                                                                                                  #                HORIZONTAL PLANE DEFINED BY THE ACTIVE VEHICLE POSITION AT CSI
026586,000077:                                                                                                  #                IGNITION.
026587,000078: 
026588,000079:                                                                                                  #        (6)     THE PERICENTER ALTITUDE OF THE ORBIT FOLLOWING CSI AND CDH
026589,000080:                                                                                                  #                MUST BE GREATER THAN 35,000 FT (LUNAR ORBIT) OR 85 NM (EARTH
026590,000081:                                                                                                  #                ORBIT) FOR SUCCESSFUL COMPLETION OF THIS PROGRAM.
026591,000082: 
026592,000083:                                                                                                  #        (7)     THE CSI AND CDH MANEUVERS ARE ORIGINALLY ASSUMED TO BE
026593,000084:                                                                                                  #                PARALLEL TO THE PLANE OF THE CSM ORBIT.  HOWEVER CREW

026596,000087:                                                                                                  #                OUT-OF-PLANE CSI MANEUVER.

026604,000095:                                                                                                  #                EXCEPT FOR CONCENTRIC FLIGHT PLAN MANEUVER SEQUENCES.
026605,000096: 
026606,000097:                                                                                                  #        (10)    THE RENDEZVOUS RADAR MAY OR MAY NOT BE USED TO UPDATE THE LM
026607,000098:                                                                                                  #                OR CSM STATE VECTORS FOR THIS PROGRAM.  IF RADAR USE IS
026608,000099:                                                                                                  #                DESIRED THE RADAR WAS TURNED ON AND LOCKED ON THE CSM BY
026609,000100:                                                                                                  #                PREVIOUS SELECTION OF P20.  RADAR SIGHTING MARKS WILL BE MADE

026611,000102:                                                                                                  #                TRACK AND UPDATE FLAGS (SEE P20).  THE RENDEZVOUS TRACKING
026612,000103:                                                                                                  #                MARK COUNTER IS ZEROED BY THE SELECTION OF P20 AND AFTER EACH
026613,000104:                                                                                                  #                THRUSTING MANEUVER.
026614,000105: 
026615,000106:                                                                                                  #        (11)    THE ISS NEED NOT BE ON TO COMPLETE THIS PROGRAM.
026616,000107: 
026617,000108:                                                                                                  #        (12)    THE OPERATION OF THE PROGRAM UTILIZES THE FOLLOWING FLAGS -
026618,000109: 
026619,000110:                                                                                                  #                        ACTIVE VEHICLE FLAG - DESIGNATES THE VEHICLE WHICH IS
026620,000111:                                                                                                  #                        DOING RENDEZVOUS THRUSTING MANEUVERS TO THE PROGRAM WHICH
026621,000112:                                                                                                  #                        CALCULATES THE MANEUVER PARAMETERS.  SET AT THE START OF
026622,000113:                                                                                                  #                        EACH RENDEZVOUS PRE-THRUSTING PROGRAM.
026623,000114: 
026624,000115:                                                                                                  #                        FINAL FLAG - SELECTS FINAL PROGRAM DISPLAYS AFTER CREW HAS
026625,000116:                                                                                                  #                        COMPLETED THE FINAL MANEUVER COMPUTATION AND DISPLAY
026626,000117:                                                                                                  #                        CYCLE.
026627,000118: 
026628,000119:                                                                                                  #                        EXTERNAL DELTA V STEERING FLAG - DESIGNATES THE TYPE OF
026629,000120:                                                                                                  #                        STEERING REQUIRED FOR EXECUTION OF THIS MANEUVER BY THE
026630,000121:                                                                                                  #                        THRUSTING PROGRAM SELECTED AFTER COMPLETION OF THIS
026631,000122:                                                                                                  #                        PROGRAM.
026632,000123: 
026633,000124:                                                                                                  #        (13)    IT IS NORMALLY REQUIRED THAT THE ISS BE ON FOR 1 HOUR PRIOR TO
026634,000125:                                                                                                  #                A THRUSTING MANEUVER.
026635,000126: 
026636,000127:                                                                                                  #        (14)    THIS PROGRAM IS SELECTED BY THE ASTRONAUT BY DSKY ENTRY -
026637,000128: 
026638,000129:                                                                                                  #                        P32 IF THIS VEHICLE IS ACTIVE VEHICLE.
