Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Luminary 131 (Apollo 13) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 001 OF AGC PROGRAM LMY99 BY NASA 2021112-061
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

026575,000042:                                                                                                  #  PURPOSE
026576,000043: 
026577,000044:                                                                                                  #        (1)     TO CALCULATE PARAMETERS ASSOCIATED WITH THE FOLLOWING
026578,000045:                                                                                                  #                CONCENTRIC FLIGHT PLAN MANEUVERS - THE CO-ELLIPTIC SEQUENCE
026579,000046:                                                                                                  #                INITIATION (CSI) MANEUVER AND THE CONSTANT DELTA ALTITUDE
026580,000047:                                                                                                  #                (CDH) MANEUVER.
026581,000048: 
026582,000049:                                                                                                  #        (2)     TO CALCULATE THESE PARAMETERS BASED UPON MANEUVER DATA
026583,000050:                                                                                                  #                APPROVED AND KEYED INTO THE DSKY BY THE ASTRONAUT.
026584,000051: 
026585,000052:                                                                                                  #        (3)     TO DISPLAY TO THE ASTRONAUT AND THE GROUND DEPENDENT VARIABLES
026586,000053:                                                                                                  #                ASSOCIATED WITH THE CONCENTRIC FLIGNT PLAN MANEUVERS FOR
026587,000054:                                                                                                  #                APPROVAL BY THE ASTRONAUT/GROUND.
026588,000055: 
026589,000056:                                                                                                  #        (4)     TO STORE THE CSI TARGET PARAMETERS FOR USE BY THE DESIRED
026590,000057:                                                                                                  #                THRUSTING PROGRAM.
026591,000058: 
026592,000059:                                                                                                  #  ASSUMPTIONS
026593,000060: 
026594,000061:                                                                                                  #        (1)     AT A SELECTED TPI TIME THE LINE OF SIGHT BETWEEN THE ACTIVE
026595,000062:                                                                                                  #                AND PASSIVE VEHICLES IS SELECTED TO BE A PRESCRIBED ANGLE (E)
026596,000063:                                                                                                  #                FROM THE HORIZONTAL PLANE DEFINED BY THE ACTIVE VEHICLE
026597,000064:                                                                                                  #                POSITION.
026598,000065: 
026599,000066:                                                                                                  #        (2)     THE TIME BETWEEN CSI IGNITION AND CDH IGNITION MUST BE
026600,000067:                                                                                                  #                COMPUTED TO BE GREATER THAN 10 MINUTES FOR SUCCESSFUL
026601,000068:                                                                                                  #                COMPLETION OF THE PROGRAM.
026602,000069: 
026603,000070:                                                                                                  #        (3)     THE TIME BETWEEN CDH IGNITION AND TPI IGNITION MUST BE
026604,000071:                                                                                                  #                COMPUTED TO BE GREATER THAN 10 MINUTES FOR SUCCESSFUL
026605,000072:                                                                                                  #                COMPLETION OF THE PROGRAM.
026606,000073: 
026607,000074:                                                                                                  #        (4)     CDH DELTA V IS SELECTED TO MINIMIZE THE VARIATION OF THE
026608,000075:                                                                                                  #                ALTITUDE DIFFERENCE BETWEEN THE ORBITS.
026609,000076: 
026610,000077:                                                                                                  #        (5)     CSI BURN IS DEFINED SUCH THAT THE IMPULSIVE DELTA V IS IN THE
026611,000078:                                                                                                  #                HORIZONTAL PLANE DEFINED BY THE ACTIVE VEHICLE POSITION AT CSI
026612,000079:                                                                                                  #                IGNITION.
026613,000080: 
026614,000081:                                                                                                  #        (6)     THE PERICENTER ALTITUDE OF THE ORBIT FOLLOWING CSI AND CDH
026615,000082:                                                                                                  #                MUST BE GREATER THAN 35,000 FT (LUNAR ORBIT) OR 85 NM (EARTH
026616,000083:                                                                                                  #                ORBIT) FOR SUCCESSFUL COMPLETION OF THIS PROGRAM.
026617,000084: 
026618,000085:                                                                                                  #        (7)     THE CSI AND CDH MANEUVERS ARE ORIGINALLY ASSUMED TO BE
026619,000086:                                                                                                  #                PARALLEL TO THE PLANE OF THE CSM ORBIT.  HOWEVER CREW
