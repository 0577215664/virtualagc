Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Luminary 131 (Apollo 13) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 001 OF AGC PROGRAM LMY99 BY NASA 2021112-061
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

057151,000033:                                                                                                  #  DISPLAYS CAN BE CLASSIFIED INTO THE FOLLOWING CATEGORIES --
057152,000034:                                                                                                  #        1.  PRIORITY DISPLAYS -- DISPLAYS WHICH TAKE PRIORITY OVER ALL OTHER DISPLAYS.  USUALLY THESE DISPLAYS ARE SENT
057153,000035:                                                                                                  #            OUT UNDER CRITICAL ALARM CONDITIONS.
057154,000036:                                                                                                  #        2.  EXTENDED VERB DISPLAYS -- ALL EXTENDED VERBS AND MARK ROUTINES SHOULD USE EXTENDED VERB (MARK) DISPLAYS.
057155,000037:                                                                                                  #        3.  NORMAL DISPLAYS -- ALL MISSION PROGRAM DISPLAYS WHICH INTERFACE WITH THE ASTRONAUT DURING THE NORMAL
057156,000038:                                                                                                  #            SEQUENCE OF EVENTS.
057157,000039:                                                                                                  #        4.  MISC. DISPLAYS -- ALL DISPLAYS NOT HANDLED BY THE DISPLAY INTERFACE ROUTINES.  THESE INCLUDE SUCH DISPLAYS AS
057158,000040:                                                                                                  #            MM DISPLAYS AND SPECIAL PURPOSE DISPLAYS HANDLED BY PINBALL.
057159,000041:                                                                                                  #        5.  ASTRONAUT INITIATED DISPLAYS -- ALL DISPLAYS INITIATED EXTERNALLY.
057160,000042: 
057161,000043:                                                                                                  #  THE FOLLOWING TERMS ARE USED TO DESCRIBE THE STATUS OF DISPLAYS --
057162,000044:                                                                                                  #        1.  ACTIVE -- THE DISPLAY WHICH IS (1) BEING DISPLAYED TO THE ASTRONAUT AND WAITING FOR A RESPONSE OR
057163,000045:                                                                                                  #            (2) WAITING FIRST IN LINE FOR THE ASTRONAUT TO FINISH USING THE DSKY OR (3) BEING DISPLAYED ON THE DSKY
057164,000046:                                                                                                  #            BUT NOT WAITING FOR A RESPONSE.
057165,000047:                                                                                                  #        2.  INACTIVE -- A DISPLAY WHICH HAS (1) BEEN ACTIVE BUT WAS INTERRUPTED BY A DISPLAY OF HIGHER PRIORITY,
057166,000048:                                                                                                  #            (2) BEEN PUT INTO THE WAITING LIST AT TIME IT WAS REQUESTED DUE TO THE FACT A HIGHER PRIORITY DISPLAY
057167,000049:                                                                                                  #            WAS ALREADY DOING, (3) BEEN INTERRUPTED BY THE ASTRONAUT (CALLED A PINBRANCH CONDITION, SINCE THIS TYPE
057168,000050:                                                                                                  #            OF INACTIVE DISPLAY IS USUALLY REACTIVATED ONLY BY PINBALL) OR (4) A DISPLAY WHICH HAS FINISHED BUT STILL
057169,000051:                                                                                                  #            HAS INFO SAVED FOR RESTART PURPOSES.
057170,000052: 
057171,000053:                                                                                                  #  DISPLAY PRIORITIES WORK AS FOLLOWS --
057172,000054:                                                                                                  #        INTERRUPTS --
057173,000055:                                                                                                  #                1.  THE ASTRONAUT CAN INTERRUPT ANY DISPLAY WITH AN EXTERNAL DISPLAY REQUEST.
057174,000056:                                                                                                  #                2.  INTERNAL DISPLAYS CAN NOT BE SENT OUT WHEN THE ASTRONAUT IS USING THE DSKY.
057175,000057:                                                                                                  #                3.  PRIORITY DISPLAYS INTERRUPT ALL OTHER TYPES OF INTERNAL DISPLAYS.  A PRIORITY DISPLAY INTERRUPTING ANOTHER
057176,000058:                                                                                                  #                    PRIORITY DISPLAY WILL CAUSE AN ABORT UNLESS BIT14 IS SET FOR THE LINUS ROUTINE.
057177,000059:                                                                                                  #                4.  A MARK DISPLAY INTERRUPTS ANY NORMAL DISPLAY.
057178,000060:                                                                                                  #                5.  A MARK THAT INTERRUPTS A MARK COMPLETELY REPLACES IT.
057179,000061: 
057180,000062:                                                                                                  #        ORDER OF WAITING DISPLAYS --
057181,000063:                                                                                                  #                1.  ASTRONAUT
057182,000064:                                                                                                  #                2.  PRIORITY
057183,000065:                                                                                                  #                3.  INTERRUPTED MARK
057184,000066:                                                                                                  #                4.  INTERRUPTED NORMAL
057185,000067:                                                                                                  #                5.  MARK TO BE REQUESTED (SEE DESCRIPTION OF ENDMARK)
057186,000068:                                                                                                  #                6.  MARK WAITING
057187,000069:                                                                                                  #                7.  NORMAL WAITING

057190,000072:                                                                                                  #  THE DISPLAY ROUTINES ARE INTENDED TO SERVE AS AN INTERFACE BETWEEN THE USER AND PINBALL.  THE
057191,000073:                                                                                                  #  FOLLOWING STATEMENTS CAN BE MADE ABOUT NORMAL DISPLAYS AND PRIORITY DISPLAYS (A DESCRIPTION OF MARK ROUTINES
057192,000074:                                                                                                  #  WILL FOLLOW LATER):
057193,000075:                                                                                                  #        1.  ALL ROUTINES THAT END IN R HAVE AN IMMEDIATE RETURN TO THE USER.  FOR ALL FLASHING DISPLAYS THIS RETURN
057194,000076:                                                                                                  #            IS TO THE USER'S CALL CADR +4.  FOR THE ONLY NON-FLASHING IMMEDIATE RETURN DISPLAY (GODSPR) THIS RETURN
057195,000077:                                                                                                  #            IS TO THE USER'S CALLING LOC +1.
057196,000078:                                                                                                  #        2.  ALL ROUTINES NOT ENDING IN R DO NOT DO AN IMMEDIATE RETURN TO THE USER.
057197,000079:                                                                                                  #        3.  ALL ROUTINES THAT END IN R START A SEPARATE JOB (MAKEPLAY) WITH USER'S JOB PRIORITY.
057198,000080:                                                                                                  #        4.  ALL ROUTINES NOT ENDING IN R BRANCH DIRECTLY TO MAKEPLAY WHICH MAKES THESE DISPLAYS A PART OF THE
057199,000081:                                                                                                  #            USER'S JOB.
057200,000082:                                                                                                  #        5.  ALL DISPLAY ROUTINES ARE CALLED VIA BANKCALL.
057201,000083:                                                                                                  #        6.  TO RESTART A DISPLAY THE USER WILL GENERALLY USE A PHASE OF ONE WITH DESIRED RESTART GROUP (SEE
057202,000084:                                                                                                  #            DESCRIPTION OF RESTARTS).
057203,000085:                                                                                                  #        7.  ALL FLASHING DISPLAYS HAVE 3 RETURNS TO THE USER FROM ASTRONAUT RESPOSES.  A TERMINATE (V34) BRANCHES
057204,000086:                                                                                                  #            TO THE USER'S CALL CADR +1.  A PROCEED (V33) BRANCHES TO THE USER'S CALL CADR +2.  AN ENTER OR RECYCLE
057205,000087:                                                                                                  #            (V32) BRANCHES TO THE USER'S CALL CADR +3.
057206,000088:                                                                                                  #        8.  ALL ROUTINES MUST BE USED UNDER EXECUTIVE CONTROL
