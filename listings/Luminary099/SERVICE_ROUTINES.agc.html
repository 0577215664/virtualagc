Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Luminary 131 (Apollo 13) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 001 OF AGC PROGRAM LMY99 BY NASA 2021112-061
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

058593,000055:                                                                                                  #  UPFLAG AND DOWNFLAG ARE ENTIRELY GENERAL FLAG SETTING AND CLEARING SUBROUTINES.  USING THEM, WHETHER OR
058594,000056:                                                                                                  #  NOT IN INTERRUPT, ONE MAY SET OR CLEAR ANY SINGLE, NAMED BIT IN ANY ERASABLE REGISTER, SUBJECT OF COURSE TO
058595,000057:                                                                                                  #  EBANK SETTING.  A "NAMED" BIT, AS THE WORD IS USED HERE, IS ANY BIT WITH A NAME FORMALLY ASSIGNED BY THE YUL
058596,000058:                                                                                                  #  ASSEMBLER.
058597,000059: 
058598,000060:                                                                                                  #  AT PRESENT THE ONLY NAMED BITS ARE THOSE IN THE FLAGWORDS.  ASSEMBLER CHANGES WILL MAKE IT POSSIBLE TO
058599,000061:                                                                                                  #  NAME ANY BIT IN ERASABLE MEMORY.
058600,000062: 
058601,000063:                                                                                                  #  CALLING SEQUENCES ARE AS FOLLOWS :-
058602,000064:                                                                                                  #                TC      UPFLAG                  TC      DOWNFLAG
058603,000065:                                                                                                  #                ADRES   NAME OF FLAG            ADRES   NAME OF FLAG
058604,000066: 
058605,000067:                                                                                                  #  RETURN IS TO THE LOCATION FOLLOWING THE "ADRES" ABOUT .58 MS AFTER THE "TC".
058606,000068:                                                                                                  #  UPON RETURN A CONTAINS THE CURRENT FLAGWRD SETTING.

058709,000171:                                                                                                  #  GENTRAN, A BLOCK TRANSFER ROUTINE
058710,000172:                                                                                                  #  WRITTEN BY D. EYLES
058711,000173:                                                                                                  #  MOD 1 BY KERNAN                               UTILITYM REV 17 11/18/67
058712,000174:                                                                                                  #  MOD 2 BY SCHULENBERG (REMOVE RELINT) SKIPPER REV 4 2/28/68
058713,000175: 
058714,000176:                                                                                                  #  THIS ROUTINE IS USEFUL FOR TRANSFERING N CONSECUTIVE ERASABLE OR FIXED QUANTITIES TO SOME OTHER N
058715,000177:                                                                                                  #  CONSECUTIVE ERASABLE LOCATIONS.  IF BOTH BLOCKS OF DATA ARE IN SWITCHABLE EBANKS, THEY MUST BE IN THE SAME ONE.
058716,000178: 
058717,000179:                                                                                                  #  GENTRAN IS CALLABLE IN A JOB AS WELL AS A RUPT.  THE CALLING SEQUENCE IS:
058718,000180:                                                                                                  #        I       CA      N-1             # # OF QUANTITIES MINUS ONE.
058719,000181:                                                                                                  #        I +1    TC      GENTRAN         # IN FIXED-FIXED.
058720,000182:                                                                                                  #        I +2    ADRES   L               # STARTING ADRES OF DATA TO BE MOVED.
058721,000183:                                                                                                  #        I +3    ADRES   M               # STARTING ADRES OF DUPLICATION BLOCK.
058722,000184:                                                                                                  #        I +4                            # RETURNS HERE.
058723,000185: 
058724,000186:                                                                                                  #  GENTRAN TAKES 25 MCT'S (300 MICROSECONDS) PER ITEM + 5 MCT'S (60 MICS) FOR ENTERING AND EXITING.
058725,000187:                                                                                                  #  A, L, AND ITEMP1 ARE NOT PRESERVED.
058726,000188: 
058727,000189:    5544                                           BLOCK    02                                    
058728,000190:    4000                                           SETLOC   FFTAG4                                
058729,000191:    4000                                           BANK                                           
058730,000192: 
058731,000193:    5544     0061                                  EBANK=   ITEMP1                                
058732,000194: 
058733,000195:    5544                                           COUNT*   $$/TRAN                               
058734,000196: 
058735,000197:    5544           00004        GENTRAN            INHINT                                         
058736,000198:    5545           54061                           TS       ITEMP1                                #  SAVE N-1
058737,000199:    5546           50002                           INDEX    Q                                     #  C(Q) = ADRES L.
058738,000200:    5547           60000                           AD       0                                     #  ADRES (L + N - 1).
058739,000201:    5550           50000                           INDEX    A                                     
058740,000202:    5551           30000                           CA       0                                     #  C(ABOVE).
058741,000203:    5552           54001                           TS       L                                     #  SAVE DATA.
058742,000204:    5553           30061                           CA       ITEMP1                                
058743,000205:    5554           50002                           INDEX    Q                                     
058744,000206:    5555           60001                           AD       1                                     #  ADRES (M + N - 1).
058745,000207:    5556           50000                           INDEX    A
