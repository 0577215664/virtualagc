Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Luminary 131 (Apollo 13) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 001 OF AGC PROGRAM LMY99 BY NASA 2021112-061
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

058634,000055:                                                                                                  #  UPFLAG AND DOWNFLAG ARE ENTIRELY GENERAL FLAG SETTING AND CLEARING SUBROUTINES.  USING THEM, WHETHER OR
058635,000056:                                                                                                  #  NOT IN INTERRUPT, ONE MAY SET OR CLEAR ANY SINGLE, NAMED BIT IN ANY ERASABLE REGISTER, SUBJECT OF COURSE TO
058636,000057:                                                                                                  #  EBANK SETTING.  A "NAMED" BIT, AS THE WORD IS USED HERE, IS ANY BIT WITH A NAME FORMALLY ASSIGNED BY THE YUL
058637,000058:                                                                                                  #  ASSEMBLER.
058638,000059: 
058639,000060:                                                                                                  #  AT PRESENT THE ONLY NAMED BITS ARE THOSE IN THE FLAGWORDS.  ASSEMBLER CHANGES WILL MAKE IT POSSIBLE TO
058640,000061:                                                                                                  #  NAME ANY BIT IN ERASABLE MEMORY.
058641,000062: 
058642,000063:                                                                                                  #  CALLING SEQUENCES ARE AS FOLLOWS :-
058643,000064:                                                                                                  #                TC      UPFLAG                  TC      DOWNFLAG
058644,000065:                                                                                                  #                ADRES   NAME OF FLAG            ADRES   NAME OF FLAG
058645,000066: 
058646,000067:                                                                                                  #  RETURN IS TO THE LOCATION FOLLOWING THE "ADRES" ABOUT .58 MS AFTER THE "TC".
058647,000068:                                                                                                  #  UPON RETURN A CONTAINS THE CURRENT FLAGWRD SETTING.

058750,000171:                                                                                                  #  GENTRAN, A BLOCK TRANSFER ROUTINE
058751,000172:                                                                                                  #  WRITTEN BY D. EYLES
058752,000173:                                                                                                  #  MOD 1 BY KERNAN                               UTILITYM REV 17 11/18/67
058753,000174:                                                                                                  #  MOD 2 BY SCHULENBERG (REMOVE RELINT) SKIPPER REV 4 2/28/68
058754,000175: 
058755,000176:                                                                                                  #  THIS ROUTINE IS USEFUL FOR TRANSFERING N CONSECUTIVE ERASABLE OR FIXED QUANTITIES TO SOME OTHER N
058756,000177:                                                                                                  #  CONSECUTIVE ERASABLE LOCATIONS.  IF BOTH BLOCKS OF DATA ARE IN SWITCHABLE EBANKS, THEY MUST BE IN THE SAME ONE.
058757,000178: 
058758,000179:                                                                                                  #  GENTRAN IS CALLABLE IN A JOB AS WELL AS A RUPT.  THE CALLING SEQUENCE IS:
058759,000180:                                                                                                  #        I       CA      N-1             # # OF QUANTITIES MINUS ONE.
058760,000181:                                                                                                  #        I +1    TC      GENTRAN         # IN FIXED-FIXED.
058761,000182:                                                                                                  #        I +2    ADRES   L               # STARTING ADRES OF DATA TO BE MOVED.
058762,000183:                                                                                                  #        I +3    ADRES   M               # STARTING ADRES OF DUPLICATION BLOCK.
058763,000184:                                                                                                  #        I +4                            # RETURNS HERE.
058764,000185: 
058765,000186:                                                                                                  #  GENTRAN TAKES 25 MCT'S (300 MICROSECONDS) PER ITEM + 5 MCT'S (60 MICS) FOR ENTERING AND EXITING.
058766,000187:                                                                                                  #  A, L, AND ITEMP1 ARE NOT PRESERVED.
