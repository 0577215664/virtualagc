Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Luminary 131 (Apollo 13) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 001 OF AGC PROGRAM LMY99 BY NASA 2021112-061
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

058766,000057:                                                                                                  #  UPFLAG AND DOWNFLAG ARE ENTIRELY GENERAL FLAG SETTING AND CLEARING SUBROUTINES.  USING THEM, WHETHER OR
058767,000058:                                                                                                  #  NOT IN INTERRUPT, ONE MAY SET OR CLEAR ANY SINGLE, NAMED BIT IN ANY ERASABLE REGISTER, SUBJECT OF COURSE TO
058768,000059:                                                                                                  #  EBANK SETTING.  A "NAMED" BIT, AS THE WORD IS USED HERE, IS ANY BIT WITH A NAME FORMALLY ASSIGNED BY THE YUL
058769,000060:                                                                                                  #  ASSEMBLER.
058770,000061: 
058771,000062:                                                                                                  #  AT PRESENT THE ONLY NAMED BITS ARE THOSE IN THE FLAGWORDS.  ASSEMBLER CHANGES WILL MAKE IT POSSIBLE TO
058772,000063:                                                                                                  #  NAME ANY BIT IN ERASABLE MEMORY.
058773,000064: 
058774,000065:                                                                                                  #  CALLING SEQUENCES ARE AS FOLLOWS :-
058775,000066:                                                                                                  #                TC      UPFLAG                  TC      DOWNFLAG
058776,000067:                                                                                                  #                ADRES   NAME OF FLAG            ADRES   NAME OF FLAG
058777,000068: 
058778,000069:                                                                                                  #  RETURN IS TO THE LOCATION FOLLOWING THE "ADRES" ABOUT .58 MS AFTER THE "TC".
058779,000070:                                                                                                  #  UPON RETURN A CONTAINS THE CURRENT FLAGWRD SETTING.

058882,000173:                                                                                                  #  GENTRAN, A BLOCK TRANSFER ROUTINE.
058883,000174:                                                                                                  #  WRITTEN BY D. EYLES
058884,000175:                                                                                                  #  MOD 1 BY KERNAN                               UTILITYM REV 17 11/18/67
058885,000176:                                                                                                  #  MOD 2 BY SCHULENBERG (REMOVE RELINT) SKIPPER REV 4 2/28/68
058886,000177: 
058887,000178:                                                                                                  #  THIS ROUTINE IS USEFULL FOR TRANSFERING N CONSECUTIVE ERASABLE OR FIXED QUANTITIES TO SOME OTHER N
058888,000179:                                                                                                  #  CONSECUTIVE ERASABLE LOCATIONS.  IF BOTH BLOCKS OF DATA ARE IN SWITCHABLE EBANKS, THEY MUST BE IN THE SAME ONE.
058889,000180: 
058890,000181:                                                                                                  #  GENTRAN IS CALLABLE IN A JOB AS WELL AS A RUPT.  THE CALLING SEQUENCE IS:
058891,000182:                                                                                                  #        I       CA      N-1             # OF QUANTITIES MINUS ONE.
058892,000183:                                                                                                  #        I +1    TC      GENTRAN         IN FIXED-FIXED.
058893,000184:                                                                                                  #        I +2    ADRES   L               STARTING ADRES OF DATA TO BE MOVED.
058894,000185:                                                                                                  #        I +3    ADRES   M               STARTING ADRES OF DUPLICATION BLOCK.
058895,000186:                                                                                                  #        I +4                            RETURNS HERE.
058896,000187: 
058897,000188:                                                                                                  #  GENTRAN TAKES 25 MCT'S (300 MICROSECONDS) PER ITEM + 5 MCT'S (60 MICS) FOR ENTERING AND EXITING.
058898,000189:                                                                                                  #  A, L AND ITEMP1 ARE NOT PRESERVED.
058899,000190: 
058900,000191:    5544                                           BLOCK    02                                    
058901,000192:    4000                                           SETLOC   FFTAG4                                
058902,000193:    4000                                           BANK                                           
058903,000194: 
058904,000195:    5544     0061                                  EBANK=   ITEMP1                                
058905,000196: 
058906,000197:    5544                                           COUNT*   $$/TRAN                               
058907,000198: 
058908,000199:    5544           00004        GENTRAN            INHINT                                         
058909,000200:    5545           54061                           TS       ITEMP1                                #  SAVE N-1.
058910,000201:    5546           50002                           INDEX    Q                                     #  C(Q) = ADRES L.
058911,000202:    5547           60000                           AD       0                                     #  ADRES (L + N - 1).
058912,000203:    5550           50000                           INDEX    A                                     
058913,000204:    5551           30000                           CA       0                                     #  C(ABOVE).
058914,000205:    5552           54001                           TS       L                                     #  SAVE DATA.
058915,000206:    5553           30061                           CA       ITEMP1                                
058916,000207:    5554           50002                           INDEX    Q                                     
058917,000208:    5555           60001                           AD       1                                     #  ADRES (M + N - 1).
