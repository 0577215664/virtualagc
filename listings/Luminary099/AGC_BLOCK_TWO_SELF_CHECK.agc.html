Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Luminary 131 (Apollo 13) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 001 OF AGC PROGRAM LMY99 BY NASA 2021112-061
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

054687,000030:                                                                                                  #  PROGRAM NAME -- SELF-CHECK                    LOG SECTION:  AGC BLOCK TWO SELF-CHECK
054688,000031:                                                                                                  #  MOD NO -- 1                                   ASSEMBLY SUBROUTINE UTILITYM REV 25
054689,000032:                                                                                                  #  MOD BY -- GAUNTT
054690,000033: 
054691,000034:                                                                                                  #  FUNCTIONAL DESCRIPTION
054692,000035:                                                                                                  #        PROGRAM HAS TWO MAIN PARTS.  THE FIRST IS SELF-CHECK WHICH RUNS AS A ZERO PRIORITY JOB WITH NO CORE SET, AS
054693,000036:                                                                                                  #        PART OF THE BACK-UP IDLE LOOP.  THE SECOND IS SHOW-BANKSUM WHICH RUNS AS A REGULAR EXECUTIVE JOB WITH ITS OWN
054694,000037:                                                                                                  #        STARTING VERB.
054695,000038: 
054696,000039:                                                                                                  #        THE PURPOSE OF SELF-CHECK IS TO CHECK OUT VARIOUS PARTS OF THE COMPUTER AS OUTLINED BELOW IN THE OPTIONS.
054697,000040: 
054698,000041:                                                                                                  #        THE PURPOSE OF SHOW-BANKSUM IS TO DISPLAY THE SUM OF EACH BANK, ONE AT A TIME.
054699,000042: 
054700,000043:                                                                                                  #        IN ALL THERE ARE 7 POSSIBLE OPTIONS IN THIS BLOCK II VERSION OF SELF-CHECK.  MORE DETAIL DESCRIPTION MAY BE
054701,000044:                                                                                                  #        FOUND IN E-2065 BLOCK II AGC SELF-CHECK AND SHOW BANKSUM BY EDWIN D. SMALLY DECEMBER 1966, AND ADDENDA 2 AND 3.
054702,000045: 
054703,000046:                                                                                                  #        THE DIFFERENT OPTIONS ARE CONTROLLED BY PUTTING DIFFERENT NUMBERS IN THE SMODE REGISTER (NOUN 27).  BELOW IS
054704,000047:                                                                                                  #        A DESCRIPTION OF WHAT PARTS OF THE COMPUTER THAT ARE CHECKED BY THE OPTIONS, AND THE CORRESPONDING NUMBER, IN
054705,000048:                                                                                                  #        OCTAL, TO LOAD INTO SMODE.
054706,000049:                                                                                                  #                +-4             ERASABLE MEMORY
054707,000050:                                                                                                  #                +-5             FIXED MEMORY
054708,000051:                                                                                                  #                +-1,2,3,6,7,10  EVERYTHING IN OPTIONS 4 AND 5.
054709,000052:                                                                                                  #                -0              SAME AS +-10 UNTIL AN ERROR IS DETECTED.
054710,000053:                                                                                                  #                +0              NO CHECK, PUTS COMPUTER INTO THE BACKUP IDLE LOOP.
054711,000054: 
054712,000055:                                                                                                  #  WARNINGS
054713,000056:                                                                                                  #        USE OF E MEMORY RESERVED FOR SELF-CHECK (EVEN IN IDLE LOOP) AS TEMP STORAGE BY OTHER PROGRAMS IS DANGEROUS.
054714,000057:                                                                                                  #        SMODE SET GREATER THAN OCT 10 PUTS COMPUTER INTO BACKUP IDLE LOOP.
054715,000058: 
054716,000059:                                                                                                  #  CALLING SEQUENCE
054717,000060:                                                                                                  #        TO CALL SELF-CHECK KEY IN
054718,000061:                                                                                                  #                V 21 N 27 E     OPTION NUMBER E
054719,000062:                                                                                                  #        TO CALL SHOW-BANKSUM KEY IN
054720,000063:                                                                                                  #                V 91 E          DISPLAYS FIRST BANK
054721,000064:                                                                                                  #                V 33 E          PROCEED, DISPLAYS NEXT BANK
054722,000065: 
054723,000066:                                                                                                  #  EXIT MODES, NORMAL AND ALARM
054724,000067:                                                                                                  #        SELF-CHECK NORMALLY CONTINUES INDEFINITELY UNLESS THERE IS AN ERROR DETECTED.  IF SO + OPTION NUMBERS PUT
054725,000068:                                                                                                  #        COMPUTER INTO BACKUP IDLE LOOP, - OPTIONS NUMBERS RESTART THE OPTION.
054726,000069: 
054727,000070:                                                                                                  #        THE -0 OPTION PROCEEDS FROM THE LINE FOLLOWING THE LINE WHERE THE ERROR WAS DETECTED.
054728,000071: 
054729,000072:                                                                                                  #        SHOW-BANKSUM PROCEEDS UNTIL A TERMINATE IS KEYED IN (V 34 E).  THE COMPUTER IS PUT INTO THE BACKUP IDLE LOOP.
054730,000073: 
054731,000074:                                                                                                  #  OUTPUT
054732,000075:                                                                                                  #  Page 1285
054733,000076:                                                                                                  #        SELF-CHECK UPON DETECTING AN ERROR LOADS THE SELF-CHECK ALARM CONSTANT (01102) INTO THE FAILREG SET AND
054734,000077:                                                                                                  #        TURNS ON THE ALARM LIGHT.  THE OPERATOR MAY THEN DISPLAY THE THREE FAILREGS BY KEYING IN V 05 N 09 E.  FOR FURTHER
054735,000078:                                                                                                  #        INFORMATION HE MAY KEY IN V 05 N 08 E, THE DSKY DISPLAY IN R1 WILL BE ADDRESS+1 OF WHERE THE ERROR WAS DETECTED,
054736,000079:                                                                                                  #        IN R2 THE BBCON OF SELF-CHECK, AND IN R3 THE TOTAL NUMBER OF ERRORS DETECTED BY SELF-CHECK SINCE THE LAST MAN
054737,000080:                                                                                                  #        INITIATED FRESH START (SLAP1).
054738,000081: 
054739,000082:                                                                                                  #        SHOW-BANKSUM STARTING WTIH BANK 0 DISPLAYS IN R1 THE BANK SUM (A +-NUMBER EQUAL TO THE BANK NUMBER), IN R2
054740,000083:                                                                                                  #        THE BANK NUMBER, AND IN R3 THE BUGGER WORD.
054741,000084: 
054742,000085:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED
054743,000086:                                                                                                  #        ACCOMPLISHED BY FRESH START
054744,000087:                                                                                                  #                SMODE SET TO +0
054745,000088: 
054746,000089:                                                                                                  #  DEBRIS
054747,000090:                                                                                                  #        ALL EXITS FROM THE CHECK OF ERASABLE (ERASCHK) RESTORE ORIGINAL CONTENTS TO REGISTERS UNDER CHECK.
054748,000091:                                                                                                  #        EXCEPTION IS A RESTART.  RESTART THAT OCCURS DURING ERASCHK RESTORES ERASABLE, UNLESS THERE IS EVIDENCE TO DOUBT
054749,000092:                                                                                                  #        E MEMORY, IN WHICH CASE PROGRAM THEN DOES A FRESH START (DOFSTART).
