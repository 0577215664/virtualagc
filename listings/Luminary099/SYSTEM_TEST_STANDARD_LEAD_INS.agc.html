Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Luminary 131 (Apollo 13) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 001 OF AGC PROGRAM LMY99 BY NASA 2021112-061
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

015549,000043:                                                                                                  #  E/BKCALL      FOR DOING BANKCALLS FROM AND RETURNING TO ERASABLE.
015550,000044: 
015551,000045:                                                                                                  #  THIS ROUTINE IS CALLABLE FROM ERASABLE OR FIXED.  LIKE BANKCALL, HOWEVER, SWITCHING BETWEEN S3 AND S4
015552,000046:                                                                                                  #  IS NOT POSSIBLE.
015553,000047: 
015554,000048:                                                                                                  #  THE CALLING SEQUENCE IS:
015555,000049: 
015556,000050:                                                                                                  #        TC      BANKCALL
015557,000051:                                                                                                  #        CADR    E/BKCALL
015558,000052:                                                                                                  #        CADR    ROUTINE         WHERE YOU WANT TO GO IN FIXED.
015559,000053:                                                                                                  #        RETURN HERE FROM DISPLAY TERMINATE, BAD STALL OR TC Q.
015560,000054:                                                                                                  #        RETURN HERE FROM DISPLAY PROCEED OR GOOD RETURN FROM STALL.
015561,000055:                                                                                                  #        RETURN HERE FROM DISPLAY ENTER OR RECYCLE.
015562,000056: 
015563,000057:                                                                                                  #  THIS ROUTINE REQUIRES TWO ERASABLES (EBUF2, +1) IN UNSWITCHED WHICH ARE UNSHARED BY INTERRUPTS AND
015564,000058:                                                                                                  #  OTHER EMEMORY PROGRAMS.
015565,000059: 
015566,000060:                                                                                                  #  A + L ARE PRESERVED THROUGH BANKCALL AND E/BKCALL.
015567,000061: 
015568,000062: 04,2576           52134        E/BKCALL           DXCH     BUF2                                  #  SAVE A,L AND GET DP RETURN.
015569,000063: 04,2577           53170                           DXCH     EBUF2                                 #  SAVE DP RETURN.
015570,000064: 04,2600           25167                           INCR     EBUF2                                 #  RETURN +1 BECAUSE DOUBLE CADR.
015571,000065: 04,2601           30006                           CA       BBANK                                 
015572,000066: 04,2602           75012                           MASK     LOW10                                 #  GET CURRENT EBANK.  (SBANK SOMEDAY)
015573,000067: 04,2603           27170                           ADS      EBUF2      +1                         #  FORM BBCON.  (WAS FBANK)
015574,000068: 04,2604           51167                           NDX      EBUF2

015576,000070: 04,2606           04622                           TC       SWCALL                                #  GO TO ROUTINE, SETTING Q TO SWRETURN
