Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Luminary 131 (Apollo 13) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 001 OF AGC PROGRAM LMY99 BY NASA 2021112-061
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

039575,000032: 
039576,000033:                                                                                                  #  OBJECTS OF MOD:       1.      TO SUPPLY COMMANDED GIMBAL ANGLES FOR NOUN 22.
039577,000034:                                                                                                  #                        2.      TO MAINTAIN CORRECT AND CURRENT THRUST
039578,000035:                                                                                                  #                                DIRECTION DATA IN ALL MODES.  THIS IS DONE BY
039579,000036:                                                                                                  #                                FETCHING FOR THE THRUST DIRECTION FILTER THE
039580,000037:                                                                                                  #                                CDUD'S IN PNGCS-AUTO, THE CDU'S IN ALL OTHER
039581,000038:                                                                                                  #                                MODES.
039582,000039:                                                                                                  #                        3.      TO SUBSTITUDE A STOPRATE FOR THE NORMAL
039583,000040:                                                                                                  #                                AUTOPILOT COMMANDS WHENEVER
039584,000041:                                                                                                  #                                1) NOT IN PNGCS-AUTO, OR
039585,000042:                                                                                                  #                                2) ENGINE IS OFF.
039586,000043: 
039587,000044:                                                                                                  #  FUNCTIONAL DESCRIPTION:
039588,000045: 
039589,000046:                                                                                                  #  FINDCDUW PROVIDES THE INTERFACES BETWEEN THE VARIOUS POWERED FLITE GUIDANCE PROGRAMS
039590,000047:                                                                                                  #  AND THE DIGITAL AUTOPILOT.  THE INPUTS TO FINDCDUW ARE THE THRUST COMMAND VECTOR
039591,000048:                                                                                                  #  AND THE WINDOW COMMAND VECTOR, AND THE OUTPUTS ARE THE GIMBAL ANGLE
039592,000049:                                                                                                  #  INCRMENTS, THE COMMANDED ATTITUDE ANGLE RATES, AND THE COMMANDED
039593,000050:                                                                                                  #  ATTITUDE LAG ANGLES (WHICH ACCOUNT FOR THE ANGLES BY WHICH THE BODY WILL
039594,000051:                                                                                                  #  LAG BEHIND A RAMP COMMAND IN ATTITUDE ANGLE DUE TO THE FINITE ANGULAR
039595,000052:                                                                                                  #  ACCELERATIONS AVAILABLE).
039596,000053: 
039597,000054:                                                                                                  #  FINDCDUW ALIGNS THE ESTIMATED THRUST VECTOR FROM THE THRUST DIRECTION
039598,000055:                                                                                                  #  FILTER WITH THE THRUST COMMAND VECTOR, AND, WHEN XDVINHIB SET,
039599,000056:                                                                                                  #  ALIGNS THE +Z HALF OF THE LM ZX PLANE WITH THE WINDOW COMMAND VECTOR.
039600,000057: 
039601,000058:                                                                                                  #  Page 909
039602,000059:                                                                                                  #  SPECIFICATIONS:
039603,000060: 
039604,000061:                                                                                                  #  INITIALIZATION:       A SINGLE INTERPRETIVE CALL TO INITCDUW IS REQUIRED
039605,000062:                                                                                                  #                        BEFORE EACH GUIDED MANEUVER USING FINDCDUW.
039606,000063: 
039607,000064:                                                                                                  #  CALL:                 INTERPRETIVE CALL TO FINDCDUW WITH THE THRUST COMMAND
039608,000065:                                                                                                  #                        VECTOR IN MPAC.  INTERPRETIVE CALL TO FINDCDUW -2 WITH
039609,000066:                                                                                                  #                        THE THRUST COMMAND VECTOR IN UNFC/2 AND NOT IN MPAC.
039610,000067: 
039611,000068:                                                                                                  #  RETURNS:              NORMAL INTERPRETIVE IN ALL CASES
039612,000069:                                                                                                  #                        1.      NORMALLY ALL AUTOPILOT CMDS ARE ISSUED.
039613,000070:                                                                                                  #                        2.      IF NOT PNGCS AUTO, DO STOPRATE AND RETURN
039614,000071:                                                                                                  #                                WITHOUT ISSUING AUTOPILOT CMDS.
039615,000072:                                                                                                  #                        3.      IF ENGINE OFF, DO STOPRATE AND RETURN WITHOUT
039616,000073:                                                                                                  #                                ISSUING AUTOPILOT CMDS.
039617,000074: 
039618,000075:                                                                                                  #  ALARMS:               00401   IF INPUTS DETERMINE AN ATTITUDE IN GIMBAL LOCK.
039619,000076:                                                                                                  #                                FINDCDUW DRIVES CDUXD AND CDUYD TO THE RQD VALUES,
039620,000077:                                                                                                  #                                BUT DRIVES CDUZD ONLY TO THE GIMBAL LOCK CONE.
039621,000078: 
039622,000079:                                                                                                  #                        00402   IF UNFC/2 OR UNWC/2 PRODUCE OVERFLOW WHEN
039623,000080:                                                                                                  #                                UNITIZED USING NORMUNIT.  FINDCDUW ISSUES
039624,000081:                                                                                                  #                                STOPRATE AS ONLY INPUT TO AUTOPILOT.
039625,000082: 
039626,000083:                                                                                                  #  INPUTS:               UNFC/2          THRUST COMMAND VECTOR, NEED NOT BE SEMI-UNIT.
039627,000084:                                                                                                  #                        UNWC/2          WINDOW COMMAND VECTOR, NEED NOT BE SEMI-UNIT.
039628,000085:                                                                                                  #                        OGABIAS         POSSIBLE BIAS FOR OUTER GIMBAL ANGLE (ZEROED IN INITCDUW), UNITS OF PI.
039629,000086:                                                                                                  #                        XOVINHIB        FLAG DENOTING X AXIS OVERRIDE INHIBITED.
039630,000087:                                                                                                  #                        CSMDOCKD        FLAG DENOTING CSM DOCKED.
039631,000088:                                                                                                  #                        STEERSW         FLAG DENOTING INSUFF THRUST FOR THRUST DIR FLTR.
