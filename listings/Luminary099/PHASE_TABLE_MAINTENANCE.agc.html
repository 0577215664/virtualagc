Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Luminary 131 (Apollo 13) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 001 OF AGC PROGRAM LMY99 BY NASA 2021112-061
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

055255,000085:                                                                                                  #  PHASCHNG IS THE MAIN WAY OF MAKING PHASE CHANGES FOR RESTARTS.  THERE ARE THREE FORMS OF PHASCHNG, KNOWN AS TYPE
055256,000086:                                                                                                  #  A, TYPE B, AND TYPE C.  THEY ARE ALL CALLED AS FOLLOWS, WHERE OCT XXXXX CONTAINS THE PHASE INFORMATION,
055257,000087:                                                                                                  #                TC      PHASCHNG
055258,000088:                                                                                                  #                OCT     XXXXX
055259,000089:                                                                                                  #  TYPE A IS CONCERNED WITH FIXED PHASE CHANGES, THAT IS, PHASE INFORMATION THAT IS STORED PERMANENTLY.  THESE
055260,000090:                                                                                                  #  OPTIONS ARE, WHERE G STANDS FOR A GROUP AND .X FOR THE PHASE,
055261,000091:                                                                                                  #        G.0             INACTIVE, WILL NOT PERMIT A GROUP G RESTART
055262,000092:                                                                                                  #        G.1             WILL CAUSE THE LAST DISPLAY TO BE REACTIVATED, USED MAINLY IN MANNED FLIGHTS
055263,000093:                                                                                                  #        G.EVEN          A DOUBLE TABLE RESTART, CAN CAUSE ANY COMBINATION OF TWO JOBS, TASKS, AND/OR
055264,000094:                                                                                                  #                        LONGCALL TO BE RESTARTED.
055265,000095:                                                                                                  #        G.ODD NOT .1    A SINGLE TABLE RESTART, CAN CAUSE EITHER A JOB, TASK, OR LONGCALL RESTART.
055266,000096: 
055267,000097:                                                                                                  #  THIS INFORMATION IS PUT INTO THE OCTAL WORD AFTER TC PHASCHNG AS FOLLOWS
055268,000098:                                                                                                  #        TL0 00P PPP PPP GGG
055269,000099:                                                                                                  #  WHERE EACH LETTER OR NUMBER STANTS FOR A BIT.  THE G'S STAND FOR THE GROUP, OCTAL 1-7, THE P'S FOR THE PHASE,
055270,000100:                                                                                                  #  OCTAL 0 - 127.  0'S MUST BE 0.  IF ONE WISHES TO HAVE THE TBASE OF GROUP G TO BE SET AT THIS TIME,
055271,000101:                                                                                                  #  T IS SET TO 1, OTHERWISE IT IS SET TO 0.  SIMILARLY IF ONE WISHES TO SET LONGBASE, THEN L IS SET TO 1, OTHERWISE
055272,000102:                                                                                                  #  IT IS SET TO 0.  SOME EXAMPLES,
055273,000103:                                                                                                  #                TC      PHASCHNG        # THIS WILL CAUSE GROUP 3 TO BE SET TO 0,
055274,000104:                                                                                                  #                OCT     00003           # MAKING GROUP 3 INACTIVE
055275,000105: 
055276,000106:                                                                                                  #                TC      PHASCHNG        # IF A RESTART OCCURS THIS WOULD CAUSE
055277,000107:                                                                                                  #                OCT     00012           # GROUP 2 TO RESTART THE LAST DISPLAY
055278,000108: 
055279,000109:                                                                                                  #                TC      PHASCHNG        # THIS SETS THE TBASE OF GROUP 4 AND IN
055280,000110:                                                                                                  #                OCT     40064           # CASE OF A RESTART WOULD START UP THE TWO
055281,000111:                                                                                                  #                                        # THINGS LOCATED IN THE DOUBLE 4.6 RESTART
055282,000112:                                                                                                  #                                        # LOCATION.
055283,000113: 
055284,000114:                                                                                                  #                TC      PHASCHNG        # THIS SETS LONGBASE AND UPON A RESTART
055285,000115:                                                                                                  #                OCT     20135           # CAUSES 5.13 TO BE RESTARTED (SINCE
055286,000116:                                                                                                  #                                        # LONGBASE WAS SET THIS SINGLE ENTRY
055287,000117:                                                                                                  #                                        # SHOULD BE A LONGCALL)
055288,000118: 
055289,000119:                                                                                                  #                TC      PHASCHNG        # SINCE BOTH TBASE4 AND LONGBASE ARE SET,
055290,000120:                                                                                                  #                OCT     60124           # 4.12 SHOULD CONTAIN BOTH A TASK AND A
055291,000121:                                                                                                  #                                        # LONGCALL TO BE RESTARTED
055292,000122: 
055293,000123:                                                                                                  #  TYPE C PHASCHNG CONTAINS THE VARIABLE TYPE OF PHASCHNG INFORMATION.  INSTEAD OF THE INFORMATION BEING IN A
055294,000124:                                                                                                  #  PERMANENT FORM, ONE STORES THE DESIRED RESTART INFORMATION IN A VARIABLE LOCATION.  THE BITS ARE AS FOLLOWS,
055295,000125:                                                                                                  #        TL0 1AD XXX CJW GGG
055296,000126:                                                                                                  #  WHERE EACH LETTER OR NUMBER STANDS FOR A BIT.  THE G'S STAND FOR THE GROUP, OCTAL 1 - 7.  IF THE RESTART IS TO
055297,000127:                                                                                                  #  BE BY WAITLIST, W IS SET TO 1, IF IT IS A JOB, J IS SET TO 1, IF IT IS A LONGCALL, C IS SET TO 1.  ONLY ONE OF
055298,000128:                                                                                                  #  THESE THREE BITS MAY BE SET.  X'S ARE IGNORED, 1 MUST BE 1, AND 0 MUST BE 0.  AGAIN T STANDS FOR THE TBASE,
055299,000129:                                                                                                  #  Page 1296
055300,000130:                                                                                                  #  AND L FOR LONGBASE.  THE BITS A AND D ARE CONCERNED WITH THE VARIABLE INFORMATION.  IF D IS SET TO 1, A PRIORITY
055301,000131:                                                                                                  #  OR DELTA TIME WILL BE READ FROM THE NEXT LOCATION AFTER THE OCTAL INFORMATION., IF THIS IS TO BE INDIRECT, THAT
055302,000132:                                                                                                  #  IS, THE NAME OF A LOCATION CONTAINING THE INFORMATION (DELTA TIME ONLY), THEN THIS IS GIVEN AS THE -GENADR OF
055303,000133:                                                                                                  #  THAT LOCATION WHICH CONTAINS THE DELTA TIME.  IF THE OLD PRIORITY OR DELTA TIME IS TO BE USED, THAT WHICH IS
055304,000134:                                                                                                  #  ALREADY IN THE VARIABLE STORAGE, THEN D IS SET TO 0.  NEXT THE A BIT IS USED.  IF IT IS SET TO 0, THE ADDRESS
055305,000135:                                                                                                  #  THAT WOULD BE RESTARTED DURING A RESTART IS THE NEXT LOCATION AFTER THE PHASE INFORMATION, THAT IS, EITHER
055306,000136:                                                                                                  #  (TC PHASCHNG) +2 OR +3, DEPENDING ON WHETHER D HAD BEEN SET OR NOT.  IF A IS SET TO 1, THEN THE ADDRESS THAT
055307,000137:                                                                                                  #  WOULD BE RESTARTED IS THE 2CADR THAT IS READ FROM THE NEXT TWO LOCATION.  EXAMPLES,
055308,000138:                                                                                                  #        AD      TC      PHASCHNG        # THIS WOULD CAUSE LOCATION AD +3 TO BE
055309,000139:                                                                                                  #        AD+1    OCT     05023           # RESTARTED BY GROUP THREE WITH A PRIORITY
055310,000140:                                                                                                  #        AD+2    OCT     23000           # OF 23.  NOTE UPON RETURNING IT WOULD
055311,000141:                                                                                                  #        AD+3                            # ALSO GO TO AD+3
055312,000142: 
055313,000143:                                                                                                  #        AD      TC      PHASCHNG        # GROUP 1 WOULD CAUSE CALLCALL TO BE
055314,000144:                                                                                                  #        AD+1    OCT     27441           # BE STARTED AS A LONGCALL FROM THE TIME
055315,000145:                                                                                                  #        AD+2    -GENADR DELTIME         # STORED IN LONGBASE (LONGBASE WAS SET) BY
055316,000146:                                                                                                  #        AD+3    2CADR   CALLCALL        # A DELTA TIME STORED IN DELTIME.  THE
055317,000147:                                                                                                  #        AD+4                            # BBCON OF THE 2CADR SHOULD CONTAIN THE E
055318,000148:                                                                                                  #        AD+5                            # BANK OF DELTIME.  PHASCHNG RETURNS TO
055319,000149:                                                                                                  #                                        # LOCATION AD+5
055320,000150: 
055321,000151:                                                                                                  #  NOTE THAT IF A VARIABLE PRIORITY IS GIVEN FOR A JOB, THE JOB WILL BE RESTARTED AS A NOVAC IF THE PRIORITY IS
055322,000152:                                                                                                  #  NEGATIVE, AS A FINDVAC IF TEH PRIORITY IS POSITIVE.
055323,000153: 
055324,000154:                                                                                                  #  TYPE B PHASCHNG IS A COMBINATION OF VARIABLE AND FIXED PHASE CHANGES.  IT WILL START UP A JOB AS INDICATED
055325,000155:                                                                                                  #  BELOW AND ALSO START UP ONE FIXED RESTART, THAT IS EITHER AN G.1 OR A G.ODD OR THE FIRST ENTRY OF G.EVEN
055326,000156:                                                                                                  #  DOUBLE ENTRY.  THE BIT INFORMATION IS AS FOLLOW,
055327,000157:                                                                                                  #        TL1 DAP PPP PPP GGG
055328,000158:                                                                                                  #  WHERE EACH LETTER OR NUMBER STANDS FOR A BIT.  THE G'S STAND FOR THE GROUP, OCTAL 1 - 7, THE P'S FOR THE FIXED
055329,000159:                                                                                                  #  PHASE INFORMATION, OCTAL 0 - 127.  1 MUST BE 1.  AND AGAIN T STANDS FOR THE TBASE AND L FOR LONGBASE.  D THIS
055330,000160:                                                                                                  #  TIME STANDS ONLY FOR PRIORITY SINCE THIS WILL BE CONSIDERED A JOB, AND IT MUST BE GIVEN DIRECTLY IF GIVEN.
055331,000161:                                                                                                  #  AGAIN A STANDS FOR THE ADDRESS OF THE LOCATION TO BE RESTARTED, 1 IF THE 2CADR IS GIVEN, OR 0 IF IT IS TO BE
055332,000162:                                                                                                  #  THE NEXT LOCATION.  (THE RETURN LOCATION OF PHASCHNG) EXAMPLES,
055333,000163:                                                                                                  #        AD      TC      PHASCHNG        # TBASE IS SET AND A RESTART CAUSE GROUP 3
055334,000164:                                                                                                  #        AD+1    OCT     56043           # TO START THE JOB AJOBAJOB WITH PRIORITY
055335,000165:                                                                                                  #        AD+2    OCT     31000           # 31 AND THE FIRST ENTRY OF 3.4SPOT (WE CAN
055336,000166:                                                                                                  #        AD+3    2CADR   AJOBAJOB        # ASSUME IT IS A TASK SINCE WE SET TBASE3)
055337,000167:                                                                                                  #        AD+4                            # UPON RETURN FROM PHASCHNG CONTROL WOULD
055338,000168:                                                                                                  #        AD+5                            # GO TO AD+5
055339,000169: 
055340,000170:                                                                                                  #        AD      TC      PHASCHNG        # UPON A RESTART THE LAST DISPLAY WOULD BE
055341,000171:                                                                                                  #        AD+1    OCT     10015           # RESTARTED AND A JOB WITH THE PREVIOUSLY
055342,000172:                                                                                                  #        AD+2                            # STORED PRIORITY WOULD BE BEGUN AT AD+2
055343,000173:                                                                                                  #                                        # BY MEANS OF GROUP 5
055344,000174:                                                                                                  #  Page 1297
055345,000175:                                                                                                  #  THE NOVAC-FINDVAC CHOICE FOR JOBS HOLDS HERE ALSO -- NEGATIVE PRIORITY CAUSES A NOVAC CALL, POSITIVE A FINDVAC.
