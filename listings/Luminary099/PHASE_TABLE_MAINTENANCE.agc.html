Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Luminary 131 (Apollo 13) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 001 OF AGC PROGRAM LMY99 BY NASA 2021112-061
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

055438,000088:                                                                                                  #  PHASCHNG IS THE MAIN WAY OF MAKING PHASE CHANGES FOR RESTARTS.  THERE ARE THREE FORMS OF PHASCHNG, KNOWN AS TYPE
055439,000089:                                                                                                  #  A, TYPE B, AND TYPE C.  THEY ARE ALL CALLED AS FOLLOWS, WHERE OCT XXXXX CONTAINS THE PHASE INFORMATION,
055440,000090:                                                                                                  #                TC      PHASCHNG
055441,000091:                                                                                                  #                OCT     XXXXX
055442,000092:                                                                                                  #  TYPE A IS CONCERNED WITH FIXED PHASE CHANGES, THAT IS, PHASE INFORMATION THAT IS STORED PERMANENTLY.  THESE
055443,000093:                                                                                                  #  OPTIONS ARE, WHERE G STANDS FOR A GROUP AND .X FOR THE PHASE,
055444,000094:                                                                                                  #        G.0             INACTIVE, WILL NOT PERMIT A GROUP G RESTART
055445,000095:                                                                                                  #        G.1             WILL CAUSE THE LAST DISPLAY TO BE REACTIVATED, USED MAINLY IN MANNED FLIGHTS
055446,000096:                                                                                                  #        G.EVEN          A DOUBLE TABLE RESTART, CAN CAUSE ANY COMBINATION OF TWO JOBS, TASKS, AND/OR
055447,000097:                                                                                                  #                        LONGCALL TO BE RESTARTED.
055448,000098:                                                                                                  #        G.ODD NOT .1    A SINGLE TABLE RESTART, CAN CAUSE EITHER A JOB, TASK, OR LONGCALL RESTART
055449,000099: 
055450,000100:                                                                                                  #  THIS INFORMATION IS PUT INTO THE OCTAL WORD AFTER TC PHASCHNG AS FOLLOWS
055451,000101:                                                                                                  #        TL0 00P PPP PPP GGG                                               .
055452,000102:                                                                                                  #  WHERE EACH LETTER OR NUMBER STANDS FOR A BIT.  THE G:S STAND FOR THE GROUP, OCTAL 1-7, THE P:S FOR THE PHASE,
055453,000103:                                                                                                  #  OCTAL 0 - 127.  0:S MUST BE 0.  IF ONE WISHES TO HAVE THE TBASE OF GROUP G TO BE SET AT THIS TIME,
055454,000104:                                                                                                  #  T IS SET TO 1, OTHERWISE IT IS SET TO 0.  SIMIARLY IF ONE WISHES TO SET LONGBASE, THEN L IS SET TO 1, OTHERWISE
055455,000105:                                                                                                  #  IT IS SET TO 0.  SOME EXAMPLES,
055456,000106:                                                                                                  #                TC      PHASCHNG        THIS WILL CAUSE GROUP 3 TO BE SET TO 0,
055457,000107:                                                                                                  #                OCT     00003           MAKING GROUP 3 INACTIVE
055458,000108: 
055459,000109:                                                                                                  #                TC      PHASCHNG        IF A RESTART OCCURS THIS WOULD CAUSE
055460,000110:                                                                                                  #                OCT     00012           GROUP 2 TO RESTART THE LAST DISPLAY
055461,000111: 
055462,000112:                                                                                                  #                TC      PHASCHNG        THIS SETS THE TBASE OF GROUP 4 AND IN
055463,000113:                                                                                                  #                OCT     40064           CASE OF A RESTART WOULD START UP THE TWO
055464,000114:                                                                                                  #                                        THINGS LOCATED IN THE DOUBLE 4.6 RESTART
055465,000115:                                                                                                  #                                        LOCATION
055466,000116: 
055467,000117:                                                                                                  #                TC      PHASCHNG        THIS SETS LONGBASE AND UPON A RESTART
055468,000118:                                                                                                  #                OCT     20135           CAUSES 5.13 TO BE RESTARTED (SINCE
055469,000119:                                                                                                  #                                        LONGBASE WAS SET THIS SINGLE ENTRY
055470,000120:                                                                                                  #                                        SHOULD BE A LONGCALL)
055471,000121: 
055472,000122:                                                                                                  #                TC      PHASCHNG        SINCE BOTH TBASE4 AND LONGBASE ARE SET,
055473,000123:                                                                                                  #                OCT     60124           4.12 SHOULD CONTAIN BOTH A TASK AND A
055474,000124:                                                                                                  #                                        LONGCALL TO BE RESTARTED
055475,000125: 
055476,000126:                                                                                                  #  TYPE C PHASCHNG CONTAINS THE VARIABLE TYPE OF PHASCHNG INFORMATION.  INSTEAD OF THE INFORMATION BEING IN A
055477,000127:                                                                                                  #  PERMANENT FORM, ONE STORES THE DESIRED RESTART INFORMATION IN A VARIABKE LOCATION.  THE BITS ARE AS FOLLOWS,
055478,000128:                                                                                                  #        TL0 1AD XXX CJW GGG
055479,000129:                                                                                                  #  WHERE EACH LETTER OR NUMBER STANDS FOR A BIT.  THE G:S STAND FOR THE GROUP, OCTAL 1 - 7.  IF THE RESTART IS TO
055480,000130:                                                                                                  #  BE BY WAITLIST, W IS SET TO 1, IF IT IS A JOB, J IS SET TO 1, IF IT IS A LONGCALL, C IS SET TO 1.  ONLY ONE OF
055481,000131:                                                                                                  #  THESE THREE BITS MAY BE SET.  X:S ARE IGNORED  1 MUST BE 1, AND 0 MUST BE 0.  AGAIN T STANDS FOR THE TBASE,

055483,000133:                                                                                                  #  AND L FOR LONGBASE.  THE BITS A AND D ARE CONCERNED WITH THE VARIABLE INFORMATION.  IF D IS SET TO 1, A PRIORITY
055484,000134:                                                                                                  #  OR DELTA TIME WILL BE READ FROM THE NEXT LOCATION AFTER THE OCTAL INFORMATION, IF THIS IS TO BE INDIRECT, THAT
055485,000135:                                                                                                  #  IS, THE NAME OF A LOCATION COMT+INING THE INFORMATION (DELTA TIME ONLY), THEN THIS IS GIVEN AS THE -GENADR OF
055486,000136:                                                                                                  #  THAT LOCATION WHICH CONTAINS THE DELTA TIME.  IF THE OLD PRIORITY OR DELTA TIME IS TO BE USED, THAT WHICH IS
055487,000137:                                                                                                  #  ALREADY IN THE VARIABLE STORAGE, THEN D IS SET TO 0.  NEXT THE A BIT IS USED.  IF IT IS SET TO 0, THE ADDRESS
055488,000138:                                                                                                  #  THAT WOULD BE RESTARTED DURING A RESTART IS THE NEXT LOCATION AFTER THE PHASE INFORMATION, THAT IS, EITHER
055489,000139:                                                                                                  #  (TC PHASCHNG) +2 OR +3, DEPENDING ON WHETHER D HAD BEEN SET OR NOT.  IF A IS SET TO 1, THEN THE ADDRESS THAT
055490,000140:                                                                                                  #  WOULD BE RESTARTED IS THE 2CADR THAT IS READ FROM THE NEXT TWO LOCATIONS.  EXAMPLES.
055491,000141:                                                                                                  #        AD      TC      PHASCHNG        THIS WOULD CAUSE LOCATION AD +3 TO BE
055492,000142:                                                                                                  #        AD+1    OCT     05023           RESTARTED BY GROUP THREE WITH A PRIORITY
055493,000143:                                                                                                  #        AD+2    OCT     23000           OF 23.  NOTE UPON RETURNING IT WOULD
055494,000144:                                                                                                  #        AD+3                            ALSO GO TO AD+3
055495,000145: 
055496,000146:                                                                                                  #        AD      TC      PHASCHNG        GROUP 1 WOULD CAUSE CAUSE CALLCALL TO
055497,000147:                                                                                                  #        AD+1    OCT     27441           BE STARTED AS A LONGCALL FROM THE TIME
055498,000148:                                                                                                  #        AD+2    -GENADR DELTIME         STORED IN LONGBASE (LONGBASE WAS SET) BY
055499,000149:                                                                                                  #        AD+3    2CADR   CALLCALL        A DELTA TIME STORED IN DELTIME.  THE
055500,000150:                                                                                                  #        AD+4                            BBCON OF THE 2CADR SHOULD CONTAIN THE E
055501,000151:                                                                                                  #        AD+5                            BANK OF DELTIME.  PHASCHNG RETURNS TO
055502,000152:                                                                                                  #                                        LOCATION AD+5

055504,000154:                                                                                                  #  NOTE THAT IF A VARIABLE PRIORITY IS GIVEN FOR A JOB, THE JOB WILL BE RESTARTED AS A NOVAC IF THE PRIORITY IS
055505,000155:                                                                                                  #  NEGATIVE, AS A FINDVAC IF THE PRIORITY IS POSITIVE.
055506,000156: 
055507,000157:                                                                                                  #  TYPE B PHASCHNG IS A COMBINATION OF VARIABLE AND FIXED PHASE CHANGES.  IT WILL START UP A JOB AS INDICATED
055508,000158:                                                                                                  #  BELOW AND ALSO START UP ONE FIXED RESTART, THAT IS EITHER AN G.1 OR A G.ODD OR THE FIRST ENTRY OF G.EVEN
055509,000159:                                                                                                  #  DOUBLE ENTRY.  THE BIT INFORMATION IS AS FOLLOWS,
055510,000160:                                                                                                  #        TL1 DAP PPP PPP GGG
055511,000161:                                                                                                  #  WHERE EACH LETTER OR NUMBER STANDS FOR A BIT.  THE G:S STAND FOR THE GROUP, OCTAL 1 - 7, THE P:S FOR THE FIXED
055512,000162:                                                                                                  #  PHASE INFORMATION, OCTAL 0 - 127.  1 MUST BE 1.  AND AGAIN T STANDS FOR THE TBASE AND L FOR LONGBASE.  D THIS
055513,000163:                                                                                                  #  TIME STANDS ONLY FOR PRIORITY SINCE THIS WILL BE CONSIDERED A JOB, AND IT MUST BE GIVEN DIRECTLY IF GIVEN.
055514,000164:                                                                                                  #  AGAIN A STANDS FOR THE ADDRESS OF THE LOCATION TO BE RESTARTED, 1 IF THE 2CADR IS GIVEN, OR 0 IF IT IS TO BE
055515,000165:                                                                                                  #  THE NEXT LOCATION.  (THE RETURN LOCATION OF PHASCHNG) EXAMPLES,
055516,000166:                                                                                                  #        AD      TC      PHASCHNG        TBASE IS SET AND A RESTART CAUSE GROUP 3
055517,000167:                                                                                                  #        AD+1    OCT     56043           TO START THE JOB AJOBAJOB WITH PRIORITY
055518,000168:                                                                                                  #        AD+2    OCT     31000           31 AND THE FIRST ENTRY OF 3.4SPOT (WE CAN
055519,000169:                                                                                                  #        AD+3    2CADR   AJOBAJOB        ASSUME IT IS A TASK SINCE WE SET TBASE3)
055520,000170:                                                                                                  #        AD+4                            UPON RETURN FROM PHASCHNG CONTROL WOULD
055521,000171:                                                                                                  #        AD+5                            GO TO AD+5
055522,000172: 
055523,000173:                                                                                                  #        AD      TC      PHASCHNG        UPON A RESTART THE LAST DISPLAY WOULD BE
055524,000174:                                                                                                  #        AD+1    OCT     10015           RESTARTED AND A JOB WITH THE PREVIOUSLY
055525,000175:                                                                                                  #        AD+2                            STORED PRIORITY WOULD BE BEGUN AT AD+2
055526,000176:                                                                                                  #                                        BY MEANS OF GROUP 5
