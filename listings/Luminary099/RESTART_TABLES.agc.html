Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Luminary 131 (Apollo 13) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 001 OF AGC PROGRAM LMY99 BY NASA 2021112-061
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

009820,000032:                                                                                                  #  --------------
009821,000033: 
009822,000034:                                                                                                  #  THERE ARE TWO FORMS OF RESTART TABLES FOR EACH GROUP.  THEY ARE KNOWN AS THE EVEN RESTART TABLES AND THE ODD
009823,000035:                                                                                                  #  RESTART TABLES.  THE ODD TABLES HAVE ONLY ONE ENTRY OF THREE LOCATIONS WHILE THE EVEN TABLES HAVE TWO ENTRIES
009824,000036:                                                                                                  #  EACH USING THREE LOCATIONS.  THE INFORMATION AS TO WHETHER IT IS A JOB, WAITLIST, OR A LONGCALL IS GIVEN BY THE
009825,000037:                                                                                                  #  WAY THINGS ARE PUT IN TO THE TABLES.
009826,000038: 
009827,000039:                                                                                                  #  A JOB HAS ITS PRIORITY STORED IN A PRDTTAB OF THE CORRECT PHASE SPOT -- A POSITIVE PRIORITY INDICATES A
009828,000040:                                                                                                  #  FINDVAC JOB, A NEGATIVE PRIORITY A NOVAC.  THE 2CADR OF THE JOB IS STORED IN THE CADRTAB.
009829,000041:                                                                                                  #  FOR EXAMPLE,
009830,000042: 
009831,000043:                                                                                                  #                5.7SPOT         OCT     23000
009832,000044:                                                                                                  #                                2CADR   SOMEJOB
009833,000045: 
009834,000046:                                                                                                  #  A RESTART OF GROUP 5 WITH PHASE SEVEN WOULD THEN CAUSE SOMEJOB TO BE RESTARTED AS A FINDVAC WITH PRIORITY 23.
009835,000047: 
009836,000048:                                                                                                  #                5.5SPOT         OCT     -23000
009837,000049:                                                                                                  #                                2CADR   ANYJOB
009838,000050: 
009839,000051:                                                                                                  #  HERE A RESTART OF GROUP 5 WITH PHASE 7 WOULD CAUSE ANYJOB TO BE RESTARTED AS A NOVAC WITH PRIORITY 23.
009840,000052:                                                                                                  #  A LONGCALL HAS ITS GENADR OF ITS 2CADR STORED NEGATIVELY AND ITS BBCON STORED POSITIVELY.  IN ITS PRDTTAB IS
009841,000053:                                                                                                  #  PLACED THE LOCATION OF A DP REGISTER THAT CONTAINS THE DELTA TIME THAT LONGCALL HAD BEEN ORIGINALLY STARTED
009842,000054:                                                                                                  #  WITH.  EXAMPLE,
009843,000055: 
009844,000056:                                                                                                  #                3.6SPOT         GENADR  DELTAT
009845,000057:                                                                                                  #                                -GENADR LONGTASK
009846,000058:                                                                                                  #                                BBCON   LONGTASK
009847,000059:                                                                                                  #                                OCT     31000
009848,000060:                                                                                                  #                                2CADR   JOBAGAIN
009849,000061: 
009850,000062:                                                                                                  #  THIS WOULD START UP LONGTASK AT THE APPROPRIATE TIME, OR IMMEDIATELY IF THE TIME HAD ALREADY PASSED.  IT SHOULD
009851,000063:                                                                                                  #  BE NOTED THAT IF DELTAT IS IN A SWITCHED E BANK, THIS INFORMATION SHOULD BE IN THE BBCON OF THE 2CADR OF THE
009852,000064:                                                                                                  #  TASK.  FROM ABOVE, WE SEE THAT THE SECOND PART O THIS PHASE WOULD BE STARTED AS A JOB WITH A PRIORITY OF 31.
009853,000065: 
009854,000066:                                                                                                  #  WAITLIST CALLS ARE IDENTIFIED BY THE FCT THAT THEIR 2CADR IS STORED NEGATIVELY.  IF PRDTTAB OF THE PHASE SPOT
009855,000067:                                                                                                  #  IS POSITIVE, THEN IT CONTAINS THE DELTA TIME, IF PRDTTAB IS NEGATIVE THEN IT IS THE -GENADR OF AN ERASABLE
009856,000068:                                                                                                  #  IF THE TIME IS STORED INDIRECTLY, THE BBCON MUST CONTAIN THE NECESSARY E BANK INFORMATION IF APPLICABLE.  WITH
009857,000069:                                                                                                  #  WAITLIST WE HAVE ONE FURTHER OPTION, IF -0 IS STORED IN PRDTTAB, IT WILL CAUSE AN IMMEDIATE RESTART OF THE
009858,000070:                                                                                                  #  TASK.  EXAMPLES,
009859,000071: 
009860,000072:                                                                                                  #                                OCT     77777           # THIS WILL CAUSE AN IMMEDIATE RESTART
009861,000073:                                                                                                  #                                -2CADR  ATASK           # OF THE TASK :ATASK:
009862,000074:                                                                                                  #        
009863,000075:                                                                                                  #                                DEC     200             # IF THE TIME OF THE 2 SECONDS SINCE DUMMY
009864,000076:                                                                                                  #                                -2CADR  DUMMY           # WAS PUT ON THE WAITLIST IS UP, IT WILL BEGIN
009865,000077:                                                                                                  #                                                        # IN 10 MS, OTHERWISE IT WILL BEGIN WHEN
009866,000078:                                                                                                  #                                                        # IT NORMALLY WOULD HAVE BEGUN.
009867,000079:                                                                                                  #  Page 239
009868,000080:                                                                                                  #                                -GENADR DTIME           # WHERE DTIME CONTAINS THE DELTA TIME
009869,000081:                                                                                                  #                                -2CADR  TASKTASK        # OTHERWISE THIS IS AS ABOVE
