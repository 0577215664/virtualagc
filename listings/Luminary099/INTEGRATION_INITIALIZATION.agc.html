051091,000032:                                                                                                  #  1.0 INTRODUCTION
051092,000033:                                                                                                  #  ----------------
051093,000034: 
051094,000035:                                                                                                  #  FROM A USER'S POINT OF VIEW, ORBITAL INTEGRATION IS ESSENTIALLY THE SAME AS THE 278 INTEGRATION
051095,000036:                                                                                                  #  PROGRAM.  THE SAME ENTRANCES TO THE PROGRAM WILL BE MAINTAINED, THE SAME STALLING ROUTINE WILL BE USED AND
051096,000037:                                                                                                  #  OUTPUT WILL STILL BE VIA THE PUSHLIST.  THE PRIMARY DIFFERENCES TO A USER INVOLVE THE ADDED CAPABILITY OF
051097,000038:                                                                                                  #  TERMINATING INTEGRATION AT A SPECIFIC FINAL RADIUS AND THE DIFFERENCE IN STATE VECTOR SCALING INSIDE AND OUTSIDE
051098,000039:                                                                                                  #  THE LUNAR SPHERE OF INFLUENCE.
051099,000040: 
051100,000041:                                                                                                  #  IN ORDER TO MAKE THE CSM(LEM)PREC AND CSM(LEM)CONIC ENTRANCES SIMILAR TO FLIGHT 278, THE INTEGRATION PROGRAM
051101,000042:                                                                                                  #  WILL ITSELF SET THE FINAL RADIUS (RFINAL) TO 0 SO THAT REACHING THE DESIRED TIME ONLY WILL TERMINATE
051102,000043:                                                                                                  #  INTEGRATION.  THE DP REGISTER RFINAL MUST BE SET BY USERS OF INTEGRVS AND INTEGRV, AND MUST BE DONE AFTER THE
051103,000044:                                                                                                  #  CALL TC INTSTALL.
051104,000045: 
051105,000046:                                                                                                  #  WHEN THE LM IS ON THE LUNAR SURFACE (INDICATED BY LUNAR SURFACE FLAG SET) CALLS TO LEMCONIC, LEMPREC, AND
051106,000047:                                                                                                  #  INTEGRV WITH VINFLAG = 0 WILL RESULT IN THE USE OF THE PLANETARY INERTIAL ORIENTATION SUBROUTINES TO PROVIDE
051107,000048:                                                                                                  #  BOTH THE LM'S POSITION AND VELOCITY IN THE REFERENCE COORDINATE SYSTEM.
051108,000049:                                                                                                  #  THE PROGRAM WILL PROVIDE OUTPUT AS IF INTEGRATION WAS USED.  THAT IS, THE PUSHLIST WILL BE SET AS NOTED BELOW AND
051109,000050:                                                                                                  #  THE PERMANENT STATE VECTOR UPDATED WHEN SPECIFIED BY AN INTEGRV CALL.
051110,000051: 
051111,000052:                                                                                                  #  USERS OF INTEGRVS DESIRING INTEGRATION (INTYPFLG = 0) SHOULD NOTE THAT THE OBLATENESS PERTURBATION COMPUTATION
051112,000053:                                                                                                  #  IN LUNAR ORBIT IS TIME DEPENDENT.  THEREFORE, THE USER SHOULD SUPPLY AN INITIAL STATE VECTOR VALID AT SOME REAL
051113,000054:                                                                                                  #  TIME AND THE DESIRED TIME (TDEC1) ALSO AT SOME REAL TIME.  FOR CONIC "INTEGRATION" THE USER MAY STILL USE ZERO
051114,000055:                                                                                                  #  AS THE INITIAL TIME AND DELTA TIME AS THE DESIRED TIME.
051115,000056: 
051116,000057:                                                                                                  #  2.0 GENERAL DESCRIPTION
051117,000058:                                                                                                  #  -----------------------
051118,000059: 
051119,000060:                                                                                                  #  THE INTEGRATION PROGRAM OPERATES AS A CLOSED INTERPRETIVE SUBROUTINE AND PERFORMS THESE FUNCTIONS --
051120,000061:                                                                                                  #        1) INTEGRATES (PRECISION OR CONIC) EITHER CSM OR LM STATE VECTOR
051121,000062:                                                                                                  #        2) INTEGRATES THE W-MATRIX
051122,000063:                                                                                                  #        3) PERMANENT OR TEMPORARY UPDATE OF THE STATE VECTOR
051123,000064: 
051124,000065:                                                                                                  #  THERE ARE SIX ENTRANCES TO THE INTEGRATION PROGRAM.  FOUR OF THESE (CSMPREC, LEMPREC, CSMCONIC, LEMCONIC) SET
051125,000066:                                                                                                  #  ALL THE FLAGS REQUIRED IN THE INTEGRATION PROGRAM ITSELF TO CAUSE THE PRECISION OR CONIC INTEGRATION (KEPLER) OF
051126,000067:                                                                                                  #  THE LM OR CSM STATE VECTOR, AS THE NAMES SUGGEST.  ONE ENTRANCE (INTEGRVS) PERMITS THE CALLING PROGRAM TO
051127,000068:                                                                                                  #  PROVIDE A STATE VECTOR TO BE INTEGRATED.  THE CALLING PROGRAM MUST SET THE FLAGS INDICATING (1) PRECISION OR
051128,000069:                                                                                                  #  CONIC INTEGRATION, (2) IN OR OUT OF LUNAR SPHERE, (3) MIDCOURSE OR NOT, AND THE INTEGRATION PROGRAM COMPLETES
051129,000070:                                                                                                  #  THE FLAG SETTING TO BYPASS W-MATRIX INTEGRATION.  THE LAST ENTRANCE (INTEGRV, USED IN GENERAL BY THE
051130,000071:                                                                                                  #  NAVIGATION PROGRAMS) PERMITS THE CALLER TO SET FIVE FLAGS (NOT MOONFLAG OR MIDFLAG) BUT NOT TO INPUT A STATE
051131,000072:                                                                                                  #  VECTOR.  ANY PROGRAM WHICH CALLS INTEGRVS OR INTEGRV MUST CALL INTSTALL BEFORE IT SETS THE INTEGRATION FLAGS
051132,000073:                                                                                                  #  AND/OR STATE VECTOR.
051133,000074: 
051134,000075:                                                                                                  #  THREE SETS OF 42 REGISTERS AND 2 FLAGS ARE USED FOR THE STATE VECTORS.  TWO SETS, WHICH MAY NOT BE OVERLAYED, ARE
051135,000076:                                                                                                  #  USED FOR THE PERMANENT STATE VECTORS FOR THE CSM AND LM.  THE THIRD SET, WHICH MAY BE OVERLAYED WHEN INTEGRATION
051136,000077:                                                                                                  #  IS NOT BEING DONE, IS USED IN THE COMPUTATIONS.
051137,000078: 
051138,000079:                                                                                                  #  THE PERMANENT STATE VECTORS WILL BE PERIODICALLY UPDATED SO THAT THE VECTORS WILL NOT BE OLDER THAN 4 TIMESTEPS.
051139,000080:                                                                                                  #  THE PERMANENT STATE VECTORS WILL ALSO BE UPDATED WHENEVER THE W-MATRIX IS INTEGRATED OR WHEN A CALLER OF INTEGRV
051140,000081:                                                                                                  #  SETS STATEFLG (THE NAVIGATION PROGRAMS P20, P22.)
051141,000082:

051145,000086:                                                                                                  #  2.1 RESTARTS
051146,000087: 
051147,000088:                                                                                                  #  PHASE CHANGES WILL BE MADE IN THE INTEGRATION PROGRAM ONLY FOR THE INTEGRV ENTRANCE (I.E., WHEN THE W-MATRIX IS
051148,000089:                                                                                                  #  INTEGRATED OR PERMANENT STATE VECTOR IS UPDATED.)  THE GROUP NUMBER USED WILL BE THAT FOR THE P20-25 PROGRAMS
051149,000090:                                                                                                  #  (I.E., GROUP2) WINCE THE INTEGRV ENTRANCE WILL ONLY BE USED BY THESE PROGRAMS.  IF A RESTART OCCURS DURING AN
051150,000091:                                                                                                  #  INTEGRATION OF THE STATE VECTOR ONLY, THE RECOVERY WILL BE TO THE LAST PHASE IN THE CALLING PROGRAM.  CALLING
051151,000092:                                                                                                  #  PROGRAMS WHICH USE THE INTEGRV OR INTEGRVS ENTRANCE OF INTEGRATION WHOULD ENSURE THAT IF PHASE CHANGING IS DONE
051152,000093:                                                                                                  #  THAT IT IS PRIOR TO SETTING THE INTEGRATION INPUTS IN THE PUSHLIST.
051153,000094:                                                                                                  #  THIS IS BECAUSE THE PUSHLIST IS LOST DURING A RESTART.
051154,000095: 
051155,000096:                                                                                                  #  2.2 SCALING
051156,000097: 
051157,000098:                                                                                                  #  THE INTEGRATION ROUTINE WILL MAINTAIN THE PERMANENT MEMORY STATE VECTORS IN THE SCALING AND UNITS DEFINED IN
051158,000099:                                                                                                  #  APPENDIX B OF THE USERS GUIDE.  THE SCALING OF THE OUTPUT POSITION VECTOR DEPENDS ON THE ORIGIN OF THE COORDINATE
051159,000100:                                                                                                  #  SYSTEM AT THE DESIRED INTEGRATION TIME.  THE COORDINATE SYSTEM TRANSFORMATION WILL BE DONE AUTOMATICALLY ON
051160,000101:                                                                                                  #  MULTIPLE TIMESTEP ENCKE INTEGRATION ONLY.  THUS IT IS POSSIBLE TO HAVE OUTPUT FROM SUCCESSIVE INTEGRATIONS IN
051161,000102:                                                                                                  #  DIFFERENT SCALING.
051162,000103:                                                                                                  #  HOWEVER, RATT, VATT WILL ALWAYS BE SCALED THE SAME.
