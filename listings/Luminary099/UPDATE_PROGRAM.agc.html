Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Luminary 131 (Apollo 13) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 001 OF AGC PROGRAM LMY99 BY NASA 2021112-061
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

059061,000044:                                                                                                  #  FUNCT. DESCR.:        P27 (THE UPDATE PROGRAM) PROCESSES COMMANDS AND DATA

059063,000046:                                                                                                  #                        THE P27 PROGRAM WILL ACCEPT UPDATES
059064,000047:                                                                                                  #                        ONLY DURING P00 FOR THE LM, AND ONLY DURINT P00,
059065,000048:                                                                                                  #                        P02, AND FRESH START FOR THE CSM
059066,000049: 
059067,000050:                                                                                                  #  CALLING SEQ:          PROGRAM IS INITIATED BY UPLINK ENTRY OF VERBS 70, 71, 72, AND 73.
059068,000051: 
059069,000052:                                                                                                  #  SUBROUTINES:          TESTXACT, NEWMODEX, NEWMODEX +3, GOXDSPF, BANKCALL, FINDVAC, INTPRET, INTSTALL, TPAGREE,
059070,000053:                                                                                                  #                        INTWAKEU, ENDEXT, POSTJUMP, FALTON, NEWPHASE, PHASCHNG
059071,000054: 
059072,000055:                                                                                                  #  NORMAL EXIT:          TC ENDEXT
059073,000056: 
059074,000057:                                                                                                  #  ALARM/ABORT:          TC FALTON FOLLOWED BY TC ENDEXT
059075,000058: 
059076,000059:                                                                                                  #  RESTARTS:             P27 IS RESTART PROTECTED IN TWO WAYS ...
059077,000060:                                                                                                  #                        1.      PRIOR TO VERIFLAG INVERSION (WHICH IS CAUSED BY THE GROUND/ASTRONAUT'S VERIFICATION OF UPDATE
059078,000061:                                                                                                  #                                DATA BY SENDING A V33E WHEN V21N02 IS FLASHING)---
059079,000062:                                                                                                  #                                NO PROTECTION EXCEPT PRE-P27 MODE IS RESTROED, COAST + ALIGN DOWNLIST IS SELECTED AND UPLINK
059080,000063:                                                                                                  #                                ACTIVITY LIGHT IS TURNED OFF.  (JUST AS IF A V34E WAS SENT DURING P27 DATA LOADS).
059081,000064:                                                                                                  #                                V70,V71,V72, OR V73 WILL HAVE TO BE COMPLETELY RESENT BY USER.
059082,000065:                                                                                                  #                        2.      AFTER VERIFLAG INVERSION (WHEN UPDATE OF THE SPECIFIED ERASABLES IS BEING PERFORMED)---
059083,000066:                                                                                                  #                                PROTECTED AGAINST RESTARTS.
059084,000067: 
059085,000068:                                                                                                  #  DEBRIS:               UPBUFF  (20D)   TEMP STORAGE FOR ADDRESSES AND CONTENTS.
059086,000069:                                                                                                  #                        UPVERB  (1)     VERB NUMBER MINUS 70D (E.G., FOR V72, UPVERB = 72D - 70D = 2)
059087,000070:                                                                                                  #                        UPOLDMOD(1)     FOR MAJOR MODE INTERRUPTED BY P27.
059088,000071:                                                                                                  #                        COMPNUMB(1)     TOTAL NUMBER OF COMPONENTS TO BE TRANSMITTED.
059089,000072:                                                                                                  #                        UPCOUNT (1)     ACTUAL NUMBER OF COMPONENTS RECEIVED.
059090,000073:                                                                                                  #                        UPTEMP  (1)     SCRATCH, BUT USUALLY CONTAINS COMPONENT NUMBER TO BE CHANGED DURING VERIFY CYCLE
