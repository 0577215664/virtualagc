Source Code

These source-code files were from the printout at the MIT Museum, with many thanks to Deborah Douglas, the Museum's Curator of Science and Technology, for releasing it to us. The printout was digitally photographed by Paul Fjeld. A team of volunteers transcribed the source code manually from these images or, where appropriate, modified already-transcribed but similar source Luminary 131 (Apollo 13) source code to agree with the images from the printout. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 001 OF AGC PROGRAM LMY99 BY NASA 2021112-061
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

059836,000031:                                                                                                  #  PROGRAM NAMES:        (1) T6JOBCHK    MOD. NO. 5      OCTOBER 2, 1967
059837,000032:                                                                                                  #                        (2) DOT6RUPT
059838,000033:                                                                                                  #  MODIFICATION BY:      LOWELL G. HULL (A.C.ELECTRONICS)
059839,000034: 
059840,000035:                                                                                                  #  THESE PROGRAMS ENABLE THE LM DAP TO CONTROL THE THRUST TIMES OF THE REACTION CONTROL SYSTEM JETS BY USING TIME6.
059841,000036:                                                                                                  #  SINCE THE LM DAP MAINTAINS EXCLUSIVE CONTROL OVER TIME6 AND ITS INTERRUPTS, THE FOLLOWING CONVENTIONS HAVE BEEN
059842,000037:                                                                                                  #  ESTABLISHED AND MUST NOT BE TAMPERED WITH:
059843,000038:                                                                                                  #        1.      NO NUMBER IS EVER PLACED INTO TIME6 EXCEPT BY LM DAP.
059844,000039:                                                                                                  #        2.      NO PROGRAM OTHER THAN LM DAP ENABLES THE TIME6 COUNTER.
059845,000040:                                                                                                  #        3.      TO USE TIME6, THE FOLLOWING SEQUENCE IS ALWAYS EMPLOYED:
059846,000041:                                                                                                  #                A.      A POSITIVE (NON-ZERO) NUMBER IS STORED IN TIME6.
059847,000042:                                                                                                  #                B.      THE TIME6 CLOCK IS ENABLED.
059848,000043:                                                                                                  #                C.      TIME6 IS INTERROGATED AND IS:
059849,000044:                                                                                                  #                        I.      NEVER FOUND NEGATIVE (NON-ZERO) OR +0.
059850,000045:                                                                                                  #                        II.     SOMETIMES FOUND POSITIVE (BETWEEN 1 AND 240D) INDICATING THAT IT IS ACTIVE.
059851,000046:                                                                                                  #                        III.    SOMETIMES FOUND POSMAX INDICATING THAT IT IS INACTIVE AND NOT ENABLED.
059852,000047:                                                                                                  #                        IV.     SOMETIMES FOUND NEGATIVE ZERO INDICATING THAT:
059853,000048:                                                                                                  #                                A.      A T6RUPT IS ABOUT TO OCCUR AT THE NEXT DINC, OR
059854,000049:                                                                                                  #                                B.      A T6RUPT IS WAITING IN THE PRIORITY CHAIN, OR
059855,000050:                                                                                                  #                                C.      A T6RUPT IS IN PROCESS NOW.
059856,000051:                                                                                                  #        4.      ALL PROGRAMS WHICH OPERATE IN EITHER INTERRUPT MODE OR WITH INTERRUPT INHIBITED MUST CALL T6JOBCHK
059857,000052:                                                                                                  #                EVERY 5 MILLISECONDS TO PROCESS A POSSIBLE WAITING T6RUPT BEFORE IT CAN BE HONORED BY THE HARDWARE.
059858,000053:                                                                                                  #       (5.      PROGRAM JTLST, IN Q,R-AXES, HANDLES THE INPUT LIST.)
059859,000054: 
059860,000055:                                                                                                  #  T6JOBCHK CALLING SEQUENCE:
059861,000056:                                                                                                  #                L       TC      T6JOBCHK
059862,000057:                                                                                                  #                L+1     (RETURN)
059863,000058: 
059864,000059:                                                                                                  #  DOT6RUPT CALLING SEQUENCE:
059865,000060:                                                                                                  #                        DXCH    ARUPT           # T6RUPT LEAD IN AT LOCATION 4004.
059866,000061:                                                                                                  #                        EXTEND
059867,000062:                                                                                                  #                        DCA     T6ADR
059868,000063:                                                                                                  #                        DTCB
059869,000064: 
059870,000065:                                                                                                  #  SUBROUTINES CALLED:   DOT6RUPT CALLS T6JOBCHK.
059871,000066: 
059872,000067:                                                                                                  #  NORMAL EXIT MODES:    T6JOBCHK RETURNS TO L +1.
059873,000068:                                                                                                  #                        DOT6RUPT TRANSFERS CONTROL TO RESUME.
059874,000069: 
059875,000070:                                                                                                  #  ALARM/ABORT MODES:    NONE.
059876,000071: 
059877,000072:                                                                                                  #  INPUT:        TIME6           NXT6ADR         OUTPUT:         TIME6           NXT6ADR         CHANNEL 5
059878,000073:                                                                                                  #                T6NEXT          T6NEXT +1                       T6NEXT          T6NEXT +1       CHANNEL 6
059879,000074:                                                                                                  #                T6FURTHA        T6FURTHA +1                     T6FURTHA        T6FURTHA +1     BIT15/CH13
059880,000075:
