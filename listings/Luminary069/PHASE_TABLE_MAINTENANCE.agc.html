Source Code

These source-code files were transcribed from scans made from Don Eyles's personal copy of Luminary 069. They were scanned at archive.org's Boston facility, and the scanning was sponsored by Onno Hommes. The code was transcribed from these scans by a team of volunteers who are referenced in the program comments. Comments from the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". In some cases, where similar code blocks exist in previously-transcribed AGC programs (primarily Luminary 99, from Apollo 11) those code blocks were used as a starting point and then corrected to agree with the Luminary 69 scans. The full scans are available at the Virtual AGC project's collection at archive.org, while more-convenient reduced-size (but reduced-quality) images are available at the main Virtual AGC website. Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 069 OF AGC PROGRAM LUMINARY BY NASA 2021112-011
	19:02 NOV. 25,1968
Note that the date is the date of the printout, not the date of the program revision.

055801,000076:                                                                                                  #  PHASCHNG IS THE MAIN WAY OF MAKING PHASE CHANGES FOR RESTARTS.  THERE ARE THREE FORMS OF PHASCHNG, KNOWN AS TYPE
055802,000077:                                                                                                  #  A, TYPE B, AND TYPE C. THEY ARE ALL CALLED AS FOLLOWS, WHERE OCT XXXXX CONTAINS THE PHASE INFORMATION,
055803,000078:                                                                                                  #                TC      PHASCHNG
055804,000079:                                                                                                  #                OCT     XXXXX
055805,000080:                                                                                                  #  TYPE A IS CONCERNED WITH FIXED PHASE CHANGES, THAT IS, PHASE INFORMATION THAT IS STORED PERMANENTLY.  THESE
055806,000081:                                                                                                  #  OPTIONS ARE, WHERE G STANDS FOR A GROUP AND .X FOR THE PHASE,
055807,000082:                                                                                                  #        G.0             INACTIVE, WILLNOT PERMIT A GROUP G RESTART
055808,000083:                                                                                                  #        G.1             WILL CAUSE THE LAST DISPLAY TO BE REACTIVATED, USED MAINLY IN MANNED FLIGHTS
055809,000084:                                                                                                  #        G.EVEN          A DOUBLE TABLE RESTART, CAN CAUSE ANY COMBINATION OF TWO JOBS, TASKS, AND/OR
055810,000085:                                                                                                  #                        LONGCALL TO BE RESTARTED.
055811,000086:                                                                                                  #        G.ODD NOT .1    A SINGLE TABLE RESTART, CAN CAUSE EITHER A JOB, TASK, OR LONGCALL RESTART
055812,000087: 
055813,000088:                                                                                                  #  THIS INFORMATION IS PUT INTO THE OCTAL WORD AFTER TC PHASCHNG AS FOLLOWS
055814,000089:                                                                                                  #        TL0 00P PPP PPP GGG                                               .
055815,000090:                                                                                                  #  WHERE EACH LETTER OR NUMBER STANDS FOR A BIT.  THE G:S STAND FOR THE GROUP, OCTAL 1-7, THE P:S FOR THE PHASE,
055816,000091:                                                                                                  #  OCTAL 0 - 127.  0:S MUST BE 0.              IF ONE WISHES TO HAVE THE TBASE OF GROUP G TO BE SET AT THIS TIME,
055817,000092:                                                                                                  #  T IS SET TO 1, OTHERWISE IT IS SET TO 0.  SIMIARLY IF ONE WISHES TO SET LONGBASE, THEN L IS SET TO 1, OTHERWISE
055818,000093:                                                                                                  #  IT IS SET TO 0.  SOME EXAMPLES,
055819,000094:                                                                                                  #                TC      PHASCHNG        THIS WILL CAUSE GROUP 3 TO BE SET TO 0,
055820,000095:                                                                                                  #                OCT     00003           MAKING GROUP 3 INACTIVE
055821,000096: 
055822,000097:                                                                                                  #                TC      PHASCHNG        IF A RESTART OCCURS THIS WOULD CAUSE
055823,000098:                                                                                                  #                OCT     00012           GROUP 2 TO RESTART THE LAST DISPLAY
055824,000099: 
055825,000100:                                                                                                  #                TC      PHASCHNG        THIS SETS THE TBASE OF GROUP 4 AND IN
055826,000101:                                                                                                  #                OCT     40064           CASE OF A RESTART WOULD START UP THE TWO
055827,000102:                                                                                                  #                                        THINGS LOCATED IN THE DOUBLE 4.6 RESTART
055828,000103:                                                                                                  #                                        LOCATION
055829,000104:                                                                                                  #                TC      PHASCHNG        THIS SETS LONGBASE AND UPON A RESTART
055830,000105:                                                                                                  #                OCT     20135           CAUSES 5.13 TO BE RESTARTED (SINCE
055831,000106:                                                                                                  #                                        LONGBASE WAS SET THIS SINGLE ENTRY
055832,000107:                                                                                                  #                                        SHOULD BE A LONGCALL)
055833,000108: 
055834,000109:                                                                                                  #                TC      PHASCHNG        SINCE BOTH TBASE4 AND LONGBASE ARE SET,
055835,000110:                                                                                                  #                OCT     60124           4.12 SHOULD CONTAIN BOTH A TASK AND A
055836,000111:                                                                                                  #                                        LONGCALL TO BE RESTARTED
055837,000112: 
055838,000113:                                                                                                  #  TYPE C PHASCHNG CONTAINS THE VARIABLE TYPE OF PHASCHNG INFORMATION. INSTEAD OF THE INFORMATION BEING IN A
055839,000114:                                                                                                  #  PERMANENT FORM, ONE STORES THE DESIRED RESTART INFORMATION IN A VARIABKE LOCATION. THE BITS ARE AS FOLLOWS,
055840,000115:                                                                                                  #        TL0 1AD XXX CJW GGG
055841,000116:                                                                                                  #  WHERE EACH LETTER OR NUMBER STANDS FOR A BIT.  THE G:S STAND FOR THE GROUP, OCTAL 1 - 7.  IF THE RESTART IS TO
055842,000117:                                                                                                  #  BE BY WAITLIST, W IS SET TO 1, IF IT IS A JOB, J IS SET TO 1, IF IT IS A LONGCALL, C IS SET TO 1.  ONLY ONE OF
055843,000118:                                                                                                  #  THESE THREE BITS MAY BE SET.  X:S ARE IGNORED  1 MUST BE 1, AND 0 MUST BE 0.  AGAIN T STANDS FOR THE TBASE,
