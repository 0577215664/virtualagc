Source Code

These source-code files were transcribed from scans made from Don Eyles's personal copy of Luminary 069. They were scanned at archive.org's Boston facility, and the scanning was sponsored by Onno Hommes. The code was transcribed from these scans by a team of volunteers who are referenced in the program comments. Comments from the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". In some cases, where similar code blocks exist in previously-transcribed AGC programs (primarily Luminary 99, from Apollo 11) those code blocks were used as a starting point and then corrected to agree with the Luminary 69 scans. The full scans are available at the Virtual AGC project's collection at archive.org, while more-convenient reduced-size (but reduced-quality) images are available at the main Virtual AGC website. Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 069 OF AGC PROGRAM LUMINARY BY NASA 2021112-011
	19:02 NOV. 25,1968
Note that the date is the date of the printout, not the date of the program revision.

043457,000026:                                                                                                  #  PROGRAM NAME - DOWN TELEMETRY PROGRAM
043458,000027:                                                                                                  #  MOD NO. - 0           TO COMPLETELY REWRITE THE DOWN TELEMETRY PROGRAM AND DOWNLINK ERASABLE DUMP PROGRAM FOR THE
043459,000028:                                                                                                  #                        PURPOSE OF SAVING APPROXIMATELY 150 WORDS OF CORE STORAGE.

043461,000030:                                                                                                  #                        LISTS LOG SECTION FOR MORE DETAILS. HOWEVER THIS CHANGE WILL NOT AFFECT THE GROUND PROCESSING
043462,000031:                                                                                                  #                        OF DOWN TELEMETRY DATA.

043468,000037:                                                                                                  #  FUNCTIONAL DESCRIPTION - THIS ROUTINE IS INITIATED BY TELEMETRY END
043469,000038:                                                                                                  #        PULSE FROM THE DOWNLINK TELEMETRY CONVERTER. THIS PULSE OCCURS
043470,000039:                                                                                                  #        AT 50 TIMES PER SEC (EVERY 20 MS) THEREFORE DODOWNTM IS
043471,000040:                                                                                                  #        EXECUTED AT THESE RATES. THIS ROUTINE SELECTS THE APPROPRIATE
043472,000041:                                                                                                  #        AGC DATA TO BE TRANSMITTED DOWNLINK AND LOADS IT INTO OUTPUT
043473,000042:                                                                                                  #        CHANNELS 34 AND 35. THE INFORMATION IS THEN GATED OUT FROM THE
043474,000043:                                                                                                  #        LGC IN SERIAL FASHION.
043475,000044:                                                                                                  #        THIS PROGRAM IS CODED FOR A 2 SECOND DOWNLIST. SINCE DOWNRUPTS
043476,000045:                                                                                                  #        OCCUR EVERY 20 MS AND 2 AGC COMPUTER WORDS CAN BE PLACED IN
043477,000046:                                                                                                  #        CHANNELS 34 AND 35 DURING EACH DOWNRUPT THE PROGRAM IS CAPABLE
043478,000047:                                                                                                  #        OF SENDING 200 AGC WORDS EVERY 2 SECONDS.
043479,000048:                                                                                                  #  CALLING SEQUENCE - NONE
043480,000049:                                                                                                  #        PROGRAM IS ENTERED VIA TCF DODOWNTM WHICH IS EXECUTED AS A
043481,000050:                                                                                                  #        RESULT OF A DOWNRUPT. CONTROL IS RETURNED VIA TCF RESUME WHICH
043482,000051:                                                                                                  #        IN EFFECT IS A RESUME.
043483,000052:                                                                                                  #  SUBROUTINES CALLED - NONE
043484,000053:                                                                                                  #  NORMAL EXIT MODE - TCF RESUME
043485,000054:                                                                                                  #  ALARM OR ABORT EXIT MODE - NONE
043486,000055:                                                                                                  #  RESTART PROTECTION:
043487,000056:                                                                                                  #        ON A FRESH START AND RESTART THE 'STARTSUB' SUBROUTINE WILL INITIALIZE THE DOWNLIST POINTER (ACTUALLY
043488,000057:                                                                                                  #        DNTMGOTO) TO THE BEGINNING OF THE CURRENT DOWNLIST (I.E. CURRENT CONTENTS OF DNLSTADR). THIS HAS THE
043489,000058:                                                                                                  #        EFFECT OF IGNORING THE REMAINDER OF THE DOWNLIST WHICH THE DOWN-TELEMETRY PROGRAM WAS WORKING ON WHEN
043490,000059:                                                                                                  #        THE RESTART (OR FRESH START) OCCURRED AND RESUME DOWN TELEMETRY FROM THE BEGINNING OF THE CURRENT
043491,000060:                                                                                                  #        DOWNLIST.
043492,000061:                                                                                                  #        ALSO OF INTEREST IS THE FACT THAT ON A RESTART THE AGC WILL ZERO DOWNLINK CHANNELS 13, 34 AND 35.
043493,000062:                                                                                                  #  DOWNLINK LIST SELECTION:
043494,000063:                                                                                                  #        THE APPROPRIATE DOWNLINK LISTS ARE SELECTED BY THE FOLLOWING:
043495,000064:                                                                                                  #        1.      FRESH START
043496,000065:                                                                                                  #        2.      V37EXXE WHERE XX = THE MAJOR MODE BEING SELECTED.
043497,000066:                                                                                                  #        3.      UPDATE PROGRAM (P27)
043498,000067:                                                                                                  #        4.      NON-V37 SELECTABLE TYPE PROGRAMS (E.G. AGS INITIALIZATION (SUNDANCE, LUMINARY) AND P61-P62
043499,000068:                                                                                                  #                TRANSITION (COLOSSUS) ETC.).
043500,000069:                                                                                                  #  DOWNLINK LIST RULES AND LIMITATIONS:
043501,000070:                                                                                                  #        READ SECTION(S) WHICH FOLLOW 'DEBRIS' WRITEUP.
043502,000071:                                                                                                  #  OUTPUT - EVERY 2 SECONDS 100 DOUBLE PRECISION WORDS (I.E. 200 LGC
043503,000072:                                                                                                  #        COMPUTER WORDS) ARE TRANSMITTED VIA DOWNLINK.
043504,000073:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED - NONE
043505,000074:                                                                                                  #        'DNTMGOTO' AND 'DNLSTADR' ARE INITIALIZED BY THE FRESH START PROGRAM.
043506,000075:                                                                                                  #  DEBRIS (ERASABLE LOCATIONS DESTROYED BY THIS PROGRAM) -
043507,000076:                                                                                                  #        LDATALST,DNTMBUFF TO DNTMBUFF +21D,TMINDEX,DNQ.

043516,000085:                                                                                                  #  NOTES REGARDING DOWNLINK LISTS ASSOCIATED WITH THIS PROGRAM:
043517,000086:                                                                                                  #  1.    DOWNLISTS. - DOWNLISTS MUST BE COMPILED IN THE SAME BANK AS THE
043518,000087:                                                                                                  #        DOWN-TELEMETRY PROGRAM. THIS IS DONE FOR EASE OF CODING, FASTER
043519,000088:                                                                                                  #        EXECUTION.
043520,000089:                                                                                                  #  2.    EACH DOWNLINK LIST CONSISTS OF A CONTROL LIST AND A NUMBER OF
043521,000090:                                                                                                  #        SUBLISTS.
043522,000091:                                                                                                  #  3.    A SUBLIST REFERS TO A SNAPSHOT OR DATA COMMON TO THE SAME OR OTHER
043523,000092:                                                                                                  #        DOWNLINK LISTS.  ANY SUBLIST CONTAINING COMMON DATA NEEDS TO BE
043524,000093:                                                                                                  #        CODED ONLY ONCE FOR THE APPLICABLE DOWNLINK LISTS.
043525,000094:                                                                                                  #  4.    SNAPSHOT SUBLISTS REFER SPECIFICALLY TO HOMOGENOUS DATA WHICH MUST BE
043526,000095:                                                                                                  #        SAVED IN A BUFFER DURING ONE DOWNRUPT.
043527,000096:                                                                                                  #  5.    THE 1DNADR FOR THE 1ST WORD OF SNAPSHOT DATA IS FOUND AT THE END
043528,000097:                                                                                                  #        OF EACH SNAPSHOT SUBLIST, SINCE THE PROGRAM CODING SENDS THIS DP WORD
043529,000098:                                                                                                  #        IMMEDIATELY AFTER STORING THE OTHERS IN THE SNAPSHOT BUFFER.
043530,000099:                                                                                                  #  6.    ALL LISTS ARE COMBINATIONS OF CODED ERASABLE ADDRESS CONSTANTS
043531,000100:                                                                                                  #        CREATED FOR THE DOWNLIST PROGRAM.
043532,000101:                                                                                                  #        A.      1DNADR                  1-WORD DOWNLIST ADDRESS.
043533,000102:                                                                                                  #                SAME AS ECADR, BUT USED WHEN THE WORD ADDRESSED IS THE LEFT
043534,000103:                                                                                                  #                HALF OF A DOUBLE-PRECISION WORD FOR DOWN TELEMETRY.
043535,000104:                                                                                                  #        B.      2DNADR - 6DNADR         N-WORD DOWNLIST ADDRESS, N = 2 - 6.
043536,000105:                                                                                                  #                SAME AS 1DNADR, BUT WITH THE 4 UNUSED BITS OF THE ECADR FORMAT
043537,000106:                                                                                                  #                FILLED IN WITH 0001-0101.  USED TO POINT TO A LIST OF N DOUBLE-
043538,000107:                                                                                                  #                PRECISION WORDS, STORED CONSECUTIVELY, FOR DOWN TELEMETRY.
043539,000108:                                                                                                  #        C.      DNCHAN                  DOWNLIST CHANNEL ADDRESS.
043540,000109:                                                                                                  #                SAME AS 1DNADR, BUT WITH PREFIX BITS 0111.  USED TO POINT TO
043541,000110:                                                                                                  #                A PAIR OF CHANNELS FOR DOWN TELEMETRY.
043542,000111:                                                                                                  #        D.      DNPTR                   DOWN TELEMETRY SUBLIST POINTER.
043543,000112:                                                                                                  #                SAME AS CAF BUT TAGGED AS A CONSTANT. USED IN CONTROL LIST TO POINT TO A SUBLIST.
043544,000113:                                                                                                  #                CAUTION--- A DNPTR CANNOT BE USED IN A SUBLIST.
043545,000114:                                                                                                  #  7.    THE WORD ORDER CODE IS SET TO ZERO AT THE BEGINNING OF EACH DOWNLIST (I.E. CONTROL LIST) AND WHEN
043546,000115:                                                                                                  #        A '1DNADR TIME2' IS DETECTED IN THE CONTROL LIST (ONLY).
043547,000116:                                                                                                  #  8.    IN THE SNAPSHOT SUBLIST ONLY, THE DNADR'S CANNOT POINT TO THE FIRST WORD OF ANY EBANK.

043550,000119:                                                                                                  #    (THE FOLLOWING POINTS MAY BE LISTED ELSEWHERE BUT ARE LISTED HERE SO IT IS CLEAR THAT THESE THINGS CANNOT BE
043551,000120:                                                                                                  #    DONE)
043552,000121:                                                                                                  #  1.    SNAPSHOT DOWNLIST:
043553,000122:                                                                                                  #        (A) CANNOT CONTAIN THE FOLLOWING ECADRS (I.E. 1DNADR'S): 0, 400, 1000, 1400, 2000, 2400, 3000, 3400.
043554,000123:                                                                                                  #        (B) CAN CONTAIN ONLY 1DNADR'S

043556,000125:                                                                                                  #        EQUIVALENT OF THE FOLLOWING ECADRS (I.E. 1DNADRS): 377, 777, 1377, 1777, 2377, 27777, 3377, 3777.
043557,000126:                                                                                                  #        (NOTE: THE TERM EQUIVALENT ' MEANT THAT THE 1DNADR TO 6DNADR WILL BE PROCESSED LIKE 1 TO 6 ECADRS)
043558,000127:                                                                                                  #  3.    CONTROL LISTS AND SUBLISTS CANNOT HAVE ENTRIES = OCTAL 00000 OR OCTAL 77777

043753,000322:                                                                                                  #  SUBROUTINE NAME- DNDUMP
043754,000323:                                                                                                  #  FUNCTIONAL DESCRIPTION - TO SEND (DUMP) ALL ERASABLE STORAGE 'N' TIMES.(N=1 TO 4). BANKS ARE SENT ONE AT A TIME
043755,000324:                                                                                                  #        EACH BANK IS PRECEEDED BY AN ID WORD, SYNCH BITS, ECADR AND TIME1 FOLLOWED BY THE 256D WORDS OF EACH
043756,000325:                                                                                                  #        EBANK. EBANKS ARE DUMPED IN ORDER (I.E. EBANK 0 FIRST, THEN EBANK1 ETC.)
043757,000326:                                                                                                  #  CALLING SEQUENCE- THE GROUND OR ASTRONAUT BY KEYING V74E CAN INITIALIZE THE DUMP.
043758,000327:                                                                                                  #        AFTER KEYING IN V74E THE CURRENT DOWNLIST WILL BE IMMEDIATELY TERMINATED AND THE DOWNLINK ERASABLE DUMP
043759,000328:                                                                                                  #        WILL BEGIN.
043760,000329:                                                                                                  #        ONCE INITIATED THE DOWNLINK ERASABLE DUMP CAN BE TERMINATED (AND INTERRUPTED DOWNLIST REINSTATED) ONLY
043761,000330:                                                                                                  #        BY THE FOLLOWING:
043762,000331:                                                                                                  #        1.      A FRESH START
043763,000332:                                                                                                  #        2.      COMPLETION OF ALL DOWNLINK DUMPS REQUESTED (ACCORDING TO BITS SET IN DUMPCNT). NOTE THAT DUMPCNT
043764,000333:                                                                                                  #                CAN BE ALTERED BY A V21N01.
043765,000334:                                                                                                  #        3.      AND INVOLUNTARILY BY A RESTART.
043766,000335:                                                                                                  #  NORMAL EXIT MODE - TCF DNPHASE1
043767,000336:                                                                                                  #  ALARM OR ABORT MODE - NONE
