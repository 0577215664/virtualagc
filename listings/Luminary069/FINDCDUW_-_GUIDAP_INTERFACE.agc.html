Source Code

These source-code files were transcribed from scans made from Don Eyles's personal copy of Luminary 069. They were scanned at archive.org's Boston facility, and the scanning was sponsored by Onno Hommes. The code was transcribed from these scans by a team of volunteers who are referenced in the program comments. Comments from the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". In some cases, where similar code blocks exist in previously-transcribed AGC programs (primarily Luminary 99, from Apollo 11) those code blocks were used as a starting point and then corrected to agree with the Luminary 69 scans. The full scans are available at the Virtual AGC project's collection at archive.org, while more-convenient reduced-size (but reduced-quality) images are available at the main Virtual AGC website. Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 069 OF AGC PROGRAM LUMINARY BY NASA 2021112-011
	19:02 NOV. 25,1968
Note that the date is the date of the printout, not the date of the program revision.

040210,000025:                                                                                                  #  PROGRAM NAME:   FINDCDUW
040211,000026: 
040212,000027:                                                                                                  #  MOD NUMBER:     1         68 07 15
040213,000028: 
040214,000029:                                                                                                  #  MOD AUTHOR:     KLUMPP
040215,000030: 
040216,000031:                                                                                                  #  OBJECTS OF MOD: 1.        TO SUPPLY COMMANDED GIMBAL ANGLES FOR NOUN 22.
040217,000032:                                                                                                  #                  2.        TO MAINTAIN CORRECT AND CURRENT THRUST
040218,000033:                                                                                                  #                            DIRECTION DATA IN ALL MODES.  THIS IS DONE BY
040219,000034:                                                                                                  #                            FETCHING FOR THE THRUST DIRECTION FILTER THE
040220,000035:                                                                                                  #                            CDUD'S IN PNGCS-AUTO, THE CDU'S IN ALL OTHER
040221,000036:                                                                                                  #                            MODES.
040222,000037:                                                                                                  #                  3.        TO SUBSTITUTE A STOPRATE FOR THE NORMAL
040223,000038:                                                                                                  #                            AUTOPILOT COMMANDS WHENEVER
040224,000039:                                                                                                  #                            1) NOT IN PNGCS-AUTO, OR
040225,000040:                                                                                                  #                            2) ENGINE IS OFF.
040226,000041: 
040227,000042:                                                                                                  #  FUNCTIONAL DESCRIPTION:
040228,000043: 
040229,000044:                                                                                                  #  FINDCDUW PROVIDES THE INTERFACES BETWEEN THE VARIOUS POWERED FLITE GUIDANCE PROGRAMS
040230,000045:                                                                                                  #  AND THE DIGITAL AUTOPILOT.  THE INPUTS TO FINDCDUW ARE THE THRUST COMMAND VECTOR
040231,000046:                                                                                                  #  AND THE WINDOW COMMAND VECTOR, AND THE OUTPUTS ARE THE GIMBAL ANGLE
040232,000047:                                                                                                  #  INCREMENTS, THE COMMANDED ATTITUDE ANGLE RATES, AND THE COMMANDED
040233,000048:                                                                                                  #  ATTITUDE LAG ANGLES (WHICH ACCOUNT FOR THE ANGLES BY WHICH THE BODY WILL
040234,000049:                                                                                                  #  LAG BEHIND A RAMP COMMAND IN ATTITUDE ANGLE DUE TO THE FINITE ANGULAR
040235,000050:                                                                                                  #  ACCELERATIONS AVAILABLE).
040236,000051: 
040237,000052:                                                                                                  #  FINDCDUW ALINES THE ESTIMATED THRUST VECTOR FROM THE THRUST DIRECTION
040238,000053:                                                                                                  #  FILTER WITH THE THRUST COMMAND VECTOR, AND, WHEN  XOVINHIB  SET,
040239,000054:                                                                                                  #  ALINES THE +Z HALF OF THE LM ZX PLANE WITH THE WINDOW COMMAND VECTOR.

040242,000057:                                                                                                  #  SPECIFICATIONS:

040257,000072:

040259,000074:                                                                                                  #                           ISSUING AUTOPILOT CMDS.
