Source Code

These source-code files were transcribed from scans made from Don Eyles's personal copy of Luminary 069. They were scanned at archive.org's Boston facility, and the scanning was sponsored by Onno Hommes. The code was transcribed from these scans by a team of volunteers who are referenced in the program comments. Comments from the original source code are prefixed with a single '#' symbol, whereas comments added later are prefixed by "##" or "###". In some cases, where similar code blocks exist in previously-transcribed AGC programs (primarily Luminary 99, from Apollo 11) those code blocks were used as a starting point and then corrected to agree with the Luminary 69 scans. The full scans are available at the Virtual AGC project's collection at archive.org, while more-convenient reduced-size (but reduced-quality) images are available at the main Virtual AGC website. Report any errors noted by creating an issue report at the Virtual AGC project's GitHub repository. Notations on the program listing read, in part:

	GAP:  ASSEMBLE REVISION 069 OF AGC PROGRAM LUMINARY BY NASA 2021112-011
	19:02 NOV. 25,1968
Note that the date is the date of the printout, not the date of the program revision.

040201,000025:                                                                                                  #  PROGRAM NAME:   FINDCDUW
040202,000026: 
040203,000027:                                                                                                  #  MOD NUMBER:     1         68 07 15
040204,000028: 
040205,000029:                                                                                                  #  MOD AUTHOR:     KLUMPP
040206,000030: 
040207,000031:                                                                                                  #  OBJECTS OF MOD: 1.        TO SUPPLY COMMANDED GIMBAL ANGLES FOR NOUN 22.
040208,000032:                                                                                                  #                  2.        TO MAINTAIN CORRECT AND CURRENT THRUST
040209,000033:                                                                                                  #                            DIRECTION DATA IN ALL MODES.  THIS IS DONE BY
040210,000034:                                                                                                  #                            FETCHING FOR THE THRUST DIRECTION FILTER THE
040211,000035:                                                                                                  #                            CDUD'S IN PNGCS-AUTO, THE CDU'S IN ALL OTHER
040212,000036:                                                                                                  #                            MODES.
040213,000037:                                                                                                  #                  3.        TO SUBSTITUTE A STOPRATE FOR THE NORMAL
040214,000038:                                                                                                  #                            AUTOPILOT COMMANDS WHENEVER
040215,000039:                                                                                                  #                            1) NOT IN PNGCS-AUTO, OR
040216,000040:                                                                                                  #                            2) ENGINE IS OFF.
040217,000041: 
040218,000042:                                                                                                  #  FUNCTIONAL DESCRIPTION:
040219,000043: 
040220,000044:                                                                                                  #  FINDCDUW PROVIDES THE INTERFACES BETWEEN THE VARIOUS POWERED FLITE GUIDANCE PROGRAMS
040221,000045:                                                                                                  #  AND THE DIGITAL AUTOPILOT.  THE INPUTS TO FINDCDUW ARE THE THRUST COMMAND VECTOR
040222,000046:                                                                                                  #  AND THE WINDOW COMMAND VECTOR, AND THE OUTPUTS ARE THE GIMBAL ANGLE
040223,000047:                                                                                                  #  INCREMENTS, THE COMMANDED ATTITUDE ANGLE RATES, AND THE COMMANDED
040224,000048:                                                                                                  #  ATTITUDE LAG ANGLES (WHICH ACCOUNT FOR THE ANGLES BY WHICH THE BODY WILL
040225,000049:                                                                                                  #  LAG BEHIND A RAMP COMMAND IN ATTITUDE ANGLE DUE TO THE FINITE ANGULAR
040226,000050:                                                                                                  #  ACCELERATIONS AVAILABLE).
040227,000051: 
040228,000052:                                                                                                  #  FINDCDUW ALINES THE ESTIMATED THRUST VECTOR FROM THE THRUST DIRECTION
040229,000053:                                                                                                  #  FILTER WITH THE THRUST COMMAND VECTOR, AND, WHEN  XOVINHIB  SET,
040230,000054:                                                                                                  #  ALINES THE +Z HALF OF THE LM ZX PLANE WITH THE WINDOW COMMAND VECTOR.

040233,000057:                                                                                                  #  SPECIFICATIONS:
040234,000058: 
040235,000059:                                                                                                  #  INITIALIZATION: A SINGLE INTERPRETIVE CALL TO  INITCDUW  IS REQUIRED
040236,000060:                                                                                                  #                  BEFORE EACH GUIDED MANEUVER USING FINDCDUW.

040245,000069:

040247,000071:                                                                                                  #                           WITHOUT ISSUING AUTOPILOT CMDS.
040248,000072: 
040249,000073:                                                                                                  #                  3.       IF ENGINE OFF, DO STOPRATE AND RETURN WITHOUT
040250,000074:                                                                                                  #                           ISSUING AUTOPILOT CMDS.
040251,000075: 
040252,000076:                                                                                                  #  ALARMS:         00401 IF INPUTS DETERMINE AN ATTITUDE IN GIMBAL LOCK.
040253,000077:                                                                                                  #                           FINDCDUW DRIVES CDUXD AND CDUYD TO THE RQD VALUES,
040254,000078:                                                                                                  #                           BUT DRIVES CDUZD ONLY TO THE GIMBAL LOCK CONE.
040255,000079:                                                                                                  #                  00402 IF UNFC/2 OR UNWC/2 PRODUCE OVERFLOW WHEN
040256,000080:                                                                                                  #                           UNITIZED USING NORMUNIT.  FINDCDUW ISSUES
040257,000081:                                                                                                  #                           STOPRATE AS ONLY INPUT TO AUTOPILOT.
040258,000082: 
040259,000083:                                                                                                  #  INPUTS:         UNFC/2   THRUST COMMAND VECTOR, NEED NOT BE SEMI-UNIT.
040260,000084:                                                                                                  #                  UNWC/2   WINDOW COMMAND VECTOR, NEED NOT BE SEMI-UNIT.
040261,000085:                                                                                                  #                  XOVINHIB FLAG DENOTING X AXIS OVERRIDE INHIBITED.
040262,000086:                                                                                                  #                  CSMDOCKD FLAG DENOTING CSM DOCKED.

040264,000088: 
040265,000089:                                                                                                  #  OUTPUTS:        DELCDUX,Y,Z

040267,000091:                                                                                                  #                  DELPEROR,+1,+2
