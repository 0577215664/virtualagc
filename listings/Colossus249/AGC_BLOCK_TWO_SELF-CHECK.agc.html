Source Code

These source-code files were derived originally from the scan from MIT's Dibner Institute's now-discontinued website titled "History of Recent Science and Technology", which in turn was originally created by Gary Neff, though his high-quality scans were unfortunately reduced in legibility when presented online. The markings on the front of the printout imply that it was the original AGC developer Norm Brodeur's copy. A scan by Ron Burkey has superceded it, made from a more-legible copy from the collection of original AGC developer Fred Martin. The source code was transcribed from those images by Ron Burkey. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	ASSEMBLE REVISION 249 OF AGC PROGRAM COLOSSUS BY NASA 2021111-041
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

058977,000039:                                                                                                  #  PROGRAM NAME -- SELF-CHECK                    LOG SECTION:  AGC BLOCK TWO SELF-CHECK

058981,000043:                                                                                                  #  FUNCTIONAL DESCRIPTION
058982,000044:                                                                                                  #        PROGRAM HAS TWO MAIN PARTS.  THE FIRST IS SELF-CHECK WHICH RUNS AS A ZERO PRIORITY JOB WITH NO CORE SET, AS
058983,000045:                                                                                                  #        PART OF THE BACK-UP IDLE LOOP.  THE SECOND IS SHOW-BANKSUM WHICH RUNS AS A REGULAR EXECUTIVE JOB WITH ITS OWN
058984,000046:                                                                                                  #        STARTING VERB.
058985,000047: 
058986,000048:                                                                                                  #        THE PURPOSE OF SELF-CHECK IS TO CHECK OUT VARIOUS PARTS OF THE COMPUTER AS OUTLINED BELOW IN THE OPTIONS.
058987,000049: 
058988,000050:                                                                                                  #        THE PURPOSE OF SHOW-BANKSUM IS TO DISPLAY THE SUM OF EACH BANK, ONE AT A TIME.
058989,000051: 
058990,000052:                                                                                                  #        IN ALL THERE ARE 7 POSSIBLE OPTIONS IN THIS BLOCK II VERSION OF SELF-CHECK.  MORE DETAIL DESCRIPTION MAY BE
058991,000053:                                                                                                  #        FOUND IN E-2065 BLOCK II AGC SELF-CHECK AND SHOW BANKSUM BY EDWIN D. SMALLY DECEMBER 1966, AND ADDENDA 2 AND 3.
058992,000054: 
058993,000055:                                                                                                  #        THE DIFFERENT OPTIONS ARE CONTROLLED BY PUTTING DIFFERENT NUMBERS IN THE SMODE REGISTER (NOUN 27).  BELOW IS
058994,000056:                                                                                                  #        A DESCRIPTION OF WHAT PARTS OF THE COMPUTER THAT ARE CHECKED BY THE OPTIONS, AND THE CORRESPONDING NUMBER, IN
058995,000057:                                                                                                  #        OCTAL, TO LOAD INTO SMODE.
058996,000058:                                                                                                  #                +-4             ERASABLE MEMORY
058997,000059:                                                                                                  #                +-5             FIXED MEMORY
058998,000060:                                                                                                  #                +-1,2,3,6,7,10  EVERYTHING IN OPTIONS 4 AND 5.
058999,000061:                                                                                                  #                -0              SAME AS +-10 UNTIL AN ERROR IS DETECTED.
059000,000062:                                                                                                  #                +0              NO CHECK, PUTS COMPUTER INTO THE BACKUP IDLE LOOP.
059001,000063: 
059002,000064:                                                                                                  #  WARNINGS
059003,000065:                                                                                                  #        USE OF E MEMORY RESERVED FOR SELF-CHECK (EVEN IN IDLE LOOP) AS TEMP STORAGE BY OTHER PROGRAMS IS DANGEROUS.
059004,000066:                                                                                                  #        SMODE SET GREATER THAN OCT 10 PUTS COMPUTER INTO BACKUP IDLE LOOP.
059005,000067: 
059006,000068:                                                                                                  #  CALLING SEQUENCE
059007,000069:                                                                                                  #        TO CALL SELF-CHECK KEY IN
059008,000070:                                                                                                  #                V 21 N 27 E     OPTION NUMBER E
059009,000071:                                                                                                  #        TO CALL SHOW-BANKSUM KEY IN
059010,000072:                                                                                                  #                V 91 E          DISPLAYS FIRST BANK
059011,000073:                                                                                                  #                V 33 E          PROCEED, DISPLAYS NEXT BANK
059012,000074: 
059013,000075:                                                                                                  #  EXIT MODES, NORMAL AND ALARM
059014,000076:                                                                                                  #        SELF-CHECK NORMALLY CONTINUES INDEFINITELY UNLESS THERE IS AN ERROR DETECTED.  IF SO + OPTION NUMBERS PUT
059015,000077:                                                                                                  #        COMPUTER INTO BACKUP IDLE LOOP, - OPTIONS NUMBERS RESTART THE OPTION.
059016,000078: 
059017,000079:                                                                                                  #        THE -0 OPTION PROCEEDS FROM THE LINE FOLLOWING THE LINE WHERE THE ERROR WAS DETECTED.
059018,000080: 
059019,000081:                                                                                                  #        SHOW-BANKSUM PROCEEDS UNTIL A TERMINATE IS KEYED IN (V 34 E).  THE COMPUTER IS PUT INTO THE BACKUP IDLE LOOP.
059020,000082: 
059021,000083:                                                                                                  #  OUTPUT

059023,000085:                                                                                                  #        SELF-CHECK UPON DETECTING AN ERROR LOADS THE SELF-CHECK ALARM CONSTANT (01102) INTO THE FAILREG SET AND
059024,000086:                                                                                                  #        TURNS ON THE ALARM LIGHT.  THE OPERATOR MAY THEN DISPLAY THE THREE FAILREGS BY KEYING IN V 05 N 09 E.  FOR FURTHER
059025,000087:                                                                                                  #        INFORMATION HE MAY KEY IN V 05 N 08 E, THE DSKY DISPLAY IN R1 WILL BE ADDRESS+1 OF WHERE THE ERROR WAS DETECTED,
059026,000088:                                                                                                  #        IN R2 THE BBCON OF SELF-CHECK, AND IN R3 THE TOTAL NUMBER OF ERRORS DETECTED BY SELF-CHECK SINCE THE LAST MAN
059027,000089:                                                                                                  #        INITIATED FRESH START (SLAP1).
059028,000090: 
059029,000091:                                                                                                  #        SHOW-BANKSUM STARTING WTIH BANK 0 DISPLAYS IN R1 THE BANK SUM (A +-NUMBER EQUAL TO THE BANK NUMBER), IN R2
059030,000092:                                                                                                  #        THE BANK NUMBER, AND IN R3 THE BUGGER WORD.
059031,000093: 
059032,000094:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED
059033,000095:                                                                                                  #        ACCOMPLISHED BY FRESH START
059034,000096:                                                                                                  #                SMODE SET TO +0
059035,000097: 
059036,000098:                                                                                                  #  DEBRIS
059037,000099:                                                                                                  #        ALL EXITS FROM THE CHECK OF ERASABLE (ERASCHK) RESTORE ORIGINAL CONTENTS TO REGISTERS UNDER CHECK.
059038,000100:                                                                                                  #        EXCEPTION IS A RESTART.  RESTART THAT OCCURS DURING ERASCHK RESTORES ERASABLE, UNLESS THERE IS EVIDENCE TO DOUBT
059039,000101:                                                                                                  #        E MEMORY, IN WHICH CASE PROGRAM THEN DOES A FRESH START (DOFSTART).
