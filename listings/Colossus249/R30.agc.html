Source Code

These source-code files were derived originally from the scan from MIT's Dibner Institute's now-discontinued website titled "History of Recent Science and Technology", which in turn was originally created by Gary Neff, though his high-quality scans were unfortunately reduced in legibility when presented online. The markings on the front of the printout imply that it was the original AGC developer Norm Brodeur's copy. A scan by Ron Burkey has superceded it, made from a more-legible copy from the collection of original AGC developer Fred Martin. The source code was transcribed from those images by Ron Burkey. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	ASSEMBLE REVISION 249 OF AGC PROGRAM COLOSSUS BY NASA 2021111-041
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

021852,000048:                                                                                                  #  1.    IF AVERAGE G IS OFF:
021853,000049:                                                                                                  #                FLASH DISPLAY V04N06.  R2 INDICATES WHICH SHIP'S STATE VECTOR IS
021854,000050:                                                                                                  #                        TO BE UPDATED.  INITIAL CHOICE IS THIS SHIP (R2-1).  ASTRONAUT
021855,000051:                                                                                                  #                        CAN CHANGE TO OTHER SHIP BY V22EXE. WHERE X IS NOT EQ 1.
021856,000052:                                                                                                  #                SELECTED STATE VECTOR UPDATED BY THISPREC (OTHPREC).
021857,000053:                                                                                                  #                CALLS SR30.1 (WHICH CALLS TFFCONMU + TFFRP/RA) TO CALCULATE
021858,000054:                                                                                                  #                        RPER (PERIGEE RADIUS), RAPO (APOGEE RADIUS), HPER (PERIGEE
021859,000055:                                                                                                  #                        HEIGHT ABOVE LAUNCH PAD OR LAUNAR LANDING SITE), HARD (APOGEE
021860,000056:                                                                                                  #                        HEIGHT AS ABOVE), TPER (TIME TO PERIGEE), TFF (TIME TO
021861,000057:                                                                                                  #                        INTERSECT 300 KFT ABOVE PAD OR 36KFT ABOVE LANDING SITE).
021862,000058:                                                                                                  #                FLASH MONITOR V16N44 (HAPO, HPER, TFF).  TFF IS -59MS59S IF IT WAS
021863,000059:                                                                                                  #                        NOT COMPUTABLE, OTHERWISE IT INCREMENTS ONCE PER SECOND.
021864,000060:                                                                                                  #                        ASTRONAUT HAS OPTION TO MONITOR TPER BY KING IN N 32 E.
021865,000061:                                                                                                  #                        DISPLAY IS IN HMS, IS NEGATIVE (AS WAS TFF), AND INCREMENTS
021866,000062:                                                                                                  #                        ONCE PER SECOND ONLY IF TFF DISPLAY WAS -59M59S.
021867,000063:                                                                                                  #        2.      IF AVERAGE G IS ON:
021868,000064:                                                                                                  #                        CALLS SR30.1 APPROX EVERY TWO SECS.  STATE VECTOR IS ALWAYS
021869,000065:                                                                                                  #                                FOR THIS VEHICLE.  V82 DOES NOT DISTURB STATE VECTOR.  RESULTS
021870,000066:                                                                                                  #                                OF SR30.1 ARE RAPO, RPER, HAPO, HPER, TPER, TFF.
021871,000067:                                                                                                  #                        FLASH MONITOR V16N44 (HAPO, HPER, TFF).
021872,000068:                                                                                                  #                        IF MODE IS P11, THEN CALL DELRSPL SO ASTRONAUT CAN MONITOR
021873,000069:                                                                                                  #                                RESULTS BY N50E.  SPLASH COMPUTATION DONE ONCE FOR TWO SECS.
021874,000070:                                                                                                  #        ADDENDUM:  HAPO AND HPER SHOULD BE CHANGED TO READ HAPOX AND HPERX IN THE
021875,000071:                                                                                                  #                ABOVE REMARKS.
