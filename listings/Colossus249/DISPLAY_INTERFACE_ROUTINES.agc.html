Source Code

This is the source-code for the Apollo 9 Command Module's Guidance Computer. These files were derived originally from the scan from MIT's Dibner Institute's now-discontinued website titled "History of Recent Science and Technology", which in turn was originally created by Gary Neff, though his high-quality scans were unfortunately reduced in legibility when presented online. The markings on the front of the printout imply that it was the original AGC developer Norm Brodeur's copy. A scan by Ron Burkey has superceded it, made from a more-legible copy from the collection of original AGC developer Fred Martin. The source code was transcribed from those images by Ron Burkey. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	ASSEMBLE REVISION 249 OF AGC PROGRAM COLOSSUS BY NASA 2021111-041
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

062094,000051:                                                                                                  #            SEQUENCE OF EVENTS.

062096,000053:                                                                                                  #            MM DISPLAYS AND SPECIAL PURPOSE DISPLAYS HANDLED BY PINBALL.

062098,000055: 
062099,000056:                                                                                                  #  THE FOLLOWING TERMS ARE USED TO DESCRIBE THE STATUS OF DISPLAYS-
062100,000057:                                                                                                  #        1.  ACTIVE - THE DISPLAY WHICH IS (1) BEING DISPLAYED TO THE ASTRONAUT AND WAITING FOR A RESPONSE OR
062101,000058:                                                                                                  #            (2) WAITING FIRST IN LINE FOR THE ASTRONAUT TO FINISH USING THE DSKY OR (3) BEING DISPLAYED ON THE DSKY
062102,000059:                                                                                                  #            BUT NOT WAITING FOR A RESPONSE.
062103,000060:                                                                                                  #        2.  INACTIVE - A DISPLAY WHICH HAS (1) BEEN ACTIVE BUT WAS INTERRUPTED BY A DISPLAY OF HIGHER PRIORITY,
062104,000061:                                                                                                  #            (2) BEEN PUT INTO THE WAITING LIST AT TIME IT WAS REQUESTED DUE TO THE FACT A HIGHER PRIORITY DISPLAY
062105,000062:                                                                                                  #            WAS ALREADY GOING, (3) BEEN INTERRUPTED BY THE ASTRONAUT (CALLED A PINBRANCH CONDITION, SINCE THIS TYPE
062106,000063:                                                                                                  #            OF INACTIVE DISPLAY IS USUALLY REACTIVATED ONLY BY PINBALL) OR (4) A DISPLAY WHICH HAS FINISHED BUT STILL
062107,000064:                                                                                                  #            HAS INFO SAVED FOR RESTART PURPOSES.

062110,000067:                                                                                                  #        INTERRUPTS -
062111,000068:                                                                                                  #                1.  THE ASTRONAUT CAN INTERRUPT ANY DISPLAY WITH AN EXTERNAL DISPLAY REQUEST.
062112,000069:                                                                                                  #                2.  INTERNAL DISPLAYS CAN NOT BE SENT OUT WHEN THE ASTRONAUT IS USING THE DSKY.
062113,000070:                                                                                                  #                3.  PRIORITY DISPLAYS INTERRUPT ALL OTHER TYPES OF INTERNAL DISPLAYS.  A PRIORITY DISPLAY INTERRUPTING ANOTHER
062114,000071:                                                                                                  #                    PRIORITY DISPLAY WILL CAUSE AN ABORT UNLESS BIT14 IS SET FOR THE LINUS ROUTINE.
062115,000072:                                                                                                  #                4.  A MARK DISPLAY INTERRUPTS ANY NORMAL DISPLAY.
