Source Code

These source-code files were derived originally from the scan from MIT's Dibner Institute's now-discontinued website titled "History of Recent Science and Technology", which in turn was originally created by Gary Neff, though his high-quality scans were unfortunately reduced in legibility when presented online. The markings on the front of the printout imply that it was the original AGC developer Norm Brodeur's copy. A scan by Ron Burkey has superceded it, made from a more-legible copy from the collection of original AGC developer Fred Martin. The source code was transcribed from those images by Ron Burkey. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	ASSEMBLE REVISION 249 OF AGC PROGRAM COLOSSUS BY NASA 2021111-041
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

061480,000040:                                                                                                  #  DISPLAYS CAN BE CLASSIFIED INTO THE FOLLOWING CATEGORIES --
061481,000041:                                                                                                  #        1.  PRIORITY DISPLAYS -- DISPLAYS WHICH TAKE PRIORITY OVER ALL OTHER DISPLAYS.  USUALLY THESE DISPLAYS ARE SENT
061482,000042:                                                                                                  #            OUT UNDER CRITICAL ALARM CONDITIONS.
061483,000043:                                                                                                  #        2.  EXTENDED VERB DISPLAYS -- ALL EXTENDED VERBS AND MARK ROUTINES SHOULD USE EXTENDED VERB (MARK) DISPLAYS.
061484,000044:                                                                                                  #        3.  NORMAL DISPLAYS -- ALL MISSION PROGRAM DISPLAYS WHICH INTERFACE WITH THE ASTRONAUT DURING THE NORMAL
061485,000045:                                                                                                  #            SEQUENCE OF EVENTS.
061486,000046:                                                                                                  #        4.  MISC. DISPLAYS -- ALL DISPLAYS NOT HANDLED BY THE DISPLAY INTERFACE ROUTINES.  THESE INCLUDE SUCH DISPLAYS AS
061487,000047:                                                                                                  #            MM DISPLAYS AND SPECIAL PURPOSE DISPLAYS HANDLED BY PINBALL.
061488,000048:                                                                                                  #        5.  ASTRONAUT INITIATED DISPLAYS -- ALL DISPLAYS INITIATED EXTERNALLY.
061489,000049: 
061490,000050:                                                                                                  #  THE FOLLOWING TERMS ARE USED TO DESCRIBE THE STATUS OF DISPLAYS --
061491,000051:                                                                                                  #        1.  ACTIVE -- THE DISPLAY WHICH IS (1) BEING DISPLAYED TO THE ASTRONAUT AND WAITING FOR A RESPONSE OR
061492,000052:                                                                                                  #            (2) WAITING FIRST IN LINE FOR THE ASTRONAUT TO FINISH USING THE DSKY OR (3) BEING DISPLAYED ON THE DSKY
061493,000053:                                                                                                  #            BUT NOT WAITING FOR A RESPONSE.
061494,000054:                                                                                                  #        2.  INACTIVE -- A DISPLAY WHICH HAS (1) BEEN ACTIVE BUT WAS INTERRUPTED BY A DISPLAY OF HIGHER PRIORITY,
061495,000055:                                                                                                  #            (2) BEEN PUT INTO THE WAITING LIST AT TIME IT WAS REQUESTED DUE TO THE FACT A HIGHER PRIORITY DISPLAY
061496,000056:                                                                                                  #            WAS ALREADY DOING, (3) BEEN INTERRUPTED BY THE ASTRONAUT (CALLED A PINBRANCH CONDITION, SINCE THIS TYPE
061497,000057:                                                                                                  #            OF INACTIVE DISPLAY IS USUALLY REACTIVATED ONLY BY PINBALL) OR (4) A DISPLAY WHICH HAS FINISHED BUT STILL
061498,000058:                                                                                                  #            HAS INFO SAVED FOR RESTART PURPOSES.
061499,000059: 
061500,000060:                                                                                                  #  DISPLAY PRIORITIES WORK AS FOLLOWS --
061501,000061:                                                                                                  #        INTERRUPTS --
061502,000062:                                                                                                  #                1.  THE ASTRONAUT CAN INTERRUPT ANY DISPLAY WITH AN EXTERNAL DISPLAY REQUEST.
061503,000063:                                                                                                  #                2.  INTERNAL DISPLAYS CAN NOT BE SENT OUT WHEN THE ASTRONAUT IS USING THE DSKY.
061504,000064:                                                                                                  #                3.  PRIORITY DISPLAYS INTERRUPT ALL OTHER TYPES OF INTERNAL DISPLAYS.  A PRIORITY DISPLAY INTERRUPTING ANOTHER
061505,000065:                                                                                                  #                    PRIORITY DISPLAY WILL CAUSE AN ABORT UNLESS BIT14 IS SET FOR TEH LINUS ROUTINE.
061506,000066:                                                                                                  #                4.  A MARK DISPLAY INTERRUPTS ANY NORMAL DISPLAY.
061507,000067:                                                                                                  #                5.  A MARK THAT INTERRUPTS A MARK COMPLETELY REPLACES IT.
061508,000068: 
061509,000069:                                                                                                  #        ORDER OF WAITING DISPLAYS --

061511,000071:                                                                                                  #                2.  PRIORITY
061512,000072:                                                                                                  #                3.  INTERRUPTED MARK

061519,000079:                                                                                                  #  THE DISPLAY ROUTINES ARE INTENDED TO SERVE AS AN INTERFACE BETWEEN THE USER AND PINBALL.  THE
061520,000080:                                                                                                  #  FOLLOWING STATEMENTS CAN BE MADE ABOUT NORMAL DISPLAYS AND PRIORITY DISPLAYS (A DESCRIPTION OF MARK ROUTINES
061521,000081:                                                                                                  #  WILL FOLLOW LATER):
061522,000082:                                                                                                  #        1.  ALL ROUTINES THAT END IN R HAVE AN IMMEDIATE RETURN TO THE USER.  FOR ALL FLASHING DISPLAYS THIS RETURN
061523,000083:                                                                                                  #            IS TO THE USER'S CALL CADR +4.  FOR THE ONLY NON-FLASHING IMMEDIATE RETURN DISPLAY (GODSPR) THIS RETURN
061524,000084:                                                                                                  #            IS TO THE USER'S CALLING LOC +1.
061525,000085:                                                                                                  #        2.  ALL ROUTINES NOT ENDING IN R DO NOT DO AN IMMEDIATE RETURN TO THE USER.

061527,000087:                                                                                                  #        4.  ALL ROUTIENS NOT ENDING IN R BRANCH DIRECTLY TO MAKEPLAY WHICH MAKES THESE DISPLAYS A PART OF THE
061528,000088:                                                                                                  #            USER'S JOB.
061529,000089:                                                                                                  #        5.  ALL DISPLAY ROUTIENS ARE CALLED VIA BANKCALL.
061530,000090:                                                                                                  #        6.  TO RESTART A DISPLAY THE USER WILL GENERALLY USE A PHASE OF ONE WITH DESIRED RESTART GROUP (SEE
061531,000091:                                                                                                  #            DESCRIPTION OF RESTARTS).
061532,000092:                                                                                                  #        7.  ALL FLASHING DISPLAYS HAVE 3 RETURNS TO THE USER FROM ASTRONAUT RESPOSES.  A TERMINATE (V34) BRANCHES
061533,000093:                                                                                                  #            TO THE USER'S CALL CADR +1.  A PROCEED (V33) BRANCHES TO THE USER'S CALL CADR +2.  AN ENTER OR RECYCLE
