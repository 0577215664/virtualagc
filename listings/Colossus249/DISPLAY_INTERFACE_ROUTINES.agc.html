Source Code

These source-code files were derived originally from the scan from MIT's Dibner Institute's now-discontinued website titled "History of Recent Science and Technology", which in turn was originally created by Gary Neff, though his high-quality scans were unfortunately reduced in legibility when presented online. The markings on the front of the printout imply that it was the original AGC developer Norm Brodeur's copy. A scan by Ron Burkey has superceded it, made from a more-legible copy from the collection of original AGC developer Fred Martin. The source code was transcribed from those images by Ron Burkey. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	ASSEMBLE REVISION 249 OF AGC PROGRAM COLOSSUS BY NASA 2021111-041
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

061484,000045:                                                                                                  #            SEQUENCE OF EVENTS.
061485,000046:                                                                                                  #        4.  MISC. DISPLAYS -- ALL DISPLAYS NOT HANDLED BY THE DISPLAY INTERFACE ROUTINES.  THESE INCLUDE SUCH DISPLAYS AS
061486,000047:                                                                                                  #            MM DISPLAYS AND SPECIAL PURPOSE DISPLAYS HANDLED BY PINBALL.
061487,000048:                                                                                                  #        5.  ASTRONAUT INITIATED DISPLAYS -- ALL DISPLAYS INITIATED EXTERNALLY.
061488,000049: 
061489,000050:                                                                                                  #  THE FOLLOWING TERMS ARE USED TO DESCRIBE THE STATUS OF DISPLAYS --
061490,000051:                                                                                                  #        1.  ACTIVE -- THE DISPLAY WHICH IS (1) BEING DISPLAYED TO THE ASTRONAUT AND WAITING FOR A RESPONSE OR
061491,000052:                                                                                                  #            (2) WAITING FIRST IN LINE FOR THE ASTRONAUT TO FINISH USING THE DSKY OR (3) BEING DISPLAYED ON THE DSKY
061492,000053:                                                                                                  #            BUT NOT WAITING FOR A RESPONSE.
061493,000054:                                                                                                  #        2.  INACTIVE -- A DISPLAY WHICH HAS (1) BEEN ACTIVE BUT WAS INTERRUPTED BY A DISPLAY OF HIGHER PRIORITY,
061494,000055:                                                                                                  #            (2) BEEN PUT INTO THE WAITING LIST AT TIME IT WAS REQUESTED DUE TO THE FACT A HIGHER PRIORITY DISPLAY
061495,000056:                                                                                                  #            WAS ALREADY DOING, (3) BEEN INTERRUPTED BY THE ASTRONAUT (CALLED A PINBRANCH CONDITION, SINCE THIS TYPE
061496,000057:                                                                                                  #            OF INACTIVE DISPLAY IS USUALLY REACTIVATED ONLY BY PINBALL) OR (4) A DISPLAY WHICH HAS FINISHED BUT STILL
061497,000058:                                                                                                  #            HAS INFO SAVED FOR RESTART PURPOSES.

061499,000060:                                                                                                  #  DISPLAY PRIORITIES WORK AS FOLLOWS --
061500,000061:                                                                                                  #        INTERRUPTS --
061501,000062:                                                                                                  #                1.  THE ASTRONAUT CAN INTERRUPT ANY DISPLAY WITH AN EXTERNAL DISPLAY REQUEST.
061502,000063:                                                                                                  #                2.  INTERNAL DISPLAYS CAN NOT BE SENT OUT WHEN THE ASTRONAUT IS USING THE DSKY.
061503,000064:                                                                                                  #                3.  PRIORITY DISPLAYS INTERRUPT ALL OTHER TYPES OF INTERNAL DISPLAYS.  A PRIORITY DISPLAY INTERRUPTING ANOTHER
061504,000065:                                                                                                  #                    PRIORITY DISPLAY WILL CAUSE AN ABORT UNLESS BIT14 IS SET FOR TEH LINUS ROUTINE.
061505,000066:                                                                                                  #                4.  A MARK DISPLAY INTERRUPTS ANY NORMAL DISPLAY.
061506,000067:                                                                                                  #                5.  A MARK THAT INTERRUPTS A MARK COMPLETELY REPLACES IT.
