Source Code

This is the source-code for the Apollo 9 Command Module's Guidance Computer. These files were derived originally from the scan from MIT's Dibner Institute's now-discontinued website titled "History of Recent Science and Technology", which in turn was originally created by Gary Neff, though his high-quality scans were unfortunately reduced in legibility when presented online. The markings on the front of the printout imply that it was the original AGC developer Norm Brodeur's copy. A scan by Ron Burkey has superceded it, made from a more-legible copy from the collection of original AGC developer Fred Martin. The source code was transcribed from those images by Ron Burkey. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	ASSEMBLE REVISION 249 OF AGC PROGRAM COLOSSUS BY NASA 2021111-041
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

063638,000088:                                                                                                  #  UPFLAG AND DOWNFLAG ARE ENTIRELY GENERAL FLAG SETTING AND CLEARING SUBROUTINES.  USING THEM, WHETHER OR
063639,000089:                                                                                                  #  NOT IN INTERRUPT, ONE MAY SET OR CLEAR ANY SINGLE, NAMED BIT IN ANY ERASABLE REGISTER, SUBJECT OF COURSE TO
063640,000090:                                                                                                  #  EBANK SETTING.  A "NAMED" BIT, AS THE WORD IS USED HERE, IS ANY BIT WITH A NAME FORMALLY ASSIGNED BY THE YUL
063641,000091:                                                                                                  #  ASSEMBLER.
063642,000092: 
063643,000093:                                                                                                  #  AT PRESENT THE ONLY NAMED BITS ARE THOSE IN THE FLAGWORDS.  ASSEMBLER CHANGES WILL MAKE IT POSSIBLE TO
063644,000094:                                                                                                  #  NAME ANY BIT IN ERASABLE MEMORY.
063645,000095: 
063646,000096:                                                                                                  #  CALLING SEQUENCES ARE AS FOLLOWS:-
063647,000097:                                                                                                  #                TC      UPFLAG                  TC      DOWNFLAG
063648,000098:                                                                                                  #                ADRES   NAME OF FLAG            ADRES   NAME OF FLAG

063754,000204:                                                                                                  #  GENTRAN, A BLOCK TRANSFER ROUTINE.
063755,000205:                                                                                                  #  WRITTEN BY D. EYLES
063756,000206:                                                                                                  #  MOD 1 BY KERNAN                               UTILITYM REV 17 11/18/67
063757,000207:                                                                                                  #  MOD 2 BY SCHULENBERG    (REMOVE RELINT)    SKIPPER REV 4 2/28/68
063758,000208: 
063759,000209:                                                                                                  #  THIS ROUTINE IS USEFULL FOR TRANSFERING N CONSECUTIVE ERASABLE OR FIXED QUANTITIES TO SOME OTHER N
063760,000210:                                                                                                  #  CONSECUTIVE ERASABLE LOCATIONS.  IF BOTH BLOCKS OF DATA ARE IN SWITCHABLE EBANKS, THEY MUST BE IN THE SAME ONE.
063761,000211: 
063762,000212:                                                                                                  #  GENTRAN IS CALLABLE IN A JOB AS WELL AS A RUPT.  THE CALLING SEQUENCE IS:
063763,000213:                                                                                                  #        I       CA      N-1             # OF QUANTITIES MINUS ONE.
063764,000214:                                                                                                  #        I +1    TC      GENTRAN         IN FIXED-FIXED.
063765,000215:                                                                                                  #        I +2    ADRES   L               STARTING ADRES OF DATA TO BE MOVED.
063766,000216:                                                                                                  #        I +3    ADRES   M               STARTING ADRES OF DUPLICATION BLOCK.
063767,000217:                                                                                                  #        I +4                            RETURNS HERE.
063768,000218: 
063769,000219:                                                                                                  #  GENTRAN TAKES 25 MCT'S (300 MICROSECONDS) PER ITEM + 5 MCT'S (60 MICS) FOR ENTERING AND EXITING.
063770,000220:                                                                                                  #  A, L AND ITEMP1 ARE NOT PRESERVED.
