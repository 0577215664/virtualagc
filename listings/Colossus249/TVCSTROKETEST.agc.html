Source Code

These source-code files were derived originally from the scan from MIT's Dibner Institute's now-discontinued website titled "History of Recent Science and Technology", which in turn was originally created by Gary Neff, though his high-quality scans were unfortunately reduced in legibility when presented online. The markings on the front of the printout imply that it was the original AGC developer Norm Brodeur's copy. A scan by Ron Burkey has superceded it, made from a more-legible copy from the collection of original AGC developer Fred Martin. The source code was transcribed from those images by Ron Burkey. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	ASSEMBLE REVISION 249 OF AGC PROGRAM COLOSSUS BY NASA 2021111-041
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

041719,000040:                                                                                                  #  NAME          STROKE TEST PACKAGE             (INCLUDING INITIALIZATION PACKAGE)
041720,000041:                                                                                                  #  ORIGINAL CODING BY OLSSON                     LOG SECTION....STROKE TEST PACKAGE
041721,000042:                                                                                                  #  MOD BY ENGEL                                  DATE....21 MARCH, 1967
041722,000043: 
041723,000044:                                                                                                  #  FUNCTIONAL DESCRIPTION....
041724,000045:                                                                                                  #        STROKE TEST PACKAGE GENERATES A WAVEFORM DESIGNED TO EXCITE BENDING
041725,000046:                                                                                                  #        STRKTSTI (STROKE TEST INITIALIZATION) IS CALLED AS A JOB BY VB68.
041726,000047:                                                                                                  #                IT INITIALIZES ALL ERASABLES REQD FOR A STROKE TEST, AND
041727,000048:                                                                                                  #                THEN TESTS FOR AN 80MS DAP.  IF 80MS IT SETS STROKER = ESTROKER
041728,000049:                                                                                                  #                FOR AN IMMEDIATE STROKE TEST, OTEHRWISE IT MERELY ENABLES
041729,000050:                                                                                                  #                A STROKE TEST BY SETTING STROKER TO -0.  THE STROKE TEST
041730,000051:                                                                                                  #                THEN AWAITS SWITCHOVER TO THE 80MS DAP WHEREUPON IT IS
041731,000052:                                                                                                  #                ENABLED AFTER AN ADDITIONAL 4 SECOND DELAY TO AVOID
041732,000053:                                                                                                  #                THE SWITCHOVER TRANSIENTS (SEE STRKCALL, STRKUP IN
041733,000054:                                                                                                  #                TVCEXECUTIVE)
041734,000055:                                                                                                  #        HACK (STROKE TEST) GENERATES THE WAVEFORM BY DUMPING PULSE BURSTS
041735,000056:                                                                                                  #                OF PROPER SIGN AND IN PROPER SEQUENCE DIRECTLY INTO
041736,000057:                                                                                                  #                TVCPITCH, WORKING IN CONJUNCITON WITH BOTH PITCH AND YAW
041737,000058:                                                                                                  #                TVC DAPS, WITH INTERMEDIAT WAITLIST CALLS.  NOTE, HOWEVER
041738,000059:                                                                                                  #                THAT THE STROKE TEST IS PERFORMED ONLY IN THE PITCH AXIS.
041739,000060:                                                                                                  #                AN EXAMPLE WAVEFORM IS GIVEN BELOW, TO DEMONSTRATE STROKE-
041740,000061:                                                                                                  #                TEST PARAMETER SELECTION.
041741,000062:                                                                                                  #        RESTARTS CAUSE TEST TO BE TERMINATED.  ANOTHER V68 REQD IF TEST
041742,000063:                                                                                                  #                IS TO BE RE-RUN.
041743,000064:                                                                                                  #        PULSE BURST SIZE IS PAD-LOADED (ESTROKER) SO THAT AMPLITUDE OF
041744,000065:                                                                                                  #                WAVEFORM CAN BE CHANGED.  THERE ARE TEN PULSE BURSTS IN
041745,000066:                                                                                                  #                THE HALF-AMPLITUDE OF THE FIRST FREQUENCY SET IN THE
041746,000067:                                                                                                  #                STANDARD WAVEFORM.  AMPLITUDE IS 10(ESTROKER)(1/42.15),
041747,000068:                                                                                                  #                NOMINALLY 50/42.15 = 1.185 DEG
041748,000069: 
041749,000070:                                                                                                  #  CALLING SEQUENCE....
041750,000071:                                                                                                  #        EXTENDED VERB 68 SETS UP STRKTSTI JOB
041751,000072:                                                                                                  #        PITCH AND YAW TVCDAPS, FINDING STROKER NON-ZERO, DO A "TC HACK"
041752,000073:                                                                                                  #        AN INTERNALLY-GENERATED WAITLIST CALL ENTERS AT "HACKWLST"
041753,000074: 
041754,000075:                                                                                                  #  NORMAL EXIT MODES....
041755,000076:                                                                                                  #        TC BUNKER ("Q" IF ENTRY FROM DAP, "TCTSKOVR" IF FROM WAITLIST) LIST
041756,000077: 
041757,000078:                                                                                                  #  SUBROUTINES CALLED....
041758,000079:                                                                                                  #        WAITLIST
041759,000080: 
041760,000081:                                                                                                  #  ALARM OR ABORT EXIT MODES....
041761,000082:                                                                                                  #        NONE
041762,000083: 
041763,000084:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED....
041764,000085:                                                                                                  #        ESTROKER (PAD-LOAD)
041765,000086:                                                                                                  #        STROKER, CADDY, REVS, CARD, N
041766,000087: 
041767,000088:                                                                                                  #  OUTPUT....
041768,000089:                                                                                                  #        STRKTSTI....INITIALIZATION FOR STROKE TEST
041769,000090:                                                                                                  #        HACK, HACKWLST....PULSE BURSTS INTO TVCPITCH VIA "ADS"
041770,000091:                                                                                                  #                          RESETS STROKER = +0 WHEN TEST COMPLETED
041771,000092: 
041772,000093:                                                                                                  #  DEBRIS....
041773,000094:                                                                                                  #        N = CADDY = +0, CARD = -0, REVS = -1
041774,000095:                                                                                                  #        BUNKER
