Source Code

This is the source-code for the Apollo 9 Command Module's Guidance Computer. These files were derived originally from the scan from MIT's Dibner Institute's now-discontinued website titled "History of Recent Science and Technology", which in turn was originally created by Gary Neff, though his high-quality scans were unfortunately reduced in legibility when presented online. The markings on the front of the printout imply that it was the original AGC developer Norm Brodeur's copy. A scan by Ron Burkey has superceded it, made from a more-legible copy from the collection of original AGC developer Fred Martin. The source code was transcribed from those images by Ron Burkey. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	ASSEMBLE REVISION 249 OF AGC PROGRAM COLOSSUS BY NASA 2021111-041
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

042133,000046:                                                                                                  #  NAME          STROKE TEST PACKAGE             (INCLUDING INITIALIZATION PACKAGE)
042134,000047:                                                                                                  #  ORIGINAL CODING BY OLSSON                     LOG SECTION....STROKE TEST PACKAGE
042135,000048:                                                                                                  #  MOD BY ENGEL                                  DATE....21 MARCH, 1967

042137,000050:                                                                                                  #  FUNCTIONAL DESCRIPTION....
042138,000051:                                                                                                  #        STROKE TEST PACKAGE GENERATES A WAVEFORM DESIGNED TO EXCITE BENDING
042139,000052:                                                                                                  #        STRKTSTI (STROKE TEST INITIALIZATION) IS CALLED AS A JOB BY VB68.
042140,000053:                                                                                                  #                IT INITIALIZES ALL ERASABLES REQD FOR A STROKE TEST, AND
042141,000054:                                                                                                  #                THEN TESTS FOR AN 80MS DAP.  IF 80MS IT SETS STROKER = ESTROKER
042142,000055:                                                                                                  #                FOR AN IMMEDIATE STROKE TEST, OTHERWISE IT MERELY ENABLES
042143,000056:                                                                                                  #                A STROKE TEST BY SETTING STROKER TO -0.  THE STROKE TEST
042144,000057:                                                                                                  #                THEN AWAITS SWITCHOVER TO THE 80MS DAP WHEREUPON IT IS
042145,000058:                                                                                                  #                ENABLED AFTER AN ADDITIONAL 4 SECOND DELAY TO AVOID
042146,000059:                                                                                                  #                THE SWITCHOVER TRANSIENTS (SEE STRKCALL, STRKUP IN
042147,000060:                                                                                                  #                TVCEXECUTIVE)
042148,000061:                                                                                                  #        HACK (STROKE TEST) GENERATES THE WAVEFORM BY DUMPING PULSE BURSTS
042149,000062:                                                                                                  #                OF PROPER SIGN AND IN PROPER SEQUENCE DIRECTLY INTO
042150,000063:                                                                                                  #                TVCPITCH, WORKING IN CONJUNCITON WITH BOTH PITCH AND YAW
042151,000064:                                                                                                  #                TVC DAPS, WITH INTERMEDIATE WAITLIST CALLS.  NOTE, HOWEVER
042152,000065:                                                                                                  #                THAT THE STROKE TEST IS PERFORMED ONLY IN THE PITCH AXIS.
042153,000066:                                                                                                  #                AN EXAMPLE WAVEFORM IS GIVEN BELOW, TO DEMONSTRATE STROKE-
042154,000067:                                                                                                  #                TEST PARAMETER SELECTION
042155,000068:                                                                                                  #        RESTARTS CAUSE TEST TO BE TERMINATED.  ANOTHER V68 REQD IF TEST
042156,000069:                                                                                                  #                IS TO BE RE-RUN.
042157,000070:                                                                                                  #        PULSE BURST SIZE IS PAD-LOADED (ESTROKER) SO THAT AMPLITUDE OF
042158,000071:                                                                                                  #                WAVEFORM CAN BE CHANGED.  THERE ARE TEN PULSE BURSTS IN
042159,000072:                                                                                                  #                THE HALF-AMPLITUDE OF THE FIRST FREQUENCY SET IN THE
042160,000073:                                                                                                  #                STANDARD WAVEFORM.  AMPLITUDE IS 10(ESTROKER)(1/42.15),
042161,000074:                                                                                                  #                NOMINALLY 50/42.15 = 1.185 DEG
042162,000075: 
042163,000076:                                                                                                  #  CALLING SEQUENCE....
042164,000077:                                                                                                  #        EXTENDED VERB 68 SETS UP STRKTSTI JOB
042165,000078:                                                                                                  #        PITCH AND YAW TVCDAPS, FINDING STROKER NON-ZERO, DO A ..TC HACK..
042166,000079:                                                                                                  #        AN INTERNALLY-GENERATED WAITLIST CALL ENTERS AT ..HACKWLST..
042167,000080: 
042168,000081:                                                                                                  #  NORMAL EXIT MODES....
042169,000082:                                                                                                  #        TC BUNKER (..Q.. IF ENTRY FROM DAP, ..TCTSKOVR.. IF FROM WAITLIST) LIST
042170,000083: 
042171,000084:                                                                                                  #  SUBROUTINES CALLED....
042172,000085:                                                                                                  #        WAITLIST
042173,000086: 
042174,000087:                                                                                                  #  ALARM OR ABORT EXIT MODES....
042175,000088:                                                                                                  #        NONE
042176,000089: 
042177,000090:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED....
042178,000091:                                                                                                  #        ESTROKER (PAD-LOAD)
042179,000092:                                                                                                  #        STROKER, CADDY, REVS, CARD, N
042180,000093: 
042181,000094:                                                                                                  #  OUTPUT....
042182,000095:                                                                                                  #        STRKTSTI...INITIALIZATION FOR STROKE TEST
042183,000096:                                                                                                  #        HACK, HACKWLST...PULSE BURSTS INTO TVCPITCH VIA..ADS..
042184,000097:                                                                                                  #                          RESETS STROKER = +0 WHEN TEST COMPLETED

042188,000101:                                                                                                  #        BUNKER
