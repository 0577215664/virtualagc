Source Code

These source-code files were derived originally from the scan from MIT's Dibner Institute's now-discontinued website titled "History of Recent Science and Technology", which in turn was originally created by Gary Neff, though his high-quality scans were unfortunately reduced in legibility when presented online. The markings on the front of the printout imply that it was the original AGC developer Norm Brodeur's copy. A scan by Ron Burkey has superceded it, made from a more-legible copy from the collection of original AGC developer Fred Martin. The source code was transcribed from those images by Ron Burkey. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	ASSEMBLE REVISION 249 OF AGC PROGRAM COLOSSUS BY NASA 2021111-041
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

059556,000093:                                                                                                  #  PHASCHNG IS THE MAIN WAY OF MAKING PHASE CHANGES FOR RESTARTS.  THERE ARE THREE FORMS OF PHASCHNG, KNOWN AS TYPE
059557,000094:                                                                                                  #  A, TYPE B, AND TYPE C.  THEY ARE ALL CALLED AS FOLLOWS, WHERE OCT XXXXX CONTAINS THE PHASE INFORMATION,
059558,000095:                                                                                                  #                TC      PHASCHNG
059559,000096:                                                                                                  #                OCT     XXXXX
059560,000097:                                                                                                  #  TYPE A IS CONCERNED WITH FIXED PHAS CHANGES, THAT IS, PHASE INFORMATION THAT IS STORED PERMANENTLY.  THESE
059561,000098:                                                                                                  #  OPTIONS ARE, WEHRE G STANDS FOR A GROUP AND .X FOR THE PHASE,
059562,000099:                                                                                                  #        G.0             INACTIVE, WILL NOT PERMIT A GROUP G RESTART
059563,000100:                                                                                                  #        G.1             WILL CAUSE THE LAST DISPLAY TO BE REACTIVATED, USED MAINLY IN MANNED FLIGHTS
059564,000101:                                                                                                  #        G.EVEN          A DOUBLE TABLE RESTART, CAN CAUSE ANY COMBINATION OF TWO JOBS, TASKS, AND/OR
059565,000102:                                                                                                  #                        LONGCALL TO BE RESTARTED.
059566,000103:                                                                                                  #        G.ODD NOT .1    A SINGLE TABLE RESTART, CAN CAUSE EITHER A JOB, TASK, OR LONGCALL RESTART.
059567,000104: 
059568,000105:                                                                                                  #  THIS INFORMATION IS PUT INTO THE OCTAL WORD AFTER TC PHASCHNG AS FOLLOWS
059569,000106:                                                                                                  #        TL0 00P PPP PPP GGG
059570,000107:                                                                                                  #  WHERE EACH LETTER OR NUMBER STANTS FOR A BIT.  THE G'S STAND FOR THE GROUP, OCTAL 1-7, THE P'S FOR THE PHASE,
059571,000108:                                                                                                  #  OCTAL 0 - 127.  0'S MUST BE 0.  IF ONE WISHES TO HAVE THE TBASE OF GROUP G TO BE SET AT THIS TIME,
059572,000109:                                                                                                  #  T IS SET TO 1, OTHERWISE IT IS SET TO 0.  SIMILARLY IF ONE WISHES TO SET LONGBASE, THEN L IS SET TO 1, OTHERWISE
059573,000110:                                                                                                  #  IT IS SET TO 0.  SOME EXAMLES,
059574,000111:                                                                                                  #                TC      PHASCHNG        # THIS WILL CAUSE GROUP 3 TO BE SET TO 0,
059575,000112:                                                                                                  #                OCT     00003           # MAKING GROUP 3 INACTIVE
059576,000113: 
059577,000114:                                                                                                  #                TC      PHASCHNG        # IF A RESTART OCCURS THIS WOULD CAUSE
059578,000115:                                                                                                  #                OCT     00012           # GROUP 2 TO RESTART THE LAST DISPLAY
059579,000116: 
059580,000117:                                                                                                  #                TC      PHASCHNG        # THIS SETS THE TBASE OF GROUP 4 AND IN
059581,000118:                                                                                                  #                OCT     40064           # CASE OF A RESTART WOULD START UP THE TWO
059582,000119:                                                                                                  #                                        # THINGS LOCATED IN THE DOUBLE 4.6 RESTART
059583,000120:                                                                                                  #                                        # LOCATION.
059584,000121: 
059585,000122:                                                                                                  #                TC      PHASCHNG        # THIS SETS LONGBASE AND UPON A RESTART
059586,000123:                                                                                                  #                OCT     20135           # CAUSES 5.13 TO BE RESTARTED (SINCE
059587,000124:                                                                                                  #                                        # LONGBASE WAS SET THIS SINGLE ENTRY
059588,000125:                                                                                                  #                                        # SHOULD BE A LONGCALL)
059589,000126: 
059590,000127:                                                                                                  #                TC      PHASCHNG        # SINCE BOTH TBASE4 AND LONGBASE ARE SET,
059591,000128:                                                                                                  #                OCT     60124           # 4.12 SHOULD CONTAIN BOTH A TASK AND A
059592,000129:                                                                                                  #                                        # LONGCALL TO BE RESTARTED
059593,000130: 
059594,000131:                                                                                                  #  TYPE C PHASCHNG CONTAINS THE VARIABLE TYPE OF PHASCHNG INFORMATION.  INSTEAD OF THE INFORMATION BEING IN A
059595,000132:                                                                                                  #  PERMANENT FORM, ONE STORES THE DESIRED RESTART INFORMATION IN A VARIABLE LOCATION.  THE BITS ARE AS FOLLOWS,
059596,000133:                                                                                                  #        TL0 1AD XXX CJW GGG
059597,000134:                                                                                                  #  WHERE EACH LETTER OR NUMBER STANDS FOR A BIT.  THE G'S STAND FOR THE GROUP, OCTAL 1 - 7.  IF THE RESTART IS TO
059598,000135:                                                                                                  #  BE BY WAITLIST, W IS SET TO 1, IF IT IS A JOB, J IS SET TO 1, IF IT IS A LONGCALL, C IS SET TO 1.  ONLY ONE OF
059599,000136:                                                                                                  #  THESE THREE BITS MAY BE SET.  X'S ARE IGNORED, 1 MUST BE 1, AND 0 MUST BE 0.  AGAIN T STANDS FOR THE TBASE,
059600,000137:                                                                                                  #  Page 1397
059601,000138:                                                                                                  #  AND L FOR LONGBASE.  THE BITS A AND D ARE CONCERNED WITH THE VARIABLE INFORMATION.  IF D IS SET TO 1, A PRIORITY
059602,000139:                                                                                                  #  OR DELTA TIME WILL BE READ FROM THE NEXT LOCATION AFTER THE OCTAL INFORMATION., IF THIS IS TO BE INDIRECT, THAT
059603,000140:                                                                                                  #  IS, THE NAME OF A LOCATION CONTAINING THE INFORMATION (DELTA TIME ONLY), THEN THIS IS GIVEN AS THE -GENADR OF
059604,000141:                                                                                                  #  THAT LOCATION WHICH CONTAINS THE DELTA TIME.  IF THE OLD PRIORITY OR DELTA TIME IS TO BE USED, THAT WHICH IS
059605,000142:                                                                                                  #  ALREADY IN THE VARIABLE STORAGE, THEN D IS SET TO 0.  NEXT THE A BIT IS USED.  IF IT IS SET TO 0, THE ADDRESS
059606,000143:                                                                                                  #  THAT WOULD BE RESTARTED DURING A RESTART IS THE NEXT LOCATION AFTER THE PHASE INFORMATION, THAT IS, EITHER
059607,000144:                                                                                                  #  (TC PHASCHNG) +2 OR +3, DEPENDING ON WHETHER D HAD BEEN SET OR NOT.  IF A IS SET TO 1, THEN THE ADDRESS THAT
059608,000145:                                                                                                  #  WOULD BE RESTARTED IS THE 2CADR THAT IS READ FROM THE NEXT TWO LOCATION.  EXAMPLES,
059609,000146:                                                                                                  #        AD      TC      PHASCHNG        # THIS WOULD CAUSE LOCATION AD +3 TO BE
059610,000147:                                                                                                  #        AD+1    OCT     05023           # RESTARTED BY GROUP THREE WITH A PRIORITY
059611,000148:                                                                                                  #        AD+2    OCT     23000           # OF 23.  NOTE UPON RETURNING IT WOULD
059612,000149:                                                                                                  #        AD+3                            # ALSO GO TO AD+3
059613,000150: 
059614,000151:                                                                                                  #        AD      TC      PHASCHNG        # GROUP 1 WOULD CAUSE CALLCALL TO BE
059615,000152:                                                                                                  #        AD+1    OCT     27441           # BE STARTED AS A LONGCALL FROM THE TIME
059616,000153:                                                                                                  #        AD+2    -GENADR DELTIME         # STORED IN LONGBASE (LONGBASE WAS SET) BY
059617,000154:                                                                                                  #        AD+3    2CADR   CALLCALL        # A DELTA TIME STORED IN DELTIME.  THE
059618,000155:                                                                                                  #        AD+4                            # BBCON OF THE 2CADR SHOULD CONTAIN THE E
059619,000156:                                                                                                  #        AD+5                            # BANK OF DELTIME.  PHASCHNG RETURNS TO
059620,000157:                                                                                                  #                                        # LOCATION AD+5
059621,000158: 
059622,000159:                                                                                                  #  NOTE THAT IF A VARIABLE PRIORITY IS GIVEN FOR A JOB, THE JOB WILL BE RESTARTED AS A NOVAC IF THE PRIORITY IS
059623,000160:                                                                                                  #  NEGATIVE, AS A FINDVAC IF TEH PRIORITY IS POSITIVE.
059624,000161: 
059625,000162:                                                                                                  #  TYPE B PHASCHNG IS A COMBINATION OF VARIABLE AND FIXED PHASE CHANGES.  IT WILL START UP A JOB AS INDICATED
059626,000163:                                                                                                  #  BELOW AND ALSO START UP ONE FIXED RESTART, THAT IS EITHER AN G.1 OR A G.ODD OR THE FIRST ENTRY OF G.EVEN
059627,000164:                                                                                                  #  DOUBLE ENTRY.  THE BIT INFORMATION IS AS FOLLOW,
059628,000165:                                                                                                  #        TL1 DAP PPP PPP GGG
059629,000166:                                                                                                  #  WHERE EACH LETTER OR NUMBER STANDS FOR A BIT.  THE G'S STAND FOR THE GROUP, OCTAL 1 - 7, THE P'S FOR THE FIXED
059630,000167:                                                                                                  #  PHASE INFORMATION, OCTAL 0 - 127.  1 MUST BE 1.  AND AGAIN T STANDS FOR THE TBASE AND L FOR LONGBASE.  D THIS
059631,000168:                                                                                                  #  TIME STANDS ONLY FOR PRIORITY SINCE THIS WILL BE CONSIDERED A JOB, AND IT MUST BE GIVEN DIRECTLY IF GIVEN.
059632,000169:                                                                                                  #  AGAIN A STANDS FOR THE ADDRESS OF THE LOCATION TO BE RESTARTED, 1 IF THE 2CADR IS GIVEN, OR 0 IF IT IS TO BE
059633,000170:                                                                                                  #  THE NEXT LOCATION.  (THE RETURN LOCATION OF PHASCHNG) EXAMPLES,
059634,000171:                                                                                                  #        AD      TC      PHASCHNG        # TBASE IS SET AND A RESTART CAUSE GROUP 3
059635,000172:                                                                                                  #        AD+1    OCT     56043           # TO START THE JOB AJOBAJOB WITH PRIORITY
059636,000173:                                                                                                  #        AD+2    OCT     31000           # 31 AND THE FIRST ENTRY OF 3.4SPOT (WE CAN
059637,000174:                                                                                                  #        AD+3    2CADR   AJOBAJOB        # ASSUME IT IS A TASK SINCE WE SET TBASE3)
059638,000175:                                                                                                  #        AD+4                            # UPON RETURN FROM PHASCHNG CONTROL WOULD
059639,000176:                                                                                                  #        AD+5                            # GO TO AD+5
059640,000177: 
059641,000178:                                                                                                  #        AD      TC      PHASCHNG        # UPON A RESTART THE LAST DISPLAY WOULD BE
059642,000179:                                                                                                  #        AD+1    OCT     10015           # RESTARTED AND A JOB WITH THE PREVIOUSLY
059643,000180:                                                                                                  #        AD+2                            # STORED PRIORITY WOULD BE BEGUN AT AD+2
059644,000181:                                                                                                  #                                        # BY MEANS OF GROUP 5
059645,000182:                                                                                                  #  Page 1398
059646,000183:                                                                                                  #  THE NOVAC-FINDVAC CHOICE FOR JOBS HOLDS HERE ALSO -- NEGATIVE PRIORITY CAUSES A NOVAC CALL, POSITIVE A FINDVAC.
