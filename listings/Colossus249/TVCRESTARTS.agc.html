Source Code

These source-code files were derived originally from the scan from MIT's Dibner Institute's now-discontinued website titled "History of Recent Science and Technology", which in turn was originally created by Gary Neff, though his high-quality scans were unfortunately reduced in legibility when presented online. The markings on the front of the printout imply that it was the original AGC developer Norm Brodeur's copy. A scan by Ron Burkey has superceded it, made from a more-legible copy from the collection of original AGC developer Fred Martin. The source code was transcribed from those images by Ron Burkey. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	ASSEMBLE REVISION 249 OF AGC PROGRAM COLOSSUS BY NASA 2021111-041
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

040557,000040:                                                                                                  #  NAME....TVCRESTART PACKAGE, CONSISTING OF REDOTVC, ENABL1, 2, CMDSOUT, PHSCHK2, ETC.
040558,000041:                                                                                                  #  LOG SECTION....TVCRESTART PACKAGE             SUBROUTINE....DAPCSM
040559,000042:                                                                                                  #  MOD BY ENGEL                                  DATE....19 OCT, 1967
040560,000043: 
040561,000044:                                                                                                  #  FUNCTIONAL DESCRIPTION....
040562,000045: 
040563,000046:                                                                                                  #       *RESTART-PROCESS THE TVC DAPS, INCLUDING PITCHDAP, YAWDAP,
040564,000047:                                                                                                  #        TVCEXECUTIVE, (?) DAP, TVCINIT4, TVCDAPON, AND STROKE TEST.
040565,000048: 
040566,000049:                                                                                                  #       *TVC RESTARTS DESERVE SPECIAL CONSIDERATION IN SEVERAL AREAS.
040567,000050:                                                                                                  #        RESTART DOWN-TIME IS IMPORTANT BECAUSE OF THE TRANSIENTS INTRODUCED
040568,000051:                                                                                                  #        BY TEH THRUST VECTOR RETURN TO THE ACTUATOR MECHANICAL NULLS
040569,000052:                                                                                                  #        FOLLOWING TVC AND OPTICS-ERROR-COUNTER-DISENABLES (CHANNEL 12).
040570,000053:                                                                                                  #        TVC USES A MIXTURE OF WAITLIST, T5, T6, AND JOB CALLS.  THERE IS
040571,000054:                                                                                                  #        FILTER MEMORY (UP TO 7TH ORDER) TO BE PROTECTED IF WILD TRANSIENTS
040572,000055:                                                                                                  #        ARE TO BE AVOIDED.  SEVERAL COUNTERS ARE INVOLVED FOR TIMING TVC
040573,000056:                                                                                                  #        EVENTS SUCH AS SWITCHOVER AND STROKE TEST STARTUPS AND RE-STARTUPS.
040574,000057:                                                                                                  #        THE TVC GAINS ARE DECREMENTED.  THE GIMBAL TRIM ESTIMATORS AND THE
040575,000058:                                                                                                  #        BODY AXIS ATTITUDE ERROR INTEGRATORS INVOLVE DIGITAL SUMMATION.
040576,000059:                                                                                                  #        DIGITAL DIFFERENTIATORS ARE INVOLVED IN THE BODY AXIS RATE ESTIMATIONS
040577,000060:                                                                                                  #        AND IN THE OUTPUTTING OF ACTUATOR COMMANDS.  THERE IS AN
040578,000061:                                                                                                  #        OFFSET-TRACKER-FILTER TO PROTECT.  ETC., ETC.
040579,000062: 
040580,000063:                                                                                                  #       *THOSE QUANTITIES WHICH MUST BE PROTECTED ARE STORED IN TEMPORARY
040581,000064:                                                                                                  #        REGISTERS AS THEY ARE COMPUTED, FOR UPDATING THE REAL REGISTERS
040582,000065:                                                                                                  #        DURING COPYCYCLES.
040583,000066: 
040584,000067:                                                                                                  #       *THE SEVERAL COPYCYCLES ARE EACH PROTECTED BY PHASE POINTS AT THEIR
040585,000068:                                                                                                  #        BEGINNING AND AT THEIR TERMINATION.  THE PHASE POINTS ARE SIMPLY
040586,000069:                                                                                                  #        "INCR" INSTRUCTIONS, EITHER "INCR TVCEXPHS" FOR COPYCYCLES
040587,000070:                                                                                                  #        IN THE TVCEXECUTIVE, OR "INCR TVCPHASE" FOR THE PITCH AND YAW
040588,000071:                                                                                                  #        COPYCYCLES.  INDEXING ON EACH OF THESE POINTERS THEN PERMITS A
040589,000072:                                                                                                  #        RETURN TO THE APPROPRIATE RESTART POINTS.
040590,000073: 
040591,000074:                                                                                                  #       *IF A RESTART OCCURS DURING EITHER COPYCYCLE, THAT COPYCYCLE IS
040592,000075:                                                                                                  #        COMPLETED.  THEN THE NORMAL TVCINIT4....DAPINIT....PITCHDAP STARTUP
040593,000076:                                                                                                  #        SEQUENCE IS CALLED UPON TO GET THINGS GOING AGAIN.
040594,000077: 
040595,000078:                                                                                                  #       *TVC-ENABLE AND OPTICS-ERROR-COUNTER ENABLE MUST BE SET ASAP
040596,000079:                                                                                                  #        (ALLOWING FOR PROCEDURAL DELAYS).  THEN THE ENGINES ARE COMMANDED
040597,000080:                                                                                                  #        TO THE P,YACTOFF TRIM VALUES.  THE DAPS ARE THEN READY TO GO ON THE
040598,000081:                                                                                                  #        AIR, WITH THE REGULAR STARTUP SEQUENCE, EITHER AT MRCLEAN FOR A
040599,000082:                                                                                                  #        COMPLETE INITIALIZATION OR AT TVCIIT4 FOR A PARTIAL INITIALIZATION.
040600,000083: 
040601,000084:                                                                                                  #       *FOR RESTARTS PRIOR TO THE SETTING OF THE T5 BITS IN IGNOVER THE
040602,000085:                                                                                                  #        PRE40.6 SECTION OF S40.6 TAKES CARE OF RE-ESTABLISHING TRIMS.
040603,000086: 
040604,000087:                                                                                                  #       *IF A RESTART OCCURS DURING THE TVCEXEC....TVCEXFIN SEQUENCE THE
040605,000088:                                                                                                  #        COMPUTATIONS WILL BE COMPLETED, STARTING AT THE APPROPRIATE RESTART
040606,000089:                                                                                                  #        POINT, AFTER THE DAPS ARE READY TO GO ON THE AIR.
040607,000090: 
040608,000091:                                                                                                  #       *IF A RESTART OCCURS PRIOR TO TVCINIT4 (TVCPHAS = -1) E.G. DURING
040609,000092:                                                                                                  #        THE EARLY DAP INITIALIZATION PHASE, THE DAP STARTUP SEQUENCE IS
040610,000093:                                                                                                  #        ENTERED AT MRCLEAN FOR A FULL INITIALIZATION.
040611,000094: 
040612,000095:                                                                                                  #       *RESTARTS ARE NOT CRITICAL TO THE ROLL DAP PERFORMANCES HENCE THE
040613,000096:                                                                                                  #        ROLL DAP IS MERELY RESTARTED.
040614,000097: 
040615,000098:                                                                                                  #       *RESTARTS DURING A STROKE TEST (STROKER IS NON-ZERO) WILL CAUSE THE
040616,000099:                                                                                                  #        STROKE TEST TO BE TERMINATED.  A NEW V68 ENTRY WILL BE REQUIRED
040617,000100:                                                                                                  #  Page 921
040618,000101:                                                                                                  #        TO GET IT GOING AGAIN (NO AUTOMATIC RESTART).
040619,000102:                                                                                                  #   
040620,000103:                                                                                                  #       *REDOTVC IS REACHED FOLLOWING ANY RESTART WHICH FINDS THE T5 BITS
040621,000104:                                                                                                  #        (BITS 15,14 OF FLAGWRD6) SET FOR TVC.  IGNOVER PREPARES TVCPHASE = -1
040622,000105:                                                                                                  #        AND TVC EXPHS = 0 JUST BEFORE SETTING THESE BITS, JUST BEFORE
040623,000106:                                                                                                  #        MAKING THE T5 CALL TO TVCDAPON.  T.V.N.G. TAKES OVER THE T5 CLOCK
040624,000107:                                                                                                  #        TO CALL RCSUP/RCSDAPON WHICH RESETS THE T5 BITS (FOR RCS) ON A
040625,000108:                                                                                                  #        NORMAL SHUTDOWN.
040626,000109: 
040627,000110:                                                                                                  #  CALLING SEQUENCE....T5, IN PARTICULAR BY ELRSKIP OF FRESH START/RESTART
040628,000111: 
040629,000112:                                                                                                  #  NORMAL EXIT MODES....RESUME, NOQRSM, POSTJUMP (TO TVCINIT4 OR MRCLEAN)
040630,000113: 
040631,000114:                                                                                                  #  ALARM OR ABORT EXIT MODES....NONE
