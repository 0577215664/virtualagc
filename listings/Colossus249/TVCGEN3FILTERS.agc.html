Source Code

This is the source-code for the Apollo 9 Command Module's Guidance Computer. These files were derived originally from the scan from MIT's Dibner Institute's now-discontinued website titled "History of Recent Science and Technology", which in turn was originally created by Gary Neff, though his high-quality scans were unfortunately reduced in legibility when presented online. The markings on the front of the printout imply that it was the original AGC developer Norm Brodeur's copy. A scan by Ron Burkey has superceded it, made from a more-legible copy from the collection of original AGC developer Fred Martin. The source code was transcribed from those images by Ron Burkey. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	ASSEMBLE REVISION 249 OF AGC PROGRAM COLOSSUS BY NASA 2021111-041
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

043029,000050:                                                                                                  #  FUNCTIONAL DESCRIPTION....
043030,000051: 
043031,000052:                                                                                                  #  THE GEN3DAP FILTER PACKAGE IS DESIGNED TO PROVIDE FLEXIBLE, LAST-MINUTE CHANGEABLE DIGITAL AUTOPILOT
043032,000053:                                                                                                  #  FILTERS FOR LEM-OFF FLIGHT.  GROUNDRULES FOR THE DESIGN AND USE OF THE PACKAGE ARE AS FOLLOWS.....
043033,000054: 
043034,000055:                                                                                                  #        1. FILTER COEFFICIENTS AND GAINS IN ERASABLE MEMORY
043035,000056:                                                                                                  #        2. UP TO THIRD-ORDER NUMERATOR OR DENOMINATOR
043036,000057:                                                                                                  #        3. OPERATIONAL FIT WITHIN THE STRUCTURE OF THE REGULAR LEM-ON DAP CODING
043037,000058:                                                                                                  #        4. DENOMINATOR POLES INSIDE THE Z-PLANE UNIT CIRCLE
043038,000059:                                                                                                  #        5. NUMERATOR ZEROS INSIDE THE Z-PLANE DOUBLE-UNIT CIRCLE
043039,000060:                                                                                                  #        6. HIGH FREQUENCY (BODE) GAIN LESS THAN 8ASCREVS, OR 8.6380088 DEG/DEG
043040,000061: 
043041,000062:                                                                                                  #  THE FILTERS ARE SHOWN IN THE FOLLOWING DIAGRAMS.....
043042,000063:                                                                                                  #  PITCH GEN3DAP FILTER..

043044,000065:                                                                                                  #                                                             ********

043049,000070:                                                                                                  #  ***************  -----------------------------------  **********( X )*********************

043069,000090:                                                                                                  #  THE IMPLEMENTING EQUATIONS FOR THESE FILTERS ARE AS FOLLOWS.....
043070,000091: 
043071,000092:                                                                                                  #        PITCH GEN3DAP....                               YAW GEN3DAP....
043072,000093:                                                                                                  #                NPD = (B+4) KPGEN3 NP0                          NYD = (B+4) KYGEN3 NY0
043073,000094:                                                                                                  #                NP0 = AP0 EP           + 4(Z-1) NP1             NY0 = AY0 EY           + 4(Z-1) NY1
043074,000095:                                                                                                  #                NY1 = AP1 EP - BP1 NP0 +  (Z-1) NP2             NY1 = AY1 EY - BY1 NY0 +  (Z-1) NY2
043075,000096:                                                                                                  #                NP2 = AP2 EP - BP2 NP0 +  (Z-1) NP3             NY2 = AY2 EY - BY2 NY0 +  (Z-1) NY3
043076,000097:                                                                                                  #                NP3 = AP3 EP - BP3 NP0                          NY3 = AY3 EY - BY3 NY0
043077,000098: 
043078,000099:                                                                                                  #  FILTER INPUTS EP AND EY ARE PICKED UP FROM REGULAR LEM-ON CODING AT ERRBTMP (UPPER WORD ONLY), THUS ARE
043079,000100:                                                                                                  #  SINGLE PRECISION QUANTITIES SCALED AT B-1 REVS.  FILTER OUTPUTS NPD AND NYD ARE LEFT IN DOUBLE PRECISION AT
043080,000101:                                                                                                  #  CMDTMP, SCALED AT 1 ASCREV, READY FOR OUTPUT PROCESSING VIA REGULAR LEM-ON CODING AT ..P,YOFFSET..
043081,000102:                                                                                                  #  FOLLOWING OUTPUT PROCESSING, RETURN TO THE GEN3DAP FILTERS IS MADE FOR CALCULATION OF THE REMAINING NODES
043082,000103:                                                                                                  #  NP1 TO NP3, OR NY1 TO NY3.  GEN3DAP FILTERS THEN RETURN TOTHE LEM-ON CODING AT ..DELBARP,Y..  FOR RESPECTIVE
043083,000104:                                                                                                  #  OFFSET-TRACKER-FILTER COMPUTATIONS AND COPYCYCLES.  NOTE THE EQUIVALENCES...NP1TMP=J5TMP, NP1=J5,
043084,000105:                                                                                                  #  NP2TMP=NSUMTMP, NP2=PNSUM, NP3TMP=DSUMTMP, NP3=PDSUM, WITH CORRESPONDING RELATIONS FOR YAW.  THUS THE COPY-
043085,000106:                                                                                                  #  CYCLE PCOPY, FROM THE GEN3DAP STANDPOINT, IS EFFECTIVE FROM PMISC-3 TO ITS END AT TC Q.  YCOPY FROM YMISC-3.
043086,000107:                                                                                                  #  SCALING OF THE FILTER NODES, COEFFICIENTS, AND GAINS WITHIN THE AGC IS AS FOLLOWS.....
043087,000108: 
043088,000109:                                                                                                  #        QUANTITY        QUANTITY        PHYS. UNITS     MAX. VALUE      SCALE AT (FOR)
043089,000110: 
043090,000111:                                                                                                  #        EP              EY              REVS            1/8             B-1 REV         (CDU SCALING)
043091,000112:                                                                                                  #        
043092,000113:                                                                                                  #        NP0             NY0             REVS            (B+1)           B+1 REV
043093,000114:                                                                                                  #        NP1             NY1             REVS            (B+3)           B+3 REV
043094,000115:                                                                                                  #        NP2             NY2             REVS            (B+3)           B+3 REV
043095,000116:                                                                                                  #        NP3             NY3             REVS            (B+3)           B+3 REV
043096,000117: 
043097,000118:                                                                                                  #        NPD             NYD             ASC REVS        (1)             1 ASCREV        (ACTUATOR CDU SCALING)
043098,000119:                                                                                                  #        
043099,000120:                                                                                                  #        KPGEN3          KYGEN3          ASCREV/REV      (8)             B+3 ASCREV/REV
043100,000121: 
043101,000122:                                                                                                  #        AP0             AY0             DIMLESS.        1               B+2
043102,000123:                                                                                                  #        AP1             AY1             DIMLESS.        6               B+4
043103,000124:                                                                                                  #        AP2             AY2             DIMLESS.        12              B+4
043104,000125:                                                                                                  #        AP3             AY3             DIMLESS.        8               B+4
043105,000126: 
043106,000127:                                                                                                  #        BP1             BY1             DIMLESS.        3               B+2
043107,000128:                                                                                                  #        BP2             BY2             DIMLESS.        3               B+2
043108,000129:                                                                                                  #        BP3             BY3             DIMLESS.        1               B+2
043109,000130: 
043110,000131:                                                                                                  #  FILTER COEFFICIENTS, GAINS, AND NODES ARE HELD IN DOUBLE PRECISION (ERASABLE) TO PERMIT CONSERVATIVE
043111,000132:                                                                                                  #  SCALING AND YET OFFSET TRUNCATION LOSSES.  THIS APPEARS NECESSARY IF FILTER FLEXIBILITY IS TO BE MAINTAINED.
043112,000133:                                                                                                  #  COMPUTATION TIME IS NOT CRITICAL.
