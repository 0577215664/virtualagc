Source Code

These source-code files were derived originally from the scan from MIT's Dibner Institute's now-discontinued website titled "History of Recent Science and Technology", which in turn was originally created by Gary Neff, though his high-quality scans were unfortunately reduced in legibility when presented online. The markings on the front of the printout imply that it was the original AGC developer Norm Brodeur's copy. A scan by Ron Burkey has superceded it, made from a more-legible copy from the collection of original AGC developer Fred Martin. The source code was transcribed from those images by Ron Burkey. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	ASSEMBLE REVISION 249 OF AGC PROGRAM COLOSSUS BY NASA 2021111-041
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

042621,000040:                                                                                                  #  PROGRAM NAME.... GEN3DAP FILTERS, CONSISTING OF NP0NODE, NP1NODE, NY0NODE, NY1NODE, ETC.
042622,000041:                                                                                                  #  LOG SECTION.... GEN3DAP FILTERS               SUBROUTINE....DAPCSM
042623,000042:                                                                                                  #  MOD BY ENGEL                                  20 OCT, 1967
042624,000043: 
042625,000044: 
042626,000045:                                                                                                  #  FUNCTIONAL DESCRIPTION....
042627,000046: 
042628,000047:                                                                                                  #  THE GEN3DAP FILTER PACKAGE IS DESIGNED TO PROVIDE FLEXIBLE, LAST-MINUTE CHANGEABLE DIGITAL AUTOPILOT
042629,000048:                                                                                                  #  FILTERS FOR LEM-OFF FLIGHT.  GROUNDRULES FOR THE DESIGN AND USE OF THE PACKAGE ARE AS FOLLOWS.....
042630,000049: 
042631,000050:                                                                                                  #        1. FILTER COEFFICIENTS AND GAINS IN ERASABLE MEMORY
042632,000051:                                                                                                  #        2. UP TO THIRD-ORDER NUMERATOR OR DENOMINATOR
042633,000052:                                                                                                  #        3. OPERATIONAL FIT WITHIN THE STRUCTURE OF TEH REGULAR LEM-ON DAP CODING
042634,000053:                                                                                                  #        4. DENOMINATOR POLES INSIDE THE Z-PLANE UNIT CIRCLE
042635,000054:                                                                                                  #        5. NUMERATOR ZEROES INSIDE THE Z-PLANE DOUBLE-UNIT CIRCLE
042636,000055:                                                                                                  #        6. HIGH FREQUENCY (BODE) GAIN LESS THAN 8ASCREVS, OR 8.6380088 DEG/DEG
042637,000056: 
042638,000057:                                                                                                  #  THE FILTERS ARE SHOWN IN THE FOLLOWING DIAGRAMS.....
042639,000058:                                                                                                  #  PITCH GEN3DAP FILTER:
042640,000059:                                                                                                  #                                                              KPGEN3
042641,000060:                                                                                                  #                                                             ********
042642,000061:                                                                                                  #                *****************************************           *
042643,000062:                                                                                                  #                *                                       *           *
042644,000063:                                                                                                  #                *               -1        -2        -3  *           *
042645,000064:                                                                                                  #   EP = ERRBTMP *    APO + AP1 Z   + AP2 Z   + AP3 Z    *    NPO          NP0 = CMDTMP  **
042646,000065:                                                                                                  #  ***************  -----------------------------------  **********( X )*********************
042647,000066:                                                                                                  #                *               -1        -2        -3  *                               **
042648,000067:                                                                                                  #                *      1 + BP1 Z   + BP2 Z   + BP3 Z    *
042649,000068:                                                                                                  #                *                                       *
042650,000069:                                                                                                  #                *****************************************
042651,000070: 
042652,000071:                                                                                                  #  YAW GEN3DAP FILTER:
042653,000072:                                                                                                  #                                                              KYGEN3
042654,000073:                                                                                                  #                                                             ********
042655,000074:                                                                                                  #                *****************************************           *

042666,000085:                                                                                                  #  THE IMPLEMENTING EQUATIONS FOR THESE FILTERS ARE AS FOLLOWS.....
042667,000086: 
042668,000087:                                                                                                  #        PITCH GEN3DAP.....                              YAW GEN3DAP....
042669,000088:                                                                                                  #                NPD = (B+4) KPGEN3 NP0                          NYD = (B+4) KYGEN3 NY0
042670,000089:                                                                                                  #                NP0 = AP0 EP           + 4(Z-1) NP1             NY0 = AY0 EY           + 4(Z-1) NY1
042671,000090:                                                                                                  #                NP1 = AP1 EP - BP1 NP0 +  (Z-1) NP2             NY1 = AY1 EY - BY1 NY0 +  (Z-1) NY2
042672,000091:                                                                                                  #                NP2 = AP3 EP - BP2 NP0 +  (Z-1) NP3             NY2 = AY2 EY - BY2 NY0 +  (Z-1) NY3
042673,000092:                                                                                                  #                NP3 = AP3 EP - BP3 NP0                          NY3 = AY3 EY - BY3 NY0
042674,000093: 
042675,000094:                                                                                                  #  FILTER INPUTS EP AND EY ARE PICKED UP FROM REGULAR LEM-ON CODING AT ERRBTMP (UPPER WORD ONLY), THUS ARE
042676,000095:                                                                                                  #  SINGLE PRECISION QUANTITIES SCALED AT B-1 REVS.  FILTER OUTPUTS NPD AND NYD ARE LEFT IN DOUBLE PRECISION AT
042677,000096:                                                                                                  #  CMDTMP, SCALED AT 1 ASCREV, READY FOR OUTPUT PROCESSING VIA REGULAR LEM-ON CODING AT "P,YOFFSET"
042678,000097:                                                                                                  #  FOLLOWING OUTPUT PROCESSING, RETURN TO THE GEN3DAP FILTERS IS MADE FOR CALCULATION OF THE REMAINING NODES
042679,000098:                                                                                                  #  NP1 TO NP3, OR NY1 TO NY3.  GEN3DAP FILTERS THEN RETURN TOTHE LEM-ON CODING AT "DELBARP,Y"  FOR RESPECTIVE
042680,000099:                                                                                                  #  OFFSET-TRACKER-FILTER COMPUTATIONS AND COPYCYCLES.  NOTE THE EQUIVALENCES...NP1TMP=J5TMP, NP1=J5,
042681,000100:                                                                                                  #  NP2TMP=NSUMTMP, NPT2=PNSUM, NP3TMP=DSUMTMP, NP3=PDSUM, WITH CORRESPONDING RELATIONS FOR YAW.  THUS THE COPY-
042682,000101:                                                                                                  #  CYCLE PCOPY, FROM THE GEN3DAP STANDPOINT, IS EFFECTIVE FROM PMISC-3 TO ITS END AT TC Q.  YCOPY FROM YMISC-3.
042683,000102:                                                                                                  #  SCALING OF THE FILTER NODES, COEFFICIENTS, AND GAINS WITHIN THE AGS IS AS FOLLOWS.....
042684,000103: 
042685,000104:                                                                                                  #        QUANTITY        QUANTITY        PHYS. UNITS     MAX. VALUE      SCALE AT (FOR)
042686,000105: 
042687,000106:                                                                                                  #        EP              EY              REVS            1/8             B-1 REV         (CDU SCALING)
042688,000107:                                                                                                  #        
042689,000108:                                                                                                  #        NP0             NY0             REVS            (B+1)           B+1 REV
042690,000109:                                                                                                  #        NP1             NY1             REVS            (B+3)           B+3 REV
042691,000110:                                                                                                  #        NP2             NY2             REVS            (B+3)           B+3 REV
042692,000111:                                                                                                  #        NP3             NY3             REVS            (B+3)           B+3 REV
042693,000112: 
042694,000113:                                                                                                  #        NPD             NYD             ASC REVS        (1)             1 ASCREV        (ACTUATOR CDU SCALING)
042695,000114:                                                                                                  #        
042696,000115:                                                                                                  #        KPGEN3          KYGEN3          ASCREV/REV      (8)             B+3 ASCREV/REV
042697,000116: 
042698,000117:                                                                                                  #        AP0             AY0             DIMLESS.        1               B+2
042699,000118:                                                                                                  #        AP1             AY1             DIMLESS.        6               B+4
042700,000119:                                                                                                  #        AP2             AY2             DIMLESS.        12              B+4
042701,000120:                                                                                                  #        AP3             AY3             DIMLESS         8               B+4
042702,000121: 
042703,000122:                                                                                                  #        BP1             BY1             DIMLESS.        3               B+2
042704,000123:                                                                                                  #        BP2             BY2             DIMLESS.        3               B+2
042705,000124:                                                                                                  #        BP3             BY3             DIMLESS.        1               B+2
042706,000125: 
042707,000126:                                                                                                  #  FILTER COEFFICIENTS, GAINS, AND NODES ARE HELD IN DOUBLE PRECISION (ERASABLE) TO PERMIT CONSERVATIVE
042708,000127:                                                                                                  #  SCALING AND YET OFFSET TRUNCATION LOSSES.  THIS APPEARS NECESSARY IF FILTER FLEXIBILITY IS TO BE MAINTAINED.
042709,000128:                                                                                                  #  COMPUTATION TIME IS NOT CRITICAL.
042710,000129: 
042711,000130:                                                                                                  #  CALLING SEQUENCE....
042712,000131:                                                                                                  #     
042713,000132:                                                                                                  #       *TC POSTJUMP....
042714,000133:                                                                                                  #        CADR NP0NODE, NP1, NY0, NY1.  SPECIFICALLY, FROM PITCHDAP OR YAWDAP
042715,000134:                                                                                                  #        (TVCDAP), AT P1FILJMP, P2FILJMP, Y1FILJMP, Y2FILJMP
042716,000135: 
042717,000136:                                                                                                  #  NORMAL EXIT MODE....
042718,000137: 
042719,000138:                                                                                                  #       *TC POSTJUMP....
042720,000139:                                                                                                  #        CADR (POFFSET, DELBARP), (YOFFSET, DELBARY).  I.E., RETURNS TO
042721,000140:                                                                                                  #        PITCHDAP OR YAWDAP AT APPROPRIATE ENTRY POINT.
042722,000141: 
042723,000142:                                                                                                  #  ALARM OR ABORT EXIT MODES....NONE
042724,000143: 
042725,000144:                                                                                                  #  SUBROUTINES CALLED.... NONE
042726,000145: 
042727,000146:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED....
042728,000147: 
042729,000148:                                                                                                  #       *AP0(SP),AP1(DP),...AP3(DP), (PITCH AND YAW) NUMERATOR COEFFICIENTS
042730,000149:                                                                                                  #        (PAD LOADS)
042731,000150: 
042732,000151:                                                                                                  #       *BP1(DP),...BP3(DP), (PITCH AND YAW) DENOMINATOR COEFFICIENTS
042733,000152:                                                                                                  #        (PAD LOADS)
042734,000153: 
042735,000154:                                                                                                  #       *KPGEN3 (S40.15 OF R03)
042736,000155: 
042737,000156:                                                                                                  #  OUTPUT....
042738,000157: 
042739,000158:                                                                                                  #       *CMDTMP (NPD, NYD) FOR OUTPUT PROCESSING BY PITCHDAP OR YAWDAP
042740,000159: 
042741,000160:                                                                                                  #       *OTHER FILTER NODES
042742,000161:
