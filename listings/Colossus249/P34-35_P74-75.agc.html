Source Code

These source-code files were derived originally from the scan from MIT's Dibner Institute's now-discontinued website titled "History of Recent Science and Technology", which in turn was originally created by Gary Neff, though his high-quality scans were unfortunately reduced in legibility when presented online. The markings on the front of the printout imply that it was the original AGC developer Norm Brodeur's copy. A scan by Ron Burkey has superceded it, made from a more-legible copy from the collection of original AGC developer Fred Martin. The source code was transcribed from those images by Ron Burkey. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	ASSEMBLE REVISION 249 OF AGC PROGRAM COLOSSUS BY NASA 2021111-041
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

019748,000050:                                                                                                  #                        THE ACTIVE/PASSIVE VEHICLE LOS AT TIG (TPI).
019749,000051:                                                                                                  #                (B)     CENTRAL ANGLE OF TRANSFER (CENTANG) FROM TIG (TPI) TO
019750,000052:                                                                                                  #                        INTERCEPT TIME (TIG (TPF)).
019751,000053:                                                                                                  #        (2)     TO CALCULATE TIG (TPI) GIVEN E OR E GIVEN TIG (TPI).
019752,000054:                                                                                                  #        (3)     TO CALCULATE THESE PARAMETERS BASED UPON MANEUVER DATA
019753,000055:                                                                                                  #                APPROVED AND KEYED INTO THE DSKY BY THE ASTRONAUT.
019754,000056:                                                                                                  #        (4)     TO DISPLAY TO THE ASTRONAUT AND THE GROUND CERTAIN DEPENDENT
019755,000057:                                                                                                  #                VARIABLES ASSOCIATED WITH THE MANEUVER FOR APPROVAL BY THE
019756,000058:                                                                                                  #                ASTRONAUT/GROUND.
019757,000059:                                                                                                  #        (5)     TO STORE THE TPI TARGET PARAMETERS FOR USE BY THE DESIRED
019758,000060:                                                                                                  #                THRUSTING PROGRAM.

019760,000062:                                                                                                  #  ASSUMPTIONS
019761,000063:                                                                                                  #        (1)     LM ONLY -- THIS PROGRAM IS BASED UPON PREVIOUS COMPLETION OF
019762,000064:                                                                                                  #                THE CONSTANT DELTA ALTITUDE (CDH) PROGRAM (P33/P73).
019763,000065:                                                                                                  #                THEREFORE --
019764,000066:                                                                                                  #                (A)     AT A SELECTED TPI TIME (NOW IN STORAGE) THE LINE OF SIGHT
019765,000067:                                                                                                  #                        BETWEEN THE ACTIVE AND PASSIVE VEHICLES WAS SELECTED TO BE
019766,000068:                                                                                                  #                        A PRESCRIBED ANGLE (E) (NOW IN STORAGE) FROM THE
019767,000069:                                                                                                  #                        HORIZONTAL PLANE DEFINED BY THE ACTIVE VEHICLE POSITION.
019768,000070:                                                                                                  #                (B)     THE TIME BETWEEN CDH IGNITION AND TPI IGNITION WAS
019769,000071:                                                                                                  #                        COMPUTED TO BE GREATER THAN 10 MINUTES.
019770,000072:                                                                                                  #                (C)     THE VARIATION OF TEH ALTITUDE DIFFERENCE BETWEEN THE
019771,000073:                                                                                                  #                        ORBITS WAS MINIMIZED.
019772,000074:                                                                                                  #                (D)     THE PERICENTER ALTITUDES OF ORBITS FOLLOWING CSI AND
019773,000075:                                                                                                  #                        CDH WERE COMPUTED TO BE GREATER THAN 35,000 FT FOR LUNAR
019774,000076:                                                                                                  #  Page 455
019775,000077:                                                                                                  #                        ORBIT OR 85 NM FOR EARTH ORBIT.
019776,000078:                                                                                                  #                (E)     THE CSI AND CDH MANEUVERS WERE ASSUMED TO BE PARALLEL TO
019777,000079:                                                                                                  #                        THE PLANE OF THE PASSIVE VEHICLE ORBIT.  HOWEVER, CREW
019778,000080:                                                                                                  #                        MODIFICATION OF DELTA V (LV) COMPONENTS MAY HAVE RESULTED
019779,000081:                                                                                                  #                        IN AN OUT-OF-PLANE MANEUVER.
019780,000082:                                                                                                  #        (2)     STATE VECTOR UPDATED BY P27 ARE DISALLOWED DURING AUTOMATIC
019781,000083:                                                                                                  #                STATE VECTOR UPDATING INITIATED BY P20 (SEE ASSUMPTION (4)).
019782,000084:                                                                                                  #        (3)     THIS PROGRAM MUST BE DONE OVER A TRACKING STATION FOR REAL
019783,000085:                                                                                                  #                TIME GROUND PARTICIPATION IN DATA INPUT AND OUTPUT.  COMPUTED
019784,000086:                                                                                                  #                VARIABLES MAY BE STORED FOR LATER VERIFICATION BY THE GROUND.
019785,000087:                                                                                                  #                THESE STORAGE CAPABILITIES ARE LIMITED ONLY TO THE PARAMETERS
019786,000088:                                                                                                  #                FOR ONE THRUSTING MANEUVER AT A TIME EXCEPT FOR CONCENTRIC
019787,000089:                                                                                                  #                FLIGHT PLAN MANEUVER SEQUENCES.
019788,000090:                                                                                                  #        (4)     THE RENDEZVOUS RADAR MAY OR MAY NOT BE USED TO UPDATE THE LM
019789,000091:                                                                                                  #                OR CSM STATE VECTORS FOR THIS PROGRAM.  IF RADAR USE IS
019790,000092:                                                                                                  #                DESIRED THE RADAR WAS TURNED ON AND LOCKED ON THE CSM BY
019791,000093:                                                                                                  #                PREVIOUS SELECTION OF P20.  RADAR SIGHTING MARKS WILL BE MADE
019792,000094:                                                                                                  #                AUTOMATICALLY APPROXIMATELY ONCE A MINUTE WHEN ENABLED BY THE
019793,000095:                                                                                                  #                TRACK AND UPDATE FLAGS (SEE P20).  THE RENDEZVOUS TRACKING
019794,000096:                                                                                                  #                MARK COUNTER IS ZEROED BY TEH SELECTION OF P20 AND AFTER EACH
019795,000097:                                                                                                  #                THRUSTING MANEUVER.
019796,000098:                                                                                                  #        (5)     THE ISS NEED NOT BE ON TO COMPLETE THIS PROGRAM.
019797,000099:                                                                                                  #        (6)     THE OPERATION OF THE PROGRAM UTILIZES THE FOLLOWING FLAGS --
019798,000100:

019800,000102:                                                                                                  #                        DOING RENDEZVOUS THRUSTING MANEUVERS TO THE PROGRAM WHICH
019801,000103:                                                                                                  #                        CALCULATES THE MANEUVER PARAMETERS.  SET AT THE START OF
019802,000104:                                                                                                  #                        EACH RENDEZVOUS PRE-THRUSTING PROGRAM.
019803,000105:

019809,000111:                                                                                                  #                        PROGRAM SELECTED AFTER COMPLETION OF THIS PROGRAM.

019812,000114:                                                                                                  #                HAVE BEEN COMPLETELY SPECIFIED, TEH VALUE OF THE ACTIVE
019813,000115:                                                                                                  #                VEHICLE CENTRAL ANGLE OF TRANSFER IS COMPUTED AND STURED.

019819,000121:                                                                                                  #  Page 456
019820,000122:                                                                                                  #                170 TO 190 DEGREES.  IF THE ANGLE IS WITHIN THIS ZONE THE

019822,000124:                                                                                                  #                UPON DELTA V AND EXPECTED MANEUVER TIME.
019823,000125:                                                                                                  #        (8)     THIS PROGRAM IS SELECTED BY TEH ASTRONAUT BY DSKY ENTRY --
