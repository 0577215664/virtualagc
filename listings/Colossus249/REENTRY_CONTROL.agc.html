These source-code files were derived originally from the scan from MIT's Dibner Institute's now-discontinued website titled "History of Recent Science and Technology", which in turn was originally created by Gary Neff, though his high-quality scans were unfortunately reduced in legibility when presented online. The markings on the front of the printout imply that it was the original AGC developer Norm Brodeur's copy. A scan by Ron Burkey has superceded it, made from a more-legible copy from the collection of original AGC developer Fred Martin. The source code was transcribed from those images by Ron Burkey. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	ASSEMBLE REVISION 249 OF AGC PROGRAM COLOSSUS BY NASA 2021111-041
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

035555,000176:                                                                                                  #  GOTOADDR CONTAINS THE ADDRESS OF THE ROLL COMMAND EQUATIONS TO THE CURRENT PHASE OF
035556,000177:                                                                                                  #  RE-ENTRY.  SEQUENCING IS AS FOLLOWS:
035557,000178: 
035558,000179:                                                                                                  #  INITROLL      ADDRESS IS SET HERE INITIALLY.  HOLDS INITIAL ROLL ATTITUDE UNTIL KAT IS EXCEEDED.  THEN HOLDS NEW ROLL
035559,000180:                                                                                                  #                ATTITUDE UNTIL  VRTHRESH  IS EXCEEDED.  THEN BRANCHES TO
035560,000181: 
035561,000182:                                                                                                  #  HUNTEST       THIS SECTION CHECKS TO SEE IF THE PREDICTED RANGE AT NOMINAL   L/D FROM PRESENT CONDITIONS IS LESS
035562,000183:                                                                                                  #                THAN THE DESIRED RANGE.
035563,000184:                                                                                                  #                        IF NOT --- A ROLL COMMAND IS GENERATED BY THE CONSTANT DRAG CONTROLLER.
035564,000185:                                                                                                  #                        IF SO  --- CONTROL AND GOTOADDR ARE SET TO UPCONTRL.
035565,000186:                                                                                                  #                USUALLY NO ITERATION IS INVOLVED EXCEPT IF THE RANGE DESIRED IS TOO LONG ON THE FIRST PASS THROUGH
035566,000187:                                                                                                  #                HUNTEST.
035567,000188: 
035568,000189:                                                                                                  #  UPCONTRL      CONTROLS ROLL DURING THE SUPER-CIRCULAR PHASE.  UPCONTRL IS TERMINATED EITHER
035569,000190:                                                                                                  #                        (A) WHEN THE DRAG (AS MEASURED BY THE PIPAS) FALLS BELOW Q7, OR
035570,000191:                                                                                                  #                        (B) IF RDOT IS NEGATIVE AND REFERENCE VL EXCEEDS V.
035571,000192:                                                                                                  #                IN CASE (A),  GOTOADDR  IS SET TO  KEP2  AND IN CASE (B), TO  PREDICT3  SKIPPING THE KEPLER PHASE OF
035572,000193:                                                                                                  #                ENTRY.
