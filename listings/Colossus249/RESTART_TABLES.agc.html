Source Code

These source-code files were derived originally from the scan from MIT's Dibner Institute's now-discontinued website titled "History of Recent Science and Technology", which in turn was originally created by Gary Neff, though his high-quality scans were unfortunately reduced in legibility when presented online. The markings on the front of the printout imply that it was the original AGC developer Norm Brodeur's copy. A scan by Ron Burkey has superceded it, made from a more-legible copy from the collection of original AGC developer Fred Martin. The source code was transcribed from those images by Ron Burkey. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	ASSEMBLE REVISION 249 OF AGC PROGRAM COLOSSUS BY NASA 2021111-041
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

008647,000041:                                                                                                  #  RESTART TABLES
008648,000042:                                                                                                  #  --------------
008649,000043: 
008650,000044:                                                                                                  #  THERE ARE TWO FORMS OF RESTART TABLES FOR EACH GROUP.  THEY ARE KNOWN AS THE EVEN RESTART TABLES AND THE ODD
008651,000045:                                                                                                  #  RESTART TABLES.  THE ODD TABLES HAVE ONLY ONE ENTRY OF THREE LOCATIONS WHILE THE EVEN TABLES HAVE TWO ENTRIES
008652,000046:                                                                                                  #  EACH USING THREE LOCATIONS.  THE INFORMATION AS TO WHETHER IT IS A JOB, WAITLIST, OR A LONGCALL IS GIVEN BY THE
008653,000047:                                                                                                  #  WAY THINGS ARE PUT IN TO THE TABLES.
008654,000048: 
008655,000049:                                                                                                  #  A JOB HAS ITS PRIORITY STORED IN A PRDTTAB OF THE CORRECT PHASE SPOT -- A POSITIVE PRIORITY INDICATES A
008656,000050:                                                                                                  #  FINDVAC JOB, A NEGATIVE PRIORITY A NOVAC.  THE 2CADR OF THE JOB IS STORED IN THE CADRTAB.
008657,000051:                                                                                                  #  FOR EXAMPLE,
008658,000052: 
008659,000053:                                                                                                  #                5.7SPOT         OCT     23000
008660,000054:                                                                                                  #                                2CADR   SOMEJOB
008661,000055: 
008662,000056:                                                                                                  #  A RESTART OF GROUP 5 WITH PHASE SEVEN WOULD THEN CAUSE SOMEJOB TO BE RESTARTED AS A FINDVAC WITH PRIORITY 23.
008663,000057: 
008664,000058:                                                                                                  #                5.5SPOT         OCT     -2300
008665,000059:                                                                                                  #                                2CADR   ANYJOB
008666,000060: 
008667,000061:                                                                                                  #  HERE A RESTART OF GROUP 5 WITH PHASE 7 WOULD CAUSE ANYJOB TO BE RESTARTED AS A NOVAC WITH PRIORITY 23.
008668,000062:                                                                                                  #  A LONGCALL HAS ITS GENADR OF ITS 2CDR STORED NEGATIVELY AND ITS BBCON STORED POSITIVELY.  IN ITS PRDTTAB IS
008669,000063:                                                                                                  #  PLACED THE LOCATION OF A DP REGISTER THAT CONTAINS THE DELTA TIME THAT LONGCALL HAD BEEN ORIGINALLY STARTED
008670,000064:                                                                                                  #  WITH.  EXAMPLE,
008671,000065: 
008672,000066:                                                                                                  #                3.6SPOT         GENADR  DELTAT
008673,000067:                                                                                                  #                                -GENADR LONGTASK
008674,000068:                                                                                                  #                                BBCON   LONGTASK
008675,000069: 
008676,000070:                                                                                                  #                                OCT     31000
008677,000071:                                                                                                  #                                2CADR   JOBAGAIN
008678,000072: 
008679,000073:                                                                                                  #  THIS WOULD START UP LONGTASK AT THE APPROPRIATE TIME, OR IMMEDIATELY IF THE TIME HAD ALREADY PASSED.  IT SHOULD
008680,000074:                                                                                                  #  BE NOTED THAT IF DELTAT IS IN A SWITCHED E BANK, THIS INFORMATION SHOULDD BE IN THE BBCON OF THE 2CADR OF THE
008681,000075:                                                                                                  #  TASK.  FROM ABOVE, WE SEE THAT THE SECOND PART O FTHIS PHASE WOULD BE STARTED AS A JOB WITH A PRIORITY OF 31.
008682,000076: 
008683,000077:                                                                                                  #  WAITLIST CALLS ARE IDENTIFIED BY THE FCT THAT THEIR 2CADR IS STORED NEGATIVELY.  IF PRDTTAB OF THE PHASE SPOT
008684,000078:                                                                                                  #  IS POSITIVE, THEN IT CONTAINS THE DELTA TIME, IF PRDTTAB IS NEGATIVE THEN IT IS THE -GENADR OF AN ERASABLE
008685,000079:                                                                                                  #  IF THE TIME IS STORED INDIRECTLY, THE BBCON MUST CONTAIN THE NECESSARY E BANK INFORMATION IF APPLICABLE.  WITH
008686,000080:                                                                                                  #  WAITLIST WE HAVE ONE FURTHER OPTION, IF -0 IS STORED IN PRDTTAB, IT WILL CAUSE AN IMMEDIATE RESTART OF THE
008687,000081:                                                                                                  #  TASK.  EXAMPLES,
008688,000082: 
008689,000083:                                                                                                  #                                OCT     77777           # THIS WILL CAUSE AN IMMEDIATE RESTART
008690,000084:                                                                                                  #                                -2CADR  ATASK           # OF THE TASK "ATASK"
008691,000085: 
008692,000086:                                                                                                  #                                DEC     200             # IF THE TIME OF THE 2 SECONDS SINCE DUMMY
008693,000087:                                                                                                  #                                -2CADR  DUMMY           # WAS PUT ON THE WAITLIST IS UP, IT WILL BEGIN
008694,000088:                                                                                                  #                                                        # IN 10 MS, OTHERWISE IT WILL BEGIN WHEN
008695,000089:                                                                                                  #                                                        # IT NORMALLY WOULD HAVE BEGUN.
