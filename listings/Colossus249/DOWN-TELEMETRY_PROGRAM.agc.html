Source Code

These source-code files were derived originally from the scan from MIT's Dibner Institute's now-discontinued website titled "History of Recent Science and Technology", which in turn was originally created by Gary Neff, though his high-quality scans were unfortunately reduced in legibility when presented online. The markings on the front of the printout imply that it was the original AGC developer Norm Brodeur's copy. A scan by Ron Burkey has superceded it, made from a more-legible copy from the collection of original AGC developer Fred Martin. The source code was transcribed from those images by Ron Burkey. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	ASSEMBLE REVISION 249 OF AGC PROGRAM COLOSSUS BY NASA 2021111-041
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

046962,000039:                                                                                                  #  PROGRAM NAME -- DOWN TELEMETRY PROGRAM
046963,000040:                                                                                                  #  MOD NO. -- 0          TO COMPLETELY REWRITE THE DOWN TELEMETRY PROGRAM AND DOWNLINK ERASABLE DUMP PROGRAM FOR THE
046964,000041:                                                                                                  #                        PURPOSE OF SAVING APPROXIMATELY 150 WORDS OF CORE STORAGE.
046965,000042:                                                                                                  #                        THIS CHANGE REQUIRES AN ENTIRELY NEW METHOD OF SPECIFYING DOWNLINK LISTS.  REFER TO DOWNLINK
046966,000043:                                                                                                  #                        LISTS LOG SECTION FOR MORE DETAILS.  HOWEVER THIS CHANGES WILL NOT AFFECT THE GROUND PROCESSING
046967,000044:                                                                                                  #                        OF DOWN TELEMETRY DATA.
046968,000045:                                                                                                  #  MOD BY -- KILROY, SMITH, DEWITT
046969,000046:                                                                                                  #  DATE -- 02 OCT 67
046970,000047:                                                                                                  #  AUTHORS -- KILROY, SMITH, DWWITT, DEWOLF, FAGIN
046971,000048:                                                                                                  #  LOG SECTION -- DOWN-TELEMETRY PROGRAM
046972,000049: 
046973,000050:                                                                                                  #  FUNCTIONAL DESCRIPTION -- THIS ROUTINE IS INITIATED BY TELEMETRY END
046974,000051:                                                                                                  #        PULSE FROM THE DOWNLINK TELEMETRY CONVERTER.  THIS PULSE OCCURS
046975,000052:                                                                                                  #        AT 50 TIMES PER SEC (EVERY 20 MS) THEREFORE DODOWNTM IS
046976,000053:                                                                                                  #        EXECUTED AT THESE RATES.  THIS ROUTINE SELECTS THE APPROPRIATE
046977,000054:                                                                                                  #        AGC DATA TO BE TRANSMITTED DOWNLINK AND LOADS IT INTO OUTPUT
046978,000055:                                                                                                  #        CHANNELS 34 AND 35.  THE INFORMATION IS THEN GATED OUT FROM THE
046979,000056:                                                                                                  #        LGC IN SERIAL FASHION.
046980,000057: 
046981,000058:                                                                                                  #        THIS PROGRAM IS CODED FOR A 2 SECOND DOWNLIST.  SINCE DOWNRUPTS
046982,000059:                                                                                                  #        OCCUR EVERY 20 MS AND 2 AGC COMPUTER WORDS CAN BE PLACED IN
046983,000060:                                                                                                  #        CHANNELS 34 AND 35 DURING EACH DOWNRUPT THE PROGRAM IS CAPABLE
046984,000061:                                                                                                  #        OF SENDING 200 AGC WORDS EVERY 2 SECONDS.
046985,000062: 
046986,000063:                                                                                                  #  CALLING SEQUENCE -- NONE
046987,000064:                                                                                                  #        PROGRAM IS ENTERED VIA TCF DODOWNTM WHICH IS EXECUTED AS A
046988,000065:                                                                                                  #        RESULT OF A DOWNRUPT.  CONTROL IS RETURNED VIA TCF RESUME WHICH
046989,000066:                                                                                                  #        IN EFFECT IS A RESUME.
046990,000067: 
046991,000068:                                                                                                  #  SUBROUTINES CALLED -- NONE
046992,000069: 
046993,000070:                                                                                                  #  NORMAL EXIT MODE -- TCF RESUME
046994,000071: 
046995,000072:                                                                                                  #  ALARM OR ABORT EXIT MODE -- NONE
046996,000073: 
046997,000074:                                                                                                  #  RESTART PROTECTION:
046998,000075:                                                                                                  #        ON A FRESH START AND RESTART THE `STARTSUB' SUBROUTINE WILL INITIALIZE THE DOWNLIST POINTER (ACTUALLY
046999,000076:                                                                                                  #        DNTMGOTO) TO THE BEGINNING OF THE CURRENT DOWNLIST (I.E., CURRENT CONTENTS OF DNLSTADR).  THIS HAS THE
047000,000077:                                                                                                  #        EFFECT OF IGNORING THE REMAINDER OF THE DOWNLIST WHICH THE DOWN-TELEMETRY PROGRAM WAS WORKING ON WHEN
047001,000078:                                                                                                  #        THE RESTART (OR FRESH START) OCCURRED AND RESUME DOWN TELEMETRY FROM THE BEGINNING OF THE CURRENT
047002,000079:                                                                                                  #        DOWNLIST.
047003,000080: 
047004,000081:                                                                                                  #        ALSO OF INTEREST IS THE FACT THAT ON A RESTART THE AGC WILL ZERO DOWNLINK CHANNELS 13, 34 AND 35.
047005,000082: 
047006,000083:                                                                                                  #  DOWNLINK LIST SELECTION:
047007,000084:                                                                                                  #        THE APPROPRIATE DOWNLINK LISTS ARE SELECTED BY THE FOLLOWING:
047008,000085:                                                                                                  #        1.      FRESH START
047009,000086:                                                                                                  #        2.      V37EXXE WHERE XX = THE MAJOR MODE BEING SELECTED.
047010,000087:                                                                                                  #        3.      UPDATE PROGRAM (P27)
047011,000088:                                                                                                  #        4.      NON-V37 SELECTABLE TYPE PROGRAMS (E.G., AGS INITIALIZATION (SUNDANCE, LUMINARY) AND P61-P62
047012,000089:                                                                                                  #                TRANSITIONS (COLOSSUS) ETC.).
047013,000090: 
047014,000091:                                                                                                  #  DOWNLINK LIST RULES AND LIMITATIONS:
047015,000092:                                                                                                  #        READ SECTION(S) WHICH FOLLOW `DEBRIS' WRITEUP.
047016,000093: 
047017,000094:                                                                                                  #  OUTPUT -- EVERY 2 SECONDS 100 DOUBLE PRECISION WORDS (I.E., 200 LGC
047018,000095:                                                                                                  #        COMPUTER WORDS) ARE TRANSMITTED VIA DOWNLINK.
047019,000096:

047032,000109:                                                                                                  #  NOTES REGARDING DOWNLINK LISTS ASSOCIATED WITH THIS PROGRAM:
047033,000110:                                                                                                  #  1.    DOWNLISTS.  DOWNLISTS MUST BE COMPILED IN THE SAME BANK AS THE
047034,000111:                                                                                                  #        DOWN-TELEMETRY PROGRAM.  THIS IS DONE FOR EASE OF CODING, FASTER
047035,000112:                                                                                                  #        EXECUTION.
047036,000113:                                                                                                  #  2.    EACH DOWNLINK LIST CONSISTES OF A CONTROL LIST AND A NUMBER OF
047037,000114:                                                                                                  #        SUBLISTS.
047038,000115:                                                                                                  #  3.    A SUBLIST REFERS TO A SNAPSHOT OR DATA COMMON TO THE SAME OR OTHER
047039,000116:                                                                                                  #        DOWNLINK LISTS.  ANY SUBLIST CONTAINING COMMON DATA NEEDS TO BE
047040,000117:                                                                                                  #        CODED ONLY ONCE FOR THE APPLICABLE DOWNLINK LISTS.
047041,000118:                                                                                                  #  4.    SNAPSHOT SUBLISTS REFER SPECIFICALLY TO HOMOGENEOUS DATA WHICH MUST BE
047042,000119:                                                                                                  #        SAVED IN A BUFFER DURING ONE DOWNRUPT.
047043,000120:                                                                                                  #  5.    THE 1DNADR FOR THE 1ST WORD OF SNAPSHOT DATA IS FOUND AT THE END
047044,000121:                                                                                                  #        OF EACH SNAPSHOT SUBLIST, SINCE THE PROGRAM CODING SENDS THIS DP WORD
047045,000122:                                                                                                  #        IMMEDIATELY AFTER STORING THE OTHERS IN THE SNAPSHOT BUFFER.
047046,000123:                                                                                                  #  6.    ALL LISTS ARE COMBINATIONS OF CODED ERASABLE ADDRESS CONSTANTS
047047,000124:                                                                                                  #        CREATED FOR THE DOWNLIST PROGRAM.
047048,000125:                                                                                                  #        A.      1DNADR                  1-WORD DOWNLIST ADDRESS.
047049,000126:                                                                                                  #                SAME AS ECADR, BUT USED WHEN THE WORD ADDRESSED IS THE LEFT
047050,000127:                                                                                                  #                HALF OF A DOUBLE-PRECISION WORD FOR DOWN TELEMETRY.
047051,000128:                                                                                                  #        B.      2DNADR - 6DNADR         N-WORD DOWNLIST ADDRESS, N = 2 - 6.
047052,000129:                                                                                                  #                SAME AS 1DNADR, BUT WTIH THE 4 UNUSED BITS OF THE ECADR FORMAT
047053,000130:                                                                                                  #                FILLED IN WITH 0001-0101.  USED TO POINT TO A LIST OF N DOUBLE-
047054,000131:                                                                                                  #                PRECISION WORDS, STORED CONSECUTIVELY, FOR DOWN TELEMETRY.
047055,000132:                                                                                                  #        C.      DNCHAN                  DOWNLIST CHANNEL ADDRESS.
047056,000133:                                                                                                  #                SAME AS 1DNADR, BUT WITH PREFIX BITS 0111.  USED TO POINT TO
047057,000134:                                                                                                  #                A PAIR OF CHANNELS FOR DOWN TELEMETRY.
047058,000135:                                                                                                  #        D.      DNPTR                   DOWN-TELEMETRY SUBLIST POINTER.
047059,000136:                                                                                                  #                SAME AS CAF BUT TAGGES AS A CONSTANT.  USED IN CONTROL LIST TO POINT TO A SUBLIST.
047060,000137:                                                                                                  #                CAUTION --- A DNPTR CANNOT BE USED IN A SUBLIST.
047061,000138:                                                                                                  #  7.    THE WORD ORDER CODE IS SET TO ZERO AT THE BEGINNING OF EACH DOWNLIST (I.E., CONTROL LIST) AND WHEN
047062,000139:                                                                                                  #        A `1DNADR TIME2' IS DETECTED IN THE CONTROL LIST (ONLY).
047063,000140:                                                                                                  #  8.    IN THE SNAPSHOT SUBLIST ONLY, THE DNADR'S CANNOT POINT TO THE FIRST WORD OF ANY EBANK.
047064,000141: 
047065,000142:                                                                                                  #  DOWNLIST LIST RESTRICTIONS:
047066,000143:                                                                                                  #  (THE FOLLOWING POINTS MAY BE LISTED ELSEWHERE BUT ARE LISTED HERE SO IT IS CLEAR THAT THESE THINGS CANNOT BE
047067,000144:                                                                                                  #  DONE)
047068,000145:                                                                                                  #  1.    SNAPSHOT DOWNLIST:
047069,000146:                                                                                                  #        (A) CANNOT CONTAIN THE FOLLOWING ECADRS (I.E., 1DNADR'S): Q, 400, 1000, 1400, 2000, 2400, 3000, 3400.
047070,000147:                                                                                                  #        (B) CAN CONTAIN ONLY 1DNADR'S
047071,000148:                                                                                                  #  2.    ALL DOWNLINKED DATA (EXCEPT CHANNELS) IS PICKED UP BY A DCA SO DOWNLINK LISTS CANNOT CONTAIN THE
047072,000149:                                                                                                  #        EQUIVALENT OF THE FOLLOWING ECADRS (I.E., IDNADRS): 377, 777, 1377, 1777, 2377, 2777, 3377, 3777.
047073,000150:                                                                                                  #        (NOTE: TE TERM `EQUIVALENT' MEANT THAT THE IDNADR TO 6DNADR WILL BE PROCESSED LIKE 1 TO 6 ECADRS)
047074,000151:                                                                                                  #  3.    CONTROL LISTS AND SUBLISTS CANNOT HAVE ENTRIES = OCTAL 00000 OR OCTAL 77777
047075,000152:                                                                                                  #  Page 1078
047076,000153:                                                                                                  #  4.    THE `1DNADR TIME2' WHICH WILL CAUSE THE DOWNLINT PROGRAM TO SET THE WORDER CODE TO 3 MUST APPEAR IN THE
047077,000154:                                                                                                  #        CONTROL SECTION OF THE DOWNLIST.
047078,000155:                                                                                                  #  5.    `DNCHAN 0' CANNOT BE USED.
047079,000156:                                                                                                  #  6.    `DNPTR 0' CANNOT BE USED.
047080,000157:                                                                                                  #  7.    DNPTR CANNOT APPEAR IN A SUBLIST.
047081,000158: 
047082,000159:                                                                                                  #  EBANK SETTINGS
047083,000160:                                                                                                  #        IN THE PROCESS OF SETTING THE EBANK (WHEN PICKING UP DOWNLINK DATA) THE DOWN TELEMETRY PROGRAM PUTS
047084,000161:                                                                                                  #        `GARBAGE' INTO BITS15-12 OF EBANK.  HUGH BLAIR-SMITH WARNS US THAT BITS15-12 OF EBANK MAY BECOME
047085,000162:                                                                                                  #        SIGNIFICANT SOMEDAY IN THE FUTURE.  IF/WHEN THAT HAPPENS, THE PROGRAM SHOULD INSURE (BY MASKING ETC.)
047086,000163:                                                                                                  #        THAT BITS 15-12 OF EBANK ARE ZERO.
047087,000164:

047090,000167:                                                                                                  #        2. NEGONE INTO SUBLIST
047091,000168:                                                                                                  #        3. NEGONE INTO DNECADR
