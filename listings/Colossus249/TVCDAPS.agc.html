Source Code

These source-code files were derived originally from the scan from MIT's Dibner Institute's now-discontinued website titled "History of Recent Science and Technology", which in turn was originally created by Gary Neff, though his high-quality scans were unfortunately reduced in legibility when presented online. The markings on the front of the printout imply that it was the original AGC developer Norm Brodeur's copy. A scan by Ron Burkey has superceded it, made from a more-legible copy from the collection of original AGC developer Fred Martin. The source code was transcribed from those images by Ron Burkey. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	ASSEMBLE REVISION 249 OF AGC PROGRAM COLOSSUS BY NASA 2021111-041
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

040826,000043:                                                                                                  #  FUNCTIONAL DESCRIPTION
040827,000044: 
040828,000045:                                                                                                  #        SELF-PERPETUATING T5 TASKS WHICH GENERATE THE COMMAND SIGNALS
040829,000046:                                                                                                  #        FOR THE PITCH AND YAW SPS GIMBAL ACTUATORS DURING TVC (SPS) BURNS,
040830,000047:                                                                                                  #        IN RESPONSE TO BODY-AXIS RATE COMMANDS FROM CROSS-PRODUCT STEERING
040831,000048:                                                                                                  #        (S40.8).  IF NO STEERING (IMPULSIVE BURNS) MAINTAINS ATTITUDE-HOLD
040832,000049:                                                                                                  #        ABOUT THE REFERENCE (INITIAL) DIRECTIONS (ZERO RATE COMMANDS).
040833,000050: 
040834,000051:                                                                                                  #        THE PITCH AND YAW LOOPS ARE SEPARATE, BUT STRUCTURED IDENTICALLY.
040835,000052:                                                                                                  #        EACH ATTITUDE-RATE LOOP INCLUDES GIMBAL ANGLE RATE DERIVATION,
040836,000053:                                                                                                  #        GIMBAL/BODY AXIS TRANSFORMATION, BODY-AXIS ATTITUDE ERROR
040837,000054:                                                                                                  #        INTEGRATION WITH ERROR LIMITING, THE CSM/LEM FILTER OR THE BRANCH
040838,000055:                                                                                                  #        POINTS FOR THE CSM-ALONE (GEN3DAP) FILTER, OUTPUT LIMITER,
040839,000056:                                                                                                  #        CG-OFFSET TRACKER FILTER, AND THE CG-TRACKER MINOR LOOP.
040840,000057: 
040841,000058:                                                                                                  #        THE DAPS ARE CYCLIC, CALLING EACH OTHER AT 1/2 THE DAP SAMPLE
040842,000059:                                                                                                  #        TIME, AS DETERMINED BY T5TVCDT.  THE ACTUATOR COMMANDS ARE
040843,000060:                                                                                                  #        REGENERATED AS ANALOG VOLTAGES BY THE OPTICS ERROR COUNTERS, WHICH
040844,000061:                                                                                                  #        TRANSMIT THE SIGNAL TO THE ACTUATOR SERVOS WHEN THERE IS PROPER CDU
040845,000062:                                                                                                  #        MODING.
040846,000063: 
040847,000064:                                                                                                  #        REFERENCES FOR THE CSM/LEM FILTER DESIGN INCLUDE R503 BY STUBBS
040848,000065:                                                                                                  #        (MIT IL OCT 1965) AND SGA MEMO R26-65 BY MARTIN (MIT IL OCT 1965).
040849,000066:                                                                                                  #        REFERENCES FOR THE CSM FILTER DESIGN (SEE GEN3DAP) INCLUDE R533 BY
040850,000067:                                                                                                  #        LU (MIT IL JUNE 1966).
040851,000068: 
040852,000069:                                                                                                  #        OPERATIONAL ASPECTS OF THE INTEGRATED CONTROL PACKAGE, WITH DESIGN-
040853,000070:                                                                                                  #        NOMINAL PARAMETER VALUES ARE DISCUSSED IN AG R336-67 BY ENGEL
040854,000071:                                                                                                  #        (MIT IL OCT 1967) AND SGA MEMO R18-67 BY SCHLUNDT (MIT IL OCT 1967)
