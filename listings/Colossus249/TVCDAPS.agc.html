Source Code

This is the source-code for the Apollo 9 Command Module's Guidance Computer. These files were derived originally from the scan from MIT's Dibner Institute's now-discontinued website titled "History of Recent Science and Technology", which in turn was originally created by Gary Neff, though his high-quality scans were unfortunately reduced in legibility when presented online. The markings on the front of the printout imply that it was the original AGC developer Norm Brodeur's copy. A scan by Ron Burkey has superceded it, made from a more-legible copy from the collection of original AGC developer Fred Martin. The source code was transcribed from those images by Ron Burkey. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	ASSEMBLE REVISION 249 OF AGC PROGRAM COLOSSUS BY NASA 2021111-041
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

041237,000051:                                                                                                  #        SELF-PERPETUATING T5 TASKS WHICH GENERATE THE COMMAND SIGNALS
041238,000052:                                                                                                  #        FOR THE PITCH AND YAW SPS GIMBAL ACTUATORS DURING TVC (SPS) BURNS,
041239,000053:                                                                                                  #        IN RESPONSE TO BODY-AXIS RATE COMMANDS FROM CROSS-PRODUCT STEERING
041240,000054:                                                                                                  #        (S40.8).  IF NO STEERING (IMPULSIVE BURNS) MAINTAINS ATTITUDE-HOLD
041241,000055:                                                                                                  #        ABOUT THE REFERENCE (INITIAL) DIRECTIONS (ZERO RATE COMMANDS).
041242,000056: 
041243,000057:                                                                                                  #        THE PITCH AND YAW LOOPS ARE SEPARATE, BUT STRUCTURED IDENTICALLY.
041244,000058:                                                                                                  #        EACH ATTITUDE-RATE LOOP INCLUDES GIMBAL ANGLE RATE DERIVATION,
041245,000059:                                                                                                  #        GIMBAL/BODY AXIS TRANSFORMATION, BODY-AXIS ATTITUDE ERROR
041246,000060:                                                                                                  #        INTEGRATION WITH ERROR LIMITING, THE CSM/LEM FILTER OR THE BRANCH
041247,000061:                                                                                                  #        POINTS FOR THE CSM-ALONE (GEN3DAP) FILTER, OUTPUT LIMITER,
041248,000062:                                                                                                  #        CG-OFFSET TRACKER FILTER, AND THE CG-TRACKER MINOR LOOP.
041249,000063: 
041250,000064:                                                                                                  #        THE DAPS ARE CYCLIC, CALLING EACH OTHER AT 1/2 THE DAP SAMPLE
041251,000065:                                                                                                  #        TIME, AS DETERMINED BY T5TVCDT.  THE ACTUATOR COMMANDS ARE
041252,000066:                                                                                                  #        REGENERATED AS ANALOG VOLTAGES BY THE OPTICS ERROR COUNTERS, WHICH
041253,000067:                                                                                                  #        TRANSMIT THE SIGNAL TO THE ACTUATOR SERVOS WHEN THERE IS PROPER CDU
041254,000068:                                                                                                  #        MODING.
041255,000069:

041257,000071:                                                                                                  #        (MIT IL OCT 1965) AND SGA MEMO R26-65 BY MARTIN (MIT IL OCT 1965).
041258,000072:                                                                                                  #        REFERENCES FOR THE CSM FILTER DESIGN (SEE GEN3DAP) INCLUDE R533 BY
041259,000073:                                                                                                  #        LU (MIT IL JUNE 1966).

041263,000077:                                                                                                  #        (MIT IL OCT 1967) AND SGA MEMO R18-67 BY SCHLUNDT (MIT IL OCT 1967)
