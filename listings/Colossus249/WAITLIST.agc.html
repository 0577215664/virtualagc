Source Code

These source-code files were derived originally from the scan from MIT's Dibner Institute's now-discontinued website titled "History of Recent Science and Technology", which in turn was originally created by Gary Neff, though his high-quality scans were unfortunately reduced in legibility when presented online. The markings on the front of the printout imply that it was the original AGC developer Norm Brodeur's copy. A scan by Ron Burkey has superceded it, made from a more-legible copy from the collection of original AGC developer Fred Martin. The source code was transcribed from those images by Ron Burkey. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	ASSEMBLE REVISION 249 OF AGC PROGRAM COLOSSUS BY NASA 2021111-041
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

051621,000047:                                                                                                  #  FUNCTIONAL DESCRIPTION --
051622,000048:                                                                                                  #        PART OF A SECTION OF PROGRAMS -- WAITLIST, TASKOVER, T3RUPT, USED TO CALL A PROGRAM (CALLED A TASK),
051623,000049:                                                                                                  #        WHICH IS TO BEGIN IN C(A) CENTISECONDS.  WAITLIST UPDATES TIME3, LST1, AND LST2.  THE MEANING OF THESE LISTS
051624,000050:                                                                                                  #        FOLLOW.
051625,000051: 
051626,000052:                                                                                                  #                ((TIME3) = 16384 -(T1-T) CENTISECONDS, (T=PRESENT TIME, T1-TIME FOR TASK1)
051627,000053: 
051628,000054:                                                                                                  #                        C(LST1)         =       -(T2-T1)+1
051629,000055:                                                                                                  #                        C(LST1 +1)      =       -(T3-T2)+1
051630,000056:                                                                                                  #                        C(LST1 +2)      =       -(T4-T3)+1
051631,000057:                                                                                                  #                                       ...
051632,000058:                                                                                                  #                        C(LST1 +6)      =       -(T8-T7)+1
051633,000059:                                                                                                  #                        C(LST1 +7)      =       -(T9-T8)+1
051634,000060: 
051635,000061:                                                                                                  #                        C(LST2)         =       2CADR OF TASK1
051636,000062:                                                                                                  #                        C(LST2 +2)      =       2CADR OF TASK2
051637,000063:                                                                                                  #                                       ...
051638,000064:                                                                                                  #                        C(LST2 +14)     =       2CADR OF TASK8
051639,000065:                                                                                                  #                        C(LST2 +16)     =       2CADR OF TASK9
051640,000066: 
051641,000067:                                                                                                  #  WARNINGS --
051642,000068:                                                                                                  #        1)      1 <= C(A) <= 16250D (1 CENTISCOND TO 162.5 SEC)
051643,000069:                                                                                                  #        2)      9 TASKS MAXIMUM
051644,000070:                                                                                                  #        3)      TASKS CALLED UNDER INTERRUPT INHIBITED
051645,000071:                                                                                                  #        4)      TASKS END BY TC TASKOVER
051646,000072: 
051647,000073:                                                                                                  #  CALLING SEQUENCE --
051648,000074:                                                                                                  #        L-1     CA      DELTAT  (TIME IN CENTISECONDS TO TASK START)
051649,000075:                                                                                                  #        L       TC      WAITLIST
051650,000076:                                                                                                  #        L+1     2CADR   DESIRED TASK.
051651,000077:                                                                                                  #        L+2     (MINOR OF 2CADR)
051652,000078:                                                                                                  #        L+3     RELINT          (RETURNS HERE)
051653,000079: 
051654,000080:                                                                                                  #  TWIDDLE --
051655,000081:                                                                                                  #        TWIDDLE IS FOR USE WHEN THE TASK BEING SET UP IS IN THE SAME EBANK AND FBANK AS THE USER.  IN
051656,000082:                                                                                                  #        SUCH CASES, IT IMPROVES UPON WAITLIST BY ELIMINATING THE NEED FOR THE BBCON HALF OF THE 2CADR,

051658,000084:                                                                                                  #        SAVING A WORD.  TWIDDLE IS LIKE WAITLIST IN EVERY RESPECT EXCEPT CALLING SEQUENCE, TO WIT,
051659,000085:                                                                                                  #                L-1     CA      DELTAT
051660,000086:                                                                                                  #                L       TC      TWIDDLE
051661,000087:                                                                                                  #                L+1     ADRES   DESIRED TASK
051662,000088:                                                                                                  #                L+2     RELINT          (RETURNS HERE)
051663,000089: 
051664,000090:                                                                                                  #  NORMAL EXIT MODES --
051665,000091:                                                                                                  #        AT L+3 OF CALLING SEQUENCE.
051666,000092: 
051667,000093:                                                                                                  #  ALARM OR ABORT EXIT MODES --
051668,000094:                                                                                                  #        TC      ABORT
051669,000095:                                                                                                  #        OCT     1203    (WAITLIST OVERFLOW -- TOO MANY TASKS)
051670,000096: 
051671,000097:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED --
051672,000098:                                                                                                  #        ACCOMPLISHED BY FRESH START --  LST2, ..., LST2 +16 = ENDTASK
051673,000099:                                                                                                  #                                        LST1, ..., LST1 +7  = NEG1/2
051674,000100: 
051675,000101:                                                                                                  #  OUTPUT --
051676,000102:                                                                                                  #        LST1 AND LST2 UPDATED WTIH NEW TASK AND ASSOCIATED TIME.
051677,000103: 
051678,000104:                                                                                                  #  DEBRIS --
051679,000105:                                                                                                  #        CENTRALS -- A,Q,L
051680,000106:                                                                                                  #        OTHER    -- WAITEXIT, WAITADR, WAITTEMP, WAITBANK
051681,000107: 
051682,000108:                                                                                                  #  DETAILED ANALYSIS OF TIMING --
051683,000109:                                                                                                  #        CONTROL WILL NOT BE RETURNED TO THE SPECIFIED ADDRESS (2CADR) IN EXACTLY DELTA T CENTISECONDS.
051684,000110:                                                                                                  #        THE APPROXIMATE TIME MAY BE CALCULATED AS FOLLOWS:
051685,000111:                                                                                                  #                LET T0 = THE TIME OF THE TC WAITLIST
051686,000112:                                                                                                  #                LET TS = T0 +147U + COUNTER INCREMENTS (SET UP TIME)
051687,000113:                                                                                                  #                LET X  = TS -(100TS)/100  (VARIANCE FROM COUNTERS)
051688,000114:                                                                                                  #                LET Y  = LENGTH OF TIME OF INHIBIT INTERRUPT AFTER T3RUPT
051689,000115:                                                                                                  #                LET Z  = LENGTH OF TIME TO PROCESS TASKS WHICH ARE DUE THIS T3RUPT BUT DISPATCHED EARLIER.
051690,000116:                                                                                                  #                         (Z=0, USUALLY).
051691,000117:                                                                                                  #                LET DELTD = THE ACTUAL TIME TAKEN TO GIVE CONTROL TO 2CADR
051692,000118:                                                                                                  #                        THEN DELTD = TS+DELTA T -X +Y +Z +1.05MS* +COUNTERS*
051693,000119:                                                                                                  #                *THE TIME TAKEN BY WAITLIST ITSELF AND THE COUNTER TICKING DURING THIS WAITLIST TIME.
051694,000120:                                                                                                  #        IN SHORT, THE ACTUAL TIME TO RETURN CONTROL TO A 2CADR IS AUGMENTED BY THE TIME TO SET UP THE TASK'S
051695,000121:                                                                                                  #        INTERRUPT, ALL COUNTERS TICKING, THE T3RUPT PROCESSING TIME, THE WAITLIST PROCESSING TIME AND THE POSSIBILITY
051696,000122:                                                                                                  #        OF OTHER TASKS INHIBITING THE INTERRUPT.
