Source Code

This is the source-code for the Apollo 9 Command Module's Guidance Computer. These files were derived originally from the scan from MIT's Dibner Institute's now-discontinued website titled "History of Recent Science and Technology", which in turn was originally created by Gary Neff, though his high-quality scans were unfortunately reduced in legibility when presented online. The markings on the front of the printout imply that it was the original AGC developer Norm Brodeur's copy. A scan by Ron Burkey has superceded it, made from a more-legible copy from the collection of original AGC developer Fred Martin. The source code was transcribed from those images by Ron Burkey. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	ASSEMBLE REVISION 249 OF AGC PROGRAM COLOSSUS BY NASA 2021111-041
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

052152,000057:                                                                                                  #        PART OF A SECTION OF PROGRAMS,- WAITLIST, TASKOVER, T3RUPT, USED TO CALL A PROGRAM, (CALLED A TASK),

052154,000059:                                                                                                  #        FOLLOW.
052155,000060: 
052156,000061:                                                                                                  #                C(TIME3) = 16384 -(T1-T) CENTISECONDS, (T=PRESENT TIME, T1-TIME FOR TASK1)
052157,000062: 
052158,000063:                                                                                                  #                        C(LST1)         =       -(T2-T1)+1
052159,000064:                                                                                                  #                        C(LST1 +1)      =       -(T3-T2)+1
052160,000065:                                                                                                  #                        C(LST1 +2)      =       -(T4-T3)+1
052161,000066:                                                                                                  #                                        .
052162,000067:                                                                                                  #                                        .
052163,000068:                                                                                                  #                        C(LST1 +6)      =       -(T8-T7)+1
052164,000069:                                                                                                  #                        C(LST1 +7)      =       -(T9-T8)+1
052165,000070: 
052166,000071:                                                                                                  #                        C(LST2)         =       2CADR OF TASK1
052167,000072:                                                                                                  #                        C(LST2 +2)      =       2CADR OF TASK2
052168,000073:                                                                                                  #                                        .
052169,000074:                                                                                                  #                                        .
052170,000075:                                                                                                  #                        C(LST2 +14)     =       2CADR OF TASK8
052171,000076:                                                                                                  #                        C(LST2 +16)     =       2CADR OF TASK9
052172,000077: 
052173,000078:                                                                                                  #  WARNINGS-
052174,000079:                                                                                                  #  --------
052175,000080:                                                                                                  #        1)      1 <= C(A) <= 16250D (1 CENTISECOND TO 162.5 SEC)
052176,000081:                                                                                                  #        2)      9 TASKS MAXIMUM
052177,000082:                                                                                                  #        3)      TASKS CALLED UNDER INTERRUPT INHIBITED
052178,000083:                                                                                                  #        4)      TASKS END BY TC TASKOVER

052181,000086:                                                                                                  #        L-1     CA      DELTAT  (TIME IN CENTISECONDS TO TASK START)
052182,000087:                                                                                                  #        L       TC      WAITLIST
052183,000088:                                                                                                  #        L+1     2CADR   DESIRED TASK
052184,000089:                                                                                                  #        L+2     (MINOR OF 2CADR)
052185,000090:                                                                                                  #        L+3     RELINT          (RETURNS HERE)

052188,000093:                                                                                                  #  -------
052189,000094:                                                                                                  #        TWIDDLE IS FOR USE WHEN THE TASK BEING SET UP IS IN THE SAME EBANK AND FBANK AS THE USER.  IN
052190,000095:                                                                                                  #        SUCH CASES, IT IMPROVES UPON WAITLIST BY ELIMINATING THE NEED FOR THE BBCON HALF OF THE 2CADR,

052192,000097:                                                                                                  #        SAVING A WORD.  TWIDDLE IS LIKE WAITLIST IN EVERY RESPECT EXCEPT CALLING SEQUENCE, TO WIT-

052194,000099:                                                                                                  #                L       TC      TWIDDLE

052213,000118:                                                                                                  #        CENTRALS - A,Q,L
052214,000119:                                                                                                  #        OTHER    - WAITEXIT, WAITADR, WAITTEMP, WAITBANK
052215,000120: 
052216,000121:                                                                                                  #  DETAILED ANALYSIS OF TIMING -
052217,000122:                                                                                                  #        CONTROL WILL NOT BE RETURNED TO THE SPECIFIED ADDRESS (2CADR) IN EXACTLY DELTA T CENTISECONDS.
052218,000123:                                                                                                  #        THE APPROXIMATE TIME MAY BE CALCULATED AS FOLLOWS
052219,000124:                                                                                                  #                LET TO = THE TIME OF THE TC WAITLIST
052220,000125:                                                                                                  #                LET TS = TO +147U + COUNTER INCREMENTS (SET UP TIME)
052221,000126:                                                                                                  #                LET X  = TS -(100TS)/100  (VARIANCE FROM COUNTERS)
052222,000127:                                                                                                  #                LET Y  = LENGTH OF TIME OF INHIBIT INTERRUPT AFTER T3RUPT
052223,000128:                                                                                                  #                LET Z  = LENGTH OF TIME TO PROCESS TASKS WHICH ARE DUE THIS T3RUPT BUT DISPATCHED EARLIER.
052224,000129:                                                                                                  #                         (Z=0, USUALLY)
052225,000130:                                                                                                  #                LET DELTD = THE ACTUAL TIME TAKEN TO GIVE CONTROL TO 2CADR
052226,000131:                                                                                                  #                        THEN DELTD = TS+DELTA T -X +Y +Z +1.05MS* +COUNTERS*
052227,000132:                                                                                                  #                *-THE TIME TAKEN BY WAITLIST ITSELF AND THE COUNTER TICKING DURING THIS WAITLIST TIME.
052228,000133:                                                                                                  #        IN SHORT, THE ACTUAL TIME TO RETURN CONTROL TO A 2CADR IS AUGMENTED BY THE TIME TO SET UP THE TASK:S
052229,000134:                                                                                                  #        INTERRUPT, ALL COUNTERS TICKING, THE T3RUPT PROCESSING TIME, THE WAITLIST PROCESSING TIME AND THE POSSIBILITY
052230,000135:                                                                                                  #        OF OTHER TASKS INHIBITING THE INTERRUPT.
