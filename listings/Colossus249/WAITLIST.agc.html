Source Code

This is the source-code for the Apollo 9 Command Module's Guidance Computer. These files were derived originally from the scan from MIT's Dibner Institute's now-discontinued website titled "History of Recent Science and Technology", which in turn was originally created by Gary Neff, though his high-quality scans were unfortunately reduced in legibility when presented online. The markings on the front of the printout imply that it was the original AGC developer Norm Brodeur's copy. A scan by Ron Burkey has superceded it, made from a more-legible copy from the collection of original AGC developer Fred Martin. The source code was transcribed from those images by Ron Burkey. Note that the page images presented online are of reduced quality, and that higher-quality images are available. Report any conversion errors or legibility problems in page images to info@sandroid.org. Notations on the program listing read, in part:

	ASSEMBLE REVISION 249 OF AGC PROGRAM COLOSSUS BY NASA 2021111-041
	20'35 OCT. 28,1968
Note that the date is the date of the printout, not the date of the program revision.

052118,000053:                                                                                                  #  FUNCTIONAL DESCRIPTION -
052119,000054:                                                                                                  #        PART OF A SECTION OF PROGRAMS,- WAITLIST, TASKOVER, T3RUPT, USED TO CALL A PROGRAM, (CALLED A TASK),
052120,000055:                                                                                                  #        WHICH IS TO BEGIN IN C(A) CENTISECONDS.  WAITLIST UPDATES TIME3, LST1 AND LST2.  THE MEANING OF THESE LISTS
052121,000056:                                                                                                  #        FOLLOW.
052122,000057: 
052123,000058:                                                                                                  #                C(TIME3) = 16384 -(T1-T) CENTISECONDS, (T=PRESENT TIME, T1-TIME FOR TASK1)
052124,000059: 
052125,000060:                                                                                                  #                        C(LST1)         =       -(T2-T1)+1
052126,000061:                                                                                                  #                        C(LST1 +1)      =       -(T3-T2)+1
052127,000062:                                                                                                  #                        C(LST1 +2)      =       -(T4-T3)+1
052128,000063:                                                                                                  #                                        .
052129,000064:                                                                                                  #                                        .
052130,000065:                                                                                                  #                        C(LST1 +6)      =       -(T8-T7)+1
052131,000066:                                                                                                  #                        C(LST1 +7)      =       -(T9-T8)+1
052132,000067: 
052133,000068:                                                                                                  #                        C(LST2)         =       2CADR OF TASK1
052134,000069:                                                                                                  #                        C(LST2 +2)      =       2CADR OF TASK2
052135,000070:                                                                                                  #                                        .
052136,000071:                                                                                                  #                                        .
052137,000072:                                                                                                  #                        C(LST2 +14)     =       2CADR OF TASK8
052138,000073:                                                                                                  #                        C(LST2 +16)     =       2CADR OF TASK9
052139,000074: 
052140,000075:                                                                                                  #  WARNINGS-
052141,000076:                                                                                                  #  --------
052142,000077:                                                                                                  #        1)      1 <= C(A) <= 16250D (1 CENTISECOND TO 162.5 SEC)
052143,000078:                                                                                                  #        2)      9 TASKS MAXIMUM
052144,000079:                                                                                                  #        3)      TASKS CALLED UNDER INTERRUPT INHIBITED
052145,000080:                                                                                                  #        4)      TASKS END BY TC TASKOVER
052146,000081: 
052147,000082:                                                                                                  #  CALLING SEQUENCE -
052148,000083:                                                                                                  #        L-1     CA      DELTAT  (TIME IN CENTISECONDS TO TASK START)
052149,000084:                                                                                                  #        L       TC      WAITLIST
052150,000085:                                                                                                  #        L+1     2CADR   DESIRED TASK
052151,000086:                                                                                                  #        L+2     (MINOR OF 2CADR)
052152,000087:                                                                                                  #        L+3     RELINT          (RETURNS HERE)
052153,000088: 
052154,000089:                                                                                                  #  TWIDDLE-
052155,000090:                                                                                                  #  -------
052156,000091:                                                                                                  #        TWIDDLE IS FOR USE WHEN THE TASK BEING SET UP IS IN THE SAME EBANK AND FBANK AS THE USER.  IN
052157,000092:                                                                                                  #        SUCH CASES, IT IMPROVES UPON WAITLIST BY ELIMINATING THE NEED FOR THE BBCON HALF OF THE 2CADR,

052159,000094:                                                                                                  #        SAVING A WORD.  TWIDDLE IS LIKE WAITLIST IN EVERY RESPECT EXCEPT CALLING SEQUENCE, TO WIT-
052160,000095:                                                                                                  #                L-1     CA      DELTAT
052161,000096:                                                                                                  #                L       TC      TWIDDLE
052162,000097:                                                                                                  #                L+1     ADRES   DESIRED TASK
052163,000098:                                                                                                  #                L+2     RELINT          (RETURNS HERE)
052164,000099:

052166,000101:                                                                                                  #        AT L+3 OF CALLING SEQUENCE
052167,000102: 
052168,000103:                                                                                                  #  ALARM OR ABORT EXIT MODES -
052169,000104:                                                                                                  #        TC      ABORT

052171,000106:

052173,000108:                                                                                                  #        ACCOMPLISHED BY FRESH START,--  LST2, ..., LST2 +16 = ENDTASK
052174,000109:                                                                                                  #                                        LST1, ..., LST1 +7  = NEG1/2

052177,000112:                                                                                                  #        LST1 AND LST2 UPDATED WITH NEW TASK AND ASSOCIATED TIME.
052178,000113: 
052179,000114:                                                                                                  #  DEBRIS -
052180,000115:                                                                                                  #        CENTRALS - A,Q,L
052181,000116:                                                                                                  #        OTHER    - WAITEXIT, WAITADR, WAITTEMP, WAITBANK
052182,000117: 
052183,000118:                                                                                                  #  DETAILED ANALYSIS OF TIMING -
052184,000119:                                                                                                  #        CONTROL WILL NOT BE RETURNED TO THE SPECIFIED ADDRESS (2CADR) IN EXACTLY DELTA T CENTISECONDS.
052185,000120:                                                                                                  #        THE APPROXIMATE TIME MAY BE CALCULATED AS FOLLOWS
052186,000121:                                                                                                  #                LET T0 = THE TIME OF THE TC WAITLIST
052187,000122:                                                                                                  #                LET TS = T0 +147U + COUNTER INCREMENTS (SET UP TIME)
052188,000123:                                                                                                  #                LET X  = TS -(100TS)/100  (VARIANCE FROM COUNTERS)
052189,000124:                                                                                                  #                LET Y  = LENGTH OF TIME OF INHIBIT INTERRUPT AFTER T3RUPT
052190,000125:                                                                                                  #                LET Z  = LENGTH OF TIME TO PROCESS TASKS WHICH ARE DUE THIS T3RUPT BUT DISPATCHED EARLIER.
052191,000126:                                                                                                  #                         (Z=0, USUALLY)
052192,000127:                                                                                                  #                LET DELTD = THE ACTUAL TIME TAKEN TO GIVE CONTROL TO 2CADR
052193,000128:                                                                                                  #                        THEN DELTD = TS+DELTA T -X +Y +Z +1.05MS* +COUNTERS*
052194,000129:                                                                                                  #                *-THE TIME TAKEN BY WAITLIST ITSELF AND THE COUNTER TICKING DURING THIS WAITLIST TIME.
052195,000130:                                                                                                  #        IN SHORT, THE ACTUAL TIME TO RETURN CONTROL TO A 2CADR IS AUGMENTED BY THE TIME TO SET UP THE TASK:S
052196,000131:                                                                                                  #        INTERRUPT, ALL COUNTERS TICKING, THE T3RUPT PROCESSING TIME, THE WAITLIST PROCESSING TIME AND THE POSSIBILITY
052197,000132:                                                                                                  #        OF OTHER TASKS INHIBITING THE INTERRUPT.
