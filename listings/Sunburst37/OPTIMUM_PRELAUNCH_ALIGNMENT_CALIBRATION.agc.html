018540,000030: 
018541,000031:                                                                                                  #  THIS LOG SECTION CONSISTS OF PRELAUNCH ALIGNMENT AND GYRO DRIFT TESTS
018542,000032:                                                                                                  #  INTEGRATED TOGETHER TO SAVE WORDS. IT IS COMPLETELY RESTART PROOFED.
018543,000033:                                                                                                  #  THE PRELAUNCH ALIGNMENT TECHNIQUE IS BASICALLY THE SAME AS IN BLOCK 1
018544,000034:                                                                                                  #  EXCEPT THAT IT HAS BEEN SIMPLIFIED IN THE SENSE THAT SMALL ANGLE APPROX.
018545,000035:                                                                                                  #  HAVE BEEN USED. THE DRIFT TESTS USE A UNIQUE IMPLEMENTATION OF THE
018546,000036:                                                                                                  #  OPTIMUM STATISTICAL FILTER. FOR A DESCRIPTION SEE E-1973.BOTH OF THESE
018547,000037:                                                                                                  #  ROUTINES USE STANDARD SYSTEM TEST LEADIN PROCEDURES. THE INITIALIZATION
018548,000038:                                                                                                  #  PROCEDURE FOR THE DRIFT TESTS IS IN THE JDC S. THE INITIALIZATION METHOD
018549,000039: 
018550,000040:                                                                                                  #  FOR GYROCOMPASS IS IN A SOON TO COME SYSTEM TEST GROUP MEMO.
018551,000041:                                                                                                  #  THIS LOG SECTION ALWAYS STARTS BY A CADR IN IMU PERF. TESTS 2 AND BEGINS
018552,000042:                                                                                                  #  AT ESTIMS. THE PIPS ARE READ EVERY .5 SEC IN COMPASS AND 1 SEC IN DRIFT
018553,000043:                                                                                                  #  TESTS. THEN IN BOTH CASES RELEVANT COMPUTATION IS DONE. THE KEY ERASABLE
018554,000044:                                                                                                  #  IS GEOCOMPS- 0 WE ARE IN A DRIFT TEST -NONZERO WE ARE IN COMPASS.
018555,000045:                                                                                                  #  THE GYROCOMPASS HAS THE CAPABILITY TO ALIGN TO ANY ORIENTATION,HAS THE
018556,000046:                                                                                                  #  CAPABILITY TO CHANGE ORIENTATIONS WHILE RUNNING,IS COMPENSATED FOR
018557,000047:                                                                                                  #  COMPONENT ERRORS,IS CAPABLE OF OPTICAL VERIFICATION( CSM ONLY).
018558,000048: 
018559,000049:                                                                                                  #  SUBROUTINES CALLED
018560,000050: 
018561,000051:                                                                                                  #  EARTHR, CALGTA,  OGC ZERO,ERTHRVSE,GCOMPZERO AND
018562,000052:                                                                                                  #  IMU COMPENSATION
018563,000053:                                                                                                  #  DURING OPTICAL VERIFICATION (CSM ONLY) ESSENTIALLY ALL OF INFLIGHT ALIGN
018564,000054:                                                                                                  #  IS CALLED IN ONE WAY OR ANOTHER. SEE THE LISTING.
018565,000055: 
018566,000056:                                                                                                  #  NORMAL EXIT
018567,000057: 
018568,000058:                                                                                                  #  DRIFT TESTS- LENGTHOT GOES TO ZERO-RETURN TO IMU PERF TEST2 CONTROL
018569,000059:                                                                                                  #  GYROCOMPASS-RECEIVE GUIDANCE REFERENCE RELEASE SIGNAL-TC MP2JOB
018570,000060:                                                                                                  #                                                       206 IMPLEMENTATION
