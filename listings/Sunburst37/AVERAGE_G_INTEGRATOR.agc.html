Source Code

These source-code files were transcribed from a printout in Don Eyles's personal collection, scanned by archive.org, and financially sponsored by Peter McDermott. A team of volunteers performed the transcription and proof-reading. The scanned page images are available at the Virtual AGC Project website, as well as higher-quality (but much larger) images at the Virtual AGC Project's collection in the Internet Archive. Report any problems by creating "issues" at the Virtual AGC Project's GitHub Repository.

Notations on the program listing read, in part:

	YUL SYSTEM FOR AGC: NEW PROGRAM SHEPATIN BY EYLES
	DEC 13, 1966

	THIS PROGRAM WAS ASSEMBLED AS A VERSION OF REVISION 37 
        OF PROGRAM SUNBURST BY LEM GROUP
Note that the date is the date of the printout, not the date of the program revision.

035550,000123:                                                                                                  #  * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * *
035551,000124: 
035552,000125:                                                                                                  #      THE MASS UPDATE ROUTINE EXTENDS FROM APPROXIMATELY HERE TO THE NEXT
035553,000126:                                                                                                  #  ROW OF ASTERISKS.   COMPUTATIONS ARE AS FOLLOWS:-
035554,000127:                                                                                                  #      (1) /AF/ = MAGNITUDE DV/DT, SCALED IN UNITS OF 2(-5) M/CS/CS.
035555,000128:                                                                                                  #      (2) PREFORCE = /AF/ MASS, APPROXIMATE PRESENT THRUST, SCALED AT
035556,000129:                                                                                                  #          SOME 3.3 POUNDS PER BIT; PREFORCE LATER BECOMES PREFORCE/2FMAX.
035557,000130:                                                                                                  #      (3) NOMINAL EXHAUST VELOCITY, VEXNOM = VEXSTEP + (PREFORCE-FSTEP)
035558,000131:                                                                                                  #          SLOPE, WHERE VEXSTEP, FSTEP, AND SLOPE ARE SELECTED FROM THE
035559,000132:                                                                                                  #          'VEXTABLE' BY THE ROUTINE BEGINNING AT VEXFINDD; THE FSTEPS
035560,000133:                                                                                                  #          AND VXSTEPS DEFINE CERTAIN DISCRETE POINTS ON THE CURVE OF
035561,000134:                                                                                                  #          EXHAUST VELOCITY VERSUS THRUST; THE ASSOCIATED SLOPE IS ALWAYS
035562,000135: 
035563,000136:                                                                                                  #          THAT FOR THE SEGMENT JUST PRECEEDING THOSE POINTS.   (UNITS
035564,000137:                                                                                                  #          FOR VEXNOM, AS FOR NEGVEX, ARE 2(6) M/CS.)
035565,000138:                                                                                                  #      (4) AREARATE, A MODERATELY ABSTRUSE (NOT TO SAY ABSURD) FUNCTION
035566,000139:                                                                                                  #          OF DELAREA (EROSION) AND PREFORCE.   (THE AREA IN QUESTION IS
035567,000140:                                                                                                  #          THAT OF THE THROAT OF THE ENGINE.)
035568,000141:                                                                                                  #      (5) DELAREA = DELAREA + AREARATE DELTAT.
035569,000142:                                                                                                  #      (6) NEGVEX = -(VEXNOM - CVEX5 DELAREA).
035570,000143:                                                                                                  #      (7) MASS = MASS + (/DELV//NEGVEX) MASS, SCALED IN UNITS OF 2(15)
035571,000144:                                                                                                  #          KILOGRAMS.
035572,000145: 
035573,000146:                                                                                                  #      THE FOLLOWING INITIALIZATIONS MUST BE MADE BEFORE LAUNCH:
035574,000147:                                                                                                  #  C(MASS) = 0, C(LEMMASS1) = SCALED MASS OF LEM CUM DPS, C(LEMMASS2) =
035575,000148:                                                                                                  #  SCALED MASS OF LEM WITHOUT DPS.   UPON LEM-SIVB SEPARATION MASS MUST
035576,000149:                                                                                                  #  BE REINITIALIZED USING LEMMASS1 AND DELAREA MUST BE SET TO DP ZERO.
035577,000150:                                                                                                  #  LATER, WHEN THE DPS PARTS COMPANY, MASS MUST BE INITIALIZED STILL
035578,000151:                                                                                                  #  AGAIN USING LEMMASS2.
035579,000152:                                                                                                  #      THE FOLLOWING IS AN INELEGANT SUPERFLUITY:-   INPUTS: MASS, DELAREA,
035580,000153:                                                                                                  #  DELV, DELTAT, EMANATIONS FROM THE VEXTABLE, A TABLESPOON OF SALT AND
035581,000154:                                                                                                  #  SEVEN BYZANTINE DODO EGGS.   OUTPUTS: /AF/ AND MASS.   DEBRIS: PREFORCE,
035582,000155:                                                                                                  #  NEGVEX, AND AREARATE (TO LEARN MORE ABOUT THESE GEMS, SEE ABOVE).
035583,000156:                                                                                                  #  SUBROUTINES CALLED: MASSMULT (A PART OF THROTTLE CONTROL) AND THE
035584,000157:                                                                                                  #  INTERPRETER.   ERASABLES: BANK 5 AT PRESENT.   EYLES WROTE THIS STUFF;
035585,000158:                                                                                                  #  HIS LAST MODIFICATION THEREOF (NUMBER N+2) WENT INTO AGC REVISION 20.
