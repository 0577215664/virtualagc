Source Code

These source-code files were transcribed from a printout in Don Eyles's personal collection, scanned by archive.org, and financially sponsored by Peter McDermott. A team of volunteers performed the transcription and proof-reading. The scanned page images are available at the Virtual AGC Project website, as well as higher-quality (but much larger) images at the Virtual AGC Project's collection in the Internet Archive. Report any problems by creating "issues" at the Virtual AGC Project's GitHub Repository.

Notations on the program listing read, in part:

	YUL SYSTEM FOR AGC: NEW PROGRAM SHEPATIN BY EYLES
	DEC 13, 1966

	THIS PROGRAM WAS ASSEMBLED AS A VERSION OF REVISION 37 
        OF PROGRAM SUNBURST BY LEM GROUP
Note that the date is the date of the printout, not the date of the program revision.

028219,000033: 
028220,000034:                                                                                                  #     THE FUNCTION OF MISSION PHASE 2 IS TO CONTROL THE SEQUENCE OF EVENTS IN THE 206 FLIGHT FROM GUIDANCE
028221,000035:                                                                                                  #  REFERENCE RELEASE THROUGH LIFTOFF TO THE SIVB BOOSTER SHUTDOWN.
028222,000036: 
028223,000037:                                                                                                  #     AT GUIDANCE REFERENCE RELEASE, THE GRR FLAG IS SET,  PREREAD IS CALLED TO BEGIN COMPUTATION OF POSITION AND
028224,000038:                                                                                                  #  VELOCITY, AND CALLS ARE MADE FOR DFI T/M CALIBRATION AND LIFTOFF
028225,000039: 
028226,000040:                                                                                                  #     WHEN PRELAUNCH DETECTS THAT THE GRR FLAG IS SET, IT TERMINATES GYROCOMPASSING AND CALLS MP2JOB. MP2JOB
028227,000041:                                                                                                  #  DISPLAYS 7 IN THE MAJOR MODE AND GOES TO MATRXJOB TO COMPUTE REFSMMAT.
028228,000042: 
028229,000043:                                                                                                  #     AT LIFTOFF, THE LGC CLOCK IS ZEROED, CALLS ARE MADE FOR THE COLD FIRE PURGE AND POST LET JETTISON PROGRAMS.
028230,000044:                                                                                                  #  THE MAJOR MODE IS CHANGED TO 11.
028231,000045: 
028232,000046:                                                                                                  #     AT POST LET JETTISON, THE DV MONITOR IS ENABLED TO DETECT BOOSTER SHUTDOWN, THE ABORT COMMAND MONITOR AND THE
028233,000047:                                                                                                  #  TUMBLE MONITOR ARE ENABLED, AND THE MAJOR MODE IS CHANGED TO 12.
028234,000048: 
028235,000049:                                                                                                  #     THE VARIOUS LMP COMMANDS REQUIRED FOR MP2 ARE SCHEDULED BY WAITLIST CALLS.
028236,000050:

028389,000203:                                                                                                  #  PROGRAM DESCRIPTION- MATRXJOB                                           DATE- 18 NOV 1966
028390,000204:                                                                                                  #  MOD NO- 1                                                               LOG SECTION- MP 2 GRR + BOOST MONITOR
028391,000205:                                                                                                  #  MOD BY- LICKLY, KERNAN                                                  ASSEMBLY- SUNBURST REVISION 8

028393,000207:                                                                                                  #  FUNCTIONAL DESCRIPTION
028394,000208: 
028395,000209:                                                                                                  #           THIS PROGRAM CONSTRUCTS THE MATRIX WHICH RELATES THE STABLE MEMBER INERTIAL FRAME TO THE REFERENCE
028396,000210:                                                                                                  #  FRAME (Z NORTH, X ALONG THE VERNAL EQUINOX.)
028397,000211: 
028398,000212:                                                                                                  #           TWO INTERMEDIATE COORDINATE SYSTEMS ARE USED: A LOCAL, EARTH FIXED, VERTICAL, SOUTH, EAST SYSTEM AND AN
028399,000213:                                                                                                  #  EARTH REFERENCE X, Y, Z SYSTEM.  IN THIS LATTER SYSTEM, THE Z AXIS IS THE EARTH'S ROTATION AXIS, THE X AXIS IS
028400,000214:                                                                                                  #  NORMAL TO Z IN THE PLANE OF Z AND THE LOCAL VERTICAL, POSITIVE IN THE DIRECTION OF V.  Y IS Z CROSS X.
028401,000215: 
028402,000216:                                                                                                  #           THE FIRST COMPUTATION IS OF AZGR, THE ANGLE BETWEEN THE REFERENCE INERTIAL AND EARTH REFERENCE X-Z
028403,000217:                                                                                                  #  PLANES (THE Z AXES ARE COINCIDENT).  AZGR IS COMPUTED BY CONVERTING THE TIME FROM THE BEGINNING OF THE EPHEMERIS
028404,000218:                                                                                                  #  YEAR TO RELEASE (TEPHEM + TPRELTER) TO REVOLUTIONS (DAYS).  THE WHOLE REVS ARE DISCARDED AND THE INITIAL ANGLE
028405,000219:                                                                                                  #  (AZ0) BETWEEN THE GREENWICH MERIDIAN AND THE REFERENCE X-Z PLANES IS ADDED.  ADDING THE LONGITUDE YIELDS AZGR.
028406,000220: 
028407,000221:                                                                                                  #           THE FOLLOWING COMPUTATIONS ARE THEN PERFORMED.
028408,000222: 
028409,000223:                                                                                                  #  LOCAL VERTICAL(ER) = COS(LATITUDE), 0, SIN(LATITUDE)  IN EARTH REFERENCE
028410,000224: 
028411,000225:                                                                                                  #  LOCAL VERTICAL(IR) = COS(LAT)COS(AZGR), COS(LAT)SIN(AZGR), SIN(LAT)  IN INERTIAL REFERENCE
028412,000226: 
028413,000227:                                                                                                  #  LOCAL EAST(IR) = NXV = -COS(LAT)SIN(AZGR), COS(LAT)COS(AZGR), 0  IN INERTIAL REFERENCE
028414,000228: 
028415,000229:                                                                                                  #  LOCAL SOUTH(IR) = E(IR) X V(IR)
028416,000230: 
028417,000231:                                                                                                  #           THE RELATIONSHIP OF THE STABLE MEMBER AXES TO THE V, S, E AXES IS GIVEN BY ZSMAZ, THE ANGLE FROM NORTH
028418,000232:                                                                                                  #  TO ZSM, AND TILT, THE ANGLE ABOUT ZSM FROM VERTICAL TO XSM.
028419,000233: 
028420,000234:                                                                                                  #  ZSM(IR) = EAST(IR)COS(ZSMAZ - 90) + SOUTH(IR)SIN(ZSMAZ - 90)
028421,000235: 
028422,000236:                                                                                                  #  YSM(IR) = (ZSM(IR) X V(IR))COS(TILT) - V(IR)SIN(TILT)
028423,000237: 
028424,000238:                                                                                                  #  XSM(IR) = YSM(IR) X ZSM(IR)
028425,000239: 
028426,000240:                                                                                                  #           THESE THREE HALF-UNIT VECTORS, XSM(IR), YSM(IR), AND ZSM(IR) ARE THE SM AXES EXPRESSED IN INERTIAL
028427,000241:                                                                                                  #  REFERENCE COORDINATES AND THEY FORM REFSMMAT, THE REFERENCE TO STABLE MEMBER MATRIX.
028428,000242: 
028429,000243:                                                                                                  #  THE INPUT (PRELAUNCH ERASABLE LOAD) REQUIREMENTS ARE:
028430,000244: 
028431,000245:                                                                                                  #  1) TEPHEM       THE TRIPLE PRECISION TIME IN CENTISECONDS FROM MIDNIGHT JULY 1, OF THE EPHEMERIS YEAR TO
028432,000246:                                                                                                  #  MIDNIGHT OF THE LAUNCH DAY (SIDEREAL CONVERTED TO MEAN SOLAR.)
028433,000247: 
028434,000248:                                                                                                  #     IT IS ASSUMED THAT DURING THE LAUNCH COUNTDOWN THE LGC CLOCK (TIME2, TIME1) WILL BE ALIGNED TO REFLECT A
028435,000249:                                                                                                  #  ZERO VALUE AT MIDNIGHT OF THE LAUNCH DAY.  IF NOT, THE DIFFERENCE MUST BE ADDED TO TEPHEM.
028436,000250: 
028437,000251:                                                                                                  #  2) TILT         THE ROTATION OF XSM ABOUT ZSM (RIGHT HAND RULE) FROM VERTICAL IN REVOLUTIONS.
028438,000252:

Page 637 

028440,000254:                                                                                                  #  3) ZSMAZ        THE ANGLE FROM NORTH TO ZSM IN REVOLUTIONS.

028443,000257: 
028444,000258:                                                                                                  #  1) REFSMMAT     THE HALF-UNIT MATRIX WHICH TRANSFORMS FROM REFERENCE INERTIAL TO SM INERTIAL.
