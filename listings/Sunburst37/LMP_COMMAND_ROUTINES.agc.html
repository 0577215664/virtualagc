Source Code

These source-code files were transcribed from a printout in Don Eyles's personal collection, scanned by archive.org, and financially sponsored by Peter McDermott. A team of volunteers performed the transcription and proof-reading. The scanned page images are available at the Virtual AGC Project website, as well as higher-quality (but much larger) images at the Virtual AGC Project's collection in the Internet Archive. Report any problems by creating "issues" at the Virtual AGC Project's GitHub Repository.

Notations on the program listing read, in part:

	YUL SYSTEM FOR AGC: NEW PROGRAM SHEPATIN BY EYLES
	DEC 13, 1966

	THIS PROGRAM WAS ASSEMBLED AS A VERSION OF REVISION 37 
        OF PROGRAM SUNBURST BY LEM GROUP
Note that the date is the date of the printout, not the date of the program revision.

033509,000023:                                                                                                  #           THE FOLLOWING SUBROUTINES ALLOW MISSION PROGRAMS TO REQUEST LMP (LEM MISSION PROGRAMMER) OUTPUTS.
033510,000024: 
033511,000025:                                                                                                  #  THE PROPER DECIMAL CODE IS ENTERED INTO A TABLE AND FROM THENCE TO CHANNEL 10 VIA T4RUPT AND ARE INCLUDED IN
033512,000026:                                                                                                  #  THE DOWNLINK.
033513,000027: 
033514,000028:                                                                                                  #           FOUR ROUTINES ARE PROVIDED:
033515,000029: 
033516,000030:                                                                                                  #                                                   TC     1LMP            CALLED UNDER EXEC OR RUPT. DELIVERS CODE
033517,000031:                                                                                                  #                                                   DEC    LMPCODE         AND RETURNS IMMEDIATELY.
033518,000032: 
033519,000033:                                                                                                  #                                                   TC     2LMP            SAME AS 1LMP BUT TWO CODES ARE
033520,000034:                                                                                                  #                                                   DEC    LMPCODE1        SENT.
033521,000035:                                                                                                  #                                                   DEC    LMPCODE2
033522,000036: 
033523,000037:                                                                                                  #                                                   TC     1LMP+DT         ASSUMES CALLED AS PART OF WAITLIST TASK.
033524,000038:                                                                                                  #                                                   DEC    LMPCODE         DELIVERS LMP CODE, DOES A VARDELAY FOR
033525,000039:                                                                                                  #                                                   DEC    DT              DT, AND THEN RETURNS UNDER WL CONTROL.
033526,000040: 
033527,000041:                                                                                                  #                                                   TC     2LMP+DT         SAME AS 1LMP+DT BUT TWO CODES SENT.
033528,000042:                                                                                                  #                                                   DEC    LMPCODE1
033529,000043:                                                                                                  #                                                   DEC    LMPCODE2
033530,000044:                                                                                                  #                                                   DEC    DT
