Source Code

These source-code files were transcribed from a printout in Don Eyles's personal collection, scanned by archive.org, and financially sponsored by Peter McDermott. A team of volunteers performed the transcription and proof-reading. The scanned page images are available at the Virtual AGC Project website, as well as higher-quality (but much larger) images at the Virtual AGC Project's collection in the Internet Archive. Report any problems by creating "issues" at the Virtual AGC Project's GitHub Repository.

Notations on the program listing read, in part:

	YUL SYSTEM FOR AGC: NEW PROGRAM SHEPATIN BY EYLES
	DEC 13, 1966

	THIS PROGRAM WAS ASSEMBLED AS A VERSION OF REVISION 37 
        OF PROGRAM SUNBURST BY LEM GROUP
Note that the date is the date of the printout, not the date of the program revision.

036642,000033:                                                                                                  #  FUNCTIONAL DESCRIPTION -
036643,000034: 
036644,000035:                                                                                                  #       THERE ARE TWO MODES OF OPERATION OF THE SECOND DPS GUIDANCE PROGRAM (2DPS).  PRIOR TO THE IGNITION
036645,000036:                                                                                                  #  SEQUENCE 2DPS IS OPERATED IN THE PRE-IGNITION MODE FOR THE PURPOSE OF PROVIDING, FOR THE IGNITION
036646,000037:                                                                                                  #  SEQUENCE, THE TIME OF INITIATION AND THE DIRECTION OF THRUST.  IMMEDIATELY AFTER THE IGNITION SEQUENCE
036647,000038:                                                                                                  #  IS COMPLETED, INCLUDING THE COMMAND OF MAXIMUM THRUST, 2DPS IS ENGAGED IN THE THRUSTING MODE FOR THE
036648,000039: 
036649,000040:                                                                                                  #  PURPOSE OF PROVIDING THRUST ACCELERATION MAGNITUDE AND DIRECTION COMMANDS NECESSARY TO CONDUCT THE
036650,000041:                                                                                                  #  SIMULATED POWERED LANDING MANEUVER.
036651,000042: 
036652,000043:                                                                                                  #       PRIOR TO LAUNCH, ERASABLE MEMORY MUST BE LOADED WITH THE SEMI-UNIT NORMAL TO THE PLANE OF THE
036653,000044:                                                                                                  #  REQUIRED ORBIT CPT6/2, AND DATA WHICH INDIRECTLY SPECIFICIES THE NOMINAL IGNITION TIME TIGNOM.  THESE
036654,000045:                                                                                                  #  DATA MAY BE ADJUSTED, AS A GROUP, VIA THE UPLINK, UNTIL 200 SECONDS PRIOR TO THE TIGNOM IMPLIED BY THE
036655,000046:                                                                                                  #  PRE-LAUNCH LOAD OR IMPLIED BY THE UPLINK DATA, WHICHEVER IS EARLIER.  A DESCRIPTION OF THE METHOD OF
036656,000047:                                                                                                  #  UPDATING TIGNOM IS IN THE PROVINCE OF THE MISSION CONTROL PROGRAM (MCP) MISSION PHASE 11 -
036657,000048:                                                                                                  #  DPS2/FITH/APS1.  THE TARGETING IS CONTROLLED BY THE SEMI-UNIT VECTOR CPT6/2 AS DESCRIBED IN THE 206 GSOP.
036658,000049: 
036659,000050: 
036660,000051:                                                                                                  #       AT APPROXIMATELY 180 SECONDS PRIOR TO TIGNOM, THE MCP ENGAGES 2DPS IN THE PRE-IGNITION MODE BY
036661,000052:                                                                                                  #  PROVIDING AN EXTRAPOLATED TIME AND STATE IN THE REGISTERS TET, RIGNTION, VIGNTION, AND TRANSFERRING
036662,000053:                                                                                                  #  CONTROL TO LOCATION PREBURN IN 2DPS.  THIS TIME AND STATE ARE AT APPROXIMATELY 50 SECONDS PRIOR TO
036663,000054:                                                                                                  #  TIGNOM.  (THE NAMES RIGNTION, VIGNTION, CHOSEN TO INDICATE USAGE WITH OTHER GUIDANCE PROGRAMS, ARE
036664,000055:                                                                                                  #  MISLEADING WITH 2DPS IN THAT THE EXTRAPOLATED STATE PROVIDED IN THESE REGISTERS IS NOT THAT EXPECTED AT
036665,000056:                                                                                                  #  IGNITION BUT RATHER AT 50 SECONDS PRIOR TO IGNITION.)  2DPS TRANSFERS THE CONTENTS OF TET TO PIPTIME
036666,000057:                                                                                                  #  AND CALLS SUBROUTINE VPATCHER WHICH TRANSFERS THE STATE TO RN, VN AND INITIALIZES FOR CALCRVG.
036667,000058:                                                                                                  #  USING CALCRVG AS A SUBROUTINE, AND INTERNAL SUBROUTINES IGNITN1 AND IGNITN2, 2DPS MOVES THE TIME AND
036668,000059:                                                                                                  #  STATE FORWARD TO A TIME GREATER THAN OR EQUAL TO TULLG-6 SECONDS AND LESS THAN OR EQUAL TO TULLG-3.8 SECONDS,
036669,000060:                                                                                                  #  LEAVING THIS CONSISTENT SET IN PIPTIME, RN, VN, WHERE TULLG IS THE PRECISE TIME FOR ULLAGE TO BE INITIATED
036670,000061:                                                                                                  #  AND IS LEFT IN THE REGISTER OF THAT NAME.  2DPS LEAVES A SEMI-UNIT VECTOR IN THE DIRECTION IN WHICH
036671,000062:                                                                                                  #  THRUST IS TO BE COMMANDED DURING THE IGNITION SEQUENCE IN THE REGISTER POINTVSM AND BRANCHES TO
036672,000063: 
036673,000064:                                                                                                  #  LOCATION RETPREB IN THE MISSION CONTROL PROGRAM.
036674,000065: 
036675,000066:                                                                                                  #       USING THE DATA LEFT BY THE PRE-IGNITION MODE OF 2DPS, THE MCP ENGAGES KALCMANU TO PROPERLY
036676,000067:                                                                                                  #  ORIENT THE SPACECRAFT FOR THE IGNITION SEQUENCE, AND ISSUES THE WAITLIST CALLS WHICH CAUSE THE
036677,000068:                                                                                                  #  READACCS, SERVICER, PGNCSMON LOOP TO BE ESTABLISHED (WITH 2DPS NOT IN THE LOOP) AND PRODUCE ULLAGE,
036678,000069:                                                                                                  #  ENGINE IGNITION, AND FINALLY MAXIMUM THRUST.  IMMEDIATELY AFTER MAXIMUM THRUST IS COMMANDED, THE MCP
036679,000070:                                                                                                  #  PLACES THE 2BCADR OF BURN (EBANK= E2DPS) IN THE AVGEXIT WHICH CAUSES 2DPS TO BE ADDED TO THE READACCS,
036680,000071:                                                                                                  #  SERVICER, PGNCSMON LOOP IMMEDIATELY FOLLOWING CALCRVG.
036681,000072: 
036682,000073:                                                                                                  #       WHEN CONTROL IS GIVEN TO 2DPS WITH THE 2BCADR OF BURN IN AVGEXIT, 2DPS OPERATES IN THE THRUSTING
036683,000074: 
036684,000075:                                                                                                  #  MODE.  IT GENERATES THE SEQUENCING REQUIRED TO CONDUCT THE SIMULATED POWERED LANDING MANEUVER AND ISSUES
036685,000076:                                                                                                  #  THE THRUST ACCELERATION MAGNITUDE AND DIRECTION COMMANDS.
036686,000077: 
036687,000078:                                                                                                  #       2DPS GENERATES THE SEQUENCING BY INCREMENTING THE HIGH ORDER PART OF AVGEXIT, BY AN APPROPRIATE
036688,000079:                                                                                                  #  INTEGER, TO ACCOMPLISH EACH CHANGE OF 2DPS PHASE.  THE INCREMENTING OF AVGEXIT CAUSES, ON THE SUBSEQUENT
036689,000080:

Page 815 

036691,000082:                                                                                                  #  PASS THROUGH 2DPS, CONTROL TO BEGIN AT A NEW POINT IN THE SECOND DPS FLITE SEQUENCE TABLE, AND
036692,000083:                                                                                                  #  CONSEQUENTLY THE CORRESPONDING 2DPS MODING TO BE ESTABLISHED.
036693,000084: 
036694,000085:                                                                                                  #       2DPS GENERATES THE THRUST ACCELERATION MAGNITUDE AND DIRECTION COMMANDS ON THE BASIS OF THE
036695,000086:                                                                                                  #  CONTENTS OF PIPTIME, RN, VN, AT THE BEGINNING OF THE PASS.  TO ISSUE THE THRUST ACCELERATION DIRECTION
036696,000087:                                                                                                  #  COMMAND, 2DPS PLACES A SEMI-UNIT VECTOR IN THE REQUIRED DIRECTION IN THE REGISTER AXISD AND CALLS
036697,000088: 
036698,000089:                                                                                                  #  FINDCDUD AS A SUBROUTINE.  TO ISSUE THE MAGNITUDE COMMAND, 2DPS PLACES THE REQUIRED MAGNITUDE IN THE
036699,000090:                                                                                                  #  REGISTER /ACF/ AND CALLS THROTCON AS A SUBROUTINE.
036700,000091: 
036701,000092:                                                                                                  #       BEFORE BUT NOT INCLUDING THE LAST PASS, 2DPS TERMINATES THE JOB PERIODICALLY STARTED BY READACCS
036702,000093:                                                                                                  #  BY BRANCHING TO ENDOFJOB.
036703,000094: 
036704,000095:                                                                                                  #       ON THE LAST PASS 2DPS PLACES THE SEMI-UNIT VECTOR ALONG THE NORMAL TO THE DESIRED ORBITAL PLANE,
036705,000096:                                                                                                  #  CPT6/2, IN THE REGISTER AXISD, CALLS FINDCDUD, AND BRANCHES TO LOCATION RETBURN IN THE MCP.  THE MCP
036706,000097:                                                                                                  #  REMOVES 2DPS FROM THE READACCS, SERVICER, PGNCSMON LOOP AND ISSUES THE WAITLIST CALLS WHICH PRODUCE THE
036707,000098:                                                                                                  #  RANDOM THROTTLING.  THE MCP DOES NOT CHANGE THE CONTENTS OF AXISD SUPPLIED BY 2DPS.  CONSEQUENTLY THE
036708,000099:                                                                                                  #  THRUST ACCELERATION DURING RANDOM THROTTLING IS NORMAL TO THE ORBITAL PLANE.
