Source Code

These source-code files were transcribed from a printout in Don Eyles's personal collection, scanned by archive.org, and financially sponsored by Peter McDermott. A team of volunteers performed the transcription and proof-reading. The scanned page images are available at the Virtual AGC Project website, as well as higher-quality (but much larger) images at the Virtual AGC Project's collection in the Internet Archive. Report any problems by creating "issues" at the Virtual AGC Project's GitHub Repository.

Notations on the program listing read, in part:

	YUL SYSTEM FOR AGC: NEW PROGRAM SHEPATIN BY EYLES
	DEC 13, 1966

	THIS PROGRAM WAS ASSEMBLED AS A VERSION OF REVISION 37 
        OF PROGRAM SUNBURST BY LEM GROUP
Note that the date is the date of the printout, not the date of the program revision.

040263,000126:                                                                                                  #           THE FOLLOWING SUBROUTINES PROVIDE TO THE BASIC PROGRAMMER ENTRY INTO AND RETURN FROM ANY INTERPRETIVE
040264,000127:                                                                                                  #  CODING WHICH DOES NOT USE THE ENTERING CONTENTS OF Q AND WHICH RETURNS VIA DANZIG. C(A) AND C(L) ARE SAVED.
040265,000128: 
040266,000129:                                                                                                  #  USER'S RESPONSIBILITY TO FILL IN ADVANCE THE APPROPRIATE OPERAND AND ADDRESS REGISTERS USED BY THE
040267,000130:                                                                                                  #  INTERPRETIVE CODING SUCH AS MPAC, BUF, ADDRWD, ETC.; AND TO CONFIRM THAT THE INTERPRETIVE CODING MEETS THE
040268,000131:                                                                                                  #  ABOVE RESTRICTIONS WITH RESPECT TO Q AND DANZIG.
040269,000132: 
040270,000133:                                                                                                  #  USEPRET AND USPRCADR MUST NOT BE USED IN INTERRUPT.
040271,000134: 
040272,000135:                                                                                                  #  1. USEPRET ACCESSES INTERPRETIVE CODING WHICH CAN BE ENTERED WITHOUT CHANGING FBANK.
040273,000136:                                                                                                  #     THE CALLING SEQUENCE IS AS FOLLOWS:
040274,000137: 
040275,000138:                                                                                                  #                                          L         TC     USEPRET
040276,000139:                                                                                                  #                                          L+1       TC,TCF INTPRETX       TC,TCF MEANS TC OR TCF
040277,000140:                                                                                                  #                                                                          INTPRETX IS THE INTERPRETIVE CODING
040278,000141:                                                                                                  #                                                                          RETURN IS TO L+2
040279,000142: 
040280,000143:                                                                                                  #  2. USPRCADR ACCESSES INTERPRETIVE CODING IN OTHER THAN THE USER'S FBANK. THE CALLING SEQUENCE IS AS FOLLOWS:
040281,000144: 
040282,000145:                                                                                                  #                                          L         TC     USPRCADR
040283,000146:                                                                                                  #                                          L+1       CADR   INTPRETX       INTPRETX IS THE INTERPRETIVE CODING
040284,000147:                                                                                                  #                                                                          RETURN IS TO L+2
040285,000148: 
040286,000149:    5326           56002        USEPRET            XCH      Q                                     #  FETCH Q, SAVING A
040287,000150:    5327           54154                           TS       LOC                                   #  L+1 TO LOC
040288,000151:    5330           30006                           CA       BBANK                                 
040289,000152:    5331           54155                           TS       BANKSET                               #  USER'S BBANK TO BANKSET
040290,000153:    5332           37746                           CA       BIT8                                  
040291,000154:    5333           54023                           TS       EDOP                                  #  EXIT INSTRUCTION TO EDOP
040292,000155:    5334           30002                           CA       Q                                     #  RETRIEVE ORIGINAL A
040293,000156:    5335           00154                           TC       LOC                                   
040294,000157: 
040295,000158:    5336           54154        USPRCADR           TS       LOC                                   #  SAVE A
040296,000159:    5337           37746                           CA       BIT8                                  
040297,000160:    5340           54023                           TS       EDOP                                  #  EXIT INSTRUCTION TO EDOP
040298,000161:    5341           30006                           CA       BBANK                                 
040299,000162:    5342           54155                           TS       BANKSET                               #  USER'S BBANK TO BANKSET
040300,000163:    5343           50002                           INDEX    Q                                     
040301,000164:    5344           30000                           CA       0
