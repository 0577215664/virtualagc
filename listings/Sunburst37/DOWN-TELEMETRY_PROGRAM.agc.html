Source Code

These source-code files were transcribed from a printout in Don Eyles's personal collection, scanned by archive.org, and financially sponsored by Peter McDermott. A team of volunteers performed the transcription and proof-reading. The scanned page images are available at the Virtual AGC Project website, as well as higher-quality (but much larger) images at the Virtual AGC Project's collection in the Internet Archive. Report any problems by creating "issues" at the Virtual AGC Project's GitHub Repository.

Notations on the program listing read, in part:

	YUL SYSTEM FOR AGC: NEW PROGRAM SHEPATIN BY EYLES
	DEC 13, 1966

	THIS PROGRAM WAS ASSEMBLED AS A VERSION OF REVISION 37 
        OF PROGRAM SUNBURST BY LEM GROUP
Note that the date is the date of the printout, not the date of the program revision.

013220,000028:                                                                                                  #  ASSEMBLY- REV 12 OF SUNBURST
013221,000029:                                                                                                  #  FUNCTIONAL DESCRIPTION- THIS ROUTINE IS INITIATED BY TELEMETRY END
013222,000030:                                                                                                  #           PULSE FROM THE DOWNLINK TELEMETRY CONVERTER. THIS PULSE OCCURS
013223,000031:                                                                                                  #           AT 50 TIMES PER SEC(EVERY 20 MS) THEREFORE DODOWNTM IS
013224,000032:                                                                                                  #           EXECUTED AT THESE RATES. THIS ROUTINE SELECTS THE APPROPRIATE
013225,000033:                                                                                                  #           LGC DATA TO BE TRANSMITTED DOWNLINK AND LOADS IT INTO OUTPUT
013226,000034:                                                                                                  #           CHANNELS 34 AND 35. THE INFORMATION IS THEN GATED OUT FROM THE
013227,000035:                                                                                                  #           LGC IN SERIAL FASHION.
013228,000036:                                                                                                  #           THIS PROGRAM IS CODED FOR A 2 SECOND DOWNLIST. SINCE DOWNRUPTS
013229,000037:                                                                                                  #           OCCUR EVERY 20 MS AND 2 LGC COMPUTER WORDS CAN BE PLACED IN
013230,000038:                                                                                                  #           CHANNELS 34 AND 35 DURING EACH DOWNRUPT THE PROGRAM IS CAPABLE
013231,000039:                                                                                                  #           OF SENDING 200 LGC WORDS EVERY 2 SECONDS.
013232,000040:                                                                                                  #  CALLING SEQUENCE- NONE
013233,000041:                                                                                                  #           PROGRAM IS ENTERED VIA TCF DODOWNTM WHICH IS EXECUTED AS A
013234,000042:                                                                                                  #           RESULT OF A DOWNRUPT. CONTROL IS RETURNED VIA TCF NOQRSM WHICH
013235,000043: 
013236,000044:                                                                                                  #           IN EFFECT IS A RESUME.
013237,000045:                                                                                                  #  SUBROUTINES CALLED- NONE
013238,000046:                                                                                                  #  NORMAL EXIT MODE- TCF NOQRSM
013239,000047:                                                                                                  #  ALARM OR ABORT EXIT MODE- NONE
013240,000048:                                                                                                  #  RESTART PROTECTION- NONE.
013241,000049:                                                                                                  #           EXCEPT THAT WHEN A RESTART DOES OCCUR :STARTSUB: WILL
013242,000050:                                                                                                  #           INITIALIZE THE DOWNLIST POINTER TO THE BEGINNING OF THE NOMINAL
013243,000051:                                                                                                  #           DOWNLIST. THIS HAS THE EFFECT OF IGNORING THE REMAINDER OF THE
013244,000052:                                                                                                  #           DOWNLIST WHICH THE DOWN-TELEMETRY PROGRAM WAS WORKING ON WHEN
013245,000053:                                                                                                  #           THE RESTART OCCURRED.
013246,000054:                                                                                                  #  OUTPUT-  EVERY 2 SECONDS 100 DOUBLE PRECISION WORDS(I.E. 200 LGC
013247,000055:                                                                                                  #           COMPUTER WORDS) ARE TRANSMITTED VIA DOWNLINK.
013248,000056:                                                                                                  #  ERASABLE INITIALIZATION REQUIRED- NONE
013249,000057:                                                                                                  #           DNTMGOTO,DNLSTADR AND EBANK ARE INITIALIZED IN STARTSUB(FRESH
